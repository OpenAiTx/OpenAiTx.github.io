Bintang Anda dapat membuat Open Ai Tx menjadi lebih baik

Pendahuluan

OpenAiTx adalah alat terjemahan otomatis 20 bahasa untuk readme & wiki proyek GitHub Anda.

Kenapa?

  • Sekali terjemah menggunakan LLM kualitas premium dan hasilnya dapat diakses bebas oleh komunitas open-source.
  • Mendukung pencarian SEO multibahasa Google/Bing, alat terjemah klien tidak bisa melakukannya.
  • Gratis & Sumber Terbuka selamanya.
  • Submit sekali dan auto-update ke depannya, sangat menghemat waktu jika Anda maintainer proyek.

Cara Mulai

  • Ganti URL GitHub dengan OpenAiTx, misal https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • Submit proyek Anda.
  • Klik dan salin badge atau teks gaya yang Anda suka.
  • Update ke file readme Anda.

atau

  • Akses https://openaitx.com
  • Submit tautan proyek Anda.
  • Klik dan salin badge atau teks gaya yang Anda suka.
  • Update ke file readme Anda.
