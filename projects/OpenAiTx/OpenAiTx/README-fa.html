معرفی

OpenAiTx ابزاری برای ترجمه خودکار ۲۰ زبان برای فایل readme و ویکی پروژه‌های شما در گیت‌هاب است.

چرا؟

  • ترجمه یک‌باره با استفاده از مدل زبانی سطح بالا و دسترسی رایگان به نتیجه برای جامعه متن‌باز.
  • پشتیبانی از جستجوی سئو چندزبانه گوگل/بینگ؛ ابزار ترجمه سمت کاربر این قابلیت را ندارد.
  • همیشه رایگان و متن‌باز.
  • ارسال یک‌باره و بروزرسانی خودکار در آینده، اگر نگهدارنده پروژه هستید زمان شما را ذخیره می‌کند.

شروع کنید

  • آدرس GitHub را با OpenAiTx جایگزین کنید، مثال: https://github.com/OpenAiTx/OpenAiTx → https://openaitx.com/OpenAiTx/OpenAiTx
  • پروژه خود را ارسال کنید.
  • سبک نشان‌ها یا متنی را که دوست دارید کلیک و کپی کنید.
  • در فایل readme خود به‌روزرسانی کنید.
