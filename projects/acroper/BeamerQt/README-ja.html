BeamerQt

BeamerQTは、スライドに関連するLaTeXコードを手動で編集することなく、Beamerプレゼンテーションの作成を容易にするユーザーフレンドリーなグラフィカルインターフェースです。レイアウトの定義、コンテンツ（テキスト、ブロック、画像など）の挿入、テーマの高度な設定の一部を構成する包括的な機能を提供します。BeamerQTは初心者から上級のLaTeXユーザーまで、コードではなく内容に集中して素晴らしいプレゼンテーションを作成することができます。

ライブラリ要件：

  • PyQt6

  • PyMuPDF

機能

BeamerQTは、LaTeXコードを追加することなく、望まれるほとんどのBeamer/LaTeX機能に簡単にアクセスできるグラフィカルユーザーインターフェースを特徴としています。

テーマエディタ（新機能！）

テンプレートのソースコードを編集します。

  • 付属ファイルの添付をサポート。
  • LaTeX生成時に添付ファイルを抽出。
  • カスタム.xmlファイルを簡単に共有可能。
  • 近い将来の新機能。

レイアウト選択

BeamerQTは、PowerPointやLibreOffice Impressなどの一般的なプレゼンテーションツールに触発された一連の定義済みレイアウトスキームを提供します。列やブロックのLaTeXコードを手動で書く代わりに、ユーザーは単にレイアウトを選択するだけで、BeamerQTが必要なBeamerの列とブロックを自動的に挿入および管理します。

主な機能は以下の通りです：

  • 選択したレイアウトに基づく列とブロックの自動作成。
  • 手動のコード編集なしで列幅を動的に調整できるスライダーコントロール。
  • 最小限のユーザー操作でスライドレイアウトのシームレスな再構成。

スライド

各スライドにはタイトル、サブタイトル、およびコンテンツ用の複数のブロックを含めることができます。さらに、スライドは新しいセクションまたはサブセクションとして設定でき、対応するセクションまたはサブセクションのタイトルがプレゼンテーションに自動挿入されます。

主な特徴：

  • スライドのタイトルとサブタイトルの簡単入力。
  • スライドをセクションまたはサブセクションとしてマークし、プレゼンテーションの構造化を支援。
  • スライドレベルの書式設定オプションの自動調整。

スライド一覧

スライド一覧は、プレゼンテーション全体の概要を提供し、各スライドの位置、番号、タイトルを表示します。セクションとサブセクションが明確にマークされており、ナビゲーションと整理を支援します。ユーザーは必要に応じてスライドの順序を変更、複製、コピー、削除でき、効率的なスライド管理を実現します。

ブロック

ブロックはBeamerQTにおける基本的なコンテンツ単位です。ツールは、Block、Alert、Example、またはプレーンテキストブロックなどの様々なブロックタイプを、シンプルなラジオボタンインターフェースでサポートします。この方法により、LaTeXコマンドを手動で記述する必要がなくなります。

ブロックの主な機能：

  • ブロックタイプの選択（Block、Alert、Example、またはプレーンテキスト）。
  • レイアウト内でのブロックの位置調整コントロール。
  • スライドからブロックを削除する専用ボタン。
  • 各ブロックにタイトルフィールドとテキスト入力エリアを備え、複数行入力をサポート。
  • ユーザー入力に応じて自動改行または手動改行の保持。
  • 複数のサブブロック

サブブロック

サブブロックは、ブロック内でより細かいコンテンツの整理を可能にします。各ブロックは少なくとも1つのサブブロック（通常はテキストベース）を含み、ユーザーは必要に応じて複数のサブブロックを追加できます。

サブブロックの機能：

  • 最大4列までの配置が可能で、水平ナビゲーションボタンでサブブロックの順序を変更可能。
  • カラム幅を調整するスライダーにより柔軟なレイアウトカスタマイズを提供。
  • 各サブブロックの配置制御（左揃え、中央揃え、右揃え、またはデフォルト）。

画像サブブロック

画像サブブロックは画像挿入用の専用サブブロックタイプです。BeamerQTはビットマップファイル（例：.jpg、.png）、ベクター画像（.svg）、および.pdfファイルをサポートします。.svg画像については、BeamerQTがInkscapeを使用して.pdf形式に変換し、最終プレゼンテーションへのシームレスな統合を実現します（Linuxシステムでのみテスト済み）。

画像サブブロックの機能：

  • サブブロックの幅の割合として調整可能な画像サイズ。
  • 一貫してスケールされたビジュアルのためのレイアウト変更への自動適応。
  • 複数の画像フォーマットに対応し、プレゼンテーションデザインの柔軟性を確保。

フロントマター

フロントマターセクションは、プレゼンテーション全体の設定を簡単に構成することを可能にします。ユーザーはプレゼンテーションのタイトル、著者名を定義し、LaTeXプリアンブルをカスタマイズできます。さらに、全体の外観や構造をより洗練させるための高度な機能も有効化可能です。例えば：

  • アスペクト比の変更（4:3から16:9へ）。
  • 各セクションのタイトルフレームの作成。
  • 各セクションのアウトラインフレームの自動生成。

LaTeX生成

ユーザーが内容とレイアウトに満足したら、Generate LaTeXボタンをクリックしてプレゼンテーションをLaTeX形式でエクスポートし、pdflatexを実行してPDFをコンパイルします。生成されたPDFは表示され、即時にレビューが可能です。

LaTeX folderボタンは出力ディレクトリを開き、さらなるカスタマイズや他のツールとの統合を可能にします。なお、各LaTeX生成時に出力フォルダ内のファイルは上書きされます。

ファイル形式

BeamerQTは.bqtファイル形式を使用します。これは基本的に必要なメタデータをすべて含む圧縮ディレクトリであり、以下を含みます：

  • プレゼンテーションの詳細を含むXMLファイル。
  • スライドのプレビュー画像。
  • （将来の機能）埋め込み画像やカスタムテーマにより移植性を確保。

この方法により、.bqtファイルは異なるシステム間で重要なデータを失うことなく簡単に共有、バックアップ、編集が可能です。

インストール

  • Windows

MikTexをインストールしてください： https://miktex.org/download

BeamerQTをインストールしてください： https://sourceforge.net/projects/beamerqt/

  • Linux
お使いのディストリビューションに応じて、python3、TexLive、Inkscapeをインストールしてください：

Debian系： `` apt-get install python3 texlive-beamer inkscape ` PyQt6とPyMuPDFをインストールする： ` pip install pyqt6 pymupdf `

BeamerQTを実行する：

このリポジトリからBeamerQTのソースコードをダウンロードします。 main.pyファイルがあるディレクトリでターミナルを開きます。 ` python3 main.py ``
