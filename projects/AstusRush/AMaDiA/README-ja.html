AMaDiA

Astusの数学表示アプリケーション

現在の機能:
  • (Shift+)AltGr+キーを使って数学記号を入力できます（AMaDiA_ReplacementTablesを参照）
  • タブ1 : 科学計算機
  • +,-,±,×,÷,√,^,°, 積分 f(x) dx, 微分 d(f(x))/dx, y'(x), 複素数, sin(), cos(), tan(), exp(), log(), π, E などで計算
  • 未知変数のある方程式を解く
  • 微分方程式を解く（特解を含む）
  • 二つの項の等価性をテスト
  • 数学Unicode記号を使って高速入力！
  • オプションで単位をオンにすると convert_to(7foot+3inch,m) のような操作が可能
  • 詳細な使い方はヘルプ/例を参照し、例を対応するタブの入力欄にコピー＆ペーストしてください
  • タブ2 : 入力をLaTeXに変換して表示
  • タブ3 : 2Dグラフのプロット（3Dおよび複素グラフは開発中）
  • タブ4 : 行列・ベクトルを使った方程式および連立一次方程式
  • システムコントロールウィンドウ : システム制御に関するすべて（開発中）
  • LANチャットプログラム（改良が必要）
  • 同一ネットワーク内の他ユーザーと方程式を共有し、協働作業を容易にすることを目的
  • サーバーとクライアントのアプリケーションが別々に存在
  • テキストおよびテキストファイルの送信が可能
  • 入力欄へのドラッグ＆ドロップでネットワーク内の他ユーザーとテキストファイルを共有
  • サーバーには一部のモデレーションツール（ミュート/キック/バン）あり
  • 注意：メッセージは現在暗号化されていません！機密データを送信しないでください！！！
  • カスタムウィンドウフレーム

インストール方法

Windowsの場合:
  • https://gitforwindows.org/ をインストール
  • https://www.anaconda.com/distribution/ をインストール
  • "AMaDiA.bat" と "AMaDiA_INSTALL.bat" の2つのファイルを作成し、WINDOWS-INSTALL内の2つのファイルのテキストを右クリックして「編集」でコピー＆ペースト
  • "AMaDiA_INSTALL.bat" をダブルクリックしてAMaDiAをインストール（インストール場所はユーザーフォルダ。両方のファイルを編集すれば変更可能）
  • 以降はAMaDiA.batをダブルクリックするとAMaDiAの更新と起動が行われます
（免責事項：これらの.batファイルは自己責任で使用してください。害を及ぼす可能性は低いと思いますが、バッチファイルの専門家ではなく、すべての事態を予見できません）
