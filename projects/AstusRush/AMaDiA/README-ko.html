AMaDiA

Astus의 수학 표현 애플리케이션

현재 기능:
  • (Shift+)AltGr+키 조합으로 수학 기호 입력 (AMaDiA_ReplacementTables 참고)
  • 탭 1 : 과학 계산기
  • +,-,±,×,÷,√,^,°, 적분 f(x) dx, 미분 d(f(x))/dx, y'(x), 복소수, sin(), cos(), tan(), exp(), log(), π, E 등으로 계산
  • 미지수가 있는 방정식 풀기
  • 미분 방정식 해법 (특수해 포함)
  • 두 식의 동등성 검사
  • 수학 유니코드 기호를 사용해 더 빠르게 입력 가능!
  • 옵션에서 단위 활성화하여 convert_to(7foot+3inch,m) 같은 변환 수행
  • 자세한 사용법은 도움말/예제 참고, 예제를 복사하여 해당 탭 입력란에 붙여넣기 가능
  • 탭 2 : 입력을 LaTeX로 변환 및 표시
  • 탭 3 : 2D 그래프 플롯 (3D 및 복소수 그래프는 작업 중)
  • 탭 4 : 행렬 및 벡터 방정식과 연립 선형 방정식
  • 시스템 제어 창 : 시스템 제어 관련 모든 기능 (작업 중)
  • LAN 채팅 프로그램 (개선 필요)
  • 동일 네트워크 내 사용자 간 방정식 공유로 협업 용이
  • 서버 및 클라이언트 애플리케이션 분리
  • 텍스트 및 텍스트 파일 전송 가능
  • 입력란에 드래그 앤 드롭으로 네트워크 내 모든 사용자와 텍스트 파일 공유
  • 서버에 일부 관리자 도구 포함 (음소거/강제퇴장/차단)
  • 경고: 메시지는 현재 암호화되지 않음! 민감한 데이터 전송 금지!!!
  • 사용자 정의 창 프레임

설치 방법

Windows에서:
  • https://gitforwindows.org/ 설치
  • https://www.anaconda.com/distribution/ 설치
  • "AMaDiA.bat"와 "AMaDiA_INSTALL.bat" 두 파일 생성 후 WINDOWS-INSTALL 폴더 내 두 파일의 텍스트를 복사하여 각각 우클릭 후 수정 선택해 붙여넣기
  • "AMaDiA_INSTALL.bat"을 더블클릭하여 AMaDiA 설치 (설치 위치는 사용자 폴더, 두 파일 편집으로 변경 가능)
  • 이후부터는 AMaDiA.bat를 더블클릭하면 업데이트 후 실행
(면책 조항: 이 .bat 파일들은 사용자의 책임 하에 사용하세요. 문제가 발생할 가능성은 낮지만, 배치 파일 전문가가 아니므로 모든 상황을 예측할 수 없습니다)

리눅스에서 [작업 중]:
  • sudo apt-install git
  • https://www.anaconda.com/distribution/ 또는 sudo apt-install ... 설치
  • [설치용 콘솔 명령어 및 업데이트/시작용 별칭 곧 제공 예정(추후?)]

"단기" 로드맵

가까운 미래에 계획된 기능들 (우선순위 순서) ("단기"라는 단어는 매우 상대적임):

  • 시스템 제어 창 개선

장기 로드맵

미래에 계획된 기능들 (우선순위 순서):
  • 탭 4를 더욱 쉽게 사용 가능하게 만들기
  • 간단한 텍스트 편집기
  • 더 나은 단위 처리 (예: 자동 단순화)
  • PDE 자동 인식 기능으로 사용자가 더 이상 "pdsolve(...)"에 직접 넣지 않아도 되도록
  • 기록 저장/불러오기
  • PDF 뷰어
