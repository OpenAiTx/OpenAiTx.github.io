🐌 Snail - Bộ khung phát triển nhanh Spring Boot nhẹ

Snail là một framework phát triển nhanh microservice hiện đại dựa trên Spring Boot 3,
cung cấp sẵn các khả năng cốt lõi như xác thực, ủy quyền, gateway, dịch vụ tài nguyên, hỗ trợ doanh nghiệp xây dựng ứng dụng cloud-native an toàn, mở rộng dễ dàng.

🚀 Sử dụng ngay | An toàn, tin cậy | Mở rộng dễ dàng | Frontend & Backend tích hợp

🌟 Tính năng

  • ✅ Dựa trên công nghệ mới nhất: Spring Boot 3.5.5 + Spring Cloud 2025.0.0 +
Spring Cloud Alibaba 2023.0.3.3
  • 🔐 Xác thực & ủy quyền OAuth2: Tích hợp Spring Security + JWT, hỗ trợ xác thực tập trung & kiểm soát quyền chi tiết
  • 🌐 Kiến trúc microservice: Tách module, gồm xác thực, ủy quyền, gateway, dịch vụ tài nguyên, hỗ trợ triển khai độc lập & mở rộng ngang
  • ⚡ Gateway hiệu năng cao: Dựa trên Spring Cloud Gateway, thực hiện định tuyến, giới hạn tốc độ, ngắt mạch, xác thực
  • 🧩 Hỗ trợ cloud-native: Tương thích Nacos, Sentinel, Seata, hỗ trợ phát hiện dịch vụ, cấu hình tập trung, giao dịch phân tán
  • 🎨 Giao diện quản trị hiện đại: Xây dựng bằng shadcn-vue/ui
mang lại giao diện quản lý tối giản, đáp ứng
  • 📚 Tài liệu đầy đủ: Bao gồm thiết kế kiến trúc, hướng dẫn triển khai, tài liệu API và hướng dẫn phát triển mở rộng
