Snail হৈছে Spring Boot 3-ৰ আধাৰত নিৰ্মিত
আধুনিক মাইক্র’চাৰ্ভিচ দ্ৰুত উন্নয়ন ফ্ৰেমৱৰ্ক, যি অটোমেটিক পৰিচয়, অনুমোদন, গেটৱে, ৰিছৰ্ছ সেৱা আদি মূল ক্ষমতা উপলব্ধ কৰে, আৰু উদ্যোগক সুৰক্ষিত, প্ৰসাৰিত কৰিব পৰা ক্লাউড-নেটিভ এপ্লিকেশ্যন নিৰ্মাণত সহায় কৰে।

🌟 বৈশিষ্ট্য

  • ✅ সৰ্বশেষ টেকন’লজি ষ্টেকৰ আধাৰত: Spring Boot 3.5.5 + Spring Cloud 2025.0.0 +
Spring Cloud Alibaba 2023.0.3.3
  • 🔐 OAuth2 পৰিচয় আৰু অনুমোদন: Spring Security + JWT ইন্টিগ্ৰেচন, একত্ৰিত পৰিচয় আৰু সূক্ষ্ম অনুমতি নিয়ন্ত্ৰণ
  • 🌐 মাইক্র’চাৰ্ভিচ স্থাপত্য: মডুলাৰ বিভাজন, পৰিচয়, অনুমোদন, গেটৱে, ৰিছৰ্ছ সেৱা অন্তর্ভুক্ত, স্বতন্ত্ৰ ডিপ্লয়মেন্ট আৰু স্কেলিং সমৰ্থন
  • ⚡ উচ্চ-দক্ষ গেটৱে: Spring Cloud Gateway-ৰ আধাৰত ৰাউটিং, সীমা নিয়ন্ত্ৰণ, ফিউজ, অথৰাইজেচন
  • 🧩 ক্লাউড-নেটিভ সমৰ্থন: Nacos, Sentinel, Seata-ৰ সৈতে সংযুক্ত, সেৱা সন্ধান, কনফিগাৰেচন চেন্টাৰ, বিতৰিত লেনদেন সমৰ্থন
  • 🎨 আধুনিক পৰিচালনা ড্যাশব’ৰ্ড: shadcn-vue/ui
ৰ আধাৰত নিৰ্মিত পৰিষ্কৃত, প্ৰতিক্ৰিয়াশীল ড্যাশব’ৰ্ড ইণ্টাৰফেইচ
  • 📚 সম্পূৰ্ণ ডকুমেণ্টেশ্যন: স্থাপত্য ডিজাইন, ডিপ্লয়মেন্ট গাইড, API ডকুমেণ্ট আৰু কাষ্টমাইজেশন নির্দেশনা অন্তর্ভুক্ত
