🐌 Snail - اسکلت سبک توسعه سریع مبتنی بر Spring Boot

Snail یک فریمورک توسعه سریع مدرن برای میکروسرویس‌ها بر پایه Spring Boot 3 است
که امکانات هسته‌ای مانند احراز هویت، مجوزدهی، گیت‌وی، سرویس منابع و غیره را به صورت آماده ارائه می‌دهد و به شرکت‌ها کمک می‌کند تا برنامه‌های ابری ایمن و مقیاس‌پذیر بسازند.

🚀 آماده استفاده | ایمن و مطمئن | قابل توسعه | یکپارچه‌سازی فرانت‌اند و بک‌اند

🌟 ویژگی‌ها

  • ✅ بر پایه جدیدترین تکنولوژی‌ها: Spring Boot 3.5.5 + Spring Cloud 2025.0.0 +
Spring Cloud Alibaba 2023.0.3.3
  • 🔐 احراز هویت و مجوزدهی OAuth2: یکپارچه با Spring Security و JWT، پشتیبانی از احراز هویت متمرکز و کنترل دسترسی با دقت بالا
  • 🌐 معماری میکروسرویس: جداسازی ماژولار، شامل ماژول‌های احراز هویت، مجوزدهی، گیت‌وی و سرویس منابع، پشتیبانی از استقرار مستقل و مقیاس‌پذیری افقی
  • ⚡ گیت‌وی پرسرعت: مبتنی بر Spring Cloud Gateway با قابلیت مسیریابی، محدودیت نرخ، قطع‌کننده مدار و اعتبارسنجی
  • 🧩 پشتیبانی از ابر بومی: سازگار با Nacos، Sentinel، Seata و پشتیبانی از کشف سرویس، مرکز پیکربندی و تراکنش توزیع‌شده
  • 🎨 پنل مدیریتی مدرن: رابط کاربری تمیز و واکنش‌گرا مبتنی بر shadcn-vue/ui
برای مدیریت پشتیبان
  • 📚 مستندات جامع: شامل طراحی معماری، راهنمای استقرار، مستندات API و راهنمای توسعه ثانویه
