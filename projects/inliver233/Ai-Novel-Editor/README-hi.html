एआई नोवेल एडिटर

PyQt6 पर आधारित एआई सहायक उपन्यास संपादक, जो बुद्धिमान स्वतः पूर्णता, परियोजना प्रबंधन, अवधारणा संगठन आदि जैसे पेशेवर लेखन उपकरण प्रदान करता है।

स्थापना

  • परियोजना क्लोन करें

git clone https://github.com/inliver233/Ai-Novel-Editor.git
cd ai-novel-editor
  • वर्चुअल वातावरण बनाना

python -m venv venv
venv\Scripts\activate     # Windows

  • निर्भरता स्थापित करें
pip install -r requirements.txt

  • प्रोग्राम चलाएँ
python src/main.py

मुख्य विशेषताएँ

परियोजना प्रबंधन

  • पदानुक्रमित दस्तावेज़ संरचना (कृति > अध्याय > दृश्य)
  • परियोजना आयात/निर्यात (TEXT, MARKDOWN, DOCX, PDF, HTML प्रारूपों का समर्थन)
  • स्वचालित बैकअप और संस्करण नियंत्रण
  • बहु-परियोजना प्रबंधन

AI पूर्ति कार्य

  • तीन पूर्ति मोड: स्वचालित पूर्ति, मैन्युअल पूर्ति (सिफारिश: एक बार टैब दबाएँ पूर्ति ट्रिगर करें, फिर टैब दबाएँ पूर्ति लागू करें), पूर्ति निष्क्रिय करें
  • तीन संदर्भ मोड:
  • त्वरित मोड: हल्का संदर्भ, तेज़ प्रतिक्रिया
  • संतुलित मोड: मध्यम संदर्भ, गुणवत्ता और गति में संतुलन
  • वैश्विक मोड: पूर्ण परियोजना संदर्भ, सर्वोत्तम परिणाम
  • कई AI सेवाओं का समर्थन: OpenAI, Claude, Tongyi Qianwen, Zhipu AI, DeepSeek, Groq आदि

रूपरेखा प्रबंधन

  • दृश्य रूपरेखा वृक्ष संरचना
  • ड्रैग-एंड-ड्रॉप क्रमबद्ध और स्तर समायोजन
  • दस्तावेज़ त्वरित नेविगेशन
  • रूपरेखा विश्लेषण और अनुकूलन सुझाव

API कॉन्फ़िगरेशन

  • एकीकृत AI कॉन्फ़िगरेशन केंद्र
  • कई सेवा प्रदाता पूर्व-सेट कॉन्फ़िगरेशन
  • कनेक्शन परीक्षण सुविधा
  • कॉन्फ़िगरेशन योजना सहेजना और आयात/निर्यात

स्मार्ट पूर्ति सेटिंग्स

  • समायोज्य ट्रिगर विलंब
  • पूर्ति लंबाई सीमा
  • संदर्भ लंबाई सेटिंग
  • स्ट्रीमिंग प्रतिक्रिया समर्थन

संकेत शब्द संपादन

  • अंतर्निहित कई लेखन टेम्पलेट्स
  • कस्टम संकेत शब्द टेम्पलेट्स
  • टेम्पलेट श्रेणी प्रबंधन
  • मोड-विशिष्ट टेम्पलेट कॉन्फ़िगरेशन

इंटरफ़ेस सुविधाएँ

  • डार्क और लाइट थीम
  • तीन-स्तंभ लेआउट (प्रोजेक्ट ट्री, एडिटर, कांसेप्ट पैनल)
  • फोल्डेबल साइडबार
  • फुलस्क्रीन लेखन मोड
  • फोकस मोड (वाक्य, पैराग्राफ, टाइपराइटर, ज़ेन मोड, इमर्सिव मोड)

खोज और प्रतिस्थापन

  • पूरे प्रोजेक्ट में टेक्स्ट खोजें
  • रेगुलर एक्सप्रेशन सपोर्ट
  • बैच रिप्लेसमेंट फीचर
  • एडवांस्ड सर्च विकल्प

अवधारणा प्रबंधन

  • पात्र, स्थान, वस्तु स्वतः पहचान
  • अवधारणा संबंध प्रबंधन
  • टैग और श्रेणी प्रणाली
  • फास्ट कांसेप्ट इनसर्ट

शॉर्टकट कुंजी

  • Ctrl+N: नया प्रोजेक्ट बनाएं
  • Ctrl+O: प्रोजेक्ट खोलें
  • Ctrl+S: दस्तावेज़ सेव करें
  • F11: फुलस्क्रीन मोड
  • Tab: AI ऑटो-कम्प्लीट मैन्युअली ट्रिगर करें (मैन्युअल मोड में)
  • Ctrl+F: खोजें
  • Ctrl+H: खोजें और प्रतिस्थापित करें
  • Ctrl+Shift+H: एडवांस्ड खोज और प्रतिस्थापन

सिस्टम आवश्यकताएँ

  • Python 3.8+
  • PyQt6
  • ऑपरेटिंग सिस्टम: Windows, macOS, Linux
