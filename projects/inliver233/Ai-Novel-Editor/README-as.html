AI Novel Editor

PyQt6-ত আধাৰিত AI সহায়ক উপন্যাস সম্পাদনা সঁজুলি, যি বুদ্ধিমান স্বয়ংসম্পূৰ্ণ, প্ৰকল্প পৰিচালনা, ধাৰণা সংগঠন আদিৰ দৰে ব্যৱসায়িক লিখনি সঁজুলি প্ৰদান কৰে।

ইনষ্টল

  • প্ৰকল্পটো ক্লোন কৰক

git clone https://github.com/inliver233/Ai-Novel-Editor.git
cd ai-novel-editor

  • ভাৰ্চুৱেল পৰিৱেশ সৃষ্টি কৰক
python -m venv venv
venv\Scripts\activate     # Windows

  • নিৰ্ভৰশীলতা স্থাপন কৰক
pip install -r requirements.txt

  • প্ৰগ্ৰামটো চলাও
python src/main.py

মুখ্য বৈশিষ্ট্যসমূহ

প্ৰকল্প ব্যৱস্থাপনা

  • স্তৰবদ্ধ নথিপত্ৰৰ গঠন (কৃতিত্ব > অধ্যায় > দৃশ্য)
  • প্ৰকল্প আমদানি আৰু ৰপ্তানি (TEXT, MARKDOWN, DOCX, PDF, HTML ফৰ্মেট সমৰ্থিত)
  • স্বয়ংক্ৰিয় সঞ্চয় আৰু সংস্কৰণ নিয়ন্ত্ৰণ
  • বহু প্ৰকল্প ব্যৱস্থাপনা

AI পূৰণ ফিচাৰ

  • তিনিটা পূৰণ ধৰণ: স্বয়ংক্ৰিয় পূৰণ, মেনুয়েল পূৰণ (প্ৰস্তাৱিত: এটা টেব চেপি এটা পূৰণ, পুনৰ টেব চেপি প্ৰয়োগ কৰক), পূৰণ নিষ্ক্ৰিয়
  • তিনিটা প্ৰসংগ ধৰণ:
  • দ্ৰুত মোড: হালকা প্ৰসংগ, দ্ৰুত প্ৰতিক্ৰিয়া
  • সন্থুলিত মোড: মধ্যম প্ৰসংগ, গুণমান আৰু গতিৰ সন্থুলন
  • গ্লোবেল মোড: সম্পূৰ্ণ প্ৰকল্প প্ৰসংগ, সৰ্বোত্তম ফলাফল
  • বহু AI সেৱা সমৰ্থিত: OpenAI, Claude, Tongyi Qianwen, Zhiyu AI, DeepSeek, Groq আদি

খাকচিকা ব্যৱস্থাপনা

  • দৃশ্যমান খাকচিকা গছৰ গঠন
  • টানিবলৈ আৰু স্তৰ সলনি কৰিবলৈ
  • দ্ৰুত নথিপত্ৰ নেভিগেচন
  • খাকচিকা বিশ্লেষণ আৰু উন্নতিৰ পৰামৰ্শ

API সংৰূপণ

  • একত্ৰিত AI সংৰূপণ কেন্দ্ৰ
  • বহু সেৱাদাতা প্ৰিসেট সংৰূপণ
  • সংযোগ পৰীক্ষা ফিচাৰ
  • সংৰূপণ সংৰক্ষণ আৰু আমদানি-ৰপ্তানি

বুদ্ধিমান পূৰণ সংৰূপণ

  • পৰিবর্তনযোগ্য ট্ৰিগাৰ দেৰী
  • পূৰণ দৈৰ্ঘ্য সীমা
  • প্ৰসংগ দৈৰ্ঘ্য সংৰূপণ
  • ষ্ট্ৰিমিং প্ৰতিক্ৰিয়া সমৰ্থন

প্ৰম্পট সম্পাদনা

  • বহুবিধ লিখনি টেমপ্লেট সংলগ্ন
  • কাষ্টম প্ৰম্পট টেমপ্লেট
  • টেমপ্লেট শ্ৰেণী ব্যৱস্থাপনা
  • মোড নিৰ্দিষ্ট টেমপ্লেট সংৰূপণ

ইণ্টাৰফেচ ফিচাৰসমূহ

  • উজ্জ্বল আৰু অন্ধকাৰ ডুয়েল থিম
  • তিনি-কলাম লেআউট (প্ৰকল্প গছ, সম্পাদনা কৰা, ধাৰণা পেনেল)
  • ভাঁজ কৰিব পৰা পাৰ্শ্ববৰ
  • সম্পূৰ্ণ স্ক্ৰীণ লেখনী মোড
  • মনোযোগ মোড (বাক্য, অনুচ্ছেদ, টাইপৰাইটাৰ, জেন মোড, ডুব মুড)

সন্ধান আৰু প্ৰতিস্থাপন

  • সমগ্ৰ প্ৰকল্পত পাঠ্য সন্ধান
  • নিয়মিত অভিব্যক্তি সমৰ্থন
  • একেলগে প্ৰতিস্থাপন সুবিধা
  • উন্নত সন্ধান বিকল্পসমূহ

ধাৰণা পৰিচালনা

  • স্বয়ংক্ৰিয়ভাৱে চৰিত্ৰ, স্থান, বস্তু চিনাক্তকৰণ
  • ধাৰণা সম্পৰ্ক ব্যৱস্থাপনা
  • টেগ আৰু শ্ৰেণীবিভাজন ব্যৱস্থা
  • ধাৰণা দ্ৰুত সন্নিবেশ

চমু চাবি

  • Ctrl+N: নতুন প্ৰকল্প
  • Ctrl+O: প্ৰকল্প খোলক
  • Ctrl+S: নথিপত্ৰ সংৰক্ষণ
  • F11: সম্পূৰ্ণ স্ক্ৰীণ মোড
  • Tab: AI সম্পূৰ্ণৰূপে ম্যানুৱেলভাৱে চলাও (ম্যানুৱেল মোডত)
  • Ctrl+F: সন্ধান
  • Ctrl+H: সন্ধান আৰু প্ৰতিস্থাপন
  • Ctrl+Shift+H: উন্নত সন্ধান আৰু প্ৰতিস্থাপন

  • অপাৰেটিং ছিষ্টেম: Windows, macOS, Linux
