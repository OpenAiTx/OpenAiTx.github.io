3. กำหนดเวลาการทำงานเอง

แก้ไข random_cron.sh เปลี่ยนเวลาที่ตรวจสอบในคำสั่ง if ให้เป็นเวลา UTC (เวลาปักกิ่ง -8 ชั่วโมง) เช่น เวลาปักกิ่ง 8 โมงเช้า คือ UTC 0 นาฬิกา เวลาที่ต้องการ -8 คือเวลา UTC

หมายเหตุ

  • ทำงานวันละ 7 ครั้ง ควบคุมโดย random_cron.sh นาทีเป็นค่าสุ่ม

  • จำนวนบัญชีและรหัสผ่านต้องตรงกัน มิฉะนั้นจะใช้ไม่ได้!!!

  • เวลาที่ตั้งต้องเป็นเวลา UTC!

  • สมัคร server酱 ได้ที่ คลิกที่นี่

  • หาก Alipay ไม่อัปเดตก้าว ให้ไปที่ Mi Fit->การตั้งค่า->บัญชี->ออกจากระบบ->ล้างข้อมูล แล้วเข้าสู่ระบบใหม่และเชื่อมโยงกับบุคคลที่สามใหม่

  • Mi Fit จะไม่อัปเดตก้าว มีเฉพาะบัญชีที่เชื่อมโยงเท่านั้นที่จะซิงค์!!!!!!

  • โปรด Fork branch หลัก เพื่อป้องกัน bug ที่ไม่จำเป็น

  • หมายเหตุ บัญชีที่ใช้ไม่ใช่ [Mi Account] แต่เป็นบัญชี [Mi Fit]
