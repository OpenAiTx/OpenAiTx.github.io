3. Настройка времени запуска

Редактируйте random_cron.sh Измените время в условии if на UTC, то есть Пекинское время минус 8. Например, 8:00 по Пекину — это 0:00 по UTC, нужное время запуска -8 и получите время UTC

Важные замечания

  • Запуск происходит семь раз в день, управляется random_cron.sh, минуты выбираются случайным образом

  • Количество аккаунтов и паролей должны совпадать, иначе не будет работать!!!

  • Время запуска должно быть в UTC!

  • Регистрация на server酱 по ссылке

  • Если шаги не обновляются в Alipay, перейдите в Mi Fit->Настройки->Аккаунт->Выйти из аккаунта->Очистить данные, затем заново войдите и повторно привяжите сторонние сервисы

  • Mi Fit не обновляет шаги, только связанные сервисы синхронизируют шаги!!!!!!

  • Используйте Fork основной ветки, чтобы избежать ненужных ошибок.

  • Обратите внимание, что аккаунт — это не [Mi аккаунт], а аккаунт [Mi Fit].
