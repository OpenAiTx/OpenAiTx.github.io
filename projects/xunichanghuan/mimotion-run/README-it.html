3. Personalizzazione dell'orario di avvio

Modifica random_cron.sh Modifica l'orario della condizione if in tempo UTC, ovvero orario Pechino -8, ad esempio le 8:00 di Pechino sono le 0:00 UTC, l'orario desiderato -8 corrisponde all'orario UTC

Note importanti

  • Si esegue 7 volte al giorno, controllato da random_cron.sh, i minuti sono valori casuali

  • Il numero di account e password deve corrispondere esattamente, altrimenti non funzionerà!!!

  • L'orario di avvio deve essere in UTC!

  • Registrazione server酱 Clicca qui

  • Se su Alipay i passi non vengono aggiornati, vai su Xiaomi Mi Fit -> Impostazioni -> Account -> Esci -> Cancella dati, poi accedi di nuovo e ricollega le terze parti

  • Xiaomi Mi Fit non aggiornerà i passi, solo quelli associati verranno sincronizzati!!!!!!

  • Si consiglia di Forkare il branch principale per evitare bug indesiderati.

  • Attenzione, l'account non è quello [Xiaomi], ma quello di [Xiaomi Mi Fit].
