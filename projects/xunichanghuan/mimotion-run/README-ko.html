3. 실행 시간 커스터마이즈

random_cron.sh를 편집 내부 if 조건문의 시간은 UTC 기준(즉, 베이징 시간-8), 예를 들어 베이징 8시는 UTC 0시, 실행 시간-8이 UTC 시간

주의사항

  • 매일 7회 실행, random_cron.sh에서 제어, 분 단위는 랜덤 값

  • 다중 계정 수와 비밀번호가 반드시 일치해야 사용 가능!!!

  • 실행 시간은 UTC 시간이어야 함!

  • server酱 가입 주소 여기 클릭

  • 알리페이 걸음 수가 업데이트 안 되면, 샤오미 운동->설정->계정->로그아웃->데이터 초기화 후 재로그인, 제3자 서비스 재연결

  • 샤오미 운동 자체는 걸음 수를 업데이트하지 않으며, 연동된 서비스만 동기화됨!!!!!!

  • 사용 시 반드시 메인 브랜치를 Fork 하세요. 불필요한 버그 방지.

  • 계정은 [샤오미 계정]이 아니라 [샤오미 운동]의 계정임을 유의하세요.
