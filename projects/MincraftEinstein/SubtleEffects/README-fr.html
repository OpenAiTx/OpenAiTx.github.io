Effets Subtils

📘 Description

Ajout de nombreux nouveaux détails subtils via des particules et quelques sons. Tous peuvent être activés ou désactivés, et certains peuvent être personnalisés davantage. Le mod inclut également un système de suppression des particules inutiles et une distance de rendu des particules. Ce mod a été conçu principalement côté client, mais quelques fonctionnalités nécessitent un accès serveur pour fonctionner, voir la FAQ pour plus de détails.

✨ Quelques fonctionnalités clés

  • Suppression des particules inutiles
  • Distance de rendu des particules
  • Les blocs liés au feu (feu, feux de camp, bougies, torches, etc.) ont des étincelles
  • Les villageois et les joueurs ronflent et ont des particules Z
  • Les slimes laissent une traînée de slime
  • Les blocs de pierre lumineuse et de redstone émettent des particules de poussière
  • Allay et Vex émettent de la magie
  • La tonte d’un mouton fait tomber du duvet
  • Les entités en feu ont des étincelles, de la fumée et des flammes
  • La fumée vanilla a été remplacée par des textures mises à jour similaires à la fumée de feu de camp
  • Les joueurs avec 3 cœurs ou moins ont un battement de cœur
  • Les joueurs avec 3 points de faim ou moins ont des gargouillements d’estomac
  • Porter une tête de mob applique le shader spectateur pour ce mob (actuellement seulement pour la tête de creeper et la tête d’Enderman de Supplemenaries)
  • Les joueurs et quelques mobs ont des nuages de poussière en courant
  • Les mobs ont des nuages de poussière lorsqu’ils touchent le sol
  • Et bien plus encore...

Une liste complète des fonctionnalités est disponible sur le wiki.

Nécessite Fzzy Config.

📚 F.A.Q

Q : Peut-on désactiver la fonctionnalité {XYZ} ?
R : Oui, toutes les fonctionnalités peuvent être désactivées dans le menu de configuration. Si vous constatez qu’une fonctionnalité/changement ne peut pas être désactivé, merci de le signaler comme un bug.

Q : Comment désactiver les Nuages de Poussière ?
R : Les configurations des nuages de poussière se trouvent dans la section de configuration Entities/Dust Clouds.

Q : Comment désactiver les particules d’objets lâchés ?
R : Les configurations des particules d’objets lâchés se trouvent dans la section Items/Item Rarity et peuvent être désactivées en modifiant le paramètre Particle Display Type à OFF.

Q : Supporterez-vous Forge ?
R : Absolument pas, Forge 1.20.1 est la seule version sur laquelle je supporterai Forge.

Q : Dans quel environnement de jeu le mod fonctionne-t-il ?
R : Le mod est conçu pour fonctionner côté client, cependant, certaines fonctionnalités et détails supplémentaires ne fonctionnent que si le mod est également installé sur le serveur. Si installé sur le serveur, tous les clients, y compris ceux sur des serveurs LAN, devront avoir le mod installé pour rejoindre. Pour une liste des fonctionnalités nécessitant un accès serveur, visitez le wiki. Note : Forge 1.20.1 ne supporte PAS le mod fonctionnant uniquement côté client.

Q : Ce mod fonctionne-t-il avec Fancy Block Particles, Particle Core, Visuality, Effective, Effectual ou Particular ?
R : Oui

Q : Acceptez-vous les suggestions/demandes de fonctionnalités ?
R : Oui, les suggestions et retours sont toujours appréciés. Vous pouvez fournir vos retours/suggestions via une issue GitHub, les commentaires CurseForge, ou sur mon serveur Discord. Merci de garder vos suggestions un peu dans l’esprit de Minecraft, par exemple.

Q : Le mod ne fonctionne pas / aucune particule n’apparaît.
R : Vérifiez vos Paramètres Vidéo pour vous assurer que le réglage des Particules n’est pas sur minimal.

Q : Allez-vous supporter Forge au-delà de la version 1.20.1 ?
R : Absolument pas
