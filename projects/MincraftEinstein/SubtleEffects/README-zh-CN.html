SubtleEffects

⭐ 195 stars Simplified Chinese by MincraftEinstein

微妙效果

📘 描述

通过粒子和一些声音添加许多新的细微细节。所有这些都可以切换，有些还能进一步自定义。该模组还内置了粒子剔除和粒子渲染距离。此模组设计主要为客户端侧，但有一些功能需要服务器访问才能工作，详情请参阅常见问题。

✨ 主要功能

  • 粒子剔除
  • 粒子渲染距离
  • 与火相关的方块（火、篝火、蜡烛、火把等）会有火花
  • 村民和玩家会打鼾并有 Z 轴粒子
  • 史莱姆留下史莱姆轨迹
  • 荧石和红石方块会发出尘埃粒子
  • 盟友和恶魂会发出魔法粒子
  • 剪羊毛时会掉落绒毛
  • 燃烧实体会有火花、烟雾和火焰
  • 原版烟雾已被类似篝火烟雾的更新纹理替代
  • 生命值低于 3 颗心的玩家会有心跳声
  • 饥饿值低于 3 的玩家会有肚子叫声
  • 戴怪物头颅会应用该怪物的观察者着色器（目前仅支持 Supplemenaries 的苦力怕头颅和末影人头颅）
  • 玩家和部分生物有奔跑时的尘土云
  • 生物落地时会有尘土云
  • 以及更多……

完整功能列表可见于维基。

需要 Fzzy Config。

📚 常见问题解答

问： {XYZ} 功能可以禁用吗？
答： 可以，所有功能都可以在配置菜单中禁用。如果发现某个功能/更改无法禁用，请报告为错误。

问： 如何禁用尘云效果？
答： 尘云的配置位于 Entities/Dust Clouds 配置部分。

问： 如何禁用掉落物品粒子？
答： 掉落物品粒子的配置位于 Items/Item Rarity 配置部分，可以通过将 Particle Display Type 改为 OFF 来禁用。

问： 会支持 Forge 吗？
答： 绝对不会，Forge 1.20.1 是我唯一支持 Forge 的版本。

问： 该模组运行于什么游戏环境？
答： 该模组设计为客户端运行，但某些额外功能和细节仅在服务器也安装模组时生效。如果安装在服务器上，所有客户端，包括局域网服务器上的客户端，都需要安装该模组才能加入。需要服务器支持的功能列表，请访问 wiki。 注意：Forge 1.20.1 不支持仅客户端运行该模组。

问： 该模组是否兼容 Fancy Block Particles、Particle Core、Visuality、Effective、Effectual 或 Particular？
答： 是的。
