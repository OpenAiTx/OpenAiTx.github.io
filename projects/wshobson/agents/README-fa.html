مجموعه عامل‌های فرعی Claude Code

یک مجموعه جامع از عامل‌های فرعی هوش مصنوعی تخصصی برای Claude Code، طراحی‌شده برای بهبود جریان‌های کاری توسعه با تخصص دامنه‌ای.

مرور کلی

این مخزن شامل ۷۵ عامل فرعی تخصصی است که قابلیت‌های Claude Code را گسترش می‌دهند. هر عامل فرعی، متخصص در یک حوزه خاص است و بر اساس زمینه به صورت خودکار یا به صورت صریح فراخوانی می‌شود. همه عامل‌ها با مدل‌های خاص Claude بر اساس پیچیدگی وظیفه برای عملکرد بهینه و مقرون‌به‌صرفه بودن پیکربندی شده‌اند.

عامل‌های فرعی موجود

توسعه و معماری

  • backend-architect - طراحی APIهای RESTful، مرزهای میکروسرویس و طرح‌های پایگاه داده
  • frontend-developer - ساخت کامپوننت‌های React، پیاده‌سازی طرح‌های واکنش‌گرا و مدیریت وضعیت سمت کلاینت
  • ui-ux-designer - ایجاد طراحی رابط کاربری، وایرفریم‌ها و سیستم‌های طراحی
  • mobile-developer - توسعه اپلیکیشن‌های React Native یا Flutter با ادغام‌های بومی
  • graphql-architect - طراحی طرح‌های GraphQL، resolverها و فدراسیون
  • architect-reviewer - بازبینی تغییرات کد جهت سازگاری معماری و الگوها

متخصصان زبان‌ها

  • python-pro - نوشتن کد پایتون ایدئوماتیک با ویژگی‌ها و بهینه‌سازی‌های پیشرفته
  • ruby-pro - نوشتن کد روبی ایدئوماتیک با متاپروگرامینگ، الگوهای Rails، توسعه gem و فریمورک‌های تست
  • golang-pro - نوشتن کد Go ایدئوماتیک با goroutineها، کانال‌ها و رابط‌ها
  • rust-pro - نوشتن Rust ایدئوماتیک با الگوهای مالکیت، lifetimes و پیاده‌سازی traitها
  • c-pro - نوشتن کد C کارآمد با مدیریت صحیح حافظه و فراخوانی‌های سیستمی
  • cpp-pro - نوشتن کد ++C ایدئوماتیک با ویژگی‌های مدرن، RAII، smart pointerها و الگوریتم‌های STL
  • javascript-pro - تسلط بر جاوااسکریپت مدرن با ES6+، الگوهای async و APIهای Node.js
  • typescript-pro - تسلط بر TypeScript با انواع پیشرفته، genericها و ایمنی شدید نوع‌دهی
  • php-pro - نوشتن کد PHP ایدئوماتیک با ویژگی‌های مدرن و بهینه‌سازی عملکرد
  • java-pro - تسلط بر جاوای مدرن با streamها، هم‌زمانی و بهینه‌سازی JVM
  • elixir-pro - نوشتن کد Elixir ایدئوماتیک با الگوهای OTP، برنامه‌نویسی تابعی و فریمورک‌های Phoenix
  • csharp-pro - نوشتن کد C# مدرن با ویژگی‌های پیشرفته و بهینه‌سازی .NET
  • scala-pro - تسلط بر توسعه Scala در سطح سازمانی با برنامه‌نویسی تابعی، سیستم‌های توزیع‌شده و پردازش داده حجیم
  • flutter-expert - تسلط بر توسعه Flutter با Dart، ویجت‌ها و ادغام‌های پلتفرم
  • unity-developer - ساخت بازی‌های Unity با اسکریپت‌های بهینه و تنظیم عملکرد
  • minecraft-bukkit-pro - تسلط بر توسعه افزونه‌های سرور Minecraft با APIهای Bukkit، Spigot و Paper
  • ios-developer - توسعه اپلیکیشن‌های بومی iOS با Swift/SwiftUI
  • sql-pro - نوشتن کوئری‌های پیچیده SQL، بهینه‌سازی برنامه‌های اجرایی و طراحی طرح‌های نرمال‌سازی‌شده

زیرساخت و عملیات

  • devops-troubleshooter - رفع اشکال مشکلات تولید، تحلیل لاگ‌ها و رفع شکست‌های استقرار
  • deployment-engineer - پیکربندی پایپ‌لاین‌های CI/CD، کانتینرهای Docker و استقرار ابری
  • cloud-architect - طراحی زیرساخت AWS/Azure/GCP و بهینه‌سازی هزینه‌های ابری
  • database-optimizer - بهینه‌سازی کوئری‌های SQL، طراحی ایندکس‌های کارآمد و مدیریت مهاجرت پایگاه داده
  • database-admin - مدیریت عملیات پایگاه داده، پشتیبان‌گیری، replication و پایش
  • terraform-specialist - نوشتن ماژول‌های پیشرفته Terraform، مدیریت فایل‌های state و پیاده‌سازی بهترین روش‌های IaC
  • incident-responder - مدیریت رخدادهای تولید با فوریت و دقت
  • network-engineer - رفع اشکال اتصال شبکه، پیکربندی تعادل بار و تحلیل الگوهای ترافیک
  • dx-optimizer - متخصص تجربه توسعه‌دهنده برای بهبود ابزارها، راه‌اندازی و جریان‌های کاری

کیفیت و امنیت

  • بازبین کد - بازبینی تخصصی کد با تمرکز عمیق بر امنیت پیکربندی و قابلیت اطمینان در محیط تولید
  • ممیز امنیتی - بررسی کد برای آسیب‌پذیری‌ها و اطمینان از انطباق با OWASP
  • خودکارساز تست - ایجاد مجموعه‌های تست جامع با تست‌های واحد، یکپارچه‌سازی و انتها به انتها
  • مهندس عملکرد - پروفایل‌گیری برنامه‌ها، بهینه‌سازی گلوگاه‌ها و پیاده‌سازی راهبردهای کشینگ
  • اشکال‌زدا - متخصص اشکال‌زدایی برای خطاها، شکست تست‌ها و رفتارهای غیرمنتظره
  • کارآگاه خطا - جستجوی الگوهای خطا، ردپاهای پشته و ناهنجاری‌ها در لاگ‌ها و پایگاه‌های کد
  • متخصص جستجو - پژوهشگر وب حرفه‌ای با استفاده از تکنیک‌های پیشرفته جستجو و سنتز

داده و هوش مصنوعی

  • دانشمند داده - متخصص تحلیل داده برای کوئری‌های SQL، عملیات BigQuery و بینش داده‌ای
  • مهندس داده - ساخت پایپ‌لاین‌های ETL، انبار داده و معماری‌های استریمینگ
  • مهندس هوش مصنوعی - ساخت برنامه‌های LLM، سیستم‌های RAG و پایپ‌لاین‌های پرامپت
  • مهندس یادگیری ماشین - پیاده‌سازی پایپ‌لاین‌های ML، سروینگ مدل و مهندسی ویژگی‌ها
  • مهندس MLOps - ساخت پایپ‌لاین‌های ML، ردیابی آزمایش‌ها و رجیستری مدل‌ها
  • مهندس پرامپت - بهینه‌سازی پرامپت‌ها برای LLMها و سیستم‌های هوش مصنوعی

حوزه‌های تخصصی

  • مستندساز API - ایجاد مشخصات OpenAPI/Swagger و مستندات توسعه‌دهندگان
  • ادغام پرداخت - ادغام Stripe، PayPal و پردازشگرهای پرداخت
  • تحلیلگر کمی - ساخت مدل‌های مالی، بک‌تست راهبردهای معاملاتی و تحلیل داده‌های بازار
  • مدیر ریسک - پایش ریسک پرتفوی، مقادیر R و محدودیت‌های موقعیت

  • مدیر زمینه - مدیریت زمینه میان چندین ایجنت و کارهای بلندمدت

مستندسازی

  • معمار مستندات - ایجاد مستندات فنی جامع از پایگاه‌های کد موجود

کسب‌وکار و بازاریابی

  • تحلیلگر کسب‌وکار - تحلیل شاخص‌ها، تهیه گزارش و ردیابی KPIها
  • بازاریاب محتوا - نگارش پست وبلاگ، محتوای شبکه‌های اجتماعی و خبرنامه ایمیلی
  • متخصص منابع انسانی - شریک برای جذب نیرو، ورود/خروج، مرخصی، عملکرد، سیاست‌های تطبیقی و روابط کارکنان
  • خودکارساز فروش - تهیه ایمیل‌های سرد، پیگیری‌ها و قالب‌های پیشنهادی
  • پشتیبان مشتری - رسیدگی به تیکت‌های پشتیبانی، پاسخ به سوالات متداول و ایمیل‌های مشتریان
  • مشاور حقوقی - تهیه سیاست‌های حریم خصوصی، شرایط استفاده، سلب مسئولیت و اطلاعیه‌های حقوقی

  • seo-structure-architect - ساختار محتوا، سلسله‌مراتب تیترها و نشانه‌گذاری اسکیما را بهینه می‌کند
  • seo-snippet-hunter - محتوا را برای اسنیپت‌های ویژه و ویژگی‌های SERP قالب‌بندی می‌کند
  • seo-content-refresher - عناصر قدیمی را شناسایی و پیشنهادات بروزرسانی محتوا ارائه می‌دهد
  • seo-cannibalization-detector - چندین صفحه را برای همپوشانی کلیدواژه و تضاد بررسی می‌کند
  • seo-authority-builder - محتوا را برای سیگنال‌های E-E-A-T و شاخص‌های اعتماد تحلیل می‌کند
  • seo-content-writer - بر اساس کلیدواژه‌ها و خلاصه‌ها، محتوای بهینه‌شده سئو می‌نویسد
  • seo-content-planner - طرح‌های محتوا، خوشه‌های موضوعی و تقویم‌ها ایجاد می‌کند

Claude Code به طور خودکار با توجه به زمینه وظیفه و توضیحات زیرعامل، وظیفه را به زیرعامل مناسب واگذار می‌کند.

"Get the performance-engineer to optimize this bottleneck"

پیکربندی مدل

از نسخه 1.0.64 کلود کد، زیرعامل‌ها می‌توانند مشخص کنند از کدام مدل کلود باید استفاده کنند. این امکان، واگذاری وظایف بر اساس پیچیدگی به‌صورت مقرون‌به‌صرفه را فراهم می‌کند:

  • پیچیدگی پایین (هایکو): وظایف ساده مانند تحلیل داده پایه، تولید مستندات و پاسخ‌های استاندارد
  • پیچیدگی متوسط (سونت): وظایف توسعه، بازبینی کد، تست و کارهای مهندسی استاندارد
  • پیچیدگی بالا (اوپوس): وظایف حیاتی مانند ممیزی امنیتی، بازبینی معماری، پاسخ به رخدادها و مهندسی هوش مصنوعی/یادگیری ماشین

مدل‌های موجود (با نام‌گذاری ساده‌شده از نسخه 1.0.64 کلود کد):
  • haiku - سریع و مقرون‌به‌صرفه برای وظایف ساده
  • sonnet - عملکرد متوازن برای بیشتر کارهای توسعه
  • opus - توانمندترین مدل برای تحلیل‌های پیچیده و وظایف حیاتی

اگر مدلی مشخص نشود، زیرعامل از مدل پیش‌فرض سیستم استفاده خواهد کرد.

الگوهای ارکستراسیون عامل

کلود کد به‌صورت خودکار عوامل را با استفاده از این الگوهای رایج هماهنگ می‌کند:

گردش کار ترتیبی

User Request → Agent A → Agent B → Agent C → Result
Example: "Build a new API feature"
backend-architect → frontend-developer → test-automator → security-auditor

Parallel Execution

User Request → Agent A + Agent B (simultaneously) → Merge Results
Example: "Optimize application performance" 
performance-engineer + database-optimizer → Combined recommendations

شاخه‌بندی شرطی

User Request → Analysis → Route to appropriate specialist
Example: "Fix this bug"
debugger (analyzes) → Routes to: backend-architect OR frontend-developer OR devops-troubleshooter

بررسی و اعتبارسنجی

Primary Agent → Review Agent → Final Result
Example: "Implement payment processing"
payment-integration → security-auditor → Validated implementation
