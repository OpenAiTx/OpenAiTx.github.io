क्लॉड कोड स्लैश कमांड्स

क्लॉड कोड के लिए प्रोडक्शन-रेडी स्लैश कमांड्स, जो बुद्धिमान ऑटोमेशन के माध्यम से विकास को तेज करती हैं।

52 कमांड्स इस प्रकार संगठित हैं:

  • 🤖 वर्कफ़्लो: जटिल कार्यों के लिए मल्टी-सबएजेंट ऑर्केस्ट्रेशन
  • 🔧 टूल्स: विशिष्ट ऑपरेशनों के लिए सिंगल-पर्पज़ यूटिलिटीज

🤝 क्लॉड कोड सबएजेंट्स की आवश्यकता है

ये कमांड क्लॉड कोड सबएजेंट्स के साथ ऑर्केस्ट्रेशन क्षमताओं के लिए काम करती हैं।

इंस्टॉलेशन

cd ~/.claude
git clone https://github.com/wshobson/commands.git
git clone https://github.com/wshobson/agents.git  # Required for subagent orchestration

उपलब्ध कमांड्स

  • 🤖 वर्कफ़्लो - जटिल कार्यों के लिए कई सबएजेंट्स का समन्वयन करें
  • 🔧 टूल्स - विशिष्ट कार्यों के लिए एकल-उद्देश्यीय कमांड्स

उपयोग

कमांड्स tools/ और workflows/ डायरेक्टरी में व्यवस्थित हैं। इन्हें डायरेक्टरी प्रीफ़िक्स के साथ उपयोग करें:

/tools:api-scaffold user management with authentication
/tools:security-scan check for vulnerabilities
/workflows:feature-development implement chat functionality

नोट: यदि आप बिना उपसर्गों के कमांड्स का उपयोग करना पसंद करते हैं, तो आप निर्देशिकाओं को समतल कर सकते हैं: 

cp tools/*.md .
cp workflows/*.md .

Claude Code स्वचालित रूप से संदर्भ के आधार पर प्रासंगिक कमांड का सुझाव देता है।

🤖 वर्कफ्लो

जटिल कार्यों के लिए मल्टी-सबएजेंट ऑर्केस्ट्रेशन:

फीचर डेवलपमेंट

  • feature-development - बैकएंड, फ्रंटएंड, टेस्टिंग और डिप्लॉयमेंट सबएजेंट्स पूरी फीचर बनाते हैं
  • full-review - कई समीक्षा सबएजेंट्स व्यापक कोड विश्लेषण प्रदान करते हैं
  • smart-fix - समस्याओं का विश्लेषण करता है और उपयुक्त विशेषज्ञ सबएजेंट्स को सौंपता है

डेवलपमेंट प्रोसेस

  • git-workflow - शाखा रणनीतियों और PR टेम्पलेट्स के साथ प्रभावी Git वर्कफ्लो लागू करता है
  • improve-agent - सबएजेंट प्रदर्शन को प्रॉम्प्ट ऑप्टिमाइजेशन के माध्यम से बढ़ाता है
  • legacy-modernize - विशेष सबएजेंट्स की मदद से लेगेसी कोडबेस को आधुनिक बनाता है
  • ml-pipeline - डेटा और ML इंजीनियरिंग सबएजेंट्स के साथ ML पाइपलाइन बनाता है
  • multi-platform - समन्वित सबएजेंट्स के साथ क्रॉस-प्लेटफार्म ऐप्स बनाता है
  • workflow-automate - CI/CD, मॉनिटरिंग, और डिप्लॉयमेंट वर्कफ्लो को स्वचालित करता है

सबएजेंट-ऑर्केस्ट्रेटेड वर्कफ्लो

  • full-stack-feature - बैकएंड, फ्रंटएंड, और मोबाइल सबएजेंट्स के साथ मल्टी-प्लेटफार्म फीचर
  • security-hardening - विशेष सबएजेंट्स के साथ सुरक्षा-प्रथम कार्यान्वयन
  • data-driven-feature - डेटा साइंस सबएजेंट्स के साथ ML-संचालित फीचर्स
  • performance-optimization - प्रदर्शन सबएजेंट्स के साथ एंड-टू-एंड ऑप्टिमाइजेशन
  • incident-response - संचालन सबएजेंट्स के साथ प्रोडक्शन घटना समाधान

🔧 उपकरण (एकल-उद्देश्य कमांड)

एआई और मशीन लर्निंग

  • ai-assistant - प्रोडक्शन-तैयार एआई असिस्टेंट्स और चैटबॉट्स बनाएं
  • ai-review - एआई/एमएल कोडबेस के लिए विशेष समीक्षा
  • langchain-agent - आधुनिक पैटर्न के साथ LangChain/LangGraph एजेंट्स बनाएं
  • ml-pipeline - MLOps के साथ एंड-टू-एंड ML पाइपलाइन बनाएं
  • prompt-optimize - प्रदर्शन और गुणवत्ता के लिए एआई प्रॉम्प्ट्स को ऑप्टिमाइज़ करें

आर्किटेक्चर और कोड गुणवत्ता

  • code-explain - जटिल कोड की विस्तृत व्याख्याएं उत्पन्न करें
  • code-migrate - कोड को भाषाओं, फ्रेमवर्क या संस्करणों के बीच माइग्रेट करें
  • refactor-clean - रखरखाव और प्रदर्शन के लिए कोड को रिफैक्टर करें
  • tech-debt - तकनीकी ऋण का विश्लेषण और प्राथमिकता निर्धारण करें

डेटा और डेटाबेस

  • data-pipeline - स्केलेबल डेटा पाइपलाइन संरचनाओं का डिजाइन करें
  • data-validation - व्यापक डेटा सत्यापन प्रणालियाँ लागू करें
  • db-migrate - उन्नत डेटाबेस माइग्रेशन रणनीतियाँ

देवऑप्स और इंफ्रास्ट्रक्चर

  • deploy-checklist - परिनियोजन कॉन्फ़िगरेशन और चेकलिस्ट उत्पन्न करें
  • docker-optimize - उन्नत कंटेनर अनुकूलन रणनीतियाँ
  • k8s-manifest - प्रोडक्शन-ग्रेड क्यूबरनेट्स डिप्लॉयमेंट्स
  • monitor-setup - व्यापक निगरानी और ऑब्ज़रवेबिलिटी सेटअप करें
  • slo-implement - सेवा स्तर उद्देश्य (SLOs) लागू करें
  • workflow-automate - विकास और संचालन वर्कफ़्लो स्वचालित करें

विकास और परीक्षण

  • api-mock - विकास और परीक्षण के लिए यथार्थवादी API मॉक बनाएं
  • api-scaffold - पूर्ण कार्यान्वयन स्टैक के साथ प्रोडक्शन-रेडी API एंडपॉइंट्स उत्पन्न करें
  • test-harness - फ्रेमवर्क डिटेक्शन के साथ व्यापक टेस्ट सूट बनाएं

सुरक्षा और अनुपालन

  • accessibility-audit - व्यापक अभिगम्यता परीक्षण और सुधार
  • compliance-check - नियामक अनुपालन सुनिश्चित करें (GDPR, HIPAA, SOC2)
  • security-scan - स्वचालित समाधान के साथ व्यापक सुरक्षा स्कैनिंग

डीबगिंग और विश्लेषण

  • debug-trace - उन्नत डीबगिंग और ट्रेसिंग रणनीतियाँ
  • error-analysis - गहन त्रुटि पैटर्न विश्लेषण और समाधान रणनीतियाँ
  • error-trace - प्रोडक्शन त्रुटियों का पता लगाएँ और निदान करें
  • issue - सुव्यवस्थित GitHub/GitLab समस्याएँ बनाएं

निर्भरता और कॉन्फ़िगरेशन

  • config-validate - एप्लिकेशन कॉन्फ़िगरेशन का सत्यापन और प्रबंधन करें
  • deps-audit - सुरक्षा और लाइसेंसिंग के लिए निर्भरताओं का ऑडिट करें
  • deps-upgrade - परियोजना निर्भरताओं का सुरक्षित रूप से उन्नयन करें

दस्तावेज़ीकरण और सहयोग

लागत अनुकूलन

  • smart-debug - डिबगर और प्रदर्शन सबएजेंट्स के साथ गहन डिबगिंग
  • multi-agent-optimize - कई सबएजेंट्स के साथ फुल-स्टैक अनुकूलन
  • context-save - प्रोजेक्ट संदर्भ को संदर्भ-प्रबंधक सबएजेंट द्वारा सुरक्षित करें

उन्नत कमांड्स

सुरक्षा और देवऑप्स

#### /security-scan

स्वचालित सुधार के साथ व्यापक सुरक्षा स्कैनिंग।

  • मल्टी-टूल स्कैनिंग: Bandit, Safety, Trivy, Semgrep, ESLint Security, Snyk
  • स्वचालित सुधार: सामान्य कमजोरियों का स्वचालित समाधान
  • CI/CD एकीकरण: GitHub Actions/GitLab CI के लिए सुरक्षा गेट्स
  • कंटेनर स्कैनिंग: इमेज कमजोरियों का विश्लेषण
  • सीक्रेट डिटेक्शन: GitLeaks और TruffleHog एकीकरण

#### /docker-optimize

एडवांस्ड कंटेनर ऑप्टिमाइजेशन रणनीतियाँ।

  • स्मार्ट ऑप्टिमाइजेशन: विश्लेषण करता है और सुधार सुझाता है
  • मल्टी-स्टेज बिल्ड्स: फ्रेमवर्क-विशिष्ट ऑप्टिमाइज्ड Dockerfiles
  • आधुनिक टूल्स: तेज बिल्ड के लिए BuildKit, Bun, UV
  • सुरक्षा सख्ती: Distroless इमेज, गैर-रूट उपयोगकर्ता
  • क्रॉस-कमांड एकीकरण: /api-scaffold आउटपुट्स के साथ कार्य करता है

#### /k8s-manifest

प्रोडक्शन-ग्रेड Kubernetes डिप्लॉयमेंट्स।

  • एडवांस्ड पैटर्न्स: Pod सुरक्षा मानक, नेटवर्क नीतियाँ, OPA
  • GitOps रेडी: FluxCD और ArgoCD कॉन्फ़िगरेशन
  • ऑब्ज़र्वेबिलिटी: Prometheus ServiceMonitors, OpenTelemetry
  • ऑटो-स्केलिंग: HPA, VPA, और क्लस्टर ऑटोस्केलर कॉन्फ़िग्स
  • सर्विस मेष: Istio/Linkerd एकीकरण

फ्रंटएंड और डेटा

#### /db-migrate

उन्नत डेटाबेस माइग्रेशन रणनीतियाँ।

  • मल्टी-डेटाबेस: PostgreSQL, MySQL, MongoDB, DynamoDB
  • शून्य-डाउनटाइम: ब्लू-ग्रीन डिप्लॉयमेंट्स, रोलिंग माइग्रेशन
  • इवेंट सोर्सिंग: CDC के लिए Kafka/Kinesis एकीकरण
  • क्रॉस-प्लेटफ़ॉर्म: पॉलीग्लॉट परसिस्टेंस को संभालता है
  • मॉनिटरिंग: माइग्रेशन हेल्थ चेक और रोलबैक
