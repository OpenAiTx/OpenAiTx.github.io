🤖 वर्कफ़्लो

जटिल कार्यों के लिए मल्टी-सबएजेंट ऑर्केस्ट्रेशन:

फीचर डेवलपमेंट

  • feature-development - बैकएंड, फ्रंटएंड, टेस्टिंग, और डिप्लॉयमेंट सबएजेंट्स द्वारा संपूर्ण फीचर्स का निर्माण
  • full-review - कई रिव्यू सबएजेंट्स द्वारा व्यापक कोड विश्लेषण
  • smart-fix - समस्याओं का विश्लेषण करता है और उपयुक्त विशेषज्ञ सबएजेंट्स को सौंपता है

डेवलपमेंट प्रक्रियाएँ

  • git-workflow - ब्रांचिंग रणनीतियों और PR टेम्पलेट्स के साथ प्रभावी Git वर्कफ़्लो लागू करता है
  • improve-agent - प्रॉम्प्ट ऑप्टिमाइज़ेशन के माध्यम से सबएजेंट प्रदर्शन को बढ़ाता है
  • legacy-modernize - विशेष सबएजेंट्स की सहायता से लेगेसी कोडबेस को आधुनिक बनाता है
  • ml-pipeline - डेटा और ML इंजीनियरिंग सबएजेंट्स के साथ ML पाइपलाइनों का निर्माण करता है
  • multi-platform - समन्वित सबएजेंट्स के साथ क्रॉस-प्लेटफ़ॉर्म ऐप्स बनाता है
  • workflow-automate - CI/CD, मॉनिटरिंग, और डिप्लॉयमेंट वर्कफ़्लो को स्वचालित करता है

सबएजेंट-ऑर्केस्ट्रेटेड वर्कफ़्लो

  • full-stack-feature - बैकएंड, फ्रंटएंड और मोबाइल सबएजेंट्स के साथ मल्टी-प्लेटफ़ॉर्म फीचर्स
  • security-hardening - विशेष सबएजेंट्स के साथ सुरक्षा-प्रथम कार्यान्वयन
  • data-driven-feature - डेटा साइंस सबएजेंट्स के साथ ML-संचालित फीचर्स
  • performance-optimization - प्रदर्शन सबएजेंट्स के साथ एंड-टू-एंड ऑप्टिमाइज़ेशन
  • incident-response - ऑप्स सबएजेंट्स के साथ प्रोडक्शन इनसिडेंट समाधान

🔧 टूल्स (एकल-उद्देश्य कमांड्स)

AI और मशीन लर्निंग

  • ai-assistant - प्रोडक्शन-रेडी AI असिस्टेंट्स और चैटबॉट्स बनाएं
  • ai-review - AI/ML कोडबेस के लिए विशेष समीक्षा
  • langchain-agent - आधुनिक पैटर्न के साथ LangChain/LangGraph एजेंट बनाएं
  • ml-pipeline - MLOps के साथ एंड-टू-एंड ML पाइपलाइनों का निर्माण करें
  • prompt-optimize - प्रदर्शन और गुणवत्ता के लिए AI प्रॉम्प्ट्स का ऑप्टिमाइजेशन करें

आर्किटेक्चर और कोड गुणवत्ता

  • code-explain - जटिल कोड की विस्तृत व्याख्या उत्पन्न करें
  • code-migrate - कोड को विभिन्न भाषाओं, फ्रेमवर्क्स, या संस्करणों में माइग्रेट करें
  • refactor-clean - अनुरक्षणीयता और प्रदर्शन के लिए कोड का रिफैक्टर करें
  • tech-debt - तकनीकी कर्ज का विश्लेषण और प्राथमिकता निर्धारण

डेटा और डेटाबेस

  • data-pipeline - स्केलेबल डेटा पाइपलाइन आर्किटेक्चर डिज़ाइन करें
  • data-validation - व्यापक डेटा सत्यापन प्रणालियाँ लागू करें
  • db-migrate - उन्नत डेटाबेस माइग्रेशन रणनीतियाँ

देवऑप्स और इन्फ्रास्ट्रक्चर

  • deploy-checklist - डिप्लॉयमेंट कॉन्फ़िगरेशन और चेकलिस्ट बनाएं
  • docker-optimize - उन्नत कंटेनर ऑप्टिमाइजेशन रणनीतियाँ
  • k8s-manifest - प्रोडक्शन-ग्रेड Kubernetes डिप्लॉयमेंट्स
  • monitor-setup - व्यापक मॉनिटरिंग और ऑब्जर्वेबिलिटी सेटअप करें
  • slo-implement - सेवा स्तर उद्देश्यों (SLOs) को लागू करें
  • workflow-automate - विकास और परिचालन वर्कफ़्लो को स्वचालित करें

विकास और परीक्षण

  • api-mock - विकास और परीक्षण के लिए यथार्थवादी API मॉक बनाएं
  • api-scaffold - संपूर्ण इम्प्लीमेंटेशन स्टैक के साथ प्रोडक्शन-रेडी API एंडपॉइंट्स बनाएं
  • test-harness - फ्रेमवर्क डिटेक्शन के साथ व्यापक टेस्ट सूट बनाएं

सुरक्षा और अनुपालन

  • accessibility-audit - व्यापक एक्सेसिबिलिटी टेस्टिंग और सुधार
  • compliance-check - नियामक अनुपालन सुनिश्चित करें (GDPR, HIPAA, SOC2)
  • security-scan - स्वचालित सुधार के साथ व्यापक सुरक्षा स्कैनिंग

डिबगिंग और विश्लेषण

  • debug-trace - उन्नत डिबगिंग और ट्रेसिंग रणनीतियाँ
  • error-analysis - गहन त्रुटि पैटर्न विश्लेषण और समाधान रणनीतियाँ
  • error-trace - प्रोडक्शन त्रुटियों का ट्रेस और निदान करें
  • issue - सुव्यवस्थित GitHub/GitLab इश्यू बनाएं

डिपेंडेंसी और कॉन्फ़िगरेशन

  • config-validate - एप्लिकेशन कॉन्फ़िगरेशन को मान्य और प्रबंधित करें
  • deps-audit - सुरक्षा और लाइसेंसिंग के लिए डिपेंडेंसी का ऑडिट करें
  • deps-upgrade - प्रोजेक्ट डिपेंडेंसी को सुरक्षित रूप से अपग्रेड करें

दस्तावेज़ीकरण और सहयोग

  • doc-generate - व्यापक दस्तावेज़ीकरण उत्पन्न करें
  • git-workflow - प्रभावी Git वर्कफ़्लो लागू करें
  • pr-enhance - गुणवत्ता जांच के साथ पुल रिक्वेस्ट्स को बेहतर बनाएं

लागत अनुकूलन

  • cost-optimize - क्लाउड और इन्फ्रास्ट्रक्चर लागत को ऑप्टिमाइज करें

ऑनबोर्डिंग और सेटअप

  • onboard - नए टीम सदस्यों के लिए विकास परिवेश सेटअप करें

सबएजेंट टूल्स

  • multi-agent-review - विशेषज्ञ सबएजेंट्स के साथ बहु-दृष्टिकोण कोड समीक्षा
  • smart-debug - डिबगर और प्रदर्शन सबएजेंट्स के साथ गहन डिबगिंग
  • multi-agent-optimize - कई सबएजेंट्स के साथ फुल-स्टैक ऑप्टिमाइजेशन
  • context-save - context-manager सबएजेंट का उपयोग कर प्रोजेक्ट संदर्भ सहेजें
  • context-restore - निरंतरता के लिए सेव्ड संदर्भ पुनर्स्थापित करें

उन्नत कमांड्स

सुरक्षा और डेवऑप्स

#### /security-scan

स्वचालित समाधान के साथ व्यापक सुरक्षा स्कैनिंग।

  • मल्टी-टूल स्कैनिंग: Bandit, Safety, Trivy, Semgrep, ESLint Security, Snyk
  • स्वचालित सुधार: सामान्य कमजोरियों का ऑटो-सुधार
  • CI/CD एकीकरण: GitHub Actions/GitLab CI के लिए सुरक्षा गेट्स
  • कंटेनर स्कैनिंग: इमेज कमजोरियों का विश्लेषण
  • सीक्रेट डिटेक्शन: GitLeaks और TruffleHog एकीकरण

#### /docker-optimize

उन्नत कंटेनर ऑप्टिमाइज़ेशन रणनीतियाँ।

  • स्मार्ट ऑप्टिमाइज़ेशन: विश्लेषण करता है और सुधार के सुझाव देता है
  • मल्टी-स्टेज बिल्ड्स: फ्रेमवर्क-विशिष्ट ऑप्टिमाइज़्ड डॉकरफाइल्स
  • आधुनिक टूल्स: तेज बिल्ड्स के लिए BuildKit, Bun, UV
  • सुरक्षा सुदृढ़ीकरण: Distroless इमेजेज, नॉन-रूट उपयोगकर्ता
  • क्रॉस-कमांड एकीकरण: /api-scaffold आउटपुट्स के साथ काम करता है

#### /k8s-manifest

प्रोडक्शन-ग्रेड क्यूबरनेट्स डिप्लॉयमेंट्स।

  • उन्नत पैटर्न्स: पाड सुरक्षा मानक, नेटवर्क पॉलिसीज, OPA
  • GitOps तैयार: FluxCD और ArgoCD कॉन्फ़िगरेशन
  • अवलोकन: Prometheus ServiceMonitors, OpenTelemetry
  • ऑटो-स्केलिंग: HPA, VPA, और क्लस्टर ऑटोस्केलर कॉन्फ़िग्स
  • सर्विस मेष: Istio/Linkerd एकीकरण

फ्रंटएंड और डेटा

#### /db-migrate

उन्नत डाटाबेस माइग्रेशन रणनीतियाँ।

  • मल्टी-डाटाबेस: PostgreSQL, MySQL, MongoDB, DynamoDB
  • शून्य-डाउनटाइम: ब्लू-ग्रीन डिप्लॉयमेंट्स, रोलिंग माइग्रेशन
  • इवेंट सोर्सिंग: CDC के लिए Kafka/Kinesis एकीकरण
  • क्रॉस-प्लेटफॉर्म: पॉलीग्लॉट परसिस्टेंस को संभालता है
  • मॉनिटरिंग: माइग्रेशन हेल्थ चेक्स और रोलबैक

कमांड चयन:

  • वर्कफ़्लोज़: जटिल फीचर्स के लिए बहु-सबएजेंट समन्वय

  • ज्ञात समस्याओं के लिए टूल्स से शुरू करें
