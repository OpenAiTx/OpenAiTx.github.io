Claude Code Slash Commando's

Productieklaar slash commando's voor Claude Code die ontwikkeling versnellen via intelligente automatisering.

52 commando's georganiseerd als:

  • 🤖 Workflows: Multi-subagent orkestratie voor complexe taken
  • 🔧 Tools: Hulpmiddelen met één doel voor specifieke handelingen

🤝 Vereist Claude Code Subagents

Deze commando's werken met de Claude Code Subagents voor orkestratiemogelijkheden.

Installatie

cd ~/.claude
git clone https://github.com/wshobson/commands.git
git clone https://github.com/wshobson/agents.git  # Required for subagent orchestration

Beveiliging & DevOps

#### /security-scan

Uitgebreide beveiligingsscans met geautomatiseerde oplossingen.

  • Multi-Tool Scanning: Bandit, Safety, Trivy, Semgrep, ESLint Security, Snyk
  • Automatische Oplossingen: Veelvoorkomende kwetsbaarheden automatisch verholpen
  • CI/CD Integratie: Beveiligingspoorten voor GitHub Actions/GitLab CI
  • Container Scanning: Analyse van kwetsbaarheden in images
  • Geheime Detectie: Integratie met GitLeaks en TruffleHog

#### /docker-optimize

Geavanceerde container optimalisatiestrategieën.

  • Slimme Optimalisatie: Analyseert en doet verbetervoorstellen
  • Multi-Stage Builds: Framework-specifiek geoptimaliseerde Dockerfiles
  • Moderne Tools: BuildKit, Bun, UV voor snellere builds
  • Beveiligingsverharding: Distroless images, non-root gebruikers
  • Cross-Command Integratie: Werkt met /api-scaffold outputs

#### /k8s-manifest

Productieklare Kubernetes-deployments.

  • Geavanceerde Patronen: Pod Security Standards, Netwerk Policies, OPA
  • GitOps Klaar: FluxCD- en ArgoCD-configuraties
  • Observability: Prometheus ServiceMonitors, OpenTelemetry
  • Auto-Schaalbaarheid: HPA, VPA en cluster autoscaler configuraties
  • Service Mesh: Integratie met Istio/Linkerd

Geavanceerde database migratiestrategieën.

Wanneer Workflows vs Tools Gebruiken

🔀 Workflows & Subagent Tools

  • Probleemoplossing: Analyseer en los problemen adaptief op
  • Meerdere perspectieven: Coördineer specialistische subagenten
  • Complexe taken: Meerstapsprocessen over verschillende domeinen
  • Onbekende oplossingen: Laat subagenten de aanpak bepalen

🛠️ Gespecialiseerde Tools

  • Infrastructuur opzetten: Configureer omgevingen
  • Codegeneratie: Maak specifieke implementaties
  • Analyse: Genereer rapporten zonder oplossingen
  • Domeintaken: Zeer gespecialiseerde handelingen

Voorbeelden:
  • "Implementeer gebruikersauthenticatiesysteem" → /feature-development
  • "Los prestatieproblemen op in de hele stack" → /smart-fix
  • "Moderniseer legacy-monoliet" → /legacy-modernize

🔧 Gebruik Tools Wanneer:

  • Specifieke expertise vereist - Duidelijke, gefocuste taak in één domein
  • Precieze controle gewenst - Wilt specifieke implementatiedetails aansturen
  • Onderdeel van handmatige workflow - Integreren in bestaande processen
  • Diepe specialisatie vereist - Uitvoering op expertniveau nodig
  • Voortbouwen op bestaand werk - Verbeteren of verfijnen van eerdere output

Voorbeelden:
  • "Maak Kubernetes-manifesten" → /k8s-manifest
  • "Scan op beveiligingslekken" → /security-scan
  • "Genereer API-documentatie" → /doc-generate

Command Format

Slash-commando’s zijn eenvoudige markdown-bestanden waarbij:

  • De bestandsnaam de commando-naam wordt (bijv. api-scaffold.md → /api-scaffold)
  • De inhoud van het bestand de prompt/instructies zijn die worden uitgevoerd bij aanroepen
  • Gebruik de $ARGUMENTS placeholder voor gebruikersinvoer

Best Practices

Commandoselectie

  • Laat Claude Code automatisch voorstellen - Analyseert de context en kiest optimale commando’s
  • Gebruik workflows voor complexe taken - Subagenten coördineren implementaties over domeinen
  • Gebruik tools voor gerichte taken - Pas specifieke commando’s toe voor doelgerichte verbeteringen

Effectief Gebruik

  • Geef uitgebreide context - Vermeld tech stack, beperkingen en vereisten
  • Koppel commando’s strategisch - Workflows → Tools → Verbeteringen
  • Bouw voort op eerdere output - Commando’s zijn ontworpen om samen te werken

Bijdragen

  • Maak een .md-bestand in workflows/ of tools/
  • Gebruik kleine letters-met-koppelteken-namen
  • Voeg $ARGUMENTS toe voor gebruikersinvoer

Problemen Oplossen

Commando niet gevonden: Controleer of bestanden in ~/.claude/commands/ staan

Workflows traag: Normaal - ze coördineren meerdere subagenten

Algemene output: Voeg meer specifieke context toe over je tech stack

Integratieproblemen: Controleer bestandslocaties en commando-volgorde

Prestatietips

Commando-selectie:

  • Workflows: Coördinatie van meerdere subagenten voor complexe features
  • Tools: Enkelvoudige bewerkingen voor specifieke taken
  • Eenvoudige aanpassingen: Blijf bij hoofdagent

Optimalisatie:
  • Begin met tools voor bekende problemen
  • Geef gedetailleerde vereisten vooraf
  • Bouw voort op eerdere commando-output
  • Laat workflows afronden voordat je aanpassingen doet

Een Nieuwe Workflow Toevoegen:

  • Focus op subagent-orkestratie en delegatielogica
  • Geef aan welke gespecialiseerde subagenten je gebruikt en in welke volgorde
  • Definieer coördinatiepatronen tussen subagenten

Een Nieuwe Tool Toevoegen:

  • Voeg volledige, productieklare implementaties toe
  • Structureer de inhoud met duidelijke secties en uitvoerbare resultaten
  • Voeg voorbeelden, best practices en integratiepunten toe
