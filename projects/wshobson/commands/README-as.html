Claude Code Slash Commands

Claude Code ৰ বাবে উৎপাদন-প্ৰস্তুত slash command সমূহ, যি বুদ্ধিমান স্বচালিতকৰণৰ জৰিয়তে উন্নয়নক দ্ৰুত কৰে।

৫২টা কমাণ্ড তলত সংগঠিত:

  • 🤖 ৱাৰ্কফ্ল': জটিল কামৰ বাবে বহু-উপ-এজেন্ট সমন্বয়
  • 🔧 টুল: নিৰ্দিষ্ট কাৰ্যৰ বাবে একক-উদ্দেশ্যৰ সহায়ক

🤝 ক্ল'ড ক'ড ছাবএজেন্টৰ প্ৰয়োজন

এই আদেশসমূহ ক্ল'ড ক'ড ছাবএজেন্টৰ সৈতে সমন্বয় ক্ষমতাৰ বাবে কাম কৰে।

  • 🤖 ৱাৰ্কফ্ল’চ - জটিল কাৰ্যৰ বাবে বহু উপ-এজেন্ট সমন্বয় কৰক
  • 🔧 টুলছ - নিৰ্দিষ্ট অপাৰেশ্যনৰ বাবে এক-মুখী আজ্ঞা

ব্যৱহাৰ

আজ্ঞাসমূহ tools/ আৰু workflows/ ডাইৰেক্টৰীত সংগঠিত। ডাইৰেক্টৰীৰ প্ৰিফিক্সৰ সৈতে ব্যৱহাৰ কৰক:

/tools:api-scaffold user management with authentication
/tools:security-scan check for vulnerabilities
/workflows:feature-development implement chat functionality

টোকা: যদি আপুনি প্ৰিফিক্স নোহোৱাকৈ কমাণ্ড ব্যৱহাৰ কৰিবলৈ পচন্দ কৰে, তেন্তে আপুনি ডিৰেক্টৰীসমূহ ফ্লেটেন কৰিব পাৰে: 

cp tools/*.md .
cp workflows/*.md .
ক্লড ক'ডে স্বয়ংক্ৰিয়ভাৱে প্ৰাসংগিক কমাণ্ডসমূহ পৰিপ্ৰেক্ষিত অনুসৰি প্ৰস্তাৱ কৰে।

🤖 ৱৰ্কফ্ল'সমূহ

জটিল কাৰ্যৰ বাবে বহু-উপ-এজেন্ট সমন্বয়:

ফিচাৰ বিকাশ

  • ফিচাৰ-বিকাশ - বেকএণ্ড, ফ্ৰন্টএণ্ড, পৰীক্ষা, আৰু ডিপ্লয়মেন্ট উপ-এজেন্টে সম্পূৰ্ণ ফিচাৰ নিৰ্মাণ কৰে
  • সম্পূৰ্ণ-সমীক্ষা - বহুবিধ সমীক্ষা উপ-এজেন্টে বিস্তৃত ক'ড বিশ্লেষণ প্ৰদান কৰে
  • চতুৰ-সমাধান - সমস্যাসমূহ বিশ্লেষণ কৰি উপযুক্ত বিশেষজ্ঞ উপ-এজেন্টলৈ প্ৰেৰণ কৰে

বিকাশ প্ৰক্ৰিয়া

  • গিট-ৱৰ্কফ্ল' - শাখা কৌশল আৰু PR টেম্পলেটৰ সৈতে কাৰ্যকৰী Git ৱৰ্কফ্ল' কাৰ্যকৰী কৰে
  • এজেন্ট-উন্নয়ন - প্ৰম্পট অপ্টিমাইজেশনৰ দ্বাৰা উপ-এজেন্টৰ কার্যক্ষমতা বৃদ্ধি কৰে
  • লেগাচি-আধুনিকীকৰণ - বিশেষ উপ-এজেন্ট ব্যৱহাৰ কৰি লেগাচি ক'ডবেছ আধুনিকীকৰণ কৰে
  • এমএল-পাইপলাইন - ডাটা আৰু এমএল ইঞ্জিনিয়াৰ উপ-এজেন্টৰে এমএল পাইপলাইন সৃষ্টি কৰে
  • বহু-প্লেটফৰ্ম - সমন্বিত উপ-এজেন্টৰ সৈতে ক্রছ-প্লেটফৰ্ম এপ নির্মাণ কৰে
  • ৱৰ্কফ্ল'-স্বয়ংক্ৰিয় - CI/CD, মনিটৰিং, আৰু ডিপ্লয়মেন্ট ৱৰ্কফ্ল' স্বয়ংক্ৰিয় কৰে

উপ-এজেন্ট-সমন্বিত ৱৰ্কফ্ল'

  • সম্পূৰ্ণ-ষ্টেক-ফিচাৰ - বেকএণ্ড, ফ্ৰন্টএণ্ড, আৰু ম'বাইল উপ-এজেন্টৰ সৈতে বহু-প্লেটফৰ্ম ফিচাৰ
  • নিরাপত্তা-শক্তিশালীকৰণ - বিশেষ উপ-এজেন্টৰ সৈতে নিৰাপত্তা-প্ৰথম বাস্তৱায়ন
  • ডাটা-ভিত্তিক-ফিচাৰ - ডাটা বিজ্ঞান উপ-এজেন্টৰ সৈতে এমএল-চালিত ফিচাৰ
  • দক্ষতা-অপ্টিমাইজেশ্বন - দক্ষতা উপ-এজেন্টৰ সৈতে সম্পূৰ্ণ অপ্টিমাইজেশ্বন
  • ঘটনা-প্ৰতিক্ৰিয়া - অপ্স উপ-এজেন্টৰে প্ৰডাকশ্যন ঘটনা সমাধান

🔧 টুল (একমাত্ৰ উদ্দেশ্যৰ কমাণ্ড)

  • এমএল-পাইপলাইন - MLOps সহ সম্পূৰ্ণ এমএল পাইপলাইন সৃষ্টি কৰক
  • প্ৰম্পট-অপ্টিমাইজ - কার্যক্ষমতা আৰু গুণগত মানৰ বাবে AI প্ৰম্পট অপ্টিমাইজ কৰক
