জটিল কামৰ বাবে বহু-সাবএজেন্ট অর্কেষ্ট্ৰেচন:

ফিচাৰ ডেভেলপমেণ্ট

  • feature-development - বেকএণ্ড, ফ্ৰণ্টএণ্ড, টেষ্টিং, আৰু ডিপ্লয়মেণ্ট সাবএজেন্টে সম্পূৰ্ণ ফিচাৰ নিৰ্মাণ কৰে
  • full-review - বহুতো ৰিভিউ সাবএজেন্টে বিস্তৃত ক'ড বিশ্লেষণ প্ৰদান কৰে
  • smart-fix - সমস্যাৰ বিশ্লেষণ কৰে আৰু উপযুক্ত বিশেষজ্ঞ সাবএজেন্টলৈ দায়িত্ব দিয়ে

ডেভেলপমেণ্ট প্ৰক্ৰিয়া

  • git-workflow - শাখা কৌশল আৰু PR টেম্প্লেটসহ ফলপ্ৰসূ Git ৱৰ্কফ্ল' প্ৰয়োগ কৰে
  • improve-agent - প্ৰম্প্ট অপ্টিমাইজেশ্যনৰ জৰিয়তে সাবএজেন্টৰ কাৰ্যক্ষমতা বৃদ্ধি কৰে
  • legacy-modernize - বিশেষজ্ঞ সাবএজেন্ট ব্যৱহাৰ কৰি লিগেচী ক'ডবেচ আধুনিক কৰে
  • ml-pipeline - ডাটা আৰু ML ইঞ্জিনিয়াৰিং সাবএজেন্টৰ সৈতে ML পাইপলাইন সৃষ্টি কৰে
  • multi-platform - সমন্বিত সাবএজেন্টৰ সহায়ত বহু-প্লেটফৰ্ম এপ নিৰ্মাণ কৰে
  • workflow-automate - CI/CD, মনিটৰিং, আৰু ডিপ্লয়মেণ্ট ৱৰ্কফ্ল' স্বয়ংক্ৰিয় কৰে

সাবএজেন্ট-অর্কেষ্ট্ৰেটেড ৱৰ্কফ্ল'চ

  • full-stack-feature - বেকএণ্ড, ফ্ৰণ্টএণ্ড, আৰু ম'বাইল সাবএজেন্টসহ বহু-প্লেটফৰ্ম ফিচাৰ
  • security-hardening - বিশেষজ্ঞ সাবএজেন্টৰ সৈতে সুৰক্ষা-প্ৰথম ব্যৱস্থাপন
  • data-driven-feature - ডাটা চায়েন্স সাবএজেন্টৰ সৈতে ML-চালিত ফিচাৰ
  • performance-optimization - কাৰ্যক্ষমতা সাবএজেন্টৰ সৈতে সম্পূৰ্ণ অপ্টিমাইজেশ্যন
  • incident-response - অপচ সাবএজেন্টৰ সহায়ত উৎপাদন ঘটনাৰ সমাধান

  • কণ্টেইনাৰ স্কেনিং: ইমেজ দুর্বলতা বিশ্লেষণ

#### /docker-optimize

উন্নত কণ্টেইনাৰ অপ্টিমাইজেশ্যন কৌশল।

  • চতুৰ অপ্টিমাইজেশ্যন: বিশ্লেষণ কৰে আৰু উন্নতিৰ পৰামৰ্শ দিয়ে
  • বহু-স্তৰীয় নিৰ্মাণ: ফ্ৰেমৱৰ্ক-নির্দিষ্ট অপ্টিমাইজ Dockerfile
  • আধুনিক সঁজুলি: BuildKit, Bun, UV-এ দ্রুত নিৰ্মাণৰ বাবে
  • সুৰক্ষা দৃঢ়ীকৰণ: Distroless ইমেজ, non-root ব্যৱহাৰকাৰী
  • আদেশ সংহতি: /api-scaffold আউটপুটৰ সৈতে কাম কৰে

#### /k8s-manifest

উৎপাদন-স্তৰৰ Kubernetes ডিপ্লয়মেন্ট।

  • উন্নত প্যাটাৰ্ণ: Pod Security Standards, Network Policies, OPA
  • GitOps-Redy: FluxCD আৰু ArgoCD কনফিগাৰেচন
  • পৰ্যবেক্ষণযোগ্যতা: Prometheus ServiceMonitors, OpenTelemetry
  • স্বয়ং-স্কেলিং: HPA, VPA, আৰু ক্লাষ্টাৰ autoscaler কনফিগ
  • Service Mesh: Istio/Linkerd সংহতি

#### /db-migrate

  • বহু-ডেটাবেচ: PostgreSQL, MySQL, MongoDB, DynamoDB
  • শূন্য-ডাউনটাইম: Blue-green ডিপ্লয়মেন্ট, rolling migration
  • ইভেন্ট সোর্সিং: CDC-ৰ বাবে Kafka/Kinesis সংহতি
  • Cross-Platform: Polyglot persistence সমলায়
  • মনিটৰিং: মাইগ্ৰেশ্যন স্বাস্থ্য পৰীক্ষা আৰু rollback
