OpenSuperClone

Scott Dwyer의 HDDSuperClone를 기반으로 한 많은 고급 기능을 갖춘 리눅스용 강력한 데이터 복구 유틸리티입니다.

✨ 기능 • 💾 설치 • 📖 문서 • ✏️ 변경 로그 • 💵 크레딧 • 📄 라이선스

✨ 기능

  • ddrescue와 유사한 고급 드라이브 복제 기능
  • 대상 데이터 추출을 위한 가상 디스크 모드
  • OS 드라이버를 우회하는 직접 AHCI/IDE 기능
  • 드라이브 자동 전원 사이클링을 위한 USB 릴레이 제어

#### 📋 계획된 변경사항

계획된 변경사항 목록은 로드맵을 참조하세요.

💾 설치

OpenSuperClone의 사전 빌드된 DEB 및 RPM 패키지는 릴리스 페이지에서 찾을 수 있습니다.

소스에서 OpenSuperClone을 설치하는 방법은 위키에서 확인할 수 있습니다.

OpenSuperClone 설치 대신에, Linux 라이브 ISO가 이제 xak77에 의해 제공됩니다.

📖 문서

OpenSuperClone 문서는 위키에서 찾을 수 있습니다. 현재 위키는 작업 중이며 완성되지 않았음을 참고하세요.

✏️ 변경 로그

#### OpenSuperClone 2.5.1 (작업 진행 중)

  • OSCDriver 2.6.7: 커널 6.14.0까지 지원 추가 (기여: piernov)

#### OpenSuperClone 2.5

  • 드라이버 설치용 DKMS 구현
  • OSCDriver 2.6.6: 커널 6.11.0까지 지원 추가 (기여: piernov)
  • USB 대용량 저장 장치 드라이버 비활성화 시 USB 연결 SCSI 드라이버도 비활성화됨
  • 대부분의 컨트롤에 툴팁 추가 및 SMART, 분석, 식별 보기 개선 등 다양한 UI 향상
  • 더 많은 알려진 SMART 속성과 식별 플래그 추가
  • 재시도 횟수를 최대 8회에서 1000회로 증가 (솔리드 스테이트 복구에 유용)

#### OpenSuperClone 2.4.1

  • UI 레이블에 잘못된 문자열이 표시되는 버그 수정

#### OpenSuperClone 2.4

  • 라이선스 관련 코드 제거
  • 빌드 시스템으로 CMake 도입
  • GTK2에서 GTK3로 전환
  • 지역화 시스템을 gettext로 교체
  • OSCDriver 2.6.1: 커널 5.19.11까지 지원 추가 (기여: piernov)
  • OSCViewer의 사소한 개선 (색상 선택기, 그래픽 변경)

💵 크레딧

이 프로젝트는 HDDSuperClone의 훌륭한 작업을 한 Scott Dwyer에게, 그리고 커널 모듈 업데이트 작업을 한 piernov에게 크레딧을 돌립니다.

이 프로젝트는 다음 라이브러리/프로그램을 사용합니다:

  • GLADE (https://glade.gnome.org/)
  • GTK (https://www.gtk.org/)
  • libconfig (https://hyperrealm.github.io/libconfig/)
  • libintl (https://www.gnu.org/software/gettext/)
  • libusb (https://libusb.info/)
  • Font Awesome (https://fontawesome.com/)
