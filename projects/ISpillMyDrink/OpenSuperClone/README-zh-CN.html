OpenSuperClone

基于Scott Dwyer 的 HDDSuperClone，功能强大的 Linux 数据恢复工具，具有多种高级功能。

✨ 功能 • 💾 安装 • 📖 文档 • ✏️ 更新日志 • 💵 鸣谢 • 📄 许可证

✨ 功能

  • 类似于 ddrescue 的先进驱动器克隆能力
  • 用于定向数据提取的虚拟磁盘模式
  • 规避操作系统驱动的直接 AHCI/IDE功能
  • 用于驱动器自动电源循环的USB 继电器控制

💾 安装

OpenSuperClone 的预编译 DEB 和 RPM 软件包可在发布页面找到。

✏️ 更新日志

#### OpenSuperClone 2.5.1（开发中）

  • OSCDriver 2.6.7：添加对内核 6.14.0 的支持（感谢piernov）

#### OpenSuperClone 2.5

  • 实现了驱动程序安装的 DKMS
  • OSCDriver 2.6.6：添加对内核 6.11.0 的支持（感谢piernov）
  • 禁用 USB 大容量存储驱动程序时也会禁用 USB 附加 SCSI 驱动程序
  • 各种界面改进，包括大多数控件的工具提示以及改进的 SMART、分析和识别视图
  • 添加了一些已知的 SMART 属性和识别标志
  • 将最大重试次数从 8 次增加到 1000 次（对固态恢复有用）

#### OpenSuperClone 2.4.1

  • 修复了界面标签显示错误字符串的错误

#### OpenSuperClone 2.4

  • 移除了许可相关代码
  • 实现了 CMake 构建系统
  • 从 GTK2 切换到 GTK3
  • 用 gettext 替换了本地化系统
  • OSCDriver 2.6.1：添加对内核 5.19.11 的支持（感谢piernov）
  • 对 OSCViewer 进行了小幅改进（颜色选择器，细微图形更改）
