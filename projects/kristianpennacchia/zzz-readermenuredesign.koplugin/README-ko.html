zzz-readermenuredesign.koplugin

KOReader의 다양한 리더 메뉴 재설계로, 사전 빠른 조회 팝업과 리더 하이라이트 메뉴를 포함합니다.

리더 하이라이트 메뉴에서 '알 수 없는' 버튼(이 플러그인에 의해 변경되지 않은 버튼)을 표시하는 옵션을 포함합니다:

이 플러그인은 또한 WordReference 플러그인 및 AI Assistant 플러그인과 같은 다른 플러그인에 대한 맞춤 UI를 제공합니다.

설치

  • 최신 릴리스를 다운로드하세요.
  • zzz-readermenuredesign.koplugin.zip 파일의 압축을 풉니다.
  • zzz-readermenuredesign.koplugin 폴더를 기기의 KOReader plugins 디렉터리에 복사합니다.
  • KOReader를 재시작합니다.

#### 문제 해결

  • 모든 아이콘이 경고(⚠️) 기호로 표시됩니다.
  • 버전 v1.0.5부터 아이콘은 자동으로 설치되어야 하지만, 여전히 문제가 발생하면 zzz-readermenuredesign.koplugin 폴더에서 아이콘을 수동으로 복사하여 KOReader 아이콘 폴더 /koreader/resources/icons에 넣어보세요.

  • 아이콘이 전혀 표시되지 않습니다.
  • 이 플러그인의 구버전을 사용하는 일부 기기에서 발생할 수 있습니다. /koreader/resources/icons/mdlight에서 이 플러그인의 SVG 아이콘을 삭제하고, 최신 릴리스의 PNG 아이콘으로 교체한 후 KOReader를 재시작하세요.

설정

  • 리더 하이라이트 메뉴:
  • 메뉴 열기 → 도구 🛠️ → 추가 도구 → 리더 메뉴 재설계 → 리더 하이라이트 메뉴에서 알 수 없는 버튼 표시.
  • 이 옵션은 리더 하이라이트 메뉴에서 '알 수 없는' 버튼(이 플러그인에서 특별한 처리가 없는 버튼)의 표시/숨기기를 전환합니다.
  • 사전 빠른 조회:
  • 메뉴 열기 → 도구 🛠️ → 추가 도구 → 리더 메뉴 재설계 → 사전 빠른 조회에서 내비 버튼 표시.
  • 이 옵션은 사전 빠른 조회에서 사전, 위키피디아 항목 등 사이를 이동하는 '내비' 버튼의 표시/숨기기를 전환합니다.
  • 또는: 왼쪽에서 오른쪽으로 스와이프하여 이동할 수도 있지만, 이 버튼들은 PC/Mac에서 더 유용합니다.
