Snel aan de slag

Er zijn verschillende opties om snel aan de slag te gaan:

  • Download de laatste release
  • Clone de repo: git clone https://github.com/twbs/bootstrap.git
  • Installeer met npm: npm install bootstrap@v5.3.6
  • Installeer met yarn: yarn add bootstrap@v5.3.6
  • Installeer met Bun: bun add bootstrap@v5.3.6
  • Installeer met Composer: composer require twbs/bootstrap:5.3.6
  • Installeer met NuGet: CSS: Install-Package bootstrap Sass: Install-Package bootstrap.sass

Lees de Aan de slag-pagina voor informatie over de frameworkinhoud, sjablonen, voorbeelden en meer.

Wat is inbegrepen

Binnen de download vind je de volgende mappen en bestanden, logisch gegroepeerd op gemeenschappelijke onderdelen en voorzien van zowel gecompileerde als geminimaliseerde varianten.

Downloadinhoud

``text bootstrap/ ├── css/ │ ├── bootstrap-grid.css │ ├── bootstrap-grid.css.map │ ├── bootstrap-grid.min.css │ ├── bootstrap-grid.min.css.map │ ├── bootstrap-grid.rtl.css │ ├── bootstrap-grid.rtl.css.map │ ├── bootstrap-grid.rtl.min.css │ ├── bootstrap-grid.rtl.min.css.map │ ├── bootstrap-reboot.css │ ├── bootstrap-reboot.css.map │ ├── bootstrap-reboot.min.css │ ├── bootstrap-reboot.min.css.map │ ├── bootstrap-reboot.rtl.css │ ├── bootstrap-reboot.rtl.css.map │ ├── bootstrap-reboot.rtl.min.css │ ├── bootstrap-reboot.rtl.min.css.map │ ├── bootstrap-utilities.css │ ├── bootstrap-utilities.css.map │ ├── bootstrap-utilities.min.css │ ├── bootstrap-utilities.min.css.map │ ├── bootstrap-utilities.rtl.css │ ├── bootstrap-utilities.rtl.css.map │ ├── bootstrap-utilities.rtl.min.css │ ├── bootstrap-utilities.rtl.min.css.map │ ├── bootstrap.css │ ├── bootstrap.css.map │ ├── bootstrap.min.css │ ├── bootstrap.min.css.map │ ├── bootstrap.rtl.css │ ├── bootstrap.rtl.css.map │ ├── bootstrap.rtl.min.css │ └── bootstrap.rtl.min.css.map └── js/ ├── bootstrap.bundle.js ├── bootstrap.bundle.js.map ├── bootstrap.bundle.min.js ├── bootstrap.bundle.min.js.map ├── bootstrap.esm.js ├── bootstrap.esm.js.map ├── bootstrap.esm.min.js ├── bootstrap.esm.min.js.map ├── bootstrap.js ├── bootstrap.js.map ├── bootstrap.min.js └── bootstrap.min.js.map `

We leveren gecompileerde CSS en JS (bootstrap.), evenals gecompileerde en geminimaliseerde CSS en JS (bootstrap.min.). Source maps (bootstrap.*.map) zijn beschikbaar voor gebruik met de ontwikkelaarstools van bepaalde browsers. Gebundelde JS-bestanden (bootstrap.bundle.js en de geminimaliseerde bootstrap.bundle.min.js) bevatten Popper.

Documentatie

De documentatie van Bootstrap, opgenomen in deze repo in de hoofdmap, is gebouwd met Astro en publiekelijk gehost op GitHub Pages op . De documentatie kan ook lokaal worden uitgevoerd.

Documentatiezoekopdrachten worden mogelijk gemaakt door Algolia's DocSearch.

Documentatie lokaal uitvoeren

  • Voer npm install uit om de Node.js-afhankelijkheden te installeren, inclusief Astro (de sitebouwer).
  • Voer npm run test uit (of een specifiek npm-script) om de gedistribueerde CSS- en JavaScript-bestanden, evenals onze doc-assets, opnieuw op te bouwen.
  • Voer in de hoofdmap /bootstrap het commando npm run docs-serve uit op de opdrachtregel.
  • Open http://localhost:9001/ in je browser, en voilà.

Lees meer over het gebruik van Astro in de documentatie.

Bijdragen

Lees onze bijdragingsrichtlijnen zorgvuldig door. Hierin staan aanwijzingen voor het openen van issues, codestandaarden en opmerkingen over ontwikkeling.

Bovendien, als je pull request JavaScript-patches of -functies bevat, moet je relevante unittests toevoegen. Alle HTML en CSS moet voldoen aan de Code Guide, onderhouden door Mark Otto.

Editorvoorkeuren zijn beschikbaar in de editorconfig voor gebruiksgemak in gangbare teksteditors. Lees meer en download plugins op .

Community

Blijf op de hoogte van de ontwikkeling van Bootstrap en chat met de projectonderhouders en leden van de community.

  • Volg @getbootstrap op X.
  • Lees en abonneer je op De Officiële Bootstrap Blog.
  • Stel vragen en verken onze GitHub Discussions.
  • Bespreek, stel vragen en meer op de community Discord of Bootstrap subreddit.
  • Chat met andere Bootstrappers op IRC. Op de server irc.libera.chat, in het #bootstrap kanaal.
  • Hulp bij implementatie kan gevonden worden op Stack Overflow (getagd bootstrap-5).
  • Ontwikkelaars zouden het trefwoord bootstrap` moeten gebruiken voor pakketten die de functionaliteit van Bootstrap aanpassen of uitbreiden bij distributie via npm of vergelijkbare distributiemechanismen voor maximale vindbaarheid.

Versiebeheer

Voor transparantie in onze releasecyclus en het streven naar achterwaartse compatibiliteit, wordt Bootstrap onderhouden volgens de Semantic Versioning-richtlijnen. Soms maken we fouten, maar we houden ons waar mogelijk aan deze regels.

Bekijk de Releases-sectie van ons GitHub-project voor de changelogs van elke releaseversie van Bootstrap. Aankondigingsposts van releases op de officiële Bootstrap-blog bevatten samenvattingen van de meest opmerkelijke wijzigingen in elke release.
