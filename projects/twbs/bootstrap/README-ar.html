البدء السريع

تتوفر عدة خيارات للبدء السريع:

  • تحميل آخر إصدار
  • استنساخ المستودع: git clone https://github.com/twbs/bootstrap.git
  • التثبيت عبر npm: npm install bootstrap@v5.3.6
  • التثبيت عبر yarn: yarn add bootstrap@v5.3.6
  • التثبيت عبر Bun: bun add bootstrap@v5.3.6
  • التثبيت عبر Composer: composer require twbs/bootstrap:5.3.6
  • التثبيت عبر NuGet: CSS: Install-Package bootstrap Sass: Install-Package bootstrap.sass

اقرأ صفحة البدء لمزيد من المعلومات حول محتويات الإطار، والقوالب، والأمثلة، والمزيد.

ما يتضمنه

عند التحميل ستجد الأدلة والملفات التالية، مصنفة منطقياً لتجميع الأصول الشائعة وتوفير نسخ مجمعة ومضغوطة.

نوفر CSS وJS مجمعة (bootstrap.) بالإضافة إلى CSS وJS مجمعة ومضغوطة (bootstrap.min.). تتوفر خرائط المصدر (bootstrap.*.map) للاستخدام مع أدوات المطور لبعض المتصفحات. ملفات JS المجمعة (bootstrap.bundle.js وbootstrap.bundle.min.js المضغوطة) تتضمن Popper.

الأعطال وطلبات الميزات

هل لديك عطل أو طلب ميزة؟ يرجى أولاً قراءة إرشادات فتح القضايا والبحث عن القضايا الحالية والمغلقة. إذا لم يتم معالجة مشكلتك أو فكرتك بعد، يرجى فتح قضية جديدة.

التوثيق

وثائق بوتستراب، المضمنة في هذا المستودع في الدليل الجذري، مبنية باستخدام Astro ومستضافة علنًا على GitHub Pages في . يمكن أيضًا تشغيل الوثائق محليًا.

البحث في الوثائق مدعوم بواسطة DocSearch من Algolia.

تشغيل الوثائق محليًا

  • شغّل npm install لتثبيت تبعيات Node.js، بما في ذلك Astro (منشئ الموقع).
  • شغّل npm run test (أو سكريبت npm محدد) لإعادة بناء ملفات CSS وJavaScript الموزعة، بالإضافة إلى أصول الوثائق.
  • من الدليل الجذري /bootstrap، شغّل npm run docs-serve في سطر الأوامر.
  • افتح http://localhost:9001/ في متصفحك، وهكذا.

تعرف أكثر على استخدام Astro من خلال قراءة وثائقه.

المساهمة

يرجى قراءة إرشادات المساهمة. تتضمن الإرشادات تعليمات لفتح القضايا، ومعايير كتابة الكود، وملاحظات حول التطوير.

بالإضافة إلى ذلك، إذا احتوى طلب الدمج الخاص بك على تعديلات أو ميزات في JavaScript، يجب أن ترفق اختبارات وحدات ذات صلة. يجب أن يتوافق كل من HTML وCSS مع دليل الكود الذي يديره Mark Otto.

تفضيلات المحرر متوفرة في ملف إعدادات المحرر لسهولة الاستخدام في محررات النصوص الشائعة. اقرأ المزيد وقم بتحميل الإضافات على .

المجتمع

احصل على التحديثات حول تطوير بوتستراب وتواصل مع مشرفي المشروع وأعضاء المجتمع.

  • تابع @getbootstrap على X.
  • اقرأ واشترك في المدونة الرسمية لبوتستراب.
  • اطرح الأسئلة واستكشف المناقشات على GitHub.
  • ناقش واطرح الأسئلة والمزيد على خادم Discord المجتمعي أو منتدى Bootstrap على Reddit.
  • تواصل مع مستخدمي بوتستراب الآخرين عبر IRC. على خادم irc.libera.chat في قناة #bootstrap.
  • قد تجد المساعدة في التنفيذ على Stack Overflow (بعلامة bootstrap-5).
  • يجب على المطورين استخدام الكلمة المفتاحية bootstrap` في الحزم التي تعدل أو تضيف إلى وظائف بوتستراب عند التوزيع عبر npm أو آليات التوزيع المشابهة لتحقيق أقصى قابلية للاكتشاف.

إصدار النسخ

لتحقيق الشفافية في دورة الإصدارات والسعي للحفاظ على التوافق مع الإصدارات السابقة، يُدار بوتستراب وفقًا لإرشادات إصدار النسخ الدلالية. أحيانًا نخطئ، لكننا نلتزم بهذه القواعد كلما أمكن ذلك.

راجع قسم الإصدارات في مشروعنا على GitHub للاطلاع على سجلات التغيير لكل إصدار من بوتستراب. منشورات الإعلان عن الإصدارات في المدونة الرسمية لبوتستراب تتضمن ملخصات لأبرز التغييرات في كل إصدار.
