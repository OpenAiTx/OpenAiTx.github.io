यह परियोजना Bilibili के index-tts पर आधारित है, जो एकल वक्ता और बहु-वक्ता सेटअप के लिए LoRA फाइन-ट्यूनिंग समाधान प्रदान करती है। इसका उद्देश्य उच्च गुणवत्ता वाले वक्ता ऑडियो संश्लेषण में लय और स्वाभाविकता को बढ़ाना है।

प्रशिक्षण एवं अनुमान

#### 1. ऑडियो टोकन और वक्ता स्थिति निष्कर्षण

# Extract tokens and speaker conditions
python tools/extract_codec.py --audio_list ${audio_list} --extract_condition
audio_list format: audio_path + transcript, separated by \t

/path/to/audio.wav 小朋友们，大家好，我是凯叔，今天我们讲一个龟兔赛跑的故事。
