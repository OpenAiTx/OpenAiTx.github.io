NotebookLM Pro 树状图

一个 Chrome 扩展，将谷歌的 NotebookLM 从简单列表转变为强大的项目管理仪表盘。

停止在无尽的平面列表中滚动。将你的来源组织成嵌套树，使用高级任务、多窗口工作流管理复杂项目，并通过禅模式专注。

重要警告 ⚠️

卸载会删除数据：此扩展使用本地存储以保护隐私和提升速度。如果你移除/卸载扩展，Chrome 会立即删除你的文件夹和任务。

解决方案：请务必使用导出按钮，在重新安装或迁移到新电脑前保存备份的 JSON 文件。

谷歌更新与可靠性：此扩展通过“读取”NotebookLM 网站工作。如果谷歌更新了他们的网站，此扩展可能会失效。

免责声明：由于这是一个免费开源项目，在我业余时间维护，我无法保证谷歌更改代码时能立即修复。请将此工具作为辅助增强，并保持重要数据备份。

项目状态与免责声明 🚧

请阅读：我是一名建设项目经理，不是专业软件开发者。我开发此工具是因为我日常工作流需要。

“我的机器上能用”：此更新（v17.9）对我个人使用稳定，但尚未在所有浏览器配置下进行全面测试。

实验功能：多窗口系统和 DOM 观察器较为复杂。你可能会遇到我未发现的错误。

反馈：如果发现错误，请在 GitHub 上提交 issue！

v17.9 新功能：生产力套件 🚀

将 NotebookLM 转变为多窗口工作区。本次更新（整合 v17.6–v17.9）带来重大工作流改进：

弹出笔记：将任意笔记在浮动窗口中打开，方便写作时并排参考。弹出窗口保留格式（只读）并与明暗主题同步。

智能任务与快速捕获：在任意笔记中选中文本，即刻显示“创建任务”按钮。此操作创建一个直接链接该源笔记的任务。

高级任务管理：将任务组织进自定义分区（可折叠分组），添加带快速选择选项的截止日期，并包含丰富描述。

优雅稳定性：新的自愈引擎，能单独禁用特定损坏的功能，而非因谷歌更新代码导致整个扩展崩溃。

功能

生产力与工作流 📋
  • 多窗口工作流程 — 点击 Studio 面板中任意笔记的“弹出”图标，将其在独立窗口中打开。非常适合双显示器设置。

  • 集成任务管理器 — 完整的项目管理工具内置于界面中：
  • 分区： 按阶段分组任务（例如，“研究”、“草稿”、“审查”）。
  • 智能链接： 由笔记创建的任务包含一个“链接”图标，可过滤视图至原始来源。
  • 优先级： 标记任务（红/黄/蓝），并按优先级或截止日期排序。

  • “禅模式” — 切换界面以隐藏 AI 聊天和侧边栏，打造无干扰的阅读/写作空间。

组织 📂

  • 嵌套文件夹 — 为来源、笔记和生成的产物（音频概览、常见问题等）创建深层结构。

  • 固定 — 将重要合同或规范固定在列表顶部。

  • 颜色编码 — 为文件夹分配颜色，实现视觉组织。

  • 批量选择 — 主复选框可瞬间选择/取消选择文件夹或整个笔记本中的所有来源。

深度搜索 🔍

  • 内容索引 — 搜索笔记中的实际文本，而不仅仅是标题。

> 注意： 索引在打开笔记时被被动创建。

  • 智能压缩 — 使用 LZ-String 压缩，在本地存储最多 5MB 的搜索索引数据而不影响性能。

可靠性 🛡️

  • 本地优先架构 — 零网络延迟。所有数据存储于您的设备。

  • 优雅降级 — 如果因网站更新导致某功能失效，扩展的其他部分仍能正常工作。

  • 样式化对话框 — 流畅且主题化的确认对话框（不再有令人不适的浏览器弹窗）。

> 注意： 每个笔记本拥有独立数据。在“项目 A”中创建的任务和文件夹不会干扰“项目 B”。

安装

加载已解压的扩展（开发者模式）

  • 下载或克隆此代码库。
  • 打开 Chrome，访问 chrome://extensions。
  • 启用 开发者模式（右上角开关）。
  • 点击 加载已解压的扩展。
  • 选择包含这些扩展文件的文件夹。
  • 进入 NotebookLM 开始整理吧！

使用

安装后，你将在 NotebookLM 中看到一个新的“项目仪表盘”界面：

  • 快速捕获： 在任意笔记中高亮文本 -> 点击悬浮的 + 按钮将其转为任务。

  • 弹出窗口： 在 Studio 面板悬停于某条笔记 -> 点击新窗口图标。

  • 任务详情： 点击任务上的描述图标添加笔记或上下文。

  • 批量操作： 使用文件夹复选框切换多个来源以供 AI 上下文窗口使用。

  • 导出/导入： 使用这些按钮备份你的结构，或迁移到新电脑。

存储与数据指南 💾

你的数据存储在哪里

由于 v17.9 优先考虑速度和隐私，你的数据存储在你的设备上：

  • 本地存储： 文件夹、任务、设置和搜索索引。

  • 无自动同步： 文件夹不会在不同电脑间自动同步。

  • 迁移数据： 要将设置迁移到笔记本电脑，使用导出按钮获取 JSON 文件，然后在新设备上导入。

> 警告： 如果你清除了 Chrome 的“站点数据”或“Cookie”，将会丢失你的文件夹。请定期导出备份。

隐私与安全

此扩展是本地优先的：

  • 所有数据均存储在您浏览器的本地存储中。

  • 不会向开发者发送任何分析或使用数据。

  • 开源： 您可以查看代码，确认没有数据离开您的设备。

详情请参见 PRIVACY_POLICY.md。

已知限制

  • 弹出编辑： 弹出窗口目前为只读。您必须使用主窗口编辑文本。

  • 下划线： 笔记中的下划线内容不会持久保存（这是 Google NotebookLM 的问题，不是扩展的问题）。

  • 隐身模式： 扩展的内存系统在隐身模式下被禁用。
