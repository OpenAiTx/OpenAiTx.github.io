크로스 플랫폼 Node.js 코드를 작성하는 방법.

왜 중요할까요: 2018 Node.js 사용자 설문조사에 따르면, Node.js 개발자의 24%가 로컬에서 Windows를 사용하고 41%가 Mac을 사용합니다. 프로덕션에서는 85%가 Linux를, 1%가 BSD를 사용합니다.
