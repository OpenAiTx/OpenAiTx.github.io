এই প্লাগইনটো ভিডিঅ' প্লেয়াৰত এপিছডৰ তালিকা যোগ কৰে, যাৰ দ্বাৰা আপুনি প্লেয়াৰ এৰি নাযাবাকৈ টিভি চ'ৰ প্ৰতিটো এপিছডৰ পূৰ্বদৰ্শন কৰিব পাৰে।

Jellyfin Media Player (JMP) Desktop Client (অপ্ৰচলিত)

নিৰ্দেশনা চাওক...

JMP সংস্কৰণ 1.11.0ৰ সৈতে অপ্ৰচলিত

কাৰণ নতুন JMP ক্লায়েন্টে চাৰ্ভাৰৰ পৰা বৰ্তমান ৱেব প্লেয়াৰ ব্যৱহাৰ কৰে, ক্লায়েন্ট ক'ডত সিধাসলাই কোনো পৰিবৰ্তন কৰাৰ প্ৰয়োজন নাই।

এইটো হৈছে ডেস্কটপ ক্লায়েন্টত স্ক্ৰিপ্ট স্থাপন কৰাৰ সুপারিশ কৰা উপায়। যদি আপুনি নিজে nativeshell.js ফাইলটো সম্পাদনা কৰিবলৈ অনুকূল নাভাৱে (পদক্ষেপ ৩ৰ পৰা ৬), তেন্তে আপুনি সম্পূৰ্ণ মুক্তি সংস্কৰণ ডাউনলোড কৰিব পাৰে, য’ত script ইতিমধ্যে nativeshell.js ফাইলত সংযোজন কৰা আছে। এতিয়াও স্পষ্ট নহয় যে, মুক্তি সংস্কৰণৰ nativeshell.js ফাইলটো স্থাপন কৰাত সম্ভাৱ্য সমস্যাসমূহ থাকিব পাৰে নে, গতিকে তলত দিয়া সকলো পদক্ষেপ অনুসৰণ কৰিবলৈ পৰামৰ্শ দিয়া হৈছে।

  • শেহতীয়া মুক্তি JMP বা JMP-full (script ইতিমধ্যে nativeshell.js ফাইলত সংযোজন কৰা আছে) ডাউনলোড কৰক।
  • zip ফাইলটো আপোনাৰ Jellyfin ডাইৰেক্টৰিত extract কৰক (উদাহৰণস্বৰূপে: C:\Program Files\Jellyfin\Jellyfin Media Player)
  • আপোনাৰ Jellyfin ডাইৰেক্টৰিত "web-client\extension" ফোল্ডাৰ পথ অনুসৰণ কৰক।
  • "nativeshell.js" ফাইলটো এখন পাঠ্য সম্পাদনকাৰীত খুলক।
  • ফাইলটোৰ ভিতৰত const plugins = []; অংশটো বিচাৰি উলিয়াওক। তালিকাৰ আৰম্ভণিত এটা নতুন শাৰী যোগ কৰক আৰু 'inPlayerEpisodePreviewPlugin', paste কৰক। অংশটো এতিয়া এইদৰে দেখিবলৈ থকা উচিত:
const plugins = [
    'inPlayerEpisodePreviewPlugin',
    'mpvVideoPlayer',
    'mpvAudioPlayer',
    'jmpInputPlugin',
    'jmpUpdatePlugin',
    'jellyscrubPlugin',
    'skipIntroPlugin'
];
  • ফাইলটো সংৰক্ষণ কৰক আৰু JMP ক্লায়েন্টটো পুনৰ আৰম্ভ কৰক।

💡 সমস্যাৰ সমাধান

1. পূৰ্বদৰ্শন বুটামটো দেখা নাযায়

এইটো সম্ভৱত index.html ফাইলটোৰ ভুল অনুমতিৰ সৈতে সম্পৰ্কিত।

সম্ভাৱ্য সমাধানসমূহৰ তালিকা চাওক...

#### 1.1 এই সমস্যাটো এৰাই file-transformation প্লাগিন ব্যৱহাৰ কৰক।

#### 1.2 ডকাৰ কণ্টেইনাৰৰ ভিতৰত মালিকানা সলনি কৰক যদি আপুনি jellyfin এটা ডকাৰ কণ্টেইনাৰত চলাইছে, তেন্তে তলত দিয়া নিৰ্দেশনাৰ দ্বাৰা মালিকানা সলনি কৰিব পাৰিব (জেলিফিনক আপোনাৰ কণ্টেইনাৰৰ নাম, user আৰু groupক কণ্টেইনাৰৰ user আৰু groupৰে প্ৰতিস্থাপন কৰক): 

docker exec -it --user root jellyfin chown user:group /jellyfin/jellyfin-web/index.html && docker restart jellyfin
আপুনি ইয়াক ব্যৱস্থাৰ আৰম্ভণিতে এটা ক্রন কাৰ্য হিচাপে চলাব পাৰে। (muisjeক ধন্যবাদ এই সমাধানত সহায় কৰাৰ বাবে)

#### 1.3 উইণ্ডোজ ইনষ্টলেশ্যনত মালিকানা পৰিৱৰ্তন কৰা

  • ইয়ালৈ যাওক: C:\Program Files\Jellyfin\Server\jellyfin-web\
  • index.htmlত সোঁ-হাতৰ ক্লিক কৰক → Properties → Security tab → Editত ক্লিক কৰক
  • তালিকাৰ পৰা আপোনাৰ ব্যৱহাৰকাৰীক বাছনি কৰক আৰু Write permission বাকচ চিহ্নিত কৰক।
  • ছাৰ্ভাৰ আৰু ক্লায়েন্ট দুয়ো পুনৰ আৰম্ভ কৰক।
(xeucক ধন্যবাদ এই সমাধানৰ বাবে)

উপৰোক্ত সমাধানবোৰৰ কোনো এটা কাম না কৰিলে, অনুগ্ৰহ কৰি পুৰণি সমস্যাবোৰ চাওক। যেনে ইয়াত অথবা ইয়াত।



আপুনি যদি কোনো ত্ৰুটিৰ সন্মুখীন হয় যি নিজে সমাধান কৰিব নোৱাৰেনে, অনায়াসে এটা issue খুলিব পাৰে।
অনুগ্ৰহ কৰি মনত ৰাখক যে প্ৰত্যেক ব্যৱস্থা পৃথক, যাৰ ফলত অপ্ৰত্যাশিত আচৰণ ঘটিব পাৰে, সেইবাবে ইয়াৰ বিষয়ে যতসম্ভব তথ্য যোগ কৰক।
