یک مرورگر فایل ساده که به دلیل علاقه من به یادگیری زبان راست ساخته شد.
این نرم‌افزار مستقل از سیستم‌عامل و بهینه‌سازی شده برای عملکرد بهتر است.



عملکرد این نرم‌افزار با استفاده از "jwalk" و "Tauri" به دست آمده است.

CoDriver برای دسترسی به فایل‌ها و پوشه‌ها از کش مسیر استفاده نمی‌کند و بنابراین عملکرد آن به زبان راست، سرعت دیسک و قدرت پردازنده بستگی دارد.

⁉️ توجه داشته باشید که این نرم‌افزار همچنان در حال توسعه است و ممکن است دارای باگ باشد!

ویژگی‌های پایه

  • مرور دایرکتوری‌ها به روشی که با آن آشنا هستید
  • کپی و جای‌گذاری، حذف، ایجاد و تغییر نام فایل‌ها و پوشه‌ها
  • جابجایی بین حالت‌های شبکه‌ای، لیستی و ستون‌های میلر
  • بستن پنجره‌های بازشو با کلید esc
  • پرش مستقیم به یک دایرکتوری با Ctrl / Cmd + G با وارد کردن مسیر
  • مرتب‌سازی آیتم‌ها در حالت لیستی بر اساس اندازه، نام یا آخرین تغییر

ویژگی‌های پیشرفته

  • فشرده‌سازی فایل‌ها و پوشه‌ها
  • zip
  • zstd
  • brotli
  • density (https://github.com/g1mv/density)
  • استخراج آرشیوها
  • rar
  • zip
  • 7zip
  • tar (.gz, .bz2)
  • density
  • رفتن به یک دایرکتوری با میانبر LAlt + 1 / 2 / 3 | (در مک‌او‌اس option + 1 / 2 / 3)
  • مسیرها را به دلخواه خود در تنظیمات پیکربندی کنید
  • ایجاد فایل با F6
  • ایجاد پوشه با F7
  • نمای دو پنجره‌ای
  • جستجوی فایل‌ها با F8
  • کپی عنصر انتخاب شده فعلی به پنجره دیگر با F5
  • انتقال عنصر انتخاب شده فعلی به پنجره دیگر با LShift + F5
  • کشیدن و رها کردن فایل‌ها به اکسپلورر برای کپی آن‌ها به دایرکتوری فعلی
  • تغییر نام گروهی انتخاب‌ها با Ctrl / Cmd + LShift + M
  • اجرای تغییر نام گروهی با Ctrl / Cmd + Return
  • پیش‌نمایش سریع فایل -> انتخاب ورودی دایرکتوری و زدن کلید فاصله.
  • فایل‌های پشتیبانی‌شده: همه فایل‌های تصویری (.jpg, .png, ...)، .pdf، .mp4، .json، .txt، .html
  • سایر آیتم‌ها یک کاشی کوچک با اطلاعاتی درباره آن نمایش می‌دهند. (مسیر، اندازه، آخرین تغییر)
  • ناوبری فوری -> شروع به تایپ کنید و ورودی‌های دایرکتوری را به صورت خودکار فیلتر کنید تا گاهی
بسیار سریع‌تر به مکان مورد نظر برسید

🖥️ ادغام FTP / SFTP

CoDriver پشتیبانی کاملاً بومی و آماده برای اتصال‌های راه دور FTP و SFTP را ارائه می‌دهد. هیچ وابستگی خارجی، لایه‌های FUSE یا مانت‌های sshfs لازم نیست! فقط کافیست سرورهای راه دور خود را مستقیماً در پنل کناری متصل کنید.

⚠️ مشکلات شناخته‌شده:

  • کشیدن و رها کردن خارج از پنجره در لینوکس همواره کار نمی‌کند
  • در ویندوز ممکن است نیاز باشد Microsoft Visual C++ Redistributable را نصب کنید
  • مجوزها در ms-windows کمی عجیب هستند
  • اگر در کپی کردن فایل‌ها یا موارد مشابه مشکل داشتید، ممکن است نیاز باشد برنامه را به عنوان مدیر اجرا کنید
  • ممکن است نیاز باشد openssl1.1 را در سیستم‌های لینوکس نصب کنید اگر برنامه اجرا نشد

چگونه مشارکت کنیم

سیستم خود را برای توسعه برنامه‌های tauri نسخه ۱ آماده کنید: پیش‌نیازهای Tauri

پس از انجام این کار فقط کافیست دستور ``git clone https://github.com/RickyDane/CoDriver` یا `gh repo clone RickyDane/CoDriver` را در محلی از سیستم خود اجرا کنید.

باید بتوانید با اجرای دستور `cargo tauri dev` در پوشه اصلی پروژه، CoDriver را اجرا و توسعه دهید.
اطمینان حاصل کنید که tauri-cli نصب شده است: `cargo install tauri-cli``

امضای نسخه نهایی

آرتیفکت‌های نسخه macOS باید امضا و تایید شوند تا Gatekeeper گزارش نکند که برنامه نصب شده آسیب دیده است. برای دریافت اسرار مورد نیاز GitHub Actions و فرمان‌های تایید، به امضا و تایید macOS مراجعه کنید.
