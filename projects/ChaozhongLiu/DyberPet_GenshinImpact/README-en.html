Quick Start

Windows Users

First time use: Download the repository here locally, unzip and double-click `DyberPet-Genshin.exe to run (The .exe program cannot leave the folder but you can create a desktop shortcut) Version update: If you are already using DyberPet desktop pet, overwrite the old version with the latest files downloaded here (Overwrite all, the data folder will be preserved, data will not be lost) Baidu Netdisk: Link | Extraction code: bs1d Quark Netdisk: Link | Extraction code: 7RD4

Linux / MacOS Users

Mac Beta Quark Netdisk: Link | Extraction code: D3VR Feel free to join the group to report bugs and questions, and get the latest updates! 233305772 Q: The program is corrupted and won't open: A: This is an issue with third-party app permissions, not actual corruption.
