PicoLoader

这是一个基于ODE的任天堂GameCube光驱改装芯片，用于启动自制软件。\ 其价格和功能与PicoBoot非常相似，但它具有多种优势。

加入Discord服务器以获得支持并讨论该改装！

它是如何工作的？

该改装通过在启动时模拟光驱，执行自制应用程序，然后重新启用光驱来工作。\ 这是通过类似FlippyDrive的柔性PCB实现的。 然而，它不是完整的光驱模拟器，仅允许像PicoBoot一样从闪存执行小型自制应用程序（例如swiss）。 要访问SD卡上的文件，需要SDGecko、SD2SP2或类似适配器，这些可以单独购买。

特性

  • 开源
  • 可保留光驱
  • 使用在线转换器简单启动大多数自制应用
  • 不会对GameCube做永久改动
  • 3D打印插头，适用于无光驱安装
  • 易于拆卸
  • Pico运行在200MHz，提升与克隆主板的兼容性

#### 焊接版本
  • 便宜且易得的组件（Raspberry Pi Pico/Pico 2 + 柔性PCB + 二极管）
  • 现货供应充足
  • 无需直接焊接到GameCube
  • 焊接简单
  • 替代安装选项：Panasonic Q、低矮型

#### 无焊版本
  • 无需焊接
  • 柔性PCB易于更换
  • 预装PicoLoader固件和Swiss
> [!TIP]
你可以从store.makstech.io购买无焊和普通版本。
