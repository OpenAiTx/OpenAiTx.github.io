Was ist json-everything?

Die Hauptphilosophie hinter json-everything ist sicherzustellen, dass die gängigen JSON-Funktionalitäten im Bereich _System.Text.Json_ gut unterstützt werden.

Der erste Schritt dazu besteht darin, zu prüfen, ob es bereits andere Projekte gibt, die eine bestimmte Funktion unterstützen.

Falls ja, wird dieser Aufwand hier wahrscheinlich nicht dupliziert. Etwas weiter unten findest du eine Liste anderer Projekte, die interessante Dinge mit JSON machen.

Andernfalls steht es zur Überlegung für dieses Projekt offen.

Was ist im Paket enthalten?

Es gibt tatsächlich mehrere Pakete. Jede Funktionalität ist in eine eigene Bibliothek aufgeteilt, sodass Sie auswählen können, welche Sie benötigen.

Alle Bibliotheken sind für .Net Standard 2.0 und .Net 8 gebaut.

JsonSchema.Net
 JSON Schema, Entwürfe 6 und höher
JsonSchema.Net.Generation
 Erzeugung von Schemata aus .Net-Typen
JsonSchema.Net.Generation.DataAnnotations
 Schema-Generierung mit Unterstützung für System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 Zufällige Instanzdatengenerierung (bereitgestellt durch Bogus)
JsonSchema.Net.Data
 Ein Vokabular für den Zugriff auf Instanz- und externe Daten
JsonSchema.Net.ArrayExt
 Ein Vokabular zur Erweiterung der Array-Validierung
JsonSchema.Net.OpenApi
 OpenApi 3.1-Vokabular-Erweiterung (verwendet von Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) und Relative JSON Pointer (Spezifikation)
JsonLogic
 JsonLogic (Webseite)
JsonE.Net
 JSON-e (Webseite)
Json.More.Net
 Allgemeine Erweiterungen, die vermutlich in System.Text.Json[.Nodes] hätten enthalten sein sollen, aber nicht enthalten sind
Yaml2JsonNode
 Konvertierungen zwischen dem YAML-Dokumentmodell in YamlDotNet und JsonNode (beide Richtungen).

Alle oben genannten Bibliotheken bieten vollständige Unterstützung für die zugehörigen Spezifikationen.

Sie können die Ergebnisse der JSON Schema Test Suite auch auf Bowtie ansehen, das die Test Suite gegen mehrere Implementierungen auf verschiedenen Plattformen ausführt.

HAFTUNGSAUSSCHLUSS Meine Bibliothek, _JsonSchema.Net_, steht in keinem Zusammenhang mit der Website https://jsonschema.net, außer dass beide hervorragende JSON Schema Tools sind.

Veraltete Pakete

Diese Pakete waren ebenfalls Teil dieser Suite, werden aber nicht mehr unterstützt.

JsonSchema.Net.CodeGeneration
 Generierung von C#-Code aus Schemata Veraltet zugunsten von Corvus.JsonSchema

_System.Text.Json_-Unterstützung durch andere Projekte

Falls Sie hier nicht finden, was Sie suchen, probieren Sie bitte eines dieser hervorragenden Projekte:

  • JsonCons.Net von @danielaparker
  • JSON Pointer
  • JSON Patch
  • JSON Merge Patch
  • JSON Path
  • JMES Path
  • Corvus.JsonSchema von @mwadams
  • JSON Schema-Validierung mit Fokus auf C#-Codegenerierung
  • (weitere folgen)

Wenn Sie JSON für etwas verwenden, das durch keine Bibliothek in dieser Suite oder durch eines der oben genannten Projekte abgedeckt wird, können Sie gerne ein Feature-Issue erstellen.

Wenn Sie ein anderes Projekt pflegen oder kennen, das _System.Text.Json_ erweitert, um coole Dinge zu tun, würde ich es gerne hier aufführen. Bitte erstellen Sie ein allgemeines Issue, um mich darüber zu informieren.
