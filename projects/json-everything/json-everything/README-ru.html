Что такое json-everything?

Основная философия json-everything заключается в обеспечении хорошей поддержки распространённой функциональности JSON в пространстве _System.Text.Json_.

Первым шагом на этом пути является проверка наличия других проектов, поддерживающих ту или иную функцию.

Если такие проекты существуют, то, скорее всего, дублировать их работу здесь не будут. Немного ниже вы найдёте список других проектов, которые делают интересные вещи с JSON.

В противном случае, возможность реализации этой функции рассматривается для данного проекта.

Что внутри?

На самом деле, внутри несколько компонентов. Каждая функция вынесена в отдельную библиотеку, так что вы можете выбрать только то, что вам нужно.

Все библиотеки построены на .Net Standard 2.0 и .Net 8.

JsonSchema.Net
 JSON Schema, черновики 6 и выше
JsonSchema.Net.Generation
 Генерация схем из .Net типов
JsonSchema.Net.Generation.DataAnnotations
 Поддержка генерации схем для System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 Генерация случайных данных экземпляра (на базе Bogus)
JsonSchema.Net.Data
 Словарь для доступа к данным экземпляра и внешним данным
JsonSchema.Net.ArrayExt
 Словарь для расширения валидации массивов
JsonSchema.Net.OpenApi
 Расширение словаря OpenApi 3.1 (используется в Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) и Relative JSON Pointer (Спецификация)
JsonLogic
 JsonLogic (Сайт)
JsonE.Net
 JSON-e (Сайт)
Json.More.Net
 Общие расширения для использования, которые, вероятно, должны были быть включены в System.Text.Json[.Nodes], но не были
Yaml2JsonNode
 Преобразования между моделью документа YAML в YamlDotNet и JsonNode (в обоих направлениях).

Все вышеперечисленные библиотеки полностью поддерживают соответствующие спецификации.

Переводы сообщений об ошибках для _JsonSchema.Net_ доступны на:

  • Итальянский it
  • Норвежский nb-NO
  • Польский pl-PL
  • Русский ru
  • Испанский es
  • Шведский sv-SE
  • Турецкий tr-TR

ПРИМЕЧАНИЕ Каждый языковой пакет предоставляется как отдельный пакет Nuget.

Вы также можете просмотреть результаты выполнения тестового набора JSON Schema на Bowtie, который запускает тестовый набор на различных реализациях и платформах.

ОТКАЗ ОТ ОТВЕТСТВЕННОСТИ Моя библиотека, _JsonSchema.Net_, не связана и не ассоциирована с сайтом https://jsonschema.net, кроме того, что оба являются отличными инструментами для работы с JSON Schema.

Если вы используете JSON для чего-то, что не покрыто библиотекой из этого набора или одним из указанных выше проектов, не стесняйтесь создать заявку на добавление функциональности.

Если вы поддерживаете или знаете о другом проекте, который расширяет _System.Text.Json_ для интересных задач, я хотел бы разместить его здесь, поэтому, пожалуйста, создайте общий issue, чтобы сообщить мне об этом.
