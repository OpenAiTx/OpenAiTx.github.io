¿Qué es json-everything?

La filosofía principal detrás de json-everything es asegurar que la funcionalidad común de JSON tenga buen soporte en el entorno de _System.Text.Json_.

El primer paso para ello es comprobar si existen otros proyectos que soporten una función determinada.

Si es así, probablemente ese esfuerzo no se duplicará aquí. Un poco más abajo encontrarás una lista de otros proyectos que están haciendo cosas interesantes con JSON.

De lo contrario, está abierto a consideración para este proyecto.

¿Qué hay en la caja?

En realidad, hay múltiples cajas. Cada funcionalidad se ha separado en su propia librería, para que puedas elegir la(s) que necesites.

Todas las librerías están construidas para .Net Standard 2.0 y .Net 8.

JsonSchema.Net
 JSON Schema, borradores 6 y superiores
JsonSchema.Net.Generation
 Generación de esquemas a partir de tipos .Net
JsonSchema.Net.Generation.DataAnnotations
 Soporte para la generación de esquemas para System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 Generación de datos de instancia aleatorios (impulsado por Bogus)
JsonSchema.Net.Data
 Un vocabulario para acceder a datos de instancia y externos
JsonSchema.Net.ArrayExt
 Un vocabulario para extender la validación de arreglos
JsonSchema.Net.OpenApi
 Extensión de vocabulario OpenApi 3.1 (utilizada por Graeae)
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) y Relative JSON Pointer (Especificación)
JsonLogic
 JsonLogic (Sitio web)
JsonE.Net
 JSON-e (Sitio web)
Json.More.Net
 Extensiones de uso general que probablemente deberían haber sido incluidas en System.Text.Json[.Nodes] pero no lo fueron
Yaml2JsonNode
 Conversiones entre el modelo de documento YAML en YamlDotNet y JsonNode (ambas direcciones).

Todas las bibliotecas anteriores ofrecen soporte completo para sus especificaciones asociadas.

NOTA Cada paquete de idioma se proporciona como su propio paquete Nuget.

También puede ver los resultados del JSON Schema Test Suite en Bowtie, que ejecuta la suite de pruebas contra múltiples implementaciones en diferentes plataformas.

AVISO LEGAL Mi biblioteca, _JsonSchema.Net_, no está relacionada ni asociada con el sitio web https://jsonschema.net, excepto que ambos son excelentes herramientas para JSON Schema.

Paquetes obsoletos

JsonSchema.Net.CodeGeneration

Soporte de _System.Text.Json_ por otros proyectos

Si no encuentra lo que busca aquí, pruebe uno de estos excelentes proyectos:

  • JsonCons.Net por @danielaparker
  • JSON Pointer
  • JSON Patch
  • JSON Merge Patch
  • JSON Path
  • JMES Path
  • Corvus.JsonSchema por @mwadams
  • Validación de JSON Schema con enfoque en generación de código C#

Si utiliza JSON para hacer algo que no está cubierto por una biblioteca de este conjunto o uno de los proyectos anteriores, siéntase libre de crear una solicitud de funcionalidad.

Si mantiene o conoce otro proyecto que extienda _System.Text.Json_ para hacer cosas interesantes, me gustaría incluirlo aquí, así que por favor cree una incidencia general para informarme.
