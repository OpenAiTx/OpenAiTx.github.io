什么是 json-everything？

json-everything 背后的主要理念是确保常见的 JSON 功能在 _System.Text.Json_ 领域中得到良好支持。

实现这一目标的第一步是检查是否存在支持某项功能的其他项目。

如果有，这项工作可能不会在此重复。稍后你会看到一些正在用 JSON 做很酷事情的其他项目列表。

否则，这项功能将开放考虑纳入本项目。

盒子里有什么？

实际上有多个盒子。每个功能都被拆分为独立的库，因此你可以根据需要选择相应的库。

所有库都基于 .Net Standard 2.0 和 .Net 8 构建。

JsonSchema.Net
 JSON Schema，支持第6版及更高
JsonSchema.Net.Generation
 从 .Net 类型生成模式
JsonSchema.Net.Api
 为 ASP.Net API 提供 JSON Schema 校验
JsonSchema.Net.Generation.DataAnnotations
 对 System.ComponentModel.DataAnnotations 的模式生成支持。
JsonSchema.Net.DataGeneration
 随机实例数据生成（由 Bogus 提供支持）
JsonSchema.Net.Data
 用于访问实例和外部数据的词汇表
JsonSchema.Net.ArrayExt
 用于扩展数组验证的词汇表
JsonSchema.Net.OpenApi
 OpenApi 3.1 词汇扩展（被 Graeae 使用）
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON 指针 (RFC 6901) 以及相对 JSON 指针 (规范)
JsonLogic
 JsonLogic (官网)
JsonE.Net
 JSON-e (官网)
Json.More.Net
 通用扩展，原本应包含在 System.Text.Json[.Nodes] 中但未被包含
Yaml2JsonNode
 YamlDotNet 的 YAML 文档模型与 JsonNode 之间的相互转换。
上述所有库都为其相关规范提供了完整支持。

_JsonSchema.Net_ 的错误消息翻译支持以下语言：

  • 意大利语 it
  • 挪威语 nb-NO
  • 波兰语 pl-PL
  • 葡萄牙语 pt
  • 俄语 ru
  • 西班牙语 es
  • 瑞典语 sv-SE
  • 土耳其语 tr-TR

注意 每个语言包均通过其独立的 Nuget 包提供。

您还可以在 Bowtie 上查看 JSON Schema 测试套件的结果，该平台在不同平台的多个实现上运行测试套件。

免责声明 我的库 _JsonSchema.Net_ 与网站 https://jsonschema.net 无任何关系或关联，除了它们都是优秀的 JSON Schema 工具。
