ما هو json-everything؟

الفلسفة الأساسية وراء json-everything هي ضمان أن تكون الوظائف الشائعة لـ JSON مدعومة بشكل جيد في مساحة _System.Text.Json_.

الخطوة الأولى نحو ذلك هي التحقق مما إذا كانت هناك مشاريع أخرى تدعم وظيفة معينة.

إذا كان الأمر كذلك، فمن المحتمل ألا يتم تكرار هذا الجهد هنا. ستجد أدناه قائمة بمشاريع أخرى تقوم بأشياء رائعة مع JSON.

وإلا، فسيتم النظر فيها لهذا المشروع.

ماذا يوجد في الصندوق؟

في الواقع هناك عدة صناديق. كل ميزة من الميزات تم تقسيمها إلى مكتبتها الخاصة، بحيث يمكنك اختيار ما تحتاج إليه فقط.

جميع المكتبات مبنية على .Net Standard 2.0 و .Net 8.

JsonSchema.Net
 JSON Schema، المسودات 6 وما بعدها
JsonSchema.Net.Generation
 توليد المخططات من أنواع .Net
JsonSchema.Net.Generation.DataAnnotations
 دعم توليد المخططات لـ System.ComponentModel.DataAnnotations.
JsonSchema.Net.DataGeneration
 توليد بيانات عشوائية للحالات (مدعوم بواسطة Bogus)
JsonSchema.Net.Data
 مفردات للوصول إلى بيانات الحالة والبيانات الخارجية
JsonSchema.Net.ArrayExt
 مفردات لتوسيع التحقق من صحة المصفوفات
JsonSchema.Net.OpenApi
 امتداد مفردات OpenApi 3.1 (يُستخدم بواسطة Graeae)
JsonPath.Net
 مسار JSON (RFC 9535)
JsonPatch.Net
 تصحيح JSON (RFC 6902)
JsonPointer.Net
 مؤشر JSON (RFC 6901) ومؤشر JSON النسبي (المواصفات)
JsonLogic
 JsonLogic (الموقع)
JsonE.Net
 JSON-e (الموقع)
Json.More.Net
 امتدادات للاستخدام العام كان يجب أن تُدرج غالباً في System.Text.Json[.Nodes] لكنها لم تدرج
Yaml2JsonNode
 تحويلات بين نموذج مستند YAML في YamlDotNet و JsonNode (في كلا الاتجاهين).

جميع المكتبات أعلاه توفر دعماً كاملاً للمواصفات المرتبطة بها.

ترجمات رسائل الأخطاء لـ _JsonSchema.Net_ متوفرة باللغات التالية:

  • الإيطالية it
  • النرويجية nb-NO
  • البولندية pl-PL
  • الروسية ru
  • الإسبانية es
  • السويدية sv-SE
  • التركية tr-TR

ملاحظة كل حزمة لغة يتم توفيرها من خلال حزمة Nuget خاصة بها.

يمكنك أيضاً عرض نتائج مجموعة اختبارات JSON Schema على Bowtie، والتي تقوم بتشغيل مجموعة الاختبار على عدة تطبيقات عبر منصات مختلفة.

تنويه مكتبتي، _JsonSchema.Net_، ليست مرتبطة أو تابعة لموقع https://jsonschema.net، باستثناء أنهما كلاهما أدوات ممتازة لـ JSON Schema.
