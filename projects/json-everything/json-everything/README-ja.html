json-everythingとは？

json-everythingの主な思想は、一般的なJSON機能に対して_System.Text.Json_空間で十分なサポートを提供することです。

その最初のステップは、特定の機能をサポートする他のプロジェクトが存在するかどうかを確認することです。

もし存在すれば、その取り組みをここで重複することはおそらくありません。もう少し下に進むと、JSONに関して素晴らしいことをしている他のプロジェクトのリストがあります。

存在しなければ、本プロジェクトでの検討対象となります。

箱の中身は？

実際には複数の箱があります。各機能は独自のライブラリとして分割されているため、必要なものだけを選択できます。

すべてのライブラリは .Net Standard 2.0 および .Net 8 に対応しています。

JsonSchema.Net
 JSON Schema、ドラフト6以降対応
JsonSchema.Net.Generation
 .Net型からスキーマ生成
JsonSchema.Net.Api
 ASP.Net APIのJSONスキーマバリデーション
JsonSchema.Net.Generation.DataAnnotations
 System.ComponentModel.DataAnnotations のスキーマ生成対応。
JsonSchema.Net.DataGeneration
 ランダムなインスタンスデータ生成（Bogus搭載）
JsonSchema.Net.Data
 インスタンスおよび外部データへのアクセスのための語彙
JsonSchema.Net.ArrayExt
 配列バリデーションを拡張するための語彙
JsonSchema.Net.OpenApi
 OpenApi 3.1 の語彙拡張（Graeae で使用）
JsonPath.Net
 JSON Path (RFC 9535)
JsonPatch.Net
 JSON Patch (RFC 6902)
JsonPointer.Net
 JSON Pointer (RFC 6901) および Relative JSON Pointer (仕様)
JsonLogic
 JsonLogic (公式サイト)
JsonE.Net
 JSON-e (公式サイト)
Json.More.Net
 一般用途の拡張。System.Text.Json[.Nodes] に本来含まれるべきだったが含まれていない機能
Yaml2JsonNode
 YamlDotNet の YAMLドキュメントモデルと JsonNode の間の変換（両方向）。
上記のすべてのライブラリは、関連する仕様への完全なサポートを提供しています。

_JsonSchema.Net_ のエラーメッセージ翻訳は以下の言語で利用可能です:

  • イタリア語 it
  • ノルウェー語 nb-NO
  • ポーランド語 pl-PL
  • ポルトガル語 pt
  • ロシア語 ru
  • スペイン語 es
  • スウェーデン語 sv-SE
  • トルコ語 tr-TR

注意 各言語パックは、それぞれ独自のNugetパッケージとして提供されています。

また、BowtieでJSON Schema Test Suiteの結果を見ることができます。Bowtieは、複数のプラットフォームにわたる様々な実装に対してテストスイートを実行します。

免責事項 私のライブラリ _JsonSchema.Net_ は、https://jsonschema.net というウェブサイトとは関係ありませんが、どちらも優れたJSON Schemaツールです。

非推奨パッケージ

これらのパッケージもこのスイートの一部でしたが、現在はサポートされていません。

JsonSchema.Net.CodeGeneration
 スキーマからC#コードを生成 Corvus.JsonSchema への移行により非推奨

他プロジェクトによる _System.Text.Json_ サポート

ここでお探しのものが見つからない場合、次の優れたプロジェクトもご利用ください:

  • JsonCons.Net （@danielaparker による）

  • JSON Pointer
  • JSON Patch
  • JSON Merge Patch
  • JSON Path
  • JMES Path
  • Corvus.JsonSchema （@mwadamsによる）
  • C#のコード生成に重点を置いたJSON Schemaバリデーション
  • （今後さらに追加予定）

もしJSONを使ってこのスイートや上記プロジェクトでカバーされていないことをしている場合は、機能追加のIssueを作成してください。

_System.Text.Json_を拡張して便利なことをしている他のプロジェクトを知っていたり、メンテナンスしている場合は、ここに掲載したいので、一般的なIssueを作成してお知らせください。
