🚀 AndroBuster

🔒 Pentesting Jadi Mudah Alat enumerasi direktori dan subdomain yang kuat, dibuat dengan Flutter

📖 Ikhtisar

AndroBuster adalah alat keamanan open-source yang terinspirasi dari GoBuster, menyediakan antarmuka modern lintas platform untuk enumerasi direktori dan subdomain. Dibangun dengan Flutter, alat ini menawarkan pengalaman pengguna yang bersih dan intuitif sekaligus tetap menjaga kekuatan dan fleksibilitas yang diharapkan dari alat keamanan profesional.

🎯 Apa yang Membuat AndroBuster Istimewa?

  • 🚀 Lintas Platform: Berjalan di Android, iOS, dan Desktop
  • 🔒 Fokus Keamanan: Dibuat khusus untuk pentester
  • 💻 UI Modern: Antarmuka cantik berbasis Flutter
  • ⚡ Performa Tinggi: Manajemen threading dan memori yang efisien
  • 📱 Mobile-First: Dioptimalkan untuk pengujian keamanan di perangkat mobile

---

🖼️ Tangkapan Layar

✨ Fitur

🗂️ Mode Enumerasi Direktori

  • Input URL Target: Penanganan URL yang bersih dengan penghapusan otomatis garis miring di akhir
  • Kode Status Negatif: Menyaring kode status HTTP yang tidak diinginkan (default: 404)
  • Ukuran Halaman Negatif: Mengecualikan respons berdasarkan panjang konten
  • Dukungan Wordlist: Input teks besar dengan kemampuan impor file
  • Kontrol Threading: Jumlah thread yang dapat dikonfigurasi untuk performa optimal
  • Pengaturan Timeout: Timeout permintaan yang dapat disesuaikan dalam milidetik

🌐 Mode Enumerasi Subdomain

  • Input Domain: Penanganan domain yang bersih (tidak perlu http/https)
  • Wordlist Subdomain: Manajemen wordlist khusus untuk penemuan subdomain
  • Pengujian Protokol: Pengujian fallback otomatis HTTPS/HTTP
  • Validasi Subdomain: Validasi nama subdomain sesuai RFC 1123
  • Filter Khusus: Kode status negatif dan ukuran halaman terpisah untuk subdomain
  • Threading & Timeout: Konfigurasi independen dari mode direktori

📋 Manajemen Hasil

  • Tab Hasil Terpadu: Melihat hasil pemindaian direktori dan subdomain
  • Format Hasil: Tampilan jelas untuk path, kode status, ukuran, dan URL
  • Salin ke Clipboard: Ekspor hasil untuk analisis eksternal
  • Hapus Hasil: Pembersihan mudah antara pemindaian
  • Jumlah Hasil: Pelacakan terpisah untuk hasil direktori dan subdomain

📁 Manajemen File

  • Impor Wordlist: Dukungan untuk file .txt hingga 50MB
  • Pratinjau Pintar: Tampilan konten berdasarkan ukuran file
  • Penanganan File Besar: Pemrosesan efisien untuk wordlist berukuran masif
  • Dukungan Format: Secara otomatis menangani komentar (#) dan baris kosong

🚀 Performa & Reliabilitas

  • Eksekusi Latar Belakang: Aplikasi tetap berjalan selama pemindaian latar belakang
  • Optimasi Memori: Penanganan wordlist besar yang efisien
  • Penanganan Error: Mekanisme fallback dan pemulihan yang baik
  • Lintas Platform: Dibangun dengan Flutter untuk Android dan iOS

📱 Instalasi

📦 Rilis Build

Saat ini, hanya rilis APK Android yang tersedia. Unduh APK terbaru dari halaman rilis.

🔧 Membangun dari Sumber

Kode sumber sekarang bersifat publik dan tersedia untuk semua orang untuk digunakan, dimodifikasi, dan berkontribusi.

📖 Panduan Penggunaan

🗂️ Enumerasi Direktori

  • Buka tab "Dir"
  • Masukkan URL target Anda
  • Konfigurasikan status kode negatif dan ukuran halaman
  • Tempel atau impor wordlist Anda
  • Atur jumlah thread dan waktu tunggu
  • Mulai pemindaian dan pantau progres

🌐 Enumerasi Subdomain

  • Buka tab "Subdomain"
  • Masukkan domain target (tanpa http/https)
  • Konfigurasikan filter khusus subdomain
  • Impor atau tempel wordlist subdomain
  • Sesuaikan pengaturan thread dan waktu tunggu
  • Mulai penemuan subdomain

📋 Manajemen Hasil

  • Lihat semua hasil di tab "Results"
  • Salin hasil ke clipboard untuk analisis eksternal
  • Bersihkan hasil antara pemindaian yang berbeda
  • Pantau progres dan penyelesaian pemindaian

---

⚙️ Konfigurasi

⚡ Jumlah Thread

  • Direkomendasikan: 10-20 thread untuk sebagian besar skenario
  • Peringatan Thread Tinggi: Peringatan jika melebihi 20 thread
  • Pertimbangan: Thread lebih tinggi dapat memicu rate limiting

⏱️ Pengaturan Timeout

  • Default: 5000ms (5 detik)
  • Penyesuaian: Naikkan untuk jaringan lambat, turunkan untuk respons lebih cepat
  • Rentang: 1000ms hingga 30000ms didukung

🚫 Filter Negatif

  • Kode Status: Daftar dipisahkan koma (misal, 404,403,500)
  • Ukuran Halaman: Daftar dipisahkan koma (misal, 0,1234,5678)
  • Tujuan: Mengecualikan respons yang tidak diinginkan dari hasil
