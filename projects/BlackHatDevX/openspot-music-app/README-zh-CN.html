# OpenSpot 音乐 🎶

通向无限音乐的门户 - 跨平台音乐流媒体

免费在所有设备上流式传输和下载高质量音乐，无广告，无需登录。

📱📺 两个惊艳应用，一个代码库

本代码库包含 两个完整的音乐流媒体应用：

🎵 OpenSpot 移动版 (React Native + Expo)

  • 平台：Android，iOS
  • 位置：openspot-music-mobile/
  • 功能：原生移动体验，支持离线下载、后台播放及持久状态

🖥️ OpenSpot 桌面版 (Electron + React)

  • 平台：macOS，Windows，Linux
  • 位置：openspot-music-electron/
  • 功能：桌面优化界面，具备全局音频控制、窗口管理和本地存储

---

✨ 主要功能（两个应用均支持）

  • 🎵 高质量流媒体：以最佳音质聆听您喜爱的曲目
  • 💾 离线下载：直接保存音乐到设备，随时离线收听
  • ❤️ 收藏与点赞：通过点赞歌曲构建您的个人收藏
  • 🔄 后台播放：在使用其他应用时继续聆听
  • 🚫 无需登录：立即使用！无需账户或注册
  • 💾 持久状态：您点赞的歌曲和最近播放的曲目会在应用重启后保存
  • 🎨 美观界面：针对各平台设计的简洁现代界面
  • 🆓 完全免费且无广告：享受无中断音乐，无任何费用或广告

📱 移动应用（React Native + Expo）

截图

下载链接

  • Android APK：OpenSpot-2.0.2-release.apk
  • iOS：即将推出（TestFlight）

开发环境搭建

cd openspot-music-app
cd openspot-music-mobile/
npm install
npx expo start

生产构建

cd openspot-music-app/
Development Build

eas build --platform android --profile development
Preview Build (APK)

eas build --platform android --profile preview
Production Build

eas build --platform android --profile production

💻 技术栈

移动应用

  • 框架: React Native 结合 Expo
  • 导航: Expo Router
  • 音频: Expo AV
  • 构建: EAS Build

桌面应用

  • 框架: Electron + React
  • 界面: Material-UI
  • 状态管理: React Context + useReducer
  • 存储: electron-store
  • 构建: electron-builder

共享部分

  • 语言: TypeScript
  • 包管理器: npm/yarn
