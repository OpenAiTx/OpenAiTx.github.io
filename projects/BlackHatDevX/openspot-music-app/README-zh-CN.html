📖 概览

本 monorepo 提供了两款完全独立的音乐流媒体应用，采用前沿技术打造：

| 应用 | 平台 | 框架 | 位置 | |-----|-----------|-----------|----------| | 📱 移动端 | Android，iOS (即将推出) | React Native + Expo | openspot-mobile/ | | 🖥️ 桌面端 | macOS，Windows/Linux (即将推出) | Tauri + React | openspot-desktop/ |

两款应用共享强大的流媒体核心，带来全设备无缝聆听体验。
