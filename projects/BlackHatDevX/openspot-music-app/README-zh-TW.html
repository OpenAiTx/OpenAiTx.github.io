📖 概述

此 monorepo 提供兩套完全獨立的音樂串流應用程式，採用最先進技術打造：

| 應用程式 | 平台 | 框架 | 位置 | |----------|------|------|------| | 📱 行動版 | Android, iOS (即將推出) | React Native + Expo | openspot-mobile/ | | 🖥️ 桌面版 | macOS, Windows/Linux (即將推出) | Tauri + React | openspot-desktop/ |

兩個應用程式共用同一強大的串流核心，讓您於所有裝置間享受無縫聆聽體驗。

🧰 技術堆疊

📱 行動版

React Native · Expo Router · Expo AV · EAS Build · IPinfo API

🖥️ 桌面版

Tauri · React · Material‑UI · React Context + useReducer
