OpenSpot Music 🎶

Sonsuz Müzik İçin Geçidiniz - Çoklu Platform Müzik Akışı

Tüm cihazlarınızda yüksek kaliteli müziği ücretsiz olarak dinleyin ve indirin, reklamsız ve giriş yapmadan.

📱📺 İki Harika Uygulama, Tek Depo

Bu depo iki tam müzik akış uygulaması içerir:

🎵 OpenSpot Mobile (React Native + Expo)

  • Platformlar: Android, iOS
  • Konum: openspot-music-mobile/
  • Özellikler: Çevrimdışı indirme, arka planda oynatma ve kalıcı durum ile yerel mobil deneyim

🖥️ OpenSpot Desktop (Electron + React)

  • Platformlar: macOS, Windows, Linux
  • Konum: openspot-music-electron/
  • Özellikler: Global ses kontrolleri, pencere yönetimi ve yerel depolama ile masaüstü için optimize edilmiş arayüz

📱 Mobil Uygulama (React Native + Expo)

Ekran Görüntüleri

İndirme Linkleri

  • Android APK: OpenSpot-2.0.2-release.apk
  • iOS: Çok Yakında (TestFlight)

Geliştirme Kurulumu

cd openspot-music-app
cd openspot-music-mobile/
npm install
npx expo start

Üretim İçin Derle

cd openspot-music-app/
Development Build

eas build --platform android --profile development
Preview Build (APK)

eas build --platform android --profile preview
Production Build

eas build --platform android --profile production

🖥️ Masaüstü Uygulaması (Electron + React)

Ekran Görüntüleri

İndirme Linkleri

  • macOS DMG: OpenSpot.Music-2.0.2-arm64.dmg
  • Windows EXE: Yakında!
  • Linux DEB: Yakında!

Geliştirme Kurulumu

cd openspot-music-app/
cd openspot-music-electron/
npm install
npm run electron-dev

Üretim için Derleme

cd openspot-music-app/
cd openspot-music-electron/
npm run build
npm run electron-pack

💻 Teknoloji Yığını

Mobil Uygulama

  • Çerçeve: React Native ve Expo
  • Navigasyon: Expo Router
  • Ses: Expo AV
  • Derleme: EAS Build

Masaüstü Uygulaması

  • Çerçeve: Electron + React
  • Arayüz: Material-UI
  • Durum: React Context + useReducer
  • Depolama: electron-store
  • Derleme: electron-builder

Ortak

  • Dil: TypeScript
  • Paket Yöneticisi: npm/yarn
