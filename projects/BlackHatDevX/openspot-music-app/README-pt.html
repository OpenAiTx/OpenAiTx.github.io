OpenSpot Music 🎶

Seu Portal para Música Ilimitada - Streaming Musical Multiplataforma

Transmita e faça download de músicas em alta qualidade gratuitamente em todos os seus dispositivos, sem anúncios e sem necessidade de login.

📱📺 Dois Apps Incríveis, Um Repositório

Este repositório contém dois aplicativos completos de streaming de música:

🎵 OpenSpot Mobile (React Native + Expo)

  • Plataformas: Android, iOS
  • Localização: openspot-music-mobile/
  • Funcionalidades: Experiência móvel nativa com downloads offline, reprodução em segundo plano e estado persistente

🖥️ OpenSpot Desktop (Electron + React)

  • Plataformas: macOS, Windows, Linux
  • Localização: openspot-music-electron/
  • Funcionalidades: Interface otimizada para desktop com controles globais de áudio, gerenciamento de janelas e armazenamento local

📱 Aplicativo Mobile (React Native + Expo)

Capturas de Tela

Links para Download

  • Android APK: OpenSpot-2.0.3-release.apk
  • iOS: Precisamos de um mantenedor

Configuração de Desenvolvimento

cd openspot-music-app
cd openspot-music-mobile/
npm install
npx expo start

Compilar para Produção

cd openspot-music-app/
Development Build

eas build --platform android --profile development
Preview Build (APK)

eas build --platform android --profile preview
Production Build

eas build --platform android --profile production

💻 Stack Tecnológico
App Mobile


  • Framework: React Native com Expo
    • 

  • Navegação: Expo Router
    • 

  • Áudio: Expo AV
    • 

  • Build: EAS Build
    • 

  • Busca de Localização: IPinfo API
App Desktop


  • Framework: Electron + React
    • 

  • UI: Material-UI
    • 

  • Estado: React Context + useReducer
    • 

  • Armazenamento: electron-store
    • 

  • Build: electron-builder
Compartilhado
