Documentación de USSO

Documentación oficial de USSO - Una plataforma SSO segura y multiinquilino.

📖 Acerca de

Este repositorio contiene la documentación completa de USSO, construida con MkDocs y Material for MkDocs.

Documentación en Vivo: docs.usso.io (próximamente)

🚀 Inicio Rápido

Requisitos Previos

  • Python 3.10+
  • pip

Desarrollo Local

  • Clona el repositorio

   git clone https://github.com/ussoio/usso-docs.git
   cd usso-docs
   ``
  • Instalar dependencias
    •    
`bash
   pip install -r requirements.txt
   `
  • Inicie el servidor de desarrollo
    •    
`bash
   mkdocs serve
   `

  • Abre tu navegador
    •    
Visita http://localhost:8000

Convenciones de Markdown

  • Use # para el título de la página (h1) - solo una vez por página
  • Use ## para secciones principales (h2)
  • Use ### para subsecciones (h3)
  • Use bloques de código con lenguaje: \\\python
  • Use admoniciones para notas/advertencias:

  !!! tip "Pro Tip"
      This is a helpful tip
  
  !!! warning "Important"
      Pay attention to this
  `
