USSO 文档

USSO 的官方文档 - 一个安全的多租户单点登录平台。

📖 关于

本仓库包含 USSO 的完整文档，使用 MkDocs 和 Material for MkDocs 构建。

在线文档：docs.usso.io（即将上线）

🚀 快速开始

先决条件

  • Python 3.10+
  • pip

本地开发

  • 克隆仓库

   git clone https://github.com/ussoio/usso-docs.git
   cd usso-docs
   ``
  • 安装依赖项
    •    
`bash
   pip install -r requirements.txt
   `
  • 启动开发服务器
    •    
`bash
   mkdocs serve
   `
  • 打开你的浏览器
    •    
访问 http://localhost:8000

一般指南
  • 使用 清晰、简单的语言

代码示例
始终提供至少3种格式的示例：
markdown === "Python"

    import requests
    
    response = requests.post(...)
    ``
=== "JavaScript"
    `javascript
    const response = await fetch(...)
    `
=== "cURL"
    `bash
    curl -X POST ...
    `

Markdown 约定

  • 使用 # 表示页面标题（h1）- 每页仅使用一次
  • 使用 ## 表示主章节（h2）
  • 使用 ### 表示子章节（h3）
  • 使用代码块并指定语言：\\\python
  • 使用提示框表示注释/警告：

  !!! tip "Pro Tip"
      This is a helpful tip
  
  !!! warning "Important"
      Pay attention to this
  `
