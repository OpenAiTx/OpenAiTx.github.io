Aperçu

Mon Blog

Qui utilise

N'hésitez pas à soumettre une Issue pour présenter votre site web.

Commencer

  • Installer le thème

  • Installer pnpm
  •      pnpm create astro@latest --template Liksone/astro-theme-sora
``

  • Entrez dans le dossier du projet et lancez pnpm dev pour démarrer le projet

  • Configurer le thème

Modifiez le fichier de configuration theme.config.ts pour personnaliser le thème. Voir Configuration.

  • Éditer les articles

Éditez dans le dossier src/content/, le format Markdown est supporté.

  • Déployer le site

Exécutez pnpm build pour construire le site, le résultat se trouve dans le dossier dist/. Vous pouvez lancer pnpm preview pour prévisualiser localement le site.

Déployez votre site sur votre propre serveur, ou consultez la documentation Astro pour le déployer sur des plateformes d'hébergement telles que Vercel, Netlify, GitHub Pages, etc.

Configuration

Le fichier de configuration est theme.config.ts.

Informations du site
