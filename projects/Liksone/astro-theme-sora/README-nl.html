Mijn Blog

Wie gebruikt het

Voel je vrij om een Issue aan te maken om je eigen website te laten zien.

Aan de slag

  • Installeer het thema

  • Installeer pnpm
  •      pnpm create astro@latest --template Liksone/astro-theme-sora
``

  • Ga naar de projectmap en voer pnpm dev uit om het project te starten

  • Thema configureren

Pas het configuratiebestand theme.config.ts aan om het thema te personaliseren. Zie Configuratie.

  • Bewerk artikelen

Bewerk in de map src/content/, ondersteund wordt het Markdown-formaat.

  • Website deployen

Voer pnpm build uit om de website te bouwen; de output staat in de map dist/. Je kunt pnpm preview uitvoeren om de website lokaal te bekijken.

Deploy de website zelf op een server, of volg de Astro documentatie om te deployen naar Vercel, Netlify, GitHub Pages en andere hostingplatforms.
