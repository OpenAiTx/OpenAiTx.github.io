시작하기

  • 테마 설치

  • pnpm 설치
  •      pnpm create astro@latest --template Liksone/astro-theme-sora
``

  • 프로젝트 디렉터리로 이동 후, pnpm dev를 실행하여 프로젝트 시작

  • 테마 설정

설정 파일 theme.config.ts를 수정하여 테마를 커스터마이즈합니다. 자세한 내용은 설정 참고.

  • 글 작성

src/content/ 디렉터리에서 작성, Markdown 형식 지원.

  • 사이트 배포

pnpm build를 실행하여 사이트를 빌드하면, 결과물이 dist/ 디렉터리에 생성됩니다. pnpm preview로 로컬에서 미리보기 가능.

서버에 직접 배포하거나, Astro 문서를 참고하여 Vercel, Netlify, GitHub Pages 등 호스팅 플랫폼에 배포할 수 있습니다.

설정

설정 파일은 theme.config.ts입니다.
