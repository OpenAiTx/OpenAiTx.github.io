من يستخدم

البدء

  • تثبيت السمة

  • تثبيت pnpm
  •      pnpm create astro@latest --template Liksone/astro-theme-sora
``

  • ادخل إلى مجلد المشروع، ثم شغّل pnpm dev لبدء المشروع

  • ضبط السمة

قم بتعديل ملف الإعدادات theme.config.ts لتخصيص السمة. لمزيد من التفاصيل راجع الإعدادات.

  • تحرير المقالات

حرر في مجلد src/content/، ويدعم صيغة Markdown.

  • نشر الموقع

شغّل pnpm build لبناء الموقع، حيث يتم حفظ الملفات الناتجة في مجلد dist/. يمكنك تشغيل pnpm preview لمعاينة الموقع محلياً.

يمكنك نشر الموقع على الخادم الخاص بك، أو الرجوع إلى دليل Astro للنشر على Vercel أو Netlify أو GitHub Pages أو منصات الاستضافة الأخرى.
