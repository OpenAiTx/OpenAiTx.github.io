Inizio

  • Installa il tema

  • Installa pnpm
  •      pnpm create astro@latest --template Liksone/astro-theme-sora
``

  • Entra nella directory del progetto ed esegui pnpm dev per avviare il progetto

  • Configura il tema

Modifica il file di configurazione theme.config.ts per personalizzare il tema. Vedi Configurazione.

  • Modifica gli articoli

Modifica nella directory src/content/, supporta il formato Markdown.

  • Distribuisci il sito

Esegui pnpm build per costruire il sito, i file generati si trovano nella cartella dist/. Puoi eseguire pnpm preview per visualizzare il sito in locale.

Distribuisci il sito sul tuo server, oppure segui la documentazione Astro per pubblicare su Vercel, Netlify, GitHub Pages e altre piattaforme.

Configurazione

Il file di configurazione è theme.config.ts.
