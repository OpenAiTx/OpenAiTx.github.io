PDFMathTranslate

PDFの科学論文の翻訳と二言語比較。

  • 📊 数式、グラフ、目次、注釈を保持 _(プレビュー)_。
  • 🌐 多言語および多様な翻訳サービスをサポート。
  • 🤖 コマンドラインツール、インタラクティブユーザーインターフェース、およびDockerを提供します。

フィードバックはGitHub IssuesまたはTelegramグループでお気軽にどうぞ。

貢献方法の詳細については、貢献ガイドをご参照ください。

プレビュー

オンラインサービス 🌟

以下のデモのいずれかを利用してアプリケーションを試すことができます：

  • パブリック無料サービス インストール不要でオンライン利用可能 _(推奨)_。
  • Immersive Translate - BabelDOC 月間1000ページ無料。 _(推奨)_
  • HuggingFaceでホストされているデモ
  • ModelScopeでホストされているデモ インストール不要。

デモの計算リソースは限られているため、乱用はご遠慮ください。

インストールと使用方法

方法

異なるユースケースに対応して、当プログラムの使用方法をいくつか提供しています：

1. UV インストール

  • Python がインストールされていること（3.10 <= バージョン <= 3.12）

  • 当パッケージをインストールする：

   pip install uv
   uv tool install --python 3.12 pdf2zh
   ``

  • 翻訳を実行し、ファイルはcurrent working directoryに生成されます:
    • 
   
`bash
   pdf2zh document.pdf
   `

2. Windows exe

  • リリースページ から pdf2zh-version-win64.zip をダウンロードします

  • 解凍して pdf2zh.exe をダブルクリックして実行します。

3. グラフィカルユーザーインターフェース

  • Pythonがインストールされていること（バージョン3.10 <= version <= 3.12）

  • 当パッケージをインストールします：
bash pip install pdf2zh 


  • ブラウザでの使用を開始する:
    •    
`bash
   pdf2zh -i
   `
  • ブラウザが自動的に起動しない場合は、次の場所に移動してください。
    •    
`bash
   http://localhost:7860/
   `
   
詳細については、GUIのドキュメントを参照してください。

4. Docker

  • プルして実行します:

`bash docker pull byaidu/pdf2zh docker run -d -p 7860:7860 byaidu/pdf2zh `

  • ブラウザで開く:

` http://localhost:7860/ ` クラウドサービスでのDockerデプロイ用:

5. Zotero プラグイン

詳細はZotero PDF2zhをご覧ください。

6. コマンドライン

  • Pythonがインストールされていること (3.10 <= バージョン <= 3.12)
  • 当パッケージをインストールしてください:

`bash pip install pdf2zh `
  • 翻訳を実行し、ファイルはcurrent working directoryに生成されます:

`bash pdf2zh document.pdf `

[!TIP]
>
- Windowsを使用していてダウンロード後にファイルが開けない場合は、vc_redist.x64.exeをインストールしてから再試行してください。
>
- Docker Hubにアクセスできない場合は、GitHub Container Registryのイメージを試してください。
`bash
docker pull ghcr.io/byaidu/pdfmathtranslate
docker run -d -p 7860:7860 ghcr.io/byaidu/pdfmathtranslate
`

インストールできない場合？

本プログラムは動作前にAIモデル(wybxc/DocLayout-YOLO-DocStructBench-onnx)を必要とし、一部のユーザーはネットワークの問題でダウンロードできない場合があります。このモデルのダウンロードに問題がある場合、以下の環境変数を使用した回避策を提供しています： shell set HF_ENDPOINT=https://hf-mirror.com 

PowerShell ユーザー向け:
shell $env:HF_ENDPOINT = https://hf-mirror.com ` もしこの解決策がうまくいかない場合や、他の問題が発生した場合は、よくある質問をご参照ください。

高度なオプション

コマンドラインで翻訳コマンドを実行すると、現在の作業ディレクトリに翻訳済みドキュメント example-mono.pdf と二言語ドキュメント example-dual.pdf が生成されます。デフォルトの翻訳サービスとしてGoogleを使用します。対応可能な翻訳サービスの詳細はこちらをご覧ください。

以下の表に、すべての高度なオプションを参考用に一覧にしています:

| オプション | 機能 | 例 | | --------------------- | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- | | files | ローカルファイル | pdf2zh ~/local.pdf | | links | オンラインファイル | pdf2zh http://arxiv.org/paper.pdf | | -i | GUIを起動 | pdf2zh -i | | -p | 部分的ドキュメント翻訳 | pdf2zh example.pdf -p 1 | | -li | ソース言語 | pdf2zh example.pdf -li en | | -lo | ターゲット言語 | pdf2zh example.pdf -lo zh | | -s | 翻訳サービス | pdf2zh example.pdf -s deepl | | -t | マルチスレッド | pdf2zh example.pdf -t 1 | | -o | 出力ディレクトリ | pdf2zh example.pdf -o output | | -f, -c | 例外処理 | pdf2zh example.pdf -f "(MS.*)" | | -cp | 互換モード | pdf2zh example.pdf --compatible | | --skip-subset-fonts | フォントサブセットをスキップ | pdf2zh example.pdf --skip-subset-fonts | | --ignore-cache | 翻訳キャッシュを無視 | pdf2zh example.pdf --ignore-cache | | --share | 公開リンク | pdf2zh -i --share | | --authorized | 認証 | pdf2zh -i --authorized users.txt [auth.html] | | --prompt | カスタムプロンプト | pdf2zh --prompt [prompt.txt] | | --onnx | [カスタムDocLayout-YOLO ONNXモデルを使用] | pdf2zh --onnx [onnx/model/path] | | --serverport | [カスタムWebUIポートを使用] | pdf2zh --serverport 7860 | | --dir | [バッチ翻訳] | pdf2zh --dir /path/to/translate/ | | --config | 設定ファイル | pdf2zh --config /path/to/config/config.json | | --serverport | [カスタムgradioサーバーポート] | pdf2zh --serverport 7860 | | --babeldoc | 実験的バックエンドBabelDOCを使用して翻訳 | pdf2zh --babeldoc -s openai example.pdf | | --mcp | MCP STDIOモードを有効化 | pdf2zh --mcp | | --sse | MCP SSEモードを有効化 | pdf2zh --mcp --sse` |
