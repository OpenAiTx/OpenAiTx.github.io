PDFMathTranslate

Traduction d’articles scientifiques au format PDF et comparaison bilingue.

  • 📊 Préservation des formules, graphiques, table des matières et annotations _(aperçu)_.
  • 🌐 Support de plusieurs langues et de divers services de traduction.
  • 🤖 Fournit un outil en ligne de commande, une interface utilisateur interactive et un Docker

N’hésitez pas à donner votre avis dans les Issues GitHub ou le Groupe Telegram.

Pour les détails sur la contribution, veuillez consulter le Guide de Contribution.

Vous pouvez essayer notre application en utilisant l’une des démos suivantes :

  • Service public gratuit en ligne sans installation _(recommandé)_.
  • Immersive Translate - BabelDOC 1000 pages gratuites par mois. _(recommandé)_

Installation et Utilisation

Méthodes

Pour différents cas d'utilisation, nous proposons des méthodes distinctes pour utiliser notre programme :

1. Installation UV

  • Python installé (3.10 <= version <= 3.12)

  • Installer notre paquet :

   pip install uv
   uv tool install --python 3.12 pdf2zh
   ``
  • Exécuter la traduction, fichiers générés dans répertoire de travail actuel :
    •    
`bash
   pdf2zh document.pdf
   `

2. Exécutable Windows

  • Téléchargez pdf2zh-version-win64.zip depuis la page des releases

  • Décompressez et double-cliquez sur pdf2zh.exe pour lancer.

3. Interface graphique utilisateur

  • Python installé (3.10 <= version <= 3.12)

  • Installez notre package :
bash pip install pdf2zh 
3. Commencez à utiliser dans le navigateur :

   `bash
   pdf2zh -i
   `

  • Si votre navigateur ne s'est pas lancé automatiquement, allez à  
    • 
   
`bash
   http://localhost:7860/
   `
   
Voir la documentation pour l'interface graphique (GUI) pour plus de détails.

4. Docker

  • Télécharger et exécuter :

`bash docker pull byaidu/pdf2zh docker run -d -p 7860:7860 byaidu/pdf2zh `

  • Ouvrir dans le navigateur :

` http://localhost:7860/ `

Pour le déploiement Docker sur un service cloud :

5. Plugin Zotero

Voir Zotero PDF2zh pour plus de détails.

6. Ligne de commande

  • Python installé (3.10 <= version <= 3.12)
  • Installer notre paquet :

`bash pip install pdf2zh `

  • Exécuter la traduction, fichiers générés dans répertoire de travail actuel :

`bash pdf2zh document.pdf `

[!ASTUCE]
>
- Si vous utilisez Windows et ne pouvez pas ouvrir le fichier après l'avoir téléchargé, veuillez installer vc_redist.x64.exe et réessayez.
>
- Si vous ne pouvez pas accéder à Docker Hub, veuillez essayer l'image sur GitHub Container Registry.
`bash
docker pull ghcr.io/byaidu/pdfmathtranslate
docker run -d -p 7860:7860 ghcr.io/byaidu/pdfmathtranslate
`

Impossible d'installer ?

Le programme actuel nécessite un modèle d'IA (wybxc/DocLayout-YOLO-DocStructBench-onnx) avant de fonctionner et certains utilisateurs ne peuvent pas le télécharger à cause de problèmes réseau. Si vous avez un problème pour télécharger ce modèle, nous proposons une solution de contournement en utilisant la variable d'environnement suivante : shell set HF_ENDPOINT=https://hf-mirror.com 

Pour l'utilisateur PowerShell :
shell $env:HF_ENDPOINT = https://hf-mirror.com ` Si la solution ne fonctionne pas pour vous / vous rencontrez d'autres problèmes, veuillez consulter la FAQ.

Options Avancées

Exécutez la commande de traduction dans la ligne de commande pour générer le document traduit example-mono.pdf et le document bilingue example-dual.pdf dans le répertoire de travail actuel. Utilisez Google comme service de traduction par défaut. Plus de services de traduction supportés sont disponibles ICI.

Dans le tableau suivant, nous listons toutes les options avancées à titre de référence :

| Option | Fonction | Exemple | | --------------------- | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- | | files | Fichiers locaux | pdf2zh ~/local.pdf | | links | Fichiers en ligne | pdf2zh http://arxiv.org/paper.pdf | | -i | Entrer dans l'interface graphique | pdf2zh -i | | -p | Traduction partielle du document | pdf2zh example.pdf -p 1 | | -li | Langue source | pdf2zh example.pdf -li en | | -lo | Langue cible | pdf2zh example.pdf -lo zh | | -s | Service de traduction | pdf2zh example.pdf -s deepl | | -t | Multi-threads | pdf2zh example.pdf -t 1 | | -o | Répertoire de sortie | pdf2zh example.pdf -o output | | -f, -c | Exceptions | pdf2zh example.pdf -f "(MS.*)" | | -cp | Mode compatibilité | pdf2zh example.pdf --compatible | | --skip-subset-fonts | Ignorer le sous-ensemble de polices | pdf2zh example.pdf --skip-subset-fonts | | --ignore-cache | Ignorer le cache de traduction | pdf2zh example.pdf --ignore-cache | | --share | Lien public | pdf2zh -i --share | | --authorized | Autorisation | pdf2zh -i --authorized users.txt [auth.html] | | --prompt | Invite personnalisée | pdf2zh --prompt [prompt.txt] | | --onnx | [Utiliser un modèle DocLayout-YOLO ONNX personnalisé] | pdf2zh --onnx [onnx/model/path] | | --serverport | [Utiliser un port WebUI personnalisé] | pdf2zh --serverport 7860 | | --dir | [Traduction par lot] | pdf2zh --dir /path/to/translate/ | | --config | Fichier de configuration | pdf2zh --config /path/to/config/config.json | | --serverport | [Port serveur gradio personnalisé] | pdf2zh --serverport 7860 | | --babeldoc | Utiliser le backend expérimental BabelDOC pour traduire | pdf2zh --babeldoc -s openai example.pdf | | --mcp | Activer le mode MCP STDIO | pdf2zh --mcp | | --sse | Activer le mode MCP SSE | pdf2zh --mcp --sse` |

Pour des explications détaillées, veuillez vous référer à notre document sur l'utilisation avancée pour la liste complète de chaque option.
