PDFMathTranslate

Traducción de artículos científicos en PDF y comparación bilingüe.

  • 📊 Conserva fórmulas, gráficos, tabla de contenidos y anotaciones _(vista previa)_.
  • 🌐 Soporta múltiples idiomas, y diversos servicios de traducción.
  • 🤖 Proporciona herramienta de línea de comandos, interfaz de usuario interactiva y Docker

No dude en enviar sus comentarios en GitHub Issues o en el Grupo de Telegram.

Para detalles sobre cómo contribuir, consulte la Guía de Contribución.

Instalación y Uso

Métodos

Para diferentes casos de uso, ofrecemos métodos distintos para usar nuestro programa:

1. Instalación UV

  • Python instalado (3.10 <= versión <= 3.12)

  • Instale nuestro paquete:

   pip install uv
   uv tool install --python 3.12 pdf2zh
   ``

  • Ejecutar la traducción, archivos generados en directorio de trabajo actual:
    • 
   
`bash
   pdf2zh document.pdf
   `

2. Windows exe

  • Descargue pdf2zh-version-win64.zip desde la página de lanzamientos

  • Descomprima y haga doble clic en pdf2zh.exe para ejecutar.

3. Interfaz gráfica de usuario

  • Python instalado (3.10 <= versión <= 3.12)

  • Instale nuestro paquete:
bash pip install pdf2zh 


  • Comience a usar en el navegador:
    •    
`bash
   pdf2zh -i
   `

  • Si su navegador no se ha iniciado automáticamente, vaya a
    • 
   
`bash
   http://localhost:7860/
   `
   
Vea la documentación para GUI para más detalles.

4. Docker

  • Descargar y ejecutar:

`bash docker pull byaidu/pdf2zh docker run -d -p 7860:7860 byaidu/pdf2zh `
  • Abrir en el navegador:

` http://localhost:7860/ `

Para despliegue de Docker en servicio en la nube:

5. Plugin de Zotero

Consulta Zotero PDF2zh para más detalles.

6. Línea de comandos

  • Python instalado (3.10 <= versión <= 3.12)
  • Instala nuestro paquete:

`bash pip install pdf2zh `

  • Ejecutar la traducción, archivos generados en directorio de trabajo actual:

`bash pdf2zh document.pdf `

[!TIP]
>
- Si estás usando Windows y no puedes abrir el archivo después de descargarlo, por favor instala vc_redist.x64.exe y vuelve a intentarlo.
>
- Si no puedes acceder a Docker Hub, prueba la imagen en GitHub Container Registry.
`bash
docker pull ghcr.io/byaidu/pdfmathtranslate
docker run -d -p 7860:7860 ghcr.io/byaidu/pdfmathtranslate
`

¿No se puede instalar?

El programa actual necesita un modelo de IA (wybxc/DocLayout-YOLO-DocStructBench-onnx) antes de funcionar y algunos usuarios no pueden descargarlo debido a problemas de red. Si tienes problemas para descargar este modelo, ofrecemos una solución alternativa usando la siguiente variable de entorno: shell set HF_ENDPOINT=https://hf-mirror.com 


Para usuarios de PowerShell:
shell $env:HF_ENDPOINT = https://hf-mirror.com ` Si la solución no funciona para usted / encontró otros problemas, por favor consulte las preguntas frecuentes.

Opciones Avanzadas

Ejecute el comando de traducción en la línea de comandos para generar el documento traducido example-mono.pdf y el documento bilingüe example-dual.pdf en el directorio de trabajo actual. Use Google como el servicio de traducción predeterminado. Puede encontrar más servicios de traducción compatibles AQUÍ.

En la siguiente tabla, enumeramos todas las opciones avanzadas para referencia:

| Opción | Función | Ejemplo | | --------------------- | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- | | files | Archivos locales | pdf2zh ~/local.pdf | | links | Archivos en línea | pdf2zh http://arxiv.org/paper.pdf | | -i | Entrar a la interfaz gráfica | pdf2zh -i | | -p | Traducción parcial del documento | pdf2zh example.pdf -p 1 | | -li | Idioma de origen | pdf2zh example.pdf -li en | | -lo | Idioma de destino | pdf2zh example.pdf -lo zh | | -s | Servicio de traducción | pdf2zh example.pdf -s deepl | | -t | Multihilos | pdf2zh example.pdf -t 1 | | -o | Directorio de salida | pdf2zh example.pdf -o output | | -f, -c | Excepciones | pdf2zh example.pdf -f "(MS.*)" | | -cp | Modo de compatibilidad | pdf2zh example.pdf --compatible | | --skip-subset-fonts | Omitir subconjunto de fuentes | pdf2zh example.pdf --skip-subset-fonts | | --ignore-cache | Ignorar caché de traducción | pdf2zh example.pdf --ignore-cache | | --share | Enlace público | pdf2zh -i --share | | --authorized | Autorización | pdf2zh -i --authorized users.txt [auth.html] | | --prompt | Indicador personalizado | pdf2zh --prompt [prompt.txt] | | --onnx | [Usar modelo personalizado DocLayout-YOLO ONNX] | pdf2zh --onnx [onnx/model/path] | | --serverport | [Usar puerto personalizado para WebUI] | pdf2zh --serverport 7860 | | --dir | [traducción por lotes] | pdf2zh --dir /path/to/translate/ | | --config | Archivo de configuración | pdf2zh --config /path/to/config/config.json | | --serverport | [puerto personalizado para servidor gradio] | pdf2zh --serverport 7860 | | --babeldoc | Usar backend experimental BabelDOC para traducir | pdf2zh --babeldoc -s openai example.pdf | | --mcp | Habilitar modo MCP STDIO | pdf2zh --mcp | | --sse | Habilitar modo MCP SSE | pdf2zh --mcp --sse` |

Para explicaciones detalladas, por favor consulte nuestro documento sobre Uso Avanzado para una lista completa de cada opción.
