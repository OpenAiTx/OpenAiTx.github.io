Excalidraw 編輯器 | 部落格 | 文件 | Excalidraw+

一個開源的虛擬手繪風格白板。
支援協作與端對端加密。


功能特色

Excalidraw 編輯器（npm 套件）支援：

  • 💯 免費與開源。
  • 🎨 無限大、以畫布為基礎的白板。
  • ✍️ 手繪風格。
  • 🌓 深色模式。
  • 🏗️ 可自訂。
  • 📷 圖像支援。
  • 😀 支援圖形庫。
  • 🌐 多語系（i18n）支援。
  • 🖼️ 匯出為 PNG、SVG 及剪貼簿。
  • 💾 開放格式 - 可將繪圖匯出為 .excalidraw json 檔案。
  • ⚒️ 多樣化工具 - 矩形、圓形、菱形、箭頭、線條、自由繪圖、橡皮擦等。
  • ➡️ 箭頭綁定與標籤箭頭。
  • 🔙 復原 / 重做。
  • 🔍 支援縮放與平移。

Excalidraw.com

託管於 excalidraw.com 的應用程式是您可以用 Excalidraw 打造的極簡展示。其 原始碼 亦包含於此儲存庫，並擁有以下功能：

  • 📡 PWA 支援（可離線運作）。
  • 🤼 即時協作。
  • 🔒 端對端加密。
  • 💾 本地優先支援（自動儲存至瀏覽器）。
  • 🔗 可分享連結（匯出為唯讀連結，方便分享）。

這些功能未來將以即插即用的外掛形式提供給 npm 套件。

快速開始

注意： 下列說明用於安裝 Excalidraw npm 套件，以便將 Excalidraw 整合到您自己的應用程式。如需本地端開發運行儲存庫，請參考我們的開發指南。

使用 npm 或 yarn 安裝套件。

npm install react react-dom @excalidraw/excalidraw

或

yarn add react react-dom @excalidraw/excalidraw
