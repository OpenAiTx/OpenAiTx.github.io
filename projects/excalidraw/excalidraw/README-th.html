คุณสมบัติ

โปรแกรมแก้ไข Excalidraw (แพ็คเกจ npm) รองรับ:

  • 💯 ฟรี & โอเพ่นซอร์ส
  • 🎨 ไวท์บอร์ดแบบผืนผ้าใบไร้ขีดจำกัด
  • ✍️ สไตล์ลายเส้นวาดมือ
  • 🌓 โหมดมืด (Dark mode)
  • 🏗️ ปรับแต่งได้
  • 📷 รองรับรูปภาพ
  • 😀 รองรับไลบรารีรูปทรง
  • 🌐 รองรับการแปลภาษา (i18n)
  • 🖼️ ส่งออกเป็น PNG, SVG & คลิปบอร์ด
  • 💾 รูปแบบไฟล์แบบเปิด - ส่งออกภาพวาดเป็นไฟล์ .excalidraw ในรูปแบบ json
  • ⚒️ เครื่องมือหลากหลาย - สี่เหลี่ยม วงกลม สี่เหลี่ยมข้าวหลามตัด ลูกศร เส้น วาดอิสระ ยางลบ...
  • ➡️ การผูกลูกศร & ลูกศรติดป้ายชื่อ
  • 🔙 ย้อนกลับ / ทำซ้ำ (Undo/Redo)
  • 🔍 รองรับการซูมและเลื่อนผ้าใบ

Excalidraw.com

แอปที่โฮสต์บน excalidraw.com เป็นตัวอย่างที่เรียบง่ายของสิ่งที่คุณสามารถสร้างได้ด้วย Excalidraw โดย ซอร์สโค้ด ของแอปเป็นส่วนหนึ่งของรีโพซิทอรีนี้ และแอปรองรับคุณสมบัติ:

  • 📡 รองรับ PWA (ใช้งานแบบออฟไลน์ได้)
  • 🤼 ทำงานร่วมกันแบบเรียลไทม์
  • 🔒 เข้ารหัสแบบครบวงจร
  • 💾 รองรับการบันทึกลงเครื่องเป็นหลัก (บันทึกอัตโนมัติในเบราว์เซอร์)
  • 🔗 ลิงก์ที่แชร์ได้ (ส่งออกเป็นลิงก์อ่านอย่างเดียวเพื่อแชร์กับผู้อื่น)

เราจะเพิ่มคุณสมบัติเหล่านี้เป็นปลั๊กอินที่สามารถติดตั้งได้ในแพ็คเกจ npm ในอนาคต

เริ่มต้นอย่างรวดเร็ว

หมายเหตุ: คำแนะนำต่อไปนี้สำหรับการติดตั้งแพ็คเกจ Excalidraw npm package เมื่อผสาน Excalidraw เข้ากับแอปของคุณเอง หากต้องการรันรีโพซิทอรีนี้เพื่อพัฒนา กรุณาดู คู่มือการพัฒนา

ใช้ npm หรือ yarn เพื่อติดตั้งแพ็คเกจ

npm install react react-dom @excalidraw/excalidraw

หรือ

yarn add react react-dom @excalidraw/excalidraw

ดูรายละเอียดเพิ่มเติมได้ที่ เอกสาร!
