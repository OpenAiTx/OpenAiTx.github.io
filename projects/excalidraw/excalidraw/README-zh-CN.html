一个开源的虚拟手绘风格白板。
支持协作并端到端加密。


功能特性

Excalidraw 编辑器（npm 包）支持：

  • 💯 免费和开源。
  • 🎨 无限画布白板。
  • ✍️ 手绘风格。
  • 🌓 深色模式。
  • 🏗️ 可定制。
  • 📷 支持图片。
  • 😀 支持形状库。
  • 🌐 支持本地化（i18n）。
  • 🖼️ 导出为 PNG、SVG 及剪贴板。
  • 💾 开放格式——可将绘图导出为 .excalidraw json 文件。
  • ⚒️ 丰富的工具——矩形、圆形、菱形、箭头、直线、自由绘制、橡皮擦等。
  • ➡️ 箭头绑定及带标签的箭头。
  • 🔙 撤销 / 重做。
  • 🔍 支持缩放和平移。

Excalidraw.com

托管在 excalidraw.com 的应用是 Excalidraw 能力的极简展示。其源代码也包含在本仓库中，应用主要特性包括：

  • 📡 支持 PWA（可离线使用）。
  • 🤼 实时协作。
  • 🔒 端到端加密。
  • 💾 本地优先（自动保存至浏览器）。
  • 🔗 可分享链接（导出为只读链接，可与他人共享）。

我们将在未来将这些功能作为即插即用插件提供给 npm 包。

快速开始

注意： 以下说明适用于将 Excalidraw 集成到你自己的应用中时安装 Excalidraw npm 包。如需本地开发运行仓库，请参考我们的开发指南。

使用 npm 或 yarn 安装该包。

npm install react react-dom @excalidraw/excalidraw

或

yarn add react react-dom @excalidraw/excalidraw
