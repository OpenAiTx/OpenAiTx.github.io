Возможности

Редактор Excalidraw (npm-пакет) поддерживает:

  • 💯 Бесплатно и с открытым исходным кодом.
  • 🎨 Бесконечный белыйboard на основе canvas.
  • ✍️ Стиль, имитирующий рисование от руки.
  • 🌓 Темный режим.
  • 🏗️ Настраиваемость.
  • 📷 Поддержка изображений.
  • 😀 Поддержка библиотек фигур.
  • 🌐 Поддержка локализации (i18n).
  • 🖼️ Экспорт в PNG, SVG и буфер обмена.
  • 💾 Открытый формат — экспорт рисунков в формате .excalidraw (json-файл).
  • ⚒️ Широкий набор инструментов — прямоугольник, круг, ромб, стрелка, линия, произвольное рисование, ластик...
  • ➡️ Привязка стрелок и стрелки с подписями.
  • 🔙 Отмена / повтор.
  • 🔍 Масштабирование и панорамирование.

Excalidraw.com

Приложение, размещенное на excalidraw.com, — это минимальная демонстрация того, что можно создать с помощью Excalidraw. Его исходный код также входит в этот репозиторий, и приложение поддерживает:

  • 📡 Поддержка PWA (работает офлайн).
  • 🤼 Совместная работа в реальном времени.
  • 🔒 Сквозное шифрование.
  • 💾 Локальная работа (автосохранение в браузере).
  • 🔗 Ссылки для общего доступа (экспорт в ссылку только для чтения, которую можно отправить другим).

В будущем мы добавим эти функции как плагины для npm-пакета.

Быстрый старт

Примечание: следующие инструкции предназначены для установки npm-пакета Excalidraw при интеграции Excalidraw в ваше собственное приложение. Для запуска репозитория локально в целях разработки обратитесь к нашему Руководству по разработке.

Используйте npm или yarn для установки пакета.

npm install react react-dom @excalidraw/excalidraw

или

yarn add react react-dom @excalidraw/excalidraw

Посмотрите нашу документацию для получения подробностей!
