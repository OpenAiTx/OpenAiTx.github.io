Özellikler

Excalidraw editörü (npm paketi) şunları destekler:

  • 💯 Ücretsiz & açık kaynak kodlu.
  • 🎨 Sonsuz, tuval tabanlı beyaz tahta.
  • ✍️ El çizimi benzeri stil.
  • 🌓 Karanlık mod.
  • 🏗️ Özelleştirilebilir.
  • 📷 Resim desteği.
  • 😀 Şekil kütüphaneleri desteği.
  • 🌐 Yerelleştirme (i18n) desteği.
  • 🖼️ PNG, SVG ve panoya dışa aktarma.
  • 💾 Açık format - çizimleri .excalidraw json dosyası olarak dışa aktarın.
  • ⚒️ Geniş araç yelpazesi - dikdörtgen, daire, elmas, ok, çizgi, serbest çizim, silgi...
  • ➡️ Ok bağlama & etiketli oklar.
  • 🔙 Geri al / Yinele.
  • 🔍 Yakınlaştırma ve kaydırma desteği.

Excalidraw.com

excalidraw.com'da barındırılan uygulama, Excalidraw ile neler oluşturabileceğinizi gösteren minimal bir vitrindir. Kaynak kodu da bu depoya dahildir ve uygulamanın özellikleri şunlardır:

  • 📡 PWA desteği (çevrimdışı çalışır).
  • 🤼 Gerçek zamanlı işbirliği.
  • 🔒 Uçtan uca şifreleme.
  • 💾 Öncelikli olarak yerelde çalışma desteği (tarayıcıya otomatik kaydetme).
  • 🔗 Paylaşılabilir bağlantılar (başkalarıyla paylaşabileceğiniz salt okunur bağlantı olarak dışa aktarım).

Bu özellikleri ileride npm paketi için eklenti olarak sunacağız.

Hızlı Başlangıç

Not: Aşağıdaki talimatlar, Excalidraw'ı kendi uygulamanıza entegre ederken Excalidraw npm paketini yüklemek içindir. Depoyu yerel olarak geliştirme amaçlı çalıştırmak için Geliştirme Rehberimize bakınız.

Paketi yüklemek için npm veya yarn kullanın.

npm install react react-dom @excalidraw/excalidraw

veya

yarn add react react-dom @excalidraw/excalidraw
