🛡️ Pack de démarrage pour l'ingénierie de détection

Une liste d'opinions des ressources essentielles pour les ingénieurs en détection en herbe.

Le but de ce pack de démarrage est de fournir une sélection organisée de ressources pour vous aider à débuter en ingénierie de détection sans vous sentir submergé. Cette liste est basée sur une expérience personnelle avec diverses technologies de détection. J'espère que cela vous sera utile ! 🚀

🔍 Comprendre les techniques des attaquants

Voir comment les attaquants atteignent leurs objectifs.
  • MITRE ATT&CK - La base de connaissances n°1 des tactiques et techniques des adversaires.
  • Top 10 ATT&CK Techniques - Une page personnalisable pour afficher les techniques ATT&CK les plus courantes.

📜 Faire Connaissance avec les Règles de Détection

Dépôts d’exemples montrant comment les détections sont structurées et appliquées.

  • Sigma - Le format générique de signature de détection.
  • Splunk Detection Rules - Une collection de règles de détection pour Splunk.
  • Elastic Detection Rules - Une collection de règles de détection pour Elastic.
  • Detection Studio - Convertir des règles Sigma en autres syntaxes de règles de détection.

🛠️ Essayer Par Vous-Même

Outils à manipuler qui sont soit open source, soit disposent d’un élément gratuit.

#### 🔒 Détection Endpoint

  • Aurora - Un agent capable d’exécuter des règles Sigma. Chargez vos règles Sigma et créez des alertes à partir de vos journaux d’événements.
  • Velociraptor - Un outil d’investigation numérique et de réponse aux incidents qui améliore votre visibilité sur vos endpoints.
  • Falco - Un outil de sécurité runtime natif cloud pour détecter les menaces dans les conteneurs.
  • Sysmon - Un simple moniteur système Windows.
  • Osquery - Un framework d’instrumentation du système d’exploitation.

#### 🌐 Détection Réseau

  • Suricata - Règles de détection conçues pour interroger le trafic réseau à la recherche d’activités suspectes.

#### 📂 Détection de Contenu de Fichiers

  • YARA - Règles de détection pour identifier et classifier des échantillons de malwares.

#### 📊 SIEM (Gestion des Informations et Événements de Sécurité)

  • Elastic Stack (ELK) - Une suite d'outils pour la recherche, la journalisation et l'analyse.
  • Wazuh - Une plateforme de surveillance de sécurité open-source.

#### ⚙️ SOAR (Orchestration, Automatisation et Réponse en Sécurité)

  • Tines - Une plateforme d'automatisation sans code pour les équipes de sécurité. Idéale pour automatiser rapidement tout type de tâches. Dispose d'un niveau gratuit.

#### 🎭 Émulation d'Adversaire

  • Adversary Emulation Library - Une bibliothèque de plans d'émulation d'adversaires.
  • MITRE Caldera - Une plateforme automatisée d'émulation d'adversaires.
  • Stratus Red Team - Un outil d'émulation d'adversaires dans le cloud.
  • Atomic Red Team - Une bibliothèque de tests simples d'émulation d'adversaires.
  • TTPForge - Un outil pour créer et gérer des TTPs.

📚 Concepts Utiles

  • Detection Engineering Behavior Maturity Model - une approche structurée pour que les équipes de sécurité fassent évoluer de manière constante leurs processus et comportements, par Elastic.
  • Alerting Detection Strategy (ADS) Framework - Un cadre simple pour construire des stratégies de détection par Palantir.
  • Summiting the Pyramid - S'appuyant sur la 'pyramide de la douleur', ce travail définit ce que signifie réellement disposer d'une détection robuste.
  • Capability Abstraction - Un de mes articles préférés - 'Capability Abstraction' de SpecterOps. Explore des concepts similaires au projet 'summiting the pyramid' ci-dessus.
