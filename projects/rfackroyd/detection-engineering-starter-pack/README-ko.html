🛡️ 탐지 엔지니어링 스타터 팩

탐지 엔지니어를 희망하는 분들을 위한 필수 리소스 목록입니다.

이 스타터 팩의 목표는 압도당하지 않고 탐지 엔지니어링을 시작하는 데 도움이 되는 엄선된 리소스를 제공하는 것입니다. 이 목록은 다양한 탐지 기술에 대한 개인적인 경험을 바탕으로 작성되었습니다. 도움이 되길 바랍니다! 🚀

🔍 공격자 기술 이해하기

공격자가 목표를 달성하는 방식을 살펴보세요.
  • MITRE ATT&CK - 적대자의 전술과 기술에 대한 최고의 지식 기반입니다.
  • Top 10 ATT&CK Techniques - 가장 일반적인 ATT&CK 기술을 사용자 지정하여 표시할 수 있는 페이지입니다.
  • Hacking the Cloud - 클라우드 중심 공격 기술을 이해하기 위한 자료 모음입니다.
  • The DFIR Report - 방어자의 관점에서 분석하고 설명한 실제 사건 사례입니다. 개인적으로 좋아하는 자료입니다.

📜 탐지 규칙 알아보기

탐지 규칙이 어떻게 구성되고 적용되는지 보여주는 예제 저장소들입니다.

  • Sigma - 일반적인 탐지 서명 형식입니다.
  • Splunk Detection Rules - Splunk용 탐지 규칙 모음입니다.
  • Elastic Detection Rules - Elastic용 탐지 규칙 모음입니다.
  • Detection Studio - Sigma 규칙을 다른 탐지 규칙 문법으로 변환합니다.

🛠️ 직접 사용해보기

오픈 소스이거나 무료 티어가 포함된 도구들입니다.

#### 🔒 엔드포인트 탐지

  • Aurora - Sigma 규칙을 실행할 수 있는 에이전트입니다. Sigma 규칙을 로드하고 이벤트 로그에서 알림을 생성할 수 있습니다.
  • Velociraptor - 엔드포인트 가시성을 높여주는 디지털 포렌식 및 사고 대응 도구입니다.
  • Falco - 컨테이너 내 위협을 탐지하는 클라우드 네이티브 런타임 보안 도구입니다.
  • Sysmon - 간단한 윈도우 시스템 모니터입니다.
  • Osquery - 운영 체제 계측 프레임워크입니다.

#### 🌐 네트워크 탐지

  • Suricata - 네트워크 트래픽에서 의심스러운 활동을 탐지하기 위한 규칙들입니다.

#### 📂 파일 내용 탐지

  • YARA - 악성코드 샘플을 식별하고 분류하기 위한 탐지 규칙입니다.

#### 📊 SIEM (보안 정보 및 이벤트 관리)

  • Elastic Stack (ELK) - 검색, 로깅, 분석을 위한 도구 모음입니다.
  • Wazuh - 오픈 소스 보안 모니터링 플랫폼입니다.

#### ⚙️ SOAR (보안 오케스트레이션 자동화 및 대응)

  • Tines - 보안 팀을 위한 코드 없는 자동화 플랫폼입니다. 빠르게 무엇이든 자동화하기에 적합합니다. 무료 등급이 있습니다.

#### 🎭 적대자 에뮬레이션

  • Adversary Emulation Library - 적대자 에뮬레이션 계획 라이브러리입니다.
  • MITRE Caldera - 자동화된 적대자 에뮬레이션 플랫폼입니다.
  • Stratus Red Team - 클라우드에서 적대자 에뮬레이션을 위한 도구입니다.
  • Atomic Red Team - 간단한 적대자 에뮬레이션 테스트 라이브러리입니다.
  • TTPForge - TTP(전술, 기술 및 절차)를 생성하고 관리하는 도구입니다.

📚 유용한 개념

  • Detection Engineering Behavior Maturity Model - Elastic에서 제공하는 보안 팀이 프로세스와 행동을 일관되게 성숙시키기 위한 구조화된 접근법입니다.
  • Alerting Detection Strategy (ADS) Framework - Palantir에서 제공하는 탐지 전략을 구축하기 위한 간단한 프레임워크입니다.
  • Summiting the Pyramid - '고통의 피라미드'를 기반으로 견고한 탐지가 무엇을 의미하는지 정의하는 작업입니다.
  • Capability Abstraction - 제가 좋아하는 글 중 하나인 SpecterOps의 'Capability Abstraction'입니다. 위의 'Summiting the Pyramid' 프로젝트와 유사한 개념을 탐구합니다.
