🔍 攻撃者の手法を理解する

攻撃者がどのように目的を達成するかを見てみましょう。

  • Top 10 ATT&CK Techniques - 最も一般的なATT&CK技術を表示するカスタマイズ可能なページ。
  • Hacking the Cloud - クラウドに焦点を当てた攻撃技術を理解するためのリソース集。
  • The DFIR Report - 守備側の視点で分析・記述された実際のインシデント。個人的なお気に入り。

📜 検出ルールを知る

検出がどのように構成され適用されるかを示すリポジトリの例。

  • Sigma - 汎用の検出シグネチャ形式。
  • Splunk Detection Rules - Splunk用の検出ルール集。
  • Elastic Detection Rules - Elastic用の検出ルール集。
  • Detection Studio - Sigmaルールを他の検出ルール構文に変換。

🛠️ 自分で試す

オープンソースまたは無料プランがあるツール。

#### 🔒 エンドポイント検出

  • Aurora - Sigmaルールを実行可能なエージェント。Sigmaルールを読み込み、イベントログからアラートを作成。
  • Velociraptor - エンドポイントの可視性を高めるデジタルフォレンジックおよびインシデント対応ツール。
  • Falco - コンテナ内の脅威を検出するクラウドネイティブなランタイムセキュリティツール。
  • Sysmon - シンプルなWindowsシステムモニター。
  • Osquery - オペレーティングシステムの計測フレームワーク。

#### 🌐 ネットワーク検出

  • Suricata - ネットワークトラフィックを調査し疑わしい活動を検出するための検出ルール。

#### 📂 ファイル内容検出

  • YARA - マルウェアサンプルの識別・分類のための検出ルール。

#### 📊 SIEM (セキュリティ情報およびイベント管理)

  • Elastic Stack (ELK) - 検索、ログ記録、分析のためのツールスイート。
  • Wazuh - オープンソースのセキュリティ監視プラットフォーム。

#### ⚙️ SOAR（セキュリティオーケストレーション自動化と対応）

  • Tines - セキュリティチーム向けのノーコード自動化プラットフォーム。迅速な自動化に最適。無料プランあり。

#### 🎭 敵対者エミュレーション

  • Adversary Emulation Library - 敵対者エミュレーションプランのライブラリ。
  • MITRE Caldera - 自動敵対者エミュレーションプラットフォーム。
  • Stratus Red Team - クラウドでの敵対者エミュレーションツール。
  • Atomic Red Team - シンプルな敵対者エミュレーションテストのライブラリ。
  • TTPForge - TTPの作成と管理のためのツール。

📚 有用な概念

  • Detection Engineering Behavior Maturity Model - Elasticによる、セキュリティチームがプロセスと行動を一貫して成熟させるための構造化されたアプローチ。
  • Alerting Detection Strategy (ADS) Framework - Palantirによる検出戦略構築のためのシンプルなフレームワーク。
  • Summiting the Pyramid - 「苦痛のピラミッド」を基に、堅牢な検出の真意を定義した研究。
  • Capability Abstraction - 私のお気に入りの記事の一つ、SpecterOpsによる「Capability Abstraction」。上記「Summiting the Pyramid」と似た概念を探求。
