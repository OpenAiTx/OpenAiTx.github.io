Godot Engine

เอนจินเกม 2D และ 3D ข้ามแพลตฟอร์ม

Godot Engine คือเอนจินเกมข้ามแพลตฟอร์มที่อัดแน่นด้วยฟีเจอร์ สำหรับสร้างเกม 2D และ 3D จากอินเทอร์เฟซเดียวกัน มันมีชุดเครื่องมือ ที่จำเป็น อย่างครบถ้วน เพื่อให้ผู้ใช้สามารถมุ่งเน้นไปที่การสร้างเกมโดยไม่ต้องเสียเวลาในการสร้างเครื่องมือขึ้นมาใหม่ เกมสามารถส่งออกได้ด้วยคลิกเดียวไปยังหลายแพลตฟอร์ม รวมถึงแพลตฟอร์มเดสก์ท็อปหลัก (Linux, macOS, Windows), แพลตฟอร์มมือถือ (Android, iOS) รวมถึงแพลตฟอร์มบนเว็บ และ คอนโซล

ฟรี โอเพนซอร์ส และขับเคลื่อนโดยชุมชน

Godot เปิดให้ใช้งานได้ฟรีและเป็นโอเพนซอร์สอย่างสมบูรณ์ภายใต้ สัญญาอนุญาต MIT ที่มีความยืดหยุ่นสูง ไม่มีข้อผูกมัด ไม่มีค่าลิขสิทธิ์ ไม่มีอะไรทั้งสิ้น เกมที่ผู้ใช้สร้างขึ้นเป็นของผู้ใช้เองทั้งหมดแม้แต่โค้ดส่วนสุดท้ายของเอนจิน การพัฒนา Godot เป็นอิสระอย่างแท้จริงและขับเคลื่อนโดยชุมชน เพื่อเปิดโอกาสให้ผู้ใช้มีส่วนร่วมในการกำหนดทิศทางของเอนจินให้ตรงกับความต้องการของตนเอง โดยได้รับการสนับสนุนจาก Godot Foundation องค์กรไม่แสวงหากำไร

ก่อนที่จะเปิดซอร์สใน กุมภาพันธ์ 2014 Godot ได้รับการพัฒนาโดย Juan Linietsky และ Ariel Manzur (ทั้งสองยังดูแลโปรเจกต์อยู่) มาหลายปีในฐานะเอนจินภายในบริษัท ซึ่งถูกนำไปใช้สร้างเกมตามสั่งหลายรายการ

การดาวน์โหลดเอนจิน

ไฟล์ไบนารี

ไฟล์ไบนารีอย่างเป็นทางการสำหรับ Godot Editor และ export templates สามารถหาได้ที่ เว็บไซต์ Godot

การคอมไพล์จากซอร์ส

ดูเอกสารทางการ สำหรับคำแนะนำในการคอมไพล์บนทุกแพลตฟอร์มที่รองรับ

ชุมชนและการมีส่วนร่วม

Godot ไม่ใช่แค่เอนจินเกม แต่ยังเป็นชุมชนของผู้ใช้และนักพัฒนาเอนจินที่เติบโตขึ้นเรื่อย ๆ ช่องทางหลักของชุมชนถูกระบุไว้ที่ หน้าแรก

วิธีที่ดีที่สุดในการติดต่อกับทีมนักพัฒนาเอนจินหลักคือการเข้าร่วม Godot Contributors Chat

หากต้องการเริ่มต้นมีส่วนร่วมกับโปรเจกต์นี้ ดูได้ที่ คู่มือการมีส่วนร่วม เอกสารนี้ยังรวมถึงแนวทางการรายงานบั๊กด้วย
