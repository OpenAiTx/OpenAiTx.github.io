storybook

⭐ 0 stars Thai by storybookjs

สร้างคอมโพเนนต์ UI ที่แข็งแกร่งได้รวดเร็วยิ่งขึ้น



Storybook คือเวิร์กช็อปสำหรับ Frontend ที่ใช้สร้างคอมโพเนนต์ UI และเพจแบบแยกส่วน ทีมงานนับพันใช้เพื่อพัฒนา UI, ทดสอบ และจัดทำเอกสาร ค้นหาข้อมูลเพิ่มเติมได้ที่ https://storybook.js.org!

เริ่มต้นใช้งาน

เยี่ยมชม เว็บไซต์ของ Storybook เพื่อเรียนรู้เพิ่มเติมและเริ่มต้นใช้งาน

เอกสารประกอบ

สามารถดูเอกสารได้ที่ เว็บไซต์เอกสารของ Storybook

ตัวอย่าง

ดู สารานุกรมคอมโพเนนต์ เพื่อดูวิธีที่ทีมงานชั้นนำใช้ Storybook

ใช้ storybook.new เพื่อสร้างโปรเจกต์ตัวอย่างอย่างรวดเร็วใน Stackblitz

Storybook มาพร้อมกับ แอดออน มากมายสำหรับออกแบบคอมโพเนนต์, จัดทำเอกสาร, ทดสอบ, สร้างการโต้ตอบ ฯลฯ API ของ Storybook ช่วยให้สามารถปรับแต่งและขยายการทำงานได้หลากหลายรูปแบบ และยังรองรับ React Native, Android, iOS และ Flutter สำหรับการพัฒนาโมบายล์อีกด้วย

ดู ตารางรองรับ Addon / Framework

เพื่อปรับปรุงประสบการณ์ของคุณอย่างต่อเนื่อง เราอาจเลิกใช้งานหรือถอดแอดออนบางรายการเพื่อสนับสนุนเครื่องมือใหม่และดีกว่า

หากคุณใช้ info/notes เราขอแนะนำให้เปลี่ยนไปใช้ docs แทน และ นี่คือคู่มือ ที่ช่วยคุณได้

หากคุณใช้ contexts เราขอแนะนำให้เปลี่ยนไปใช้ toolbars และ นี่คือคู่มือ ที่ช่วยคุณได้

หากคุณใช้ addon-storyshots เราขอแนะนำให้เปลี่ยนไปใช้ test-runner ของ Storybook และ นี่คือคู่มือ ที่ช่วยคุณได้

สคริปต์สำหรับนักพัฒนา

Storybook จัดระเบียบแบบ monorepo สคริปต์ที่มีประโยชน์ ได้แก่:

#### yarn start

รัน Storybook แซนด์บ็อกซ์พร้อม test stories

#### yarn task

เช่นเดียวกับด้านบน แต่ให้ตัวเลือกปรับแต่งแซนด์บ็อกซ์ (เช่น เลือกเฟรมเวิร์กอื่น)

#### yarn lint

ตรวจสอบ boolean ว่าโค้ดเป็นไปตามกฎ linting หรือไม่ - ใช้ remark & eslint

  • yarn lint:js - ตรวจสอบ js
  • yarn lint:md - ตรวจสอบ markdown + ตัวอย่างโค้ด
  • yarn lint:js --fix - แก้ไข js อัตโนมัติ

#### yarn test

ตรวจสอบ boolean ว่ายูนิตเทสต์ทั้งหมดผ่าน - ใช้ jest

  • yarn run test --core --watch - รัน core tests ในโหมด watch
