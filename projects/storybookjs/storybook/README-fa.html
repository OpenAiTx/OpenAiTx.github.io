شروع به کار

برای کسب اطلاعات بیشتر و شروع کار با Storybook به وب‌سایت Storybook مراجعه کنید.

مستندات

مستندات را می‌توانید در سایت مستندات Storybook بیابید.

مثال‌ها

برای مشاهده نحوه استفاده تیم‌های پیشرو از Storybook به دایره‌المعارف کامپوننت‌ها مراجعه کنید.

از storybook.new استفاده کنید تا به سرعت یک پروژه نمونه در Stackblitz ایجاد نمایید.

Storybook با مجموعه‌ای از افزونه‌ها برای طراحی کامپوننت، مستندسازی، تست، تعاملی بودن و غیره ارائه می‌شود. API استوری‌بوک امکان پیکربندی و توسعه به روش‌های مختلف را فراهم می‌کند. حتی برای توسعه موبایل با React Native، اندروید، iOS و Flutter نیز گسترش یافته است.

اسکریپت‌های توسعه

Storybook به صورت یک مونوریپو سازماندهی شده است. اسکریپت‌های مفید عبارتند از:

#### yarn start

اجرای یک Storybook الگوی sandbox با داستان‌های تستی

#### yarn task

مانند بالا، اما گزینه‌هایی برای سفارشی‌سازی sandbox (مثلاً انتخاب فریم‌ورک‌های دیگر) می‌دهد.

#### yarn lint

بررسی بولی انطباق کد با قوانین lint - از remark و eslint استفاده می‌کند.

  • yarn lint:js - بررسی کد جاوااسکریپت
  • yarn lint:md - بررسی markdown و نمونه‌های کد
  • yarn lint:js --fix - رفع خودکار مشکلات js

#### yarn test

بررسی بولی عبور همه تست‌های واحد - از jest استفاده می‌کند.

  • yarn run test --core --watch - اجرای تست‌های core در حالت watch
