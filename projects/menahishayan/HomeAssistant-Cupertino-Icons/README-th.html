ไอคอน Home Assistant Cupertino

ไอคอน Apple SF Symbols สำหรับ Home Assistant! (มากกว่า 3000 ไอคอน)

                       
                       

การติดตั้ง

HACS (แนะนำ)

  • แถบ HACS > เมนู > Custom Repository
  • วาง URL ของรีโปนี้
  • เลือก Integration ในดรอปดาวน์
  • ติดตั้ง cupertino-icons จาก HACS
  • รีสตาร์ท Home Assistant
  • Home Assistant > การตั้งค่า > การผสานรวม > เพิ่ม > Cupertino Icons > ติดตั้ง

ไอคอนควรพร้อมใช้งานใน Home Assistant แล้ว หากยังไม่แสดง ให้ลองรีเฟรชหน้า ใช้โหมดส่วนตัว หรือรีสตาร์ท Home Assistant

ติดตั้งด้วยตนเอง

  • คัดลอก custom_components/cupertino ไปยังโฟลเดอร์ custom_components ของคุณ
  • รีสตาร์ท Home Assistant
  • Home Assistant > การตั้งค่า > การผสานรวม > เพิ่ม > Cupertino Icons > ติดตั้ง
ไอคอนควรพร้อมใช้งานใน Home Assistant แล้ว หากยังไม่แสดง ให้ลองรีเฟรชหน้า ใช้โหมดส่วนตัว หรือรีสตาร์ท Home Assistant

การใช้งาน

คำนำหน้า: ios ตัวอย่าง: ios:lightbulb-fill | ios:cloud-moon-rain | ios:bed-double-fill

ไอคอนหลายสี! (v3.0+)

                       
                       

SF Symbols 3+ รองรับไอคอนแบบหลายสี ฟังก์ชันนี้ได้ถูกนำมาใช้ในคลังนี้เพื่อรองรับไอคอนหลายสีใน Home Assistant!

เพียงแค่เพิ่มคำต่อท้าย #color, #invert, #color-invert หรือ #fullcolor เพื่อเปลี่ยนสีของไอคอน! (เครดิตแก่ thomasloven สำหรับแรงบันดาลใจ)

ตัวอย่าง:

  • ios:gear-badge-xmark#color
  • ios:gear-badge-xmark#color-invert
  • ios:gear-badge-xmark#fullcolor

อ้างอิงไอคอน

อัปเดต v3.0

  • Home Assistant Cupertino Icons ได้อัปเดตเป็น SF Symbols v3.0 พร้อม 3167 ไอคอน! (ไม่รวมไอคอนที่ติดลิขสิทธิ์)
  • รองรับตัวเลือกไอคอนของ Home Assistant (2021.11.0+)

SF Symbols | Apple SF Symbols Reference | SFSymbols.com

คุณสามารถดาวน์โหลดแอป MacOS จาก Apple เพื่อใช้อ้างอิงได้สะดวกยิ่งขึ้น

หมายเหตุ: อักขระขีดเส้นใต้ (\_) และจุด (.) ต้องเปลี่ยนเป็นขีดกลาง (-)

การเปลี่ยนแปลงที่ส่งผลกระทบ

เริ่มต้นตั้งแต่ v3.0+ คลังนี้ได้เปลี่ยนจาก repo สำหรับ lovelace/frontend เป็น integration เต็มรูปแบบ หากคุณอัปเกรดจากเวอร์ชันเก่า:
  • HACS > ถอนการติดตั้ง "cupertino-icons"
  • HACS > เมนู > Custom Repository
  • ลบ repo นี้
  • ทำตามขั้นตอน Install ด้านบน

ข้อจำกัดทางกฎหมาย

ไอคอนบางรายการเป็นลิขสิทธิ์ของ Apple และไม่อนุญาตให้นำไปแก้ไข ไอคอนเหล่านี้จะไม่ถูกส่งออกและไม่มีอยู่ในคลังนี้ ดังนั้นจะไม่ถูกนำมาใช้
