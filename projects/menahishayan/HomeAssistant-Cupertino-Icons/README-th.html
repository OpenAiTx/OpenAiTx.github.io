Home Assistant Cupertino Icons (SF Symbols)

ไอคอน Apple SF Symbols สำหรับ Home Assistant! (มากกว่า 3000 ไอคอน)

                       
                       

การติดตั้ง

HACS (แนะนำ)

  • แถบ HACS > เมนู > Custom Repository
  • วาง URL ของ repo นี้
  • เลือก Integration ในดรอปดาวน์
  • ติดตั้ง cupertino-icons จาก HACS
  • รีสตาร์ท Home Assistant
  • Home Assistant > การตั้งค่า > การผสานรวม > เพิ่ม > Cupertino Icons > ติดตั้ง

ไอคอนควรใช้งานได้ใน Home Assistant แล้ว หากไม่แสดงผล ให้ลองรีเฟรชหน้าเว็บ ใช้โหมดส่วนตัว หรือรีสตาร์ท Home Assistant

คู่มือการติดตั้งด้วยตนเอง

  • คัดลอก custom_components/cupertino ไปยังโฟลเดอร์ custom_components ของคุณ
  • รีสตาร์ท Home Assistant
  • Home Assistant > การตั้งค่า > การผสานรวม > เพิ่ม > Cupertino Icons > ติดตั้ง

ไอคอนควรใช้งานได้ใน Home Assistant แล้ว หากไม่แสดงผล ให้ลองรีเฟรชหน้าเว็บ ใช้โหมดส่วนตัว หรือรีสตาร์ท Home Assistant

การใช้งาน

คำนำหน้า: ios ตัวอย่าง: ios:lightbulb-fill | ios:cloud-moon-rain | ios:bed-double-fill

ไอคอนหลายสี

                       
                       

SF Symbols 3+ รองรับไอคอนแบบหลายสี ฟังก์ชันนี้ได้ถูกนำมาใช้ในรีโปนี้เพื่อรองรับไอคอนหลายสีใน Home Assistant!

เพียงแค่เพิ่มคำต่อท้าย #color, #invert, #color-invert หรือ #fullcolor เพื่อเปลี่ยนสีไอคอน! (เครดิต thomasloven สำหรับแรงบันดาลใจ)

ตัวอย่าง:

  • ios:gear-badge-xmark#color
  • ios:gear-badge-xmark#color-invert
  • ios:gear-badge-xmark#fullcolor

อ้างอิงไอคอน

SF Symbols | Apple

คุณสามารถดาวน์โหลดแอป MacOS จาก Apple เพื่อดูอ้างอิงได้ดียิ่งขึ้น

หมายเหตุ: เครื่องหมายขีดล่าง(\_) และจุด(.) ต้องเปลี่ยนเป็นขีดกลาง(-)

การเปลี่ยนแปลงสำคัญ

เริ่มตั้งแต่ v3.0+ รีโปนี้ได้เปลี่ยนจากรีโป lovelace/frontend เป็นการรวมระบบเต็มรูปแบบ หากคุณอัปเกรดจากเวอร์ชันเก่า:
  • HACS > ถอนการติดตั้ง "cupertino-icons"
  • HACS > เมนู > Custom Repository
  • ลบรีโปนี้
  • ทำตามคำแนะนำการติดตั้งด้านบน

ข้อจำกัดทางกฎหมาย
