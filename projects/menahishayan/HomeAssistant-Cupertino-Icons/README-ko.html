Home Assistant Cupertino 아이콘

Home Assistant용 Apple SF Symbols 아이콘! (3000개 이상 아이콘)

                       
                       

설치

HACS (권장)

  • HACS 탭 > 메뉴 > Custom Repository
  • 이 저장소의 URL을 붙여넣기
  • 드롭다운에서 Integration 선택
  • HACS에서 cupertino-icons 설치
  • Home Assistant 재시작
  • Home Assistant > 설정 > 통합 > 추가 > Cupertino Icons > 설치

이제 아이콘을 Home Assistant에서 사용할 수 있습니다. 만약 표시되지 않으면 페이지 새로고침, 시크릿 브라우징 또는 Home Assistant 재시작을 시도하세요.

수동 설치

  • custom_components/cupertino를 custom_components 폴더에 복사
  • Home Assistant 재시작
  • Home Assistant > 설정 > 통합 > 추가 > Cupertino Icons > 설치

사용법

접두사: ios 예시: ios:lightbulb-fill | ios:cloud-moon-rain | ios:bed-double-fill

다중 색상 아이콘! (v3.0+)

                       
                       

SF Symbols 3+는 다중 색상 아이콘을 지원합니다. 이 기능이 이 저장소에 이식되어 Home Assistant에서도 다중 색상 아이콘을 사용할 수 있습니다!

아이콘 색상을 변경하려면 Suffix로 #color, #invert, #color-invert 또는 #fullcolor를 추가하세요! (영감 제공: thomasloven)

예시:

  • ios:gear-badge-xmark#color
  • ios:gear-badge-xmark#color-invert
  • ios:gear-badge-xmark#fullcolor

아이콘 참조

v3.0 업데이트

  • Home Assistant Cupertino Icons가 이제 SF Symbols v3.0으로 업데이트되어 3167개의 아이콘을 제공합니다! (저작권 아이콘 제외)
  • Home Assistant의 아이콘 선택기(2021.11.0+)를 지원합니다

SF Symbols | Apple SF Symbols Reference | SFSymbols.com

더 나은 참조를 위해 Apple에서 MacOS 앱을 다운로드할 수 있습니다.

참고: 언더스코어(\_)와 점(.)은 하이픈(-)으로 대체되어야 합니다.

주요 변경 사항

v3.0+부터 이 저장소는 lovelace/프론트엔드 저장소에서 완전한 통합 형태로 변경되었습니다. 이전 버전에서 업그레이드하는 경우:
  • HACS > "cupertino-icons" 제거
  • HACS > 메뉴 > Custom Repository
  • 이 저장소 삭제
  • 위의 설치 지침을 따르세요

법적 고지

특정 아이콘은 Apple의 저작권에 의해 보호되며, 수정하여 사용할 수 없습니다. 이러한 아이콘은 내보내지 않았으며 이 저장소에 포함되어 있지 않습니다. 또한 포함되지 않을 예정입니다.
