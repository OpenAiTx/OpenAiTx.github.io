হোম এছিষ্টেণ্ট কুপৰ্টিনো আইকনছ (SF Symbols)

হোম এছিষ্টেণ্টৰ বাবে এপল SF Symbols আইকনছ! (৩০০০+ আইকনছ)

                       
                       

ইনষ্টল কৰক

HACS (পৰামৰ্শ দিয়া)

  • HACS টেব > মেনু > Custom Repository
  • এই ৰেপ'ৰ URLটো পেষ্ট কৰক
  • ড্ৰপডাউনত Integration বাছনি কৰক
  • HACS-ৰ পৰা cupertino-icons ইনষ্টল কৰক
  • হোম এছিষ্টেন্ট পুনৰ আৰম্ভ কৰক
  • হোম এছিষ্টেন্ট > ছেটিংছ > ইন্টেগ্ৰেছনছ > যোগ কৰক > Cupertino Icons > ইনষ্টল কৰক

আইকনসমূহ এতিয়া Home Assistant-ত ব্যৱহাৰ কৰিব পাৰি। যদি এইটো দেখা নাযায়, পৃষ্ঠাটো ৰিফ্ৰেশ কৰক, ব্যক্তিগত ব্রাউজিং কৰক অথবা Home Assistant পুনৰ আৰম্ভ কৰক।

ম্যানুৱেল

  • custom_components/cupertino-ক আপোনাৰ custom_components ফোল্ডাৰলৈ কপি কৰক
  • Home Assistant পুনৰ আৰম্ভ কৰক
  • Home Assistant > Settings > Integrations > Add > Cupertino Icons > Install

বহু-ৰং আইকন

                       
                       

SF Symbols 3+ বহু-ৰঙা আইকনসমূহক সমৰ্থন কৰে। এই ফিচাৰটো এই ৰিপ'ত যোগ কৰা হৈছে যাতে Home Assistant-ত বহু-ৰঙা আইকনসমূহ ব্যৱহাৰ কৰিব পৰা যায়!

আইকনৰ ৰং সলনি কৰিবলৈ suffix #color, #invert, #color-invert অথবা #fullcolor যোগ কৰক! (প্ৰেৰণা thomasloven ক্ৰেডিট)

  • ios:gear-badge-xmark#color
  • ios:gear-badge-xmark#color-invert
