Intel GPU 공식 커뮤니티 이슈 트래커입니다.

Windows Intel 드라이버 전용이며, Linux Intel mesa 드라이버는 mesa gitlab을 이용하세요.

규칙

  • 사용자/Intel 직원에게 불쾌감을 주는 댓글을 피하세요
  • 일부 수정은 필요한 작업량에 따라 시간이 걸릴 수 있으니 인내심을 가지세요
