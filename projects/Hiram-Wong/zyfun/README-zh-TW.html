軟體介紹

zyfun是一款免費、極簡、全能的跨平台影音管家，一站式管理 T1-T4 全類型資源，內建多核心播放器隨心切換。獨家老板鍵，一鍵隱身摸魚無憂。用觀影發現世界！

🌴 注意事項

  • 僅於此 Github 發布，請勿上當受騙；請各管理者不要宣傳及引流本軟體。
  • 強烈倡導合法觀影，本軟體僅作為播放工具，不涉及資源存儲或分發。
  • 僅供個人學習交流之用，24小時內請自覺卸載，勿作商業用途。
  • 前身來源於 ZY-Player，圖示由 @fourbeauty 設計。
  • 在開始使用前，請務必詳讀並同意用戶協議，確保遵守相關規定。

💻 跨平台說明

  • MacOS：
  • 資料庫/檔案/插件/日誌 路徑：~/Library/Application Support/zyfun/
  • 執行檔 路徑：~/.zy/bin/
  • Linux：
  • 資料庫/檔案/插件/日誌 路徑：~/.config/zyfun/
  • 執行檔 路徑：~/.zy/bin/
  • Windows：
  • 資料庫/檔案/插件/日誌 路徑：%USERPROFILE%\AppData\Roaming\zyfun\
  • 執行檔 路徑：%USERPROFILE%\.zy\bin\

展開查看安裝問題

#### MacOS 安裝後打開提示「檔案已損壞」

sudo spctl --master-disable
sudo xattr -cr /Applications/zyfun.app

#### Linux Appimage 執行失敗

以 root 身份執行且未加上 --no-sandbox 參數是不被支援的

./zyfun.AppImage --no-sandbox

dlopen()：載入 libfuse.so.2 時發生錯誤

sudo apt-get install libfuse2

因初始化期間發生錯誤，正在結束 GPU 程序

xhost +

#### Linux deb包安裝失敗

未安裝套件 libnss3-1d

sudo apt-get install libnss3-1d

套件 libxss1 未安裝

sudo apt-get install libxss1

#### Linux rpm套件安裝失敗

依賴性檢查失敗，需要 libXScrnSaver

sudo yum install epel-release
sudo yum install libXScrnSaver

依賴性檢查失敗，需要 xdg-utils

sudo yum install xdg-utils
