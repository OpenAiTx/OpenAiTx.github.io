소개

vue-element-admin은(는) 관리 인터페이스를 위한 프로덕션 수준의 프론트엔드 솔루션입니다. vue를 기반으로 하며, UI 툴킷으로 element-ui를 사용합니다.

vue-element-admin은(는) vue의 최신 개발 스택을 기반으로 하며, 내장 i18n 솔루션, 엔터프라이즈 애플리케이션을 위한 전형적인 템플릿, 다양한 뛰어난 기능들을 제공합니다. 대규모 및 복잡한 싱글 페이지 애플리케이션(SPA) 구축을 도와줍니다. 여러분의 요구가 무엇이든, 이 프로젝트가 도움이 될 것이라 믿습니다.

v4.1.0+ 버전 이후로 기본 master 브랜치는 i18n을 지원하지 않습니다. i18n 브랜치를 사용해 주세요. master 브랜치와 동기화됩니다.

현재 버전은 vue-cli 기반의 v4.0+입니다. 문제가 있다면 issue를 남겨주세요. 구버전 사용을 원하면 tag/3.11.0 브랜치로 전환하세요. 해당 버전은 vue-cli에 의존하지 않습니다.

이 프로젝트는 구버전 브라우저(예: IE)를 지원하지 않습니다. 필요 시 polyfill을 직접 추가하세요.

준비 사항

Node.js와 git을 로컬에 설치해야 합니다. 본 프로젝트는 ES2015+, vue, vuex, vue-router, vue-cli, axios 및 element-ui를 기반으로 하며, 모든 요청 데이터는 Mock.js로 시뮬레이션합니다. 이런 기술을 미리 이해하고 학습해두면 프로젝트 사용에 큰 도움이 됩니다.

시작하기

# 프로젝트 클론
git clone https://github.com/PanJiaChen/vue-element-admin.git
프로젝트 디렉토리로 이동

cd vue-element-admin
의존성 설치

npm install
개발 모드 실행

npm run dev

자동으로 http://localhost:9527 이 열립니다.

빌드

# 테스트 환경용 빌드
npm run build:stage
프로덕션 환경용 빌드

npm run build:prod

고급 기능

# 배포 환경 미리보기
npm run preview
배포 환경 미리보기 + 정적 리소스 분석

npm run preview -- --report
코드 포맷 검사

npm run lint
코드 포맷 검사 및 자동 수정

npm run lint -- --fix

자세한 내용은 문서를 참고하세요.
