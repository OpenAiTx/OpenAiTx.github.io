Whispering

การถอดเสียงอย่างไร้รอยต่อ ขับเคลื่อนโดย Whisper Model ของ OpenAI

เกี่ยวกับ

Whispering เป็นแอปพลิเคชันถอดเสียงแบบโอเพ่นซอร์สที่ให้ฟังก์ชันแปลงเสียงเป็นข้อความได้ทั่วโลก พร้อมตัวเลือก เช่น คีย์ลัดและการคัดลอก-วางอัตโนมัติ เพื่อให้การถอดเสียงเป็นไปอย่างไร้รอยต่อที่สุด

เบื้องหลังขับเคลื่อนด้วยโมเดล Whisper ของ OpenAI ซึ่งมีความแม่นยำมากกว่าฟีเจอร์ถอดเสียงที่ติดมากับระบบปฏิบัติการอย่างมีนัยสำคัญ

หมายเหตุสำคัญ: Whispering ถูกออกแบบมาเพื่อเป็นเครื่องมือถอดเสียง ไม่ใช่เครื่องมือบันทึกเสียง สำหรับการบันทึกเสียงที่ยาวและต้องการความน่าเชื่อถือสูง แนะนำให้ใช้แอปบันทึกเสียงเฉพาะทางในโทรศัพท์หรืออุปกรณ์ที่ใช้ API บันทึกเสียงของระบบ เพื่อให้ได้คุณภาพเสียงและความเสถียรสูงสุด

คุณสมบัติเด่น

  • ถอดเสียงได้ทั่วโลก: ใช้งานฟังก์ชันแปลงเสียงเป็นข้อความของ Whisper ได้ทุกที่ ด้วยคีย์ลัดหรือคลิกเพียงสองครั้ง
  • รองรับหลายแพลตฟอร์ม:
  • แอปเดสก์ท็อป: ให้ฟีเจอร์ถอดเสียงทั่วทั้งแอปพลิเคชัน
  • ส่วนขยายเบราว์เซอร์: ถอดเสียงได้จากเบราว์เซอร์โดยสื่อสารกับเว็บแอป
  • ผสานเข้ากับแชท: ส่วนขยายเบราว์เซอร์เพิ่มปุ่มบันทึกเสียงใน ChatGPT และ Claude บนเว็บไซต์ เพื่อให้สามารถป้อนเสียงและถอดเสียงได้โดยตรงในอินเทอร์เฟซแชท
  • จัดการถอดเสียง: ตรวจสอบและแก้ไขข้อความถอดเสียงในแอป Whispering เพื่อความถูกต้องและชัดเจน
  • คัดลอกคลิปบอร์ดอัตโนมัติ: เมื่อถอดเสียงเสร็จ ข้อความจะถูกคัดลอกไปยังคลิปบอร์ดโดยอัตโนมัติ พร้อมตัวเลือกวางอัตโนมัติ

ข้อมูลของฉันถูกจัดเก็บอย่างไร?

Whispering จะจัดเก็บข้อมูลให้มากที่สุดไว้ในเครื่องของคุณ ทั้งไฟล์เสียงและข้อความถอดเสียง วิธีนี้เพื่อความเป็นส่วนตัวและความปลอดภัยสูงสุดในการจัดเก็บข้อมูล สรุปการจัดการข้อมูลดังนี้:

  • จัดเก็บในเครื่อง: ไฟล์เสียงและข้อความถอดเสียงจะถูกเก็บใน IndexedDB ซึ่งใช้เก็บข้อมูลแบบ blob และข้อมูลทั้งหมด เช่น ข้อความและถอดเสียง

  • บริการถอดเสียง: ข้อมูลเดียวที่ถูกส่งออกนอกเครื่อง คือเสียงที่คุณเลือกใช้บริการถอดเสียงภายนอก โดยคุณมีตัวเลือกดังนี้:
  • บริการภายนอก เช่น OpenAI หรือ Groq
  • บริการถอดเสียงในเครื่อง เช่น faster-whisper-server ที่ทุกอย่างอยู่ในอุปกรณ์ของคุณ

  • ตั้งค่ากำหนดเองได้: คุณสามารถเลือกบริการถอดเสียงในเมนูตั้งค่าเพื่อความเป็นส่วนตัวสูงสุด

แอปเดสก์ท็อป

ดาวน์โหลดและติดตั้งแอป Whispering สำหรับเดสก์ท็อป ตามขั้นตอนของระบบปฏิบัติการที่คุณใช้:

#### Windows

  • ดาวน์โหลดไฟล์ติดตั้งที่ลงท้ายด้วย .msi จาก หน้า releases ล่าสุด
  • เปิดไฟล์ .msi ที่ดาวน์โหลดมาเพื่อเริ่มติดตั้ง
  • หากมีคำเตือนว่าไฟล์ไม่ปลอดภัย ให้คลิก More Info -> Run Anyway เพื่อดำเนินการต่อ
  • ทำตามคำแนะนำบนหน้าจอจนติดตั้งเสร็จสิ้น

#### macOS

สำหรับ macOS ให้ทำตามขั้นตอนดังนี้:

  • ดาวน์โหลดไฟล์ติดตั้ง:
  • ไปที่ หน้า releases ล่าสุด
  • เลือกไฟล์ที่เหมาะสม:
  • สำหรับ Apple Silicon: Whispering_x.x.x_aarch64.dmg
  • สำหรับ Intel: Whispering_x.x.x_x64.dmg

  • ติดตั้งแอปพลิเคชัน:
  • เปิดไฟล์ .dmg ที่ดาวน์โหลดมา
  • ลากแอป Whispering ไปยังโฟลเดอร์ Applications

  • เปิดใช้งาน Whispering:
  • เปิดแอป Whispering จากโฟลเดอร์ Applications
  • หากพบคำเตือนเกี่ยวกับนักพัฒนาไม่ได้รับการยืนยัน:
  • คลิก Cancel
  • คลิกขวาที่แอปใน Finder แล้วเลือก Open

  • แก้ไขปัญหา (สำหรับ Apple Silicon เท่านั้น):
หากพบข้อผิดพลาด "Whispering" is damaged and can't be opened:
  • เปิด Terminal
  • รันคำสั่งนี้:
``bash xattr -cr /Applications/Whispering.app `
  • ลองเปิดแอปอีกครั้ง

หลังจากทำตามขั้นตอนข้างต้น Whispering จะพร้อมใช้งานบน macOS ของคุณ

#### Linux

สำหรับ Linux มีหลายทางเลือกในการติดตั้ง เลือกวิธีที่เหมาะกับสภาพแวดล้อมของคุณ:

  • AppImage:

  • ดาวน์โหลดไฟล์ .AppImage จาก หน้า releases ล่าสุด
  • ทำให้ไฟล์สามารถรันได้:
`bash chmod +x whispering_x.x.x_amd64.AppImage `
  • รัน AppImage:
`bash ./whispering_x.x.x_amd64.AppImage `

  • แพ็กเกจ DEB (Debian/Ubuntu):
  • ดาวน์โหลดไฟล์ .deb จาก หน้า releases ล่าสุด
  • ติดตั้งแพ็กเกจด้วย dpkg:
`bash sudo dpkg -i whispering_x.x.x_amd64.deb `
  • แก้ไขปัญหา dependency:
`bash sudo apt-get install -f `

หลังจากติดตั้งแล้ว แอปจะพร้อมใช้งาน

ส่วนขยาย Chrome

หลังติดตั้งส่วนขยาย Chrome คุณจะพบไอคอน Whispering ในแถบส่วนขยายของ Chrome คลิกไอคอนนี้เพื่อเปิดส่วนขยาย จากนั้นคลิกปุ่มไมโครโฟนเพื่อบันทึกเสียงของคุณ และคลิกปุ่มสี่เหลี่ยมเมื่อเสร็จ ข้อความถอดเสียงจะแสดงในช่องข้อความด้านล่าง

หากต้องการใช้ฟีเจอร์ ChatGPT หรือ Claude ให้เข้าไปที่หน้าเว็บ ChatGPT หรือ Claude จะเห็นปุ่มบันทึกเสียงใหม่ในอินเทอร์เฟซแชท คลิกเพื่อเริ่มและหยุดบันทึก ข้อความถอดเสียงจะถูกแทรกอัตโนมัติในช่องป้อนข้อความของแชท

หากต้องการใช้คีย์ลัด กด Control + Shift + X หรือ Command + Shift + X (สามารถปรับเปลี่ยนได้ในหน้าตั้งค่าคีย์ลัดของ Chrome) เพื่อเริ่มบันทึกเสียงจากเว็บไซต์ใดก็ได้ ข้อความถอดเสียงจะถูกคัดลอกไปยังคลิปบอร์ดและวางในช่องป้อนข้อความปัจจุบันโดยอัตโนมัติ

ส่วนขยาย Chrome จะสื่อสารกับ whispering.bradenwong.com และจะพยายามเปิดแท็บเบื้องหลังโดยอัตโนมัติหากไม่พบแท็บอยู่แล้ว ข้อผิดพลาดส่วนใหญ่เกิดจากการสื่อสารนี้ล้มเหลว เช่น ในกรณีที่แท็บเข้าสู่โหมดพัก

แอปเดสก์ท็อป

หลังจากติดตั้งแอปเดสก์ท็อป Whispering ให้กด Control/Command + Shift + ; (สามารถปรับได้ในเมนูตั้งค่า) เพื่อเริ่มบันทึกเสียงจากที่ใดก็ได้บนเดสก์ท็อป ข้อความถอดเสียงจะถูกคัดลอกไปยังคลิปบอร์ดและวางโดยอัตโนมัติ ทั้งสองฟีเจอร์นี้สามารถเปิดหรือปิดได้ในเมนูตั้งค่า

เทคโนโลยีที่ใช้สร้าง

#### เว็บและเดสก์ท็อป

แอป Whispering ถูกสร้างขึ้นด้วยเทคโนโลยีและไลบรารีต่อไปนี้:

  • Svelte 5: ไลบรารี UI ที่เน้นความตอบสนอง
  • SvelteKit: สำหรับการกำหนดเส้นทางและสร้างเว็บไซต์แบบ static ใช้กับทั้งเว็บไซต์และ frontend ของแอป Tauri
  • Tauri: เฟรมเวิร์กแอปเดสก์ท็อป
  • Effect-TS: เพิ่มความสามารถแบบฟังก์ชันนัลและความปลอดภัยของชนิดข้อมูลสูงสุด โดยผลลัพธ์และข้อผิดพลาดรวมอยู่ในลายเซ็นของฟังก์ชัน
  • Svelte Sonner: ไลบรารี toast สำหรับ Svelte ใช้แสดงข้อผิดพลาดที่เกิดจาก Effect-TS ด้วยฟังก์ชัน renderAsToast
  • TanStack Table: สำหรับจัดการตารางข้อมูลทั้งหมด
  • IndexedDB: API ระดับต่ำสำหรับเก็บข้อมูลขนาดใหญ่ในเบราว์เซอร์ ซิงก์กับ Tanstack Table
  • ShadCN-Svelte: ไลบรารี UI ที่เลือกใช้
  • TailwindCSS: เฟรมเวิร์ก CSS แบบ utility-first สำหรับสร้าง UI อย่างรวดเร็ว
  • Turborepo: สำหรับจัดการ monorepo ให้ apps/app และ apps/extension ใช้โค้ดร่วมกัน เพื่อลดการซ้ำซ้อนของโค้ดและรักษาแหล่งข้อมูลเดียว
  • Rust: สำหรับขยายฟีเจอร์เดสก์ท็อป เช่น ใช้ crate enigo เพื่อจัดการการวางข้อความอัตโนมัติ
  • Vercel: โฮสต์ที่เหมาะสำหรับโปรเจกต์ส่วนตัวและรวมกับ Turborepo ได้ดี
  • Zapsplat.com: ไลบรารีเสียงเอฟเฟกต์ปลอดค่าลิขสิทธิ์

#### ส่วนขยาย

ส่วนขยาย Chrome ของ Whispering สร้างขึ้นโดยใช้:

  • Plasmo: เฟรมเวิร์กสำหรับสร้างส่วนขยาย Chrome เราใช้ relay flow เพื่อสื่อสารกับเว็บไซต์ Whispering
  • Effect-TS: เพิ่มความสามารถแบบฟังก์ชันนัลและความปลอดภัยของชนิดข้อมูลสูงสุด โดยผลลัพธ์และข้อผิดพลาดรวมอยู่ในลายเซ็นของฟังก์ชัน
  • React: ไลบรารี UI สำหรับส่วนขยาย Chrome เนื่องจาก Plasmo ยังไม่รองรับ Svelte 5
  • ShadCN: ไลบรารี UI สำหรับส่วนขยาย Chrome
  • TailwindCSS: เฟรมเวิร์ก CSS แบบ utility-first สำหรับการสร้างส่วนติดต่อผู้ใช้ที่กำหนดเองได้อย่างรวดเร็ว
  • Chrome API: Chrome extension API
  • Zapsplat.com: คลังเอฟเฟกต์เสียงที่ไม่มีลิขสิทธิ์

รัน Whispering ในโหมดพัฒนาแบบ Local

ในการตั้งค่าโปรเจกต์บนเครื่องของคุณ ให้ทำตามขั้นตอนดังนี้:

  • โคลน repository: git clone https://github.com/braden-w/whispering.git
  • เปลี่ยนไปยังไดเรกทอรีโปรเจกต์: cd whispering
  • ติดตั้ง dependencies ที่จำเป็น: pnpm i

เพื่อรันแอป Whispering บนเดสก์ท็อปและเว็บไซต์ในโหมดพัฒนา:

  • เปลี่ยนไปยังไดเรกทอรีแอป: cd apps/app
  • รันเซิร์ฟเวอร์พัฒนา: pnpm tauri dev

แอปเดสก์ท็อปควรเปิดขึ้นโดยอัตโนมัติสำหรับการพัฒนาแบบ local หากต้องการพัฒนาเว็บแอป ให้เปิดเบราว์เซอร์ของคุณและไปที่ http://localhost:5173

เพื่อรันส่วนขยาย Whispering Chrome ในโหมดพัฒนา:

  • เปลี่ยนไปยังไดเรกทอรี extension: cd apps/extension
  • รันเซิร์ฟเวอร์พัฒนา: pnpm dev --target=chrome-mv3

ในการพัฒนาส่วนขยาย Chrome ให้โหลดเข้า Chrome โดยไปที่ chrome://extensions เปิดโหมดนักพัฒนา แล้วโหลดโฟลเดอร์ apps/extension/build/{platform}-{manifest-version}-dev เป็นส่วนขยายแบบ unpacked

สร้าง Executable ด้วยตัวเอง

หากคุณกังวลเกี่ยวกับความน่าเชื่อถือของตัวติดตั้ง หรืออยากควบคุมได้มากขึ้น คุณสามารถสร้าง executable ด้วยตัวเองได้เสมอ ซึ่งอาจต้องตั้งค่ามากขึ้น แต่จะช่วยให้มั่นใจว่าคุณรันโค้ดที่คุณคาดหวังได้ นี่คือข้อดีของซอฟต์แวร์โอเพ่นซอร์ส!

Chrome

  • เปลี่ยนไปยังไดเรกทอรี extension: cd apps/extension
  • ติดตั้ง dependencies ที่จำเป็น: pnpm i
  • รัน Plasmo build: pnpm plasmo build --target=chrome-mv3
  • ผลลัพธ์จะอยู่ใน apps/extension/build/chrome-mv3-prod ซึ่งสามารถโหลดเข้า Chrome เป็นส่วนขยายแบบ unpacked ได้
  • หรือจะ build สำหรับ Chrome Web Store ด้วยคำสั่ง: pnpm plasmo build --target=chrome-mv3 --release

Firefox

  • เปลี่ยนไปยังไดเรกทอรี extension: cd apps/extension
  • ติดตั้ง dependencies ที่จำเป็น: pnpm i
  • รัน Plasmo build: pnpm plasmo build --target=firefox-mv3
  • ผลลัพธ์จะอยู่ใน apps/extension/build/firefox-mv3-prod ซึ่งสามารถโหลดเข้า Chrome เป็นส่วนขยายแบบ unpacked ได้
  • หรือจะ build สำหรับ Chrome Web Store ด้วยคำสั่ง: pnpm plasmo build --target=firefox-mv3 --release

Desktop

  • เปลี่ยนไปยังไดเรกทอรีแอป: cd apps/app
  • ติดตั้ง dependencies ที่จำเป็น: pnpm i
  • รัน Tauri Build: pnpm tauri build
  • คุณสามารถหา executable ได้ในไดเรกทอรี apps/app/target/release
