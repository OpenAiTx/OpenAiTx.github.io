ioBroker.vw-connect

测试::

ioBroker 的 vw-connect 适配器

适用于 VW We Connect、We Connect ID、We Charge、myAudi、Skoda Connect、Seat Connect 和 We Connect Go 的适配器

请将您的系统更新到 Node 10。

使用方法

使用 remote control 下的状态远程控制您的汽车。 普通刷新是从 VAG 云接收数据的轮询间隔 强制刷新用于非电动车，强制刷新次数由 VAG 限制，直到汽车再次启动。 行程数据仅对非电动车可用。

您可以在 .climater.settings.targetTemperature.content 设置空调温度
