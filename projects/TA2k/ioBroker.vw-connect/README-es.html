Adaptador vw-connect para ioBroker

Adaptador para VW We Connect, We Connect ID, We Charge, myAudi, Skoda Connect, Seat Connect y We Connect Go

Por favor actualice su sistema a Node 10.

Uso

Use el estado bajo control remoto para controlar su coche a distancia. El refresco normal es el intervalo de sondeo para recibir datos de la nube VAG El refresco forzado es para coches no eléctricos para forzar una actualización, este número está limitado por VAG hasta que el coche se encienda de nuevo. Los datos del viaje solo están disponibles para coches no eléctricos.

Puede configurar la temperatura de climatización en .climater.settings.targetTemperature.content
