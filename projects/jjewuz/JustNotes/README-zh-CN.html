JustNotes

JustNotes 是一个简单的安卓记笔记应用，就像之前成千上万个一样。应用使用 Room 数据库。

此仓库已归档，因为后续更新将变为商业版本。

访问 JustNotes Lite（完全开源版本）。

功能

  • [x] Material You 表现设计
  • [x] 文字格式化
  • [x] 导入和导出 .txt 文档
  • [x] 使用系统生物识别锁定应用
  • [x] 主屏幕小部件
  • [x] 使用 PIN 码锁定笔记
  • [x] 本地和云备份
  • [x] 更改文字颜色
  • [x] 笔记标签
  • [x] 提醒（带通知）
  • [x] 通知中的笔记
  • [x] 更改笔记背景（在 5.8.2 版本）
