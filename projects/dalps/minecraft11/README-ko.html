Minecraft11

Minecraft11은 마인크래프트 몹과 블록을 특징으로 하는 Windows 11용 재미있고 무료인 폴더 아이콘 모음입니다.

.svg 및 .ico 파일이 모두 제공됩니다. 파일 탐색기에서 아이콘을 사용하려면 ico 폴더 (MacOS 사용자는 icns 폴더)의 내용만 필요합니다.

아이콘 벡터 편집에 관심이 있다면 Inkscape를 사용하세요.

이 다소 우스꽝스러운 프로젝트는 davidvkimball의 아이콘과 Folder11 컬렉션에서 영감을 받았습니다.

멋지네요! 어떻게 설치하나요?

Windows에서는 다음과 같이 진행됩니다:

  • 저장소를 클론하거나 ZIP 아카이브로 다운로드하세요 (페이지 상단의 큰 밝은 녹색 "Code" 버튼 > Download ZIP)
  • ico 폴더를 안전한 위치에 압축 해제하세요, 예를 들어 내 홈 폴더나 바탕화면에. 이 위치를 꼭 기억하세요!
  • 사용자 지정할 폴더를 마우스 오른쪽 버튼으로 클릭하세요
  • 속성 > 사용자 지정 > 아이콘 변경 > 찾아보기 > 앞서 언급한 위치에서 .ico 파일 선택
  • 완료, 폴더 아이콘이 변경된 것을 확인할 수 있습니다!

직접 아이콘 만들기!

또한 아이콘의 기반으로 사용할 수 있는 픽셀 완벽한 빈 폴더 템플릿이 있습니다. 자유롭게 사용하시고, 기여도 환영합니다 🙂
