安装（Termux / Nethunter）

git clone https://github.com/TnYtCoder/InstaFans
cd InstaFans
chmod +x *
pip3 install -r requirements.txt

现在，您需要您的用户名和密码

然后通过运行替换credentials.ini中的用户名和密码 

nano credentials.ini
[Instagram Credentials]
username
password
只需替换您的凭据，且不要加"双引号

运行方式 → 

python3 instafans.py
