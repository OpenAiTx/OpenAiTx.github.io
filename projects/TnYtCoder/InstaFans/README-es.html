Instalación (Termux / Nethunter)

git clone https://github.com/TnYtCoder/InstaFans
cd InstaFans
chmod +x *
pip3 install -r requirements.txt

Ahora, Usted Necesita Su Nombre de Usuario y Contraseña

Luego Reemplace Su nombre de usuario y contraseña en credentials.ini ejecutando 

nano credentials.ini
[Instagram Credentials]
username
password
Solo reemplace sus credenciales sin las comillas dobles "`

Para ejecutar → 

python3 instafans.py
