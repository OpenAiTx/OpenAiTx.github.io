インストール（Termux / Nethunter）

git clone https://github.com/TnYtCoder/InstaFans
cd InstaFans
chmod +x *
pip3 install -r requirements.txt

今、あなたはユーザー名とパスワードが「必要」です

次に、credentials.iniのユーザー名とパスワードを置き換えて実行してください 

nano credentials.ini
[Instagram Credentials]
username
password
"を使わずに認証情報を置き換えてください

実行方法 → 

python3 instafans.py
