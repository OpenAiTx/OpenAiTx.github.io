Longhorn Bluesky

Un cliente de Bluesky de terceros con un tema de Windows Longhorn.

Próximas características (sin orden específico)

  • Reproducción de video
  • Soporte adecuado para alta DPI (sin errores, solo íconos de baja calidad)
  • Soporte para 2FA
  • Adjuntar texto alternativo
  • Edición de perfil
  • Configuraciones cambiables
  • Notificaciones, Chat, Feeds y Listas
  • Finalizar Explorar
  • Reportar publicaciones y usuarios
  • Creación y eliminación de cuentas
  • Soporte para GIFs de Tenor
  • Visualización como invitado
  • Finalizar Ayuda y Soporte
  • Temas para el cliente
  • ¿Temas para perfil?
  • ...y las características restantes de Bluesky

Compatibilidad

✔ Windows 11 / Server 2025 ✔ Windows 10 / Server (2016/2019/2022) ✔ Windows 8.x / Server 2012 ✔ Windows 7 / Server 2008 R2 🟡* Windows Vista / Windows Server 2008 (Con SP2) (No probado) (.NET 4.7.2 normalmente no es compatible con Windows Vista, pero creo que podría haber una forma de instalarlo.) ❓ Betas de Windows / Builds Insider (Podrían funcionar, pero no se garantiza compatibilidad) ❌ Windows Longhorn (Pre-reinicio) (Lo siento) ❌ Windows XP o anteriores
