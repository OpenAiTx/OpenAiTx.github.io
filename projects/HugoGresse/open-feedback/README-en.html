https://openfeedback.io/

Open Feedback is an open SaaS platform for organizers and speakers to gather user feedback. It connects to one of 3 data sources: OpenFeedback database, hoverboard-v2, or your own API, to retrieve all the sessions & speakers. Anonymous users can give constructive feedback on sessions, and admins can manage the voting form, comments, languages, sessions and speakers directly within the GUI. Organizations also have centralized user management and a common voting form.

Demo: https://openfeedback.io/eaJnyMXD3oNfhrrnBYDT

Features

  • Generate unique & public read URL for an event
  • List all sessions, search by tag, speakers, tracks
  • Users give feedback (not a rating) on any sessions anonymously (it uses Firebase anonymous login)
  • Voting forms are 100% customizable and can be translated into any languages
  • Display user feedback, with live feedback as an option
  • Full dashboard statistics & admin interface
  • Comments moderation
  • Admin editors management
  • Organization: manage users, rights, voting form and logos
