피드백을 쉽게 수집하세요

커뮤니티 · 문서

https://openfeedback.io/

Open Feedback은 주최자와 연사들이 사용자 피드백을 수집할 수 있는 오픈 SaaS 플랫폼입니다. OpenFeedback 데이터베이스, hoverboard-v2 또는 자체 API 중 하나에 연결하여 모든 세션과 연사를 가져옵니다. 익명 사용자는 세션에 대해 건설적인 피드백을 제공할 수 있으며, 관리자는 GUI 내에서 투표 양식, 댓글, 언어, 세션 및 연사를 직접 관리할 수 있습니다. 조직은 또한 중앙 집중식 사용자 관리와 공통 투표 양식을 가집니다.

데모: https://openfeedback.io/eaJnyMXD3oNfhrrnBYDT

기능

  • 이벤트별 고유하고 공개된 읽기 URL 생성
  • 모든 세션 목록, 태그, 연사, 트랙별 검색
  • 사용자가 익명으로 (Firebase 익명 로그인 사용) 어떤 세션에든 피드백(평가 아님) 제공
  • 투표 양식은 100% 사용자 정의 가능하며 모든 언어로 번역 가능
  • 사용자 피드백 표시, 옵션으로 실시간 피드백 제공
  • 완전한 대시보드 통계 및 관리자 인터페이스
  • 댓글 중재
  • 관리자 편집자 관리
  • 조직: 사용자, 권한, 투표 양식 및 로고 관리

기여하시겠습니까?

Open Feedback을 좋은 상태로 유지하기 위해 기여하고 싶다면, 여기로 가세요:

contribute-open-feedback.netlify.app

  • Kotlin SDK _Gérard Paligo와 Martin Bonnin 제작_: https://github.com/paug/openfeedback-android-sdk
