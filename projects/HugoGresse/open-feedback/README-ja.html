簡単にフィードバックを収集

コミュニティ · ドキュメント

https://openfeedback.io/

Open Feedback は、主催者や講演者がユーザーのフィードバックを収集するためのオープンなSaaSプラットフォームです。OpenFeedbackデータベース、hoverboard-v2、または独自のAPIのいずれか3つのデータソースに接続して、すべてのセッションと講演者を取得します。匿名ユーザーはセッションに対して建設的なフィードバックを提供でき、管理者は投票フォーム、コメント、言語、セッションおよび講演者をGUI内で直接管理できます。組織も集中管理されたユーザー管理と共通の投票フォームを持ちます。

デモ: https://openfeedback.io/eaJnyMXD3oNfhrrnBYDT

特徴

  • イベントのためのユニークで公開可能な読み取り用URLを生成
  • すべてのセッションをリスト表示、タグ、講演者、トラックで検索可能
  • ユーザーは匿名で任意のセッションにフィードバック（評価ではなく）を提供（Firebase匿名ログインを使用）
  • 投票フォームは100%カスタマイズ可能で、任意の言語に翻訳可能
  • ユーザーのフィードバックを表示、ライブフィードバックもオプションで提供
  • 完全なダッシュボード統計と管理インターフェース
  • コメントのモデレーション
  • 管理者エディターの管理
  • 組織：ユーザー、権限、投票フォーム、ロゴの管理

コミュニティSDK

  • Kotlin SDK _作成者：Gérard Paligo と Martin Bonnin_: https://github.com/paug/openfeedback-android-sdk
