Maha

⭐ 210 stars French by TRoboto





Une bibliothèque de traitement de texte arabe destinée à être utilisée dans des applications de TAL

Maha est une bibliothèque de traitement de texte spécialement développée pour traiter le texte arabe. La version bêta peut être utilisée pour nettoyer et analyser des textes, fichiers et dossiers avec ou sans capacité de streaming.

Si vous avez besoin d’aide ou souhaitez discuter de sujets liés à Maha, n’hésitez pas à rejoindre notre serveur Discord. Si vous souhaitez soumettre un rapport de bogue ou une demande de fonctionnalité, veuillez ouvrir une issue.

Installation

Il suffit d’exécuter la commande suivante pour installer Maha :

pip install mahad # pronounced maha d

Pour l'installation à partir des sources, consultez la documentation.
