laravel-tinker

⭐ 242 stars Korean by Roboroads

laravel artisan tinker을 사용하는 것처럼 PhpStorm/IDEA에서 PHP 코드를 쉽게 작성하고 실행하세요. IDE의 자동완성, 구문 강조, 오류 검사를 통해 향상된 Laravel 클래스에 대한 완전한 접근 권한으로 코드 스니펫을 실행하는 편리함을 누리세요.

모든 플러그인 액션은 도구 -> Laravel Tinker를 통해 접근하거나 기본 단축키 ctrl + Shift + T를 사용하여 Tinker가 지능적으로 사용자의 의도(열기, 실행 등)를 판단하도록 하세요.

  • 💡 GitHub 이슈: 기능 요청 및 버그 신고 제출
  • 🙏 Ko-Fi / GitHub 후원: 프로젝트가 도움이 되셨다면 후원해 주세요 ❤️
  • 🔌 플러그인 페이지: PHPStorm 또는 IDEA용 다운로드
  • 🔗 공유: 플러그인을 네트워크에 공유하세요

이 플러그인은 Laravel 프로젝트 외부에서도 작동합니다 - 기본 기능을 위해 프로젝트에 psy/psysh가 포함되어 있어야 합니다(일부 제한 사항 있음).
