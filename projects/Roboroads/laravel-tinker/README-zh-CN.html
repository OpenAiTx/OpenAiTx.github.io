laravel-tinker

⭐ 242 stars Simplified Chinese by Roboroads

轻松在 PhpStorm/IDEA 中直接编写并执行 PHP 代码，就像使用 laravel artisan tinker 一样。享受运行代码片段的便利，全面访问你的 Laravel 类，并借助 IDE 的自动补全、语法高亮和错误检查功能提升体验。

通过 工具 -> Laravel Tinker 访问所有插件操作，或使用默认快捷键 ctrl + Shift + T，让 Tinker 智能判断你的意图（打开、运行等）。

  • 💡 GitHub 问题：提交功能请求和错误报告
  • 🙏 Ko-Fi / GitHub 赞助：如果觉得有用，请支持该项目 ❤️
  • 🔌 插件页面：下载适用于 PHPStorm 或 IDEA
  • 🔗 分享：与您的网络分享该插件

该插件也适用于非 Laravel 项目——只需确保项目中已安装 psy/psysh 以实现基本功能（存在一定限制）。
