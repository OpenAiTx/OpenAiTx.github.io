OpenBMC 文件

OpenBMC 計畫 是一個由 Linux 基金會主導的計畫， 目標是為主機板管理控制器（BMCs）製作一個可自訂、開源的韌體堆疊。 本倉庫包含有關 OpenBMC 的整體文件。每個元件的倉庫中也可能有元件專屬的文件。

功能列表 文件列出了此計畫的主要功能，並附有更多資訊的連結。
