إطار عمل FastAPI، عالي الأداء، سهل التعلم، سريع في البرمجة، جاهز للإنتاج

التوثيق: https://fastapi.tiangolo.com

كود المصدر: https://github.com/fastapi/fastapi

FastAPI هو إطار عمل ويب حديث وسريع (عالي الأداء) لبناء واجهات برمجة التطبيقات (APIs) باستخدام بايثون، ويعتمد على تلميحات أنواع بايثون القياسية.

الميزات الرئيسية هي:

  • سريع: أداء عالي جداً، يقارن بـ NodeJS و Go (بفضل Starlette وPydantic). من أسرع أطر بايثون المتوفرة.
سريع في البرمجة: يزيد سرعة تطوير الميزات بحوالي 200% إلى 300%. أخطاء أقل: يقلل حوالي 40% من الأخطاء البشرية (أخطاء المطورين).
  • بديهي: دعم رائع من المحررات. إكمال تلقائي في كل مكان. وقت أقل في تصحيح الأخطاء.
  • سهل: مصمم ليكون سهل الاستخدام والتعلم. وقت أقل في قراءة التوثيقات.
  • مختصر: تقليل تكرار الشيفرة. ميزات متعددة من كل تعريف معلمة. أخطاء أقل.
  • متين: احصل على شيفرة جاهزة للإنتاج. مع توثيق تفاعلي تلقائي.
  • معتمد على المعايير: مبني (ومتوافق بالكامل) مع المعايير المفتوحة لواجهات برمجة التطبيقات: OpenAPI (المعروفة سابقاً بـ Swagger) و JSON Schema.

* تقدير مبني على اختبارات لفريق تطوير داخلي، لبناء تطبيقات إنتاجية.

المتطلبات

FastAPI يقف على أكتاف العمالقة:

  • Starlette للأجزاء المتعلقة بالويب.
  • Pydantic للأجزاء المتعلقة بالبيانات.

التثبيت

أنشئ وفعّل بيئة افتراضية ثم ثبت FastAPI:

$ pip install "fastapi[standard]"
---> 100%

ملاحظة: تأكد من وضع "fastapi[standard]" بين علامات اقتباس لضمان عمله في جميع الطرفيات.

مثال

أنشئه

أنشئ ملفًا باسم main.py يحتوي على:

from typing import Union
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}

أو استخدم async def...

إذا كان كودك يستخدم async / await، استخدم async def:

الاعتمادات

يعتمد FastAPI على Pydantic وStarlette.

اعتمادات standard

عند تثبيت FastAPI باستخدام pip install "fastapi[standard]" يأتي مع مجموعة standard من الاعتمادات الاختيارية:

يستخدمها Pydantic:

  • email-validator - للتحقق من صحة البريد الإلكتروني.

يستخدمها Starlette:

  • httpx - مطلوب إذا كنت تريد استخدام TestClient.
  • jinja2 - مطلوب إذا كنت تريد استخدام إعداد القوالب الافتراضي.
  • python-multipart - مطلوب إذا كنت تريد دعم "تحليل" النماذج، مع request.form().

يستخدمها FastAPI / Starlette:

  • uvicorn - لخادم تشغيل وتقديم تطبيقك. يتضمن ذلك uvicorn[standard]، والذي يتضمن بعض الاعتمادات (مثل uvloop) اللازمة لأداء عالي.
  • fastapi-cli - لتوفير أمر fastapi.

بدون اعتمادات standard

إذا كنت لا ترغب في تضمين الاعتمادات الاختيارية standard، يمكنك التثبيت باستخدام pip install fastapi بدلاً من pip install "fastapi[standard]".

اعتمادات اختيارية إضافية

هناك بعض الاعتمادات الإضافية التي قد ترغب في تثبيتها.

اعتمادات Pydantic الاختيارية الإضافية:

  • pydantic-settings - لإدارة الإعدادات.
  • pydantic-extra-types - لأنواع إضافية لاستخدامها مع Pydantic.

اعتمادات FastAPI الاختيارية الإضافية:

  • orjson - مطلوب إذا كنت تريد استخدام ORJSONResponse.
  • ujson - مطلوب إذا كنت تريد استخدام UJSONResponse`.
