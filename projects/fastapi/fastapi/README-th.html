FastAPI เฟรมเวิร์ก ประสิทธิภาพสูง เรียนรู้ง่าย เขียนโค้ดเร็ว พร้อมใช้งานในระดับโปรดักชัน

เอกสารประกอบ: https://fastapi.tiangolo.com

ซอร์สโค้ด: https://github.com/fastapi/fastapi

FastAPI เป็นเฟรมเวิร์กเว็บที่ทันสมัย เร็ว (ประสิทธิภาพสูง) สำหรับสร้าง API ด้วย Python โดยอิงกับ type hints มาตรฐานของ Python

คุณสมบัติเด่น:

  • เร็ว: ประสิทธิภาพสูงมาก ใกล้เคียงกับ NodeJS และ Go (ขอบคุณ Starlette และ Pydantic) หนึ่งในเฟรมเวิร์ก Python ที่เร็วที่สุด
เขียนโค้ดเร็ว: เพิ่มความเร็วในการพัฒนาฟีเจอร์ประมาณ 200% ถึง 300% บั๊กน้อยลง: ลดข้อผิดพลาดจากนักพัฒนาประมาณ 40%
  • เข้าใจง่าย: รองรับ editor ได้ดี Completion ทุกที่ ใช้เวลาน้อยลงกับการ debug
  • ใช้งานง่าย: ออกแบบมาให้ใช้งานและเรียนรู้ง่าย ใช้เวลาอ่านเอกสารน้อยลง
  • โค้ดสั้น: ลดการเขียนโค้ดซ้ำซ้อน หลายฟีเจอร์จากการประกาศ parameter เดียวกัน บั๊กน้อยลง
  • แข็งแกร่ง: ได้โค้ดที่พร้อมใช้งานในโปรดักชัน พร้อมเอกสาร interactive อัตโนมัติ
  • ยึดตามมาตรฐาน: พัฒนาตาม (และเข้ากันได้กับ) มาตรฐานเปิดสำหรับ API: OpenAPI (หรือที่รู้จักในชื่อ Swagger) และ JSON Schema

* ประมาณการจากผลการทดสอบกับทีมพัฒนาภายใน ขณะสร้างแอปพลิเคชันโปรดักชัน

การติดตั้ง

สร้างและเปิดใช้งาน virtual environment แล้วติดตั้ง FastAPI ด้วยคำสั่ง:

$ pip install "fastapi[standard]"
---> 100%

หมายเหตุ: ให้ใส่ "fastapi[standard]" ในเครื่องหมายคำพูด เพื่อให้ใช้งานได้กับทุกเทอร์มินัล

สร้างไฟล์

สร้างไฟล์ main.py ดังนี้:

from typing import Union
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}

ไลบรารีที่เกี่ยวข้อง

FastAPI อาศัย Pydantic และ Starlette

ไลบรารี standard ที่เกี่ยวข้อง

เมื่อติดตั้ง FastAPI ด้วย pip install "fastapi[standard]" จะมาพร้อมกับ dependencies กลุ่ม standard ดังนี้:

สำหรับ Pydantic:

  • email-validator - สำหรับการตรวจสอบอีเมล

สำหรับ Starlette:

  • httpx - ต้องใช้ถ้าต้องการ TestClient
  • jinja2 - ต้องใช้ถ้าต้องการ template configuration ค่าเริ่มต้น
  • python-multipart - ต้องใช้หากรองรับ "parsing" form ด้วย request.form()

สำหรับ FastAPI / Starlette:

  • uvicorn - สำหรับเซิร์ฟเวอร์ที่รันและให้บริการแอปของคุณ (รวมถึง uvicorn[standard] ที่มี dependencies สำหรับประสิทธิภาพสูงเช่น uvloop)
  • fastapi-cli - เพื่อคำสั่ง fastapi

หากไม่ต้องการ dependencies standard

หากไม่ต้องการ dependencies กลุ่ม standard ให้ติดตั้งด้วย pip install fastapi แทน pip install "fastapi[standard]"

ไลบรารีเสริมเพิ่มเติม

มี dependencies เสริมอื่นๆ ที่อาจต้องการติดตั้ง

สำหรับ Pydantic:

  • pydantic-settings - สำหรับจัดการ settings
  • pydantic-extra-types - สำหรับ type เสริมกับ Pydantic

สำหรับ FastAPI:

  • orjson - สำหรับ ORJSONResponse
  • ujson - สำหรับ UJSONResponse`
