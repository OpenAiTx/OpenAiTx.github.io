Framework FastAPI, hiệu năng cao, dễ học, lập trình nhanh, sẵn sàng cho môi trường sản xuất

Tài liệu: https://fastapi.tiangolo.com

Mã nguồn: https://github.com/fastapi/fastapi

FastAPI là một framework web hiện đại, nhanh (hiệu năng cao), dùng để xây dựng API với Python dựa trên các type hint chuẩn của Python.

Các tính năng chính:

  • Nhanh: Hiệu năng rất cao, ngang hàng với NodeJS và Go (nhờ vào Starlette và Pydantic). Một trong những framework Python nhanh nhất hiện nay.
Lập trình nhanh: Tăng tốc độ phát triển tính năng lên khoảng 200% đến 300%. Ít lỗi hơn: Giảm khoảng 40% lỗi do con người (lập trình viên) gây ra.
  • Trực quan: Hỗ trợ tốt cho editor. Tự hoàn thành ở mọi nơi. Giảm thời gian debug.
  • Dễ dàng: Thiết kế để dễ sử dụng và dễ học. Giảm thời gian đọc tài liệu.
  • Ngắn gọn: Giảm thiểu lặp lại mã nguồn. Nhiều tính năng từ mỗi khai báo tham số. Ít lỗi hơn.
  • Vững chắc: Được chuẩn bị sẵn sàng cho môi trường sản xuất. Có tài liệu tương tác tự động.
  • Dựa trên tiêu chuẩn: Dựa trên (và hoàn toàn tương thích với) các tiêu chuẩn mở cho API: OpenAPI (trước đây gọi là Swagger) và JSON Schema.

* ước tính dựa trên kiểm thử với nhóm phát triển nội bộ, xây dựng ứng dụng sản xuất.

Typer, FastAPI dành cho CLI

Nếu bạn đang xây dựng một ứng dụng CLI để dùng trong terminal thay vì web API, hãy xem qua Typer.

Typer là "em út" của FastAPI. Và nó hướng tới việc trở thành FastAPI của CLI. ⌨️ 🚀

Yêu cầu

FastAPI đứng trên vai những người khổng lồ:

  • Starlette cho phần web.
  • Pydantic cho phần dữ liệu.

Cài đặt

Tạo và kích hoạt một virtual environment sau đó cài đặt FastAPI:

$ pip install "fastapi[standard]"
---> 100%

Lưu ý: Hãy chắc chắn đặt "fastapi[standard]" trong dấu ngoặc kép để đảm bảo hoạt động trên mọi terminal.

Ví dụ

Tạo ứng dụng

Tạo file main.py với nội dung:

from typing import Union
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}

Phụ thuộc

FastAPI phụ thuộc vào Pydantic và Starlette.

Phụ thuộc standard

Khi bạn cài FastAPI bằng pip install "fastapi[standard]" nó sẽ kèm theo nhóm phụ thuộc tùy chọn standard:

Được Pydantic sử dụng:

  • email-validator - để kiểm tra email.

Được Starlette sử dụng:

  • httpx - Bắt buộc nếu bạn muốn dùng TestClient.
  • jinja2 - Bắt buộc nếu bạn muốn dùng cấu hình template mặc định.
  • python-multipart - Bắt buộc nếu bạn muốn hỗ trợ "parsing" form, với request.form().

Được FastAPI / Starlette sử dụng:

  • uvicorn - server để load và phục vụ ứng dụng của bạn. Bao gồm uvicorn[standard], tích hợp các phụ thuộc (ví dụ uvloop) cần thiết để phục vụ hiệu năng cao.
  • fastapi-cli - cung cấp lệnh fastapi.

Không dùng phụ thuộc standard

Nếu bạn không muốn bao gồm các phụ thuộc tùy chọn standard, bạn có thể cài bằng pip install fastapi thay vì pip install "fastapi[standard]".

Các phụ thuộc tùy chọn bổ sung

Có một số phụ thuộc bổ sung bạn có thể muốn cài thêm.

Phụ thuộc tùy chọn bổ sung cho Pydantic:

  • pydantic-settings - quản lý cấu hình.
  • pydantic-extra-types - các kiểu dữ liệu bổ sung cho Pydantic.

Phụ thuộc tùy chọn bổ sung cho FastAPI:

  • orjson - Bắt buộc nếu bạn muốn dùng ORJSONResponse.
  • ujson - Bắt buộc nếu bạn muốn dùng UJSONResponse`.
