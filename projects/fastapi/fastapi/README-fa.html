فریمورک FastAPI، با عملکرد بالا، یادگیری آسان، کدنویسی سریع، آماده برای محیط تولید

مستندات: https://fastapi.tiangolo.com

کد منبع: https://github.com/fastapi/fastapi

FastAPI یک فریمورک مدرن و سریع (با عملکرد بالا) برای ساخت APIها با پایتون، مبتنی بر تایپ هینت‌های استاندارد پایتون است.

ویژگی‌های کلیدی:

  • سریع: عملکرد بسیار بالا، هم‌تراز با NodeJS و Go (به لطف Starlette و Pydantic). یکی از سریع‌ترین فریمورک‌های پایتون موجود.
کدنویسی سریع: سرعت توسعه قابلیت‌ها را حدود ۲۰۰٪ تا ۳۰۰٪ افزایش دهید. باگ کمتر: حدود ۴۰٪ خطاهای انسانی (توسعه‌دهنده) را کاهش دهید.
  • مستقیم و شهودی: پشتیبانی عالی از ویرایشگر. تکمیل خودکار همه جا. زمان کمتری برای دیباگ.
  • آسان: طراحی شده برای استفاده و یادگیری آسان. زمان کمتر برای خواندن مستندات.
  • کوتاه: حداقل تکرار کد. قابلیت‌های متعدد از هر تعریف پارامتر. باگ کمتر.
  • قدرتمند: کد آماده برای محیط تولید. با مستندسازی تعاملی خودکار.
  • استاندارد محور: مبتنی بر استانداردهای باز APIها و کاملاً سازگار با آن‌ها: OpenAPI (که قبلاً Swagger نامیده می‌شد) و JSON Schema.

* برآورد بر اساس تست‌های تیم توسعه داخلی، هنگام ساخت برنامه‌های تولیدی.

نصب

یک محیط مجازی ایجاد و فعال کرده و سپس FastAPI را نصب کنید:

$ pip install "fastapi[standard]"
---> 100%

نکته: مطمئن شوید که "fastapi[standard]" را داخل کوتیشن قرار دهید تا در تمام ترمینال‌ها به درستی کار کند.

مثال

ایجاد فایل

فایلی به نام main.py ایجاد کنید با محتوای زیر:

from typing import Union
from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: Union[str, None] = None):
    return {"item_id": item_id, "q": q}

from fastapi import FastAPI from pydantic import BaseModel

app = FastAPI()

class Item(BaseModel): name: str price: float is_offer: Union[bool, None] = None

@app.get("/") def read_root(): return {"Hello": "World"}

@app.get("/items/{item_id}") def read_item(item_id: int, q: Union[str, None] = None): return {"item_id": item_id, "q": q}

@app.put("/items/{item_id}") def update_item(item_id: int, item: Item): return {"item_name": item.name, "item_id": item_id}

هشدار لو رفتن داستان!: آموزش - راهنمای کاربر شامل موارد زیر است:

  • تعریف پارامترها از مکان‌های مختلف مانند: هدرها، کوکی‌ها، فیلدهای فرم و فایل‌ها.
  • نحوه تعیین محدودیت‌های اعتبارسنجی مانند maximum_length یا regex.
  • یک سیستم تزریق وابستگی بسیار قدرتمند و آسان برای استفاده (Dependency Injection).
  • امنیت و احراز هویت شامل پشتیبانی از OAuth2 با توکن‌های JWT و احراز هویت HTTP Basic.

  • ادغام GraphQL با Strawberry و کتابخانه‌های دیگر.
  • بسیاری قابلیت‌های اضافی (به لطف Starlette) مانند:
  • WebSockets
  • تست‌های بسیار آسان مبتنی بر HTTPX و pytest

  • Cookie Sessions
  • ...و موارد بیشتر.

بنچمارک‌های مستقل TechEmpower نشان می‌دهند که برنامه‌های FastAPI که با Uvicorn اجرا می‌شوند جزو سریع‌ترین فریمورک‌های پایتون موجود هستند، و فقط بعد از Starlette و Uvicorn (که داخلی توسط FastAPI استفاده می‌شوند) قرار دارند. (*)

وابستگی‌ها

FastAPI به Pydantic و Starlette وابسته است.

وابستگی‌های standard

وقتی FastAPI را با دستور pip install "fastapi[standard]" نصب می‌کنید، شامل گروه وابستگی‌های اختیاری standard نیز می‌شود:

مورد استفاده در Pydantic:

  • email-validator - برای اعتبارسنجی ایمیل.

مورد استفاده در Starlette:

  • httpx - مورد نیاز اگر می‌خواهید از TestClient استفاده کنید.
  • jinja2 - مورد نیاز اگر می‌خواهید از پیکربندی قالب پیش‌فرض استفاده کنید.
  • python-multipart - مورد نیاز اگر می‌خواهید از تجزیه فرم با request.form() پشتیبانی کنید.

مورد استفاده در FastAPI / Starlette:

  • uvicorn - برای سروری که برنامه شما را بارگذاری و ارائه می‌کند. این شامل uvicorn[standard] است که وابستگی‌هایی مانند uvloop را برای عملکرد بالا دارد.
  • fastapi-cli - برای ارائه دستور fastapi.

بدون وابستگی‌های standard

اگر نمی‌خواهید وابستگی‌های اختیاری standard را نصب کنید، می‌توانید با pip install fastapi به جای pip install "fastapi[standard]" نصب نمایید.

وابستگی‌های اختیاری اضافی

برخی وابستگی‌های اضافی وجود دارد که ممکن است بخواهید نصب کنید.

وابستگی‌های اختیاری اضافی برای Pydantic:

  • pydantic-settings - برای مدیریت تنظیمات.
  • pydantic-extra-types - برای تایپ‌های اضافی قابل استفاده با Pydantic.

وابستگی‌های اختیاری اضافی برای FastAPI:

  • orjson - مورد نیاز اگر بخواهید از ORJSONResponse استفاده کنید.
  • ujson - مورد نیاز اگر بخواهید از UJSONResponse` استفاده کنید.
