🎯 MemGallery란 무엇인가?

MemGallery는 당신의 모든 디지털 삶을 저장하고 회상하는 데 도움을 주는 스마트하고 프라이버시 중심의 메모리 뱅크입니다. 오디오, 이미지, 텍스트, URL에 대한 지능형 파이프라인으로, 순간적인 생각이나 중요한 스크린샷을 다시는 잃지 않도록 보장합니다.

전통적인 앱과 달리, MemGallery는 제로 클라우드 정책에 기반합니다:

  • 완전한 프라이버시: 데이터는 기기 내에만 저장됩니다. 클라우드 동기화나 추적 없음.
  • 회원가입 불필요: 즉시 당신의 삶을 인덱싱 시작 가능.
  • AI 기반 지능: 원하는 엔진 선택 가능—로컬 모델, Google Gemini, Groq (Q), 또는 OpenAI 호환 제공자 사용.

🧠 능동적 지능

MemGallery는 단순히 데이터를 저장하는 것을 넘어 이해합니다. 우리의 능동적 AI 엔진은 자동으로:
  • 모든 것을 인덱싱: 이미지 스캔, 오디오 전사, URL 요약 수행.
  • 행동 추출: 기억 속에 숨겨진 약속, 할 일, 알림 식별.
  • 스마트 회상: 간단하고 자연스러운 언어 검색으로 즉시 무엇이든 찾기.

  • 스마트 알림: 기억 속에서 발견된 다가오는 마감일과 이벤트에 대해 알림 받기

🔍 검색 및 조직

🤔 왜 MemGallery인가?

문제점

우리는 매일 수백 개의 정보—스크린샷, 대화, 기사, 알림—를 접합니다. 기존 갤러리와 노트 앱은 이들을 분리된 파일로 취급합니다. 지난주 특정 스크린샷이나 링크를 찾는 것은 종종 번거롭습니다.

해결책

MemGallery는 안드로이드 기기를 다음과 같은 지능형 기억 시스템으로 변환합니다:

  • 절대 잊지 않음: 스크린샷 자동 캡처 및 웹 링크 정리
  • 문맥 이해: AI가 이미지와 오디오의 내용을 읽고 이해
  • 모두 검색 가능: 자연어로 원하는 모든 것 검색
  • 실행 가능한 항목 추출: 자동으로 이벤트 감지 후 작업 목록에 추가
  • 오프라인 우선 작동: 데이터는 사용자가 처리 선택할 때까지 기기에만 저장

📥 설치

  • 최신 MemGallery-v*.apk 파일 다운로드

  • Android 7.0 (API 24) 이상
  • AI 처리를 위한 인터넷 연결
  • 약 50MB 저장 공간

🔑 설정

1. Google AI Studio API 키 받기

MemGallery는 AI 처리를 위해 Google Gemini API 키가 필요합니다. 대부분 개인용으로 무료입니다.

  • Google AI Studio 방문
  • Google 계정으로 로그인
  • "Get API Key" -> "Create API Key" 클릭
  • 키 복사

2. MemGallery 구성

  • MemGallery 실행 후 온보딩 완료
  • 요청 시 API 키 붙여넣기
  • 필요한 권한 허용 (카메라, 마이크, 알림)
  • (선택 사항) 설정에서 "자동 스크린샷 인덱스" 활성화

📱 사용법

추억 캡처하기

  • 스크린샷: 평소처럼 스크린샷을 찍으세요. MemGallery가 자동으로 감지하고 처리합니다.
  • 웹 북마크: 브라우저에서 공유 -> MemGallery를 눌러 링크를 저장하고 분석하세요.
  • 빠른 캡처: + 버튼을 사용하거나 앱 아이콘을 길게 눌러 특정 유형(오디오, 텍스트, 사진)을 추가하세요.

작업 관리

  • 작업 탭으로 이동하세요 (설정에서 활성화된 경우).
  • 기억에서 AI가 감지한 작업과 이벤트를 확인하세요.
  • 캘린더 뷰를 사용해 일정을 한눈에 보세요.

사용자 설정

  • 설정 -> 외관에서 AMOLED 모드를 전환하거나 앱 테마를 변경하세요.
  • 고급 설정에서 AI에 맞춤 시스템 프롬프트를 제공할 수 있습니다 (예: "간결하게", "기술적 세부사항에 집중").

🛠️ 기술 스택

  • 언어: Kotlin
  • UI: Jetpack Compose (Material 3)
  • 아키텍처: MVVM + 클린 아키텍처
  • DI: Hilt
  • AI: Google Gen AI SDK를 통한 Google Gemini 2.5 Flash
  • 데이터베이스: Room
  • 백그라운드: WorkManager 및 포그라운드 서비스
  • 웹 파싱: Jsoup

🔒 개인정보 보호

  • 로컬 저장: 모든 기억과 작업은 기기에 로컬로 저장됩니다.
  • 보안 AI: 콘텐츠는 분석을 위해서만 Google의 Gemini API로 전송되며, 공개 모델 학습에는 사용되지 않습니다 (기업 수준의 개인정보 보호).
  • 암호화된 키: API 키는 Android의 EncryptedSharedPreferences를 사용하여 안전하게 저장됩니다.
