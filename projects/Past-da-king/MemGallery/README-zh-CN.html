📱 应用展示

🎯 什么是 MemGallery？

MemGallery 是一个智能且以隐私为先的记忆库，旨在帮助您存储和回忆整个数字生活。它是一个用于您的音频、图像、文本和网址的智能管道，确保您不再丢失稍纵即逝的想法或重要的截图。

与传统应用不同，MemGallery 建立在零云端原则上：

  • 完全隐私：您的数据保存在设备上。无云同步，无追踪。
  • 无需注册：立即开始索引您的生活。
  • AI驱动智能：选择您的引擎——使用本地模型、Google Gemini、Groq (Q) 或任何兼容 OpenAI的提供商。

🧠 主动智能

MemGallery 不仅仅存储数据；它理解数据。我们的主动 AI 引擎会自动：
  • 索引一切：扫描您的图片，转录您的音频，摘要您的网址。
  • 提取动作：识别隐藏在记忆中的约会、待办事项和提醒。
  • 智能回忆：使用简单自然语言搜索，瞬间找到任何内容。

🤔 为什么选择 MemGallery？

问题所在

我们每天都会遇到数百条信息——截图、对话、文章、提醒。传统的图库和笔记应用将这些视为彼此独立的文件。想要找到上周的那张特定截图或链接通常很麻烦。

解决方案

MemGallery 将您的 Android 设备转变为智能记忆系统，能够：

  • 永不遗忘：自动捕捉截图并整理网页链接
  • 理解上下文：AI 读取并理解您图片和音频的内容
  • 一切可搜索：使用自然语言查找任何内容
  • 提取可执行项：自动检测事件并添加到任务列表
  • 优先离线工作：数据保留在设备上，直到您选择处理

📥 安装

选项 1：下载 APK（推荐）

  • 访问 Releases 页面
  • 下载最新的 MemGallery-v*.apk 文件
  • 在你的安卓设备上打开 APK

最低要求：
  • 安卓 7.0（API 24）或更高版本
  • 需要互联网连接以进行 AI 处理
  • 约 50MB 存储空间

---

🔑 设置

1. 获取 Google AI Studio API 密钥

MemGallery 需要 Google Gemini API 密钥 进行 AI 处理。大多数个人使用是免费的。

  • 访问 Google AI Studio
  • 使用你的 Google 账号登录
  • 点击 “获取 API 密钥” -> “创建 API 密钥”
  • 复制该密钥

2. 配置 MemGallery

  • 启动 MemGallery 并完成引导
  • 在提示时粘贴你的 API 密钥
  • 授予必要权限（相机、麦克风、通知）
  • （可选）在设置中启用 “自动索引截图”

📱 使用

捕捉记忆

  • 截图：像往常一样截图。MemGallery会自动检测并处理截图。
  • 网页书签：在浏览器中，点击分享 -> MemGallery保存并分析链接。
  • 快速捕获：使用+按钮或长按应用图标添加特定类型（音频、文本、照片）。

任务管理

  • 进入任务标签页（如果在设置中启用）。
  • 查看AI从记忆中检测到的任务和事件。
  • 使用日历视图一目了然地查看日程。

自定义

  • 进入设置 -> 外观切换AMOLED模式或更改应用主题。
  • 使用高级设置为AI提供自定义系统提示（例如，“简明扼要”，“关注技术细节”）。

🛠️ 技术栈

  • 语言：Kotlin
  • 界面：Jetpack Compose（Material 3）
  • 架构：MVVM + 清洁架构
  • 依赖注入：Hilt
  • AI：通过Google Gen AI SDK使用Google Gemini 2.5 Flash
  • 数据库：Room
  • 后台：WorkManager和前台服务
  • 网页解析：Jsoup

---

🔒 隐私

  • 本地存储：所有记忆和任务均存储在您的设备本地。
  • 安全AI：内容仅发送到Google的Gemini API进行分析，不用于训练公共模型（企业级隐私）。
  • 加密密钥：您的API密钥通过Android的EncryptedSharedPreferences安全存储。
