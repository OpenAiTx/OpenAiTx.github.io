備註：

重要：需要 Android 15（SDK 35）以獲得完整的 Vulkan 1.4 支援

請使用 Ubuntu 24.04 或任何基於其的發行版

在繼續之前請確保你的網路連線穩定

如果你的 ISP 限速，請使用 VPN

請確保你的 Android 版本為 14` 或以上，否則將無法安裝

安裝前請確保你已安裝最新的 MAGISK / KERNELSU
