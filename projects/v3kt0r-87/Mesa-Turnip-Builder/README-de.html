HINWEISE :

Wichtig : Android 15 (SDK 35) wird für vollständige Vulkan 1.4-Unterstützung benötigt

Bitte verwenden Sie Ubuntu 24.04 oder eine darauf basierende Distribution

STELLEN SIE SICHER, DASS SIE EINE STABILE INTERNETVERBINDUNG HABEN, BEVOR SIE FORTFAHREN

VERWENDEN SIE EIN VPN, WENN IHR ISP IHRE GESCHWINDIGKEIT DROSSELT

Stellen Sie sicher, dass Ihre Android-Version 14` oder höher ist, andernfalls können Sie nicht installieren

Stellen Sie sicher, dass Sie das neueste MAGISK / KERNELSU vor der Installation haben
