NOTE :

Importante : Android 15 (SDK 35) è necessario per il pieno supporto a Vulkan 1.4

Si consiglia di usare Ubuntu 24.04 o qualsiasi altra distro basata su di esso

ASSICURATI DI AVERE UNA CONNESSIONE INTERNET STABILE PRIMA DI PROCEDERE

USA UNA VPN SE IL TUO ISP LIMITA LA VELOCITÀ

Assicurati che la versione Android sia 14` o superiore, altrimenti non potrai installare
