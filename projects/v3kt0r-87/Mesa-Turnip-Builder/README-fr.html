Mesa Stable / RC + Android NDK

Script Bash simple qui vise à construire un pilote turnip pour MAGISK/KERNELSU ou EMULATEUR

Quoi de neuf : 🔥

Cliquez ici

Comment construire localement 🤔

Il suffit de cloner ce dépôt et d’utiliser BASH pour construire

`` bash build-turnip.sh `

NOTES :

Important : Android 15 (SDK 35) est nécessaire pour le support complet de Vulkan 1.4

Veuillez utiliser Ubuntu 24.04 ou toute autre distribution basée dessus

ASSUREZ-VOUS D'AVOIR UNE CONNEXION INTERNET STABLE AVANT DE COMMENCER

UTILISEZ UN VPN SI VOTRE FAI RALENTIT VOTRE DÉBIT

Assurez-vous que votre version d'Android est 14` ou supérieure sinon vous ne pourrez pas installer

Assurez-vous d'avoir la dernière version de MAGISK / KERNELSU avant l'installation
