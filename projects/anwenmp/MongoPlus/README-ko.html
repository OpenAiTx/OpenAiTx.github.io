MongoPlus

⭐ 66 stars Korean by anwenmp

소개：

#### 🔥🔥🔥MyBatisPlus 방식을 사용하여 우아하게 MongoDB 조작하기

  • #### 바람에 잠입하는 밤
        기존 Mongo 프레임워크와 함께 사용할 수 있으며, 비침입적임
  • #### 개발 간소화
        간단한 설정만으로 빠른 CRUD 작업 가능, 체인 호출이 더 편리함

장점：

  • 비침입성: 오직 기능 강화만 수행하며 기존 프로젝트에 영향 없이 부드럽게 도입 가능
  • 낮은 비용: 시작 시 기본 CRUD 자동 주입, 성능 손실 거의 없음, 객체 지향 직접 조작 가능
  • 강력한 CRUD 작업: 공통 서비스로 소량 설정만으로 단일 테이블 대부분 CRUD 작업 수행 가능, 강력한 조건 생성기로 다양한 요구 충족
  • 람다 호출 지원: 람다 표현식으로 각종 쿼리 조건을 편리하게 작성, 필드명 오기입 걱정 없음
  • 자동 키 생성 지원: 최대 5가지 키 생성 전략 지원(분산 유니크 ID 생성기 - Sequence 포함), 자유로운 설정으로 키 문제 완벽 해결
  • 엔티티 클래스 없이도 지원
  • 동적 데이터 소스 지원
  • 논리 삭제 지원, 전체 컬렉션 업데이트 및 삭제 방지, 자동 채우기 등 기능 지원

빠른 개발

MyBatisPlus는 MyBatis처럼 너무 많은 SQL 문을 작성할 필요가 없듯이, Mongo-Plus도 마찬가지입니다!

프로젝트는 머피 시큐리티에 접속됨
