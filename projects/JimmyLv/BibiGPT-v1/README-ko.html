작동 방식

이 프로젝트는 OpenAI ChatGPT API (특히 gpt-3.5-turbo)와 Vercel Edge functions의 스트리밍, 그리고 Redis 캐시 및 속도 제한을 위한 Upstash를 사용합니다. Bilibili 영상의 콘텐츠를 가져와 GPT-3 API에 프롬프트로 보내 요약을 요청한 뒤, Vercel Edge function을 통해 응답을 애플리케이션에 스트리밍합니다.

비용 절감

이와 같은 프로젝트는 비용이 많이 들 수 있으므로, 자신의 버전을 만들고 공개하려는 경우 비용 절감을 위해 다음 세 가지를 권장합니다:

  • [x] 1. 사이트 남용을 막기 위한 속도 제한 구현
  • [x] 2. 비싼 AI 재생성을 피하기 위한 캐싱 구현
  • [x] 3. summarize 엣지 함수에서 text-dacinci-003 대신 text-curie-001 사용

로컬 실행

레포를 클론한 후, OpenAI에서 계정을 만들고 API 키를 .env라는 파일에 넣으세요.

그런 다음, 커맨드 라인에서 애플리케이션을 실행하면 http://localhost:3000에서 이용할 수 있습니다.

구체적인 실행 절차는 이 문서에 설명되어 있습니다 - 중국어 버전

npm run dev
