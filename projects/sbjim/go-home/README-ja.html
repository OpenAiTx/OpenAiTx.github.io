今回の更新説明：

今回の更新は基本的に底層アルゴリズムを書き直し、monent.js、jquery、bootstrap3 サードパーティライブラリを導入

  • [√] 出退勤時間のカスタマイズ可能
  • [√] 退勤までの残り時間表示
  • [√] 週末までの残り時間表示
  • [√] 月末までの残り時間表示
  • [√] 週休二日制、単休の週末タイプ追加
  • [ x ] 給料支払日追加 - 未完成
