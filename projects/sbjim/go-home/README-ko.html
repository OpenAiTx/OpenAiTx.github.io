이번 업데이트 설명：

이번 업데이트는 기본적으로 하위 알고리즘을 재작성하고, moment.js, jquery, bootstrap3 타사 라이브러리를 도입했습니다.

  • [√] 출퇴근 시간 사용자 정의 가능
  • [√] 퇴근까지 남은 시간 표시
  • [√] 주말까지 남은 시간 표시
  • [√] 월말까지 남은 시간 표시
  • [√] 주말 유형에 주 5일(토,일 휴무), 주 6일(일 휴무) 추가
  • [ x ] 급여 지급 시간 추가 - 미완료

계획된 버전 업그레이드 v2.1.1：

+ 퇴근 10분 전에 알림 전송 사용자에게 퇴근 시간이 다가옴을 알림

인터페이스 효과 화면

플러그인 설명

+ 기본 출근 시간은 9:00 + 기본 퇴근 시간은 18:00 + 기본 주말 유형은 주 5일 근무(토,일 휴무)

신버전 업데이트 설명

+ moment js 날짜 프레임워크 도입 + 하위 날짜 계산 최적화 + 출퇴근 시간 사용자 정의 기능 추가 + 주말 유형 추가
