⚡ Schnelle Integration

Verwenden Sie das iframe-Tag in HTML-Webseiten für eine schnelle Integration:

🐳 Deployment mit Docker

  • Image herunterladen:

docker pull zyyzyykk/kkterminal

  • Container erstellen und für Port-Mapping ausführen: -p port:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • Benutzerdefiniertes Artword: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • Zugriff im Browser: http://server-ip:3000/

💡 Funktionsbeschreibung

  • kkTerminal ist ein Terminal für Web-SSH-Verbindungen. Klicken Sie auf das Terminalsymbol oben links und wählen Sie die Verbindungseinstellungen, um eine SSH-Verbindung herzustellen

  • Unterstützung der schnellen Integration in Drittanbieter-Webseiten über das iframe-Tag

  • Unterstützung der individuellen Terminal-Konfiguration über URL-Parameter

  • Unterstützung der Internationalisierung und Sprachumschaltung zwischen Chinesisch und Englisch

  • Unterstützung der lokalen PC-Bereitstellung, öffnet beim Starten automatisch das Browserfenster

  • Unterstützung der adaptiven Fenstergröße und chinesischer Eingabe

  • Unterstützung benutzerdefinierter Einstellungen, sodass Benutzer Hintergrund-/Vordergrundfarbe, Schriftgröße, Cursor-Stil, TCode für das Terminal usw. wählen können

  • Unterstützung von Neustarts: Nach Änderung der SSH-Verbindungseinstellungen oder benutzerdefinierten Einstellungen erfolgt ein automatischer Neustart, oder es kann nach Trennung der SSH-Verbindung manuell neugestartet werden

  • Unterstützung von Kopieren und Einfügen:

  • Kopieren: Wie beim Git-Terminal wird Text automatisch beim Markieren kopiert

  • Einfügen: Wie beim Cmd-Terminal, Rechtsklick zum Einfügen (Benötigt Browser-Zugriffsberechtigung)

  • Unterstützung der Dateiverwaltung, öffnen Sie das Dateiverwaltungsmodul zum Anzeigen, Entpacken, Hoch- und Herunterladen von Dateien/Ordnern

  • Unterstützung von Tastenkombinationen wie Mehrfach-/Gesamtauswahl, Kopieren und Einfügen, Ausschneiden, Auswahlwechsel, Öffnen usw.

  • Unterstützung der Dateiansicht und -bearbeitung, Datei ändern und mit ctrl+s auf den Remote-Server speichern

  • Unterstützung von TCode (Terminal Code), mit dem automatisierte Workflows ähnlich wie Shell-Skripte durch benutzerdefinierten TCode realisiert werden können

  • Unterstützung der Aufzeichnungs- und Cloud-Synchronisationsfunktionen

  • Unterstützung von Kooperieren-, Überwachungs- und Docker-Funktionen
