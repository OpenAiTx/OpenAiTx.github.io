⚡ Hızlı Entegrasyon

HTML web sayfalarında iframe etiketi kullanarak hızlı entegrasyon:

🐳 Docker ile Dağıtım

  • İmajı çekin:

docker pull zyyzyykk/kkterminal

  • Port eşlemesiyle bir konteyner oluşturup çalıştırın: -p port:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • Özel sanat yazısı: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • Tarayıcıdan erişim: http://server-ip:3000/

💡 Fonksiyon Açıklaması

  • kkTerminal, Web SSH bağlantısı için bir terminaldir. Sol üst köşedeki terminal simgesine tıklayın ve bağlantı ayarlarını seçerek bir SSH bağlantısı kurun

  • Web sayfalarında iframe etiketinin kullanılmasını destekler ve üçüncü parti sitelere hızlı entegrasyon sağlar

  • URL Parametreleri aracılığıyla terminal yapılandırmasının özelleştirilmesini destekler

  • Uluslararasılaştırma ve Çince-İngilizce dil geçişini destekler

  • Yerel PC kurulumunu destekler, başlatıldığında otomatik olarak tarayıcı penceresini açar

  • Uyumlu pencere boyutu ve Çince giriş desteği

  • Arka plan/ön plan rengi, yazı tipi boyutu, imleç gösterim stili, terminalde TCode etkinleştirme gibi özel tercihler ayarlanabilir

  • Yeniden başlatma desteği: SSH bağlantı ayarlarını veya özel tercihleri değiştirdikten sonra otomatik olarak yeniden başlatılır, veya SSH bağlantısı kesildikten sonra manuel olarak yeniden başlatılabilir

  • Kopyala ve yapıştır desteği:

  • Kopyala: Git terminalinde olduğu gibi, metni seçtiğinizde otomatik olarak kopyalanır

  • Yapıştır: Cmd terminalinde olduğu gibi, sağ tıklayarak yapıştırılır (tarayıcı erişim izni gerektirir)

  • Dosya yönetimini destekler, Dosya Yönetimi Modülünü açarak dosya/klasörleri görüntüleyin, sıkıştırmasını açın, yükleyin ve indirin

  • Birden çok/tüm dosya seçimi, kopyala & yapıştır, kes, seçim değiştir, aç gibi kısayol tuş işlemlerini destekler

  • Dosya gezintisi ve düzenlemesini destekler, dosyayı değiştirip ctrl+s ile uzak sunucuya kaydedin

  • TCode (Terminal Code) desteği, Özelleştirilmiş TCode ile Shell betiklerine benzer otomatikleştirilmiş İş Akışı elde edilebilir

  • İşlem kaydı ve bulut senkronizasyon fonksiyonlarını destekler

  • İşbirliği, İzleme ve Docker fonksiyonlarını destekler
