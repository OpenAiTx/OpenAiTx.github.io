🐳 استقرار با Docker

  • دریافت ایمیج:

docker pull zyyzyykk/kkterminal

  • ایجاد و اجرای کانتینر با نگاشت پورت: -p port:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • سفارشی‌سازی واژه هنری: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • دسترسی در مرورگر: http://server-ip:3000/

💡 توضیحات عملکرد

  • kkTerminal یک ترمینال برای اتصال SSH تحت وب است. روی آیکن ترمینال در گوشه بالا سمت چپ کلیک کرده و تنظیمات اتصال را انتخاب کنید تا ارتباط SSH برقرار شود.

  • پشتیبانی از استفاده از تگ iframe در صفحات وب برای ادغام سریع در سایت‌های شخص ثالث

  • پشتیبانی از سفارشی‌سازی تنظیمات ترمینال از طریق پارامترهای URL

  • پشتیبانی از بین‌المللی‌سازی و امکان تغییر زبان بین چینی و انگلیسی

  • پشتیبانی از استقرار محلی روی PC، هنگام اجرا به صورت خودکار پنجره مرورگر را باز می‌کند

  • پشتیبانی از اندازه پنجره تطبیقی و ورودی چینی

  • پشتیبانی از تنظیمات ترجیحات سفارشی، امکان انتخاب رنگ پس‌زمینه/پیش‌زمینه، اندازه فونت، سبک نمایش مکان‌نما، فعال‌سازی TCode برای ترمینال و غیره

  • پشتیبانی از راه‌اندازی مجدد: پس از تغییر تنظیمات اتصال SSH یا تنظیمات ترجیحی، به صورت خودکار راه‌اندازی مجدد می‌شود یا پس از قطع ارتباط SSH می‌توان به صورت دستی راه‌اندازی مجدد کرد

  • پشتیبانی از کپی و چسباندن:

  • کپی: مشابه ترمینال Git، انتخاب متن به صورت خودکار آن را کپی می‌کند

  • چسباندن: مشابه ترمینال Cmd، با راست کلیک چسباندن انجام می‌شود (نیاز به مجوز مرورگر دارد)

  • پشتیبانی از مدیریت فایل، باز کردن ماژول مدیریت فایل برای مشاهده، استخراج، بارگذاری و دانلود فایل‌ها/پوشه‌ها

  • پشتیبانی از عملیات کلیدهای میانبر مانند انتخاب چندتایی/همه فایل‌ها، کپی و چسباندن، برش، تغییر انتخاب، باز کردن و غیره

  • پشتیبانی از مرور و ویرایش فایل، ویرایش فایل و ذخیره آن در سرور راه دور با استفاده از ctrl+s

  • پشتیبانی از TCode (کد ترمینال)، که می‌تواند گردش کار خودکار مشابه اسکریپت‌های Shell را از طریق TCode سفارشی ایجاد کند

  • پشتیبانی از ضبط عملیات و همگام‌سازی ابری

  • پشتیبانی از عملکردهای همکاری، نظارت و Docker
