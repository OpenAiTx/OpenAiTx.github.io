🐳 使用 Docker 部署

  • 拉取镜像：

docker pull zyyzyykk/kkterminal

  • 创建并运行容器进行端口映射：-p 本地端口:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • 自定义艺术字：-e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • 浏览器访问：http://服务器IP:3000/

💡 功能说明

  • kkTerminal 是一个 Web SSH 连接终端，点击左上角终端图标并选择连接设置即可建立 SSH 连接

  • 支持在网页中使用 iframe 标签，快速集成到第三方网站

  • 支持通过URL 参数自定义终端配置

  • 支持国际化和中英文语言切换

  • 支持本地电脑部署，启动后自动打开浏览器窗口

  • 支持窗口自适应和中文输入

  • 支持自定义偏好设置，用户可选择背景/前景色、字体大小、光标显示样式、开启终端 TCode 等

  • 支持重启：修改 SSH 连接设置或自定义偏好设置后自动重启，SSH 断开后可手动重启

  • 支持复制粘贴：

  • 复制：与 Git 终端一致，选中文字自动复制
  • 粘贴：与 Cmd 终端一致，右键粘贴（需浏览器授权）

  • 支持文件管理，打开文件管理模块可查看、解压、上传和下载文件/文件夹

  • 支持多选/全选、复制粘贴、剪切、切换选择、打开等快捷键操作

  • 支持文件浏览与编辑，修改后使用 ctrl+s 保存到远程服务器

  • 支持 TCode (终端代码)，可通过自定义 TCode 实现类似 Shell 脚本的自动化流程

  • 支持操作录制与云同步功能

  • 支持协作、监控和 Docker 功能
