💡 功能說明

  • kkTerminal 是一個用於 Web SSH 連線的終端機，點擊左上角終端機圖示並選擇連線設置，即可建立 SSH 連線

  • 支援在網頁中使用 iframe 標籤，快速整合至第三方網站

  • 支援通過 URL 參數 自訂終端機配置

  • 支援國際化及中英文語言切換

  • 支援本地 PC 部署，啟動時自動開啟瀏覽器視窗

  • 支援視窗大小自適應及中文輸入

  • 支援自訂偏好設置，允許用戶選擇背景/前景色、字體大小、游標顯示樣式、終端機啟用 TCode 等

  • 支援重啟：修改 SSH 連線設置或自訂偏好設置後會自動重啟，或 SSH 連線斷開後可手動重啟

  • 支援複製與貼上：

  • 複製：與 Git 終端機一致，選取文字即自動複製
  • 貼上：與 Cmd 終端機一致，右鍵貼上（需瀏覽器授權）

  • 支援檔案管理，開啟檔案管理模組可查看、解壓、上傳及下載檔案/資料夾

  • 支援快捷鍵操作，如多選/全選、複製貼上、剪下、選擇切換、開啟等

  • 支援檔案瀏覽與編輯，修改檔案後可通過 ctrl+s 儲存至遠端伺服器

  • 支援 TCode (Terminal Code)，可通過自訂 TCode 實現類 Shell 腳本的自動化工作流

  • 支援操作記錄及雲端同步功能

  • 支援協作、監控及 Docker 功能
