🐳 النشر باستخدام Docker

  • سحب الصورة:

docker pull zyyzyykk/kkterminal

  • أنشئ وشغل حاوية مع تعيين المنفذ: -p port:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • تخصيص كلمة الفن: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • الوصول من المتصفح: http://server-ip:3000/

💡 وصف الوظائف

  • kkTerminal هو طرفية لاتصال SSH عبر الويب. انقر على أيقونة الطرفية في الزاوية العلوية اليسرى واختر إعدادات الاتصال لإنشاء اتصال SSH

  • يدعم استخدام وسم iframe في صفحات الويب للدمج السريع في مواقع الطرف الثالث

  • يدعم تخصيص إعدادات الطرفية من خلال معاملات الرابط

  • يدعم التدويل والتبديل بين اللغتين الصينية والإنجليزية

  • يدعم النشر المحلي على الحاسب وفتح نافذة المتصفح تلقائياً عند بدء التشغيل

  • يدعم التكيف مع حجم النافذة وإدخال اللغة الصينية

  • يدعم إعدادات التفضيلات المخصصة، بحيث يمكن للمستخدمين اختيار لون الخلفية/المقدمة، حجم الخط، نمط عرض المؤشر، تفعيل TCode للطرفية، إلخ

  • يدعم إعادة التشغيل: سيتم إعادة التشغيل تلقائياً بعد تعديل إعدادات اتصال SSH أو إعدادات التفضيلات المخصصة، أو يمكن إعادة التشغيل يدوياً بعد قطع اتصال SSH

  • يدعم النسخ واللصق:

  • النسخ: مثل طرفية Git، تحديد النص سيتم نسخه تلقائياً

  • اللصق: مثل طرفية Cmd، انقر بزر الماوس الأيمن للصق (يتطلب إذن وصول المتصفح)

  • يدعم إدارة الملفات، افتح وحدة إدارة الملفات لعرض وفك ضغط ورفع وتنزيل الملفات/المجلدات

  • يدعم اختصارات لوحة المفاتيح مثل تحديد ملفات متعددة/الكل، نسخ ولصق، قص، تبديل التحديد، فتح، إلخ

  • يدعم تصفح الملفات وتحريرها، تعديل الملف وحفظه على الخادم البعيد باستخدام ctrl+s

  • يدعم TCode (كود الطرفية)، والذي يمكنه تنفيذ سير عمل تلقائي شبيه بسكربتات Shell عبر TCode مخصص

  • يدعم تسجيل العمليات والمزامنة السحابية

  • يدعم وظائف التعاون، المراقبة وDocker
