🐳 Deploy con Docker

  • Scarica l'immagine:

docker pull zyyzyykk/kkterminal

  • Crea ed esegui un container per il mapping della porta: -p porta:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • Personalizza la scritta artistica: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • Accesso dal browser: http://ip-server:3000/

💡 Descrizione delle Funzionalità

  • kkTerminal è un terminale per la connessione Web SSH. Clicca sull'icona del terminale in alto a sinistra e seleziona le impostazioni di connessione per stabilire una connessione SSH

  • Supporta l'utilizzo dei tag iframe nelle pagine web per una rapida integrazione in siti di terze parti

  • Supporta la personalizzazione della configurazione del terminale tramite Parametri URL

  • Supporta l'internazionalizzazione e il cambio lingua tra cinese e inglese

  • Supporta il deploy locale su PC, all'avvio apre automaticamente la finestra del browser

  • Supporta la dimensione adattiva della finestra e l'input cinese

  • Supporta impostazioni preferenze personalizzate, permettendo agli utenti di scegliere colore di sfondo/colore primo piano, dimensione font, stile del cursore, abilitare TCode per il terminale, ecc

  • Supporta il riavvio: si riavvia automaticamente dopo la modifica delle impostazioni di connessione SSH o delle preferenze personalizzate, oppure può essere riavviato manualmente dopo la disconnessione SSH

  • Supporta copia e incolla:

  • Copia: Come nel terminale Git, selezionando il testo viene copiato automaticamente

  • Incolla: Come nel terminale Cmd, tasto destro per incollare (richiede permessi del browser)

  • Supporta la gestione dei file, apri il Modulo Gestione File per visualizzare, decomprimere, caricare e scaricare file/cartelle

  • Supporta operazioni tramite tasti di scelta rapida come selezione multipla/tutti i file, copia e incolla, taglia, cambio selezione, apertura, ecc

  • Supporta la navigazione e la modifica dei file, modifica il file e salvalo su un server remoto usando ctrl+s

  • Supporta TCode (Terminal Code), che può realizzare Workflow automatizzati simili a script Shell tramite TCode personalizzato

  • Supporta funzioni di registrazione delle operazioni e sincronizzazione cloud

  • Supporta le funzioni Cooperate, Monitor e Docker
