🐳 การติดตั้งด้วย Docker

  • ดึงอิมเมจ:

docker pull zyyzyykk/kkterminal

  • สร้างและรันคอนเทนเนอร์สำหรับการแมปพอร์ต: -p port:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • กำหนดคำศิลป์เอง: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • เข้าถึงผ่านเบราว์เซอร์: http://server-ip:3000/

💡 คำอธิบายฟังก์ชัน

  • kkTerminal เป็นเทอร์มินัลสำหรับเชื่อมต่อ Web SSH คลิกไอคอนเทอร์มินัลที่มุมซ้ายบนและเลือกการตั้งค่าการเชื่อมต่อเพื่อสร้างการเชื่อมต่อ SSH

  • รองรับการใช้แท็ก iframe ในหน้าเว็บเพื่อเชื่อมต่ออย่างรวดเร็วกับเว็บไซต์ภายนอก

  • รองรับการปรับแต่งการตั้งค่าเทอร์มินัลผ่าน พารามิเตอร์ URL

  • รองรับการแปลภาษาและสลับระหว่างภาษาจีนกับอังกฤษ

  • รองรับการติดตั้งในเครื่อง PC เปิดหน้าต่างเบราว์เซอร์โดยอัตโนมัติเมื่อเริ่มต้นใช้งาน

  • รองรับการปรับขนาดหน้าต่างอัตโนมัติและรองรับการพิมพ์ภาษาจีน

  • รองรับการตั้งค่าความต้องการส่วนตัว ผู้ใช้สามารถเลือกสีพื้นหลัง/ตัวอักษร ขนาดฟอนต์ สไตล์การแสดงผลเคอร์เซอร์ เปิดใช้งาน TCode สำหรับเทอร์มินัล ฯลฯ

  • รองรับการรีสตาร์ท: ระบบจะรีสตาร์ทอัตโนมัติหลังจากแก้ไขการตั้งค่า SSH หรือการตั้งค่าความต้องการส่วนตัว หรือรีสตาร์ทเองได้หลังเชื่อมต่อ SSH ขาด

  • รองรับการคัดลอกและวาง:

  • คัดลอก: เหมือนเทอร์มินัล Git เลือกข้อความจะคัดลอกโดยอัตโนมัติ

  • วาง: เหมือนเทอร์มินัล Cmd คลิกขวาเพื่อวาง (ต้องอนุญาตการเข้าถึงของเบราว์เซอร์)

  • รองรับการจัดการไฟล์ เปิดโมดูล File Management เพื่อดู แตกไฟล์ อัปโหลดและดาวน์โหลดไฟล์/โฟลเดอร์

  • รองรับการใช้งานคีย์ลัด เช่น เลือกหลายไฟล์/ทั้งหมด คัดลอกและวาง ตัด สลับการเลือก เปิด ฯลฯ

  • รองรับการเรียกดูและแก้ไขไฟล์ แก้ไขไฟล์และบันทึกไปยังเซิร์ฟเวอร์ระยะไกลด้วย ctrl+s

  • รองรับ TCode (Terminal Code) ซึ่งสามารถสร้าง Workflow อัตโนมัติคล้ายกับ Shell script ด้วย TCode ที่กำหนดเอง

  • รองรับการบันทึกการทำงานและซิงค์ข้อมูลกับคลาวด์

  • รองรับฟังก์ชัน Cooperate, Monitor และ Docker

เรียนผู้ใช้ทุกท่าน

ขอแจ้งให้ทราบว่าเนื่องด้วยข้อจำกัดด้านเวลาและพลังงาน ผมจะไม่สามารถดูแล kkTerminal ต่อไปในอนาคต

หากคุณสนใจใน kkTerminal หรืออยากสานต่อการดูแล สามารถติดต่อผมหรือแจ้ง issue ได้

หากคุณพบข้อผิดพลาดหรืออยากพัฒนาฟีเจอร์ใหม่ กรุณาแจ้ง issue ได้เช่นกัน

สุดท้ายนี้ขอขอบคุณสำหรับการสนับสนุน kkTerminal และหวังว่า kkTerminal จะเป็นประโยชน์กับคุณจริง ๆ
