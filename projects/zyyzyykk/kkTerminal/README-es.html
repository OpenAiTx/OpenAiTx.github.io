🐳 Despliegue con Docker

  • Descargar la imagen:

docker pull zyyzyykk/kkterminal

  • Crear y ejecutar un contenedor para el mapeo de puertos: -p puerto:3000

docker run -d --name kkterminal -p 3000:3000 zyyzyykk/kkterminal

  • Personalizar palabra artística: -e TITLE="ArtWord"

docker run -d --name kkterminal -p 3000:3000 -e TITLE="kkbpro" zyyzyykk/kkterminal

  • Acceder desde el navegador: http://server-ip:3000/

💡 Descripción de funciones

  • kkTerminal es un terminal para conexión SSH vía Web. Haz clic en el icono del terminal en la esquina superior izquierda y selecciona la configuración de conexión para establecer una conexión SSH

  • Soporta el uso de etiquetas iframe en páginas web para una integración rápida en sitios web de terceros

  • Soporta la personalización de la configuración del terminal mediante Parámetros de URL

  • Soporta internacionalización y cambio de idioma entre chino e inglés

  • Soporta despliegue local en PC, abriendo automáticamente la ventana del navegador al iniciar

  • Soporta tamaño de ventana adaptable y entrada en chino

  • Soporta configuración personalizada de preferencias, permitiendo al usuario elegir el color de fondo/frente, tamaño de fuente, estilo de visualización del cursor, habilitar TCode para el terminal, etc.

  • Soporta reinicio: Se reiniciará automáticamente después de modificar la configuración de conexión SSH o las preferencias personalizadas, o se puede reiniciar manualmente tras desconectar la conexión SSH

  • Soporta copiar y pegar:

  • Copiar: Igual que el terminal de Git, al seleccionar texto se copia automáticamente

  • Pegar: Igual que el terminal de Cmd, clic derecho para pegar (requiere permiso de acceso del navegador)

  • Soporta gestión de archivos, abre el módulo de gestión de archivos para ver, descomprimir, subir y descargar archivos/carpetas

  • Soporta operaciones con teclas rápidas como selección múltiple/todos los archivos, copiar y pegar, cortar, cambiar selección, abrir, etc.

  • Soporta exploración y edición de archivos, modificar el archivo y guardarlo en el servidor remoto usando ctrl+s

  • Soporta TCode (Terminal Code), que permite lograr flujos de trabajo automatizados similares a scripts Shell mediante TCode personalizado

  • Soporta funciones de grabación de operaciones y sincronización en la nube

  • Soporta funciones de Cooperar, Monitor y Docker
