BiliAnalysis

Léger · Simple · Open source · Modulaire

Obtenez des liens directs pour les vidéos et les lives de Bilibili via un script Tampermonkey et une extension de navigateur.

Fonctionnalités

  • Prise en charge de l’analyse des vidéos et des salles de live
  • Propose une analyse locale et une analyse cloud
  • Menu contextuel pour copier rapidement les liens directs
  • Conception modulaire, installation à la demande

Choix de version

| Version | Méthode d’analyse | Description | | :--- | :--- | :--- | | BiliCloudAnalysis | Cloud (recommandé) | Stable et fiable, résout les problèmes d’analyse 1080P+ | | BiliAnalysis | Locale | Rapide, mais certaines vidéos HD peuvent échouer | | BiliRoomAnalysis | Locale | Spécifique aux salles de live | | Plugin menu contextuel | Extension navigateur | Comme la version cloud, sans Tampermonkey |

  • Téléchargez Plugin d'analyse clic droit.zip et décompressez-le
  • Activez le "Mode développeur" sur la page des extensions du navigateur

Résultat comme sur l’image


Mode d’emploi

Faites un clic droit sur le titre d’une vidéo sur le site Bilibili, puis sélectionnez "Analyse vidéo" pour obtenir le lien direct.

Remarque importante : L’analyse cloud dépend d’un serveur public, veuillez ne pas en abuser. Si vous l’utilisez dans un monde VRChat ou sur un site, conservez les informations de l’auteur original.

Projets pris en charge

Le service d’analyse cloud engendre des coûts de serveur, si vous trouvez cela utile, n’hésitez pas à soutenir par un don.
