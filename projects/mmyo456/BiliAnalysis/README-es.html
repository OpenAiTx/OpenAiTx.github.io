BiliAnalysis

Ligero · Sencillo · Código abierto · Modular

Obtén enlaces directos de videos y transmisiones en vivo de Bilibili mediante scripts de Tampermonkey y extensiones de navegador.

Características

  • Soporta análisis de videos y salas en vivo
  • Proporciona análisis local y en la nube
  • Menú contextual para copiar enlaces directos rápidamente
  • Diseño modular, instala según necesidad

Selección de versión

| Versión | Método de análisis | Descripción | | :--- | :--- | :--- | | BiliCloudAnalysis | En la nube (recomendado) | Estable y confiable, resuelve problemas con análisis 1080P+ | | BiliAnalysis | Local | Rápido, pero puede fallar con algunos videos HD | | BiliRoomAnalysis | Local | Exclusivo para salas en vivo | | Plugin de análisis con clic derecho | Extensión de navegador | Igual que la versión en la nube, sin necesidad de Tampermonkey |

Extensión del navegador

  • Descargue Plugin de análisis con clic derecho.zip y descomprímalo
  • En la página de extensiones del navegador, active el "modo desarrollador"
  • Cargue la extensión descomprimida
  • Consulte Tutorial de carga de extensiones en Edge

Cómo usar

Haga clic derecho en el título del video en el sitio web de Bilibili y seleccione "Análisis de video" para obtener el enlace directo.

Aviso importante: El análisis en la nube depende de servidores públicos, por favor no abuse. Si lo usa en mundos o sitios web de VRChat, mantenga la información del autor original.

Proyectos compatibles

El servicio de análisis en la nube implica costos de servidor; si le es útil, le invitamos a apoyar con una donación.
