BiliAnalysis

Lightweight · Simple · Open Source · Modular

Userscript and browser extension for obtaining direct links of Bilibili videos and live streams.

Features

  • Supports video and live room parsing
  • Offers local parsing and cloud parsing
  • Quick copy direct link from right-click menu
  • Modular design, install as needed

Version Selection

| Version | Parsing Method | Description | | :--- | :--- | :--- | | BiliCloudAnalysis | Cloud (Recommended) | Stable and reliable, solves 1080P+ parsing issues | | BiliAnalysis | Local | Fast, but some HD videos may fail | | BiliRoomAnalysis | Local | For live rooms only | | Right-click Parsing Plugin | Browser Extension | Same as cloud version, no userscript needed |

Browser Extension

  • Download RightClickParsingPlugin.zip and unzip it
  • Enable "Developer Mode" on the browser extensions page
  • Load the unpacked extension
  • See Edge Extension Loading Tutorial for details
Effect as shown


Usage

Right-click on the video title on the Bilibili website and select "Video Parsing" to get the direct link.

Important Note: Cloud parsing depends on public servers; please do not abuse it. When using in VRChat worlds or websites, please keep the original author information.
