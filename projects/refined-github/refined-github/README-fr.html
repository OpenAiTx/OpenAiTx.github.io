Refined GitHub

Extension de navigateur qui simplifie l’interface de GitHub et ajoute des fonctionnalités utiles

Nous utilisons beaucoup GitHub et remarquons de nombreuses petites gênes que nous aimerions corriger. Donc, ici, il y a des dragons.

Nous espérons que GitHub remarquera et implémentera certaines de ces améliorations tant attendues. Donc, si vous appréciez l’une de ces améliorations, veuillez ouvrir une discussion sur GitHub feedback ou contacter le support GitHub à ce sujet.

GitHub Enterprise est également pris en charge : Comment l’activer.

Les marques GITHUB et REFINED GITHUB sont la propriété de GitHub, Inc. et sont utilisées sous licence.

  • [](# "same-page-links") Garde les liens sur la même page au lieu d’ouvrir systématiquement dans un nouvel onglet. "Ouvrir dans un nouvel onglet" doit rester un choix utilisateur, pas le comportement par défaut.

Anciennement inclus dans Refined GitHub

GitHub s’est inspiré de Refined GitHub et a implémenté en natif des dizaines de ses fonctionnalités, 🎉 elles ont donc été retirées de cette extension. Vous pouvez aussi voir toutes les anciennes fonctionnalités de Refined GitHub dans une seule liste.

Personnalisation

La plupart des fonctionnalités peuvent être désactivées si elles sont basées sur JavaScript et vous pouvez surcharger notre CSS avec le vôtre dans les options de l’extension.

Nous sommes ravis de recevoir des suggestions et contributions, mais sachez que ce projet est très subjectif. La barre pour ajouter des fonctionnalités est haute. Les utilisateurs ne seront jamais tous d’accord. Cela étant dit, nous restons ouverts à la discussion.
