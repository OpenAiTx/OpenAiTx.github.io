Refined GitHub

Browserextensie die de GitHub-interface vereenvoudigt en nuttige functies toevoegt

Wij gebruiken GitHub veel en merken veel ergernissen op die we graag willen oplossen. Dus hier zijn draken.

We hopen dat GitHub sommige van deze broodnodige verbeteringen zal opmerken en implementeren. Dus als je een van deze verbeteringen waardeert, open dan een discussie op GitHub feedback of neem contact op met GitHub support om het voor elkaar te krijgen.

GitHub Enterprise wordt ook ondersteund: Hoe schakel je het in.

De handelsmerken GITHUB en REFINED GITHUB zijn eigendom van GitHub, Inc. en worden gebruikt onder licentie.

GitHub heeft inspiratie gehaald uit Refined GitHub en tientallen functies ervan native geïmplementeerd, 🎉 dus zijn ze uit deze extensie verwijderd. Je kunt ook alle voorgaande functies van Refined GitHub in één lijst bekijken.

Aanpassen

De meeste functies kunnen uitgeschakeld worden als ze JavaScript-gebaseerd zijn en je kunt onze CSS overschrijven met je eigen in de extensie-opties.

We ontvangen graag suggesties en bijdragen, maar wees je ervan bewust dat dit een sterk uitgesproken project is. Er is een hoge drempel voor het toevoegen van functies. Gebruikers zijn het altijd ergens niet mee eens. Dat gezegd hebbende, staan we open voor discussie.
