Refined GitHub

Браузерное расширение, которое упрощает интерфейс GitHub и добавляет полезные функции

Мы много работаем с GitHub и замечаем множество неудобств, которые хотим исправить. Так что берегитесь, здесь могут быть «драконы».

Мы надеемся, что GitHub обратит внимание и внедрит некоторые из этих столь необходимых улучшений. Если вам нравятся какие-либо из этих улучшений, пожалуйста, откройте обсуждение на GitHub feedback или свяжитесь с поддержкой GitHub, чтобы предложить их реализацию.

Также поддерживается GitHub Enterprise: Как включить.

Торговые марки GITHUB и REFINED GITHUB принадлежат GitHub, Inc. и используются по лицензии.

Кастомизация

Большинство функций можно отключить, если они реализованы на JavaScript, а также вы можете переопределить наш CSS своим в настройках расширения.

Мы всегда рады предложениям и контрибьюциям, но учтите, что это проект с выраженной позицией. Порог для добавления новых функций довольно высок. Пользователи всегда с чем-то не согласятся. Тем не менее, мы открыты для обсуждений.
