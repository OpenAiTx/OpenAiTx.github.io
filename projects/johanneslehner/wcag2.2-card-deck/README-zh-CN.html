WCAG 2.2 卡片组

最后更新：2026年1月25日（版本 1.2） 访问 WCAG 2.2 卡片组的 Figma 社区文件

这套卡片组是什么？

WCAG 2.2 卡片组起初是一个周末项目，旨在将多个优秀的无障碍资源整合成一个工具包：
  • 所有 WCAG 2.2 成功标准的摘要
  • 以易于理解的描述呈现
  • 按实用主题分组以便测试（基于 Andrew Hick 的“WCAG 2.2 主题地图”、“WCAG 2.2 主题地图 - AAA 版”和“WCAG 决策树”）
  • 包含满足每条标准受益对象的指示（基于 Charlie Triplett 的 Atomic A11Y、Appt.org 和 IBM 无障碍需求）
  • 突出主要负责的角色（基于 GOV.UK 的 WCAG 入门指南 及墨尔本大学的“项目角色的无障碍责任”清单）
  • 建议哪些成功标准组合使用效果最佳

如何使用？

使用 WCAG 2.2 卡片组开展研讨会、无障碍审查、培训，或任何你能用得上的场合。

接下来计划？

感谢 LinkedIn 上初次发布后收到的精彩反馈，我已有计划在未来版本中添加以下功能：
  • 更多语言（已计划：日语、阿拉伯语、瑞典语、挪威语、加泰罗尼亚语——更多语言陆续加入）
  • 添加方法描述（如何最大化利用卡片组，例如培训/学习或无障碍审查）
  • 添加可用于测试每条成功标准的工具（包括相关外部资源）
  • 包含可与卡片组配合使用的工具参考
  • WCAG 2.2 卡片组的无障碍 HTML 版本（托管于公共 GitHub 仓库；采用 CC BY-SA 4.0 许可）
  • WCAG 卡片组海报
  • “WCAG 2.2 主题地图”海报（将卡片可视化为地铁线路）
  • 每条成功标准的好/坏示例（支持自主改进或辅导式培训）
  • 关于忽视标准会造成哪些障碍的信息
  • … 以及更多
