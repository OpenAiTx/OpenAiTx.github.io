##### 목차로 돌아가기

설치

macOS 또는 Linux에서는 [Homebrew][homebrew]를 이용해 The Fuck을 설치할 수 있습니다:

brew install thefuck

Ubuntu / Mint에서는 다음 명령어로 The Fuck을 설치하세요: 

sudo apt update
sudo apt install python3-dev python3-pip python3-setuptools
pip3 install thefuck --user

FreeBSD에서는 다음 명령어로 The Fuck을 설치하세요: 

pkg install thefuck

ChromeOS에서는 chromebrew를 사용하여 설치할 수 있습니다: 

crew install thefuck

Arch 기반 시스템에서는 다음 명령어로 설치하세요: 

sudo pacman -S thefuck

기타 시스템에서는 pip을 이용해 The Fuck을 설치하세요:

pip install thefuck

또는 OS 패키지 매니저(OS X, Ubuntu, Arch)를 사용할 수도 있습니다.

# 아래 명령어를 .bash_profile, .bashrc, .zshrc 또는 기타 셸 시작 스크립트에 추가하는 것이 권장됩니다:

eval $(thefuck --alias)

원하는 별칭을 사용할 수 있습니다. 예: 월요일에는 이렇게:

eval $(thefuck --alias FUCK)

또는 Bash, Zsh, Fish, Powershell, tcsh 등 셸 설정 파일에 추가하세요.

변경 사항은 새 셸 세션에서만 적용됩니다. 즉시 적용하려면 source ~/.bashrc(또는 .zshrc 등) 명령어를 실행하세요.

확인 없이 수정된 명령어를 실행하려면 --yeah 옵션(또는 짧게 -y, 또는 특별히 화가 났다면 --hard)을 사용하세요:

fuck --yeah

성공할 때까지 명령어를 재귀적으로 수정하려면 -r 옵션을 사용하세요:

fuck -r

##### 목차로 돌아가기

업데이트

pip3 install thefuck --upgrade

참고: The Fuck v1.34부터 별칭 기능이 변경되었습니다.

삭제

The Fuck을 제거하려면 설치 과정을 반대로 진행하세요:

  • Bash, Zsh, Fish, Powershell, tcsh 등 셸 설정 파일에서 thefuck 별칭 줄을 삭제하거나 주석 처리
  • 패키지 관리자(brew, pip3, pkg, crew, pip 등)로 바이너리 삭제
