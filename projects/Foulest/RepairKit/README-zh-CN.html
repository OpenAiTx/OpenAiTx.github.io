RepairKit

RepairKit 是一个基于Java的全能Windows修复和维护工具包。

安装与设置指南

功能

自动修复

自动执行全面的系统清理和修复，包括：

  • 创建系统还原点
  • 删除恶意系统策略
  • 运行注册表和系统调整
  • 移除易受攻击的Windows功能
  • 应用最佳的Windows Defender设置
  • 清理难以发现的垃圾文件
  • 移除预装的臃肿软件
  • 修复磁盘问题和损坏的Windows文件
  • 更新过时的已安装程序
修复中的每个功能都可完全自定义，允许您根据需要启用或禁用特定的修复，方法是使用面板上的复选框，或位于 config 文件夹中的配置文件。只有在知道自己在做什么时才修改这些文件。

注意： 之前提供的 移除恶意软件 选项现已移除。运行 Windows Defender 调优 现在改为应用最佳的 Windows Defender 设置。手动扫描选项可通过“实用程序”面板中的 Emsisoft 或 Sophos 扫描选项使用。

实用程序

访问系统维护的基本软件工具，包括：

  • CPU-Z：识别系统硬件详情。
  • HWMonitor：监控硬件温度和规格。
  • Emsisoft Scan：使用 Emsisoft 紧急工具扫描恶意软件。
  • Sophos Scan：使用 Sophos 扫描与清理扫描恶意软件。
  • TreeSize：分析和管理磁盘内容。
  • Everything：极速文件搜索引擎。
  • CrystalDiskInfo：监控硬盘驱动器的健康状态。
  • CrystalDiskMark：测试硬盘驱动器的读写速度。
  • Autoruns：Windows 启动管理器的替代工具。
  • Process Explorer：Windows 任务管理器的替代工具。
  • Process Monitor：监控系统活动和进程。
  • BlueScreenView：查看和分析 Windows 蓝屏死机转储。
  • NVCleanstall：轻量级 NVIDIA 显卡驱动更新工具。
  • DisplayDriverUninstaller：移除显示驱动及相关包。
  • Recuva：已删除文件的数据恢复工具。
  • DiskGenius：磁盘分区管理与数据恢复工具。

此外，RepairKit 提供了有用的软件工具链接，包括：

  • 7-Zip：开源文件归档工具链接。
  • Bitwarden：开源密码管理器链接。
  • Sophos Home：Sophos Home 杀毒软件链接。
  • uBlock Origin：强力广告拦截浏览器扩展链接。
  • Osprey：浏览器保护扩展链接。
  • Notepad++：开源文本编辑器链接。
  • Twinkle Tray：显示器亮度控制软件链接。
  • FanControl：风扇速度控制软件链接。

下载与运行

不确定如何下载和运行 RepairKit？请按照以下步骤操作：

  • 从发布页面下载最新版本的 RepairKit（点击 RepairKit-X.X.X.zip 文件）。
  • 将 ZIP 文件解压到电脑上的一个文件夹。你可以使用类似 7-Zip 的程序进行解压。
  • 双击 RepairKit-X.X.X.exe 文件运行程序。
  • 如果 Windows Defender SmartScreen 阻止应用，点击“更多信息”，然后选择“仍然运行”。
  • 在用户账户控制提示中点击“是”。

RepairKit 现在将打开，你可以开始使用其功能。

编译

  • 克隆仓库。
  • 打开命令提示符/终端，进入仓库目录。
  • 在 Windows 上运行 gradlew createExe，在 macOS 或 Linux 上运行 ./gradlew createExe。
  • 生成的 RepairKit-X.X.X.zip 文件将在 build 文件夹中。
