安装与设置指南

功能特色

自动修复

自动执行全面的系统清理与修复操作，包括：

  • 创建系统还原点
  • 删除恶意系统策略
  • 运行注册表和系统优化
  • 移除易受攻击的Windows功能
  • 应用最佳的Windows Defender设置
  • 清理难以发现的垃圾文件
  • 移除预装的臃肿软件
  • 修复磁盘问题及损坏的Windows文件
  • 更新过时的已安装程序

修复中的每一项功能都可以完全自定义，您可以通过面板上的复选框或config文件夹中的配置文件 选择启用或禁用特定修复项。仅当您了解相关内容时再修改这些文件。

注意： 先前提供的移除恶意软件选项已被移除。运行Windows Defender优化现在
应用最佳的Windows Defender设置。您也可以在实用程序面板中通过Emsisoft或Sophos
扫描选项手动进行查杀。

实用程序

访问系统维护所需的核心软件工具，包括：

  • CPU-Z：识别您的系统硬件详细信息。
  • HWMonitor：监控硬件温度与规格。
  • Emsisoft Scan：使用 Emsisoft Emergency Kit 扫描恶意软件。
  • Sophos Scan：使用 Sophos Scan & Clean 扫描恶意软件。
  • TreeSize：分析并管理磁盘内容。
  • Everything：超快的文件搜索引擎。
  • CrystalDiskInfo：监控您的磁盘驱动器健康和状态。
  • CrystalDiskMark：测试磁盘驱动器的读写速度。
  • Autoruns：Windows 启动管理器的替代方案。
  • Process Explorer：Windows 任务管理器的替代方案。
  • Process Monitor：监控系统活动与进程。
  • BlueScreenView：查看和分析 Windows 蓝屏崩溃转储。
  • NVCleanstall：轻量级 NVIDIA 显卡驱动更新工具。
  • DisplayDriverUninstaller：卸载显示驱动及相关包。
  • Recuva：用于恢复已删除文件的数据恢复工具。
  • DiskGenius：磁盘分区管理与数据恢复工具。

此外，RepairKit 提供了实用软件工具的链接，包括：

  • 7-Zip：开源文件压缩工具的链接。
  • Bitwarden：开源密码管理器的链接。
  • Sophos Home：Sophos Home 杀毒软件的链接。
  • uBlock Origin：强力广告拦截浏览器扩展的链接。
  • Osprey：浏览器保护扩展的链接。
  • Notepad++：开源文本编辑器的链接。
  • Twinkle Tray：显示器亮度调节软件的链接。

  • FanControl：风扇速度控制软件链接。

下载与运行

不知道如何下载和运行 RepairKit？请按照以下步骤操作：

  • 从 Releases 页面 下载最新版本的 RepairKit（点击 RepairKit-X.X.X.zip 文件）。
  • 将 ZIP 文件解压到电脑上的某个文件夹。你可以使用 7-Zip 等程序来解压内容。
  • 双击 RepairKit-X.X.X.exe 文件运行程序。
  • 如果 Windows Defender SmartScreen 阻止了该应用，请点击 更多信息，然后点击 仍然运行。
  • 在用户账户控制提示中点击 是。

RepairKit 现在会打开，你可以开始使用它的功能了。

编译

  • 克隆该仓库。
  • 打开命令提示符/终端，定位到仓库目录。
  • 在 Windows 上运行 gradlew createExe，或在 macOS 或 Linux 上运行 ./gradlew createExe。
  • 构建好的 RepairKit-X.X.X.zip 文件将在 build 文件夹中。
