RepairKit

インストール & セットアップガイド

機能

自動修復

包括的なシステムクリーンアップと修復を自動的に実行します。内容は以下の通りです：

  • システムの復元ポイントの作成
  • 悪意のあるシステムポリシーの削除
  • レジストリとシステムの調整の実行
  • 脆弱なWindows機能の削除
  • 最適なWindows Defenderの設定適用
  • 見つけにくいジャンクファイルのクリーンアップ
  • プレインストールされた不要ソフトの削除
  • ディスクの問題と破損したWindowsファイルの修復
  • 古くなったインストール済みプログラムの更新
修復の各機能は完全にカスタマイズ可能で、パネルのチェックボックスまたはconfigフォルダー内の設定ファイルを使って必要に応じて特定の修復を有効または無効にできます。これらのファイルは、内容を理解している場合のみ変更してください。

注意： 以前利用可能だったマルウェア除去オプションは削除されました。代わりにWindows Defenderの調整を実行が最適なWindows Defenderの設定を適用します。手動スキャンはUseful ProgramsパネルのEmsisoftまたはSophosスキャンオプションで利用可能です。

Useful Programs（便利なプログラム）

システムメンテナンスに必要なソフトウェアツールにアクセスできます：

  • CPU-Z：システムのハードウェア情報を特定します。
  • HWMonitor：ハードウェアの温度や仕様を監視します。
  • Emsisoft Scan：Emsisoft Emergency Kitを使ってマルウェアをスキャンします。
  • Sophos Scan：Sophos Scan & Cleanを使ってマルウェアをスキャンします。
  • TreeSize：ディスク内容の分析と管理を行います。
  • Everything：超高速のファイル検索エンジンです。
  • CrystalDiskInfo：ディスクドライブの健康状態とステータスを監視します。
  • CrystalDiskMark：ディスクドライブの読み書き速度をベンチマークします。
  • Autoruns：Windowsのスタートアップマネージャーの代替ツールです。
  • Process Explorer：Windowsタスクマネージャーの代替ツールです。
  • Process Monitor：システム活動とプロセスを監視します。
  • BlueScreenView：WindowsのBSODクラッシュダンプを表示・分析します。
  • NVCleanstall：軽量のNVIDIAグラフィックスカードドライバーアップデーターです。
  • DisplayDriverUninstaller：ディスプレイドライバーとパッケージを削除します。
  • Recuva：削除されたファイルのデータ復旧ツールです。
  • DiskGenius：ディスクパーティション管理とデータ復旧ツールです。

さらに、RepairKit は以下の便利なソフトウェアツールへのリンクも提供しています：

  • 7-Zip：オープンソースのファイルアーカイバへのリンク。
  • Bitwarden：オープンソースのパスワードマネージャーへのリンク。
  • Sophos Home：Sophos Home ウイルス対策ソフトウェアへのリンク。
  • uBlock Origin：強力な広告ブロックブラウザ拡張機能へのリンク。
  • Osprey：ブラウザ保護拡張機能へのリンク。
  • Notepad++：オープンソースのテキストエディタへのリンク。
  • Twinkle Tray：モニターの明るさ調整ソフトウェアへのリンク。
  • FanControl：ファンスピード制御ソフトウェアへのリンク。

ダウンロードと実行

RepairKit のダウンロードと実行方法がわからない場合は、以下の手順に従ってください：

  • 最新版の RepairKit を
Releases ページ からダウンロードします（RepairKit-X.X.X.zip ファイルをクリック）。
  • ZIP ファイルをコンピュータのフォルダーに解凍します。7-Zip などのプログラムを使うと解凍が簡単です。
  • RepairKit-X.X.X.exe ファイルをダブルクリックしてプログラムを実行します。
  • Windows Defender SmartScreen がアプリをブロックした場合は、「詳細情報」をクリックしてから「実行」を選択します。
  • ユーザーアカウント制御のプロンプトで「はい」をクリックします。

これで RepairKit が起動し、機能を使い始めることができます。

コンパイル

  • リポジトリをクローンします。
  • コマンドプロンプト／ターミナルをリポジトリのディレクトリで開きます。
  • Windowsでは gradlew createExe を、macOSまたはLinuxでは ./gradlew createExe を実行します。
  • ビルドされた RepairKit-X.X.X.zip ファイルは build フォルダにあります。
