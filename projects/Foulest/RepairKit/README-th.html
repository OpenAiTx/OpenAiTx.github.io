คุณสมบัติ

การซ่อมแซมอัตโนมัติ

ดำเนินการล้างระบบและซ่อมแซมอย่างครบถ้วนโดยอัตโนมัติ รวมถึง:

  • สร้างจุดคืนค่าระบบ (Restore Point)
  • ลบนโยบายระบบที่เป็นอันตราย
  • ปรับแต่งรีจิสทรีและระบบ
  • ลบฟีเจอร์ Windows ที่มีความเสี่ยง
  • ปรับตั้งค่าสำหรับ Windows Defender ให้เหมาะสม
  • ลบไฟล์ขยะที่หาเจอยาก
  • ลบซอฟต์แวร์ที่มาพร้อมเครื่องที่ไม่ต้องการ
  • ซ่อมแซมปัญหาดิสก์และไฟล์ Windows ที่เสียหาย
  • อัปเดตโปรแกรมที่ติดตั้งที่ล้าสมัย

ทุกฟังก์ชันในการซ่อมแซมสามารถปรับแต่งได้อย่างเต็มที่ คุณสามารถเปิดหรือปิดแต่ละรายการซ่อมแซมตามต้องการโดยใช้ ช่องทำเครื่องหมายบนแผงควบคุม หรือไฟล์การกำหนดค่าในโฟลเดอร์ config โปรดแก้ไขไฟล์เหล่านี้เฉพาะเมื่อคุณมั่นใจในสิ่งที่ทำ

หมายเหตุ: ตัวเลือก Remove Malware ที่เคยมีได้ถูกนำออกแล้ว ขณะนี้ Run Windows Defender Tweaks
จะใช้การตั้งค่าสำหรับ Windows Defender ที่เหมาะสมแทน คุณสามารถสแกนด้วยตนเองผ่านโปรแกรม Emsisoft หรือ Sophos
ได้ที่ตัวเลือก Scan ในแผงโปรแกรมที่มีประโยชน์

โปรแกรมที่มีประโยชน์

เข้าถึงเครื่องมือซอฟต์แวร์สำคัญสำหรับการบำรุงรักษาระบบ รวมถึง:

  • CPU-Z: ตรวจสอบรายละเอียดฮาร์ดแวร์ของระบบคุณ
  • HWMonitor: ตรวจสอบอุณหภูมิและข้อมูลฮาร์ดแวร์
  • Emsisoft Scan: สแกนมัลแวร์ด้วย Emsisoft Emergency Kit
  • Sophos Scan: สแกนมัลแวร์ด้วย Sophos Scan & Clean
  • TreeSize: วิเคราะห์และจัดการเนื้อหาในดิสก์
  • Everything: เครื่องมือค้นหาไฟล์ที่รวดเร็วมาก
  • CrystalDiskInfo: ตรวจสอบสุขภาพและสถานะฮาร์ดดิสก์ของคุณ
  • CrystalDiskMark: ทดสอบความเร็วอ่าน/เขียนของฮาร์ดดิสก์
  • Autoruns: ทางเลือกสำหรับตัวจัดการเริ่มต้นระบบ Windows
  • Process Explorer: ทางเลือกสำหรับตัวจัดการงาน Windows
  • Process Monitor: ตรวจสอบกิจกรรมและกระบวนการของระบบ
  • BlueScreenView: ดูและวิเคราะห์ไฟล์ BSOD crash dumps ของ Windows
  • NVCleanstall: ตัวอัปเดตไดรเวอร์การ์ดจอ NVIDIA แบบน้ำหนักเบา
  • DisplayDriverUninstaller: ลบไดรเวอร์จอและแพ็คเกจออก
  • Recuva: เครื่องมือกู้คืนข้อมูลสำหรับไฟล์ที่ถูกลบ
  • DiskGenius: ตัวจัดการพาร์ติชันและกู้คืนข้อมูลในดิสก์

นอกจากนี้ RepairKit ยังมีลิงก์ไปยังเครื่องมือซอฟต์แวร์ที่มีประโยชน์ รวมถึง:

  • 7-Zip: ลิงก์ไปยังโปรแกรมบีบอัดไฟล์แบบโอเพ่นซอร์ส
  • Bitwarden: ลิงก์ไปยังตัวจัดการรหัสผ่านแบบโอเพ่นซอร์ส
  • Sophos Home: ลิงก์ไปยังซอฟต์แวร์ป้องกันไวรัส Sophos Home
  • uBlock Origin: ลิงก์ไปยังส่วนขยายบล็อกโฆษณาที่ทรงพลังสำหรับเบราว์เซอร์
  • Osprey: ลิงก์ไปยังส่วนขยายปกป้องเบราว์เซอร์
  • Notepad++: ลิงก์ไปยังโปรแกรมแก้ไขข้อความแบบโอเพ่นซอร์ส
  • Twinkle Tray: ลิงก์ไปยังซอฟต์แวร์ควบคุมความสว่างของหน้าจอ

  • FanControl: ลิงก์ไปยังซอฟต์แวร์ควบคุมความเร็วพัดลม

ทางลัดระบบ

เข้าถึงยูทิลิตี้สำคัญของ Windows ได้อย่างรวดเร็ว เช่น:

  • แอป & ฟีเจอร์
  • แอปเริ่มต้น
  • Windows Update
  • Windows Security
  • ตัวจัดการงาน
  • และอื่นๆ...

ดาวน์โหลดและเรียกใช้

ไม่แน่ใจว่าดาวน์โหลดและเรียกใช้ RepairKit อย่างไร? ทำตามขั้นตอนเหล่านี้:

  • ดาวน์โหลดเวอร์ชันล่าสุดของ RepairKit จาก
หน้า Releases (คลิกไฟล์ RepairKit-X.X.X.zip)
  • แตกไฟล์ ZIP ไปยังโฟลเดอร์บนคอมพิวเตอร์ของคุณ คุณสามารถใช้โปรแกรมอย่าง 7-Zip เพื่อแตกไฟล์
  • ดับเบิลคลิกที่ไฟล์ RepairKit-X.X.X.exe เพื่อเรียกใช้โปรแกรม
  • หาก Windows Defender SmartScreen บล็อกแอป ให้คลิก ข้อมูลเพิ่มเติม แล้วคลิก เรียกใช้ต่อไป
  • คลิก ใช่ ที่พรอมต์ User Account Control

ขณะนี้ RepairKit จะเปิดและคุณสามารถเริ่มใช้งานฟีเจอร์ต่างๆ ได้

การคอมไพล์

  • โคลนรีโพซิทอรี
  • เปิดพรอมต์คำสั่ง/เทอร์มินัลที่ไดเรกทอรีของรีโพซิทอรี
  • รัน gradlew createExe บน Windows หรือ ./gradlew createExe บน macOS หรือ Linux
  • ไฟล์ RepairKit-X.X.X.zip ที่สร้างเสร็จจะอยู่ในโฟลเดอร์ build
