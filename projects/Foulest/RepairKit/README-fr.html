Guide d'installation et de configuration

Fonctionnalités

Réparations automatiques

Effectue automatiquement un nettoyage et une réparation complète du système, incluant :

  • Création d’un point de restauration système
  • Suppression des politiques système malveillantes
  • Exécution d’optimisations du registre et du système
  • Suppression des fonctionnalités Windows vulnérables
  • Application des paramètres optimaux de Windows Defender
  • Nettoyage des fichiers indésirables difficiles à trouver
  • Suppression des logiciels préinstallés inutiles
  • Réparation des problèmes de disque et des fichiers Windows corrompus
  • Mise à jour des programmes installés obsolètes

Chaque fonction de la réparation est entièrement personnalisable, vous permettant d’activer ou de désactiver des réparations spécifiques selon vos besoins en utilisant soit les cases à cocher sur le panneau, soit les fichiers de configuration situés dans le dossier config. Modifiez ces fichiers uniquement si vous savez ce que vous faites.

Remarque : L’option Supprimer les logiciels malveillants précédemment disponible a été supprimée. Exécuter les réglages de Windows Defender applique désormais
les paramètres optimaux de Windows Defender à la place. Des options d’analyse manuelle sont disponibles via les options de scan Emsisoft ou Sophos
dans le panneau Programmes utiles.

Programmes utiles

Accédez aux outils logiciels essentiels pour la maintenance du système, notamment :

  • CPU-Z : Identifiez les détails matériels de votre système.
  • HWMonitor : Surveillez les températures et spécifications du matériel.
  • Emsisoft Scan : Analysez les logiciels malveillants avec Emsisoft Emergency Kit.
  • Sophos Scan : Analysez les logiciels malveillants avec Sophos Scan & Clean.
  • TreeSize : Analysez et gérez le contenu des disques.
  • Everything : Un moteur de recherche de fichiers ultra-rapide.
  • CrystalDiskInfo : Surveillez la santé et le statut de votre disque dur.
  • CrystalDiskMark : Testez les vitesses de lecture/écriture de votre disque dur.
  • Autoruns : Une alternative au gestionnaire de démarrage Windows.
  • Process Explorer : Une alternative au gestionnaire des tâches Windows.
  • Process Monitor : Surveillez l’activité système et les processus.
  • BlueScreenView : Visualisez et analysez les vidages mémoire des BSOD Windows.
  • NVCleanstall : Un utilitaire léger de mise à jour des pilotes de cartes graphiques NVIDIA.
  • DisplayDriverUninstaller : Supprimez les pilotes et packages d’affichage.
  • Recuva : Un outil de récupération de données pour fichiers supprimés.
  • DiskGenius : Un gestionnaire de partitions de disque et outil de récupération de données.

De plus, RepairKit fournit des liens vers des outils logiciels utiles, notamment :

  • 7-Zip : Lien vers l’archiver de fichiers open-source.
  • Bitwarden : Lien vers le gestionnaire de mots de passe open-source.
  • Sophos Home : Lien vers le logiciel antivirus Sophos Home.
  • uBlock Origin : Lien vers l’extension de navigateur puissante pour bloquer les publicités.
  • Osprey : Lien vers l’extension de protection du navigateur.
  • Notepad++ : Lien vers l’éditeur de texte open-source.
  • Twinkle Tray : Lien vers le logiciel de contrôle de la luminosité des moniteurs.
  • FanControl : Lien vers le logiciel de contrôle de la vitesse des ventilateurs.

Télécharger et Exécuter

Vous ne savez pas comment télécharger et exécuter RepairKit ? Suivez ces étapes :

  • Téléchargez la dernière version de RepairKit depuis
la page des Releases (cliquez sur le fichier RepairKit-X.X.X.zip).
  • Extrayez le fichier ZIP dans un dossier sur votre ordinateur. Vous pouvez utiliser un programme comme 7-Zip pour extraire
le contenu.
  • Double-cliquez sur le fichier RepairKit-X.X.X.exe pour lancer le programme.
  • Si Windows Defender SmartScreen bloque l’application, cliquez sur Plus d’informations puis sur Exécuter quand même.
  • Cliquez sur Oui dans la fenêtre de contrôle de compte utilisateur.

RepairKit va maintenant s’ouvrir, et vous pouvez commencer à utiliser ses fonctionnalités.

Compilation

  • Clonez le dépôt.
  • Ouvrez une invite de commande/terminal dans le répertoire du dépôt.
  • Exécutez gradlew createExe sous Windows, ou ./gradlew createExe sous macOS ou Linux.
  • Le fichier RepairKit-X.X.X.zip généré se trouvera dans le dossier build.
