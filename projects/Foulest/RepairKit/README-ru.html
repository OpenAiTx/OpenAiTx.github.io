Руководство по установке и настройке

Возможности

Автоматический ремонт

Автоматически выполняет комплексную очистку и восстановление системы, включая:

  • Создание точки восстановления системы
  • Удаление вредоносных политик системы
  • Запуск твиков реестра и системы
  • Удаление уязвимых компонентов Windows
  • Применение оптимальных настроек Windows Defender
  • Очистка труднонаходимых временных файлов
  • Удаление предустановленного мусора
  • Исправление ошибок диска и поврежденных файлов Windows
  • Обновление устаревших установленных программ

Каждая функция ремонта полностью настраиваема, позволяя включать или отключать конкретные операции с помощью чекбоксов на панели или конфигурационных файлов в папке config. Изменяйте эти файлы только если точно знаете, что делаете.

Примечание: Ранее доступная опция Удалить вредоносное ПО была удалена. Теперь Твики Windows Defender
применяют оптимальные параметры Windows Defender. Опции ручного сканирования доступны через Emsisoft или Sophos
в панели "Полезные программы".

Полезные программы

Получите доступ к необходимым программам для обслуживания системы, включая:

  • CPU-Z: Определите характеристики аппаратного обеспечения вашей системы.
  • HWMonitor: Следите за температурой и характеристиками железа.
  • Emsisoft Scan: Сканируйте систему на вредоносное ПО с помощью Emsisoft Emergency Kit.
  • Sophos Scan: Сканируйте систему на вредоносное ПО с помощью Sophos Scan & Clean.
  • TreeSize: Анализируйте и управляйте содержимым диска.
  • Everything: Сверхбыстрый поисковик файлов.
  • CrystalDiskInfo: Контролируйте состояние и здоровье накопителя.
  • CrystalDiskMark: Тестируйте скорость чтения/записи
вашего диска.
  • Autoruns: Альтернатива Диспетчеру автозагрузки Windows.
  • Process Explorer: Альтернатива
Диспетчеру задач Windows.
  • Process Monitor: Мониторинг активности системы и
процессов.
  • BlueScreenView: Просмотр и анализ дампов аварийных синих экранов Windows.
  • NVCleanstall: Легковесная утилита обновления драйверов NVIDIA.
  • DisplayDriverUninstaller: Удаление
драйверов и пакетов видеокарт.
  • Recuva: Программа для восстановления удалённых файлов.
  • DiskGenius: Менеджер разделов и инструмент восстановления данных.

Кроме того, RepairKit предоставляет ссылки на полезные программы, включая:

  • 7-Zip: Ссылка на открытый архиватор файлов.
  • Bitwarden: Ссылка на открытый менеджер паролей.
  • Sophos Home: Ссылка на антивирус Sophos Home.
  • uBlock Origin: Ссылка на мощное расширение-блокировщик рекламы.
  • Osprey: Ссылка на расширение для защиты браузера.
  • Notepad++: Ссылка на открытый текстовый редактор.
  • Twinkle Tray: Ссылка на программу управления яркостью монитора.

  • FanControl: Ссылка на программу для управления скоростью вентиляторов.

Загрузка и запуск

Не знаете, как скачать и запустить RepairKit? Следуйте этим шагам:

  • Скачайте последнюю версию RepairKit с
страницы релизов (нажмите на файл RepairKit-X.X.X.zip).
  • Извлеките ZIP-файл в папку на вашем компьютере. Для извлечения можно использовать программу 7-Zip.
  • Дважды кликните на файл RepairKit-X.X.X.exe, чтобы запустить программу.
  • Если Windows Defender SmartScreen блокирует приложение, нажмите Подробнее, затем Выполнить в любом случае.
  • Нажмите Да в запросе контроля учетных записей.

RepairKit теперь откроется, и вы можете начать использовать его функции.

Сборка

  • Клонируйте репозиторий.
  • Откройте командную строку/терминал в каталоге репозитория.
  • Выполните gradlew createExe в Windows или ./gradlew createExe в macOS или Linux.
  • Готовый файл RepairKit-X.X.X.zip будет в папке build.
