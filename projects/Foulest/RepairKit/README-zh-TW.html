安裝與設置指南

功能介紹

自動修復

自動執行全面的系統清理與修復，包括：

  • 建立系統還原點
  • 刪除惡意系統政策
  • 執行登錄檔與系統優化
  • 移除高風險的 Windows 功能
  • 套用最佳化的 Windows Defender 設定
  • 清理難以發現的垃圾檔案
  • 移除預載的臃腫軟體
  • 修復磁碟問題與損壞的 Windows 檔案
  • 更新過時的已安裝程式

修復中的每一個功能都可以完全自訂，您可以透過面板上的勾選框或位於 config 資料夾中的設定檔 啟用或停用特定修復項目。僅在您確定操作時才修改這些檔案。

注意： 先前提供的 移除惡意軟體 選項已經移除。現在的 執行 Windows Defender 優化
會套用最佳的 Windows Defender 設定。您可於實用程式面板中，使用 Emsisoft 或 Sophos
掃描選項執行手動掃描。

實用程式

存取系統維護必備的軟體工具，包括：

  • CPU-Z：識別您的系統硬體詳細資訊。
  • HWMonitor：監控硬體溫度及規格。
  • Emsisoft Scan：使用 Emsisoft Emergency Kit 掃描惡意軟體。
  • Sophos Scan：使用 Sophos Scan & Clean 掃描惡意軟體。
  • TreeSize：分析並管理磁碟內容。
  • Everything：超高速檔案搜尋引擎。
  • CrystalDiskInfo：監控您的硬碟健康狀態。
  • CrystalDiskMark：測試硬碟讀寫
速度。
  • Autoruns：Windows 啟動管理器替代品。
  • Process Explorer：Windows 工作管理員替代品。
  • Process Monitor：監控系統活動及
程序。
  • BlueScreenView：查看與分析 Windows BSOD 當機檔案。
  • NVCleanstall：輕量級 NVIDIA 顯示卡
驅動程式更新器。
  • DisplayDriverUninstaller：移除
顯示驅動程式及套件。
  • Recuva：刪除檔案資料救援工具。
  • DiskGenius：磁碟分割管理與資料救援工具。

此外，RepairKit 提供實用軟體工具連結，包括：

  • 7-Zip：開源檔案壓縮工具連結。
  • Bitwarden：開源密碼管理器連結。
  • Sophos Home：Sophos Home 防毒軟體連結。
  • uBlock Origin：強大廣告阻擋瀏覽器擴充連結。
  • Osprey：瀏覽器防護擴充連結。
  • Notepad++：開源文字編輯器連結。
  • Twinkle Tray：螢幕亮度控制軟體連結。

  • FanControl：風扇轉速控制軟體連結。

下載與執行

不確定如何下載並執行 RepairKit？請依照以下步驟：

  • 從發行頁面下載最新版本的 RepairKit（點擊 RepairKit-X.X.X.zip 檔案）。
  • 將 ZIP 壓縮檔解壓縮到電腦上的資料夾。你可以使用 7-Zip 等程式來解壓縮內容。
  • 雙擊 RepairKit-X.X.X.exe 檔案以執行程式。
  • 若 Windows Defender SmartScreen 阻擋此應用程式，請點擊「更多資訊」再選擇「仍要執行」。
  • 在使用者帳戶控制提示時點選「是」。

RepairKit 現已開啟，您可以開始使用其功能。

編譯說明

  • 複製（clone）此儲存庫。
  • 開啟命令提示字元或終端機，切換到儲存庫目錄。
  • 在 Windows 執行 gradlew createExe，或在 macOS/Linux 執行 ./gradlew createExe。
  • 編譯後的 RepairKit-X.X.X.zip 檔案將會在 build 資料夾中。
