Installations- & Einrichtungsanleitung

Funktionen

Automatische Reparaturen

Führt automatisch eine umfassende Systembereinigung und Reparatur durch, einschließlich:

  • Erstellen eines Systemwiederherstellungspunkts
  • Löschen schädlicher Systemrichtlinien
  • Ausführen von Registry- und Systemoptimierungen
  • Entfernen anfälliger Windows-Funktionen
  • Anwenden optimaler Windows Defender-Einstellungen
  • Bereinigen schwer auffindbarer Junk-Dateien
  • Entfernen vorinstallierter Bloatware
  • Reparieren von Festplattenproblemen und beschädigten Windows-Dateien
  • Aktualisieren veralteter installierter Programme

Jede Funktion der Reparatur ist vollständig anpassbar, sodass Sie bestimmte Reparaturen nach Bedarf entweder über die Kontrollkästchen im Panel oder die Konfigurationsdateien im config-Ordner aktivieren oder deaktivieren können. Ändern Sie diese Dateien nur, wenn Sie wissen, was Sie tun.

Hinweis: Die zuvor verfügbare Option Remove Malware wurde entfernt. Windows Defender Tweaks ausführen wendet nun
stattdessen optimale Windows Defender-Einstellungen an. Manuelle Scan-Optionen sind über die Emsisoft- oder Sophos-
Scan-Optionen im Bereich Nützliche Programme verfügbar.

Nützliche Programme

Zugriff auf wichtige Software-Tools für Systemwartung, darunter:

  • CPU-Z: Erkennen Sie Details Ihrer Systemhardware.
  • HWMonitor: Überwachen Sie Hardware-Temperaturen und Spezifikationen.
  • Emsisoft Scan: Malware-Scan mit dem Emsisoft Emergency Kit.
  • Sophos Scan: Malware-Scan mit Sophos Scan & Clean.
  • TreeSize: Analysieren und verwalten Sie Festplatteninhalte.
  • Everything: Eine ultraschnelle Dateisuche.
  • CrystalDiskInfo: Überwachen Sie den Zustand und Status Ihrer Festplatte.
  • CrystalDiskMark: Benchmark Ihrer Festplatte für Lese-/Schreibgeschwindigkeiten.
  • Autoruns: Eine Alternative zum Windows-Startmanager.
  • Process Explorer: Eine Alternative zum Windows Task-Manager.
  • Process Monitor: Überwachen Sie Systemaktivitäten und Prozesse.
  • BlueScreenView: Anzeigen und Analysieren von Windows BSOD Crash-Dumps.
  • NVCleanstall: Ein leichtgewichtiger NVIDIA-Grafiktreiber-Updater.
  • DisplayDriverUninstaller: Entfernen von Display-Treibern und -Paketen.
  • Recuva: Ein Tool zur Wiederherstellung gelöschter Dateien.
  • DiskGenius: Ein Festplatten-Partitionsmanager und Tool zur Datenwiederherstellung.

Zusätzlich bietet RepairKit Links zu nützlichen Software-Tools, darunter:

  • 7-Zip: Link zum quelloffenen Dateiarchivierer.
  • Bitwarden: Link zum quelloffenen Passwortmanager.
  • Sophos Home: Link zur Sophos Home Antivirus-Software.
  • uBlock Origin: Link zur leistungsstarken Adblocker-Browsererweiterung.
  • Osprey: Link zur Browser-Schutz-Erweiterung.
  • Notepad++: Link zum quelloffenen Texteditor.
  • Twinkle Tray: Link zur Software zur Monitor-Helligkeitssteuerung.

  • FanControl: Link zur Software zur Lüftersteuerung.

Herunterladen und Ausführen

Nicht sicher, wie Sie RepairKit herunterladen und ausführen? Folgen Sie diesen Schritten:

  • Laden Sie die neueste Version von RepairKit von der
Release-Seite herunter (klicken Sie auf die Datei RepairKit-X.X.X.zip).
  • Entpacken Sie die ZIP-Datei in einen Ordner auf Ihrem Computer. Sie können ein Programm wie 7-Zip zum Entpacken verwenden.
  • Doppelklicken Sie auf die Datei RepairKit-X.X.X.exe, um das Programm zu starten.
  • Falls der Windows Defender SmartScreen die App blockiert, klicken Sie auf Weitere Informationen und dann auf Trotzdem ausführen.
  • Klicken Sie auf Ja in der Benutzerkontensteuerungs-Abfrage.

RepairKit wird nun geöffnet und Sie können die Funktionen verwenden.

Kompilieren

  • Klonen Sie das Repository.
  • Öffnen Sie eine Eingabeaufforderung/Terminal im Repository-Verzeichnis.
  • Führen Sie gradlew createExe unter Windows oder ./gradlew createExe unter macOS oder Linux aus.
  • Die erstellte Datei RepairKit-X.X.X.zip befindet sich im Ordner build.
