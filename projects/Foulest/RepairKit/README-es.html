RepairKit

RepairKit es un kit de herramientas todo en uno basado en Java para reparación y mantenimiento de Windows.

Guía de Instalación y Configuración

Características

Reparaciones Automáticas

Realice automáticamente una limpieza y reparación integral del sistema, incluyendo:

  • Crear un Punto de Restauración del sistema
  • Eliminar políticas del sistema maliciosas
  • Ejecutar ajustes del registro y del sistema
  • Quitar características vulnerables de Windows
  • Aplicar configuraciones óptimas para Windows Defender
  • Limpiar archivos basura difíciles de encontrar
  • Eliminar bloatware preinstalado
  • Reparar problemas de disco y archivos de Windows corruptos
  • Actualizar programas instalados obsoletos
Cada función en la reparación es completamente personalizable, lo que le permite habilitar o deshabilitar reparaciones específicas según sea necesario utilizando ya sea las casillas de verificación en el panel, o los archivos de configuración ubicados en la carpeta config. Modifique estos archivos solo si sabe lo que está haciendo.

Nota: La opción Eliminar Malware previamente disponible ahora ha sido eliminada. Ejecutar ajustes de Windows Defender ahora
aplica configuraciones óptimas de Windows Defender en su lugar. Las opciones de escaneo manual están disponibles utilizando las opciones de escaneo de Emsisoft o Sophos
en el panel de Programas Útiles.

Programas Útiles

Acceda a herramientas de software esenciales para el mantenimiento del sistema, incluyendo:

  • CPU-Z: Identifique los detalles de hardware de su sistema.
  • HWMonitor: Monitoree las temperaturas y especificaciones del hardware.
  • Emsisoft Scan: Escanee en busca de malware usando Emsisoft Emergency Kit.
  • Sophos Scan: Escanee en busca de malware usando Sophos Scan & Clean.
  • TreeSize: Analice y administre el contenido del disco.
  • Everything: Un motor de búsqueda de archivos ultra rápido.
  • CrystalDiskInfo: Monitoree la salud y estado de su disco duro.
  • CrystalDiskMark: Evalúe las velocidades de lectura/escritura de su disco duro.
  • Autoruns: Una alternativa al Administrador de inicio de Windows.
  • Process Explorer: Una alternativa al Administrador de tareas de Windows.
  • Process Monitor: Monitoree la actividad del sistema y procesos.
  • BlueScreenView: Vea y analice volcados de fallos BSOD de Windows.
  • NVCleanstall: Un actualizador ligero para controladores de tarjetas gráficas NVIDIA.
  • DisplayDriverUninstaller: Elimine controladores y paquetes de pantalla.
  • Recuva: Una herramienta de recuperación de datos para archivos eliminados.
  • DiskGenius: Un gestor de particiones de disco y herramienta de recuperación de datos.

Además, RepairKit proporciona enlaces a herramientas de software útiles, que incluyen:

  • 7-Zip: Enlace al archivador de archivos de código abierto.
  • Bitwarden: Enlace al gestor de contraseñas de código abierto.
  • Sophos Home: Enlace al software antivirus Sophos Home.
  • uBlock Origin: Enlace a la potente extensión de navegador para bloqueo de anuncios.
  • Osprey: Enlace a la extensión de protección para navegador.
  • Notepad++: Enlace al editor de texto de código abierto.
  • Twinkle Tray: Enlace al software para control de brillo del monitor.
  • FanControl: Enlace al software para control de velocidad del ventilador.

Descargar y ejecutar

¿No sabes cómo descargar y ejecutar RepairKit? Sigue estos pasos:

  • Descarga la última versión de RepairKit desde
la página de lanzamientos (haz clic en el archivo RepairKit-X.X.X.zip).
  • Extrae el archivo ZIP en una carpeta de tu computadora. Puedes usar un programa como 7-Zip para extraer
el contenido.
  • Haz doble clic en el archivo RepairKit-X.X.X.exe para ejecutar el programa.
  • Si Windows Defender SmartScreen bloquea la aplicación, haz clic en Más información y luego en Ejecutar de todos modos.
  • Haz clic en Sí en el aviso de Control de cuentas de usuario.

Ahora RepairKit se abrirá y podrás empezar a usar sus funciones.

Compilación

  • Clona el repositorio.
  • Abre un símbolo del sistema/terminal en el directorio del repositorio.
  • Ejecuta gradlew createExe en Windows, o ./gradlew createExe en macOS o Linux.
  • El archivo RepairKit-X.X.X.zip generado estará en la carpeta build.
