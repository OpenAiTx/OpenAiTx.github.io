Guía de Instalación y Configuración

Características

Reparaciones Automáticas

Realiza automáticamente una limpieza y reparación exhaustiva del sistema, incluyendo:

  • Creación de un Punto de Restauración del sistema
  • Eliminación de políticas maliciosas del sistema
  • Ejecución de ajustes en el registro y el sistema
  • Eliminación de características vulnerables de Windows
  • Aplicación de configuraciones óptimas para Windows Defender
  • Limpieza de archivos basura difíciles de encontrar
  • Eliminación de bloatware preinstalado
  • Reparación de problemas de disco y archivos corruptos de Windows
  • Actualización de programas instalados desactualizados

Cada función en la reparación es totalmente personalizable, permitiéndote activar o desactivar reparaciones específicas según lo requieras utilizando ya sea las casillas de verificación en el panel, o los archivos de configuración ubicados en la carpeta config. Solo modifica estos archivos si sabes lo que estás haciendo.

Nota: La opción Eliminar Malware que estaba disponible anteriormente ha sido eliminada. Ejecutar ajustes de Windows Defender ahora
aplica configuraciones óptimas de Windows Defender en su lugar. Opciones de escaneo manual están disponibles usando las opciones de escaneo Emsisoft o Sophos
en el panel de Programas Útiles.

Programas Útiles

Accede a herramientas esenciales de software para el mantenimiento del sistema, incluyendo:

  • CPU-Z: Identifica los detalles de hardware de tu sistema.
  • HWMonitor: Monitorea temperaturas y especificaciones del hardware.
  • Emsisoft Scan: Escanea en busca de malware con Emsisoft Emergency Kit.
  • Sophos Scan: Escanea en busca de malware con Sophos Scan & Clean.
  • TreeSize: Analiza y gestiona el contenido del disco.
  • Everything: Un motor de búsqueda de archivos ultra-rápido.
  • CrystalDiskInfo: Monitorea el estado y la salud de tu disco.
  • CrystalDiskMark: Realiza pruebas de velocidad de lectura/escritura
de tu disco.
  • Autoruns: Una alternativa al Administrador de inicio de Windows.
  • Process Explorer: Una alternativa al
Administrador de tareas de Windows.
  • Process Monitor: Monitorea la actividad y los procesos del sistema.
  • BlueScreenView: Visualiza y analiza volcados de errores de Windows BSOD.
  • NVCleanstall: Actualizador liviano de drivers de tarjetas NVIDIA.
  • DisplayDriverUninstaller: Elimina
controladores y paquetes de pantalla.
  • Recuva: Una herramienta de recuperación de datos para archivos eliminados.
  • DiskGenius: Gestor de particiones de disco y herramienta de recuperación de datos.

Además, RepairKit ofrece enlaces a herramientas de software útiles, incluyendo:

  • 7-Zip: Enlace al archivador de archivos de código abierto.
  • Bitwarden: Enlace al gestor de contraseñas de código abierto.
  • Sophos Home: Enlace al software antivirus Sophos Home.
  • uBlock Origin: Enlace a la poderosa extensión bloqueadora de anuncios.
  • Osprey: Enlace a la extensión de protección del navegador.
  • Notepad++: Enlace al editor de texto de código abierto.
  • Twinkle Tray: Enlace al software de control de brillo del monitor.

  • FanControl: Enlace al software de control de velocidad de ventiladores.

Descargar y ejecutar

¿No sabes cómo descargar y ejecutar RepairKit? Sigue estos pasos:

  • Descarga la última versión de RepairKit desde
la página de lanzamientos (haz clic en el archivo RepairKit-X.X.X.zip).
  • Extrae el archivo ZIP en una carpeta de tu computadora. Puedes usar un programa como 7-Zip para extraer
el contenido.
  • Haz doble clic en el archivo RepairKit-X.X.X.exe para ejecutar el programa.
  • Si Windows Defender SmartScreen bloquea la app, haz clic en Más información y luego en Ejecutar de todas formas.
  • Haz clic en Sí en el aviso de Control de cuentas de usuario.

RepairKit se abrirá y podrás comenzar a usar sus funciones.

Compilación

  • Clona el repositorio.
  • Abre una terminal o símbolo del sistema en el directorio del repositorio.
  • Ejecuta gradlew createExe en Windows o ./gradlew createExe en macOS o Linux.
  • El archivo RepairKit-X.X.X.zip generado estará en la carpeta build.
