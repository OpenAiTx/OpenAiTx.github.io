การติดตั้ง

Linux

NixOS (Flake)

คุณสามารถใช้ flake.nix ในรีโปนี้ได้

ในไฟล์ flake.nix ของคุณ: 

{
  inputs = {
    nixpkgs.url = "github:nixos/nixpkgs/nixpkgs-unstable";
    ytdlp-gui = {
      url = "github:bksalman/ytdlp-gui";
    };
  };
    outputs = { nixpkgs, ytdlp-gui, ...}:
    let
      system = "x86_64-linux";
      pkgs = import nixpkgs {
        inherit system;
        overlays = [
          ytdlp-gui.overlay
        ];
      };
    in
    {
      ...snip

จากนั้นคุณสามารถเพิ่มเป็นแพ็กเกจปกติ ไม่ว่าจะเป็นสำหรับ home-manager หรือ nixosConfiguration

Fedora

ดาวน์โหลดแพ็กเกจ rpm จากหน้าปล่อยเวอร์ชัน แล้วติดตั้งด้วยคำสั่ง `sudo dnf localinstall

Ubuntu

ดาวน์โหลดแพ็กเกจ deb จากหน้าปล่อยเวอร์ชัน แล้วติดตั้งด้วยคำสั่ง sudo apt install ./

Arch

มีให้ใน AUR ytdlp-gui

การแจกจ่ายอื่นๆ

#### 1- ดาวน์โหลด yt-dlp โดย

      ก- จากรีโปของการแจกจ่ายของคุณ

      ข- หรือดาวน์โหลด binary แล้วย้ายไปยังโฟลเดอร์ bin ของคุณ และทำให้เป็นไฟล์ที่สามารถรันได้โดยใช้คำสั่ง chmod +x

#### 2- ดาวน์โหลด ffmpeg หรือ ffmpeg-free จากรีโปของการแจกจ่ายของคุณ

#### 3- ดาวน์โหลด binary ของ ytdlp-gui จาก หน้าปล่อยเวอร์ชัน

Windows

##### เพียงดาวน์โหลดไฟล์ zip จากหน้าปล่อยเวอร์ชัน แตกไฟล์ในโฟลเดอร์ย่อย แล้วเริ่มต้น ytdlp-gui.exe

การสร้างจากซอร์สโค้ด

เพื่อสร้างจากซอร์สโค้ด คุณต้องมี cargo และ rustc คุณสามารถติดตั้งผ่าน rustup (ตัวจัดการ toolchain ของ Rust) หรือจากรีโปของการแจกจ่ายของคุณก็ได้

หลังจากนั้นให้รันคำสั่งต่อไปนี้: 

# โคลนรีโปไปยังโฟลเดอร์ "ytdlp-gui"
git clone https://github.com/BKSalman/ytdlp-gui

เข้าไปในโฟลเดอร์

cd ytdlp-gui

คุณสามารถสร้างโปรเจคโดยใช้คำสั่งนี้

cargo build

หรือสร้างในโหมด release เพื่อประสิทธิภาพที่ดีกว่า

cargo build -r
จากนั้นไฟล์ไบนารีจะอยู่ที่ /target/debug/ytdlp-gui หรือ /target/release/ytdlp-gui

และคุณสามารถรันได้โดยตรง: 

# จากโฟลเดอร์โปรเจค
./target/release/ytdlp-gui

หรือใช้ cargo: 

cargo r

หรือสำหรับโหมด release

cargo r -r

การตั้งค่า

สำหรับเวอร์ชัน v0.2.2+ แอปพลิเคชันจะบันทึกการตั้งค่าไว้ในไดเรกทอรีการตั้งค่าเริ่มต้นของแพลตฟอร์ม/ระบบปฏิบัติการนั้นๆ ที่ /ytdlp-gui/config.toml

ไฟล์เริ่มต้นจะมีลักษณะดังนี้:

# ตัวเลือก

นี่คือไดเรกทอรีของ bin ไม่ใช่ bin เอง


bin_path = "" # (0.2.4)
bin_dir = "" # (0.2.5+) หากไม่ตั้งค่า คำสั่งจะเป็น yt-dlp `
ตัวเลือก

download_folder = "" # ค่าเริ่มต้น = "~/Videos"
[options]
video_resolution = "FullHD" # ตัวเลือก: "Sd" "Hd" "FullHD" "TwoK" "FourK"
video_format = "Mp4" # ตัวเลือก: "Mp4" "Mkv" "Webm"
audio_quality = "Good" # ตัวเลือก: "Best" "Good" "Medium" "Low"
audio_format = "Mp3" #  ตัวเลือก: "Mp3" "Wav" "Vorbis" "M4a" "Opus"

หมายเหตุ: ตัวเลือกคุณภาพ/รูปแบบจะถูกบันทึกโดยอัตโนมัติเมื่อกดปุ่มดาวน์โหลด
