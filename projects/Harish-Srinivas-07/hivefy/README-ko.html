Hivefy 🎧

Hivefy는 Flutter로 개발된 FOSS, Spotify에서 영감을 받은, 광고 없는, 오프라인 사용 가능한 음악 앱으로, 비공식 JioSaavn API를 사용합니다. 트렌딩 차트, 앨범, 플레이리스트, 다국어 콘텐츠를 탐색하세요 — 모두 오픈 소스, 개인정보 보호를 존중하며, 완벽하게 사용자 맞춤형입니다. 🎵

Android용 다운로드 😍

⚠️ Hivefy를 설치하려면 기기 설정에서 “알 수 없는 출처에서 설치”가 활성화되어 있는지 확인하세요.

✨ 기능

🎨 현대적인 경험

  • Spotify에서 영감을 받은 UI 및 애니메이션
  • 동적인 Material You 테마
  • SpotifyMix를 사용한 적응형 아이콘 및 타이포그래피

🎧 강력한 플레이어

  • audio_service를 통한 백그라운드 재생
  • 스와이프 제스처 및 시각적 탐색 컨트롤
  • 애니메이션 미니플레이어 → 전체 플레이어 전환

💾 오프라인 우선

  • 음악을 로컬에 다운로드 및 저장 (앨범 / 재생목록 / 곡)
  • 유효하지 않은 파일 자동 정리
  • 백그라운드 다운로드 알림

🔍 스마트 검색 및 라이브러리

  • 통합 글로벌 검색 (곡, 앨범, 아티스트, 재생목록)
  • Hive DB 기반의 지속적인 라이브러리
  • 인터넷 없이도 완벽 작동

⚙️ 설정 및 사용자화

  • 서버 선택기 (메인 / 백업 / 미러)
  • 캐시 관리 및 다운로드 모니터
  • 테마 전환, 저장소 정리 등

🚀 시작하기

사전 준비

  • Flutter SDK (>=3.7.2)
  • Flutter용 Android Studio / VS Code 설정
  • Java JDK 17+ (앱 서명을 위해)
  • Android 7.0+ 실행 중인 기기 / 에뮬레이터

설정 및 실행 

git clone https://github.com/Harish-Srinivas-07/hivefy.git
cd hivefy
flutter pub get
flutter run

🧩 기술 스택

| 카테고리 | 패키지 | | --------- | -------------------------------------------------------------------------------------- | | 상태 | flutter_riverpod | | 오디오 | just_audio, audio_service, just_audio_background | | 저장소 | hive, shared_preferences, path_provider | | UI/UX | shimmer, cached_network_image, figma_squircle, flashy_tab_bar2, readmore | | 네트워크 | dio, http, html_unescape | | 시스템 | flutter_local_notifications, disk_space_plus, package_info_plus | | 유틸 | toastification, permission_handler, internet_connection_checker_plus |

💡 개발자 노트

  • 디버그 빌드는 DebugServer 엔드포인트 사용 (서명 불필요)
  • 릴리즈 빌드는 ProGuard로 자동 최적화
  • 모든 데이터 모델은 이스케이프 해제 정규화를 포함한 toJson 및 fromJson 포함
  • 백엔드 로직은 services/에서, UI 흐름은 views/에서 수정 가능
