Hivefy 🎧

Hivefy は FOSS、Spotifyに触発された、広告なし、オフライン対応の音楽アプリ で、Flutter と 非公式のJioSaavn API を使用して構築されています。 トレンドチャート、アルバム、プレイリスト、多言語コンテンツを探求—すべて オープンソース、プライバシー尊重、完全カスタマイズ可能。🎵

Android向けダウンロード 😍

⚠️ Hivefy をインストールするには、デバイスの設定で「提供元不明のアプリのインストール」を有効にしてください。

✨ 特徴

🎨 モダンな体験

  • Spotify にインスパイアされた UI とアニメーション
  • ダイナミックな Material You テーマ
  • SpotifyMix を使った適応アイコンとタイポグラフィ

🎧 強力なプレイヤー

  • audio_service によるバックグラウンド再生
  • スワイプジェスチャーと視覚的なシークコントロール
  • アニメーション付きミニプレイヤー → フルプレイヤーへの遷移

💾 オフラインファースト

  • 音楽をローカルにダウンロード・保存（アルバム / プレイリスト / 曲）
  • 無効ファイルの自動クリーンアップ
  • バックグラウンドダウンロード通知

🔍 スマート検索＆ライブラリ

  • 統合されたグローバル検索（曲、アルバム、アーティスト、プレイリスト）
  • Hive DB による永続的なライブラリ
  • インターネット接続なしで完全に動作

⚙️ 設定とカスタマイズ

  • サーバーセレクター（メイン / バックアップ / ミラー）
  • キャッシュ管理＆ダウンロードモニター
  • テーマ切替、ストレージのクリアなど

🚀 はじめに

前提条件

  • Flutter SDK (>=3.7.2)
  • Flutter用のAndroid Studio / VS Codeのセットアップ
  • Java JDK 17+（アプリ署名用）
  • Android 7.0+を搭載したデバイスまたはエミュレーター

セットアップと実行

git clone https://github.com/Harish-Srinivas-07/hivefy.git
cd hivefy
flutter pub get
flutter run

🧩 技術スタック

| カテゴリ | パッケージ | | ----------- | ---------------------------------------------------------------------------------- | | 状態管理 | flutter_riverpod | | オーディオ | just_audio, audio_service, just_audio_background | | ストレージ | hive, shared_preferences, path_provider | | UI/UX | shimmer, cached_network_image, figma_squircle, flashy_tab_bar2, readmore | | ネットワーク | dio, http, html_unescape | | システム | flutter_local_notifications, disk_space_plus, package_info_plus | | ユーティリティ | toastification, permission_handler, internet_connection_checker_plus |

💡 開発者ノート

  • デバッグビルド は DebugServer エンドポイントを使用（署名不要）
  • リリースビルド は自動的に ProGuard で最適化されます
  • すべてのデータモデルにはアンエスケープ正規化を含む toJson と fromJson が含まれています
  • バックエンドのロジックは services/、UIのフローは views/ 内で変更可能です
