Arco Admin

An out-of-the-box, feature-rich mid-to-backend management system framework, a standardized project with the latest technology stack.

Simplified Chinese | English

This project’s CDN acceleration and security protection are sponsored and supported by Tencent EdgeOne

Features

+ ⚡️ Keep up with the latest versions of Vue and Vite. + 👍🏻 Vue + official recommended Vue Router + Pinia state management. + 🔩 Enterprise-level design ecosystem component library ArcoDesign produced by ByteDance. + 🎨 Less CSS preprocessor. + 📚 Use i18n + i18n-ally to implement a simpler, standardized automated internationalization solution. + ⌨️ Built-in multiple business scenario pages, secondary encapsulated components, framework support configuration for easy personalized customization. + 🧚🏻 The project has fully adopted the ESM standard modular syntax, following the major unification trend in modern front-end development. + 📦 Continuously optimize configuration to keep up with updates, deprecations, and feature merges of various dependencies and Vite to achieve the best packaging optimization effect.
