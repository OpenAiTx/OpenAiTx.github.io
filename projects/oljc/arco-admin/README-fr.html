Arco Admin

Un cadre de système de gestion back-office riche en fonctionnalités et prêt à l'emploi, un projet normé avec la dernière pile technologique.

Chinois simplifié | Anglais

L'accélération CDN et la protection de sécurité de ce projet sont sponsorisées par Tencent EdgeOne

Caractéristiques

+ ⚡️ Toujours à jour avec les dernières versions de Vue et Vite. + 👍🏻 Vue + le Vue Router officiel recommandé + gestionnaire Pinia. + 🔩 Bibliothèque de composants d'écosystème de design d'entreprise ArcoDesign développée par ByteDance. + 🎨 Préprocesseur CSS Less. + 📚 Utilisation de i18n + i18n-ally pour une solution d’internationalisation automatisée plus simple et standardisée. + ⌨️ Intégration de nombreuses pages de scénarios métier, composants encapsulés, support du cadre pour une personnalisation facile. + 🧚🏻 Le projet utilise entièrement la syntaxe modulaire standard ESM, suivant la grande tendance unifiée du développement front-end moderne. + 📦 Optimisation continue des configurations pour suivre les mises à jour, suppressions, et fusions de fonctionnalités des dépendances et de Vite afin d’obtenir une optimisation de bundling optimale.
