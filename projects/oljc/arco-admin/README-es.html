Arco Admin

Un marco de sistema de administración de backend listo para usar y con muchas funciones, un proyecto estándar con la última pila tecnológica.

Chino simplificado | Inglés

La aceleración CDN y la protección de seguridad de este proyecto están patrocinadas por Tencent EdgeOne

Características

+ ⚡️ Mantenerse al día con las últimas versiones de Vue y Vite. + 👍🏻 Vue + el oficial recomendado Vue Router + el gestor Pinia. + 🔩 Biblioteca de componentes del ecosistema de diseño empresarial ArcoDesign creada por ByteDance. + 🎨 Preprocesador CSS Less. + 📚 Uso de i18n + i18n-ally para una solución de internacionalización automática más simple y estandarizada. + ⌨️ Páginas para múltiples escenarios de negocio, componentes reempaquetados, configuración del marco para personalización fácil. + 🧚🏻 El proyecto adopta completamente la sintaxis modular estándar ESM, siguiendo la gran tendencia unificada del desarrollo frontend moderno. + 📦 Optimización continua de la configuración para seguir las actualizaciones, deprecaciones y combinaciones de dependencias y Vite, para lograr la mejor optimización de empaquetado.

Versión

  • Versión internacionalizada: rama i18n

Normas

Mejorar la calidad del código, reducir errores y asegurar la coherencia en el estilo de código en todo el proyecto te permite gestionar más fácilmente la calidad y el estilo del código.

+ Normas de código: ESLint + Stylelint + Prettier
Comprobación y normalización estática de la sintaxis, estilo y formato del código mediante normas predefinidas para garantizar la calidad y legibilidad del código

+ Normas de commit: Husky + Lint-Staged + Commitlint
Al hacer commit, se asegura la calidad y consistencia del código mediante comprobaciones de estilo, formateo y validación de mensajes preestablecidas
