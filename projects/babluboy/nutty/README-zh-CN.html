Nutty

一个网络工具

作者：Siddhartha Das

这是一个为 elementary OS 设计的简单应用程序，用于提供网络相关方面的基本信息。所展示信息以如下标签页形式呈现。

  • 我的信息：提供设备网卡的基础和详细信息
  • 使用情况：以两种视图展示网络数据使用情况——历史使用与当前使用

  • 速度：检查上传和下载速度，并获取到主机的路由时间
  • 端口：提供本地设备上活动端口及其所用应用程序的信息
  • 设备：监控、警报并提供连接到网络的其他设备的信息

请访问 Nutty 官网以获取功能详情、快捷键、Ubuntu 及其他支持发行版的安装指南等信息：
https://babluboy.github.io/nutty/

从源码构建、测试与安装

你需要以下依赖项来进行构建：

  • libgranite-dev
  • libnotify-dev
  • libxml2-dev
  • libxml2-utils
  • libgee-0.8-dev
  • libgtk-3-dev
  • libsqlite3-dev
  • meson
  • valac

并需要以下依赖项来运行：
  • net-tools
  • nethogs
  • nmap
  • traceroute
  • vnstat
  • curl
  • wireless-tools
  • iproute2
  • pciutils

有时在安装时 vnstat 并未自动启动，请使用相应的初始化系统命令启动 vnstat 守护进程，例如：sudo systemctl enable vnstat

运行 meson build 以配置构建环境，并运行 ninja test 进行构建

git clone https://github.com/babluboy/nutty.git
cd nutty
meson build --prefix=/usr
cd build
ninja

安装时，使用 sudo ninja install，然后执行 com.github.babluboy.nutty

sudo ninja install
com.github.babluboy.nutty
要卸载，请进入构建目录并运行

sudo ninja uninstall

从 Deb 包安装

要从 .deb 包安装 Nutty，请确保已安装上述所有 Nutty 的运行时依赖（无需构建依赖）。然后运行以下命令 

sudo dpkg -i .deb

要卸载通过 .deb 包安装的 Nutty，请运行 

sudo apt autoremove com.github.babluboy.nutty
如果您想直接构建 Nutty 的 .deb 软件包，请先安装上述构建依赖项，然后运行

git clone https://github.com/babluboy/nutty.git
cd nutty
dpkg-buildpackage -us -uc
cd ..
