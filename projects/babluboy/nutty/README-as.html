Nutty

এখন নেটৱৰ্ক ইউটিলিটি

লেখক: সিদ্ধার্থ দাস

elementary OS ৰ বাবে তৈয়াৰ কৰা এটা সাধাৰণ এপ্লিকেশ্যন যি নেটৱৰ্ক সম্পৰ্কীয় বিষয়সমূহৰ প্ৰয়োজনীয় তথ্য প্ৰদান কৰে। তথ্যসমূহ তলত উল্লেখ কৰা ট্যাবসমূহত দেখুওৱা হৈছে।

  • My Info: ডিভাইচৰ নেটৱৰ্ক কাৰ্ডৰ মৌলিক আৰু বিস্তৃত তথ্য প্ৰদান কৰে
  • Usage: নেটৱৰ্ক ডেটা ব্যৱহাৰ দুটা দৃশ্যত দেখুৱায় - ইতিবৃত্তমূলক ব্যৱহাৰ আৰু বৰ্তমান ব্যৱহাৰ
  • গতি: আপল'ড আৰু ডাউনল'ড গতিবিধি পৰীক্ষা কৰক আৰু কোনো হষ্টলৈ ৰুট সময় জানক
  • পোৰ্টছ: সক্ৰিয় পোৰ্ট আৰু স্থানীয় ডিভাইচত সিহঁত ব্যৱহাৰ কৰা এপ্লিকেশ্যনসমূহৰ বিষয়ে তথ্য দিয়ে
  • ডিভাইচসমূহ: নেটৱৰ্কত সংযুক্ত অন্য ডিভাইচসমূহ মনিটৰ, এলাৰ্ট আৰু তথ্য প্ৰদান কৰে

বিৱৰণ, চমু পথ, উবুন্টু আৰু অন্যান্য সমৰ্থিত ডিষ্ট্ৰোৰ ইনষ্টলেচন গাইড আদিৰ বাবে Nutty ৱেবছাইট চাওক :
https://babluboy.github.io/nutty/

উৎসৰ পৰা নিৰ্মাণ, পৰীক্ষা আৰু ইনষ্টলেচন

নিৰ্মাণৰ বাবে তলত দিয়া নিৰ্ভৰশীলতাসমূহৰ প্ৰয়োজন:

  • libgranite-dev
  • libnotify-dev
  • libxml2-dev
  • libxml2-utils
  • libgee-0.8-dev
  • libgtk-3-dev
  • libsqlite3-dev
  • meson
  • valac

চলাবলৈ এই নিৰ্ভৰশীলতাসমূহৰ প্ৰয়োজন:
  • net-tools
  • nethogs
  • nmap
  • traceroute
  • vnstat
  • curl
  • wireless-tools
  • iproute2
  • pciutils

কেতিয়াবা vnstat ইনষ্টল কৰাৰ পিছত আৰম্ভ নহয়, vnstat ডিমন আৰম্ভ কৰিবলৈ উপযুক্ত init system কমাণ্ড ব্যৱহাৰ কৰক, যেনে sudo systemctl enable vnstat

নিৰ্মাণ পৰিৱেশ কনফিগাৰ কৰিবলৈ meson build চলাওক আৰু নিৰ্মাণ কৰিবলৈ ninja test চলাওক

git clone https://github.com/babluboy/nutty.git
cd nutty
meson build --prefix=/usr
cd build
ninja
ইন্সটল কৰিবলৈ, sudo ninja install ব্যৱহাৰ কৰক, তাৰ পাছত com.github.babluboy.nutty দ্বাৰা এক্সিকিউট কৰক

sudo ninja install
com.github.babluboy.nutty
অনইনষ্টল কৰিবলৈ, বিল্ড ডাইৰেক্টৰীত যাওক আৰু চলাও

sudo ninja uninstall

ডেব পেকেজৰ পৰা ইনষ্টলেশন

.deb পেকেজৰ পৰা Nutty ইনষ্টল কৰিবলৈ, নিশ্চিত কৰক যে আপুনি Nutty-ৰ ওপৰৰ সকলো এক্সিকিউশন ডিপেণ্ডেন্সী ইনষ্টল কৰিছে (বিল্ড ডিপেণ্ডেন্সী লাগিব নালাগে)। তাৰ পিছত তলৰ নিৰ্দেশটো চলাও 

sudo dpkg -i .deb

আৰু .deb পেকেজৰ জৰিয়তে সংস্থাপন কৰা Nutty আনইনষ্টল কৰিবলৈ এইটো চলাও 

sudo apt autoremove com.github.babluboy.nutty
আপুনি যদি Nutty-ৰ .deb পেকেজটো নিজে সৰ্বসাক্ষাৎকৈ বনাব খুজে, তেন্তে ওপৰৰ নিৰ্মাণ নিৰ্ভৰশীলসমূহ ইনষ্টল কৰক আৰু তাৰ পিছত চলাও

git clone https://github.com/babluboy/nutty.git
cd nutty
dpkg-buildpackage -us -uc
cd ..
