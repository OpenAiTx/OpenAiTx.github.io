Un dépôt avec des milliers de presets de couleurs prédéfinis pour ReShade

Qu'est-ce qu'un LUT ?

Les LUTs (Table de correspondance des couleurs) sont des presets de couleur intégrés que vous pouvez utiliser dans n'importe quel jeu où vous pouvez installer ReShade.

De quoi ai-je besoin pour que ça fonctionne ?

+ ReShade + Le fichier _BaseLUT.fxh fourni avec ce dépôt + Les fichiers fournis avec l'installation par défaut de ReShade. + ReShade.fxh + ReShadeUI.fxh + Les fichiers .fx et .png avec des noms de fichiers correspondants.

Installation

  • Téléchargez le dépôt.
  • Déplacez les fichiers .fx que vous voulez (vous n'avez pas besoin de tous, juste ceux que vous souhaitez) dans le dossier Shaders à l'intérieur de votre dossier d'installation de ReShade.
  • Déplacez les fichiers .png correspondants dans le dossier Textures à l'intérieur de votre dossier d'installation de ReShade.
  • Terminé !

Astuces et conseils

Vous pouvez combiner autant de LUTs que vous le souhaitez, mais sachez que vous commencerez à voir un fort banding des couleurs avec 2 LUTs ou plus actives en même temps.

Changez l'ordre de chargement du shader pour obtenir des résultats différents.

https://user-images.githubusercontent.com/23023051/118404928-719f4d80-b64b-11eb-964d-09fa9b60fe8b.mp4

Placez les LUTs entre les 2 shaders PCGI pour changer les couleurs du GI.

https://user-images.githubusercontent.com/23023051/118405513-9cd76c00-b64e-11eb-915f-edffe52f1dae.mp4

Maintenant vous pouvez utiliser MLUT avec la profondeur et peindre le ciel de toutes les couleurs de l’arc-en-ciel \o/

https://user-images.githubusercontent.com/23023051/146495932-cdbec80d-e532-4300-9a88-323bf675bc10.mp4
