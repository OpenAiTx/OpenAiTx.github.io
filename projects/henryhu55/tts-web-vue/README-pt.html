Características

  • 🌐 Totalmente web, sem necessidade de instalar aplicativo de desktop
  • 🔊 Suporta diversas sínteses de voz da Microsoft
  • 🚀 Suporta integração com API TTS88
  • 🆓 Suporta TTS gratuito, com cota diária grátis
  • 🧠 Suporta geração de texto pelo OpenAI
  • 🌍 Suporta vários idiomas: chinês, inglês, espanhol
  • 🔐 Identificação de impressão digital do navegador, experiência do usuário mais segura

Configuração de API

Esta versão web atualmente suporta serviço TTS gratuito, API TTS88 e API OpenAI:

Configuração do serviço TTS gratuito

  • Encontre a opção "Serviço TTS gratuito" na página de configurações
  • O endereço do servidor TTS gratuito já está configurado por padrão
  • Você pode visualizar o saldo gratuito restante do dia e o tempo de reinicialização
  • Cada cliente do navegador possui uma cota gratuita independente

Configuração da API TTS88

  • Encontre a opção de configuração "URL de API de terceiros" na página de configurações
  • Insira seu endereço da API TTS88
  • Se houver uma chave de API, insira-a no campo "Chave da API TTS88"

Configuração da API OpenAI

  • Encontre a opção de configuração OpenAI na página de configurações
  • Insira sua chave de API OpenAI
  • Selecione o modelo a ser utilizado (padrão: gpt-3.5-turbo)
  • Se estiver usando auto-hospedagem ou proxy, pode definir uma URL Base de API personalizada

Observações

  • Os dados são armazenados apenas no armazenamento local do navegador; atualizar ou limpar o cache não afeta outros usuários
  • Os arquivos de áudio convertidos podem ser reproduzidos diretamente no navegador ou baixados para o computador local
  • O serviço TTS gratuito tem limite diário de uso; ao exceder, é necessário aguardar a reinicialização no dia seguinte
