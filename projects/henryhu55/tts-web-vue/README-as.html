ইণ্টাৰফেচ পুনৰ্নিমাণ আৰু ফিচাৰ বৃদ্ধি (v2.2.0)

  • 📱 UI পুনৰ্নিমাণ: শীৰ্ষ নেভিগেশ্যন স্থিৰ, টেক্সটবক্স চুইটিং মোডত, ম'বাইলত উপযোগিতা উন্নত
  • 🔤 SSML বৃদ্ধি: নতুন SSML ফৰ্মেটিং ফিচাৰ, ছেটিং পৰিবৰ্তনত SSML স্বয়ংক্ৰিয়ভাৱে আপডেট
  • 🎵 প্লেয়াৰ উন্নতি: প্লেয়াৰ মূখ্য ইণ্টাৰফেচত সংযোজন, অডিঅ' প্লে আৰু ডাউনলোডৰ অভিজ্ঞতা উন্নত
  • 🎞️ ছাবটাইটেল ফিচাৰ: অনলাইন ছাবটাইটেল সৃষ্টি ফিচাৰ যোগ, বিষয়বস্তু সৃষ্টি দক্ষতা বৃদ্ধি
  • 🌓 থীম উন্নতি: ডাৰ্ক মোডত ইণ্টাৰফেচ উন্নত, গ্ল'বল থীম সলনি যোগ
  • 🛠️ উন্নত ছেটিং: বিনামূলীয়া TTS ছেটিং ইণ্টাৰফেচ উন্নত, স্পীড আৰু পিচ ছেটিং যোগ
  • 💡 ইন্টাৰেক্টিভ অভিজ্ঞতা: টুলটিপ ফিচাৰ যোগ, ব্যৱহাৰকাৰীৰ কাৰ্য্যপদ্ধতি উন্নত
  • 🔄 ডিপেণ্ডেন্সি আপডেট: Vue 3.5.11লৈ, ElementPlus 2.9.9লৈ আপডেট
  • 🌐 লিংক আপডেট: GitHub ৰেপ' ঠিকনা আপডেট

বিনামূলীয়া TTS সেৱা সংযুক্তি (v2.1.0)

  • 🆕 বিনামূলীয়া TTS ফিচাৰ: API কীৰ প্ৰয়োজন নাই, প্ৰতি দিন বিনামূলীয়া ক্ৰেডিট উপলব্ধ
  • 👤 ব্ৰাউজাৰ ফিঙ্গাৰপ্ৰিন্ট চিনাক্তকৰণ: প্ৰতিজন ব্যৱহাৰকাৰীৰ বাবে সম অধিকার নিশ্চিতকৰণ
  • 🔄 ক্ৰেডিট অৱস্থা দেখুৱাই: অবশিষ্ট বিনামূলীয়া চৰিত্ৰ আৰু ৰিছেট সময় দেখুৱাই
  • 🌐 চাৰ্ভাৰ অৱস্থা পৰীক্ষা: বিনামূলীয়া TTS চাৰ্ভাৰৰ সংযোগ অৱস্থা স্বয়ংক্ৰিয়ভাৱে পৰীক্ষা
  • 🛡️ উন্নত ত্ৰুটি ব্যৱস্থাপনা: অধিক বন্ধুত্বপূর্ণ ত্ৰুটি আৰু অৱস্থা প্ৰতিক্ৰিয়া
  • 🎛️ বিনামূলীয়া TTS ছেটিং পেনেল: ব্যৱহাৰকাৰীৰ বাবে ছেটিং আৰু ক্ৰেডিট তথ্য চাবলৈ সুবিধা

ইণ্টাৰফেচ আৰু ব্যৱহাৰকাৰী অভিজ্ঞতা উন্নতি (v2.0.0)

  • ✨ ছেটিং এলাকা উন্নতি: ছেটিং পেনেল ভাঁজ কৰিব পাৰি, "Voice Settings" শিৰোনামত ক্লিক কৰিলে খুলিব/বন্ধ হব
  • 🔘 স্মাৰ্ট বুটাম স্থান: "Convert" বুটাম শীৰ্ষত, স্ক্ৰল নকৰাকৈ ক্লিক কৰিব পাৰি
  • 💬 অৱস্থা সংকেত বৃদ্ধি: স্পষ্ট কনভাৰ্শন অৱস্থা টেক্সট যোগ
  • 🎨 দৃশ্যগত উন্নতি: বুটামত লাইটিং এনিমেচন, ব্যৱহাৰকাৰীৰ অভিজ্ঞতা বৃদ্ধি
  • 📱 প্ৰতিক্ৰিয়াশীল লে-আউট উন্নতি: বিভিন্ন স্ক্ৰিনত দেখুৱাবলৈ উন্নত

এই সকল উন্নতিসমূহে ব্যৱহাৰকাৰী দক্ষতা যথেষ্ট বৃদ্ধি কৰিছে, বিশেষকৈ পৰম্পৰাগত ডিজাইনত ব্যৱহাৰকাৰীক পৃষ্ঠাৰ তলত স্ক্ৰল কৰি "Convert" বুটাম ক্লিক কৰিব লগা সমস্যা সমাধান কৰিছে। এতিয়া, ব্যৱহাৰকাৰী যিকোনো স্থানৰ পৰা সহজে কনভাৰ্ট প্ৰক্ৰিয়া আৰম্ভ কৰিব পাৰে।

স্থাপন

নির্মাণ সম্পূর্ণ হোৱাৰ পাছত, dist ডাইৰেক্টৰীৰ সমগ্ৰ সামগ্ৰী যিকোনো স্থিৰ Web ছাৰ্ভাৰত স্থাপন কৰক।

API সংৰূপ

এই Web সংস্কৰণত বৰ্তমান বিনামূলীয়া TTS সেৱা, TTS88 API আৰু OpenAI API সমৰ্থন কৰে:

বিনামূলীয়া TTS সেৱা সংৰূপ

  • ছেটিংছ পৃষ্ঠাত "বিনামূলীয়া TTS সেৱা" বিকল্পটো বিচাৰি পাব
  • ব্যৱস্থাই বিনামূলীয়া TTS ছাৰ্ভাৰৰ ঠিকনা ডিফল্ট ৰূপে সংৰূপ কৰিছে
  • আপুনি আজিৰ বাকী বিনামূলীয়া সীমা আৰু পুনৰ চাৰ্জ সময় চাব পাৰে
  • প্ৰতিটো ব্ৰাউজাৰ ক্লায়েণ্টে স্বতন্ত্ৰ বিনামূলীয়া সীমা লাভ কৰে

TTS88 API সংৰূপ

  • ছেটিংছ পৃষ্ঠাত "তৃতীয় পক্ষ API URL" ছেটিং বিকল্পটো বিচাৰি পাব
  • আপোনাৰ TTS88 API ঠিকনা টাইপ কৰক
  • যদি API চাবি থাকে, "TTS88 API চাবি" ক্ষেত্ৰত টাইপ কৰক

OpenAI API সংৰূপ

  • ছেটিংছ পৃষ্ঠাত OpenAI ছেটিং বিকল্পটো বিচাৰি পাব
  • আপোনাৰ OpenAI API চাবি টাইপ কৰক
  • ব্যৱহাৰ কৰিবলৈ মডেল নিৰ্বাচন কৰক (ডিফল্ট gpt-3.5-turbo)
  • যদি স্ব-হোষ্ট বা প্ৰক্সি ব্যৱহাৰ কৰে, কাষ্টম API Base URL ছেট কৰিব পাৰে

লক্ষ্য ৰখা বিষয়

  • ডাটা কেৱল ব্ৰাউজাৰৰ লোকেল ষ্ট'ৰেজত সংৰক্ষণ হয়, ৰিফ্ৰেছ বা কেছ মচিলে অন্য ব্যৱহাৰকাৰীৰ প্ৰভাৱ নহয়
  • ৰূপান্তৰিত অডিঅ' ফাইল ব্ৰাউজাৰত প্ৰতক্ষ্যভাৱে প্লে বা লোকেলত ডাউনলোড কৰিব পাৰি
  • বিনামূলীয়া TTS সেৱাত দৈনিক ব্যৱহাৰ সীমা থাকে, অধিক হলে পৰৱৰ্তী দিনলৈ পুৱাই পুনৰ চাৰ্জ হ'ব
