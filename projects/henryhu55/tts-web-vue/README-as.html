  • 🎵 প্লেয়াৰ উন্নয়ন: মূল ইণ্টাৰফেছত প্লেয়াৰ সংযোজন, অডিঅ' প্লে আৰু ডাউনলোড অভিজ্ঞতা উন্নয়ন
  • 🎞️ চাবটাইটেল ফিচাৰ: অনলাইন চাবটাইটেল নিৰ্মাণ ফিচাৰ, বিষয়বস্তু সৃষ্টি দক্ষতা বৃদ্ধি
  • 🌓 থীম উন্নয়ন: ডাৰ্ক মুড ইণ্টাৰফেছ উন্নয়ন, গ্লোবেল থীম চুইচ যোগ
  • 🛠️ এডভান্স্ড ছেটিংছ: বিনামূলীয়া TTS ছাৰ্ভিছ ইণ্টাৰফেছ উন্নয়ন, স্পিচ ৰেট আৰু পিচ ছেটিং সংযোজন
  • 💡 ইন্টাৰেক্টিভ অভিজ্ঞতা: টুলটিপ ফিচাৰ সংযোজন, ব্যৱহাৰকাৰী অপাৰেশ্যন পথনিদৰ্শন উন্নয়ন
  • 🔄 ডিপেণ্ডেন্সি আপডেট: Vue 3.5.11লৈ আপডেট, ElementPlus 2.9.9লৈ আপডেট
  • 🌐 লিংক আপডেট: GitHub ৰিপ'জিট'ৰি ঠিকনা আপডেট

বিনামূলীয়া TTS ছাৰ্ভিছ ইন্টেগ্ৰেচন (v2.1.0)

  • 🆕 বিনামূলীয়া TTS ফিচাৰ: API কীৰ প্ৰয়োজন নাই, প্ৰতিদিনে বিনামূলীয়া সীমা
  • 👤 ব্ৰাউজাৰ ফিংগাৰপ্ৰিন্ট চিনাক্তকৰণ: প্ৰতিগৰাকী ব্যৱহাৰকাৰীৰ ন্যায্য বিনামূলীয়া সীমা ব্যৱহাৰ নিশ্চিত
  • 🔄 সীমা অৱস্থা প্ৰদৰ্শন: বাকী বিনামূলীয়া চেৰেক্টাৰ আৰু পুনৰুদ্ধাৰৰ সময় দেখুৱায়
  • 🌐 ছাৰ্ভাৰ অৱস্থা চেকিং: বিনামূলীয়া TTS ছাৰ্ভাৰৰ সংযোগ অৱস্থা স্বয়ংক্ৰিয় চেক
  • 🛡️ উন্নত ত্ৰুটি ব্যৱস্থাপনা: অধিক বন্ধুত্বপূর্ণ ত্ৰুটি মেচেজ আৰু অৱস্থা ফিডব্যাক
  • 🎛️ বিনামূলীয়া TTS ছেটিং পেনেল: ব্যৱহাৰকাৰীয়ে ছেটিং আৰু সীমা তথ্য সহজে চাব পাৰে

ইণ্টাৰফেছ আৰু ব্যৱহাৰকাৰী অভিজ্ঞতা উন্নয়ন (v2.0.0)

  • ✨ ছেটিং অঞ্চল উন্নয়ন: ছেটিং পেনেলত ভাঁজ কৰাৰ সুবিধা, "শব্দ ছেটিং" শিৰোনামত ক্লিক কৰিলে বিস্তাৰ/ভাঁজ হব
  • 🔘 বুদ্ধিমান বুটামৰ স্থান: "পৰিবৰ্তন আৰম্ভ কৰক" বুটামটো ওপৰত স্থানান্তৰ, স্ক্ৰল নকৰাকৈ ক্লিক কৰিব পাৰিব
  • 💬 স্থিতি সংকেত উন্নয়ন: স্পষ্ট পৰিবৰ্তনৰ স্থিতি বুজাব পৰা শব্দ যোগ কৰা হৈছে
  • 🎨 দৃশ্য প্ৰভাৱ উন্নয়ন: বুটামত পোহৰৰ এনিমেচন যোগ, ব্যৱহাৰকাৰীৰ অভিজ্ঞতা বৃদ্ধি
  • 📱 প্ৰতিক্ৰিয়াশীল বিন্যাস উন্নয়ন: বিভিন্ন স্ক্ৰীন মাপত উত্তম প্ৰদৰ্শনৰ বাবে উন্নয়ন

এই উন্নয়নসমূহে ব্যৱহাৰ কাৰ্যক্ষমতা যথেষ্ট বৃদ্ধি কৰিছে, বিশেষকৈ পৰম্পৰাগত ডিজাইনত ব্যৱহাৰকাৰীকে "পৰিবৰ্তন আৰম্ভ কৰক" বুটামটোত ক্লিক কৰিবলৈ পৃষ্ঠাৰ তলত স্ক্ৰল কৰিব লগা সমস্যা সমাধান কৰিছে। এতিয়া, ব্যৱহাৰকাৰী পৃষ্ঠাৰ যিকোনো স্থানৰ পৰা সহজে পৰিবৰ্তন আৰম্ভ কৰিব পাৰে।

স্থাপন

নির্মাণ সম্পূর্ণ হোৱাৰ পাছত, dist ডাইৰেক্টৰীৰ সমগ্ৰ সামগ্ৰী যিকোনো স্থিৰ Web ছাৰ্ভাৰত স্থাপন কৰক।

API সংৰূপ

এই Web সংস্কৰণত বৰ্তমান বিনামূলীয়া TTS সেৱা, TTS88 API আৰু OpenAI API সমৰ্থন কৰে:

বিনামূলীয়া TTS সেৱা সংৰূপ

  • ছেটিংছ পৃষ্ঠাত "বিনামূলীয়া TTS সেৱা" বিকল্পটো বিচাৰি পাব
  • ব্যৱস্থাই বিনামূলীয়া TTS ছাৰ্ভাৰৰ ঠিকনা ডিফল্ট ৰূপে সংৰূপ কৰিছে
  • আপুনি আজিৰ বাকী বিনামূলীয়া সীমা আৰু পুনৰ চাৰ্জ সময় চাব পাৰে
  • প্ৰতিটো ব্ৰাউজাৰ ক্লায়েণ্টে স্বতন্ত্ৰ বিনামূলীয়া সীমা লাভ কৰে

TTS88 API সংৰূপ

  • ছেটিংছ পৃষ্ঠাত "তৃতীয় পক্ষ API URL" ছেটিং বিকল্পটো বিচাৰি পাব
  • আপোনাৰ TTS88 API ঠিকনা টাইপ কৰক
  • যদি API চাবি থাকে, "TTS88 API চাবি" ক্ষেত্ৰত টাইপ কৰক

OpenAI API সংৰূপ

  • ছেটিংছ পৃষ্ঠাত OpenAI ছেটিং বিকল্পটো বিচাৰি পাব
  • আপোনাৰ OpenAI API চাবি টাইপ কৰক
  • ব্যৱহাৰ কৰিবলৈ মডেল নিৰ্বাচন কৰক (ডিফল্ট gpt-3.5-turbo)
  • যদি স্ব-হোষ্ট বা প্ৰক্সি ব্যৱহাৰ কৰে, কাষ্টম API Base URL ছেট কৰিব পাৰে

লক্ষ্য ৰখা বিষয়

  • ডাটা কেৱল ব্ৰাউজাৰৰ লোকেল ষ্ট'ৰেজত সংৰক্ষণ হয়, ৰিফ্ৰেছ বা কেছ মচিলে অন্য ব্যৱহাৰকাৰীৰ প্ৰভাৱ নহয়
  • ৰূপান্তৰিত অডিঅ' ফাইল ব্ৰাউজাৰত প্ৰতক্ষ্যভাৱে প্লে বা লোকেলত ডাউনলোড কৰিব পাৰি
  • বিনামূলীয়া TTS সেৱাত দৈনিক ব্যৱহাৰ সীমা থাকে, অধিক হলে পৰৱৰ্তী দিনলৈ পুৱাই পুনৰ চাৰ্জ হ'ব

এই ছফ্টৱেৰ আৰু ক'ড কেৱল ব্যক্তিগত অধ্যয়ন আৰু পৰীক্ষাৰ বাবে, ডাউনলোড কৰাৰ ২৪ ঘণ্টাৰ ভিতৰত ডিলিট কৰক, ব্যৱসায়িক উদ্দেশ্যে ব্যৱহাৰ কৰিব নোৱাৰি, নতুবা ফলাফলৰ বাবে নিজে দায়ী থাকিব। যিকোনো অবৈধ ব্যৱহাৰৰ বাবে হোৱা আইনী ফলাফলে মোৰ সৈতে সম্পৰ্ক নাথাকে। এই ছফ্টৱেৰ কেতিয়াও মাচুল লোৱা নহ'ব, যদি আপুনি এই ছফ্টৱেৰ ব্যৱহাৰ কৰাৰ আগতে অতিৰিক্ত মাচুল দিবলৈ বাধ্য হয়, বা ক'ড বা প্ৰস্তুত ছফ্টৱেৰ ক্ৰয় কৰে, তেন্তে আপুনি নিশ্চয় ঠগবাজিত পৰিছে!
