TTS-Vue إصدار الويب

🎤 أداة تحويل النص إلى كلام من مايكروسوفت، إصدار الويب، تم تطويرها باستخدام Vue + ElementPlus + Vite.

مثال على الموقع

https://web.tts88.top

واجهة الإصدار الجديد

واجهة الإصدار القديم

الميزات الأساسية

  • 🌐 واجهة ويب بالكامل، لا حاجة لتثبيت تطبيق سطح المكتب
  • 🔊 يدعم عدة أنواع من تحويل النص إلى كلام من مايكروسوفت
  • 🚀 يدعم تكامل واجهة برمجة تطبيقات TTS88
  • 🆓 يدعم تحويل النص إلى كلام مجاني، مع حصة يومية مجانية
  • 🧠 يدعم توليد النصوص عبر OpenAI
  • 🌍 يدعم لغات متعددة: الصينية، الإنجليزية، الإسبانية
  • 🔐 التعرف على بصمة المتصفح، تجربة مستخدم أكثر أماناً

آخر التحديثات

إعادة هيكلة الواجهة وتعزيز الوظائف (v2.2.0)

  • 📱 إعادة هيكلة الواجهة: تثبيت شريط التنقل العلوي، صندوق النص بطريقة التثبيت، تحسين دعم الجوال
  • 🔤 تعزيز SSML: إضافة تنسيق SSML تلقائي، وتحديث SSML حسب الإعدادات تلقائياً
  • 🎵 تحسين المشغل: دمج المشغل في الواجهة الرئيسية وتحسين تجربة تشغيل وتحميل الصوتيات
  • 🎞️ ميزة الترجمة: إضافة وظيفة توليد الترجمة الفورية عبر الإنترنت، لتعزيز إنتاجية المحتوى
  • 🌓 تحسين الثيمات: تحسين الواجهة في الوضع الليلي، وإضافة تبديل الثيم العالمي
  • 🛠️ إعدادات متقدمة: تحسين واجهة خدمة TTS المجانية، إضافة إعدادات السرعة والنبرة
  • 💡 تجربة تفاعل محسّنة: إضافة تلميحات الأدوات لتعزيز إرشاد المستخدم
  • 🔄 تحديث التبعيات: تحديث Vue إلى 3.5.11، وتحديث ElementPlus إلى 2.9.9
  • 🌐 تحديث الروابط: تحديث عنوان مستودع GitHub

دمج خدمة TTS المجانية (v2.1.0)

  • 🆕 ميزة TTS المجانية: لا حاجة لمفتاح API، يوجد حصة مجانية يومية
  • 👤 التعرف على بصمة المتصفح: لضمان الاستخدام العادل لكل مستخدم
  • 🔄 عرض حالة الحصة: عرض عدد الأحرف المجانية المتبقية ووقت إعادة التعيين بشكل فوري
  • 🌐 فحص حالة الخادم: فحص حالة اتصال خادم TTS المجاني تلقائياً
  • 🛡️ تحسين معالجة الأخطاء: رسائل خطأ أكثر وضوحاً وردود حالة أفضل
  • 🎛️ لوحة إعدادات TTS المجانية: تسهيل ضبط الحصة ومشاهدتها للمستخدمين

تحسين الواجهة وتجربة المستخدم (v2.0.0)

  • ✨ تحسين منطقة الإعدادات: لوحة الإعدادات تدعم الطي، يمكن النقر على عنوان "إعدادات الصوت" للتوسيع/الطي
  • 🔘 موضع زر الذكاء: تم نقل زر "بدء التحويل" إلى الأعلى، لا حاجة للتمرير للنقر عليه
  • 💬 تعزيز تلميحات الحالة: إضافة نص تلميحات حالة التحويل بشكل بديهي
  • 🎨 تحسين التأثيرات البصرية: إضافة تأثيرات ضوئية للأزرار لتعزيز تجربة المستخدم
  • 📱 تحسين التخطيط المتجاوب: تحسين العرض على أحجام الشاشات المختلفة

لقد حسّنت هذه التعديلات بشكل كبير من كفاءة الاستخدام، خاصةً بحل مشكلة التصميم التقليدي حيث كان المستخدم يحتاج للتمرير إلى أسفل الصفحة للنقر على زر "بدء التحويل". الآن، بغض النظر عن مكان المستخدم في الصفحة، يمكنه بسهولة بدء عملية التحويل.

النشر

بعد الانتهاء من البناء، قم بنشر محتويات دليل dist على أي خادم ويب ثابت.

إعدادات API

يدعم هذا الإصدار من الويب حالياً خدمة TTS المجانية، وواجهة TTS88 API وواجهة OpenAI API:

إعداد خدمة TTS المجانية

  • في صفحة الإعدادات، ابحث عن خيار "خدمة TTS المجانية"
  • النظام مضبوط افتراضياً على عنوان خادم TTS المجاني
  • يمكنك عرض الحصة المجانية المتبقية لهذا اليوم ووقت إعادة التعيين
  • كل متصفح عميل لديه حصة مجانية مستقلة

إعداد TTS88 API

  • في صفحة الإعدادات، ابحث عن خيار إعداد "رابط API طرف ثالث"
  • أدخل عنوان TTS88 API الخاص بك
  • إذا كان لديك مفتاح API، أدخله في حقل "مفتاح TTS88 API"

إعداد OpenAI API

  • في صفحة الإعدادات، ابحث عن خيار إعدادات OpenAI
  • أدخل مفتاح OpenAI API الخاص بك
  • اختر النموذج الذي تريد استخدامه (الافتراضي هو gpt-3.5-turbo)
  • إذا كنت تستخدم استضافة ذاتية أو وكيل، يمكنك ضبط عنوان API Base URL مخصص

ملاحظات هامة

  • يتم تخزين البيانات فقط في تخزين المتصفح المحلي، إعادة التحميل أو مسح ذاكرة التخزين المؤقت لن تؤثر على المستخدمين الآخرين
  • يمكن تشغيل ملفات الصوت المحولة مباشرة في المتصفح أو تنزيلها محلياً
  • خدمة TTS المجانية لديها حد استخدام يومي، إذا تجاوزت الحد يجب الانتظار حتى إعادة التعيين في اليوم التالي
