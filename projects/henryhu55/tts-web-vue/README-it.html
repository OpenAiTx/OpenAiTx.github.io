Configurazione API

Questa versione Web attualmente supporta il servizio TTS gratuito, l’API TTS88 e l’API OpenAI:

Configurazione del servizio TTS gratuito

  • Nella pagina delle impostazioni trova l’opzione "Servizio TTS gratuito"
  • Il sistema è preconfigurato con l’indirizzo del server TTS gratuito
  • Puoi visualizzare il limite gratuito residuo del giorno e il tempo di reset
  • Ogni client browser ha una quota gratuita indipendente

Configurazione API TTS88

  • Nella pagina delle impostazioni trova l’opzione "URL API di terze parti"
  • Inserisci il tuo indirizzo API TTS88
  • Se hai una chiave API, inseriscila nel campo "Chiave API TTS88"

Configurazione API OpenAI

  • Nella pagina delle impostazioni trova l’opzione di configurazione OpenAI
  • Inserisci la tua chiave API OpenAI
  • Seleziona il modello da utilizzare (predefinito gpt-3.5-turbo)
  • Se utilizzi un proxy o un’istanza self-hosted, puoi impostare una API Base URL personalizzata

Note

  • I dati sono archiviati solo nello storage locale del browser; aggiornare o cancellare la cache non influenzerà altri utenti
  • I file audio convertiti possono essere riprodotti direttamente nel browser o scaricati in locale
  • Il servizio TTS gratuito ha un limite giornaliero di utilizzo, superato il quale occorre attendere il reset del giorno successivo
