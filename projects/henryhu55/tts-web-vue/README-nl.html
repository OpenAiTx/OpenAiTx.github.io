API-configuratie

Deze webversie ondersteunt momenteel gratis TTS-diensten, de TTS88 API en de OpenAI API:

Configuratie van gratis TTS-dienst

  • Zoek de optie "Gratis TTS-dienst" op de instellingenpagina
  • Standaard is er een gratis TTS-serveradres geconfigureerd
  • U kunt het resterende gratis tegoed van de dag en de reset-tijd bekijken
  • Elke browserclient heeft een eigen gratis tegoed

TTS88 API-configuratie

  • Zoek de instelling "Derde partij API URL" op de instellingenpagina
  • Voer uw TTS88 API-adres in
  • Indien u een API-sleutel heeft, vul deze in bij het veld "TTS88 API-sleutel"

OpenAI API-configuratie

  • Zoek de OpenAI-instellingen op de instellingenpagina
  • Voer uw OpenAI API-sleutel in
  • Kies het model dat u wilt gebruiken (standaard is gpt-3.5-turbo)
  • Bij gebruik van zelfgehoste of proxy-oplossing kunt u een aangepaste API Base URL instellen

Opmerkingen

  • Gegevens worden alleen lokaal in de browser opgeslagen, verversen of cache wissen beïnvloedt andere gebruikers niet
  • Omgezette audiobestanden kunnen direct in de browser worden afgespeeld of naar de computer worden gedownload
  • De gratis TTS-dienst heeft een dagelijks gebruikslimiet; na overschrijding moet u wachten tot de volgende dag voor reset
