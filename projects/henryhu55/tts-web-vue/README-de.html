API-Konfiguration

Diese Webversion unterstützt derzeit den kostenlosen TTS-Dienst, die TTS88 API und die OpenAI API:

Konfiguration des kostenlosen TTS-Dienstes

  • Öffnen Sie die Einstellungen und suchen Sie die Option "Kostenloser TTS-Dienst"
  • Das System ist standardmäßig mit der Adresse des kostenlosen TTS-Servers konfiguriert
  • Sie können das verbleibende kostenlose Tageskontingent und die Rücksetzzeit einsehen
  • Jeder Browser-Client verfügt über ein eigenes kostenloses Kontingent

TTS88 API-Konfiguration

  • Öffnen Sie die Einstellungen und suchen Sie die Option "Drittanbieter-API-URL"
  • Geben Sie Ihre TTS88 API-Adresse ein
  • Falls Sie einen API-Schlüssel besitzen, tragen Sie ihn in das Feld "TTS88 API-Schlüssel" ein

OpenAI API-Konfiguration

  • Öffnen Sie die Einstellungen und suchen Sie die OpenAI-Einstellungen
  • Geben Sie Ihren OpenAI API-Schlüssel ein
  • Wählen Sie das gewünschte Modell aus (Standard: gpt-3.5-turbo)
  • Bei Verwendung von Self-Hosting oder Proxy können Sie eine benutzerdefinierte API-Basis-URL einstellen

Hinweise

  • Daten werden ausschließlich im lokalen Speicher des Browsers abgelegt; ein Aktualisieren oder Löschen des Caches hat keine Auswirkungen auf andere Nutzer
  • Die konvertierten Audiodateien können direkt im Browser abgespielt oder lokal heruntergeladen werden
  • Für den kostenlosen TTS-Dienst gilt ein tägliches Nutzungslimit; bei Überschreitung muss bis zur Rücksetzung am Folgetag gewartet werden
