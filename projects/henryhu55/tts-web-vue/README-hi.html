नवीनतम अपडेट

इंटरफ़ेस पुनर्निर्माण और फ़ंक्शन संवर्द्धन (v2.2.0)

  • 📱 UI पुनर्निर्माण: शीर्ष नेविगेशन को फिक्स किया गया, टेक्स्टबॉक्स अब स्टिकी मोड में, मोबाइल अनुकूलन बेहतर किया गया
  • 🔤 SSML संवर्द्धन: नया SSML फॉर्मेटिंग फ़ीचर, सेटिंग बदलने पर SSML अपने आप अपडेट
  • 🎵 प्लेयर अनुकूलन: मुख्य इंटरफ़ेस में प्लेयर इंटीग्रेट किया, ऑडियो प्लेबैक और डाउनलोड अनुभव सुधारा
  • 🎞️ सबटाइटल फ़ीचर: ऑनलाइन सबटाइटल जनरेट करने की सुविधा जोड़ी, कंटेंट क्रिएशन को तेज किया
  • 🌓 थीम अनुकूलन: डार्क मोड इंटरफ़ेस बेहतर, ग्लोबल थीम स्विच जोड़ा गया
  • 🛠️ एडवांस्ड सेटिंग्स: फ्री TTS सेवा इंटरफ़ेस अनुकूलित, स्पीच स्पीड और पिच सेटिंग जोड़ी गई
  • 💡 इंटरएक्टिव अनुभव: टूलटिप फ़ीचर जोड़ा, यूज़र गाइडेंस बेहतर किया
  • 🔄 डिपेंडेंसी अपग्रेड: Vue को 3.5.11 और ElementPlus को 2.9.9 में अपग्रेड किया गया
  • 🌐 लिंक अपडेट: GitHub रिपॉजिटरी लिंक अपडेट किया

फ्री TTS सेवा इंटीग्रेशन (v2.1.0)

  • 🆕 फ्री TTS फ़ीचर: बिना API कुंजी के, प्रतिदिन फ्री कोटा उपलब्ध
  • 👤 ब्राउज़र फिंगरप्रिंट पहचान: प्रत्येक यूज़र को फेयर फ्री कोटा गारंटी
  • 🔄 कोटा स्टेटस डिस्प्ले: शेष फ्री कैरेक्टर और रीसेट टाइम रियल टाइम दिखाएं
  • 🌐 सर्वर स्टेटस चेक: फ्री TTS सर्वर कनेक्शन अपने आप जांचें
  • 🛡️ बेहतर एरर हैंडलिंग: ज्यादा फ्रेंडली एरर मैसेज व स्टेटस फीडबैक
  • 🎛️ फ्री TTS सेटिंग पैनल: यूज़र के लिए कोटा जानकारी सेट/देखना आसान

इंटरफ़ेस और यूज़र एक्सपीरियंस अनुकूलन (v2.0.0)

  • ✨ सेटिंग एरिया अनुकूलन: सेटिंग पैनल फोल्डेबल, "स्पीच सेटिंग" टाइटल पर क्लिक कर एक्सपैंड/फोल्ड करें
  • 🔘 स्मार्ट बटन पोजिशन: "कन्वर्ट स्टार्ट" बटन ऊपर, स्क्रोल की जरूरत नहीं
  • 💬 स्टेटस प्रॉम्प्ट संवर्द्धन: कन्वर्ज़न स्टेटस के लिए स्पष्ट टेक्स्ट जोड़ा गया
  • 🎨 विजुअल इफेक्ट सुधार: बटन में लाइट इफेक्ट ऐनिमेशन, यूज़र अनुभव बेहतर
  • 📱 रिस्पॉन्सिव लेआउट अनुकूलन: विभिन्न स्क्रीन साइज पर डिस्प्ले में सुधार

इन सुधारों ने उपयोग दक्षता को बहुत बढ़ाया है, खासकर उस पारंपरिक डिज़ाइन समस्या का समाधान किया जहां "कन्वर्ट स्टार्ट" बटन दबाने के लिए यूज़र को पेज के नीचे स्क्रोल करना पड़ता था। अब, चाहे यूज़र पेज पर कहीं भी हो, वे आसानी से कन्वर्ज़न शुरू कर सकते हैं।

API कॉन्फ़िगरेशन

यह वेब संस्करण वर्तमान में मुफ्त TTS सेवा, TTS88 API और OpenAI API का समर्थन करता है:

मुफ्त TTS सेवा कॉन्फ़िगरेशन

  • सेटिंग पेज में "मुफ्त TTS सेवा" विकल्प खोजें
  • सिस्टम में डिफ़ॉल्ट रूप से मुफ्त TTS सर्वर पता कॉन्फ़िगर किया गया है
  • आप आज के शेष मुफ्त कोटा और रीसेट समय देख सकते हैं
  • प्रत्येक ब्राउज़र क्लाइंट का अपना अलग मुफ्त कोटा होता है

TTS88 API कॉन्फ़िगरेशन

  • सेटिंग पेज में "थर्ड पार्टी API URL" सेटिंग विकल्प खोजें
  • अपना TTS88 API पता दर्ज करें
  • यदि आपके पास API कुंजी है, तो उसे "TTS88 API कुंजी" फ़ील्ड में दर्ज करें

OpenAI API कॉन्फ़िगरेशन

  • सेटिंग पेज में OpenAI सेटिंग विकल्प खोजें
  • अपना OpenAI API कुंजी दर्ज करें
  • उपयोग करने के लिए मॉडल का चयन करें (डिफ़ॉल्ट gpt-3.5-turbo है)
  • यदि सेल्फ-होस्ट या प्रॉक्सी का उपयोग करते हैं, तो कस्टम API बेस URL सेट कर सकते हैं

ध्यान दें

  • डेटा केवल ब्राउज़र के लोकल स्टोरेज में ही सहेजा जाता है, रिफ्रेश या कैश क्लियर करने पर अन्य उपयोगकर्ताओं पर कोई असर नहीं होगा
  • कन्वर्टेड ऑडियो फाइल ब्राउज़र में सीधा प्ले या लोकल में डाउनलोड की जा सकती है
  • मुफ्त TTS सेवा पर दैनिक उपयोग सीमा है, सीमा पार करने पर अगले दिन रीसेट तक इंतजार करें
