नवीनतम अपडेट

इंटरफ़ेस पुनर्निर्माण और फीचर संवर्धन (v2.2.0)

  • 📱 UI पुनर्निर्माण: स्थायी शीर्ष नेविगेशन, टेक्स्ट बॉक्स के लिए स्टिकी मोड, मोबाइल अनुकूलन बेहतर
  • 🔤 SSML संवर्धन: नया SSML फॉर्मेटिंग फीचर, सेटिंग बदलने पर स्वतः SSML अपडेट
  • 🎵 प्लेयर अनुकूलन: मुख्य इंटरफ़ेस में प्लेयर एकीकृत, ऑडियो प्ले और डाउनलोड अनुभव बेहतर
  • 🎞️ सबटाइटल फीचर: ऑनलाइन सबटाइटल जनरेशन नया फीचर, कंटेंट निर्माण में वृद्धि
  • 🌓 थीम अनुकूलन: डार्क मोड में इंटरफ़ेस प्रदर्शन सुधार, ग्लोबल थीम स्विचिंग जोड़ा गया
  • 🛠️ एडवांस्ड सेटिंग्स: मुफ्त TTS सेवा इंटरफ़ेस अनुकूलित, स्पीच स्पीड और पिच कॉन्फ़िगरेशन नया
  • 💡 इंटरएक्टिव अनुभव: टूलटिप फीचर जोड़ा गया, यूजर ऑपरेशन गाइड बेहतर
  • 🔄 डिपेंडेंसी अपग्रेड: Vue अपडेट किया गया 3.5.11 पर, ElementPlus अपडेट 2.9.9 पर
  • 🌐 लिंक अपडेट: GitHub रिपॉजिटरी पता अपडेट किया गया

मुफ्त TTS सेवा एकीकरण (v2.1.0)

  • 🆕 मुफ्त TTS फीचर: API कुंजी की आवश्यकता नहीं, प्रतिदिन मुफ्त कोटा उपलब्ध
  • 👤 ब्राउज़र फिंगरप्रिंट पहचान: हर यूजर को निष्पक्ष मुफ्त कोटा सुनिश्चित करें
  • 🔄 कोटा स्थिति प्रदर्शन: शेष मुफ्त कैरेक्टर और रीसेट टाइम रियलटाइम दिखाएं
  • 🌐 सर्वर स्थिति जांच: मुफ्त TTS सर्वर कनेक्शन स्थिति स्वतः जांचें
  • 🛡️ एरर हैंडलिंग संवर्धन: अधिक मित्रवत त्रुटि संकेत और स्थिति फीडबैक
  • 🎛️ मुफ्त TTS सेटिंग पैनल: यूजर्स के लिए कॉन्फ़िगरेशन और कोटा जानकारी देखना आसान

इंटरफ़ेस और यूजर एक्सपीरियंस अनुकूलन (v2.0.0)

  • ✨ सेटिंग क्षेत्र अनुकूलन: सेटिंग पैनल अब फोल्ड किया जा सकता है, "वॉयस सेटिंग्स" शीर्षक पर क्लिक करने से यह खुलता/बंद होता है
  • 🔘 स्मार्ट बटन स्थान: "कन्वर्शन शुरू करें" बटन अब शीर्ष पर है, क्लिक करने के लिए स्क्रॉल करने की आवश्यकता नहीं है
  • 💬 स्थिति संकेत बढ़ोतरी: स्पष्ट रूप से कन्वर्शन की स्थिति बताने वाला टेक्स्ट जोड़ा गया है
  • 🎨 दृश्य प्रभाव सुधार: बटन में लाइट इफेक्ट एनिमेशन जोड़ा गया, जिससे यूजर अनुभव बेहतर हुआ है
  • 📱 उत्तरदायी लेआउट अनुकूलन: विभिन्न स्क्रीन साइज पर डिस्प्ले को बेहतर किया गया है

ये सुधार उपयोग की दक्षता को काफी बढ़ाते हैं, खासकर पारंपरिक डिज़ाइन में जहाँ यूजर को "कन्वर्शन शुरू करें" बटन पर क्लिक करने के लिए पेज के नीचे स्क्रॉल करना पड़ता था। अब, यूजर चाहे पेज पर कहीं भी हों, कन्वर्शन प्रक्रिया आसानी से शुरू की जा सकती है।

API कॉन्फ़िगरेशन

यह वेब संस्करण वर्तमान में मुफ्त TTS सेवा, TTS88 API और OpenAI API का समर्थन करता है:

मुफ्त TTS सेवा कॉन्फ़िगरेशन

  • सेटिंग पेज में "मुफ्त TTS सेवा" विकल्प खोजें
  • सिस्टम में डिफ़ॉल्ट रूप से मुफ्त TTS सर्वर पता कॉन्फ़िगर किया गया है
  • आप आज के शेष मुफ्त कोटा और रीसेट समय देख सकते हैं
  • प्रत्येक ब्राउज़र क्लाइंट का अपना अलग मुफ्त कोटा होता है

TTS88 API कॉन्फ़िगरेशन

  • सेटिंग पेज में "थर्ड पार्टी API URL" सेटिंग विकल्प खोजें
  • अपना TTS88 API पता दर्ज करें
  • यदि आपके पास API कुंजी है, तो उसे "TTS88 API कुंजी" फ़ील्ड में दर्ज करें

OpenAI API कॉन्फ़िगरेशन

  • सेटिंग पेज में OpenAI सेटिंग विकल्प खोजें
  • अपना OpenAI API कुंजी दर्ज करें
  • उपयोग करने के लिए मॉडल का चयन करें (डिफ़ॉल्ट gpt-3.5-turbo है)
  • यदि सेल्फ-होस्ट या प्रॉक्सी का उपयोग करते हैं, तो कस्टम API बेस URL सेट कर सकते हैं

ध्यान दें

  • डेटा केवल ब्राउज़र के लोकल स्टोरेज में ही सहेजा जाता है, रिफ्रेश या कैश क्लियर करने पर अन्य उपयोगकर्ताओं पर कोई असर नहीं होगा
  • कन्वर्टेड ऑडियो फाइल ब्राउज़र में सीधा प्ले या लोकल में डाउनलोड की जा सकती है
  • मुफ्त TTS सेवा पर दैनिक उपयोग सीमा है, सीमा पार करने पर अगले दिन रीसेट तक इंतजार करें

ध्यान दें

यह सॉफ्टवेयर और कोड केवल व्यक्तिगत अध्ययन और परीक्षण के लिए है, कृपया डाउनलोड के 24 घंटे के भीतर हटा दें, व्यावसायिक उपयोग के लिए नहीं है, अन्यथा परिणाम के लिए स्वयं जिम्मेदार होंगे। किसी भी गैरकानूनी उपयोग से उत्पन्न कानूनी परिणामों के लिए मेरी कोई जिम्मेदारी नहीं होगी। यह सॉफ्टवेयर कभी भी शुल्क नहीं लेगा, यदि आप इस सॉफ़्टवेयर के उपयोग से पहले कोई अतिरिक्त शुल्क देते हैं, या सोर्स कोड या तैयार सॉफ़्टवेयर के लिए भुगतान करते हैं, तो आपको निश्चित रूप से धोखा दिया गया है!

कृपया स्रोत का उल्लेख करें। समूह में शामिल होने, निजी संदेश आदि के माध्यम से सॉफ़्टवेयर का रिपॉजिटरी, डाउनलोड लिंक या इंस्टॉलेशन पैकेज प्राप्त करने के लिए दूसरों को प्रेरित करना मना है।
