API Yapılandırması

Bu Web sürümü şu anda ücretsiz TTS hizmeti, TTS88 API ve OpenAI API'yi desteklemektedir:

Ücretsiz TTS Hizmeti Yapılandırması

  • Ayarlar sayfasında "Ücretsiz TTS Hizmeti" seçeneğini bulun
  • Sistem varsayılan olarak ücretsiz TTS sunucu adresiyle yapılandırılmıştır
  • O gün kalan ücretsiz kullanım hakkınızı ve sıfırlama zamanını görüntüleyebilirsiniz
  • Her tarayıcı istemcisinin bağımsız ücretsiz kullanım hakkı vardır

TTS88 API Yapılandırması

  • Ayarlar sayfasında "Üçüncü Parti API URL" ayarını bulun
  • TTS88 API adresinizi girin
  • Bir API anahtarınız varsa, "TTS88 API Anahtarı" alanına girin

OpenAI API Yapılandırması

  • Ayarlar sayfasında OpenAI ayarları seçeneğini bulun
  • OpenAI API anahtarınızı girin
  • Kullanmak istediğiniz modeli seçin (varsayılan gpt-3.5-turbo)
  • Kendi sunucunuzu veya proxy'nizi kullanıyorsanız, özel bir API Base URL belirleyebilirsiniz

Dikkat Edilecekler

  • Veriler yalnızca tarayıcı yerel depolamasında saklanır; sayfayı yenilemek veya önbelleği temizlemek diğer kullanıcıları etkilemez
  • Dönüştürülen ses dosyaları doğrudan tarayıcıda oynatılabilir veya yerel olarak indirilebilir
  • Ücretsiz TTS hizmetinin günlük kullanım limiti vardır; aşılırsa ertesi gün sıfırlanmasını beklemeniz gerekir
