Konfiguracja API

Ta wersja webowa obsługuje obecnie darmową usługę TTS, TTS88 API oraz OpenAI API:

Konfiguracja darmowej usługi TTS

  • Na stronie ustawień znajdź opcję "Darmowa usługa TTS"
  • System domyślnie skonfigurował adres serwera darmowej usługi TTS
  • Możesz sprawdzić pozostały dzienny limit darmowy oraz czas resetu
  • Każda przeglądarka posiada niezależny limit darmowy

Konfiguracja TTS88 API

  • Na stronie ustawień znajdź opcję ustawień "URL API strony trzeciej"
  • Wprowadź adres swojego TTS88 API
  • Jeśli posiadasz klucz API, wpisz go w pole "Klucz TTS88 API"

Konfiguracja OpenAI API

  • Na stronie ustawień znajdź opcję ustawień OpenAI
  • Wprowadź swój klucz OpenAI API
  • Wybierz model do użycia (domyślnie gpt-3.5-turbo)
  • W przypadku korzystania z hostingu własnego lub proxy, można ustawić niestandardowy API Base URL

Uwagi

  • Dane są przechowywane wyłącznie w lokalnej pamięci przeglądarki, odświeżenie lub wyczyszczenie pamięci podręcznej nie wpłynie na innych użytkowników
  • Przekonwertowane pliki audio można odtwarzać bezpośrednio w przeglądarce lub pobrać na dysk lokalny
  • Darmowa usługa TTS posiada dzienny limit wykorzystania, po jego przekroczeniu należy poczekać na reset następnego dnia
