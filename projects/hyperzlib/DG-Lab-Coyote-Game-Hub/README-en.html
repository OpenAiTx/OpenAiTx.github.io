Usage (Windows Binary Distribution)

  • Download ``coyote-game-hub-windows-amd64-dist.zip` from Releases: Click to go
  • Extract and run `start.bat` to start the server

Usage (Linux/MacOS Command Line)

  • Install nodejs (for Linux, nvm is recommended; for Mac use `brew install node@22`)
  • Download `coyote-game-hub-nodejs-server.zip` from Releases: Click to go
  • Run `node server/index.js` in the extracted directory

Usage (Compile and Use)

(The following example uses `pnpm` to install dependencies; you can also use `npm` or `yarn`)

  • Enter the `server` directory and run `pnpm install` to install dependencies

  • Enter the `frontend` directory and run `pnpm install` to install dependencies

  • Run `pnpm install` in the project root directory to install dependencies, then run `npm run build` to compile the project

  • Run `npm start` in the project root directory to start the server

  • Open `http://localhost:8920` in the browser to see the control panel

Project Structure

  • `server`: Server-side code
  • `frontend``: Front-end code
