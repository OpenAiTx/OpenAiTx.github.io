C 程式設計專案集

一組為初學者設計的 C 程式專案，讓您學習並練習各種程式設計概念，從基礎演算法到網路程式設計。每個專案皆為獨立單元，包含完整文件與建置系統。

專案總覽

1. SHA-512 實作

一個加密雜湊函式的實作，展示：
  • 位元操作
  • 複雜演算法
  • 記憶體管理
  • 檔案輸入輸出操作

2. HTTP 伺服器

基礎 HTTP 伺服器實作，涵蓋：
  • Socket 程式設計
  • HTTP 協定基礎
  • 檔案提供服務
  • 網路通訊

3. UDP 伺服器-用戶端

簡單的 UDP 通訊系統，教導：
  • UDP socket 程式設計
  • 用戶端-伺服器架構
  • 網路協定
  • 訊息交換

4. 連接埠掃描器

網路診斷工具，展示：
  • TCP socket 程式設計
  • 網路掃描技術
  • 錯誤處理
  • 命令列參數解析

5. Ping 工具實作

自訂 Ping 工具，涵蓋：
  • ICMP 協定
  • Raw socket 程式設計
  • 網路診斷
  • 時間測量

6. 井字遊戲

經典遊戲實作，教導：
  • 多人遊戲伺服器
  • Socket 程式設計
  • 遊戲邏輯
  • 使用者輸入處理

7. 聊天系統

即時聊天應用，展示：
  • TCP socket 程式設計
  • 多線程
  • 即時通訊
  • 用戶端-伺服器架構

8. 詞法分析器

簡易編譯器前端組件，展示：
  • 原始碼的詞法分析
  • 解析技術
  • 編譯器設計原理
  • 字串操作

9. 算術編譯器

簡單編譯器，輸入算術運算式並產生類組合語言程式碼：
  • 詞法分析（標記化）
  • 運算式解析與評估
  • 組合語言程式碼生成
  • 編譯器建構基礎

10. 小行星遊戲

終端機遊戲，控制飛船避開墜落的小行星：

  • 遊戲狀態管理
  • 使用者輸入處理
  • 終端機圖形顯示
  • 即時遊戲機制
  • 簡易碰撞偵測

開始使用

本倉庫中每個專案皆為獨立單元，包含：

  • 原始碼
  • Makefile 或 CMake 配置
  • README，含建置與使用說明
  • 實作細節

開始使用任一專案：
  • 進入專案目錄
  • 閱讀專案的 README.md
  • 遵照建置說明操作
  • 按描述執行程式

先決條件

  • GCC 編譯器（C 專案）
  • G++ 編譯器（C++ 專案）
  • Make 工具
  • CMake（C/C++ 專案）
  • 基本的 C/C++ 程式設計理解
  • 對網路專案：理解基本的網路概念

推薦書籍

要開始學習 C 程式設計，我推薦這本綜合性書籍： The C Programming Language - 學習 C 程式語言基礎和最佳實踐的必備資源。

學習路徑

這些專案按難度遞增排列：

  • 從井字遊戲開始，學習基本程式設計概念
  • 接著實作 SHA-512 演算法
  • 嘗試 UDP 伺服器-客戶端，了解基礎網路
  • 進階到 HTTP 伺服器和聊天系統，學習更複雜的網路技術
  • 最後探索埠掃描器和 Ping，掌握進階網路概念
