Vectarine

Vectarine 是一个专注于超快速原型制作、易用性和乐趣的游戏引擎。

目标重要性排序

  • 你的时间很宝贵
  • Luau 脚本：即时重载和强类型支持
  • 引擎内置资源，快速测试
  • 示例库：从可用模板开始
  • 强大的调试工具：减少调试时间
  • 不限制创造力
  • 访问底层原语
  • 支持 3D 和 2D
  • 性能：以 60 帧渲染数百万实体
  • 覆盖广泛受众
  • 支持网页、Windows、Linux、MacOS
  • 通过共享小体积的压缩包分发你的游戏
  • 免费且开源

开始制作游戏

从 发布页下载引擎的最新版本。

此外，你还需要：

  • 一个文本编辑器。我推荐 Visual Studio Code，但你也可以用记事本或其他任何编辑器。
  • （可选）为你的编辑器安装 Luau 扩展

详见 手册，了解如何使用 Vectarine 制作游戏。 如果您更喜欢观看而不是阅读，这里有一个视频介绍，帮助您快速入门。

如果您有任何问题或想与其他开发者交流，欢迎加入我们的Discord服务器。

以下是有关如何使用和改进引擎的信息。

参与引擎开发

有很多方式可以为引擎做出贡献！

您可以改进文档

  • 您可以为 luau-api 中的各个函数编写文档
  • 您可以澄清并补充手册的章节内容
  • 您可以为图库添加新的示例

您可以修改引擎本身

  • 您可以改进编辑器
  • 您可以为运行时添加新的 API

有关引擎的技术信息，请参阅贡献指南
