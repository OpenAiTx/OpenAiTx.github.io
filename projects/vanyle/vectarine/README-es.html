Vectarine

Vectarine es un motor de juegos enfocado en la creación rápida de prototipos, facilidad de uso y diversión.

Objetivos por importancia

  • Tu tiempo es valioso
  • Scripting Luau: Recarga instantánea y tipado fuerte
  • Activos integrados en el motor para pruebas rápidas
  • Galería de ejemplos: comienza con plantillas funcionales
  • Herramientas de depuración potentes: pierde menos tiempo en bugs
  • No limites la creatividad
  • Acceso a primitivas de bajo nivel
  • Soporte 3d y 2d
  • Rendimiento: Renderiza millones de entidades a 60 fps
  • Alcanzar una amplia audiencia
  • Soporta Web, Windows, Linux, MacOS
  • Distribuye tu juego compartiendo un zip con una huella pequeña.
  • Gratis y de código abierto

Comenzando a crear juegos

Descarga la última versión del motor desde la página de lanzamientos.

Adicionalmente, necesitarás:

  • Un editor de texto. Recomiendo Visual Studio Code, pero puedes usar Notepad o cualquier otro.
  • (Opcional) Instalar una extensión de Luau para tu editor

Consulta El manual para obtener información detallada sobre cómo crear juegos con Vectarine Si prefieres ver en lugar de leer, hay una presentación en video para comenzar.

No dudes en unirte a nuestro servidor de Discord si tienes alguna pregunta o quieres charlar con otros desarrolladores.

A continuación se muestra información sobre cómo trabajar y mejorar el motor.

Ayudando con el motor

¡Hay muchas formas de contribuir al motor!

Puedes mejorar la documentación

  • Puedes documentar funciones individuales dentro de luau-api
  • Puedes aclarar y agregar secciones del manual
  • Puedes agregar nuevos ejemplos a la galería

Puedes cambiar el motor mismo

  • Puedes mejorar el editor
  • Puedes agregar nuevas APIs al runtime

Consulta CONTRIBUTING para información técnica sobre el motor

Tenemos muchas funciones planeadas en la lista TODO, ¡así que puedes elegir issues fáciles para comenzar si quieres ayudar!

Tenemos un servidor de Discord donde puedes discutir funciones y obtener ayuda de otros desarrolladores
