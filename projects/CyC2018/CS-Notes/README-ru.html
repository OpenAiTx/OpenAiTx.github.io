Верстка

Содержание заметок оформлено в соответствии с Руководством по типографике китайского текста, чтобы обеспечить читаемость. Для вставки изображений не используется синтаксис ![](), а применяется тег . Во-первых, это позволяет контролировать размер отображения изображения, а во-вторых, GFM не поддерживает центрирование картинок с помощью
![]()
, поэтому для центрирования используется
. Онлайн-инструмент для верстки: Text-Typesetting.
