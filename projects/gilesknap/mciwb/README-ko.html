mciwb

마인크래프트 인터랙티브 월드 빌더 ===================================

이 프로젝트는 파이썬 프로그래밍 언어를 재미있게 배우기 위한 목적입니다.

경험 있는 파이썬 프로그래머도 이 라이브러리를 사용하여 인터랙티브 파이썬 기능이 포함된 마인크래프트 월드를 만들 수 있습니다.

소스 | :---: | :---: PyPI | pip install mciwb 문서 | 릴리스 |

빠른 시작 ---------

파이썬 학습을 시작하려면, 소개를 참고하세요.

기존 파이썬 개발자는, API 문서를 참고하세요.

예제 빌드 ---------

작동하는 포트쿨리스가 있는 파고다와 성은 모두 파이썬과 현재 버전의 MCIWB로 프로그래밍 방식으로 생성되었습니다.

https://github.com/gilesknap/mciwb/assets/964827/026b3551-f514-403a-bebe-8d168c7170b0

목표 -----

Minecraft Interactive World Builder의 목표는 다음과 같습니다:

  • Python을 사용하여 Minecraft 세계 내에서 무엇이든 생성할 수 있습니다.
  • 세계 내에서 이벤트가 발생할 때 Python 코드를 호출합니다.
  • iPython 프롬프트를 사용하여 Python 명령어를 입력하며 세계와 상호작용합니다.
  • 플레이어가 명령어 표지판을 설치하거나 스위치/레버를 활성화하여 Python 코드를 실행하게 합니다.

Minecraft 서버와 Python 개발자 환경을 설정하는 방법에 대한 모든 지침이 포함되어 있습니다. Python 프로그래밍과 Minecraft Interactive World Builder 기능을 소개하는 단계별 튜토리얼이 있습니다.

.. note::

이 프로젝트는 진행 중입니다. 현재 Python의 기본을 학습할 수 있는 충분한 튜토리얼과 Python 코드로 Minecraft와 상호작용하는 재미를 누릴 수 있는 기능이 있습니다.

튜토리얼을 계속 확장하고 라이브러리에 더 많은 기능을 추가할 예정입니다.

크레딧 -------

이 프로젝트는 Minecraft RCON용 클라이언트 라이브러리를 제공하는 mcipc 없이는 불가능했을 것입니다.

자세한 문서는 https://gilesknap.github.io/mciwb 에서 확인하세요.
