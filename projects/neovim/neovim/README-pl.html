neovim

Dokumentacja | Czat

Neovim to projekt, który ma na celu zdecydowaną refaktoryzację Vim, aby:

  • Uprościć utrzymanie i zachęcić do współpracy
  • Rozdzielić pracę pomiędzy wielu deweloperów
  • Umożliwić [zaawansowane interfejsy użytkownika] bez modyfikacji rdzenia
  • Zmaksymalizować rozszerzalność

Zobacz stronę wiki Wprowadzenie oraz [Roadmap] po więcej informacji.

Funkcje --------

  • Nowoczesne GUI
  • Dostęp do API
z dowolnego języka, w tym C/C++, C#, Clojure, D, Elixir, Go, Haskell, Java/Kotlin, JavaScript/Node.js, Julia, Lisp, Lua, Perl, Python, Racket, Ruby, Rust
  • Wbudowany, skryptowalny emulator terminala
  • Asynchroniczna kontrola zadań
  • Współdzielone dane (shada) pomiędzy wieloma instancjami edytora
  • Obsługa katalogów bazowych XDG
  • Kompatybilny z większością wtyczek Vim, w tym wtyczkami Ruby i Python

Zobacz [:help nvim-features][nvim-features] dla pełnej listy oraz [:help news][nvim-news], aby poznać ważne zmiany w najnowszej wersji!

Instalacja z pakietu --------------------

Gotowe pakiety dla Windows, macOS i Linux znajdują się na stronie Wydania.

[Zarządzane pakiety] są dostępne w [Homebrew], [Debian], [Ubuntu], [Fedora], [Arch Linux], [Void Linux], [Gentoo] i wielu innych!

Instalacja ze źródeł --------------------

Szczegóły znajdziesz w BUILD.md oraz na stronie obsługiwane platformy.

Budowanie wykorzystuje CMake, ale Makefile jest dostępny dla wygody. Po zainstalowaniu zależności uruchom poniższe polecenie.

make CMAKE_BUILD_TYPE=RelWithDebInfo sudo make install

Aby zainstalować do niestandardowej lokalizacji:

make CMAKE_BUILD_TYPE=RelWithDebInfo CMAKE_INSTALL_PREFIX=/pelna/sciezka/ make install

Wskazówki CMake dotyczące inspekcji budowy:

  • cmake --build build --target help wyświetla wszystkie cele budowania.
  • build/CMakeCache.txt (lub cmake -LAH build/) zawiera rozstrzygnięte wartości wszystkich zmiennych CMake.
  • build/compile_commands.json pokazuje pełne wywołania kompilatora dla każdej jednostki translacji.

Przechodzenie z Vim --------------------

Zobacz :help nvim-from-vim po instrukcje.

Struktura projektu --------------

├─ cmake/ Narzędzia CMake ├─ cmake.config/ Definicje CMake ├─ cmake.deps/ Podprojekt pobierający i budujący zależności (opcjonalnie) ├─ runtime/ Wtyczki i dokumentacja ├─ src/nvim/ Kod źródłowy aplikacji (zobacz src/nvim/README.md) │ ├─ api/ Podsystem API │ ├─ eval/ Podsystem Vimscript │ ├─ event/ Podsystem pętli zdarzeń │ ├─ generators/ Generowanie kodu (przed kompilacją) │ ├─ lib/ Ogólne struktury danych │ ├─ lua/ Podsystem Lua │ ├─ msgpack_rpc/ Podsystem RPC │ ├─ os/ Niskopoziomowy kod platformowy │ └─ tui/ Wbudowany interfejs tekstowy └─ test/ Testy (zobacz test/README.md)
