🧠 C.O.R.E.

संदर्भित अवलोकन और पुनःस्मरण इंजन

C.O.R.E एक पोर्टेबल मेमोरी ग्राफ़ है जो आपके एलएलएम इंटरैक्शनों और व्यक्तिगत डेटा से बना है, जिससे आपकी सारी संदर्भ और कार्यप्रवाह की इतिहास किसी भी एआई टूल के लिए एक डिजिटल दिमाग की तरह सुलभ हो जाती है। इससे बार-बार संदर्भ साझा करने की आवश्यकता समाप्त हो जाती है। इसका उद्देश्य है:

  • एकीकृत, पोर्टेबल मेमोरी: आसानी से संदर्भ जोड़ें और याद करें, और अपनी मेमोरी को Claude, Cursor, Windsurf आदि जैसे ऐप्स में जोड़ें।
  • संबंधात्मक, केवल तथ्यों से आगे: CORE आपके ज्ञान को व्यवस्थित करता है, तथ्यों के साथ-साथ उनके संबंधों को भी संग्रहीत करता है, जिससे असली दिमाग जैसा गहरा और समृद्ध स्मरण मिलता है।
  • उपयोगकर्ता के स्वामित्व वाली: आप तय करते हैं कि क्या रखना है, अपडेट या हटाना है, और अपनी मेमोरी को अपनी पसंद के टूल्स में साझा करें और विक्रेता लॉक-इन से मुक्त रहें।

  • मेमोरी के साथ बातचीत: त्वरित अंतर्दृष्टि और समझ के लिए मेमोरी से प्रश्न पूछें

CORE MCP को Cursor के साथ जोड़ना
  • CORE डैशबोर्ड खोलें और API सेक्शन में जाकर एक नया API टोकन जनरेट करें।
    • 

  • Cursor में जाएँ: Settings → Tools & Integrations → New MCP Server।
    • 

  • नीचे दिए गए कॉन्फ़िगरेशन फॉर्मेट का उपयोग करके CORE MCP सर्वर जोड़ें। सुनिश्चित करें कि आप API_TOKEN वैल्यू को चरण 1 में जनरेट किए गए टोकन से बदल दें।
    •    
Cursor में जोड़ने के लिए MCP कॉन्फ़िगरेशन

   `json
   {
     "mcpServers": {
       "memory": {
         "command": "npx",
         "args": ["-y", "@redplanethq/core-mcp"],
         "env": {
           "API_TOKEN": "YOUR_API_TOKEN_HERE",
           "API_BASE_URL": "https://core.heysol.ai",
           "SOURCE": "cursor"
         }
       }
     }
   }
   `

  • सेटिंग्स -> यूज़र रूल्स -> नया रूल -> पर जाएँ और नीचे दिए गए नियम को जोड़ें ताकि आपकी सभी चैट इंटरैक्शन कोर मेमोरी में सहेजी जा सकें।
