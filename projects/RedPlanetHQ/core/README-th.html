🧠 C.O.R.E.

Contextual Observation & Recall Engine

C.O.R.E คือกราฟหน่วยความจำแบบพกพาที่สร้างจากการโต้ตอบกับ llm และข้อมูลส่วนตัวของคุณ ทำให้บริบทและประวัติการทำงานทั้งหมดของคุณเข้าถึงได้กับเครื่องมือ AI ใด ๆ เช่นเดียวกับสมองดิจิทัลของคุณ ซึ่งช่วยขจัดความจำเป็นในการแบ่งปันบริบทซ้ำ ๆ เป้าหมายคือเพื่อให้:

  • หน่วยความจำรวมที่พกพาได้: เพิ่มและเรียกคืนบริบทได้อย่างราบรื่น เชื่อมโยงหน่วยความจำของคุณกับแอปต่าง ๆ เช่น Claude, Cursor, Windsurf และอื่น ๆ
  • ความสัมพันธ์ ไม่ใช่แค่ข้อเท็จจริงเรียบ ๆ: CORE จัดระเบียบความรู้ของคุณ เก็บทั้งข้อเท็จจริงและความสัมพันธ์ เพื่อหน่วยความจำที่ลึกซึ้งยิ่งขึ้นเหมือนสมองจริง
  • ผู้ใช้เป็นเจ้าของ: คุณตัดสินใจว่าจะเก็บ อัปเดต หรือลบอะไร และแบ่งปันหน่วยความจำของคุณกับเครื่องมือที่ต้องการ พร้อมเป็นอิสระจากการผูกขาดผู้ให้บริการ

ดูการสาธิต C.O.R.E

🧩 คุณสมบัติเด่น

  • กราฟหน่วยความจำ: มองเห็นว่าข้อเท็จจริงและความชอบของคุณเชื่อมโยงกันอย่างไร
  • แชทกับหน่วยความจำ: ถามคำถามเกี่ยวกับหน่วยความจำเพื่อรับข้อมูลและความเข้าใจทันที
  • Plug n Play: ใช้หน่วยความจำ CORE กับแอปต่าง ๆ เช่น Cursor, Claude ได้ทันที

☁️ การตั้งค่า C.O.R.E Cloud

  • สมัครใช้งาน Core Cloud แล้วเริ่มสร้างกราฟหน่วยความจำของคุณ
  • เพิ่มข้อความที่คุณต้องการบันทึกในหน่วยความจำ เมื่อคลิกปุ่ม + Add กราฟหน่วยความจำของคุณจะถูกสร้างขึ้น
  • เชื่อมต่อ Core Memory MCP กับ Cursor

  • Docker
  • คีย์ API ของ OpenAI

> หมายเหตุ: เรากำลังปรับปรุงการรองรับโมเดล Llama อย่างต่อเนื่อง ในขณะนี้ C.O.R.E ยังไม่ให้ผลลัพธ์ที่ดีที่สุดกับโมเดล Llama แต่เรากำลังพัฒนาเพื่อให้สามารถใช้งานและได้ผลลัพธ์ที่ดียิ่งขึ้นในอนาคตอันใกล้นี้

การเชื่อมต่อ CORE MCP กับ Cursor
  • เปิดแดชบอร์ด CORE และไปที่ส่วน API เพื่อสร้าง API token ใหม่
    • 

  • ใน Cursor ไปที่: Settings → Tools & Integrations → New MCP Server
    • 

  • เพิ่มเซิร์ฟเวอร์ CORE MCP โดยใช้รูปแบบการกำหนดค่าด้านล่างนี้ อย่าลืมแทนที่ค่า API_TOKEN ด้วยโทเคนที่คุณสร้างในขั้นตอนที่ 1
    •    
ตัวอย่างการกำหนดค่า MCP ที่จะเพิ่มใน Cursor

   `json
   {
     "mcpServers": {
       "memory": {
         "command": "npx",
         "args": ["-y", "@redplanethq/core-mcp"],
         "env": {
           "API_TOKEN": "YOUR_API_TOKEN_HERE",
           "API_BASE_URL": "https://core.heysol.ai",
           "SOURCE": "cursor"
         }
       }
     }
   }
   `

  • ไปที่ การตั้งค่า-> กฎผู้ใช้ -> กฎใหม่ -> และเพิ่มกฎด้านล่างนี้เพื่อให้แน่ใจว่าการสนทนาทั้งหมดของคุณถูกจัดเก็บไว้ในหน่วยความจำ CORE
After every interaction, update the memory with the user's query and the assistant's response to core-memory mcp. sessionId should be the uuid of the conversation ``
