ระบบปฏิบัติการ AI ส่วนตัวของคุณ

ไม่ใช่แชทบอทที่คุณเปิดใช้ แต่เป็น AI ที่เปิดตลอดเวลา คอยสังเกตอยู่เสมอ ตั้งชื่อมัน กำหนดรูปแบบมัน เชื่อมต่อมันกับทุกสิ่งที่คุณใช้ เข้าถึงมันได้ทุกวิธีที่คุณทำงาน โอเพ่นซอร์ส โฮสต์เอง เป็นของคุณตลอดไป

ดูการทำงานจริง

ชมวิดีโอที่ CORE รับงานจากข้อความธรรมดา รวบรวมข้อมูลจาก GitHub และหน่วยความจำ วางแผนการทำงาน เรียกใช้เซสชัน Claude Code และเปิด PR:

เฝ้าดูอยู่เสมอ พร้อมเสมอ

เครื่องมือ AI ส่วนใหญ่มักรอให้ถูกเรียกใช้งาน แต่ CORE เฝ้าดูอยู่ตลอดเวลา

เพียงเชื่อมต่อกับแอปของคุณ CORE จะคอยตรวจสอบทุกกิจกรรม ไม่ว่าจะเป็นอีเมลจากลูกค้า Issue ใหม่ใน GitHub แจ้งเตือนจาก Sentry หรือประชุมจบลง CORE จะเห็น ตรวจสอบหน่วยความจำและทักษะที่ตั้งไว้ แล้วจัดการหรือแจ้งคุณให้ตัดสินใจเอง โดยที่คุณไม่ต้องสั่งงาน มันจะสังเกตเห็นเอง

เมื่อติดตั้งปลั๊กอิน CORE ใน Claude Code, Codex หรือ Cursor การสนทนากับเอเจนต์ของคุณก็จะถูกติดตามด้วย หัวข้อที่คุยกัน การตัดสินใจ โค้ดที่เขียน ทั้งหมดนี้จะถูกบันทึกลงกราฟความรู้ในหน่วยความจำ งานถัดไปที่ CORE รับรู้ได้ทันทีว่าเซสชันก่อนเกิดอะไรขึ้น

เมื่อ CORE ลงมือทำ จะดำเนินการโดยตรง ไม่ว่าจะตอบอีเมล อัปเดต Issue ใน Linear เปิด PR ใน GitHub ส่งข้อความ Slack รันคำสั่งในเทอร์มินัล ควบคุมเบราว์เซอร์ หรือเริ่มเซสชัน Claude Code หรือ Codex จากทุกอินเทอร์เฟซ ส่งข้อความ WhatsApp จากสนามบินแล้ว CORE สามารถเริ่มเซสชันโค้ดและเปิด PR ให้ก่อนขึ้นเครื่อง

คุณเป็นผู้ตัดสินใจว่ามันจะทำงานเองเมื่อไร และควรรอคำสั่งคุณเมื่อไร จัดการแจ้งเตือน Sentry อัตโนมัติแต่ถามก่อนทุกครั้งที่จะ merge อนุมัติแผนก่อนเริ่มโค้ด Require การยืนยันก่อนส่งอีเมลใด ๆ ระดับความอัตโนมัติคุณเป็นผู้กำหนดได้เอง ต่อแอป ต่อภารกิจ ต่อการกระทำ

สี่วิธีในการเข้าถึง

แชทบอทมีหนึ่งอินเทอร์เฟซ ระบบปฏิบัติการมีหลายแบบ

เสียง กด Ctrl+Option บน Mac แล้วพูดสิ่งที่ต้องการให้ทำ CORE จะทำงานเบื้องหลังไม่ขัดจังหวะโฟลว์ของคุณ

Scratchpad เปิดหน้าประจำวันแล้วเขียนว่า [ ] แก้บั๊ก auth จาก issue #47 CORE จะรับรู้ภายใน 3 นาที โหลดบริบทจากรีโปและหน่วยความจำ แล้วร่างแผนงาน

การส่งข้อความ WhatsApp, Slack, Telegram ส่งงานจากสนามบิน จากโทรศัพท์ หรือจากบนเตียง CORE มีข้อมูลบริบทครบถ้วนไม่ว่าข้อความจะมาจากที่ใด

แชท เปิดแดชบอร์ดแล้วพูดคุยโดยตรง เหมือนคุยกับผู้ช่วยทั่วไป เมื่อคุณต้องการถาม-ตอบก่อนมอบหมายงาน

หนึ่ง AI, สี่ช่องทาง, ข้อมูลและบริบทเดียวกันในทุกช่อง

ปรับแต่งให้เป็นของคุณ

ตั้งชื่อให้มัน เลือกวิธีพูด กำหนดกฎที่ต้องปฏิบัติตามทุกที่

เลือกบุคลิกสำเร็จรูปจากห้าแบบ — ประสิทธิภาพแบบ TARS, ความเป็นทางการแบบ Alfred, ความอบอุ่นแบบ Hudson — หรือแต่งเอง บุคลิกจะสะท้อนในทุกงาน ทุกแผน ทุกข้อความที่ส่งแทนคุณ

เลือกเสียงสำหรับการสนทนาแบบพูด หลังจากนั้นมันจะพูดและปฏิบัติตัวตามที่คุณตั้งค่าไว้ในทุกอินเทอร์เฟซ

CORE ขณะปฏิบัติงาน

พูด แล้วกลับมาดู PR

กด Ctrl+Option, พูด: "แก้ race condition ใน checkout flow จาก issue #312"

CORE โหลด issue ดึง commit และเธรด Slack ที่เกี่ยวข้อง ร่างแผน และเริ่ม Claude Code session คุณกลับมาเจอ diff โดยที่ไม่ต้องนั่งหน้าคอม

เขียนงานกลางคืน ทบทวนตอนเช้า

Scratchpad: [ ] ทำงานผ่านงานค้างคืนนี้ เริ่มตั้งแต่ 23.00 น.

CORE ดึงข้อมูลจาก Linear, GitHub และหน่วยความจำ, จัดลำดับความสำคัญ, และดำเนินการขณะคุณหลับ คืนที่ราบรื่นรอให้คุณตรวจสอบ เซสชั่นที่ติดขัดจะกลับมาพร้อมคำถามสั้น ๆ ไม่ใช่แท็บที่หยุดนิ่ง

ส่งข้อความถึงมันจากที่ไหนก็ได้

WhatsApp จากสนามบิน: "ส่งรีแฟกเตอร์ระบบยืนยันตัวตน"

CORE รู้สาขา, บริบท, และความชอบของคุณแล้ว มันทำงานอยู่ใน Railway และเริ่มเซสชั่นก่อนที่คุณจะขึ้นเครื่อง

ตรวจสอบการแจ้งเตือนก่อนจะกลายเป็นเหตุการณ์

Sentry แจ้งเตือนเวลา 2am. CORE ตรวจสอบ, ดึง Trace ที่เกี่ยวข้องและเหตุการณ์ก่อนหน้าจากหน่วยความจำ, เสนอวิธีแก้ไข, และแจ้งคุณทาง Slack: "Issue #847, มีข้อเสนอแก้ไข, รอรีวิวของคุณ" คุณอนุมัติผ่านมือถือ

รับสรุปที่รู้จักสัปดาห์ของคุณอยู่แล้ว

งานประจำ, ทุกเช้า 8am. CORE ดึงข้อมูลจากอีเมล, GitHub, Linear, และ Slack, สรุปเฉพาะที่ต้องการความสนใจ, ข้ามสิ่งที่ไม่จำเป็น, และเปลี่ยน follow-up เป็นงานโดยอัตโนมัติ

เริ่มต้นอย่างรวดเร็ว

โอเพ่นซอร์สและโฮสต์เองได้ ข้อมูลของคุณยังคงอยู่ในโครงสร้างพื้นฐานของคุณ

เลือกเส้นทางของคุณ:

| ฉันต้องการ... | วิธีการ | |---|---| | ทดลองใช้งานบนเครื่องของฉัน | รันการติดตั้งแบบขั้นตอนเดียวด้านล่าง (ต้องมี Docker) | | ติดตั้งบนเซิร์ฟเวอร์หรือ VPS | ติดตั้งด้วย Railway เพียงคลิกเดียว | | ใช้งานแอป Mac | เข้าร่วมรายชื่อรอ |

ติดตั้งและเริ่มต้น CORE:

npm install -g @redplanethq/corebrain && corebrain setup
ตัวช่วยตั้งค่าจะขอไดเรกทอรีติดตั้ง, ผู้ให้บริการ AI, คีย์ API, และโมเดลแชท จากนั้นจะสร้างข้อมูลลับ, เริ่มต้นสแต็ก, และเปิด http://localhost:3033

โดยปกติการติดตั้งในเครื่องใช้เวลาสองสามนาทีหลังจาก Docker ทำงานแล้ว

หรือปรับใช้บน Railway:

เชื่อมต่อเกตเวย์ เพื่อให้ CORE สามารถรันเอเจนต์โค้ดดิ้ง, ควบคุมเบราว์เซอร์ของคุณ, และเข้าถึงโฟลเดอร์ในเครื่อง:

corebrain login
corebrain gateway setup
ข้อกำหนด: Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

คู่มือการโฮสต์เองฉบับเต็ม

ภารกิจแรกของคุณ (2 นาทีหลังจากติดตั้ง):

  • เปิด Scratchpad (หน้าประจำวันของคุณที่ http://localhost:3033)
  • พิมพ์ [ ] สรุปปัญหา GitHub ที่เปิดอยู่ของฉัน หรือภารกิจใด ๆ ที่คุณมักจะทำเอง
  • CORE จะรับงานนี้ภายใน 3 นาที รวบรวมบริบทจากแอปที่เชื่อมต่อ และร่างแผนให้
  • อนุมัติแผน และ CORE จะดำเนินการและนำผลลัพธ์กลับมาให้

เชื่อมต่อแอปแรกของคุณ

เอกสาร

  • Memory - กราฟความรู้แบบชั่วคราว, การจำแนกข้อเท็จจริง, การค้นหาตามเจตนา
  • Scratchpad - พื้นที่ประจำวันสำหรับเริ่มภารกิจและไอเดีย
  • Tasks - แผน, สถานะ, งานประจำ, และบริบทตามภารกิจ
  • Toolkit - กว่า 1000 การกระทำใน 50+ แอปผ่าน MCP
  • CORE Agent - ทริกเกอร์, หน่วยความจำ, เครื่องมือ, และการดำเนินงาน
  • Gateway - การเข้าถึงผ่าน WhatsApp, Slack, Telegram, อีเมล, เว็บ และ API
  • Skills - คำแนะนำที่ใช้ซ้ำสำหรับเวิร์กโฟลว์ที่ทำซ้ำได้
  • Self-hosting - คู่มือการติดตั้งฉบับสมบูรณ์
  • Changelog - สิ่งที่มีการอัปเดต

Benchmark

CORE ทำได้ 88.24% ความแม่นยำเฉลี่ยบน LoCoMo benchmark ครอบคลุม single-hop, multi-hop, open-domain และ temporal reasoning ดูรายละเอียดผลลัพธ์และการเปรียบเทียบ baseline ได้ที่ repo ของ benchmark

  • ใช้ TLS 1.3 ในการส่งข้อมูล

  • ข้อมูลของคุณจะไม่ถูกนำไปใช้ในการฝึกโมเดล
  • โฮสต์ด้วยตัวเองเพื่อการแยกตัวที่สมบูรณ์
