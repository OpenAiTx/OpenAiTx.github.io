🧠 C.O.R.E.

Contextual Observation & Recall Engine

C.O.R.E คือ กราฟความจำแบบพกพาที่สร้างขึ้นจากการโต้ตอบกับ llm และข้อมูลส่วนตัวของคุณ ทำให้บริบทและประวัติการทำงานทั้งหมดของคุณเข้าถึงได้จากเครื่องมือ AI ใดๆ เสมือนสมองดิจิทัลของคุณ ซึ่งช่วยขจัดความจำเป็นในการแบ่งปันบริบทซ้ำๆ จุดมุ่งหมายคือเพื่อมอบ:

  • หน่วยความจำแบบรวมศูนย์และพกพาได้: เพิ่มและเรียกคืนบริบทได้อย่างราบรื่น และเชื่อมต่อหน่วยความจำของคุณข้ามแอป เช่น Claude, Cursor, Windsurf และอื่นๆ
  • ความสัมพันธ์ ไม่ใช่แค่ข้อเท็จจริงแยกเดี่ยว: CORE จัดระเบียบความรู้ของคุณ โดยจัดเก็บทั้งข้อเท็จจริงและความสัมพันธ์เพื่อสร้างหน่วยความจำที่ลึกซึ้งและสมบูรณ์ยิ่งขึ้นเหมือนสมองจริง
  • ผู้ใช้เป็นเจ้าของ: คุณตัดสินใจได้ว่าจะเก็บ อัปเดต หรือลบอะไร รวมถึงแบ่งปันหน่วยความจำกับเครื่องมือที่คุณต้องการ และเป็นอิสระจากการผูกขาดผู้ให้บริการ

ชมตัวอย่าง C.O.R.E

🧩 คุณสมบัติหลัก

  • กราฟความจำ: เห็นภาพว่าข้อเท็จจริงและความชอบของคุณเชื่อมโยงกันอย่างไร
  • แชทกับความจำ: ถามคำถามเกี่ยวกับความจำเพื่อข้อมูลเชิงลึกและความเข้าใจทันที
  • Plug n Play: ใช้งาน CORE memory กับแอปอย่าง Cursor, Claude ได้ทันที

☁️ การตั้งค่า C.O.R.E Cloud

  • สมัครใช้งาน Core Cloud และเริ่มสร้างกราฟความจำของคุณ
  • เพิ่มข้อความที่คุณต้องการบันทึกในหน่วยความจำ เมื่อคลิกปุ่ม + Add กราฟความจำของคุณจะถูกสร้างขึ้น
  • เชื่อมต่อ Core Memory MCP กับ Cursor

#### ข้อกำหนดเบื้องต้น

  • Docker
  • OpenAI API Key

> หมายเหตุ: เรากำลังพัฒนาเพื่อรองรับโมเดล Llama ให้ดียิ่งขึ้น ขณะนี้ C.O.R.E ยังไม่สามารถให้ผลลัพธ์ที่ดีที่สุดกับโมเดล Llama แต่เรากำลังดำเนินการเพื่อให้รองรับและใช้งานได้ดีขึ้นในอนาคตอันใกล้

  • คัดลอก Environment Variables

คัดลอกไฟล์ environment ตัวอย่างไปที่ .env:

   cp .env.example .env
   ``

  • สร้างบัญชีด้วย Magic Link

  • กรอกอีเมลของคุณแล้วคลิกปุ่ม Send a Magic Link

  • คัดลอก magic link จากบันทึกในเทอร์มินัล แล้วเปิดลิงก์ในเบราว์เซอร์ของคุณ

การเชื่อมต่อ CORE MCP กับ Cursor
  • เปิดแดชบอร์ด CORE และไปที่ส่วน API เพื่อสร้าง API token ใหม่
    • 

  • ใน Cursor ไปที่: Settings → Tools & Integrations → New MCP Server
    • 

  • เพิ่มเซิร์ฟเวอร์ CORE MCP โดยใช้รูปแบบการกำหนดค่าด้านล่างนี้ อย่าลืมแทนที่ค่า API_TOKEN ด้วยโทเคนที่คุณสร้างในขั้นตอนที่ 1
    •    
ตัวอย่างการกำหนดค่า MCP ที่จะเพิ่มใน Cursor

   `json
   {
     "mcpServers": {
       "memory": {
         "command": "npx",
         "args": ["-y", "@redplanethq/core-mcp"],
         "env": {
           "API_TOKEN": "YOUR_API_TOKEN_HERE",
           "API_BASE_URL": "https://core.heysol.ai",
           "SOURCE": "cursor"
         }
       }
     }
   }
   `

  • ไปที่ การตั้งค่า-> กฎผู้ใช้ -> กฎใหม่ -> และเพิ่มกฎด้านล่างนี้เพื่อให้แน่ใจว่าการสนทนาทั้งหมดของคุณถูกจัดเก็บไว้ในหน่วยความจำ CORE
After every interaction, update the memory with the user's query and the assistant's response to core-memory mcp. sessionId should be the uuid of the conversation ``
