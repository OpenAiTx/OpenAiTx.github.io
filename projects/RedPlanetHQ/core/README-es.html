Visión general

Problema

Los desarrolladores pierden tiempo volviendo a explicar el contexto a las herramientas de IA. ¿Llegaste al límite de tokens en Claude? Empieza de nuevo y pierdes todo. ¿Cambias de ChatGPT/Claude a Cursor? Explica tu contexto otra vez. Tus conversaciones, decisiones y conocimientos desaparecen entre sesiones. Con cada nueva herramienta de IA, el coste del cambio de contexto aumenta.

Solución - CORE (Motor de Observación y Recuperación Contextual)

CORE es una capa de memoria unificada y persistente de código abierto para todas tus herramientas de IA. Tu contexto te acompaña de Cursor a Claude, de ChatGPT a Claude Code. Un solo grafo de conocimiento recuerda quién dijo qué, cuándo y por qué. Conéctate una vez, recuerda en todas partes. Deja de gestionar el contexto y empieza a construir.

🚀 Comienza

Construye tu grafo de memoria unificado en 5 minutos:

  • Regístrate en core.heysol.ai y crea tu cuenta
  • Agrega tu primera memoria - comparte contexto sobre ti

  • Visualiza tu grafo de memoria y observa cómo CORE forma automáticamente conexiones entre hechos
  • Pruébalo - pregunta "¿Qué sabes sobre mí?" en la sección de conversación
  • Conecta tus herramientas:
  • Claude y Cursor - programación con contexto
  • CLaude Code CLI y Gemini CLI - programación en terminal con memoria
  • Agrega la extensión de navegador - lleva tu memoria a cualquier sitio web
  • Linear, Github - añade contexto de proyecto automáticamente

Cómo CORE crea memoria

La canalización de ingestión de CORE tiene cuatro fases diseñadas para capturar el contexto evolutivo:

  • Normalización: Vincula nueva información al contexto reciente, divide documentos largos en fragmentos coherentes manteniendo referencias cruzadas, y estandariza términos para que cuando CORE extraiga conocimiento, trabaje con una entrada limpia y contextualizada en lugar de texto desordenado.
  • Extracción: Extrae significado del texto normalizado identificando entidades (personas, herramientas, proyectos, conceptos), convirtiéndolas en declaraciones con contexto, fuente y tiempo, y mapeando relaciones. Por ejemplo, “Escribimos CORE en Next.js” se convierte en: Entidades (Core, Next.js), Declaración (CORE fue desarrollado usando Next.js), y Relación (fue desarrollado usando).
  • Resolución: Detecta contradicciones, rastrea cómo evolucionan las preferencias, y preserva múltiples perspectivas con procedencia en vez de sobrescribirlas, para que la memoria refleje todo tu recorrido, no solo la última instantánea.
  • Integración de Grafo: Conecta entidades, declaraciones y episodios en un grafo de conocimiento temporal que vincula hechos con su contexto e historia, convirtiendo datos aislados en una red viva de conocimiento que los agentes pueden usar realmente.

El resultado: En lugar de una base de datos plana, CORE te brinda una memoria que crece y evoluciona contigo, preservando contexto, evolución y propiedad para que los agentes puedan realmente usarla.

Cómo CORE recuerda desde la memoria

Cuando le haces una pregunta a CORE, no solo busca texto; explora todo tu grafo de conocimiento para encontrar las respuestas más útiles.

  • Búsqueda: CORE revisa la memoria desde múltiples ángulos a la vez: búsqueda por palabras clave para coincidencias exactas, búsqueda semántica para ideas relacionadas aunque se expresen diferente, y recorrido de grafo para seguir enlaces entre conceptos conectados.
  • Reordenar: Los resultados obtenidos se reordenan para resaltar los más relevantes y diversos, asegurando que no solo veas coincidencias obvias sino también conexiones más profundas.
  • Filtrado: CORE aplica filtros inteligentes basados en tiempo, confiabilidad y fuerza de relación, para que solo surja el conocimiento más significativo.
  • Salida: Obtienes tanto hechos (declaraciones claras) como episodios (el contexto original de donde provienen), así que el recuerdo siempre está fundamentado en el contexto, el tiempo y la historia.

El resultado: CORE no solo recuerda hechos, los recuerda en el contexto, tiempo e historia adecuados, para que los agentes respondan como tú recordarías.

🔒 Seguridad

CORE toma la seguridad en serio. Implementamos prácticas de seguridad estándar de la industria para proteger tus datos:

  • Cifrado de Datos: Todos los datos en tránsito (TLS 1.3) y en reposo (AES-256)
  • Autenticación: OAuth 2.0 y autenticación por enlace mágico
  • Control de Acceso: Aislamiento basado en espacios de trabajo y permisos basados en roles
  • Reporte de Vulnerabilidades: Por favor reporta problemas de seguridad a harshith@tegon.ai

Para información detallada de seguridad, consulta nuestra Política de Seguridad.
