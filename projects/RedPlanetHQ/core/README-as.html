কাম কৰিবলৈ চাওক

COREয়ে এটা সাধাৰণ পাঠ-কাৰ্য গ্ৰহণ কৰে, GitHub আৰু স্মৃতিৰ পৰা প্ৰসংগ সংগ্ৰহ কৰে, কামৰ পৰিকল্পনা কৰে, Claude Code চেশন চলায়, আৰু এটা PR খোলে:

সদায় চকু ৰাখে। সদায় সাজু।

বেশী AI টুলৰ বাবে অনুৰোধৰ অপেক্ষা থাকে। COREয়ে চকু ৰাখে।

এইটো আপোনাৰ এপচমূহত সংযোগ কৰক আৰু ই সকলোৰ ওপৰত সক্ৰিয়তা পৰ্যবেক্ষণ কৰে। এজন ক্লায়েন্টৰ পৰা ইমেইল আহে, এটা GitHub ইস্যু নিৰ্ধাৰিত হয়, Sentry এলাৰ্ট যায়, এটা মিটিং শেষ হয়। COREয়ে এইটো দেখে, আপোনাৰ স্মৃতি আৰু নিৰ্ধাৰিত কৌশল পৰীক্ষা কৰে, আৰু নিজেই ইয়াক সমাধান কৰে বা আপোনাৰ বিচাৰলৈ আগবঢ়ায়। আপুনি ইয়াক আৰম্ভ নকৰে। ই নিজেই লক্ষ্য কৰে।

Claude Code, Codex, বা Cursor-ত CORE প্লাগইন স্থাপন কৰক আৰু আপোনাৰ এজেন্ট কথোপকথনসমূহও COREয়ে পৰ্যবেক্ষণ কৰে। আলোচনা কৰা প্ৰসংগ, কৰা সিদ্ধান্ত, লিখা ক'ড, সকলোয়ে স্মৃতি জ্ঞান গ্ৰাফত যোগ হয়। COREয়ে যি পৰবর্তী কাৰ্য গ্ৰহণ কৰে সেইটো ইতিমধ্যে শেষ চেশনটোত কি হৈছিল জানে।

COREয়ে কাৰ্য কৰে, সোজাসুজি কৰে। ইমেইল উত্তৰ দিব পাৰে, Linear ইস্যু আপডেট কৰিব পাৰে, GitHub PR জমা দিব পাৰে, Slack বাৰ্তা পাঠাব পাৰে, টার্মিনেল কমাণ্ড চলাব পাৰে, এটা ব্ৰাউজাৰ চালাব পাৰে, আৰু যিকোনো ইণ্টাৰফেচৰ পৰা Claude Code বা Codex চেশন আৰম্ভ কৰিব পাৰে। বিমানবন্দৰত WhatsApp বাৰ্তা পাঠাও আৰু COREয়ে ক'ডিং চেশন চলাই থাকিব আৰু PR খুলি থাকিব আপোনাৰ উৰাৰ আগতে।

আপুনি নিৰ্ণয় কৰে ক'ত ই নিজৰ পৰা কাম কৰে আৰু ক'ত আপোনাৰ আহ্বানৰ অপেক্ষা কৰে। Sentry এলাৰ্টসমূহ স্বয়ংক্ৰিয়ভাৱে হেণ্ডল কৰক, কিন্তু সদায় মাৰ্জ কৰাৰ আগতে সোধক। ক'ডিং ছেছন আৰম্ভ হোৱাৰ আগতে এটা প্লান অনুমোদন কৰক। যিকোনো ইমেইল পঠিওৱাৰ আগতে নিশ্চিতকৰণৰ প্ৰয়োজন। স্বায়ত্তশাসনৰ স্তৰ আপোনাৰ ইচ্ছামতে, প্ৰতিটো টাস্ক, প্ৰতিটো এপ, প্ৰতিটো কাৰ্য্য অনুসৰি নিৰ্ধাৰণ কৰক।

চাৰি টা উপায়ত ইয়াক পাব

এটা চেটবটত এটা ইন্টাৰফেচ থাকে। এটা OS-ত বহুত থাকে।

ভয়েছ। Mac-ত Ctrl+Option টিপি ক'ব লাগিব কি কৰিব লাগে। CORE-এ ই আপোনাৰ ফ্ল' ভংগ নকৰাকৈ ব্যাকগ্ৰাউণ্ডত চলায়।

Scratchpad. আপোনাৰ দৈনিক পৃষ্ঠাত খুলক আৰু লিখক [ ] Fix the auth bug from issue #47। CORE-এ ৩ মিনিটৰ ভিতৰত এইটো ধৰি লৈ, ৰেপ' আৰু মেম'ৰিৰ পৰা প্ৰসংগ লোড কৰে আৰু এটা প্লান খচৰা কৰে।

Messaging. WhatsApp, Slack, Telegram। বিমানবন্দৰ, ফোন, বিচনাত থাকোঁতেই এটা টাস্ক পঠিয়াওক। CORE-এ আপোনাৰ সম্পূৰ্ণ প্ৰসংগ লৈ থাকে, বার্তা যিখন ঠাইৰ পৰা আহে সেইটো নির্বিশেষে।

Chat. ডেশব'ৰ্ড খুলি সোধক, যিকোনো সহকাৰীৰ দৰে। দায়িত্ব দিয়াৰ আগতে সামনা-সামনি কথোপকথন কৰিবলৈ ইচ্ছা কৰিলে।

এটা AI, চাৰি টা ইন্টাৰফেচ, সকলোত একেটা মেম'ৰি আৰু প্ৰসংগ।

ইয়াক নিজৰ দৰে বনাওক

এটা নাম দিয়ক। কিদৰে কথা কয় বাছনি কৰক। যিকোনো ঠাইত অনুসৰণ কৰিবলগীয়া নিয়মবোৰ স্থাপন কৰক।

পাঁচটা সংলগ্ন ব্যক্তিত্বৰ ভিতৰত বাছনি কৰক — TARS-ৰ শুকান দক্ষতা, Alfred-ৰ বিশ্বস্ত নিয়মিততা, Hudson-ৰ উষ্ণ ব্যৱহাৰিকতা — বা নিজৰ ব্যক্তিত্ব লিখক। এই ব্যক্তিত্বই ই চলোৱা প্ৰতিটো টাস্ক, খচৰা কৰা প্ৰতিটো প্লান, আপোনাৰ হৈ পঠিওৱা প্ৰতিটো বার্তাত ওলাই আহে।

শব্দেৰে কথোপকথনৰ বাবে ইয়াৰ স্বৰ বাছনি কৰক। তাৰ পৰা, সকলো ইন্টাৰফেচত ইয়াৰ আওয়াজ আৰু আচৰণ আপোনাৰ কনফিগাৰৰ সৈতে মিলি থাকে।

| মই কৰিব খোজো... | কেনেকৈ | |---|---| | মোৰ মেশিনত চেষ্টা কৰক | তলৰ এক-পদক্ষেপ ইনষ্টল চলাওক (Docker প্ৰয়োজন) | | ছাৰ্ভাৰ বা VPS-ত স্থাপন কৰক | এক-ক্লিক Railway স্থাপন | | Mac এপ ব্যৱহাৰ কৰক | প্ৰত্যাশা তালিকাত যোগদান কৰক |

CORE ইনষ্টল আৰু আৰম্ভ কৰক:

npm install -g @redplanethq/corebrain && corebrain setup
চেটআপ উইজাৰ্ডে ইনষ্টল ডাইৰেক্টৰি, AI প্ৰভাইডাৰ, API কী, আৰু চেট মডেল বিচাৰে। ই গোপন তথ্য সৃষ্টি কৰে, ষ্টেক আৰম্ভ কৰে, আৰু http://localhost:3033 খোলে।

অধিকাংশ স্থানীয় ইনষ্টলেশ্যন ডকৰ চলি থাকিলে কেইমিনিটমান সময় লয়।

অথবা Railway-ত ডিপ্লয় কৰক:

এটা গেটৱে সংযোগ কৰক যাতে CORE-এ কোডিং এজেণ্ট চলাব পাৰে, আপোনাৰ ব্ৰাউজাৰ নিয়ন্ত্ৰণ কৰিব পাৰে, আৰু স্থানীয় ফ'ল্ডাৰসমূহত প্ৰৱেশ কৰিব পাৰে:

corebrain login
corebrain gateway setup
আৱশ্যকতাসমূহ: Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

  • CORE 3 মিনিটৰ ভিতৰত এইটো ধৰি লয়, সংযুক্ত এপসমূহৰ পৰা প্ৰসংগ সংগ্ৰহ কৰে, আৰু এটা পৰিকল্পনা খচৰা কৰে
  • পৰিকল্পনাটো অনুমোদন কৰক আৰু CORE এইটো চলায় আৰু ফলাফল উভতি আনে

  • Memory - সাময়িক জ্ঞান গ্ৰাফ, তথ্য শ্ৰেণীবিভাজন, উদ্দেশ্য-চালিত উদ্ধাৰ
  • Scratchpad - দৈনিক পৃষ্ঠত য'ত কাম আৰু আইডিয়াসমূহ আৰম্ভ হয়
  • Tasks - পৰিকল্পনা, অৱস্থা, পুনৰাবৃত্তিৰ কাম, আৰু টাস্ক-স্কোপড প্ৰসংগ
  • Toolkit - MCP ৰ জৰিয়তে ৫০+ এপত ১০০০+ কাৰ্য
  • CORE Agent - ট্রিগাৰ, স্মৃতি, টুল, আৰু কাৰ্য সম্পাদন
  • Gateway - WhatsApp, Slack, Telegram, ইমেইল, ৱেব, আৰু API ৰ জৰিয়তে প্ৰৱেশ
  • Skills - পুনৰাবৃত্তিযোগ্য ৱাৰ্কফ্ল'ৰ বাবে পুনঃব্যৱহাৰযোগ্য নিৰ্দেশনা
  • Self-hosting - সম্পূৰ্ণ ডিপ্লয়মেন্ট গাইড
  • Changelog - কি কি ৰিলিজ হৈছে
