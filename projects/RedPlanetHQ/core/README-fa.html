مرور کلی

مسئله

توسعه‌دهندگان زمان زیادی را صرف توضیح مجدد زمینه به ابزارهای هوش مصنوعی می‌کنند. به محدودیت توکن در Claude رسیدید؟ باید از نو شروع کنید و همه چیز را از دست بدهید. از ChatGPT/Claude به Cursor رفتید؟ باید دوباره زمینه را توضیح دهید. مکالمات، تصمیمات و بینش‌های شما بین جلسات ناپدید می‌شوند. با هر ابزار هوش مصنوعی جدید، هزینه تغییر زمینه بیشتر می‌شود.

راه حل - CORE (موتور مشاهده و یادآوری زمینه‌ای)

CORE یک لایه حافظه یکپارچه، منبع‌باز و پایدار برای تمامی ابزارهای هوش مصنوعی شماست. زمینه شما از Cursor تا Claude تا ChatGPT تا Claude Code با شما همراه است. یک گراف دانش به خاطر می‌سپارد چه کسی، چه زمانی و چرا چه چیزی گفته است. یک بار متصل شوید، همه جا به خاطر بسپارید. مدیریت زمینه را کنار بگذارید و شروع به ساخت کنید.

🚀 شروع کار

گراف حافظه یکپارچه خود را در ۵ دقیقه بسازید:

  • ثبت‌نام کنید در core.heysol.ai و حساب خود را بسازید
  • اولین حافظه خود را اضافه کنید - اطلاعاتی درباره خودتان به اشتراک بگذارید

  • گراف حافظه خود را مشاهده کنید و ببینید چگونه CORE به طور خودکار بین حقایق ارتباط ایجاد می‌کند
  • آزمایش کنید - در بخش گفتگو بپرسید «درباره من چه می‌دانی؟»
  • به ابزارهای خود متصل شوید:
  • Claude و Cursor - کدنویسی با زمینه
  • Claude Code CLI و Codex CLI - کدنویسی مبتنی بر ترمینال با حافظه
  • افزونه مرورگر را اضافه کنید - حافظه خود را به هر وبسایتی ببرید
  • Linear، Github - افزودن زمینه پروژه به صورت خودکار

🧩 ویژگی‌های کلیدی

🧠 حافظه یکپارچه و قابل حمل:

حافظه خود را در Cursor، Windsurf، Claude Desktop، Claude Code، Gemini CLI، Kiro AWS، VS Code و Roo Code از طریق MCP اضافه و فراخوانی کنید

🕸️ گراف دانش زمانی و متجسم‌شده:

داستان پشت هر حقیقت را به خاطر بسپارید—پیگیری کنید چه کسی، چه زمانی و چرا چیزی را گفته است با روابط غنی و منشأ کامل، نه فقط ذخیره‌سازی سطحی

🌐 افزونه مرورگر:

گفتگوها و محتوای ChatGPT، Grok، Gemini، توییتر، یوتیوب، پست‌های وبلاگ و هر صفحه وب را مستقیماً در حافظه CORE خود ذخیره کنید.

نحوه استفاده از افزونه

  • افزونه را دانلود کنید از فروشگاه Chrome.
  • وارد داشبورد CORE شوید
  • به تنظیمات (پایین سمت چپ) بروید
  • به کلید API → تولید کلید جدید → نام‌گذاری “extension.”
  • افزونه را باز کنید، کلید API خود را جای‌گذاری کنید و ذخیره نمایید.

چگونه CORE حافظه ایجاد می‌کند

خط لوله دریافت CORE چهار مرحله دارد که برای ثبت زمینه در حال تحول طراحی شده‌اند:

  • نرمال‌سازی: اطلاعات جدید را به زمینه اخیر پیوند می‌دهد، اسناد طولانی را به بخش‌های منسجم تقسیم می‌کند و ارجاعات متقابل را حفظ می‌کند، و اصطلاحات را استاندارد می‌سازد تا زمانی که CORE دانش را استخراج می‌کند، با ورودی تمیز و زمینه‌دار کار کند نه متن بی‌نظم.
  • استخراج: با شناسایی نهادها (افراد، ابزارها، پروژه‌ها، مفاهیم) معنا را از متن نرمال‌شده بیرون می‌کشد، آن‌ها را به گزاره‌هایی با زمینه، منبع و زمان تبدیل می‌کند و روابط را نگاشت می‌کند. برای مثال، «ما CORE را در Next.js نوشتیم» تبدیل می‌شود به: نهادها (Core، Next.js)، گزاره (CORE با استفاده از Next.js توسعه داده شد)، و رابطه (توسعه داده شد با).
  • حل و فصل: تناقضات را شناسایی می‌کند، ردیابی می‌کند که ترجیحات چگونه تغییر می‌کنند، و دیدگاه‌های متعدد را با منشأ حفظ می‌کند تا حافظه منعکس‌کننده کل مسیر شما باشد، نه فقط آخرین تصویر.
  • ادغام گراف: نهادها، گزاره‌ها و اپیزودها را به یک گراف دانش زمانی متصل می‌کند که واقعیت‌ها را به زمینه و تاریخچه‌شان پیوند می‌دهد و داده‌های جدا را به یک شبکه زنده دانش تبدیل می‌کند که عامل‌ها واقعاً می‌توانند از آن استفاده کنند.

نتیجه: به جای یک پایگاه داده تخت، CORE به شما حافظه‌ای می‌دهد که با شما رشد و تغییر می‌کند - با حفظ زمینه، تکامل و مالکیت تا عامل‌ها واقعاً بتوانند از آن استفاده کنند.

چگونه CORE از حافظه بازیابی می‌کند

وقتی از CORE سوالی می‌پرسید، فقط متن را جستجو نمی‌کند - بلکه در کل گراف دانش شما کندوکاو می‌کند تا مفیدترین پاسخ‌ها را پیدا کند.

  • جستجو: CORE حافظه را همزمان از چند زاویه بررسی می‌کند - جستجوی کلیدواژه برای تطابق‌های دقیق، جستجوی معنایی برای ایده‌های مرتبط حتی اگر متفاوت بیان شده باشند، و پیمایش گراف برای دنبال کردن پیوندهای بین مفاهیم متصل.
  • رتبه‌بندی مجدد: نتایج بازیابی‌شده دوباره مرتب می‌شوند تا مرتبط‌ترین و متنوع‌ترین موارد برجسته شوند و مطمئن شوید فقط تطابق‌های آشکار را نمی‌بینید بلکه ارتباطات عمیق‌تر را هم مشاهده می‌کنید.
  • فیلتر کردن: CORE فیلترهای هوشمند بر اساس زمان، قابلیت اعتماد و قدرت رابطه اعمال می‌کند تا فقط معنادارترین دانش ظاهر شود.
  • خروجی: شما هم اطلاعات (بیانیه‌های واضح) و هم اپیزودها (زمینه اصلی که از آن آمده‌اند) را دریافت می‌کنید، بنابراین یادآوری همیشه مبتنی بر زمینه، زمان و داستان است.

نتیجه: CORE فقط اطلاعات را یادآوری نمی‌کند - بلکه آن‌ها را در زمینه، زمان و داستان درست بازیابی می‌کند تا عامل‌ها بتوانند همانطور که شما به یاد می‌آورید پاسخ دهند.

🔒 امنیت

CORE امنیت را جدی می‌گیرد. ما برای محافظت از داده‌های شما، شیوه‌های امنیتی استاندارد صنعت را اجرا می‌کنیم:

  • رمزگذاری داده‌ها: تمام داده‌های در حال انتقال (TLS 1.3) و در حالت سکون (AES-256)
  • احراز هویت: OAuth 2.0 و احراز هویت با لینک جادویی
  • کنترل دسترسی: جداسازی مبتنی بر فضای کاری و مجوزهای مبتنی بر نقش
  • گزارش‌دهی آسیب‌پذیری: لطفاً مشکلات امنیتی را به harshith@tegon.ai گزارش دهید

برای اطلاعات کامل امنیتی، به سیاست امنیتی ما مراجعه کنید.
