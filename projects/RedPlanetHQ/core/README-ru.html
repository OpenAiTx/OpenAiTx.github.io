🧠 C.O.R.E.

Contextual Observation & Recall Engine

C.O.R.E — это портативный граф памяти, построенный на основе ваших взаимодействий с LLM и личных данных, делающий весь ваш контекст и историю работы доступными для любого AI-инструмента, словно цифровой мозг. Это устраняет необходимость повторного предоставления контекста. Цель — предоставить:

  • Единая, портативная память: Легко добавляйте и вспоминайте контекст, объединяйте свою память между такими приложениями, как Claude, Cursor, Windsurf и другими.
  • Реляционная, а не просто факты: CORE структурирует ваши знания, храня не только факты, но и их взаимосвязи для более глубокой и богатой памяти, как настоящий мозг.
  • Память принадлежит пользователю: Вы решаете, что сохранять, обновлять или удалять, а также делиться памятью между нужными инструментами, без привязки к одному поставщику.

🧩 Ключевые возможности

  • Граф памяти: Визуализируйте, как ваши факты и предпочтения связаны между собой
  • Чат с памятью: Задавайте вопросы о памяти для мгновенного получения информации и понимания
  • Plug n Play: Мгновенно используйте память CORE в таких приложениях, как Cursor, Claude

  • Зарегистрируйтесь в Core Cloud и начните строить свой граф памяти.
  • Добавьте текст, который хотите сохранить в памяти. После нажатия на кнопку + Add будет сгенерирован ваш граф памяти.

💻 Локальная установка C.O.R.E

#### Необходимые условия

  • Docker
  • Ключ API OpenAI

> Примечание: Мы активно работаем над улучшением поддержки моделей Llama. В данный момент C.O.R.E не обеспечивает оптимальных результатов с моделями на базе Llama, но мы стремимся обеспечить лучшую совместимость и качество вывода в ближайшем будущем.

Подключение CORE MCP к Cursor
  • Откройте панель CORE и перейдите в раздел API для создания нового API-токена.
    • 

  • В Cursor перейдите: Settings → Tools & Integrations → New MCP Server.
    • 

  • Добавьте сервер CORE MCP, используя приведенный ниже формат конфигурации. Не забудьте заменить значение API_TOKEN на токен, созданный на шаге 1.
    •    
Конфигурация MCP для добавления в Cursor

   `json
   {
     "mcpServers": {
       "memory": {
         "command": "npx",
         "args": ["-y", "@redplanethq/core-mcp"],
         "env": {
           "API_TOKEN": "YOUR_API_TOKEN_HERE",
           "API_BASE_URL": "https://core.heysol.ai",
           "SOURCE": "cursor"
         }
       }
     }
   }
   `

  • Перейдите в Настройки -> Пользовательские правила -> Новое правило -> и добавьте следующее правило, чтобы все ваши чат-взаимодействия сохранялись в памяти CORE
After every interaction, update the memory with the user's query and the assistant's response to core-memory mcp. sessionId should be the uuid of the conversation ``
