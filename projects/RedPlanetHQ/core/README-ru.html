Что такое CORE

CORE — это открытый операционный слой для работы, ориентированной на ИИ.

Вы пишете, что нужно сделать, в черновике. CORE берет задание, загружает контекст из памяти и подключенных приложений, составляет план, запускает нужного агента через шлюз, решает препятствия по мере возможности и возвращается, когда требуется человеческое решение.

Это не чат-бот, которому вы постоянно даёте подсказки. Это слой, который запоминает, координирует, действует и эскалирует.

Архитектура

| | | |---|---| | Следит | Контекст из ваших AI-бесед через MCP, 50+ подключённых приложений и, на Mac, из любого приложения, которому вы явно предоставили доступ. | | Запоминает | Граф знаний, который отслеживает не только, что вы сказали, но и что вы решили, когда и почему, в каждом инструменте и разговоре. | | Действует | Совершает прямые действия в ваших подключённых приложениях и делегирует длительную работу кодовым и браузерным агентам через шлюз. | | Политики | Потоки утверждения, правила эскалации, планы и журналы аудита, чтобы автономия оставалась подконтрольной. |

CORE решает, что может делать безопасно, спрашивает перед чувствительными действиями и оставляет след, который вы можете проверить.

CORE в действии

Делегируйте задачу по кодированию и вернитесь к PR.

Сообщите CORE, что нужно сделать. Он собирает контекст из вашего репозитория, приложений и памяти, составляет план, запускает сессию Claude Code или Codex, решает проблемы там, где может, и возвращает PR. Вы проверяете результат после завершения.

[ ] Исправить гонку в процессе оформления заказа из задачи #312

Освободите свой бэклог, пока вы спите.

Задайте регулярную задачу для обработки бэклога в определённое время. CORE работает с ним, пока вы офлайн. Готовые задачи ждут проверки утром. Застрявшие сессии возвращаются с конкретным вопросом, а не с зависшей вкладкой.

[ ] Проработать ночной бэклог начиная с 23:00

Расследуйте оповещения до того, как они станут помехой.

Задайте регулярную задачу для мониторинга Sentry, логов или любого источника оповещений. Когда что-то срабатывает, CORE исследует, собирает связанные трассы и прошлые инциденты и решает, стоит ли обработать или эскалировать.

Оповещение Sentry срабатывает в 2 часа ночи. CORE исследует, предлагает решение и отправляет вам сообщение в Slack для проверки.

Получайте утренний бриф, который знает вашу работу.

Задайте регулярную задачу для сбора данных из email, GitHub, Linear и Slack каждое утро. CORE суммирует, что требует внимания, пропускает то, что не важно, и автоматически превращает последующие действия в задачи.

Делегируйте откуда угодно.

Создавайте задачи из Slack, WhatsApp, Telegram, email или веба. Шлюз работает в Docker или Railway, так что CORE может подхватить работу даже когда ваш ноутбук закрыт.

Чем НЕ является CORE

| | | |---|---| | Не обёртка для RAG. | Память — это не просто встроенные фрагменты. Это граф знаний, который отслеживает, что вы решили, когда и почему. | | Не конструктор процессов. | Нет перетаскивания DAG. Вы пишете, что нужно сделать. CORE сам определяет процесс и спрашивает, когда требуется решение. | | Не очередной Devin. | CORE предлагает планы, вы утверждаете. CORE просит разблокировки, вы решаете. CORE приносит PR, вы проверяете. Агенты не сливают самостоятельно. | | Не закрытый облачный ассистент. | CORE — это open source, самохостинг, независимость от моделей и ориентация на вашу инфраструктуру. |

Быстрый старт

Открытый исходный код и самохостинг. Ваши данные остаются в вашей инфраструктуре.

Установите и запустите CORE:

npm install -g @redplanethq/corebrain && corebrain setup
Мастер установки запрашивает каталог установки, поставщика ИИ, API-ключ и модель чата. Он генерирует секреты, запускает стек и открывает http://localhost:3033.

Большинство локальных установок занимают несколько минут после запуска Docker.

Или разверните на Railway:

Подключите шлюз, чтобы CORE мог управлять вашим браузером, запускать агентов для программирования и получать доступ к локальным папкам:

corebrain login
corebrain gateway setup

Требования: Docker 20.10+, Docker Compose 2.20+, 4 vCPU / 8GB RAM

Во что мы верим

  • Чат — это интерфейс. Не операционная система.

  • Работа должна переходить от намерения к действию без вашего участия как связующего звена.
