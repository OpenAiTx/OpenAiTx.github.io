ClickHouse® 是一個開源的列式資料庫管理系統，能夠實時生成分析性數據報告。

如何安裝（Linux, macOS, FreeBSD）

curl https://clickhouse.com/ | sh

實用連結

  • 官方網站 在主頁上提供 ClickHouse 的快速高層次概述。
  • ClickHouse Cloud 由創建者和維護者打造的 ClickHouse 服務。
  • 教學 展示如何設定及查詢一個小型 ClickHouse 叢集。
  • 文件 提供更深入的資訊。
  • YouTube 頻道 有大量關於 ClickHouse 的影片內容。
  • Slack 和 Telegram 可與 ClickHouse 用戶進行即時聊天。
  • 部落格 包含各種 ClickHouse 相關文章，以及活動公告與報告。
  • Bluesky 和 X 提供短訊新聞。
  • 程式碼瀏覽器 (github.dev) 具備語法高亮，由 github.dev 提供支持。
  • 聯絡方式 可協助解答您的任何問題。
