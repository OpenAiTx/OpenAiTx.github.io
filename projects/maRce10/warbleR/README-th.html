warbleR: เพิ่มประสิทธิภาพการวิเคราะห์เสียงชีวภาพ ================

warbleR ถูกออกแบบมาเพื่อ อำนวยความสะดวกในการวิเคราะห์โครงสร้างของสัญญาณเสียงสัตว์ใน R ผู้ใช้สามารถรวบรวมไฟล์บันทึกเสียงนกที่เข้าถึงได้แบบเปิด หรือป้อนข้อมูลของตนเอง เข้าสู่เวิร์กโฟลว์ที่ช่วยให้แสดงภาพสเปกโตรแกรมและ วัดค่าพารามิเตอร์เสียง warbleR ใช้งาน เครื่องมือวิเคราะห์เสียงพื้นฐานจากแพ็กเกจ seewave และมีเครื่องมือใหม่ สำหรับวิเคราะห์โครงสร้างเสียง เครื่องมือเหล่านี้รองรับ การวิเคราะห์ชุดข้อมูลเสียงแบบกลุ่ม

คุณสมบัติหลักของแพ็กเกจ ได้แก่:

  • เครื่องมือหลากหลายสำหรับวัดโครงสร้างเสียง
  • การใช้ลูปเพื่อประมวลผลสัญญาณเสียงตามที่อ้างอิงใน
ตารางเลือก
  • การสร้างภาพในไดเรกทอรีทำงานพร้อมสเปกโตรแกรมเพื่อ
ให้ผู้ใช้จัดระเบียบข้อมูลและตรวจสอบการวิเคราะห์เสียง

แพ็กเกจนี้มีฟังก์ชันเพื่อ:

  • สำรวจและดาวน์โหลดไฟล์บันทึกเสียงนกจาก Xeno‐Canto
  • สำรวจ จัดระเบียบ และจัดการไฟล์เสียงหลายไฟล์
  • ตรวจจับสัญญาณโดยอัตโนมัติ (ทั้งในด้านความถี่และเวลา) (แต่ควรตรวจสอบแพ็กเกจ R
ohun สำหรับการใช้งานที่ละเอียดและเป็นมิตรยิ่งขึ้น)
  • สร้างสเปกโตรแกรมของบันทึกเสียงทั้งหมดหรือสัญญาณแต่ละรายการ

  • รันมาตรการโครงสร้างสัญญาณเสียงที่แตกต่างกัน
  • ประเมินประสิทธิภาพของวิธีการวัดผล
  • จัดทำรายการสัญญาณ
  • อธิบายลักษณะโครงสร้างระดับต่าง ๆ ในสัญญาณเสียง
  • วิเคราะห์เชิงสถิติการประสานเสียงคู่
  • รวมฐานข้อมูลและตารางคำอธิบายประกอบ

ฟังก์ชันส่วนใหญ่สามารถขนานงานได้ ซึ่ง ช่วยกระจายงานไปยังโปรเซสเซอร์หลายตัวเพื่อเพิ่มประสิทธิภาพการคำนวณ มีเครื่องมือสำหรับประเมินประสิทธิภาพของการวิเคราะห์ในแต่ละขั้นตอนด้วย

การติดตั้ง

ติดตั้ง/โหลดแพ็กเกจจาก CRAN ดังนี้:

`` r install.packages("warbleR")

load package

library(warbleR)

github คุณจะต้องใช้แพ็กเกจ R
remotes:
r remotes::install_github("maRce10/warbleR")

load package

library(warbleR) ``

การใช้งาน

แพ็กเกจนี้มีเอกสารประกอบหลายฉบับที่อธิบายคุณสมบัติหลักของแพ็กเกจ โดย แนะนำ warbleR ให้ภาพรวมของฟังก์ชันการทำงานของแพ็กเกจ เอกสารประกอบ รูปแบบข้อมูลการกำกับคำอธิบาย ให้รายละเอียดเกี่ยวกับรูปแบบที่จำเป็นสำหรับข้อมูลนำเข้าการกำกับคำอธิบาย นอกจากนี้ยังมี เอกสารประกอบเพิ่มเติมของแพ็กเกจ อีกสามฉบับพร้อมตัวอย่างวิธีการจัดระเบียบฟังก์ชันในเวิร์กโฟลว์การวิเคราะห์เสียง

คำอธิบายเต็มของแพ็กเกจ (แม้จะค่อนข้างเก่า) สามารถดูได้ใน บทความวารสารนี้

แพ็กเกจอื่นๆ

แพ็กเกจ seewave และ tuneR มีฟังก์ชันหลากหลายสำหรับวิเคราะห์และปรับแต่งเสียง โดยส่วนใหญ่จะทำงานกับวัตถุคลื่นเสียงที่นำเข้าแล้วในสภาพแวดล้อม R แพ็กเกจ baRulho เน้นการวัดการเสื่อมสภาพของสัญญาณเสียงที่เกิดจากถิ่นที่อยู่อาศัย โดยมีรูปแบบข้อมูลนำเข้าและส่งออกคล้ายกับของ warbleR แพ็กเกจ Rraven ช่วยแลกเปลี่ยนข้อมูลระหว่าง R และ ซอฟต์แวร์วิเคราะห์เสียง Raven (Cornell Lab of Ornithology) และมีประโยชน์มากในการนำ Raven มาใช้เป็นเครื่องมือกำกับคำอธิบายในเวิร์กโฟลว์การวิเคราะห์เสียงใน R แพ็กเกจ ohun ใช้สำหรับตรวจจับเหตุการณ์เสียงโดยอัตโนมัติ ให้ฟังก์ชันสำหรับวินิจฉัยและปรับแต่งกระบวนการตรวจจับ dynaSpec ช่วยสร้างสเปกโตรแกรมแบบไดนามิก (เช่น วิดีโอสเปกโตรแกรม)

การอ้างอิง

กรุณาอ้างอิง warbleR ดังนี้

Araya-Salas, M. และ Smith-Vidaurre, G. (2017), *warbleR: an r package to streamline analysis of animal acoustic signals*. Methods Ecol Evol. 8, 184-191.

หมายเหตุ: กรุณาอ้างอิงแพ็กเกจ tuneR และ seewave ด้วย หากคุณใช้ฟังก์ชันที่สร้างสเปกโตรแกรมหรือวัดเสียง
