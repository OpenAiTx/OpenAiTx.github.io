Cosas Que Todo Programador Debería Saber :thinking:

Una colección de cosas (principalmente técnicas) que todo desarrollador de software debería saber.

:point_up: Estos son recursos que puedo recomendar a cualquier programador sin importar su nivel de habilidad o stack tecnológico

Altamente subjetivo :bomb:. No respaldado por la ciencia. No tiene un orden particular :recycle:

¿Te gusta? :star: dale una estrella y comparte con un desarrollador amigo. ¿No te gusta? Mira el perrito :dog:

*P.D. No necesitas saber todo esto de memoria para ser programador. ¡Pero conocer estas cosas te ayudará a mejorar! :muscle:*

P.P.D. ¡Se aceptan contribuciones!
