नीचे बताए गए तरीकों में से कोई एक चुनें

[!TIP]
यदि आप Markdown को इमेज में बदलने की सुविधा चाहते हैं, तो nonebot-plugin-deepseek[image] इंस्टॉल करें

[सिफारिश] nb-cli का उपयोग कर इंस्टॉल करें Bot की रूट डायरेक्टरी में कमांड लाइन खोलें, और निम्नलिखित कमांड दर्ज करें

nb plugin install nonebot-plugin-deepseek

पैकेज प्रबंधक का उपयोग करके इंस्टॉल करें

pip install nonebot-plugin-deepseek

or, use poetry

poetry add nonebot-plugin-deepseek

or, use pdm

pdm add nonebot-plugin-deepseek

NoneBot परियोजना की मुख्य निर्देशिका में स्थित कॉन्फ़िगरेशन फ़ाइल खोलें, और [plugin] अनुभाग में अतिरिक्त लिखें

plugins = ["nonebot_plugin_deepseek"]

⚙️ कॉन्फ़िगरेशन

प्रोजेक्ट की कॉन्फ़िगरेशन फ़ाइल में नीचे दी गई सेटिंग्स जोड़ें

[!note]
api_key कृपया DeepSeek ओपन प्लेटफॉर्म से प्राप्त करें
enable_models CustomModel संरचना की डिक्शनरी है, यदि स्थानीय मॉडल को जोड़ने की आवश्यकता नहीं है तो इसमें बदलाव आवश्यक नहीं
यदि स्थानीय मॉडल जोड़ना हो तो देखें: 👉 Wiki - स्थानीय मॉडल को जोड़ना

| कॉन्फ़िगरेशन | आवश्यक | डिफ़ॉल्ट मान | विवरण | | :------------------------------: | :---: | :------------------------------------------------------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------: | | deepseek__api_key | हाँ | नहीं | API कुंजी | | deepseek__enable_models | नहीं | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | सक्षम किए गए मॉडल कॉन्फ़िगरेशन विवरण | | deepseek__prompt | नहीं | नहीं | मॉडल प्रीसेट | | deepseek__stream | नहीं | False | क्या स्ट्रीमिंग सक्षम करें | | deepseek__timeout | नहीं | {"api_request": 100, "user_input": 60} | टाइमआउट सेटिंग | | deepseek__md_to_pic | नहीं | False | क्या Markdown को इमेज में बदलें | | deepseek__enable_send_thinking | नहीं | False | क्या Thinking Chain भेजें |

🎉 उपयोग

[!note]
कृपया अपने COMMAND_START और ऊपर दिए गए कॉन्फ़िगरेशन की जांच करें। यहां डिफ़ॉल्ट रूप से / का उपयोग किया गया है

सहायता

deepseek --help
         --balance
         [...content] 
           --use-model [model]
           --with-context
           --render | -r
         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']

问答

/deepseek [内容]

शॉर्टकट कमांड: /ds [सामग्री] या टेक्स्ट संदेश का उत्तर दें

बहु-चरण संवाद

/deepseek --with-context [内容]

त्वरित निर्देश: /ds --with-context [सामग्री] /बहु-चरण संवाद

गहन विचार

/deepseek [内容] --use-model deepseek-v4-pro

快捷指令：/深度思考 [内容]

余额

权限：SUPERUSER

/deepseek --balance

快捷指令：/ds --balance /余额

设置

权限：设置默认模型 指令仅 SUPERUSER 可用

# 查看支持的模型列表
/deepseek model -l|--list

设置默认模型

/deepseek model --set-default [模型名]

设置 Markdown 转图片

/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']

शॉर्टकट निर्देश: /मॉडल सूची /डिफ़ॉल्ट मॉडल सेट करें [मॉडल नाम]

कस्टम शॉर्टकट निर्देश

यह सुविधा Alconna शॉर्टकट निर्देश पर निर्भर करती है। कस्टम निर्देश में COMMAND_START नहीं होता, आवश्यकता होने पर इसे मैन्युअल रूप से भरना होगा

# 增加
/deepseek --shortcut <自定义指令> /deepseek

删除

/deepseek --shortcut delete <自定义指令>

列出

/deepseek --shortcut list

例子:

user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
