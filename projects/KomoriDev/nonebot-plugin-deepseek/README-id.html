💿 Instalasi

Pilih salah satu metode yang disebutkan di bawah ini

[!TIP]
Jika ingin mengaktifkan fitur konversi Markdown ke gambar, perlu memasang nonebot-plugin-deepseek[image]

[Direkomendasikan] Instalasi menggunakan nb-cli Buka command line di direktori root Bot, lalu masukkan perintah berikut untuk menginstal

nb plugin install nonebot-plugin-deepseek

Instalasi menggunakan manajer paket

pip install nonebot-plugin-deepseek

or, use poetry

poetry add nonebot-plugin-deepseek

or, use pdm

pdm add nonebot-plugin-deepseek

Buka file konfigurasi di direktori root proyek NoneBot, lalu tambahkan penulisan pada bagian [plugin]

plugins = ["nonebot_plugin_deepseek"]

⚙️ Konfigurasi

Tambahkan konfigurasi pada tabel di bawah ini ke dalam file konfigurasi proyek

[!note]
api_key silakan dapatkan dari DeepSeek Platform Terbuka
enable_models adalah kamus dengan struktur CustomModel, jika tidak perlu mengintegrasikan model lokal maka tidak perlu diubah
Jika ingin mengintegrasikan model lokal, silakan lihat: 👉 Wiki - Integrasi Model Lokal

| Konfigurasi | Wajib | Nilai Default | Keterangan | | :-----------------------------------: | :----: | :--------------------------------------------------------------------: | :-----------------------------------------------------------------------------------------------------------------------------------------------------------: | | deepseek__api_key | Ya | Tidak ada | API Key | | deepseek__enable_models | Tidak | [{ "name": "deepseek-v4-flash" }, { "name": "deepseek-v4-pro" }] | Model yang diaktifkan Penjelasan Konfigurasi | | deepseek__prompt | Tidak | Tidak ada | Preset Model | | deepseek__stream | Tidak | False | Apakah streaming diaktifkan | | deepseek__timeout | Tidak | {"api_request": 100, "user_input": 60} | Pengaturan Timeout | | deepseek__md_to_pic | Tidak | False | Apakah Markdown ke Gambar diaktifkan | | deepseek__enable_send_thinking | Tidak | False | Apakah mengirim rantai pemikiran |

🎉 Penggunaan

[!note]
Silakan periksa COMMAND_START dan item konfigurasi di atas. Di sini default menggunakan /

Bantuan

deepseek --help
         --balance
         [...content] 
           --use-model [model]
           --with-context
           --render | -r
         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']

问答

/deepseek [内容]

Perintah cepat: /ds [konten] atau balas pesan teks

Percakapan Multi-putaran

/deepseek --with-context [内容]

Perintah cepat: /ds --with-context [konten] /percakapan multi-putaran

Pemikiran Mendalam

/deepseek [内容] --use-model deepseek-v4-pro

快捷指令：/深度思考 [内容]

余额

权限：SUPERUSER

/deepseek --balance

快捷指令：/ds --balance /余额

设置

权限：设置默认模型 指令仅 SUPERUSER 可用

# 查看支持的模型列表
/deepseek model -l|--list

设置默认模型

/deepseek model --set-default [模型名]

设置 Markdown 转图片

/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']

Perintah cepat: /daftar-model /atur-model-default [nama model]

Perintah Cepat Kustom

Fitur ini bergantung pada Alconna Perintah Cepat. Perintah kustom tidak menggunakan COMMAND_START, jika diperlukan harus diisi secara manual

# 增加
/deepseek --shortcut <自定义指令> /deepseek

删除

/deepseek --shortcut delete <自定义指令>

列出

/deepseek --shortcut list

例子:

user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
