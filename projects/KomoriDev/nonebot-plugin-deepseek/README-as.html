💿 স্থাপন

তলত উল্লেখ কৰা পদ্ধতিসমূহৰ যিকোনো এটা বাছি লব পাৰে

[!TIP]
Markdown-টো চিত্ৰলৈ ৰূপান্তৰ কৰিবলৈ, nonebot-plugin-deepseek[image] স্থাপন কৰিব লাগিব

[সুপাৰিশ] nb-cli ব্যৱহাৰ কৰি স্থাপন কৰক Bot-ৰ মূল ডাইৰেক্টৰীত কমাণ্ড লাইন খোলক, তলৰ নিৰ্দেশনা প্ৰৱিষ্ট কৰক

nb plugin install nonebot-plugin-deepseek

পেকেজ মেনেজাৰ ব্যৱহাৰ কৰি ইনষ্টল কৰক

pip install nonebot-plugin-deepseek

or, use poetry

poetry add nonebot-plugin-deepseek

or, use pdm

pdm add nonebot-plugin-deepseek
NoneBot প্ৰকল্পৰ মূল ডিৰেক্টৰীৰ কনফিগাৰেচন ফাইল খোলক, [plugin] অংশত অতিৰিক্তভাৱে লিখক

plugins = ["nonebot_plugin_deepseek"]

api_key অনুগ্ৰহ কৰি DeepSeek খোলা প্লাটফৰ্ম ৰ পৰা সংগ্ৰহ কৰক
enable_models হৈছে CustomModel গঠনযুক্ত এটা অভিধান, স্থানীয় মডেল সংযোগৰ প্ৰয়োজন নোহোৱালে এইটো সলনি কৰাৰ প্ৰয়োজন নাই
স্থানীয় মডেল সংযোগ কৰিবলৈ, অনুগ্ৰহ কৰি চাওক: 👉 Wiki - স্থানীয় মডেল সংযোগ

🎉 ব্যৱহাৰ

[!note]
অনুগ্ৰহ কৰি আপোনাৰ COMMAND_START আৰু ওপৰৰ সংৰূপসমূহ পৰীক্ষা কৰক। ইয়াত ডিফল্ট ৰূপে / ব্যৱহাৰ কৰা হৈছে

সহায়

deepseek --help
         --balance
         [...content] 
           --use-model [model]
           --with-context
           --render | -r
         model
           --list | -l
           --set-default [model]
           --render-markdown ['enable' | 'disable' | 'on' | 'off']

问答

/deepseek [内容]

দ্ৰুত আদেশ: /ds [সমগ্ৰী] বা পাঠ্য বার্তা উত্তৰ কৰক

বহু-পৰ্যায় আলোচনা

/deepseek --with-context [内容]

দ্ৰুত নির্দেশনা: /ds --with-context [সমগ্ৰী] /বহু-চক্ৰ সংলাপ

গভীৰ চিন্তা

/deepseek [内容] --use-model deepseek-v4-pro

快捷指令：/深度思考 [内容]

余额

权限：SUPERUSER

/deepseek --balance

快捷指令：/ds --balance /余额

设置

权限：设置默认模型 指令仅 SUPERUSER 可用

# 查看支持的模型列表
/deepseek model -l|--list

设置默认模型

/deepseek model --set-default [模型名]

设置 Markdown 转图片

/deepseek model --render-markdown ['enable' | 'disable' | 'on' | 'off']

দ্ৰুত নিৰ্দেশনা: /মডেল তালিকা /ডিফল্ট মডেল ছেট কৰক [মডেল নাম]

কাষ্টম দ্ৰুত নিৰ্দেশনা

এই বৈশিষ্ট্যটো Alconna দ্ৰুত নিৰ্দেশনাৰ ওপৰত নিৰ্ভৰ কৰে। কাষ্টম নিৰ্দেশনাত COMMAND_START নাথাকে, প্ৰয়োজন হ'লে নিজেৰে যোগ দিব লাগিব

# 增加
/deepseek --shortcut <自定义指令> /deepseek

删除

/deepseek --shortcut delete <自定义指令>

列出

/deepseek --shortcut list

例子:

user: /deepseek --shortcut /chat /deepseek --use-model deepseek-v4-flash
bot: deepseek::deepseek 的快捷指令: "/chat" 添加成功
user: /chat
bot: (使用模型 deepseek-v4-flash)
