वेबचैट

किसी भी वेबसाइट पर किसी से भी चैट करें

यह एक गुमनाम चैट ब्राउज़र एक्सटेंशन है जो विकेंद्रीकृत और सर्वर रहित है, जिसमें एंड-टू-एंड एन्क्रिप्टेड संचार के लिए WebRTC का उपयोग किया गया है। यह गोपनीयता को प्राथमिकता देता है, क्योंकि सभी डेटा स्थानीय रूप से संग्रहीत होते हैं।

इसका उद्देश्य किसी भी वेबसाइट पर चैट रूम की सुविधा जोड़ना है, जिससे आप कभी अकेले महसूस नहीं करेंगे।

इंस्टॉल करें

स्टोर से इंस्टॉल करें

  • क्रोम वेब स्टोर
  • एज वेब स्टोर
  • फायरफॉक्स ऐडऑन

मैनुअल इंस्टॉलेशन

  • GitHub रिपॉजिटरी पर जाएँ (रिलीज़)
  • "Assets" बटन पर क्लिक करें और "web-chat-*.zip" चुनें
  • ZIP फ़ाइल को अपने कंप्यूटर के एक फ़ोल्डर में निकालें
  • अपने ब्राउज़र में एक्सटेंशन प्रबंधन पृष्ठ खोलें (आमतौर पर chrome://extensions/)
  • "डेवलपर मोड" सक्षम करें
  • "लोड अनपैक्ड" पर क्लिक करें और जो फ़ोल्डर आपने निकाला है, उसे चुनें

उपयोग

एक्सटेंशन इंस्टॉल करने के बाद, आपको किसी भी वेबसाइट के नीचे-दाएँ कोने में एक भूत आइकन दिखाई देगा। उस पर क्लिक करें, और आप उसी साइट पर दूसरों के साथ खुशी-खुशी चैट कर सकते हैं!

दिग्गजों के कंधों पर खड़े होकर

विकेंद्रीकृत चैट के अच्छे विचार के अलावा, इसमें कुछ शानदार तकनीकों का भी उपयोग किया गया है।

  • remesh: जावास्क्रिप्ट में एक फ्रेमवर्क जो DDD सिद्धांतों को लागू करता है, जिससे UI और लॉजिक का सही अलगाव प्राप्त होता है। इससे UI भाग को आसानी से लागू किया जा सकता है, जैसे कि इसे Vue में फिर से लिखना, क्योंकि यह UI से स्वतंत्र है।

  • shadcn/ui: एक सुंदर UI लाइब्रेरी और नो-इंस्टॉल अवधारणा की अग्रणी, जो स्टाइल कस्टमाइज़ करने में अतुलनीय सुविधा प्रदान करती है।

  • wxt: यह ब्राउज़र एक्सटेंशन बनाने के लिए अब तक का सबसे अच्छा फ्रेमवर्क है जिसका मैंने उपयोग किया है।

  • ~~trystero: विकेंद्रीकृत संचार को लागू करने के लिए मुख्य निर्भरता, जो IPFS, टोरेंट, Nostr आदि जैसे विकेंद्रीकृत नेटवर्क से कनेक्शन सक्षम करता है।~~
  • Artico: एक लचीला लाइब्रेरी सेट जो आपको अपनी खुद की WebRTC-आधारित सॉल्यूशन बनाने में मदद करता है

  • ugly-avatar: इसका उपयोग शानदार रैंडम अवतार बनाने के लिए करें।
