Gemini CLI

यह रिपोजिटरी Gemini CLI को शामिल करती है, जो एक कमांड-लाइन AI वर्कफ़्लो टूल है जो आपके टूल्स से कनेक्ट होता है, आपके कोड को समझता है और आपके वर्कफ़्लो को तेज़ करता है।

Gemini CLI के साथ आप:

  • Gemini के 1M टोकन कॉन्टेक्स्ट विंडो में और उससे परे बड़े कोडबेस को क्वेरी और एडिट कर सकते हैं।
  • Gemini की मल्टीमोडल क्षमताओं का उपयोग करते हुए PDF या स्केच से नए ऐप्स जेनरेट कर सकते हैं।
  • ऑपरेशनल कार्यों को स्वचालित कर सकते हैं, जैसे पुल रिक्वेस्ट को क्वेरी करना या जटिल रीबेस को संभालना।
  • टूल्स और MCP सर्वर का उपयोग कर नई क्षमताएँ जोड़ सकते हैं, जिसमें Imagen, Veo या Lyria के साथ मीडिया जेनरेशन शामिल है।
  • अपने क्वेरी को Google Search टूल के साथ ग्राउंड कर सकते हैं, जो Gemini में बिल्ट-इन है।

त्वरित प्रारंभ

  • पूर्व-आवश्यकताएँ: सुनिश्चित करें कि आपके पास Node.js संस्करण 18 या उससे अधिक स्थापित है।
  • CLI चलाएँ: अपने टर्मिनल में निम्नलिखित कमांड चलाएँ:

   npx https://github.com/google-gemini/gemini-cli
   ``
   या इसे इंस्टॉल करें:
   `bash
   npm install -g @google/gemini-cli
   gemini
   `
  • एक रंग थीम चुनें

  • प्रमाणीकृत करें: जब संकेत मिले, तो अपने व्यक्तिगत Google खाते के साथ साइन इन करें। इससे आपको Gemini के साथ प्रति मिनट 60 मॉडल अनुरोध और प्रति दिन 1,000 मॉडल अनुरोध करने की अनुमति मिलेगी।
अब आप Gemini CLI का उपयोग करने के लिए तैयार हैं!

यदि आपको किसी विशेष मॉडल का उपयोग करना है या अधिक अनुरोध क्षमता चाहिए, तो आप API कुंजी का उपयोग कर सकते हैं:
  • Google AI Studio से एक कुंजी जनरेट करें।

  • इसे अपने टर्मिनल में पर्यावरण चर (environment variable) के रूप में सेट करें। YOUR_API_KEY को अपनी जनरेट की गई कुंजी से बदलें।
    •    
`bash
   export GEMINI_API_KEY="YOUR_API_KEY"
   `
अन्य प्रमाणीकरण विधियों के लिए, जिसमें Google Workspace खाते शामिल हैं, authentication गाइड देखें।
उदाहरण
एक बार CLI चलने के बाद, आप अपने शेल से Gemini के साथ इंटरैक्ट करना शुरू कर सकते हैं।
आप एक नई डायरेक्टरी से प्रोजेक्ट शुरू कर सकते हैं:
sh cd new-project/ gemini
एक Gemini Discord बोट लिखें जो एक FAQ.md फ़ाइल से प्रश्नों के उत्तर देती है, जिसे मैं प्रदान करूंगा

या किसी मौजूदा प्रोजेक्ट के साथ काम करें:
sh git clone https://github.com/google-gemini/gemini-cli cd gemini-cli gemini
कल किए गए सभी परिवर्तनों का सारांश दें
