এটা শক্তিশালী ব্ৰাউজাৰ এক্সটেনচন, যি প্ৰম্পট শব্দ ব্যৱস্থাপনা আৰু দ্ৰুত ইনপুটত কেন্দ্ৰিত। এইটো ব্যৱহাৰকাৰীক প্ৰম্পট শব্দৰ ডাটাবেইজ সৃষ্টি, পৰিচালনা আৰু সংগঠিত কৰিবলৈ সহায় কৰে আৰু যিকোনো ৱেবপেইজৰ ইনপুট বাকচত আগতীয়াকৈ সংৰক্ষিত প্ৰম্প্ট সমল দ্ৰুতভাৱে সুমুৱাব পাৰে, যাৰ ফলত কামৰ দক্ষতা বৃদ্ধি হয়।

ব্ৰাউজাৰ প্লাগইন কেৱল ৱেব সংস্কৰণতেই ব্যৱহাৰ কৰিব পাৰি, সেয়ে একে সুবিধাসম্পন্ন এটা Raycast প্লাগইন মুক্ত উৎস হিচাপে উপলব্ধ, যি ব্ৰাউজাৰ প্লাগইনৰ ডাটা ফৰ্মেটৰ লগত সদৃশ আৰু একে JSON ডাটাৰ সহায়ত নির্বিঘ্নে স্থানান্তৰ কৰিব পাৰে।

✨ ফিচাৰ বৈশিষ্ট্যসমূহ

  • 📚 প্ৰম্পট শব্দ ব্যৱস্থাপনা: আপোনাৰ প্ৰম্পট শব্দবোৰ সহজে সৃষ্টি, সম্পাদনা আৰু ব্যৱস্থাপনা কৰক
  • 🚀 দ্ৰুত ইনপুট: যিকোনো ৱেবপৃষ্ঠাৰ ইনপুট বাকচত /p টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী দ্ৰুতকৈ আৰম্ভ কৰক
  • ⌨️ প্ৰম্পট শব্দ চয়নকাৰী খোলাৰ বাবে কিবোৰ্ড চাৰ্টকাট কনফিগাৰ আৰু নিৰ্বাচিত টেক্সট প্ৰম্পট শব্দ হিচাপে সংৰক্ষণৰ সুবিধা
  • 📑 সোঁ-চিকন মেনুতেও নিৰ্বাচিত টেক্সট প্ৰম্পট শব্দ হিচাপে সংৰক্ষণৰ সুবিধা
  • 🎯 কাষ্টম প্ৰম্পট শব্দৰ সুবিধা, তাৰ ভিতৰত শ্ৰেণী, শিৰোনাম, সমল, টেগ আৰু ভেৰিয়েবল
  • 🧑‍💻 ভেৰিয়েবল সমৰ্থন: প্ৰম্পট শব্দত {{ভেৰিয়েবল নাম}} ফৰ্মেটৰ ভেৰিয়েবল ব্যৱহাৰ কৰিব পাৰি, ব্যৱহাৰৰ সময়ত নির্দিষ্ট মান পূৰণ কৰিব পাৰি
  • 💾 ডাটা বেকআপ: প্ৰম্পট শব্দ ডাটাবেসৰ এক্সপোৰ্ট আৰু ইম্পোৰ্ট সমৰ্থন, ক্ৰছ ডিভাইচ স্থানান্তৰ আৰু বেকআপৰ সুবিধা
  • 🔗 ক্লাউড ছিঞ্চ: Notion ডাটাবেস বা Gitee/GitHub Gist ৰ সৈতে প্ৰম্পট শব্দ ডাটাবেস ছিঞ্চ কৰাৰ সুবিধা
  • 🔍 প্ৰম্পট শব্দৰ সন্ধান আৰু ফিল্টাৰ কৰাৰ সুবিধা
  • 🌙 চিস্টেমৰ লাইট-ডাৰ্ক থিমটো অটোমেটিকেলি এডাপ্ট কৰে

🚀 ব্যৱহাৰ পদ্ধতি

  • দ্ৰুত আৰম্ভ: যিকোনো ৱেবপৃষ্ঠাৰ টেক্সট ইনপুট বাকচত /p টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী আৰম্ভ কৰক
  • চাৰ্টকাটে চয়নকাৰী খোলক: Ctrl+Shift+P (Windows/Linux) বা Command+Shift+P (macOS) ব্যৱহাৰ কৰি প্ৰম্পট শব্দ চয়নকাৰী খোলক
  • প্ৰম্পট শব্দ চয়ন: পপ-আপ চয়নকাৰীত পৰা প্ৰয়োজনীয় প্ৰম্পট শব্দ ক্লিক কৰক, ইয়াৰ পৰা ইনপুট বাকচত স্বয়ংক্ৰিয়ভাৱে ইনচাৰ্ট হয়
  • দ্ৰুত সংৰক্ষণ: যিকোনো টেক্সট চয়ন কৰি Ctrl+Shift+S (Windows/Linux) বা Command+Shift+S (macOS) ব্যৱহাৰ কৰি প্ৰম্পট শব্দ হিচাপে দ্ৰুত সংৰক্ষণ কৰক
  • সোঁ-চিকন মেনু সংৰক্ষণ: যিকোনো টেক্সট চয়ন কৰি সোঁ-চিকন ক্লিক কৰক আৰু "এই প্ৰম্পট শব্দ সংৰক্ষণ" চয়ন কৰক, চয়ন কৰা সমল প্ৰম্পট শব্দ হিচাপে সংৰক্ষণ হ'ব
  • প্ৰম্পট শব্দ ডাটাবেস এক্সপোৰ্ট: ব্যৱস্থাপনা পৃষ্ঠাত "এক্সপোৰ্ট" বুটাম ক্লিক কৰি সকলো প্ৰম্পট শব্দ JSON ফাইল হিচাপে লোকেলত সংৰক্ষণ কৰক
  • প্ৰম্পট শব্দ ডাটাবেস ইম্পোৰ্ট: ব্যৱস্থাপনা পৃষ্ঠাত "ইম্পোৰ্ট" বুটাম ক্লিক কৰি লোকেল JSON ফাইলৰ পৰা প্ৰম্পট শব্দ ইম্পোৰ্ট কৰক (মিলোৱা বা বৰ্তমান প্ৰম্পট শব্দ আচ্ছাদিত কৰাৰ সুবিধা)

📸 ইন্টাৰফেচ পূৰ্বদৰ্শন

Quick Prompt সোজা আৰু বন্ধুত্বপূর্ণ ইন্টাৰফেচ প্ৰদান কৰে, যি আপোনাক প্ৰম্পট শব্দ সহজে ব্যৱস্থাপনা আৰু ব্যৱহাৰ কৰিবলৈ সহায় কৰে।

প্ৰম্পট শব্দ চয়নকাৰী

/p চাৰ্টকাট কমাণ্ড বা চাৰ্টকাট কিবোৰ্ডে যিকোনো ইনপুট বাকচত দ্ৰুতকৈ প্ৰম্পট শব্দ চয়নকাৰী খুলি, আপোনাক সহজে প্ৰম্পট শব্দ চয়ন আৰু ইনচাৰ্ট কৰিবলৈ সুবিধা দিয়ে।

প্ৰম্পট শব্দ ব্যৱস্থাপনা পৃষ্ঠা

ব্যৱস্থাপনা পৃষ্ঠাত, আপুনি নতুন প্ৰম্পট শব্দ সৃষ্টি, বৰ্তমান প্ৰম্পট শব্দ সম্পাদনা, টেগ যোগ আৰু শ্ৰেণীভুক্ত ব্যৱস্থাপনা কৰিব পাৰি। ইন্টাৰফেচ সৰল, স্পষ্ট আৰু ব্যৱহাৰ সুবিধাজনক।

সোঁ-মাউচ বুটাম মেনু সংৰক্ষণ

ওয়েবপৃষ্ঠা এখনত যিকোনো পাঠ্য নিৰ্বাচন কৰক, সোঁ-মাউচ বুটামত ক্লিক কৰি তৎক্ষণাত সেইটো প্ৰম্প্ট হিচাপে সংৰক্ষণ কৰিব পাৰি, যাৰ ফলত কামৰ গতি বৃদ্ধি পায়।

প্ৰম্প্ট ভেৰিয়েবল ইনপুট

প্ৰম্প্টত ভেৰিয়েবল সংহতিৰ সুবিধা আছে, নিৰ্বাচন কৰাৰ পাছত প্ৰাসংগিক ভেৰিয়েবল মান ইনপুট কৰাৰ বাবে পপ-আপ দেখা যায়।

⚙️ ব্যক্তিগতকৰণ সংহতি

  • এক্সটেনচনৰ আইকনটোত ক্লিক কৰক, তাৰ পাছত "প্ৰম্প্ট ব্যৱস্থাপনা" বুটামত ক্লিক কৰক
  • ব্যৱস্থাপনা পৃষ্ঠাত, আপুনি কৰিব পাৰে:
  • নতুন প্ৰম্প্ট সংযোজন কৰক
  • বিদ্যমান প্ৰম্প্ট সম্পাদনা কৰক
  • অপ্রয়োজনীয় প্ৰম্প্ট আঁতৰাওক
  • প্ৰম্প্টক শিতানৰ বাবে টেগ যোগ কৰক
  • প্ৰম্প্ট লাইব্ৰেৰী ৰপ্তানি কৰি বেকআপ কৰক
  • পূৰ্বৰ বেকআপ কৰা প্ৰম্প্ট লাইব্ৰেৰী আমদানি কৰক
