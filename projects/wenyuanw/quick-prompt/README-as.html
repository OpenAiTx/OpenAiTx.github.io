Quick Prompt

এটা শক্তিশালী ব্ৰাউজাৰ এক্সটেনশ্যন, যি প্ৰম্প্ট শব্দ ব্যৱস্থাপনা আৰু দ্ৰুত ইনপুটত কেন্দ্ৰীভূত। এইটো ব্যৱহাৰকাৰীক প্ৰম্প্ট শব্দৰ ডাটাবেছ সৃষ্টি, ব্যৱস্থাপনা আৰু সংগঠিত কৰাত সহায় কৰে, আৰু যিকোনো ৱেবপেজ ইনপুট বাকচত ছেট কৰা প্ৰম্প্ট বিষয়বস্তু তৎক্ষণাত সন্নিবিষ্ট কৰিবলৈ সহায় কৰে, যাৰ দ্বাৰা কামৰ সক্ষমতা বৃদ্ধি পায়।

✨ বৈশিষ্ট্য সমূহ

  • 📚 প্ৰম্পট শব্দ ব্যৱস্থাপনা: আপোনাৰ প্ৰম্পট শব্দ সংৰক্ষণাগাৰ সহজে সৃষ্টি, সম্পাদনা আৰু পৰিচালনা কৰক
  • 🚀 দ্ৰুত ইনপুট: যিকোনো ৱেবপৃষ্ঠাৰ ইনপুট বাক্সত /p টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী তৎক্ষণাৎ খুলিব
  • ⌨️ প্ৰম্পট শব্দ চয়নকাৰী খুলিবলৈ আৰু নিৰ্বাচিত পাঠ্য প্ৰম্পট শব্দ হিচাপে সংৰক্ষণ কৰিবলৈ কিবোৰ্ড শ্বটকাট কনফিগাৰ কৰক
  • 📑 সোঁমুৰি মেনুতে নিৰ্বাচিত পাঠ্যক প্ৰম্পট শব্দ হিচাপে সোঁমুৰি সংৰক্ষণ কৰাৰ সুবিধা
  • 🎯 কাষ্টম প্ৰম্পট শব্দৰ সমৰ্থন, ইয়াত শ্ৰেণী, শিৰোনাম, বিষয়বস্তু, সংলগ্ন, টেগ আৰু ভেৰিয়েবল অন্তৰ্ভুক্ত
  • 🧑‍💻 ভেৰিয়েবল সমৰ্থন: প্ৰম্পট শব্দত {{ভেৰিয়েবল নাম}} আকারৰ ভেৰিয়েবল ব্যৱহাৰ কৰিব পাৰি, ব্যৱহাৰ কৰাৰ সময়ত নির্দিষ্ট মান লিখিব পাৰি
  • 💾 ডাটা ব্যাকআপ: প্ৰম্পট শব্দ সংৰক্ষণাগাৰ ডাটাৰ এক্সপৰ্ট আৰু ইমপৰ্ট সমৰ্থন, ডিভাইচৰ মাজত স্থানান্তৰ আৰু ব্যাকআপ সহজ
  • 🔗 ক্লাউড ছিংক: প্ৰম্পট শব্দ সংৰক্ষণাগাৰ WebDAV ছিংক, Notion ডাটাবেছ, Gitee/GitHub Gist ছিংক সমৰ্থন
  • 🔍 প্ৰম্পট শব্দৰ সন্ধান আৰু ফিল্টাৰ সমৰ্থন
  • 🌙 সিষ্টেমৰ লাইট/ডাৰ্ক থীম স্বয়ংক্ৰিয়ভাৱে মানি চলা

🚀 ব্যৱহাৰ পদ্ধতি

  • দ্ৰুত সূচনা: যিকোনো ৱেবপৃষ্ঠাৰ পাঠ্য ইনপুট বাক্সত /p টাইপ কৰি প্ৰম্পট শব্দ চয়নকাৰী খুলক
  • শ্বটকাটে চয়নকাৰী খোলা: Ctrl+Shift+P (Windows/Linux) অথবা Command+Shift+P (macOS) ব্যৱহাৰ কৰি চয়নকাৰী খোলা
  • প্ৰম্পট শব্দ বাছনি: চয়নকাৰীত প্ৰয়োজনীয় প্ৰম্পট শব্দ ক্লিক কৰক, ই স্বয়ংক্ৰিয়ভাৱে ইনপুট বাক্সত সোমাই যাব
  • দ্ৰুত সংৰক্ষণ: যিকোনো পাঠ্য নিৰ্বাচন কৰাৰ পিছত Ctrl+Shift+S (Windows/Linux) অথবা Command+Shift+S (macOS) ব্যৱহাৰ কৰি দ্ৰুত সংৰক্ষণ কৰক
  • সোঁমুৰি মেনু সংৰক্ষণ: পাঠ্য নিৰ্বাচন কৰাৰ পিছত সোঁমুৰি ক্লিক কৰি "এই প্ৰম্পট শব্দ সংৰক্ষণ কৰক" বাছি সংৰক্ষণ কৰক
  • সংৰক্ষণাগাৰ এক্সপৰ্ট: পৰিচালনা পৃষ্ঠাত "এক্সপৰ্ট" বুটন ক্লিক কৰি সকলো প্ৰম্পট শব্দ JSON ফাইলত লোকেলত সংৰক্ষণ কৰক
  • সংৰক্ষণাগাৰ ইমপৰ্ট: পৰিচালনা পৃষ্ঠাত "ইমপৰ্ট" বুটন ক্লিক কৰি লোকেল JSON ফাইলৰ পৰা প্ৰম্পট শব্দ ইমপৰ্ট কৰক (মিলোৱা বা বিদ্যমান শব্দৰ ওপৰত লিখা সমৰ্থন)

📸 ইণ্টাৰফেচ পূৰ্বদৰ্শন

Quick Prompt এ সহজ, বোধগম্য ইউজাৰ ইণ্টাৰফেচ প্ৰদান কৰে, যাৰ সহায়ত আপুনি প্ৰম্পট শব্দ সহজে ব্যৱহাৰ আৰু পৰিচালনা কৰিব পাৰে।

প্ৰম্পট শব্দ চয়নকাৰী

/p দ্ৰুত নির্দেশনা বা শ্বটকাট ব্যৱহাৰ কৰি যিকোনো ইনপুট বাক্সত দ্ৰুত প্ৰম্পট শব্দ চয়নকাৰী খুলিব পাৰি, যাৰ সহায়ত সহজে প্ৰয়োজনীয় শব্দ বাছি ইনপুটত যোগ কৰিব পাৰি।

প্ৰম্পট শব্দ পৰিচালনা পৃষ্ঠা

পৰিচালনা পৃষ্ঠাত আপুনি নতুন প্ৰম্পট শব্দ সৃষ্টি, বিদ্যমান শব্দ সম্পাদনা, টেগ যোগ, আৰু শ্ৰেণীভুক্ত পৰিচালনা কৰিব পাৰে। ইণ্টাৰফেচ সহজ আৰু ব্যৱহাৰযোগ্য।

সোঁমুৰি মেনু সংৰক্ষণ

ওয়েবপৃষ্ঠাৰ যিকোনো পাঠ্য চয়ন কৰক, সোঁমোজা ক্লিক কৰি সেয়া তৎক্ষণাত্‌ প্ৰম্প্ট শব্দ হিচাপে সংৰক্ষণ কৰিব পাৰিব, যাৰফলে কামৰ গতি বৃদ্ধি হয়।

প্ৰম্প্ট শব্দ ভেৰিয়েবল ইনপুট

প্ৰম্প্ট শব্দসমূহত ভেৰিয়েবল সংহতি কৰা যায়, চয়ন কৰাৰ পাছত পপ-আপত সংশ্লিষ্ট ভেৰিয়েবল মান প্ৰৱিষ্ট কৰিব পাৰি।

⚙️ ব্যক্তিগতকৃত সংহতি

  • এক্সটেনচন আইকনটোত ক্লিক কৰক, তাৰ পাছত "প্ৰম্প্ট শব্দ ব্যৱস্থাপনা" বুটামত ক্লিক কৰক
  • ব্যৱস্থাপনা পৃষ্ঠাত, আপুনি কৰিব পাৰিব:
  • নতুন প্ৰম্প্ট শব্দ যোগ কৰা
  • বৰ্তমান থকা প্ৰম্প্ট শব্দ সম্পাদনা কৰা
  • অপ্ৰয়োজনীয় প্ৰম্প্ট শব্দ মচি দিয়া
  • প্ৰম্প্ট শব্দসমূহত টেগ যোগ কৰি শ্ৰেণীবদ্ধ কৰা
  • প্ৰম্প্ট শব্দ ডাটাবেজ এক্সপোৰ্ট কৰি বেকআপ লোৱা
  • আগতে সংৰক্ষিত প্ৰম্প্ট শব্দ ডাটাবেজ ইম্পোৰ্ট কৰা
