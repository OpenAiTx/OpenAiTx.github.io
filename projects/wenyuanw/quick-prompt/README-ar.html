إضافة قوية للمتصفح تركز على إدارة الكلمات الدلالية والإدخال السريع. تساعد المستخدمين على إنشاء وإدارة وتنظيم مكتبة الكلمات الدلالية، كما تتيح إدراج محتوى المطالبات المعد مسبقًا بسرعة في أي حقل إدخال على صفحات الويب، مما يعزز كفاءة العمل.

نظرًا لأن إضافات المتصفح يمكن استخدامها فقط في الويب، تم فتح مصدر إضافة Raycast ذات الوظائف المماثلة، وهي متوافقة مع تنسيق بيانات إضافة المتصفح وتدعم الانتقال السلس باستخدام نفس بيانات JSON.

✨ الميزات الوظيفية

  • 📚 إدارة العبارات المقترحة: أنشئ، حرر وأدر مكتبة العبارات المقترحة لديك بسهولة
  • 🚀 إدخال سريع: فعّل أداة اختيار العبارات بسرعة عبر كتابة /p في أي حقل إدخال نصي على صفحات الويب
  • ⌨️ دعم تهيئة اختصارات لوحة المفاتيح لفتح أداة اختيار العبارات أو حفظ النص المحدد كعبارة مقترحة
  • 📑 دعم حفظ النص المحدد كعبارة مقترحة مباشرة عبر قائمة الزر الأيمن
  • 🎯 دعم تخصيص العبارات المقترحة، بما يشمل التصنيف، العنوان، المحتوى، العلامات والمتغيرات
  • 🧑‍💻 دعم المتغيرات: يمكن استخدام متغيرات بصيغة {{اسم المتغير}} ضمن العبارات، مع إمكانية تحديد القيم عند الاستخدام
  • 💾 نسخ احتياطي للبيانات: دعم تصدير واستيراد مكتبة العبارات لتسهيل النقل أو النسخ الاحتياطي بين الأجهزة
  • 🔗 مزامنة سحابية: دعم مزامنة مكتبة العبارات مع قاعدة بيانات Notion أو Gitee/GitHub Gist
  • 🔍 دعم البحث وتصفية العبارات المقترحة
  • 🌙 التوافق التلقائي مع وضع الإضاءة أو الظلام حسب النظام

🚀 طريقة الاستخدام

  • تشغيل سريع: اكتب /p في أي حقل إدخال نصي على صفحة الويب لتفعيل أداة اختيار العبارات المقترحة
  • اختصار لوحة المفاتيح: استخدم Ctrl+Shift+P (ويندوز/لينكس) أو Command+Shift+P (ماك) لفتح أداة اختيار العبارات
  • اختيار العبارة: اختر العبارة المطلوبة من القائمة المنبثقة وسيتم إدراجها تلقائيًا في حقل الإدخال الحالي
  • حفظ عبارة بسرعة: بعد تحديد أي نص، استخدم Ctrl+Shift+S (ويندوز/لينكس) أو Command+Shift+S (ماك) لحفظه كعبارة مقترحة
  • حفظ من قائمة الزر الأيمن: بعد تحديد النص، انقر بزر الفأرة الأيمن واختر "حفظ هذه العبارة" لحفظ النص المحدد كعبارة مقترحة
  • تصدير مكتبة العبارات: في صفحة الإدارة، انقر على زر "تصدير" لتصدير جميع العبارات كملف JSON محلي
  • استيراد مكتبة العبارات: في صفحة الإدارة، انقر على زر "استيراد" لاستيراد العبارات من ملف JSON محلي (يدعم الدمج أو الاستبدال)

📸 معاينة الواجهة

يقدم Quick Prompt واجهة مستخدم بديهية وسهلة الاستخدام لإدارة العبارات المقترحة واستخدامها بكل يسر.

أداة اختيار العبارات المقترحة

باستخدام أمر /p أو اختصار لوحة المفاتيح، يمكنك استدعاء أداة اختيار العبارات المقترحة في أي حقل إدخال وإدراج العبارة المطلوبة بسهولة.

صفحة إدارة العبارات المقترحة

في صفحة الإدارة، يمكنك إنشاء عبارات جديدة، تعديل العبارات الحالية، إضافة علامات، وإدارة التصنيفات. الواجهة واضحة وسهلة الاستخدام.

حفظ من قائمة النقر بزر الماوس الأيمن

ما عليك سوى تحديد أي نص في صفحة الويب، ثم النقر بزر الماوس الأيمن لحفظه بسرعة ككلمة مفتاحية، مما يعزز كفاءة العمل.

إدخال متغيرات الكلمة المفتاحية

تدعم الكلمات المفتاحية إعداد متغيرات، وبعد الاختيار ستظهر نافذة منبثقة لإدخال القيم المطلوبة للمتغيرات.

⚙️ إعدادات مخصصة

  • انقر على أيقونة الإضافة، ثم انقر على زر "إدارة الكلمات المفتاحية"
  • في صفحة الإدارة يمكنك:
  • إضافة كلمات مفتاحية جديدة
  • تعديل الكلمات المفتاحية الحالية
  • حذف الكلمات المفتاحية غير المرغوب فيها
  • إضافة تصنيفات للكلمات المفتاحية لتنظيمها
  • تصدير قاعدة بيانات الكلمات المفتاحية للنسخ الاحتياطي
  • استيراد قاعدة بيانات النسخ الاحتياطي للكلمات المفتاحية السابقة

البناء من المصدر

  • استنساخ المستودع
``bash git clone https://github.com/wenyuanw/quick-prompt.git cd quick-prompt `
  • تثبيت الاعتمادات
`bash pnpm install `
  • التطوير والبناء

`bash # 开发模式 (Chrome) pnpm dev # 开发模式 (Firefox) pnpm dev:firefox # 构建扩展 (Chrome) pnpm build # 构建扩展 (Firefox) pnpm build:firefox `

تثبيت الامتداد الذي تم إنشاؤه

#### كروم / إيدج

  • افتح صفحة إدارة الامتدادات (chrome://extensions أو edge://extensions)
  • فعّل "وضع المطور"
  • انقر على "تحميل الامتداد غير المضغوط"
  • اختر مجلد المشروع .output/chrome-mv3/

#### فايرفوكس
  • افتح about:debugging
  • انقر على "هذا الفايرفوكس"
  • انقر على "تحميل إضافة مؤقتة"
  • اختر ملف manifest.json داخل مجلد .output/firefox-mv2/ الخاص بالمشروع
