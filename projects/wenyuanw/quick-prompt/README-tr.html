Quick Prompt

Güçlü bir tarayıcı uzantısı, prompt yönetimi ve hızlı veri girişi üzerine odaklanır. Kullanıcıların prompt havuzları oluşturmasına, yönetmesine ve düzenlemesine yardımcı olur; ayrıca herhangi bir web sayfası giriş kutusunda önceden ayarlanmış Prompt içeriğini hızlıca ekleyerek iş verimliliğini artırır.

✨ Özellikler

  • 📚 Prompt Yönetimi: Prompt kitaplığınızı kolayca oluşturun, düzenleyin ve yönetin
  • 🚀 Hızlı Giriş: Herhangi bir web sayfası giriş kutusunda /p ile prompt seçiciye hızlıca erişin
  • ⌨️ Kısayol tuşları ile prompt seçici açma & seçilen metni prompt olarak kaydetme desteği
  • 📑 Sağ tık menüsü ile seçili metni doğrudan prompt olarak kaydetme desteği
  • 🎯 Özelleştirilebilir promptlar; kategori, başlık, içerik, ek dosya, etiket ve değişken tanımlama desteği
  • 🧑‍💻 Değişken Desteği: Prompt içinde {{değişken_adı}} formatında değişken kullanılabilir, kullanırken değer girilebilir
  • 💾 Veri Yedekleme: Prompt kitaplığı dışa ve içe aktarılabilir; cihazlar arası taşıma ve yedekleme kolaylığı
  • 🔗 Bulut Senkronizasyonu: Prompt kitaplığı WebDAV senkronizasyonu, Notion veritabanı, Gitee/GitHub Gist senkronizasyonu desteği
  • 🔍 Prompt arama ve filtreleme özelliği desteği
  • 🌙 Sistemin aydınlık/karanlık temasına otomatik uyum

🚀 Kullanım Yöntemi

  • Hızlı Erişim: Herhangi bir web sayfası metin giriş kutusunda /p yazarak prompt seçiciyi açın
  • Kısayol ile Açma: Ctrl+Shift+P (Windows/Linux) veya Command+Shift+P (macOS) ile prompt seçiciyi açın
  • Prompt Seçimi: Açılan seçiciden istediğiniz prompta tıklayın, otomatik olarak mevcut giriş kutusuna eklenir
  • Promptı Hızlı Kaydetme: Herhangi bir metni seçtikten sonra Ctrl+Shift+S (Windows/Linux) veya Command+Shift+S (macOS) ile prompt olarak kaydedin
  • Sağ Tık Menüsüyle Kaydetme: Metni seçtikten sonra sağ tıklayıp "Bu promptı kaydet" seçeneğini tıklayın, seçilen içerik prompt olarak kaydedilir
  • Prompt Kitaplığını Dışa Aktarma: Yönetim sayfasında "Dışa Aktar" butonuna tıklayın, tüm promptları JSON dosyası olarak yerel bilgisayara kaydedin
  • Prompt Kitaplığını İçe Aktarma: Yönetim sayfasında "İçe Aktar" butonuna tıklayın, yerel JSON dosyasından promptları içe aktarın (mevcut promptlarla birleştirme veya üzerine yazma desteği)

📸 Arayüz Önizlemesi

Quick Prompt, kolay yönetim ve kullanım için sezgisel ve kullanıcı dostu bir arayüz sunar.

Prompt Seçici

/p hızlı komutu veya kısayol ile herhangi bir giriş kutusunda prompt seçiciyi açabilir ve ihtiyacınız olan promptları kolayca seçip ekleyebilirsiniz.

Prompt Yönetim Sayfası

Yönetim sayfasında yeni promptlar oluşturabilir, mevcutları düzenleyebilir, etiket ekleyebilir ve kategorilere göre yönetebilirsiniz. Arayüz sade ve kullanımı kolaydır.

Sağ Tık Menüsüyle Kaydetme

Web sayfasında herhangi bir metni seçin, sağ tıklayın ve onu hızlıca bir istem olarak kaydedin, iş verimliliğinizi artırın.

İstem Değişkeni Girişi

İstemler değişken yapılandırmasını destekler, seçtikten sonra ilgili değişken değerini açılır pencerede girin.

⚙️ Özelleştirilmiş Yapılandırma

  • Eklenti simgesine tıklayın ve ardından "İstemleri Yönet" düğmesine basın
  • Yönetim sayfasında şunları yapabilirsiniz:
  • Yeni istemler ekleyin
  • Mevcut istemleri düzenleyin
  • Gereksiz istemleri silin
  • İstemlere etiket ekleyerek sınıflandırın
  • İstem kitaplığını dışa aktararak yedekleyin
  • Daha önce yedeklenen istem kitaplığını içe aktarın

Kaynak koddan derleme

  • Depoyu klonlayın
``bash git clone https://github.com/wenyuanw/quick-prompt.git cd quick-prompt `
  • Bağımlılıkların kurulumu
`bash pnpm install `
  • Geliştirme ve Derleme

`bash # 开发模式 (Chrome) pnpm dev # 开发模式 (Firefox) pnpm dev:firefox # 构建扩展 (Chrome) pnpm build # 构建扩展 (Firefox) pnpm build:firefox `

Derlenmiş Eklentinin Kurulumu

#### Chrome / Edge

  • Eklenti yönetim sayfasını açın (chrome://extensions veya edge://extensions)
  • "Geliştirici Modu"nu etkinleştirin
  • "Paketlenmemiş eklenti yükle"ye tıklayın
  • Projenin .output/chrome-mv3/ dizinini seçin

#### Firefox
  • about:debugging adresini açın
  • "Bu Firefox" seçeneğine tıklayın
  • "Geçici eklenti yükle"ye tıklayın
  • Projenin .output/firefox-mv2/ dizinindeki manifest.json dosyasını seçin
