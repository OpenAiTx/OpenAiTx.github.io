Quick Prompt

Een krachtige browserextensie, gericht op promptbeheer en snelle invoer. Helpt gebruikers bij het maken, beheren en organiseren van een promptbibliotheek, en bij het snel invoegen van vooraf ingestelde prompts in elk invoerveld op een webpagina om de efficiëntie te verhogen.

✨ Functies en Kenmerken

  • 📚 Promptbeheer: Eenvoudig prompts aanmaken, bewerken en beheren in je promptbibliotheek
  • 🚀 Snelle invoer: Roep de promptkiezer snel op in elk webpagina-invoerveld met /p
  • ⌨️ Ondersteuning voor sneltoetsen om promptkiezer te openen & geselecteerde tekst als prompt op te slaan
  • 📑 Ondersteuning voor opslaan van geselecteerde tekst als prompt via het rechtermuisklikmenu
  • 🎯 Ondersteuning voor aangepaste prompts, waaronder categorie, titel, inhoud, bijlagen, labels en variabelen
  • 🧑‍💻 Variabelenondersteuning: Gebruik variabelen in prompts met het formaat {{variabelenaam}}, vul specifieke waarden in bij gebruik
  • 💾 Gegevensback-up: Ondersteunt exporteren en importeren van de promptbibliotheek voor eenvoudige migratie en back-up tussen apparaten
  • 🔗 Cloudsynchronisatie: Promptbibliotheek ondersteunt synchronisatie via WebDAV, Notion-database, Gitee/GitHub Gist
  • 🔍 Ondersteuning voor zoeken en filteren van prompts
  • 🌙 Past zich automatisch aan het licht- of donkerthema van het systeem aan

🚀 Gebruiksaanwijzing

  • Snelle trigger: Typ /p in een tekstinvoerveld op elke webpagina om de promptkiezer te openen
  • Promptkiezer openen met sneltoets: Gebruik Ctrl+Shift+P (Windows/Linux) of Command+Shift+P (macOS) om de promptkiezer te openen
  • Prompt kiezen: Klik op de gewenste prompt in de pop-upkiezer, deze wordt automatisch in het huidige invoerveld ingevoegd
  • Prompt snel opslaan: Selecteer willekeurige tekst en gebruik Ctrl+Shift+S (Windows/Linux) of Command+Shift+S (macOS) om snel als prompt op te slaan
  • Opslaan via rechtermuisklikmenu: Selecteer tekst, klik met rechts en kies "Prompt opslaan" om de geselecteerde inhoud als prompt op te slaan
  • Promptbibliotheek exporteren: Klik op de knop "Exporteren" op de beheerpagina om alle prompts als JSON-bestand lokaal op te slaan
  • Promptbibliotheek importeren: Klik op de knop "Importeren" op de beheerpagina om prompts uit een lokaal JSON-bestand te importeren (ondersteunt samenvoegen of overschrijven van bestaande prompts)

📸 Interfacevoorbeeld

Quick Prompt biedt een intuïtieve en gebruiksvriendelijke interface om eenvoudig prompts te beheren en te gebruiken.

Promptkiezer

Gebruik /p als snelcommando of een sneltoets om de promptkiezer snel op te roepen in elk invoerveld, zodat je eenvoudig de gewenste prompt kunt selecteren en invoegen.

Promptbeheerpagina

Op de beheerpagina kun je nieuwe prompts aanmaken, bestaande prompts bewerken, labels toevoegen en categorieën beheren. De interface is overzichtelijk en eenvoudig te bedienen.

Opslaan via rechtermuisklikmenu

Selecteer gewoon een willekeurige tekst op de webpagina en klik met de rechtermuisknop om deze snel op te slaan als prompt, waardoor de efficiëntie wordt verhoogd.

Prompt variabelen invoer

Prompts ondersteunen het configureren van variabelen; na selectie verschijnt een pop-up om de bijbehorende waarde in te voeren.

⚙️ Aangepaste configuratie

  • Klik op het extensie-icoon en klik vervolgens op de knop "Beheer prompts"
  • Op de beheerpagina kun je:
  • Nieuwe prompts toevoegen
  • Bestaande prompts bewerken
  • Onnodige prompts verwijderen
  • Labels toevoegen om prompts te categoriseren
  • Promptbibliotheek exporteren voor back-up
  • Eerder geback-upte promptbibliotheek importeren

📦 Installatiehandleiding

Installeren vanuit de app store

Downloaden van GitHub Releases

  • Ga naar de GitHub Releases pagina
  • Download het nieuwste kant-en-klare extensiepakket
  • Pak het gedownloade bestand uit
  • Installeer de gebouwde extensie volgens de onderstaande instructies

Bouwen vanuit de broncode

  • Clone de repository
``bash git clone https://github.com/wenyuanw/quick-prompt.git cd quick-prompt `
  • Afhankelijkheden installeren
`bash pnpm install `
  • Ontwikkeling en bouw

`bash # 开发模式 (Chrome) pnpm dev # 开发模式 (Firefox) pnpm dev:firefox # 构建扩展 (Chrome) pnpm build # 构建扩展 (Firefox) pnpm build:firefox `

Geïnstalleerde extensie bouwen

#### Chrome / Edge

  • Open de extensiebeheerpagina (chrome://extensions of edge://extensions)
  • Schakel "Ontwikkelaarsmodus" in
  • Klik op "Uitgepakte extensie laden"
  • Selecteer de .output/chrome-mv3/ map van het project

#### Firefox
  • Open about:debugging
  • Klik op "Deze Firefox"
  • Klik op "Tijdelijke add-on laden"
  • Selecteer het bestand manifest.json in de .output/firefox-mv2/ map van het project
