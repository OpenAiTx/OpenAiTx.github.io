پرومپت سریع

یک افزونه قدرتمند مرورگر که بر مدیریت پرومپت و ورود سریع تمرکز دارد. این ابزار به کاربران کمک می‌کند تا بانک پرومپت‌های خود را ایجاد، مدیریت و سازماندهی کنند و محتوای پرومپت از پیش تعیین‌شده را در هر کادر متنی وب‌سایت به سرعت وارد کرده و بهره‌وری را افزایش دهند.

✨ ویژگی‌های اصلی

  • 📚 مدیریت پریمپت‌ها: به راحتی کتابخانه پریمپت‌های خود را ایجاد، ویرایش و مدیریت کنید
  • 🚀 ورود سریع: با استفاده از /p در هر کادر ورودی وب، انتخابگر پریمپت را به سرعت فراخوانی کنید
  • ⌨️ پشتیبانی از پیکربندی کلیدهای میانبر برای باز کردن انتخابگر پریمپت و ذخیره متن انتخابی به عنوان پریمپت
  • 📑 پشتیبانی از ذخیره مستقیم متن انتخاب‌شده به عنوان پریمپت از طریق منوی راست‌کلیک
  • 🎯 پشتیبانی از پریمپت‌های سفارشی، شامل دسته‌بندی، عنوان، محتوا، پیوست، برچسب و متغیر
  • 🧑‍💻 پشتیبانی از متغیرها: در پریمپت‌ها می‌توانید از متغیر با فرمت {{نام متغیر}} استفاده کنید و هنگام استفاده مقدار دلخواه وارد کنید
  • 💾 پشتیبان‌گیری داده: پشتیبانی از خروجی و وارد کردن کتابخانه پریمپت برای انتقال و پشتیبان‌گیری بین دستگاه‌ها
  • 🔗 همگام‌سازی ابری: کتابخانه پریمپت از همگام‌سازی با WebDAV، پایگاه داده Notion، و همگام‌سازی Gitee/GitHub Gist پشتیبانی می‌کند
  • 🔍 پشتیبانی از جستجو و فیلتر پریمپت‌ها
  • 🌙 تطبیق خودکار با تم روشن یا تاریک سیستم

🚀 روش استفاده

  • فراخوانی سریع: با وارد کردن /p در هر کادر متنی وب، انتخابگر پریمپت را فعال کنید
  • باز کردن انتخابگر با میانبر: با Ctrl+Shift+P (ویندوز/لینوکس) یا Command+Shift+P (مک) انتخابگر پریمپت را باز کنید
  • انتخاب پریمپت: روی پریمپت مورد نظر در انتخابگر کلیک کنید تا به طور خودکار در کادر ورودی فعلی درج شود
  • ذخیره سریع پریمپت: پس از انتخاب هر متنی، با Ctrl+Shift+S (ویندوز/لینوکس) یا Command+Shift+S (مک) آن را سریعاً به عنوان پریمپت ذخیره کنید
  • ذخیره با منوی راست‌کلیک: پس از انتخاب متن، کلیک راست کرده و گزینه "ذخیره این پریمپت" را انتخاب کنید تا متن به عنوان پریمپت ذخیره شود
  • خروجی گرفتن از کتابخانه پریمپت: در صفحه مدیریت روی دکمه "خروجی" کلیک کنید تا همه پریمپت‌ها به صورت فایل JSON در دستگاه شما ذخیره شود
  • وارد کردن کتابخانه پریمپت: در صفحه مدیریت روی دکمه "وارد کردن" کلیک کنید و از فایل JSON محلی پریمپت‌ها را وارد نمایید (پشتیبانی از ادغام یا جایگزینی پریمپت‌های فعلی)

📸 پیش‌نمایش رابط کاربری

Quick Prompt یک رابط کاربری شهودی و دوستانه ارائه می‌دهد تا بتوانید به راحتی پریمپت‌های خود را مدیریت و استفاده کنید.

انتخابگر پریمپت

با دستور سریع /p یا کلید میانبر می‌توانید انتخابگر پریمپت را در هر کادر ورودی به سرعت باز کرده و پریمپت موردنظر را انتخاب و درج کنید.

صفحه مدیریت پریمپت

در صفحه مدیریت می‌توانید پریمپت جدید بسازید، پریمپت‌های قبلی را ویرایش کنید، برچسب اضافه کنید و دسته‌بندی انجام دهید. رابط کاربری ساده و عملیات آسان است.

ذخیره با منوی راست‌کلیک

کافیست هر متنی را در صفحه وب انتخاب کنید و با راست‌کلیک آن را به سرعت به عنوان یک عبارت راهنما ذخیره کنید تا بهره‌وری کاری افزایش یابد.

ورودی متغیرهای عبارت راهنما

عبارت راهنما قابلیت پیکربندی متغیر را دارد، پس از انتخاب پنجره‌ای برای وارد کردن مقدار متغیر باز می‌شود.

⚙️ پیکربندی سفارشی

  • روی آیکون افزونه کلیک کنید، سپس دکمه "مدیریت عبارت‌های راهنما" را بزنید.
  • در صفحه مدیریت، شما می‌توانید:
  • عبارت راهنمای جدید اضافه کنید
  • عبارات راهنمای موجود را ویرایش کنید
  • عبارات راهنمای غیرضروری را حذف کنید
  • برای دسته‌بندی عبارات راهنما، برچسب اضافه کنید
  • بانک عبارات راهنما را برای پشتیبان‌گیری صادر کنید
  • بانک عبارات راهنمای قبلی را وارد کنید

📦 راهنمای نصب

نصب از فروشگاه برنامه‌ها

دریافت از GitHub Releases

  • به صفحه GitHub Releases مراجعه کنید
  • جدیدترین نسخه بسته افزونه را دانلود کنید
  • فایل دانلود شده را از حالت فشرده خارج کنید
  • طبق دستورالعمل زیر، افزونه ساخته‌شده را نصب کنید

ساخت از روی کد منبع

  • مخزن را کلون کنید
``bash git clone https://github.com/wenyuanw/quick-prompt.git cd quick-prompt `
  • نصب وابستگی‌ها
`bash pnpm install `
  • توسعه و ساخت

`bash # 开发模式 (Chrome) pnpm dev # 开发模式 (Firefox) pnpm dev:firefox # 构建扩展 (Chrome) pnpm build # 构建扩展 (Firefox) pnpm build:firefox `

نصب افزونه‌ی ساخته‌شده

#### Chrome / Edge

  • صفحه مدیریت افزونه‌ها را باز کنید (chrome://extensions یا edge://extensions)
  • "حالت توسعه‌دهنده" را فعال کنید
  • روی "بارگذاری افزونه‌ی از حالت فشرده خارج شده" کلیک کنید
  • پوشه .output/chrome-mv3/ پروژه را انتخاب کنید

#### Firefox
  • about:debugging را باز کنید
  • روی "این Firefox" کلیک کنید
  • روی "بارگذاری موقت افزونه" کلیک کنید
  • فایل manifest.json موجود در پوشه .output/firefox-mv2/ پروژه را انتخاب کنید
