🎬 yt-dlp ভিডিঅ' একেলগে ডাউনলোড কৰাৰ উপকৰণ (Windows / Linux সমৰ্থিত)

yt-dlp আধাৰিত এক-ক্লিক ভিডিঅ' ডাউনলোড স্ক্ৰিপ্ট, NodeSeek ডাঙৰজনৰ অংশীদাৰিত্ব ৰ পৰা অনুপ্ৰাণিত।
একক ভিডিঅ' আৰু একেলগে ডাউনলোড সমৰ্থন কৰে, Windows আৰু মুখ্য Linux বিতৰণ (Debian / Ubuntu / Alpine / CentOS) ৰ লগত সঁজুলিকৃত।

✨ ফিচাৰৰ বৈশিষ্ট্য

  • 📥 এবাৰে চলাও: হস্তচালিত সংৰূপণৰ প্ৰয়োজন নাই, নিৰ্দেশনা অনুসৰি ইনপুট দিলে ডাউনলোড আৰম্ভ হয়
  • 🍪 কাষ্টম কুকি সমৰ্থন কৰে: লগইন কৰিব লাগিবলগীয়া ভিডিঅ'ৰ বাবে উপযোগী (⚠️ এই ফিচাৰটো এতিয়াও পৰীক্ষা কৰা হোৱা নাই, Bug থাকিব পাৰে)
  • 📂 কাষ্টম আউটপুট ডিৰেক্টৰি সমৰ্থন কৰে: নিৰ্দিষ্ট ফোল্ডাৰত সহজে সংৰক্ষণ কৰক
  • 📃 বাল্ক ডাউনলোড সমৰ্থন কৰে: urls.txt ফাইলৰ সংযোগসমূহ স্বয়ংক্ৰিয়ভাৱে পঢ়ি ডাউনলোড কৰে
  • ⚙️ স্বয়ংক্ৰিয় ডিপেণ্ডেন্সি ইনষ্টল: Linux পৰিবেশত yt-dlp আৰু ffmpeg স্বয়ংক্ৰিয়ভাৱে ইনষ্টল হয়

🪟 Windows ব্যৱহাৰৰ পদ্ধতি

  • তলত দিয়া ফাইলসমূহ ডাউনলোড কৰক:

  • yt-dlp.exe
  • ffmpeg Windows সংস্কৰণ

  • Extract কৰাৰ পিছত, তলত দিয়া ফাইলসমূহ একেটা ডিৰেক্টৰিত (যেনে yt-dlp ফোল্ডাৰ) ৰাখক:

  • yt-dlp.exe
  • ffmpeg.exe
  • ffprobe.exe
  • yt-dlp.bat (স্ক্ৰিপ্ট ফাইল)

  • yt-dlp.bat-ত ডাবল ক্লিক কৰক, নিৰ্দেশনা অনুসৰি ভিডিঅ' ডাউনলোড কৰক।
  • config/config.ini সম্পাদনা কৰি ডিফল্ট সংৰূপণ সলনি কৰক

🐧 Linux ব্যৱহাৰৰ পদ্ধতি(Technology lion ডাঙৰীয়াৰ নতুন স্ক্ৰিপ্টত এই একে ফিচাৰৰ স্ক্ৰিপ্ট অন্তর্ভুক্ত আছে)

সমৰ্থিত প্ৰণালী: Debian / Ubuntu / Alpine / CentOS

এটিমাত্র ক্লিকত স্থাপন বা আপডেট কৰি চলাও:

bash -c "$(curl -fsSL https://raw.githubusercontent.com/nodeloc666/yt-dlp-script/main/install.sh)"

অপসৰণৰ প্ৰয়োজন হলে:

bash -c "$(curl -fsSL https://raw.githubusercontent.com/nodeloc666/yt-dlp-script/main/uninstall.sh)"

📱 Android ব্যৱহাৰৰ পৰামৰ্শ

  • Seal App ব্যৱহাৰ কৰক, Android-ত yt-dlp ফিচাৰ উপভোগ কৰিবলৈ।
  • termux-ত Debian ইনষ্টল কৰিও ডাউনলোড কৰিব পৰা যায়, কিন্তু ইয়াৰ প্ৰয়োজন নাই, Seal-এ খুব ভাল কাম কৰে

⚠️ মনোযোগৰ বিষয়সমূহ

  • বাল্ক ডাউনলোড মড: স্ক্ৰিপ্টে স্বয়ংক্ৰিয়ভাৱে বৰ্তমান ডাইৰেক্টৰিৰ config\urls.txt পঢ়িব, প্ৰতিটো লাইনত এটা ভিডিঅ' লিংক। ফাইল নথকা হ'লে, স্বয়ংক্ৰিয়ভাৱে সৃষ্টি হ'ব।
  • ৰিছৰ্চ ব্যৱহাৰৰ সূচনা: কম মেমৰি থকা ডিভাইচত একেলগে বহুত ডাঙৰ ভিডিঅ' ডাউনলোড নকৰিব, বা অধিক থ্ৰেড একেলগে চলাব নালাগে।
  • CentOS বিশেষ সূচনা: পৰিৱেশ বিশেষভাৱে ভিন্ন, সম্পূৰ্ণৰূপে পৰীক্ষা হোৱা নাই, প্ৰডাকশ্যন পৰিৱেশত ব্যৱহাৰ নকৰিব। বাকী প্ৰধান অপাৰেটিং চিষ্টেমসমূহত পৰীক্ষা কৰি দেখা হৈছে।
  • Linux সংস্কৰণৰ সৰু বাগ: স্ক্ৰিপ্টে হোমপেইজলৈ চলি আহিলে, এবাৰ ডিপেণ্ডেন্সি চেক কৰে, কোড চলিব পাৰিলেই হ'ল বুলি ধৰি লৈছোঁ, সলনি কৰা নাই
  • Windows সংস্কৰণৰ সৰু বাগ: আন পৃষ্ঠাৰ পৰা হোমপেইজলৈ আহি পুনৰ বাছনি কৰোঁতে, ডিফল্ট অপচন নোপোৱা হ'ব পাৰে, তাৰপিছত লুপত আৱদ্ধ হয়, সক্ষমতা সীমিত, ভালকৈ সংশোধন কৰিব পৰা নাই
দুটা সমাধান:
  • এক্সিট কৰি পুনৰ চলাওক
  • ডিফল্ট নলব, নম্বৰ দিয়ে বাছনি কৰক
