Vous pouvez contribuer

Que pouvez-vous contribuer ?

  • Toute note et tout sujet d'examen lié aux __cours de première année__ du programme 2022 de RVCE peuvent être contribué.
  • La taille du fichier ne doit pas dépasser 100 Mo. Si vous souhaitez ajouter un manuel, ajoutez uniquement les chapitres pertinents aux unités du cours.

Comment contribuer ?

  • Si vous êtes familier avec l'utilisation de GitHub :
+ Si vous souhaitez ajouter des documents à un cours qui n'existe pas dans ce dépôt, créez un dossier et envoyez une demande de push. + Sinon, ajoutez simplement les fichiers dans le dossier du cours et envoyez une demande de push.
  • Si vous n'êtes pas familier avec l'utilisation de GitHub, veuillez consulter le manuel d'instructions
  • Si vous ne souhaitez pas utiliser GitHub, veuillez remplir ce formulaire Lien du formulaire

__Vous recevrez des crédits.__

Ce dépôt contient également quelques chapitres sélectionnés pour tous les cours de mathématiques de première année, du manuel Higher Engineering Mathematics du Dr BS Grewal. Ces chapitres sont : 1) Algèbre linéaire 2) Calcul différentiel 3) Différentiation partielle 4) Intégrales multiples 5) Calcul vectoriel 6) Équations différentielles linéaires 7) Applications des EDL 8) Équations aux dérivées partielles 9) Applications des EDP 10) Transformées de Laplace 11) Méthodes statistiques 12) Méthodes numériques 13) Différences finies et interpolation 14) Différentiation et intégration numériques 15) Résolution numérique des EDO
