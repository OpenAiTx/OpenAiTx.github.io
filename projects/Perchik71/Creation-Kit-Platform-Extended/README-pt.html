Uma coleção de modificações, aprimoramentos e recursos de engenharia reversa para o Creation Kit da Bethesda.

Descrição

CKPE é uma plataforma poderosa de código compartilhado que faz inúmeras correções e melhorias no editor Creation Kit da Bethesda, inclui suporte ao editor para jogos como Skyrim Special Edition, Fallout 4, Starfield, sendo um sucessor dos projetos SSE CKFixes, SSE CKFixes Update, FO4 CKFixes, além do Unicode patch for SSE CK 1.5.73 que deu início a tudo para o autor perchik71.

Instalação

Instalação Nova

Baixe qualquer jogo suportado e instale o Creation Kit. Baixe a última versão do CKPE para seu jogo. Extraia todo o conteúdo no diretório do jogo, de modo que ckpe_loader.exe fique na mesma pasta que CreationKit.exe.

Atualização

Baixe a última versão do CKPE para seu jogo. Extraia todo o conteúdo no diretório do jogo, de modo que ckpe_loader.exe fique na mesma pasta que CreationKit.exe. Concorde em substituir todos os arquivos.

Compilação

Usuários comuns só precisam deste método para acessar recursos mais recentes que ainda não foram lançados. Abra o projeto no Visual Studio 2022 ou mais recente. Compile o projeto no modo Release ou Release-NoAVX2. Em seguida, mova o resultado da compilação para o diretório do jogo, de modo que ckpe_loader.exe fique na mesma pasta que CreationKit.exe. Se for solicitado a substituir arquivos antigos, concorde em substituir todos.

Automatizar builds (Beta)

Baixe o build do último commit do CKPE para seu jogo. Extraia todo o conteúdo no diretório do jogo, de modo que ckpe_loader.exe fique na mesma pasta que CreationKit.exe. Erros possíveis.
