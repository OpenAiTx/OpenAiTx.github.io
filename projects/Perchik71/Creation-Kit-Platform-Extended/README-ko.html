Bethesda의 Creation Kit을 위한 수정, 향상, 리버스 엔지니어링된 리소스 모음입니다.

설명

CKPE는 공유 소스의 강력한 플랫폼으로, Bethesda의 Creation Kit 에디터에 다양한 수정 및 개선을 제공하며, Skyrim Special Edition, Fallout 4, Starfield와 같은 게임의 에디터 지원을 포함합니다. 이는 SSE CKFixes, SSE CKFixes Update, FO4 CKFixes 프로젝트 및 저자 perchik71이 시작한 Unicode patch for SSE CK 1.5.73의 후속작입니다.

설치

>[!중요] >Regular 또는 noavx2 릴리스? >Steam에서 CPU가 avx2를 지원하는지 확인하세요. Steam을 열고 왼쪽 상단의 도움말 메뉴에 마우스를 올려 시스템 정보를 선택한 후 AVX2를 검색하세요. >지원된다면 regular 릴리스를 사용하세요. >지원되지 않는다면 noavx2 릴리스를 사용하세요.

신규 설치

지원되는 게임을 다운로드하고 Creation Kit를 설치하세요. 게임에 맞는 CKPE의 최신 릴리스를 다운로드하세요. 모든 파일을 게임 디렉토리에 추출하여 ckpe_loader.exe가 CreationKit.exe와 같은 폴더에 있도록 하세요.

업데이트

게임에 맞는 CKPE의 최신 릴리스를 다운로드하세요. 모든 파일을 게임 디렉토리에 추출하여 ckpe_loader.exe가 CreationKit.exe와 같은 폴더에 있도록 하세요. 모든 파일을 교체하는 것에 동의하세요.

컴파일

일반 사용자는 아직 공개되지 않은 최신 기능을 이용할 때만 이 방법을 사용하면 됩니다. Visual Studio 2022 이상에서 프로젝트를 열고, Release 또는 Release-NoAVX2 모드로 빌드하세요. 빌드 결과물을 게임 디렉토리로 이동하여 ckpe_loader.exe가 CreationKit.exe와 같은 폴더에 있도록 하세요. 기존 파일을 교체하라는 요청이 있으면 모두 교체에 동의하세요.
