Bethesda의 Creation Kit을 위한 수정, 향상, 리버스 엔지니어드 리소스 모음입니다.

설명

CKPE는 공유 소스의 강력한 플랫폼으로, Bethesda의 Creation Kit 에디터에 많은 수정과 개선을 제공하며, Skyrim Special Edition, Fallout 4, Starfield 등 게임의 에디터 지원을 포함합니다. 이는 SSE CKFixes, SSE CKFixes Update, FO4 CKFixes 프로젝트의 후속작이며, 모든 것이 시작된 SSE CK 1.5.73의 유니코드 패치와 perchik71 저자의 작품입니다.

설치

신규 설치

지원되는 게임 중 하나를 다운로드하고 Creation Kit를 설치합니다. 해당 게임용 CKPE의 최신 릴리스를 다운로드합니다. 모든 파일을 게임 디렉터리에 압축 해제하고, ckpe_loader.exe가 CreationKit.exe와 동일한 폴더에 있도록 합니다.

업데이트

해당 게임용 CKPE의 최신 릴리스를 다운로드합니다. 모든 파일을 게임 디렉터리에 압축 해제하고, ckpe_loader.exe가 CreationKit.exe와 동일한 폴더에 있도록 합니다. 모든 파일 교체에 동의합니다.

컴파일

일반 사용자는 아직 출시되지 않은 최신 기능을 이용하려면 이 방법을 사용해야 합니다. Visual Studio 2022 이상에서 프로젝트를 열고 Release 또는 Release-NoAVX2 모드로 빌드합니다. 빌드 결과물을 게임 디렉터리로 옮기고, ckpe_loader.exe가 CreationKit.exe와 동일한 폴더에 있도록 합니다. 이전 파일 교체 요청이 있으면 모두 교체에 동의합니다.

자동 빌드(Beta)

해당 게임용 CKPE의 최신 커밋 빌드를 다운로드합니다. 모든 파일을 게임 디렉터리에 압축 해제하고, ckpe_loader.exe가 CreationKit.exe와 동일한 폴더에 있도록 합니다. 발생 가능한 오류.
