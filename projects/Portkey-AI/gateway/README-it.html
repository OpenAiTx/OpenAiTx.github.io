Avvio rapido (2 minuti)

1. Configura il tuo AI Gateway

# Esegui il gateway in locale (necessita Node.js e npm)
npx @portkey-ai/gateway
Il Gateway è in esecuzione su http://localhost:8787/v1
La Console Gateway è in esecuzione su http://localhost:8787/public/

Guide al deployment:   Portkey Cloud (Consigliato)   Docker   Node.js   Cloudflare   Replit   Altri...

2. Effettua la tua prima richiesta

# pip install -qU portkey-ai
from portkey_ai import Portkey
Client compatibile con OpenAI

client = Portkey(
    provider="openai", # oppure 'anthropic', 'bedrock', 'groq', ecc
    Authorization="sk-*" # la API key del provider
)
Effettua una richiesta tramite il tuo AI Gateway

client.chat.completions.create(
    messages=[{"role": "user", "content": "Che tempo fa?"}],
    model="gpt-4o-mini"
)

Librerie supportate:   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   Altro..

Nella Gateway Console (http://localhost:8787/public/) puoi vedere tutti i tuoi log locali in un unico posto.

3. Routing & Guardrail

Le Config nel gateway LLM ti permettono di creare regole di instradamento, aggiungere affidabilità e impostare guardrail. 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
Collega la config al client

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Rispondi casualmente con Apple o Bat"}]
)
Questo risponderà sempre con "Bat" perché il guardrail nega tutte le risposte contenenti "Apple". La config di retry farà fino a 5 tentativi prima di rinunciare.

Funzionalità principali

Instradamento affidabile

  • Fallbacks: Passa a un altro provider o modello in caso di richieste fallite utilizzando il gateway LLM. Puoi specificare gli errori che attivano il fallback. Migliora l’affidabilità della tua applicazione.
  • Ritenti automatici: Ritenta automaticamente le richieste fallite fino a 5 volte. Una strategia di backoff esponenziale distanzia i tentativi per evitare il sovraccarico della rete.
  • Bilanciamento del carico: Distribuisci le richieste LLM tra più chiavi API o provider AI con pesi per garantire alta disponibilità e prestazioni ottimali.
  • Timeout delle richieste: Gestisci LLM indisciplinati e latenze impostando timeout granulari sulle richieste, consentendo la terminazione automatica delle richieste che superano una durata specificata.
  • Gateway LLM multi-modale: Chiama modelli di visione, audio (text-to-speech e speech-to-text) e generazione immagini da più provider — tutto utilizzando la firma OpenAI familiare.
  • API in tempo reale: Chiama le API in tempo reale lanciate da OpenAI tramite il server integrato websockets.

Sicurezza & Accuratezza

  • Guardrails: Verifica che i tuoi input e output LLM rispettino i controlli specificati. Scegli tra oltre 40 guardrails preconfigurati per garantire la conformità agli standard di sicurezza e accuratezza. Puoi portare i tuoi guardrails oppure scegliere tra i nostri numerosi partner.
  • Gestione sicura delle chiavi: Usa le tue chiavi oppure genera chiavi virtuali al volo.
  • Controllo accessi basato sui ruoli: Controllo granulare degli accessi per i tuoi utenti, workspace e chiavi API.
  • Conformità & Privacy dei dati: Il gateway AI è conforme a SOC2, HIPAA, GDPR e CCPA.

Gestione dei costi

  • Smart caching: Memorizza le risposte degli LLM per ridurre i costi e migliorare la latenza. Supporta caching semplice e semantico*.
  • Analytics sull’utilizzo: Monitora e analizza l’utilizzo di AI e LLM, inclusi volume richieste, latenza, costi e tassi di errore.
  • Ottimizzazione dei provider*: Passa automaticamente al provider più conveniente in base ai pattern di utilizzo e ai modelli di prezzo.

Collaborazione & Flussi di lavoro

  • Supporto agenti: Integra senza soluzione di continuità con framework agent popolari per costruire applicazioni AI complesse. Il gateway si integra perfettamente con Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow e anche Agenti personalizzati.
  • Gestione template di prompt*: Crea, gestisci e versiona i tuoi template di prompt in modo collaborativo tramite un playground universale.


* Disponibile nelle versioni hosted ed enterprise
