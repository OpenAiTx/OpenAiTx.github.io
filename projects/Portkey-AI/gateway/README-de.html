Schnellstart (2 Minuten)

1. Richten Sie Ihr AI Gateway ein

# Gateway lokal ausführen (benötigt Node.js und npm)
npx @portkey-ai/gateway
Das Gateway läuft unter http://localhost:8787/v1
Die Gateway-Konsole ist unter http://localhost:8787/public/ erreichbar

Bereitstellungsanleitungen:   Portkey Cloud (Empfohlen)   Docker   Node.js   Cloudflare   Replit   Andere...

2. Senden Sie Ihre erste Anfrage

# pip install -qU portkey-ai
from portkey_ai import Portkey
OpenAI-kompatibler Client

client = Portkey(
    provider="openai", # oder 'anthropic', 'bedrock', 'groq', etc
    Authorization="sk-*" # Der API-Schlüssel des Providers
)
Anfrage über Ihr AI Gateway stellen

client.chat.completions.create(
    messages=[{"role": "user", "content": "Wie ist das Wetter?"}],
    model="gpt-4o-mini"
)

Unterstützte Bibliotheken:   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   Mehr..

In der Gateway-Konsole (http://localhost:8787/public/) können Sie alle lokalen Logs an einem Ort einsehen.

3. Routing & Guardrails

Mit Configs im LLM Gateway können Sie Routing-Regeln erstellen, Zuverlässigkeit erhöhen und Guardrails einrichten. 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
Konfiguration an den Client anhängen

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Antworte zufällig mit Apfel oder Fledermaus"}]
)
Dies würde immer mit "Fledermaus" antworten, da die Guardrail alle Antworten mit "Apple" verweigert. Die Retry-Konfiguration versucht es 5 Mal, bevor sie aufgibt.

Kernfunktionen

Zuverlässiges Routing

  • Fallbacks: Fallback auf einen anderen Anbieter oder ein anderes Modell bei fehlgeschlagenen Anfragen über das LLM-Gateway. Sie können die Fehler angeben, bei denen der Fallback ausgelöst werden soll. Verbessert die Zuverlässigkeit Ihrer Anwendung.
  • Automatische Wiederholungen: Fehlgeschlagene Anfragen werden automatisch bis zu 5-mal wiederholt. Eine exponentielle Backoff-Strategie verteilt die Wiederholungsversuche, um eine Überlastung des Netzwerks zu verhindern.
  • Lastenausgleich: Verteilen Sie LLM-Anfragen auf mehrere API-Schlüssel oder KI-Anbieter mit Gewichtung, um eine hohe Verfügbarkeit und optimale Leistung zu gewährleisten.
  • Anfrage-Timeouts: Verwalten Sie widerspenstige LLMs und Latenzen, indem Sie granulare Anfrage-Timeouts einrichten, wodurch Anfragen, die eine bestimmte Dauer überschreiten, automatisch beendet werden.
  • Multimodales LLM-Gateway: Rufen Sie Vision-, Audio- (Text-to-Speech & Speech-to-Text) und Bildgenerierungsmodelle von mehreren Anbietern auf — alles mit der vertrauten OpenAI-Signatur.
  • Echtzeit-APIs: Rufen Sie von OpenAI gestartete Echtzeit-APIs über den integrierten Websocket-Server auf.

Sicherheit & Genauigkeit

  • Guardrails: Überprüfen Sie Ihre LLM-Ein- und Ausgaben, um sicherzustellen, dass sie Ihren definierten Prüfungen entsprechen. Wählen Sie aus über 40 vorgefertigten Guardrails, um die Einhaltung von Sicherheits- und Genauigkeitsstandards zu gewährleisten. Sie können eigene Guardrails einbringen oder aus unseren zahlreichen Partnern wählen.
  • Sichere Schlüsselverwaltung: Verwenden Sie eigene Schlüssel oder generieren Sie virtuelle Schlüssel nach Bedarf.
  • Rollenbasierte Zugriffskontrolle: Granulare Zugriffskontrolle für Ihre Benutzer, Workspaces und API-Schlüssel.
  • Compliance & Datenschutz: Das KI-Gateway ist SOC2-, HIPAA-, DSGVO- und CCPA-konform.

Kostenmanagement

  • Intelligentes Caching: Zwischenspeichern von LLM-Antworten zur Kostensenkung und Verbesserung der Latenz. Unterstützt einfaches und semantisches* Caching.
  • Nutzungsanalysen: Überwachen und analysieren Sie Ihre KI- und LLM-Nutzung, einschließlich Anfragevolumen, Latenz, Kosten und Fehlerquoten.
  • Anbieteroptimierung*: Automatischer Wechsel zum kosteneffizientesten Anbieter basierend auf Nutzungsverhalten und Preismodellen.

Zusammenarbeit & Workflows

  • Agenten-Unterstützung: Nahtlose Integration mit beliebten Agenten-Frameworks zum Aufbau komplexer KI-Anwendungen. Das Gateway integriert sich nahtlos mit Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow und sogar eigenen Agenten.
  • Prompt-Template-Management*: Erstellen, verwalten und versionieren Sie Ihre Prompt-Templates gemeinsam über einen universellen Prompt-Playground.


* Verfügbar in gehosteten und Enterprise-Versionen
