AI Gateway

#### Routeer naar 250+ LLMs met 1 snelle & gebruiksvriendelijke API

Docs | Enterprise | Hosted Gateway | Changelog | API Referentie


De AI Gateway is ontworpen voor snelle, betrouwbare & veilige routering naar 1600+ taal-, visie-, audio- en beeldmodellen. Het is een lichtgewicht, open-source en enterprise-ready oplossing waarmee je binnen 2 minuten met elk taalmodel kunt integreren.

  • [x] Bliksemsnel (<1ms latency) met een kleine footprint (122kb)
  • [x] Beproefd in de praktijk, met meer dan 10 miljard tokens verwerkt per dag
  • [x] Enterprise-ready met verbeterde beveiliging, schaalbaarheid en aangepaste implementaties


#### Wat kun je doen met de AI Gateway?

  • Integreer met elke LLM binnen 2 minuten - Snelstart
  • Voorkom downtime door automatische herhalingen en fallbacks
  • Schaal AI-apps met load balancing en conditionele routering
  • Bescherm je AI-implementaties met guardrails
  • Ga verder dan tekst met multi-modale mogelijkheden
  • Ontdek ten slotte agentische workflow integraties

Snelstart (2 min)

1. Installeer je AI Gateway

# Draai de gateway lokaal (vereist Node.js en npm)
npx @portkey-ai/gateway
De Gateway draait op http://localhost:8787/v1
De Gateway Console draait op http://localhost:8787/public/

Deployments gidsen:   Portkey Cloud (Aanbevolen)   Docker   Node.js   Cloudflare   Replit   Overig...

2. Doe je eerste verzoek

# pip install -qU portkey-ai
from portkey_ai import Portkey
OpenAI compatibele client

client = Portkey(
    provider="openai", # of 'anthropic', 'bedrock', 'groq', etc
    Authorization="sk-*" # de provider API sleutel
)
Doe een verzoek via je AI Gateway

client.chat.completions.create(
    messages=[{"role": "user", "content": "What's the weather like?"}],
    model="gpt-4o-mini"
)

Ondersteunde bibliotheken:   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   Meer..

Op de Gateway Console (http://localhost:8787/public/) kun je al je lokale logs op één plek bekijken.

3. Routing & Guardrails

Configs in de LLM gateway stellen je in staat om routeringsregels te maken, betrouwbaarheid toe te voegen en guardrails in te stellen. 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
Voeg de config toe aan de client

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Reply randomly with Apple or Bat"}]
)
Dit zal altijd antwoorden met "Bat" omdat de guardrail alle antwoorden met "Apple" weigert. De retry-config zal 5 keer opnieuw proberen voordat hij opgeeft.

Kernfunctionaliteiten

Betrouwbare Routering

  • Fallbacks: Vallen op een andere provider of model bij mislukte verzoeken via de LLM-gateway. Je kunt de fouten specificeren waarop de fallback moet worden geactiveerd. Verhoogt de betrouwbaarheid van je applicatie.
  • Automatische Herhalingen: Mislukte verzoeken automatisch tot 5 keer opnieuw proberen. Een exponentiële backoff-strategie spreidt de herhalingspogingen om netwerkoverbelasting te voorkomen.
  • Load Balancing: Verspreid LLM-verzoeken over meerdere API-sleutels of AI-providers met gewichten om hoge beschikbaarheid en optimale prestaties te garanderen.
  • Request Timeouts: Beheer onhandelbare LLM's & latenties door gedetailleerde request timeouts in te stellen, waarmee verzoeken die een bepaalde duur overschrijden automatisch worden beëindigd.
  • Multi-modale LLM Gateway: Roep vision-, audio- (tekst-naar-spraak & spraak-naar-tekst) en beeldgeneratiemodellen aan van meerdere providers — allemaal met de vertrouwde OpenAI-signature.
  • Realtime API's: Roep realtime API's aan die door OpenAI zijn gelanceerd via de geïntegreerde websockets-server.

Veiligheid & Nauwkeurigheid

  • Guardrails: Controleer je LLM-inputs en -outputs om te voldoen aan je opgegeven controles. Kies uit de 40+ vooraf gebouwde guardrails om te voldoen aan beveiligings- en nauwkeurigheidsnormen. Je kunt je eigen guardrails toevoegen of kiezen uit onze vele partners.
  • Veilig Sleutelbeheer: Gebruik je eigen sleutels of genereer virtuele sleutels on-the-fly.
  • Rolgebaseerde toegangscontrole: Gedetailleerde toegangscontrole voor je gebruikers, workspaces en API-sleutels.
  • Compliance & Gegevensprivacy: De AI-gateway voldoet aan SOC2, HIPAA, GDPR en CCPA.

Kostenbeheer

  • Slimme caching: Cache antwoorden van LLM's om kosten te verlagen en latentie te verbeteren. Ondersteunt eenvoudige en semantische* caching.
  • Gebruik Analytics: Monitor en analyseer je AI- en LLM-gebruik, inclusief het aantal verzoeken, latentie, kosten en foutpercentages.
  • Provider optimalisatie*: Schakel automatisch over naar de meest kosteneffectieve provider op basis van gebruikspatronen en prijsmodellen.

Samenwerking & Workflows

  • Agents Support: Integreer naadloos met populaire agent-frameworks om complexe AI-applicaties te bouwen. De gateway integreert naadloos met Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow, en zelfs Custom Agents.
  • Prompt Template Management*: Maak, beheer en versieer je prompt-templates gezamenlijk via een universele prompt-playground.


* Beschikbaar in hosted en enterprise versies
