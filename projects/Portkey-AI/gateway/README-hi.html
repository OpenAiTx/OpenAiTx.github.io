1. अपना एआई गेटवे सेटअप करें

# गेटवे को लोकली चलाएं (Node.js और npm की आवश्यकता है)
npx @portkey-ai/gateway
गेटवे http://localhost:8787/v1 पर चल रहा है
गेटवे कंसोल http://localhost:8787/public/ पर चल रहा है

डिप्लॉयमेंट गाइड्स:   Portkey Cloud (अनुशंसित)   Docker   Node.js   Cloudflare   Replit   अन्य...

2. अपनी पहली रिक्वेस्ट करें

# pip install -qU portkey-ai
from portkey_ai import Portkey
OpenAI संगत क्लाइंट

client = Portkey(
    provider="openai", # या 'anthropic', 'bedrock', 'groq', आदि
    Authorization="sk-*" # प्रोवाइडर API कुंजी
)
अपने एआई गेटवे के माध्यम से अनुरोध करें

client.chat.completions.create(
    messages=[{"role": "user", "content": "What's the weather like?"}],
    model="gpt-4o-mini"
)

सपोर्टेड लाइब्रेरीज़:   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   More..

गेटवे कंसोल (http://localhost:8787/public/) पर आप अपने सभी लोकल लॉग्स एक ही जगह देख सकते हैं।

3. रूटिंग और गार्डरेल्स

LLM गेटवे में Configs आपको रूटिंग रूल्स बनाने, विश्वसनीयता जोड़ने और गार्डरेल्स सेटअप करने की सुविधा देता है। 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
कॉन्फ़िग को क्लाइंट से जोड़ें

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Reply randomly with Apple or Bat"}]
)
यह हमेशा "Bat" के साथ उत्तर देगा क्योंकि गार्डरेल सभी "Apple" वाले उत्तरों को अस्वीकार कर देता है। रिट्राई कॉन्फ़िग 5 बार कोशिश करेगा।

रिपोर्ट प्राप्त करें

मुख्य विशेषताएँ

विश्वसनीय रूटिंग

  • फ़ॉलबैक: LLM गेटवे का उपयोग करके असफल अनुरोधों पर किसी अन्य प्रदाता या मॉडल पर फ़ॉलबैक करें। आप यह निर्दिष्ट कर सकते हैं कि किन त्रुटियों पर फ़ॉलबैक ट्रिगर करना है। आपकी एप्लिकेशन की विश्वसनीयता में सुधार करता है।
  • स्वचालित पुनः प्रयास: असफल अनुरोधों को स्वचालित रूप से 5 बार तक पुनः प्रयास करें। एक एक्सपोनेंशियल बैकऑफ रणनीति पुनः प्रयासों के बीच अंतराल रखती है ताकि नेटवर्क ओवरलोड न हो।
  • लोड बैलेंसिंग: LLM अनुरोधों को कई API कुंजियों या AI प्रदाताओं के बीच भार के अनुसार वितरित करें, ताकि उच्च उपलब्धता और अनुकूलित प्रदर्शन सुनिश्चित हो।
  • अनुरोध टाइमआउट: ग्रैन्युलर अनुरोध टाइमआउट सेट करके अनियंत्रित LLMs और विलंबता का प्रबंधन करें, जिससे निर्दिष्ट अवधि से अधिक अनुरोधों का स्वचालित रूप से समापन हो सके।
  • मल्टी-मोडल LLM गेटवे: विज़न, ऑडियो (टेक्स्ट-टू-स्पीच और स्पीच-टू-टेक्स्ट), और इमेज जनरेशन मॉडल्स को कई प्रदाताओं से कॉल करें — वह भी परिचित OpenAI सिग्नेचर का उपयोग करते हुए।
  • रीयलटाइम API: OpenAI द्वारा लॉन्च किए गए रीयलटाइम API को इंटीग्रेटेड वेब सॉकेट्स सर्वर के माध्यम से कॉल करें।

सुरक्षा और सटीकता

  • गार्डरेल्स: अपने LLM इनपुट और आउटपुट को आपके निर्दिष्ट मानकों के अनुसार सत्यापित करें। सुरक्षा और सटीकता मानकों के पालन के लिए 40+ प्री-बिल्ट गार्डरेल्स में से चुनें। आप अपने स्वयं के गार्डरेल्स ला सकते हैं या हमारे कई साझेदारों में से चुन सकते हैं।
  • सुरक्षित कुंजी प्रबंधन: अपनी स्वयं की कुंजियों का उपयोग करें या चलते-फिरते वर्चुअल कुंजी जनरेट करें।
  • भूमिका-आधारित एक्सेस कंट्रोल: अपने उपयोगकर्ताओं, वर्कस्पेस और API कुंजियों के लिए ग्रैन्युलर एक्सेस कंट्रोल।
  • अनुपालन और डेटा गोपनीयता: AI गेटवे SOC2, HIPAA, GDPR, और CCPA के अनुरूप है।

लागत प्रबंधन

  • स्मार्ट कैशिंग: लागत कम करने और विलंबता सुधारने के लिए LLMs से प्राप्त प्रतिक्रियाओं को कैश करें। सरल और सेमांटिक* कैशिंग को सपोर्ट करता है।
  • उपयोग विश्लेषण: अपने AI और LLM उपयोग की निगरानी और विश्लेषण करें, जिसमें अनुरोध मात्रा, विलंबता, लागत और त्रुटि दरें शामिल हैं।
  • प्रदाता अनुकूलन*: उपयोग पैटर्न और मूल्य निर्धारण मॉडल के आधार पर सबसे किफायती प्रदाता पर स्वचालित रूप से स्विच करें।

सहयोग और वर्कफ़्लो

  • एजेंट्स सपोर्ट: जटिल AI एप्लिकेशन बनाने के लिए लोकप्रिय एजेंट फ्रेमवर्क्स के साथ निर्बाध रूप से एकीकृत करें। गेटवे Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow, और यहाँ तक कि Custom Agents के साथ सहजता से एकीकृत होता है।
  • प्रॉम्प्ट टेम्पलेट प्रबंधन*: एक सार्वभौमिक प्रॉम्प्ट प्लेग्राउंड के माध्यम से अपने प्रॉम्प्ट टेम्पलेट्स को सहयोगात्मक रूप से बनाएं, प्रबंधित करें और संस्करण नियंत्रित करें।


* होस्टेड और एंटरप्राइज़ वर्शन में उपलब्ध

गेटवे एंटरप्राइज़ वर्शन

अपने AI ऐप को अधिक विश्वसनीय और फॉरवर्ड कंपैटिबल बनाएं, साथ ही पूर्ण डेटा सुरक्षा और गोपनीयता सुनिश्चित करें।

✅  सुरक्षित कुंजी प्रबंधन - भूमिका-आधारित एक्सेस कंट्रोल और ट्रैकिंग के लिए
✅  सरल एवं सेमांटिक कैशिंग - बार-बार पूछे गए प्रश्नों को तेज़ी से सर्व करने और लागत बचाने के लिए
✅  एक्सेस कंट्रोल और इनबाउंड नियम - यह नियंत्रित करने के लिए कि कौन से IPs और Geos आपकी डिप्लॉयमेंट्स से कनेक्ट कर सकते हैं
✅  PII रीडैक्शन - आपके अनुरोधों से संवेदनशील डेटा को स्वचालित रूप से हटाने के लिए, ताकि अनजाने में डेटा एक्सपोज़र रोका जा सके
✅  SOC2, ISO, HIPAA, GDPR अनुपालन - सर्वोत्तम सुरक्षा प्रथाओं के लिए
✅  पेशेवर सहायता - साथ ही फ़ीचर प्राथमिकता
