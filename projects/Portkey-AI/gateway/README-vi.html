Cổng AI Gateway

#### Kết nối tới hơn 250+ LLM chỉ với 1 API nhanh & thân thiện

Docs | Enterprise | Hosted Gateway | Changelog | API Reference


Cổng AI Gateway được thiết kế cho việc định tuyến nhanh, đáng tin cậy & an toàn tới hơn 1600+ mô hình ngôn ngữ, hình ảnh, âm thanh và ảnh. Đây là một giải pháp mã nguồn mở, nhẹ và sẵn sàng cho doanh nghiệp, cho phép bạn tích hợp với bất kỳ mô hình ngôn ngữ nào chỉ trong dưới 2 phút.

  • [x] Cực nhanh (độ trễ <1ms) với dung lượng cực nhỏ (122kb)
  • [x] Đã được kiểm thử thực tế, với hơn 10 tỷ tokens xử lý mỗi ngày
  • [x] Sẵn sàng cho doanh nghiệp với bảo mật nâng cao, khả năng mở rộng và triển khai tùy chỉnh


#### Bạn có thể làm gì với AI Gateway?

  • Tích hợp với bất kỳ LLM nào chỉ dưới 2 phút - Khởi động nhanh
  • Ngăn ngừa downtime với tự động thử lại và fallbacks
  • Mở rộng ứng dụng AI với cân bằng tải và định tuyến điều kiện
  • Bảo vệ triển khai AI của bạn với guardrails
  • Không chỉ văn bản mà còn hỗ trợ đa phương thức
  • Khám phá tích hợp agentic workflow

Khởi động nhanh (2 phút)

1. Thiết lập AI Gateway của bạn

# Chạy gateway trên máy (yêu cầu Node.js và npm)
npx @portkey-ai/gateway
Gateway chạy tại http://localhost:8787/v1
Gateway Console chạy tại http://localhost:8787/public/

Hướng dẫn triển khai:   Portkey Cloud (Khuyến nghị)   Docker   Node.js   Cloudflare   Replit   Khác...

2. Gửi yêu cầu đầu tiên của bạn

# pip install -qU portkey-ai
from portkey_ai import Portkey
Client tương thích OpenAI

client = Portkey(
    provider="openai", # hoặc 'anthropic', 'bedrock', 'groq', v.v.
    Authorization="sk-*" # API key của nhà cung cấp
)
Gửi yêu cầu qua AI Gateway của bạn

client.chat.completions.create(
    messages=[{"role": "user", "content": "Thời tiết hôm nay thế nào?"}],
    model="gpt-4o-mini"
)

Thư viện hỗ trợ:   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   Thêm..

Trên Gateway Console (http://localhost:8787/public/) bạn có thể xem tất cả log local ở một nơi.

3. Routing & Guardrails

Configs trong gateway LLM cho phép bạn tạo quy tắc định tuyến, bổ sung độ tin cậy và thiết lập guardrails. 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
Gắn config vào client

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Trả lời ngẫu nhiên với Apple hoặc Bat"}]
)
Lệnh này sẽ luôn trả lời với "Bat" vì guardrail từ chối mọi phản hồi chứa "Apple". Cấu hình retry sẽ thử lại 5 lần trước khi dừng.

LLMs in Prod'25

Những góc nhìn từ việc phân tích hơn 2 nghìn tỷ tokens, trên 90+ khu vực và 650+ nhóm trong môi trường sản xuất. Bạn sẽ nhận được gì từ báo cáo này:

  • Xu hướng định hình việc ứng dụng AI và sự phát triển của các nhà cung cấp LLM.
  • Chuẩn hiệu năng để tối ưu tốc độ, chi phí và độ tin cậy.
  • Chiến lược để mở rộng hệ thống AI ở quy mô sản xuất.

Nhận báo cáo

Tính năng cốt lõi

Định tuyến đáng tin cậy

  • Phương án dự phòng (Fallbacks): Tự động chuyển sang nhà cung cấp hoặc mô hình khác khi yêu cầu bị lỗi thông qua LLM gateway. Bạn có thể chỉ định các lỗi sẽ kích hoạt phương án dự phòng. Nâng cao độ tin cậy cho ứng dụng của bạn.
  • Tự động thử lại (Automatic Retries): Tự động thử lại các yêu cầu bị lỗi lên tới 5 lần. Chiến lược exponential backoff giúp giãn cách các lần thử lại để tránh quá tải mạng.
  • Cân bằng tải (Load Balancing): Phân phối các yêu cầu LLM qua nhiều khóa API hoặc nhà cung cấp AI với trọng số để đảm bảo tính sẵn sàng cao và hiệu suất tối ưu.
  • Giới hạn thời gian yêu cầu (Request Timeouts): Quản lý LLM & độ trễ bằng cách thiết lập giới hạn thời gian cụ thể cho từng yêu cầu, cho phép tự động dừng các yêu cầu vượt quá khoảng thời gian quy định.
  • Cổng LLM đa phương tiện (Multi-modal LLM Gateway): Gọi các mô hình thị giác, âm thanh (chuyển văn bản thành giọng nói & ngược lại), và tạo ảnh từ nhiều nhà cung cấp — tất cả đều sử dụng chữ ký OpenAI quen thuộc.
  • Realtime APIs: Gọi các API realtime do OpenAI phát hành thông qua server websocket tích hợp.

Bảo mật & Độ chính xác

  • Guardrails: Xác minh đầu vào và đầu ra của LLM để tuân thủ các kiểm tra bạn đã chỉ định. Chọn từ hơn 40 guardrail dựng sẵn để đảm bảo tuân thủ các tiêu chuẩn bảo mật và chính xác. Bạn có thể tự mang guardrail hoặc chọn từ nhiều đối tác của chúng tôi.
  • Quản lý khóa an toàn: Dùng khóa của bạn hoặc tạo khóa ảo tức thời.
  • Kiểm soát truy cập dựa trên vai trò: Kiểm soát truy cập chi tiết cho người dùng, workspace và khóa API.
  • Tuân thủ & Bảo mật dữ liệu: AI gateway tuân thủ SOC2, HIPAA, GDPR và CCPA.

Quản lý chi phí

  • Bộ nhớ đệm thông minh (Smart caching): Lưu đệm phản hồi từ LLM để giảm chi phí và cải thiện độ trễ. Hỗ trợ cache đơn giản và semantic*.
  • Phân tích sử dụng (Usage analytics): Giám sát và phân tích việc sử dụng AI và LLM, bao gồm số lượng yêu cầu, độ trễ, chi phí và tỷ lệ lỗi.
  • Tối ưu hóa nhà cung cấp*: Tự động chuyển sang nhà cung cấp tiết kiệm nhất dựa trên mô hình sử dụng và giá cả.

Hợp tác & Quy trình làm việc

  • Hỗ trợ Agents: Tích hợp liền mạch với các framework agent phổ biến để xây dựng ứng dụng AI phức tạp. Gateway tích hợp dễ dàng với Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow, và cả Custom Agents.
  • Quản lý mẫu Prompt*: Tạo, quản lý và phiên bản hóa các mẫu prompt một cách cộng tác thông qua playground prompt chung.


* Có sẵn trên phiên bản hosted và enterprise
