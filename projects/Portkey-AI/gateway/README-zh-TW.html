快速上手（2 分鐘）

1. 設定你的 AI Gateway

# 在本地運行 Gateway（需先安裝 Node.js 和 npm）
npx @portkey-ai/gateway
Gateway 運行於 http://localhost:8787/v1
Gateway 控制台運行於 http://localhost:8787/public/

部署指南：   Portkey Cloud（推薦）   Docker   Node.js   Cloudflare   Replit   其他...

2. 發送你的第一個請求

# pip install -qU portkey-ai
from portkey_ai import Portkey
OpenAI 相容的 client

client = Portkey(
    provider="openai", # 或 'anthropic', 'bedrock', 'groq', 等等
    Authorization="sk-*" # 供應商 API 金鑰
)
透過 AI Gateway 發送請求

client.chat.completions.create(
    messages=[{"role": "user", "content": "What's the weather like?"}],
    model="gpt-4o-mini"
)

支援的函式庫：   JS   Python   REST   OpenAI SDKs   Langchain   LlamaIndex   Autogen   CrewAI   更多..

在 Gateway 控制台 (http://localhost:8787/public/) 中，你可以集中查看所有本地日誌。

3. 路由與防護欄

LLM gateway 的 Configs 讓你可以建立路由規則、增強可靠性並設置防護欄。 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
將 config 附加到 client

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Reply randomly with Apple or Bat"}]
)
由於防護欄會拒絕所有包含 "Apple" 的回應，因此這裡總是回覆 "Bat"。重試設定會在放棄前重試 5 次。

你可以在 AI Gateway 的 config 中做更多進階設定。查看更多範例 →

核心功能

可靠路由

  • 故障轉移（Fallbacks）：當請求失敗時，通過 LLM Gateway 自動切換到其他供應商或模型。你可以指定觸發故障轉移的錯誤類型，提升應用程式的可靠性。
  • 自動重試（Automatic Retries）：自動重試失敗的請求，最多可達 5 次。採用指數退避策略，以避免網路過載。
  • 負載平衡（Load Balancing）：將 LLM 請求分配到多個 API 金鑰或 AI 供應商，通過權重分配，確保高可用性與最佳效能。
  • 請求逾時（Request Timeouts）：通過設置細緻的請求逾時，管理 LLM 的異常與延遲，自動終止超時的請求。
  • 多模態 LLM Gateway：使用熟悉的 OpenAI 介面，從多家供應商調用視覺、音訊（文字轉語音與語音轉文字）、圖像生成模型。
  • 即時 API：通過整合 websockets 伺服器，調用 OpenAI 推出的即時 API。

安全性與準確性

  • 防護軌道（Guardrails）：驗證你的 LLM 輸入與輸出是否符合指定檢查。可從 40 多個預建防護軌道中選擇，以確保符合安全與準確標準。你也可以自訂防護軌道或選用我們的多家合作夥伴。
  • 安全金鑰管理：可使用自己的金鑰，或即時產生虛擬金鑰。
  • 基於角色的存取控制：為用戶、工作區及 API 金鑰提供細緻存取控制。
  • 合規性與資料隱私：AI Gateway 符合 SOC2、HIPAA、GDPR 及 CCPA 標準。

成本管理

  • 智慧快取：快取 LLM 回應以降低成本並提升延遲表現。支援簡單及語意*快取。
  • 使用分析：監控並分析你的 AI 與 LLM 使用情況，包括請求量、延遲、成本及錯誤率。
  • 供應商最佳化*：根據使用模式與價格自動切換至最具成本效益的供應商。

協作與工作流

  • 代理支援：可無縫整合主流代理框架，構建複雜 AI 應用。Gateway 可與 Autogen、CrewAI、LangChain、LlamaIndex、Phidata、Control Flow，甚至自訂代理順利整合。
  • 提示模板管理*：透過通用提示遊樂場協作建立、管理與版本化提示模板。


* 僅在雲端與企業版提供
