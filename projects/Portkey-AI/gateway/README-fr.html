Passerelle IA

#### Accédez à plus de 250 LLMs avec une API rapide et conviviale

Docs | Entreprise | Gateway Hébergée | Changelog | Référence API


La Passerelle IA est conçue pour un routage rapide, fiable et sécurisé vers plus de 1600 modèles de langage, vision, audio et image. C’est une solution légère, open-source et prête pour l’entreprise qui vous permet de vous connecter à n’importe quel modèle de langage en moins de 2 minutes.

  • [x] Ultra rapide (<1ms de latence) avec une empreinte minime (122kb)
  • [x] Éprouvée en production, avec plus de 10 milliards de tokens traités chaque jour
  • [x] Prête pour l’entreprise avec sécurité renforcée, passage à l’échelle et déploiements personnalisés


#### Que pouvez-vous faire avec la Passerelle IA ?

  • Connectez-vous à n’importe quel LLM en moins de 2 minutes - Démarrage rapide
  • Évitez les interruptions grâce aux reprises automatiques et solutions de repli
  • Faites évoluer vos applications IA avec répartition de charge et routage conditionnel
  • Protégez vos déploiements IA avec des garde-fous
  • Allez au-delà du texte avec des capacités multimodales
  • Enfin, explorez les intégrations de workflows agentiques

Démarrage rapide (2 min)

1. Installez votre Passerelle IA

# Lancez la passerelle localement (nécessite Node.js et npm)
npx @portkey-ai/gateway
La Passerelle fonctionne sur http://localhost:8787/v1
La Console de la Passerelle est accessible sur http://localhost:8787/public/

Guides de déploiement :   Portkey Cloud (Recommandé)   Docker   Node.js   Cloudflare   Replit   Autres...

2. Faites votre première requête

# pip install -qU portkey-ai
from portkey_ai import Portkey
Client compatible OpenAI

client = Portkey(
    provider="openai", # ou 'anthropic', 'bedrock', 'groq', etc
    Authorization="sk-*" # la clé API du provider
)
Effectuer une requête via votre Passerelle IA

client.chat.completions.create(
    messages=[{"role": "user", "content": "Quel temps fait-il ?"}],
    model="gpt-4o-mini"
)

Bibliothèques supportées :   JS   Python   REST   SDK OpenAI   Langchain   LlamaIndex   Autogen   CrewAI   Plus...

Sur la Console de la Passerelle (http://localhost:8787/public/), vous pouvez visualiser tous vos logs locaux au même endroit.

3. Routage & Garde-fous

Les Configs dans la passerelle LLM vous permettent de créer des règles de routage, d'ajouter de la fiabilité et de configurer des garde-fous. 
config = {
  "retry": {"attempts": 5},
  "output_guardrails": [{
    "default.contains": {"operator": "none", "words": ["Apple"]},
    "deny": True
  }]
}
Attacher la config au client

client = client.with_options(config=config)
client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Réponds aléatoirement avec Apple ou Bat"}]
)
Cela répondra toujours "Bat" car le garde-fou refuse toutes les réponses contenant "Apple". La config retry tentera 5 fois avant d'abandonner.

Vous pouvez faire beaucoup plus avec les configs dans votre passerelle IA. Voir les exemples →

Fonctionnalités principales

Routage fiable

  • Reprises automatiques (Fallbacks) : Basculez vers un autre fournisseur ou modèle lors de requêtes échouées en utilisant la passerelle LLM. Vous pouvez spécifier les erreurs qui déclenchent la reprise. Améliore la fiabilité de votre application.
  • Reprises automatiques (Automatic Retries) : Relancez automatiquement les requêtes échouées jusqu'à 5 fois. Une stratégie de backoff exponentiel espace les tentatives pour éviter la surcharge du réseau.
  • Répartition de charge (Load Balancing) : Distribuez les requêtes LLM entre plusieurs clés API ou fournisseurs d’IA avec des pondérations pour garantir une haute disponibilité et des performances optimales.
  • Délai d'expiration des requêtes (Request Timeouts) : Gérez les LLM imprévisibles & les latences en configurant des délais d’expiration granulaires, permettant la résiliation automatique des requêtes dépassant une durée spécifiée.
  • Passerelle LLM multimodale (Multi-modal LLM Gateway) : Appelez des modèles de vision, audio (texte-parole & parole-texte) et de génération d’images de plusieurs fournisseurs — tout en utilisant la signature OpenAI familière.
  • APIs temps réel (Realtime APIs) : Appelez les APIs temps réel lancées par OpenAI via le serveur websockets intégré.

Sécurité & Précision

  • Garde-fous (Guardrails) : Vérifiez les entrées et sorties de vos LLM pour respecter vos contrôles spécifiés. Choisissez parmi plus de 40 garde-fous préconstruits pour garantir la conformité aux normes de sécurité et de précision. Vous pouvez apporter vos propres garde-fous ou choisir parmi nos nombreux partenaires.
  • Gestion sécurisée des clés : Utilisez vos propres clés ou générez des clés virtuelles à la volée.
  • Contrôle d’accès basé sur les rôles : Contrôle d’accès granulaire pour vos utilisateurs, espaces de travail et clés API.
  • Conformité & Confidentialité des données : La passerelle IA est conforme SOC2, HIPAA, RGPD et CCPA.

Gestion des coûts

  • Cache intelligent : Mettez en cache les réponses des LLM pour réduire les coûts et améliorer la latence. Prend en charge le cache simple et sémantique*.
  • Analyse d’utilisation : Surveillez et analysez votre utilisation de l’IA et des LLM, y compris le volume de requêtes, la latence, les coûts et les taux d’erreurs.
  • Optimisation des fournisseurs* : Basculez automatiquement vers le fournisseur le plus rentable selon les modèles d’utilisation et de tarification.

Collaboration & Workflows

  • Support des Agents : Intégrez facilement les frameworks d’agents populaires pour créer des applications IA complexes. La passerelle s’intègre parfaitement avec Autogen, CrewAI, LangChain, LlamaIndex, Phidata, Control Flow, et même Agents personnalisés.
  • Gestion des modèles de prompts* : Créez, gérez et versionnez vos modèles de prompts de manière collaborative via un bac à sable universel.


* Disponible dans les versions hébergées et entreprise
