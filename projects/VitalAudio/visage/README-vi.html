Visage: Thư viện UI kết hợp với lập trình sáng tạo

Visage là một thư viện C++ đa nền tảng, tăng tốc GPU cho UI gốc và đồ họa 2D. Nó kết hợp cấu trúc của một framework UI với các tính năng của các thư viện đồ họa sáng tạo.

Ví dụ cơ bản đầy đủ

#include 
int main() {
  visage::ApplicationWindow app;
  app.onDraw() = &app {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };
  app.show(800, 600); // Mở cửa sổ với kích thước 800 x 600 pixel
  app.runEventLoop(); // Chạy vòng lặp sự kiện cửa sổ. Trả về khi cửa sổ đóng.
  return 0;
}

  • Chế độ hòa trộn (Blend Modes)
  • Bloom
  • Gradient

Tính năng UI

  • Chuẩn hóa sự kiện
   Hỗ trợ đa nền tảng cho chuẩn hóa nhập liệu bàn phím và chuột.

  • Chuẩn hóa cửa sổ
   Hỗ trợ đa nền tảng cho mở và xử lý cửa sổ.

  • Nhập liệu văn bản
   Nhập liệu văn bản Unicode với chỉnh sửa nhiều dòng.

  • ✨ Biểu tượng cảm xúc (Emojis) ✨
   Nếu bạn thích loại này 🤷

  • Kết xuất một phần
   Chỉ vẽ lại vùng bị thay đổi để tối ưu hiệu suất.

Tính năng đồ họa

  • Chuyển động mượt mà
   Khung hình mới hiển thị theo tần số quét của màn hình và hoạt ảnh mượt mà.

  • Nhóm hình dạng tự động
   Tự động nhóm các hình dạng để kết xuất GPU hiệu quả.

  • Chế độ hòa trộn
   Hỗ trợ hòa trộn các lớp với chế độ cộng, trừ hoặc bằng cách vẽ mặt nạ tùy chỉnh cho UI thông qua.

  • Shader
   Viết shader một lần và chuyển đổi cho Direct3D, Metal và OpenGL.

  • Hiệu ứng đi kèm
   Hiệu ứng thời gian thực như làm mờ lớn và bloom.

  • Độ chính xác điểm ảnh
   Truy cập kích thước điểm ảnh thiết bị đảm bảo kết xuất chính xác không bị mờ.

Nền tảng hỗ trợ

  • Windows: Direct3D11
  • MacOS: Metal
  • Linux: Vulkan
  • Web/Emscripten: WebGL
