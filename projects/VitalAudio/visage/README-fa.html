Visage: کتابخانه رابط کاربری ملاقات با برنامه‌نویسی خلاقانه

Visage یک کتابخانه C++ شتاب‌یافته توسط GPU و چندسکویی برای رابط کاربری بومی و گرافیک دوبعدی است. این کتابخانه ساختار یک چارچوب رابط کاربری را با ویژگی‌های کتابخانه‌های گرافیکی خلاقانه ادغام می‌کند.

مثال کامل پایه

#include 
int main() {
  visage::ApplicationWindow app;
  app.onDraw() = &app {
    canvas.setColor(0xffff00ff);
    canvas.fill(0, 0, app.width(), app.height());
  };
  app.show(800, 600); // باز می‌شود به عنوان پنجره‌ای با ابعاد ۸۰۰ در ۶۰۰ پیکسل
  app.runEventLoop(); // رویدادهای پنجره را اجرا می‌کند. وقتی پنجره بسته شود بازمی‌گردد.
  return 0;
}

ویژگی‌های رابط کاربری

  • نرمال‌سازی رویدادها
   پشتیبانی چندسکویی برای نرمال‌سازی ورودی صفحه‌کلید و ماوس.

  • نرمال‌سازی پنجره
   پشتیبانی چندسکویی برای باز کردن و مدیریت پنجره‌ها.

  • ورود متن
   ورود متن یونیکد با ویرایش متن چندخطی

  • ✨ ایموجی‌ها ✨
   اگر طرفدار این چیزها هستید 🤷

  • رندر جزئی
   فقط مناطق تغییر یافته را بازنقاشی می‌کند برای عملکرد بهینه.

ویژگی‌های گرافیکی

  • حرکت روان
   فریم‌های جدید با نرخ تازه‌سازی مانیتور نمایش داده می‌شوند و انیمیشن‌ها روان هستند

  • گروه‌بندی خودکار اشکال
   اشکال را به طور خودکار برای رندر کارآمد روی GPU گروه‌بندی می‌کند.

  • حالت‌های ترکیب
   پشتیبانی از ترکیب لایه‌ها با حالت‌های جمعی، تفریقی یا با رسم ماسک سفارشی برای عبور رابط کاربری

  • شیدرها
   شیدرها را یکبار بنویسید و آن‌ها را برای Direct3D، Metal و OpenGL تبدیل کنید

  • افکت‌های گنجانده شده
   افکت‌های زمان واقعی مانند بلور بزرگ و بلوم

  • دقت پیکسلی
   دسترسی به اندازه پیکسل دستگاه تضمین می‌کند که رندر دقیق بدون تار شدن انجام شود.

پلتفرم‌های پشتیبانی شده

  • ویندوز: Direct3D11
  • مک‌اواس: Metal
  • لینوکس: Vulkan
  • وب/ام‌اسکریپتن: WebGL
