Introdução

Esta lista foi originalmente um clone de StackOverflow - Lista de Livros de Programação Gratuitamente Disponíveis com contribuições de Karan Bhangui e George Stocker.

A lista foi movida para o GitHub por Victor Felder para atualização e manutenção colaborativa. Ela cresceu e se tornou um dos repositórios mais populares do GitHub.

       

A Free Ebook Foundation agora administra o repositório, uma organização sem fins lucrativos dedicada a promover a criação, distribuição, arquivamento e sustentabilidade de ebooks gratuitos. Doações para a Free Ebook Foundation são dedutíveis de impostos nos EUA.
