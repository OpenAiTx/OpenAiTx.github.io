Einführung

Diese Liste war ursprünglich ein Klon von StackOverflow - List of Freely Available Programming Books mit Beiträgen von Karan Bhangui und George Stocker.

Die Liste wurde von Victor Felder für eine kollaborative Pflege und Aktualisierung auf GitHub übertragen. Sie ist zu einem der beliebtesten Repositories auf GitHub geworden.

       

Die Free Ebook Foundation verwaltet nun das Repository. Sie ist eine gemeinnützige Organisation, die sich der Förderung der Erstellung, Verbreitung, Archivierung und Nachhaltigkeit von kostenlosen E-Books widmet. Spenden an die Free Ebook Foundation sind in den USA steuerlich absetzbar.
