はじめに

このリストはもともと StackOverflow - List of Freely Available Programming Books のクローンであり、Karan Bhangui と George Stocker の貢献によって作成されました。

このリストは、共同での更新・保守のために Victor Felder により GitHub へ移動されました。現在では GitHubで最も人気のあるリポジトリ の一つとなっています。

       

現在、このリポジトリは Free Ebook Foundation によって管理されています。これは、無料電子書籍の作成・配布・保存・持続可能性の促進を目的とした非営利団体です。Free Ebook Foundation への寄付 は米国で税控除の対象となります。
