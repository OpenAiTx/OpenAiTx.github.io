介紹

本清單最初為 StackOverflow - List of Freely Available Programming Books 的分支，由 Karan Bhangui 和 George Stocker 共同貢獻。 此清單由 Victor Felder 移至 GitHub 以便協作更新與維護，現已成為 GitHub 最受歡迎的倉庫之一。
       
Free Ebook Foundation 現負責管理本倉庫，該機構是一個致力於促進免費電子書創作、分發、歸檔及可持續發展的非營利組織。捐款給 Free Ebook Foundation 在美國可享有稅賦減免。
