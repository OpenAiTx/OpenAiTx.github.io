🚀 為什麼我會建立這個模組

我發現很多人不是在販售洩漏的 keybox，就是付錢購買那些聲稱能通過強大 Play Integrity 的模組，但其實只提供了洩漏的 keybox。我建立這個模組，就是要讓你獲得真正、可用的 keybox，而且完全免費，沒有隱藏費用、沒有詐騙，只有正規存取，還有多項實用功能。🚫🔑

ℹ️ 模組資訊

本模組提供下列功能：

  • ✅ 更新 keybox.xml
  • 🗽 根據你的 TEE 狀態更新 target.txt
  • 🥷 切換 Shamiko 模式（透過模組開關）
  • 👻 切換 NoHello 模式（透過模組開關）
  • 🛠️ 將所有自定義 ROM 偵測套件新增至 SusFS 路徑
  • ⛔ 預設停用 EU 注入器
  • ⛔ 停用 Pixel ROM 偽裝
  • 🔐 偽裝加密狀態
  • 🔑 偽裝 ROM release key
  • 😋 偽裝 SE Linux 狀態
  • 🕵️ 偵測異常行為以協助除錯
  • 🎨 更多功能請參見 WebUI

🗽 前置需求

請確認在使用本模組前，已經安裝以下模組：

  • Play Integrity Fork
  • Tricky Store

🛡️ 應用簽名與安全性說明

在較早的版本中，應用程式是用測試金鑰簽署的，因此部分安全性偵測器會將其標記為潛在有害應用。

從 模組 v3+ 開始，應用已改為用私有發行金鑰簽署。 🔒 雖然功能沒有任何變動，但改用正規金鑰後已經解決此問題，目前沒有再出現誤判。

🐾 Meow Assistant 的目的

Meow Assistant 旨在提升易用性與透明度。

它會在以下情況提供彈出訊息：

  • ✅ 你在 WebView 內點擊任何選項時
  • ⚙️ 你透過 動作按鈕 執行任何腳本時

這能協助你隨時掌握觸發的動作，並提升整體使用體驗。

🔍 有問題？點擊這裡

  • 無法給應用授予 root 權限？停用 IntegrityBox 模組，它會將 shamiko/nohello 切換為 黑名單模式 並顯示 root 權限
  • Shamiko 不支援 magisk delta（kitsune mask）
  • Shamiko 僅支援 Zygisk Next
  • 如果 Play Integrity 無法通過，請正確隱藏 root。如果你用的是自定義 ROM，務必關閉內建的 gms spoofing。如何關閉？這取決於你用的 ROM，請加入 ROM 的討論群組並詢問 如何關閉。
  • 如果官方 magisk 發生完整性問題，請切換到 Magisk Alpha。
