Este portafolio ha sido reescrito en TypeScript, renovado con la capacidad de respuesta y accesibilidad en mente, y para probar algunas bibliotecas nuevas.

Cambios desde la V2.0:

  • 🏙️ TypeScript
  • 🔍 Barra de búsqueda, potenciada por Fuse.js
  • ⌨️ Navegación por teclado
  • 🗞️ Se añadió una sección de certificados bajo "acerca de"
  • 😎 Actualizado con las últimas bibliotecas UI de moda (shadcn, tailwindcss, vite)
  • 🐛 Corregidos algunos errores de vibración, como en la sección de Experiencia

Consulta versiones anteriores de mi portafolio:

  • V2.0: github, despliegue (Chakra-UI, PWA frontend)
  • V1.0: github (react-bootstrap, cliente/servidor)

💼 Prerrequisitos

  • Node.js
  • pnpm

⚙️ Instalación

  • Clona el código más reciente en la rama master
  • Instala las dependencias
  • pnpm i
  • ¡Ejecuta el servidor web!
  • pnpm dev

🎈 Uso

  • Navega a http://localhost:5173 en tu navegador

💫 Despliegue

Importa tu proyecto a Vercel. Ahora tendrá despliegues automáticos al hacer push.
