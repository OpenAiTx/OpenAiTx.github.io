💼 先决条件

  • Node.js
  • pnpm

⚙️ 安装

  • 克隆 master 分支上的最新代码
  • 安装依赖
  • pnpm i
  • 运行 web 服务器！
  • pnpm dev

🎈 使用

  • 在浏览器中访问 http://localhost:5173

💫 部署

将您的项目导入 Vercel。现在，每次推送都会自动部署。

⛏️ 构建工具

  • React
  • shadcn
  • TailwindCSS
  • Vite
