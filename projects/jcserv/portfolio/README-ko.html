🚀 포트폴리오 V3.0

[!TIP]
이 포트폴리오를 사용하고 싶으신가요? 위의 포크 버튼을 클릭하세요!
>
거의 처음부터 모든 지루한 설정을 갖추고 시작하고 싶으신가요? 이 스타터 템플릿을 확인하세요.

이 포트폴리오는 TypeScript로 다시 작성되었으며, 반응성 및 접근성을 염두에 두고 개편되었고, 새로운 라이브러리를 시도해 보았습니다.

V2.0에서 변경된 사항:

  • 🏙️ TypeScript
  • 🔍 Fuse.js를 이용한 검색창
  • ⌨️ 키보드 내비게이션
  • 🗞️ about 섹션 아래에 자격증 섹션 추가
  • 😎 최신 트렌디 UI 라이브러리로 업데이트 (shadcn, tailwindcss, vite)
  • 🐛 Experience 섹션 등 일부 흔들림 버그 수정

이전 버전의 포트폴리오 보기:

  • V2.0: github, 배포 (Chakra-UI, 프론트엔드 PWA)
  • V1.0: github (react-bootstrap, 클라이언트/서버)

💼 필수 조건

  • Node.js
  • pnpm

⚙️ 설치 방법

  • 마스터 브랜치에서 최신 코드를 클론하세요
  • 의존성 설치
  • pnpm i
  • 웹 서버 실행!
  • pnpm dev

🎈 사용법

  • 브라우저에서 http://localhost:5173 으로 이동하세요

💫 배포

프로젝트를 Vercel에 가져오세요. 이제 푸시 시 자동 배포가 이루어집니다.

⛏️ 사용 기술

  • React
  • shadcn
  • TailwindCSS
  • Vite
