🚀 Portfolio V3.0

[!TIP]
Vous souhaitez utiliser ce portfolio ? Cliquez sur le bouton fork ci-dessus !

Vous voulez construire à partir de (presque) zéro, avec toutes les choses ennuyeuses configurées ? Découvrez ce modèle de démarrage.

Ce portfolio a été réécrit en TypeScript, repensé avec la réactivité et l’accessibilité en tête, et pour essayer de nouvelles bibliothèques.

Changements par rapport à la V2.0 :

  • 🏙️ TypeScript
  • 🔍 Barre de recherche, propulsée par Fuse.js
  • ⌨️ Navigation au clavier
  • 🗞️ Ajout d’une section certificats sous « à propos »
  • 😎 Mise à jour avec les dernières bibliothèques UI tendance (shadcn, tailwindcss, vite)
  • 🐛 Correction de certains bugs de saccades, comme dans la section Expérience

Voir les versions précédentes de mon portfolio :

  • V2.0 : github, déploiement (Chakra-UI, frontend PWA)
  • V1.0 : github (react-bootstrap, client/serveur)

💼 Prérequis

  • Node.js
  • pnpm

⚙️ Installation

  • Clonez le code le plus récent sur la branche master
  • Installez les dépendances
  • pnpm i
  • Lancez le serveur web !
  • pnpm dev

🎈 Utilisation

  • Naviguez vers http://localhost:5173 dans votre navigateur

💫 Déploiement

Importez votre projet dans Vercel. Il bénéficiera désormais de déploiements automatisés lors des pushs.

⛏️ Réalisé Avec

  • React
  • shadcn
  • TailwindCSS
  • Vite
