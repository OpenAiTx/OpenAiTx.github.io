아이폰을 AirPlay 수신기로 사용하세요
제작자: neon443

testflight



AirAP는 iOS용으로 Swift로 작성된 완전한 네이티브 AirPlay 서버입니다. 본질적으로, AirAP를 사용하면 아이폰을 iTunes나 Mac에서 AirPlay 수신기로 사용할 수 있어, 아이폰으로 기기의 소리를 재생할 수 있습니다.

AirAP란?

Mac, Apple TV 또는 다른 iOS 기기에서 아이폰으로 오디오를 스트리밍하고 싶었던 적이 있나요? AirAP는 iOS에서 네이티브로 실행되는 완전한 AirPlay 서버를 구현하여 이를 가능하게 합니다. 설치가 완료되면, 아이폰은 시스템 환경설정의 오디오 패널, 음악 앱, 또는 기타 AirPlay 호환 앱에서 사용 가능한 AirPlay 대상 장치로 표시됩니다.

이 개념이 처음엔 다소 뒤바뀐 것처럼 느껴질 수도 있습니다. 보통은 아이폰에서 다른 장치로 스트리밍하는 데 익숙하기 때문이죠. 하지만 반대로 하고 싶을 때도 의외로 많습니다. 예를 들어, 밤 늦게 Mac을 사용하면서 이어폰을 낀 아이폰으로 오디오를 전송해 다른 사람을 방해하지 않으려 할 때(안녕하세요 👋), 오디오 앱을 테스트하는 개발자로서 다양한 출력 장치 간 전환이 필요할 때, 혹은 오래된 유선 스피커를 재활용하고 싶을 때 등입니다.

AirAP 설치하기

체험해보고 싶다면, 이 TestFlight 링크를 열어 AirAP를 설치하고 안내에 따라 진행하세요. 설치가 완료되면 AirAP를 실행하고, 아이폰이 스트리밍하려는 기기와 동일한 Wi-Fi 네트워크에 연결되어 있는지 확인하세요. 아이폰이 자동으로 AirPlay 장치 목록에 나타나 오디오 수신 준비가 완료됩니다. 만약 나타나지 않는다면 앱을 다시 시작해보세요.

컴파일하기

homebrew가 설치되어 있는지 확인하세요

brew install carthage
git clone https://github.com/neon443/AirAP
cd AirAP
carthage checkout
open AirAP.xcodeproj
After adding your Team ID in Project > AirAP > Signing and Capabilities, hit Command + R to build and run!
