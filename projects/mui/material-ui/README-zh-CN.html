Material UI

Material UI 是一个全面的 React 组件库，采用我们独立实现的 Google Material Design 设计体系。 它已经经过了上千名开源贡献者超过十年的严格实战检验，因此被世界上一些最优秀的产品团队所信赖。

Material UI 的核心功能可通过 MUI X 扩展，MUI X 是一套用于高级用例的复杂组件集。 Toolpad 基于 Material UI 构建，提供全栈组件和低代码内部工具构建器。

Joy UI

本仓库还包含 Joy UI，这是一个实验性的组件库，实现了我们内部的 Joy Design 设计体系。 Joy UI 目前处于 beta 阶段，_开发已暂时搁置_。 如果你从零开始新项目，建议优先选择 Material UI，因为我们能保证持续支持。

请注意，维护者目前主要精力集中在其它项目上，可能无法及时响应与 Joy UI 相关的问题或拉取请求。

查看 Joy UI 文档。
