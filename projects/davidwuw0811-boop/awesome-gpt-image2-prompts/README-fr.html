🚀 Mode d'emploi

  • Visitez la page en ligne
  • Entrez un mot-clé dans la barre de recherche (supporte le chinois et l'anglais)
  • Cliquez sur une étiquette de catégorie pour filtrer
  • Une fois le prompt souhaité trouvé, cliquez sur le bouton "Copier"
  • Collez-le dans GPT Image 2 pour l'utiliser

📝 Format des prompts

Chaque prompt contient les informations suivantes :

  • Étiquette en chinois - Catégorie et description, pour faciliter la recherche des utilisateurs chinois
  • Prompt en anglais - Texte complet du prompt en anglais
  • Auteur - Auteur original du prompt
  • Source - Dépôt de données source
  • Image de référence - Certains prompts nécessitent le téléchargement d'une image de référence
