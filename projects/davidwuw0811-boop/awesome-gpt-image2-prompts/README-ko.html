🚀 사용 방법

  • 온라인 웹페이지 방문
  • 검색창에 키워드 입력 (중국어 및 영어 지원)
  • 분류 태그 클릭하여 필터링
  • 필요한 프롬프트 찾은 후 "복사" 버튼 클릭
  • GPT Image 2에 붙여넣어 사용

📝 프롬프트 형식

각 프롬프트는 다음 정보를 포함합니다:

  • 영어 프롬프트 - 완전한 영어 프롬프트 텍스트
  • 저자 - 프롬프트 원저자
  • 출처 - 데이터 출처 저장소
  • 참고 이미지 - 일부 프롬프트는 참고 이미지 업로드 필요
