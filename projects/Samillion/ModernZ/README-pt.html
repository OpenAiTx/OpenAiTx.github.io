ModernZ - Uma OSC Elegante Alternativa para mpv

Uma OSC elegante e moderna para mpv, este projeto é um fork do ModernX projetado para aprimorar a funcionalidade adicionando mais recursos, tudo isso preservando os padrões principais da OSC do mpv.

Personalização

Você pode alterar o tema do ícone para fluent ou material para combinar com seu gosto e estilo usando a opção icon_theme no seu modernz.conf

Não gosta de botões e texto brancos? Você tem controle total para personalizar cores e layouts de botões para refletir perfeitamente seu estilo.

Consulte a seção Personalização de Cores no guia de configuração para detalhes sobre como personalizar cores e botões.

Menus Interativos

ModernZ suporta o console/select embutido do mpv (v0.40+) para playlist, legendas, navegação por capítulos e mais.

Instalação

  • Desative a OSC padrão

  • Adicione osc=no no seu mpv.conf

  • Copie os Arquivos

  • Coloque modernz.lua no diretório de scripts do mpv
  • Coloque fluent-system-icons.ttf e material-design-icons.ttf no diretório de fontes do mpv
  • (OPCIONAL) Coloque thumbfast.lua no diretório de scripts do mpv

  • Locais

   Linux:   ~/.config/mpv/
   Windows: C:/Users/%username%/AppData/Roaming/mpv/
   macOS:   ~/Library/Application Support/mpv/

Traduções

ModernZ está disponível atualmente em inglês, mas você pode facilmente mudá-lo para seu idioma preferido! Veja como:

  • Baixe o pacote de locale:

Pegue o arquivo modernz-locale.json deste repositório. Este arquivo contém traduções para vários idiomas.

  • Adicione os locales ao mpv:

Copie o arquivo modernz-locale.json baixado para a pasta /script-opts do seu mpv.

  • Escolha seu idioma:

Há duas formas de definir seu idioma preferido:

  • Recomendado: Use o arquivo modernz.conf

`ini # Exemplo de configuração em modernz.conf # Definir idioma para Chinês Simplificado language=zh `

  • Alternativo: Edite o script modernz.lua

Abra o modernz.lua e localize a seção user_opts perto do início. Altere o valor language para o código do idioma desejado:

`lua local user_opts = { -- Geral language = "en", -- Altere para o código do idioma preferido ... } `

Extras

Os seguintes scripts são criados e mantidos por mim, sinta-se à vontade para usá-los se forem úteis para você.

  • Pause-Indicator-Lite - Um script simples que exibe um indicador na pausa

  • ytdlAutoFormat - Um script simples para mpv que altera automaticamente o ytdl-format (yt-dlp) para domínios especificados.

  • BoxtoWide - Um script simples para mpv que altera automaticamente a proporção 4:3 de arquivos/streams de vídeo para 16:9.

Para ainda mais scripts úteis, confira o mpv User Scripts Wiki. Ele oferece uma ampla gama de scripts contribuídos pela comunidade para melhorar sua experiência com o mpv.

Por que mais um fork?

  • Para adicionar muitos recursos em: Personalização de Cores, Opções e Integração de Locale
  • Para integrar console e select na osc, o que inspirou o mpv a aplicar isso na osc padrão. ref [#1, #2]
  • Para adicionar um layout dedicado para imagens. [detalhes]
  • Para refazer o projeto completamente para corresponder aos padrões da osc padrão do mpv, garantindo compatibilidade
  • Para eliminar bugs antigos e redundâncias no código
  • O que permite que outros forks Modern usem o ModernZ como base, como zydezu/ModernX. [Referência]

Essencialmente, para manter e reviver a origem do modern-osc`. Dito isso, o ModernZ ainda utiliza partes do código antigo, e todos os autores e colaboradores dos forks anteriores e atuais merecem crédito (incluindo o osc padrão do mpv), por isso eles são mencionados em detalhes.
