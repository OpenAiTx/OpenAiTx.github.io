Master Duel 社区中文翻译补丁

原项目地址 | 原发布页 | API来源

自2023年5月初版本起，MD修改了缓存文件的调用方式，原有的翻译补丁失效，故在原项目的基础上进行了修改。

本项目是在作者 @叶lf / @Timelic 工作的基础上进行的，在此表示感谢！另感谢 @拓王神 对数据解析部分提供的支持。

使用方式

前往Releases页面下载MDTR_vx.x.x.exe，同时参考本项目Wiki页面使用。

开发相关

本项目使用 UnityPy 解包，使用 flet 制作界面。编译环境为Python 3.10。

#### 准备

pip install -r requirements.txt

#### 开发

无界面

python3 index.py

flet 界面

flet run interface.py

#### 打包

PowerShell:

./release.ps1
