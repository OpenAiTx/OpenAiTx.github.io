Master Duel コミュニティ日本語翻訳パッチ

元プロジェクトURL | 元リリースページ | APIソース

2023年5月初旬のバージョンから、MDはキャッシュファイルの呼び出し方法を変更し、既存の翻訳パッチが無効になったため、元のプロジェクトをベースに修正を行いました。

本プロジェクトは作者 @叶lf / @Timelic の作業を基にしており、ここに感謝の意を表します！また、データ解析部分での支援をいただいた @拓王神 にも感謝します。

使用方法

ReleasesページからMDTR_vx.x.x.exeをダウンロードし、本プロジェクトのWikiページを参照してください。

開発関連

本プロジェクトは UnityPy を使ってアンパックし、flet でUIを制作しています。コンパイル環境はPython 3.10です。

#### 準備

pip install -r requirements.txt

#### 开发

无界面

python3 index.py

flet インターフェース

flet run interface.py

#### 打包

PowerShell:

./release.ps1
