備考：これはテスト版で多くの問題や機能不足がありますので、dockerやexeなどのワンクリックインストール・デプロイ方法は提供しておらず、ソースコードからの起動方法のみ提供しています。一定の技術基盤があり、新しいものを試したいユーザー向けです。今後、バージョンが十分に更新・安定したらワンクリックデプロイ方法を提供予定です。ご理解ください。

❗ 使用前に必読：

  • 現バージョンはゲームウィンドウが常に最前面にある必要がありますが、遮蔽は可能です
  • 現バージョンには初版のGUIインターフェースがあり、使用したい場合は python main_gui.py を起動してください
  • 現バージョンは自動収穫、自動除草/駆除/水やり、自動植え付け（テスト中）、自動盗み、自動支援除草/駆除/水やり、毎日無料肥料の自動受け取り機能を実装済みで、関連機能は継続的にテスト・最適化中です
  • 画像テンプレートマッチングは様々な要因でマッチしない、誤マッチする場合があるため、使用状況に応じてパラメータの閾値を調整する必要があります。閾値調整方法は以下を参照してください：threshold_adjust.txt
  • キーリスニングを設定し、ロボットのショートカット操作を可能にしています。ショートカット：Ctrl+cで即停止・終了、pで一時停止/再開

システム要件

  • Windows 10以上のOS（Linuxなど他OSは未対応）
  • ローカルにPython（conda環境推奨）をインストール済み、Pythonバージョンは3.9以上
  • NTアーキテクチャのデスクトップ版QQがインストールされており、農場ページが開けること

展開・試用手順

  • 1、git cloneで本プロジェクトをローカルに取得し、PC版QQにログイン
  • 2、プロジェクトのルートディレクトリでターミナルを開く（任意でconda環境切替）、コマンド pip install -r requirement.txt で依存パッケージをインストール
  • 3、ルートディレクトリの config.ini 設定ファイルを確認し、必要に応じて設定項目を変更
  • 4、コマンド python main.py でロボットを起動し、ターミナルで正常起動を確認（GUI版起動は python main_gui.py）
  • 5、農場ゲームのウィンドウを開く（現バージョンはゲームウィンドウが遮蔽されない状態にする必要あり）
  • 6、プログラムが自動でマウスを操作し、各種機能を自動化
