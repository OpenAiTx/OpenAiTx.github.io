QQ클래식 농장 로봇(AI 비전 버전)

CV 비전 매칭 기반의 초판이 개발 완료되었으며, Alaph 버전이 출시되었습니다. 주의: 이는 초기 테스트 버전으로 기능이 완전하지 않으며 지속적으로 개발 및 최적화 중입니다. 주로 일부 사용자들이 테스트용으로 사용하여 버그를 발견하고 의견을 제공하는 데 목적이 있습니다.

최신 기능 업데이트 로그는 다음에서 확인하세요: update_cv.txt

사용 중 문제 발생 시 issue를 제기할 수 있습니다: issue문제구역

기술 교류는 Discord에서 가능합니다: 농민 교류 채널

아래는 CV 버전의 배포 및 체험 방법입니다

참고: 이 버전은 테스트 버전이므로 여러 문제 및 기능 누락이 있을 수 있습니다. 따라서 docker, exe 등 원클릭 설치 및 배포 방법을 제공하지 않고, 소스코드 실행 방식만 제공합니다. 일정 수준 이상의 기술적 배경이 있고 새로운 기능을 체험하고자 하는 사용자들을 위한 것입니다. 이후 버전이 충분히 최적화되고 안정화되면 원클릭 배포 솔루션을 제공할 예정이니 양해 부탁드립니다.

❗ 사용 전 필독:

  • 현재 버전은 게임 창이 반드시 실시간으로 전면에 유지되어야 하지만, 가려지는 것은 허용됩니다.
  • 현재 버전에는 초판 GUI 인터페이스가 구현되어 있으며, 사용을 원하면 python main_gui.py를 실행하세요.
  • 현재 버전은 자동 수확, 자동 제초/해충 제거/물주기, 자동 심기(테스트 중), 자동 도둑질, 자동 제초/해충 제거/물주기 도움, 자동 매일 무료 비료 수령 기능을 구현했으며 관련 기능은 계속 테스트 및 최적화 중입니다.
  • 이미지 템플릿 매칭은 여러 요인에 의해 매칭 실패 또는 오매칭이 발생할 수 있으며, 사용 환경에 맞게 관련 파라미터 임계값을 조정해야 합니다. 임계값 파라미터 조정 방법은 다음을 참고하세요: threshold_adjust.txt
  • 키보드 리스너를 설정하여 로봇을 빠르게 제어할 수 있습니다. 단축키: Ctrl+c를 누르면 즉시 로봇을 중지하고 종료하며, p는 임시 일시정지/재개 기능입니다.

시스템 요구사항

  • Windows 10 이상 운영체제 (Linux 등 기타 시스템은 아직 미지원)
  • 로컬에 Python 설치(권장: conda 환경), Python 버전 ≥ 3.9
  • NT 아키텍처의 데스크톱 QQ 설치 및 농장 페이지 접속 가능

배포 및 체험 절차

  • 1. git clone으로 본 프로젝트를 로컬에 복제하고 PC버전 QQ에 로그인
  • 2. 프로젝트 루트 디렉터리에서 터미널을 열고(선택 사항: conda 환경 활성화) pip install -r requirement.txt 명령어로 의존성 설치
  • 3. 루트 디렉터리의 config.ini 설정 파일을 개인 필요에 맞게 수정
  • 4. python main.py 명령어로 로봇 실행 및 터미널에서 정상 실행 확인(GUI는 python main_gui.py)
  • 5. 농장 게임 창을 열고(현재 버전은 게임 창이 가려지지 않아야 함)
  • 6. 프로그램이 자동으로 마우스를 제어하여 관련 기능 버튼 클릭 및 자동화 수행
