arcade-songs

아케이드 게임의 노래와 악보를 검색할 수 있는 유틸리티 사이트입니다.

사전 요구 사항

  • Node.js (LTS 버전 이상)
  • Yarn

설치

# install dependencies
yarn install
  • .env.example 파일을 .env로 복사한 후 필수 항목을 작성하세요.

개발

# serve with hot reload at localhost
yarn run dev
serve with hot reload on local network

yarn run dev:host
  • 각 디렉토리에 대한 자세한 설명은 Nuxt 문서를 참조하세요.

프로덕션 빌드

SSG를 이용한 정적 호스팅 (권장)

# Build the application (if needed), generate every route as a HTML file and statically export to dist/ directory (used for static hosting).
yarn run generate
serve the dist/ directory like your static hosting would do (Netlify, Vercel, Surge, etc), great for testing before deploying.

yarn run serve

SSR을 이용한 서버 호스팅

# Build and optimize your application with webpack for production.
yarn run build
Start the production server (after running nuxt build). Use it for Node.js hosting like Heroku, Digital Ocean, etc.

yarn run start

기여 방법

arcade-songs는 현재 개발자와 번역가 모두의 번역 작업을 용이하게 하기 위해 설계된 웹 도구 Weblate를 사용하여 번역되고 있습니다.

번역에서 문제가 발견되면 다음 방법 중 하나로 개선에 도움을 줄 수 있습니다:

  • Weblate에서 직접 편집하기
  • 문제를 지적하기 위해 이슈 열기
  • 수정 사항을 제출하기 위한 풀 리퀘스트 생성

번역 파일은 locales/에 위치해 있습니다.

아이디어나 제안이 있다면 이슈 열기 또는 문의 양식 사용을 통해 전달할 수 있습니다.
