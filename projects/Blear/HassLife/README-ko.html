Home Assistant의 컴포넌트

이 Home Assistant 컴포넌트는 티몰 지니, 샤오아이, 샤오두 스피커를 HomeAssistant 플랫폼의 장비와 연동하여 장비를 제어하고 장비 상태를 조회 및 보고할 수 있습니다.

주의: 이 플러그인은 제어 명령 전송과 장비 상태 보고만 담당하며, 장비 추가는 HassLife 플랫폼에서 설정해야 합니다.

설치

1. 플러그인 설치

##### 방법1: hacs 설치【추천!】 >>사이드바 HACS -> 통합 -> hasslife 검색 후 다운로드 클릭

##### 방법2: 수동 다운로드 플러그인 다운로드 >>hasslife 폴더를 다운로드하여 /custom_components/ 경로에 저장합니다. custom_components 폴더가 없으면 직접 생성 후 HomeAssistant를 재시작합니다.

  • homeassistant 구성 디렉터리는 어디인가요?

>>Windows 사용자: %APPDATA%\.homeassistant

>>Linux 기반 사용자: locate .homeassistant/configuration.yaml 명령어 실행 후 찾은 .homeassistant 폴더가 구성 디렉터리입니다.

>>Synology Docker 사용자: Docker - 이미지 - homeassistant - 고급 설정 - 볼륨에서 /config에 매핑된 경로가 구성 디렉터리입니다.

2. 플러그인 구성

##### 방법1: 통합 UI 구성 >>HomeAssistant 사이드바의 구성 -> 통합 -> 우측 하단 통합 추가 -> hasslife 검색 후 클릭 -> 등록한 hasslife 계정과 비밀번호 입력 후 제출 클릭, 플러그인 최초 실행 시 시간이 필요하니 기다려 주세요.

해당 통합이 UI 구성을 지원하지 않는 경우, Home Assistant 웹사이트에서 링크로 접속했다면 최신 버전의 Home Assistant를 실행 중인지 확인하세요. 수동으로 계정 구성을 추가한 후 HomeAssistant를 재시작할 수 있습니다.

사용 단계：

  • HassLife 계정 등록 https://hass.blear.cn 에 접속하여 계정을 등록하고 로그인
  • 플러그인 설치: 설치 후 HomeAssistant 재시작, 통합 추가 페이지로 이동하여 HassLife 통합을 추가하고 HassLife 계정으로 로그인합니다.
  • HassLife 웹사이트 페이지로 다시 돌아가서, 플러그인 설정에 문제가 없다면 개인 센터 페이지에 플러그인 상태가 온라인으로 표시됩니다.
  • 장치 관리 메뉴를 클릭하고, 오른쪽 녹색의 Homeassistant 장치 추가 버튼을 클릭한 후, 추가하려는 장치를 선택하고 제출하세요.
  • 장치 목록에서 구성하려는 장치를 선택하고 구성을 클릭한 후 필요에 따라 접속할 스마트 스피커 플랫폼을 선택하고 장치 유형 및 음성 제어 이름을 설정하세요.
  • 해당 플랫폼의 스킬 바인딩 페이지로 이동하여 HassLife 계정을 바인딩하면, 문제 없이 추가한 장치를 볼 수 있습니다.
