Home Assistantのコンポーネント

このHome Assistantのコンポーネントは、天猫精霊、小愛同学、小度音箱をHomeAssistantプラットフォーム上のデバイスに接続し、デバイスの制御や状態の問い合わせ報告を行うことができます。

注意：本プラグインは制御およびデバイス状態の報告のみを担当し、デバイスの追加はHassLifeプラットフォームで設定する必要があります。

インストール

1.プラグインのインストール

##### 方法1:hacsインストール【推奨！】 >>サイドバーのHACS -> 統合 -> 検索でhasslifeを見つけてダウンロードをクリック

##### 方法2:手動ダウンロードプラグインダウンロード >>hasslifeフォルダをダウンロードし、/custom_components/ディレクトリに保存します。custom_componentsディレクトリがなければ自分で作成し、その後HomeAssistantを再起動してください。

  • homeassistantの設定ディレクトリはどこ？

>>Windowsユーザー: %APPDATA%\.homeassistant

>>Linux系ユーザー: locate .homeassistant/configuration.yamlコマンドを実行して見つかった.homeassistantフォルダが設定ディレクトリです。

>>Synology Dockerユーザー: Docker - イメージ - homeassistant - 詳細設定 - ボリュームに入り、/configに対応するパスが設定ディレクトリです。

2.プラグインの設定

##### 方法1:統合画面UI設定 >>HomeAssistantのサイドバーで設定 -> 統合 -> 右下の統合を追加 -> hasslifeを検索してクリック -> 登録済みのhasslifeアカウントとパスワードを入力し送信してください。プラグインの初回起動には時間がかかるため、しばらくお待ちください。

この統合がUIによる設定をサポートしていない場合や、Home Assistantのウェブサイトからリンクをクリックしてアクセスした場合は、最新版のHome Assistantを使用していることを確認してください。 アカウント設定を手動で追加し、HomeAssistantを再起動してください。

使用手順：

1.HassLifeアカウント登録 https://hass.blear.cn にアクセスし、アカウント登録とログインを行う 2.プラグインをインストール：インストール後HomeAssistantを再起動し、統合追加画面に移動してHassLife統合を追加し、HassLifeアカウントでログインする。
  • HassLifeのウェブサイトページに戻り、プラグインの設定に問題がなければ、個人センターページにプラグインの状態が「オンライン」と表示されます。
  • デバイス管理メニューをクリックし、右側の緑色の「Homeassistantデバイスを追加」ボタンをクリックし、追加したいデバイスを選択して送信します。
  • デバイスリストから設定したいデバイスを選び、「設定」をクリックし、必要に応じて接続したいスマートスピーカープラットフォームを選択し、デバイスタイプと音声制御名を設定します。
  • 対応するプラットフォームのスキルバインドページにアクセスし、HassLifeアカウントをバインドすると、問題なければ追加したデバイスが表示されます。

天猫精霊設定例

  • 最新版の「天猫精霊」アプリをインストールする
  • スマホブラウザで HassLife https://hass.blear.cn にアクセスし、hasslifeアカウントでログインする
  • 個人センターページで「天猫精霊スキル入口」をクリックする
  • 「プラットフォームアカウントをバインド」をクリックする
  • HassLifeアカウントでログインする
  • 承認を確認し、精霊の家タブに戻ると追加したデバイスが表示されます

小愛同学設定例

  • HassLifeのデバイスリストページに入り、必要なデバイスを追加し、対応するデバイスタイプと名前を設定する
  • 最新版の「米家」アプリをインストールする
  • 「米家」アプリを開く
  • 「マイ」タブをクリックする
  • 「その他プラットフォームデバイス」をクリックする
  • 右上の「追加」をクリックする
  • 「HassLife」を見つけてクリックする
  • 「アカウントをバインド」をクリックする
  • HassLifeアカウントでログインする
  • 承認を確認し、デバイスを同期する

小度設定例

  • HassLifeのデバイスリストページに入り、必要なデバイスを追加し、対応するデバイスタイプと名前を設定する
  • 最新版の「小度」アプリをインストールする
  • 「小度」アプリを開く
  • 右上のプラスボタンをクリックする
  • 「デバイスを追加」をクリックする
  • 「HassLife」を検索してクリックする
  • HassLifeアカウントでログインする
  • 承認を確認し、デバイスを同期する
