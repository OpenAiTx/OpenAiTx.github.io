DeTikZify
TikZ와 함께 과학 도표 및 스케치를 위한 그래픽 프로그램 합성

고품질 과학 도표를 만드는 것은 시간이 많이 걸리고 어려울 수 있습니다, 비록 아이디어를 종이에 스케치하는 것은 비교적 쉽다 하더라도. 또한, 의미 정보를 보존하지 않는 형식으로 저장된 기존 도표를 재현하는 것도 동일하게 복잡합니다. 이 문제를 해결하기 위해, 우리는 DeTikZify를 소개합니다. 이 새로운 멀티모달 언어 모델은 스케치와 기존 도표를 기반으로 의미를 보존하는 TikZ 그래픽 프로그램을 자동으로 합성합니다. 우리는 또한 추가 학습 없이 DeTikZify가 반복적으로 출력을 개선할 수 있도록 하는 MCTS 기반 추론 알고리즘을 도입합니다.

뉴스

2025-06-30: 우리는 최신 모델인 DeTik*Zifyv2.5 (8b)를 공개했습니다. 이 모델은 DeTikZifyv2를 기반으로 자기 피드백을 이용한 강화 학습 (RLSF)을 적용하였습니다. 우리는 이를 Hugging Face Space의 새로운 기본 모델로 설정했습니다. 자세한 내용은 모델 카드를 참고하세요. 2025-06-26: Tik*Zero가 ICCV 2025에서 하이라이트 페이퍼로 채택되었습니다! 2025-03-17: 우리는 Tik*Zero 어댑터를 공개했습니다. 이 어댑터는 DeTikZifyv2 (8b)에 직접 연결되어 제로샷 텍스트 조건부 기능을 제공합니다. 또한 추가 엔드투엔드 미세조정이 적용된 TikZero+도 공개했습니다. 자세한 내용은 우리의 논문과 아래 사용법을 참조하세요. 2024-12-05: 우리는 이전 버전들을 모두 능가하는 새로운 DeTik*Zify 모델 DeTikZifyv2 (8b)를 공개했습니다. 자세한 내용은 모델 카드를 확인하세요. 2024-09-24: DeTik*Zify가 NeurIPS 2024에서 스포트라이트 페이퍼로 선정되었습니다!

설치

[!TIP]
설치나 자체 하드웨어에서 추론 중 문제가 발생하면, 우리의 Hugging Face
Space를 방문하는 것을 고려하세요
(스페이스 재시작에 최대 30분이 소요될 수 있습니다). 긴 대기열이 있을 경우,
유료 개인 GPU 런타임으로 복제하거나
Docker를 사용해 로컬 실행을 할 수 있습니다.
또한 Google Colab에서 데모를 시도할 수 있습니다.
하지만 환경 설정에 시간이 걸릴 수 있으며, 무료 티어는 1b 모델 추론만 지원합니다.

DeTikZify의 파이썬 패키지는 pip를 통해 쉽게 설치할 수 있습니다:

pip install 'detikzify[legacy] @ git+https://github.com/potamides/DeTikZify'
[legacy] 추가 항목은 DeTikZifyv1 모델을 사용할 계획이 있을 때만 필요합니다. DeTikZifyv2만 사용할 계획이라면 제거할 수 있습니다. 포함된 예제를 실행하는 것이 목표라면, 저장소를 클론한 후 다음과 같이 편집 가능한 모드로 설치하는 것이 더 쉽습니다:

git clone https://github.com/potamides/DeTikZify
pip install -e DeTikZify[examples]

사용법
[!TIP]


대화형 사용과 일반적인 사용 팁을 위해,


명령줄에서 직접 시작할 수 있는 웹 UI를 확인하는 것을 권장합니다.


모든 옵션 목록은 --help를 사용하세요:


`sh


python -m detikzify.webui --light


`
필요한 모든 종속 항목이 설치되어 있다면, 컴파일, 렌더링, TikZ 그래픽 저장,
그리고 MCTS 기반 추론과 같은 DeTikZify의 모든 기능을 프로그래밍 인터페이스를 통해 사용할 수 있습니다:

DeTikZify 예시
python from operator import itemgetter

from detikzify.model import load from detikzify.infer import DetikzifyPipeline

image = "https://w.wiki/A7Cc" pipeline = DetikzifyPipeline(*load( model_name_or_path="nllg/detikzify-v2.5-8b", device_map="auto", torch_dtype="bfloat16", ))

generate a single TikZ program

fig = pipeline.sample(image=image)

if it compiles, rasterize it and show it

if fig.is_rasterizable: fig.rasterize().show()

run MCTS for 10 minutes and generate multiple TikZ programs

figs = set() for score, fig in pipeline.simulate(image=image, timeout=600): figs.add((score, fig))

save the best TikZ program

best = sorted(figs, key=itemgetter(0))[-1][1] best.save("fig.tex")

generate a single TikZ program

fig = pipeline.sample(text=caption)

if it compiles, rasterize it and show it

python from detikzify.model import load, load_adapter from detikzify.infer import DetikzifyPipeline

generate a single TikZ program

감사의 글

DeTikZify 모델 아키텍처의 구현은 LLaVA와 AutomaTikZ (v1), 그리고 Idefics 3 (v2)를 기반으로 합니다. 저희 MCTS 구현은 VerMCTS를 기반으로 합니다. TikZero 아키텍처는 Flamingo 와 LLaMA 3.2-Vision에서 영감을 받았습니다.
