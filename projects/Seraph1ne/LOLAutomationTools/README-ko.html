소개

LOLAutomationTools는 리그 오브 레전드(LOL)의 기능 확장 도구입니다; 일부 상황에서 양손을 사용할 수 없거나 잠시 자리를 비워야 할 때 간단한 클라이언트 조작을 도와 시간을 절약할 수 있도록 설계되었습니다; 소프트웨어는 플레이어의 게임 이해와 플레이 방식을 과도하게 간섭하지 않으므로 일부 기능은 여전히 수동 설정이 필요합니다(예: 챔피언 룬 설정); 과도한 제한은 무감각을 초래하고 게임 본연의 즐거움을 잃게 합니다; 즐겁고 긍정적인 게임 플레이를 통해 조화로운 환경을 함께 만들어 갑시다.

선택 영웅

선택 영웅과 금지 영웅을 사용하기 전, 반드시 설정 페이지에서 편집 작업(스위치 항목 뒤 버튼)을 해야 합니다.

라인 설정에 추가된 영웅 항목은 일부 사전 선택 슬롯이 있는 경기 모드에서만 적용됩니다(예: 랭크 게임, 익스트림 라이트닝), 랜덤은 랜덤 영웅 모드에서만 적용됩니다(예: 극지 대난투, 무한 난투, 금지 영웅 설정에는 랜덤 설정 페이지가 포함되지 않음), 일반은 그 외 모든 모드에서 적용됩니다(앞서 열거한 모드 제외, 사용 규칙을 위함), 자동 선택 영웅 실행 시 선택 영웅 설정 항목에 따라 위에서 아래 순서대로 선택됩니다.

사전 선택 영웅

이 기능의 실행 로직은 선택 영웅과 기본적으로 같으며, 차이는 사전 선택 영웅은 금지 단계가 있는 모드에서만 적용된다는 점입니다. 예를 들어 랭크 게임과 일부 금지 영웅이 필요한 재미 모드가 해당됩니다.

금지 영웅

금지 영웅 설정 페이지는 선택 영웅 설정 페이지와 기본적으로 동일하며, 오른쪽 상단에 모드 전환 버튼이 추가되어 있습니다. 버튼을 클릭하면 금지 모드로 전환할 수 있습니다.

일반 금지 모드 : 이 모드의 동작 논리는 선택 영웅 설정 페이지와 동일하며, 자동 금지 영웅 실행 시 금지할 영웅이 설정되어 있지 않으면 Ban 단계 남은 시간이 0초일 때 공백 Ban을 수행합니다.

연동 금지 모드 : 설정 목록의 항목이 선택 영웅 설정의 해당 영웅으로 대체되며, 항목을 선택한 상태에서 왼쪽 영웅 항목을 더블 클릭하면 해당 영웅이 영웅의 금지 그룹에 추가됩니다. 자동 금지 영웅 실행 시 선택 영웅 설정 항목 순서에 따라 금지하며, 모든 선택 영웅 설정의 금지 그룹에 금지할 영웅이 없으면 일반 금지 모드 논리에 따라 금지합니다.

팀원 사전 선택 무시를 활성화하면 자동 금지 영웅 실행 시 팀 사전 선택 검사를 건너뜁니다. 이 옵션은 활성화하지 않는 것이 좋습니다.

대국 수락

매칭 후 자동으로 대국을 수락합니다.

지연 대국 수락 옵션을 활성화하면 대국 수락은 수락 카운트다운 마지막 1초에만 자동으로 대국을 수락합니다.

대국 생성

대국 생성 버튼을 통해 수동으로 대국을 생성하거나 이 스위치를 켜 자동으로 대국을 생성할 수 있습니다.

자동 대국 생성은 더 많은 설정에서 생성할 대국 모드를 선택해야 하며, 이 옵션은 설정에 저장되지 않습니다.

대국 찾기

자동으로 대국을 찾습니다. 이 옵션은 대국 모드 방에서 자신이 소대 소유자(방장)일 때만 유효합니다.

시간 초과 검사 옵션을 활성화하면 매칭 대기열에서 예상 대기 시간을 초과해 지정한 간격만큼 대기 시 자동으로 대국 찾기를 다시 시도합니다.

칭찬 건너뛰기

대국 종료 시 팀원의 칭찬 페이지를 건너뜁니다.

결산 건너뛰기

대국 종료 시 결산 대기(검은 화면 로딩) 또는 경기 후 결산 페이지를 건너뛰고 대국 모드 방으로 돌아갑니다.

교환 기능

주 스위치와 세 개의 독립 스위치가 포함되어 있으며, 추가 설정이 필요합니다. 교환 기능 더 많은 설정에서 무작위 영웅 모드 시 팀원과 영웅 교환 여부 또는 소집 모드에서 팀원과 층(영웅 선택 순서) 및 사전 선택 위치를 교환할지 설정할 수 있습니다.

영웅 교환과 위치 교환 설정은 비교적 간단하여 여기서는 자세히 설명하지 않습니다.

교환 층의 더 많은 설정에서 교환 순서(즉, 층)와 교환 모드를 설정할 수 있습니다.

교환 모드：

  • 목표 순서만: 지정한 순서와 일치하는 팀원의 요청을 수락합니다.
  • 목표를 포함한 우선 순서: ≤ 지정한 순서의 팀원 요청을 수락합니다.
  • 목표를 포함한 연기 순서: ≥ 지정한 순서의 팀원 요청을 수락합니다.
  • 목표 순서 기반 범위: 순서 ≤ (지정 순서 - 범위) 및 순서 ≥ (지정 순서 + 범위)의 팀원 요청을 수락합니다.

또한, 기타 설정에는 두 가지 교환 거부 옵션이 있으며, 이 두 옵션이 활성화된 경우 팀원이 영웅이나 포지션 교환 요청 시 판단 조건에 부합하지 않으면 자동으로 교환 요청을 거부합니다. 그렇지 않으면 아무 작업도 하지 않으며, 직접 교환 수락 여부를 결정할 수 있습니다.

금지 선택 영웅 방지 퇴장

이 항목을 활성화하면 BP 단계 마지막 1초에 영웅 선택 또는 영웅 금지가 실행됩니다.

기능 설정 페이지에서 금지 선택 영웅 방지 퇴장 기능의 모드 설정을 변경할 수 있습니다.

잠시 자리 비움 자동 응답

이 항목은 주 스위치이며, 추가 설정이 필요합니다. 더보기 구성 페이지에서 떠날 때 친구 메시지 또는 게임 초대에 자동으로 응답할지 설정할 수 있으며(기본 활성화 상태), 응답할 텍스트 내용을 설정할 수 있습니다.

추가 창

이 항목을 활성화하면 일부 클라이언트 단계에서 추가 창이 표시됩니다. 또한 이 스위치 옆 버튼을 통해 추가 창의 구성 페이지에 들어갈 수 있으며, 여기서 현재 사용 가능한 엔터테인먼트 모드에서 영웅의 밸런스 속성(있는 경우)을 확인하고 각 모드에서 사용할 룬 구성과 클라이언트 독립 구성을 설정할 수 있습니다.

추가 창 구성 페이지

룬 구성 페이지에서 영웅의 룬 구성을 추가하거나 업데이트하려면, 게임 클라이언트의 소장품 - 룬에서 룬 페이지를 선택해 편집 후 저장하고, 오른쪽 상단에서 소환사 기술을 변경한 뒤 구성 추가 버튼을 누르거나 업데이트할 룬 구성 항목에서 현재 게임 구성을 이 구성으로 업데이트 버튼을 눌러 룬 구성을 추가하거나 업데이트해야 합니다.

자동 종료 대국

간단 명료하게, 게임 대국 종료 시 크리스탈 종료 애니메이션 재생 중에 게임 프로세스를 자동 종료합니다.

자동 재접속 대국

게임 재접속이 필요할 때 재접속을 시도합니다.

자동 카드 선택(추가 확장)

현재 게임에서 사용하는 영웅이 카드 마스터일 경우, Z/X/C 키를 사용해 자동으로 파란/빨간/노란 카드를 선택합니다.

이 기능은 그래픽 처리와 키보드 시뮬레이션 방식을 사용하여 구현되었으며, 현재 비교적 많이 사용되는 두 가지 해상도(1920x1080/2560x1440)만 테스트되었습니다. 다른 해상도는 별도의 설정이 없으면 정상적으로 작동하지 않습니다(특히 16:9 비율이 아닌 해상도). 감지는 일정한 필터 범위를 허용하지만, 이 기능 사용 시 필터를 켜지 않는 것을 권장하며, 그렇지 않으면 일부 감지가 무효화될 수 있습니다.
