Pendahuluan

LOLAutomationTools adalah alat ekstensi fungsional untuk League of Legends; digunakan untuk membantu pemain melakukan beberapa operasi sederhana pada klien saat tangan tidak bisa digunakan atau harus meninggalkan sementara, sehingga dapat menghemat sedikit waktu; perangkat lunak ini tidak akan terlalu banyak mengintervensi pemahaman dan cara bermain pemain terhadap permainan, jadi beberapa fitur masih harus dikonfigurasi secara manual terlebih dahulu (seperti konfigurasi rune hero), terlalu banyak pembatasan akan membuat pemain jenuh dan kehilangan kesenangan dari permainan itu sendiri; bermainlah dengan positif dan bahagia, bangun lingkungan yang harmonis bersama.

Pilih Hero

Sebelum menggunakan Pilih Hero dan Nonaktifkan Hero, Anda perlu melakukan pengeditan di halaman pengaturan (tombol di sebelah sakelar).

Hero yang ditambahkan pada pengaturan jalur hanya berlaku di beberapa mode pertandingan dengan slot pra-pilih (seperti Ranked dan Ultimate Blitz), acak hanya berlaku di mode Hero Acak (seperti ARAM, URF, konfigurasi nonaktifkan hero tidak termasuk halaman pengaturan acak), umum berlaku di semua mode lainnya (tidak termasuk mode yang disebutkan sebelumnya, ini untuk standarisasi penggunaan), saat melakukan pemilihan hero otomatis, akan mengikuti pengaturan pada Pilih Hero dan urut dari atas ke bawah.

Pra-pilih Hero

Logika fungsi ini pada dasarnya sama dengan Pilih Hero, perbedaannya adalah Pra-pilih Hero hanya berlaku pada mode dengan fase nonaktifkan, seperti Ranked dan beberapa mode hiburan yang membutuhkan nonaktifkan hero.

Nonaktifkan Hero

Halaman konfigurasi Nonaktifkan Hero pada dasarnya sama dengan halaman konfigurasi Pilih Hero, hanya ada tombol peng sopan mode di pojok kanan atas, klik tombol untuk beralih mode nonaktifkan.

Mode Nonaktifkan Normal: Logika operasi mode ini sama dengan halaman konfigurasi Pilih Hero, saat melakukan nonaktifkan hero otomatis, jika tidak ada hero yang dikonfigurasi untuk dinonaktifkan, maka akan melakukan Ban kosong ketika waktu fase Ban tersisa 0 detik.

Mode Nonaktifkan Kelompok: Daftar konfigurasi akan diganti dengan hero yang sesuai pada konfigurasi Pilih Hero, saat memilih sebuah item, klik dua kali pada hero di sebelah kiri untuk menambahkannya ke Grup Nonaktifkan hero tersebut, saat melakukan nonaktifkan hero otomatis, akan mengikuti urutan pengaturan pada Pilih Hero, jika semua Grup Nonaktifkan pada konfigurasi Pilih Hero tidak memiliki hero untuk dinonaktifkan, maka akan mengikuti logika Mode Nonaktifkan Normal.

Jika Abaikan Pra-pilih Tim diaktifkan, maka saat melakukan nonaktifkan hero otomatis akan melewati deteksi pra-pilih tim. Tidak disarankan mengaktifkan opsi ini.

Terima Pertandingan

Secara otomatis menerima pertandingan setelah pencocokan pertandingan ditemukan.

Setelah opsi Tunda Terima Pertandingan diaktifkan, Terima Pertandingan hanya akan menerima pertandingan secara otomatis pada detik terakhir dari hitungan mundur penerimaan.

Buat Pertandingan

Anda dapat membuat pertandingan secara manual melalui tombol Buat Pertandingan atau mengaktifkan pengaturan ini untuk membuat pertandingan secara otomatis.

Buat Pertandingan Otomatis memerlukan pemilihan mode pertandingan yang ingin dibuat pada bagian Pengaturan Lainnya, dan opsi ini tidak akan disimpan ke dalam konfigurasi.

Cari Pertandingan

Secara otomatis mencari pertandingan, opsi ini hanya berlaku ketika Anda berada di ruang mode pertandingan dan menjadi pemilik tim (host).

Jika opsi Deteksi Timeout diaktifkan, maka akan secara otomatis mencari ulang pertandingan setelah berada di antrean pencocokan melebihi estimasi waktu antrean pada interval yang ditentukan.

Lewati Pujian

Melewati halaman pujian rekan tim saat pertandingan berakhir.

Lewati Penyelesaian

Melewati proses tunggu penyelesaian (layar hitam berputar memuat) atau halaman penyelesaian pasca pertandingan, lalu kembali ke ruang mode pertandingan.

Fitur Tukar

Terdiri dari sakelar utama dan tiga sakelar independen, Anda juga perlu melakukan pengaturan lainnya. Pada pengaturan lanjutan Fitur Tukar, atur apakah akan menukar hero dengan rekan tim pada mode acak, atau menukar lantai (urutan pemilihan hero) dan posisi prapilih di mode draft.

Pengaturan tukar hero dan tukar posisi relatif sederhana, tidak dibahas lebih lanjut di sini.

Pada pengaturan lanjutan Tukar Lantai, Anda dapat mengatur urutan (lantai) yang ingin ditukar dan mode penukarannya.

Mode Tukar:

  • Hanya urutan target: menerima permintaan rekan tim yang sesuai dengan urutan yang ditentukan.
  • Prioritas termasuk target: menerima permintaan rekan tim dengan urutan ≤ urutan yang ditentukan.
  • Urutan setelah termasuk target: menerima permintaan rekan tim dengan urutan ≥ urutan yang ditentukan.
  • Rentang berdasarkan urutan target: menerima permintaan rekan tim dengan urutan ≤ (urutan yang ditentukan - rentang) dan urutan ≥ (urutan yang ditentukan + rentang).

Selain itu, pada Pengaturan Lainnya terdapat dua opsi tolak tukar, jika diaktifkan maka permintaan tukar hero atau posisi oleh rekan tim yang tidak memenuhi kriteria akan otomatis ditolak, jika tidak tidak ada tindakan, Anda dapat memutuskan sendiri apakah ingin menerima penukaran.

Pencegahan Keluar Saat Ban/Pick Hero

Jika diaktifkan, pada detik terakhir fase BP akan otomatis melakukan pilih hero atau ban hero.

Kamu dapat mengubah mode fitur Pencegahan Keluar Saat Ban/Pick Hero di halaman pengaturan fitur.

Balas Otomatis Saat Sementara Pergi

Switch ini adalah saklar utama, kamu masih perlu melakukan pengaturan lain. Pada halaman konfigurasi Lainnya, atur apakah akan otomatis membalas pesan teman atau undangan pertandingan saat kamu keluar (masing-masing secara default dalam keadaan aktif), dan atur isi teks balasannya.

Jendela Tambahan

Setelah diaktifkan, jendela tambahan akan muncul di beberapa tahap klien. Selain itu, kamu bisa masuk ke halaman konfigurasi dari tombol di sebelah switch ini, dan di situ juga bisa melihat atribut keseimbangan hero dalam mode hiburan yang tersedia (jika ada), serta mengatur konfigurasi rune dan pengaturan klien secara terpisah untuk setiap mode.

Halaman Konfigurasi Jendela Tambahan

Jika kamu ingin menambah atau memperbarui konfigurasi rune hero pada halaman konfigurasi rune, kamu perlu memilih satu halaman rune di koleksi - rune di klien game, edit dan simpan, lalu dari pojok kanan atas ubah skill summoner dan tekan tombol Tambah Konfigurasi atau tombol Perbarui Konfigurasi Game Saat Ini ke Konfigurasi Ini pada item konfigurasi rune yang ingin diperbarui untuk menambah atau memperbarui konfigurasi rune.

Jendela Tambahan

Tutup Otomatis Setelah Pertandingan

Sederhana dan langsung, secara otomatis menutup proses game ketika animasi crystal end selesai setelah pertandingan berakhir.

Reconnect Otomatis ke Pertandingan

Akan mencoba melakukan reconnect otomatis ketika game membutuhkan koneksi ulang.

Auto Pick Kartu (Ekstra Tambahan)

Jika menggunakan hero Twisted Fate di game, gunakan tombol Z/X/C untuk otomatis memilih kartu Biru/Merah/Kuning.

Fitur ini menggunakan pengolahan grafis dan simulasi keyboard, hanya diuji di dua resolusi umum (1920x1080/2560x1440), resolusi lain tidak dapat digunakan secara normal jika belum dikonfigurasi sendiri (terutama resolusi di luar rasio 16:9), deteksi mengizinkan rentang filter tertentu, namun disarankan saat menggunakan fitur ini jangan aktifkan filter agar deteksi tidak gagal.
