DailyAnimeList - MyAnimeList Client

A Flutter based MyAnimeList client with a clean and simple UI.

Download

Note: The app has been delisted from the Google Play Store. Please use GitHub Releases or F-Droid to download the app.

Features

Explore the extensive MyAnimeList database to discover your favorite anime and manga. Engage in forums with millions of discussions at your fingertips. Access detailed reviews and receive numerous recommendations for all anime and manga on MyAnimeList.

Screenshots

⚡️ 动漫 & 漫画 ⚡️

  • 发现季节性动漫、即将上映的热门动漫、最受欢迎动漫、动漫排名列表、历年最爱等。

  • 访问动漫/漫画简介、相关和推荐内容。

  • 阅读详细的动漫评论并查看动漫/漫画统计数据。

⚡️ 主题和自定义 ⚡️

  • 选择4种不同的暗色主题：春、夏、秋和冬。

  • 自定义底部导航栏并调整缓存更新频率。

⚡️ MyAnimeList 论坛 ⚡️

  • 参与专注于 MyAnimeList、动漫 & 漫画及综合讨论的论坛。

⚡️ 高级搜索栏 ⚡️

  • 使用 "@" 和 "#" 在搜索栏中直接进行高级搜索，轻松便捷。

  • 通过缓存机制体验更快的搜索加载时间。

⚡️ 用户专属功能 ⚡️

  • 快速编辑/更新您的动漫/漫画列表。

  • 按标题、列表评分、开始日期和更新日期排序您的列表。

入门指南

  • 运行此项目，您需要在机器上安装 Flutter。
  • 将此仓库克隆到您的机器：git clone
  • 使用 dal-code.code-workspace 在 VSCode 中打开项目
  • 运行脚本 update_gradle.sh 以准备好 Gradle 文件用于开发
  • 运行 flutter pub get 安装依赖项
  • 使用 flutter run 或在 VSCode 中按 F5 启动应用

额外设置

  • 运行用户任务 - intl generate 生成本地化文件
  • 运行用户任务 - build runner 生成必要文件
  • 运行用户任务 - python translate 生成翻译文件
