📝 描述

Simple Columns 插件允许你在 Obsidian 中轻松创建可调整大小和自定义的列。无论你是在组织任务、拆分内容还是可视化想法，这个插件都提供了简单的 Markdown 语法，帮助你快速调整和构建布局。

⚙️ 使用方法

要创建一个简单的列布局，你可以：
  • 在笔记中右键点击并从子菜单中选择相应选项。

  • 使用命令面板直接插入列。例如，输入“Add 2 columns”即可在笔记中插入一个两列布局。

当你创建列时，插件会生成带有唯一 ID 的以下 Markdown：

id: b52707b0-ce15-6458-825d-32615b4b7h85
===
Column 1
===
Column 2

示例：使用代码块创建列

要创建包含代码块的列布局，请使用以下语法（以4个反引号开头和结尾：```）： 

``columns
编号：b52707b0-ce15-6458-825d-32615b4b7h85
===
列 1
javascript console.log("Hello world") 
===
第2栏
 `


示例：通过 YAML 指定列宽
您可以使用 YAML 为特定列设置自定义宽度。指定了比例的列将优先分配宽度，未定义比例的列将平分剩余空间。
id: b52707b0-ce15-6458-825d-32615b4b7h85 column-1-ratio: 60% === Column 1 hello === Column 2 ` 解释：
  • column-1-ratio：60% → 第一列占容器宽度的60%。
  • 第二列（未指定）自动填充剩余的40%。
  • 你可以为多列定义比例（column-2-ratio，column-3-ratio 等），最多支持4列。

✨ 功能

  • 唯一列ID：每列生成唯一ID，确保你可以独立地样式化或引用各列。
  • 轻松调整列宽：列宽完全可调，让你对内容布局拥有完全控制。
  • 响应式布局：列根据屏幕大小自动调整，提供整洁且可适应的布局。
  • 高度可定制：列完全可调节，方便你轻松控制大小、间距和对齐方式以满足需求。

🔧 全局设置

全局设置允许你为整个Obsidian库中的所有列定义默认样式和行为。这些设置可以在插件设置中访问和配置。

容器边框

你可以自定义列容器的边框外观以增强可见性。

  • 显示容器边框：切换此设置以显示每个列容器的边框。
  • 边框宽度：设置列容器边框的宽度。
  • 边框颜色：选择边框颜色。
  • 边框透明度：使用RGBA颜色码调整边框透明度。alpha值（最后一个数字）控制透明度。

调整器设置

你也可以自定义列调整器的外观和行为。

  • 显示调整器：切换此设置以显示或隐藏列之间的调整器。调整器仅在鼠标悬停于列分隔线时出现。
  • 调整器宽度：设置调整器元素的宽度。
  • 调整器颜色：选择调整器颜色。
  • 调整器透明度：使用RGBA颜色码调整调整器透明度。

高级设置

  • 重置样式：此操作将所有列样式，包括边框设置和调整器样式，恢复为默认值。
  • 清除本地存储：此操作将移除所有自定义设置，并将插件恢复到初始状态。如遇问题或想重新开始，请使用此选项。

⚡️ 本地设置

本地设置允许您对笔记中的各个列应用更细粒度的自定义。

可用的本地设置：

  • 重置所有样式：将该列的所有自定义样式恢复为默认设置。
  • 显示边框
  • 设置边框颜色和透明度
  • 显示调整器：切换列调整器的显示。调整器仅在悬停时出现。
  • 设置调整器颜色和透明度
  • 设置文本颜色
  • 设置背景颜色和透明度
  • 设置文本对齐：设置列中文本的对齐方式（例如，左对齐、居中、右对齐`）。

📦 安装

该插件现已在 Obsidian 社区插件中提供！🎉

  • 在 Obsidian 中，进入设置 > 社区插件。
  • 点击浏览并搜索 Simple Columns。
  • 点击安装，然后启用该插件。
