📝 설명

Simple Columns 플러그인은 Obsidian에서 노트 내 쉽게 크기 조절 및 사용자 지정 가능한 컬럼을 만들 수 있게 해줍니다. 작업을 정리하거나, 내용을 분할하거나, 아이디어를 시각화할 때 이 플러그인은 간단한 마크다운 문법으로 레이아웃을 빠르게 조정하고 구성할 수 있게 합니다.

⚙️ 사용 방법

간단한 컬럼 레이아웃을 만들려면 다음 중 하나를 선택하세요:
  • 노트 내에서 우클릭 후 하위 메뉴에서 옵션을 선택합니다.

  • 명령 팔레트를 사용해 컬럼을 직접 삽입합니다. 예를 들어 "2개의 컬럼 추가"를 입력하면 노트에 2단 컬럼 레이아웃을 삽입합니다.

컬럼을 만들면 플러그인은 각 컬럼에 고유 ID가 포함된 다음 마크다운을 생성합니다:

id: b52707b0-ce15-6458-825d-32615b4b7h85
===
Column 1
===
Column 2

예제: 코드 블록으로 열 만들기

코드 블록을 열 안에 배치하는 레이아웃을 만들려면, 다음 구문을 사용하세요 (4개의 백틱 ``` 으로 시작하고 끝냅니다):


id: b52707b0-ce15-6458-825d-32615b4b7h85
===
열 1
javascript console.log("Hello world") 
===
열 2


예제: YAML을 통한 열 너비 지정
YAML을 사용하여 특정 열에 대한 사용자 지정 너비를 설정할 수 있습니다. 비율이 지정된 열이 우선 적용되며, 비율이 정의되지 않은 열은 남은 공간을 균등하게 나눕니다.
id: b52707b0-ce15-6458-825d-32615b4b7h85 column-1-ratio: 60% === Column 1 hello === Column 2 ` 설명:
  • column-1-ratio: 60% → 컬럼 1이 컨테이너 너비의 60%를 차지합니다.
  • 컬럼 2 (명시되지 않음)는 나머지 40%를 자동으로 채웁니다.
  • 여러 컬럼에 대해 비율을 정의할 수 있습니다 (column-2-ratio, column-3-ratio 등), 최대 4개 컬럼까지 가능합니다.

✨ 기능

  • 고유 컬럼 ID: 각 컬럼은 고유한 ID로 생성되어, 컬럼을 독립적으로 스타일링하거나 참조할 수 있습니다.
  • 쉽게 크기 조절 가능한 컬럼: 컬럼은 완전히 크기 조절이 가능하여 콘텐츠 레이아웃을 완벽하게 제어할 수 있습니다.
  • 반응형 레이아웃: 컬럼은 화면 크기에 따라 자동으로 조정되어 깔끔하고 적응력 있는 레이아웃을 제공합니다.
  • 높은 맞춤 설정 가능: 컬럼은 완전히 조절 가능하여 크기, 간격, 정렬을 쉽게 제어할 수 있습니다.

🔧 전역 설정

전역 설정은 Obsidian 볼트 내 모든 컬럼에 대해 기본 스타일과 동작을 정의할 수 있게 합니다. 이 설정은 플러그인 설정에서 접근하고 구성할 수 있습니다.

컨테이너 테두리

더 나은 가시성을 위해 컬럼 컨테이너의 테두리 모양을 사용자화할 수 있습니다.

  • 컨테이너 테두리 표시: 이 설정을 토글하여 각 컬럼 컨테이너 주위에 테두리를 표시합니다.
  • 테두리 두께: 컬럼 컨테이너 테두리의 두께를 설정합니다.
  • 테두리 색상: 테두리 색상을 선택합니다.
  • 테두리 투명도: RGBA 색상 코드를 사용하여 테두리의 투명도를 조절합니다. 알파 값(마지막 숫자)이 투명도를 제어합니다.

리사이저 설정

컬럼 리사이저의 모양과 동작도 사용자화할 수 있습니다.

  • 리사이저 표시: 이 설정을 토글하여 컬럼 사이의 리사이저를 표시하거나 숨깁니다. 리사이저는 컬럼 구분선 위에 마우스를 올릴 때만 나타납니다.
  • 리사이저 너비: 리사이저 요소의 너비를 설정합니다.
  • 리사이저 색상: 리사이저 색상을 선택합니다.
  • 리사이저 투명도: RGBA 색상 코드를 사용하여 리사이저의 투명도를 조절합니다.

고급 설정

  • 스타일 초기화: 테두리 설정 및 리사이저 스타일을 포함한 모든 컬럼 스타일을 기본값으로 되돌립니다.
  • 로컬 저장소 초기화: 모든 사용자 설정을 제거하고 플러그인을 초기 상태로 복원합니다. 문제가 발생하거나 새로 시작하고 싶을 때 이 옵션을 사용하세요.

⚡️ 로컬 설정

로컬 설정을 통해 노트 내 개별 열에 대해 더 세밀한 사용자 지정을 적용할 수 있습니다.

사용 가능한 로컬 설정:

  • 모든 스타일 초기화: 열에 적용된 모든 사용자 지정 스타일을 기본 설정으로 되돌립니다.
  • 테두리 표시
  • 테두리 색상 및 투명도 설정
  • 리사이저 표시: 열 리사이저의 표시를 전환합니다. 리사이저는 마우스 오버 시에만 나타납니다.
  • 리사이저 색상 및 투명도 설정
  • 텍스트 색상 설정
  • 배경 색상 및 투명도 설정
  • 텍스트 정렬 설정: 열 내 텍스트 정렬을 지정합니다(예: 왼쪽, 가운데, 오른쪽`).

📦 설치

이 플러그인은 이제 Obsidian 커뮤니티 플러그인에서 사용할 수 있습니다! 🎉

  • Obsidian에서 설정 > 커뮤니티 플러그인으로 이동합니다.
  • 찾아보기 클릭 후 Simple Columns를 검색합니다.
  • 설치를 클릭하고 플러그인을 활성화합니다.
