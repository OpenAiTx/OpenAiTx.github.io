Deno (/ˈdiːnoʊ/, telaffuz dee-no) güvenli varsayılanlar ve harika bir geliştirici deneyimi sunan bir JavaScript, TypeScript ve WebAssembly çalışma zamanıdır. V8, Rust ve Tokio üzerine kuruludur.

Deno çalışma zamanı hakkında daha fazla bilgiyi dökümantasyonda bulabilirsiniz.

Kurulum

Aşağıdaki komutlardan birini kullanarak sisteminize Deno çalışma zamanını kurun. Deno'yu kurmanın birçok yolu olduğunu unutmayın - kapsamlı kurulum seçenekleri listesini burada bulabilirsiniz.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

İlk Deno programınız

Deno birçok farklı uygulama için kullanılabilir, ancak en yaygın olarak web sunucuları oluşturmak için kullanılır. server.ts adında bir dosya oluşturun ve aşağıdaki TypeScript kodunu ekleyin:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Sunucunuzu aşağıdaki komutla çalıştırın:

deno run --allow-net server.ts

Bu, yerel bir web sunucusunu http://localhost:8000 adresinde başlatmalıdır.
