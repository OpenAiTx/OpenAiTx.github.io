Deno (/ˈdiːnoʊ/, uitgesproken als dee-no) is een JavaScript-, TypeScript- en WebAssembly-runtime met veilige standaardinstellingen en een uitstekende ontwikkelaarservaring. Het is gebouwd op V8, Rust en Tokio.

Leer meer over de Deno-runtime in de documentatie.

Installatie

Installeer de Deno-runtime op uw systeem met een van de onderstaande commando's. Merk op dat er verschillende manieren zijn om Deno te installeren – een uitgebreide lijst van installatie-opties is te vinden hier.

Shell (Mac, Linux):

curl -fsSL https://deno.land/install.sh | sh

PowerShell (Windows):

irm https://deno.land/install.ps1 | iex

Homebrew (Mac):

brew install deno

Chocolatey (Windows):

choco install deno

WinGet (Windows):

winget install --id=DenoLand.Deno

Bouwen en installeren vanuit broncode

Volledige instructies voor het bouwen van Deno vanuit de broncode zijn te vinden hier.

Uw eerste Deno-programma

Deno kan voor veel verschillende toepassingen worden gebruikt, maar wordt het vaakst gebruikt om webservers te bouwen. Maak een bestand genaamd server.ts en voeg de volgende TypeScript-code toe:

Deno.serve((_req: Request) => {
  return new Response("Hello, world!");
});

Voer uw server uit met het volgende commando:

deno run --allow-net server.ts

Dit zou een lokale webserver moeten starten op http://localhost:8000.

Leer meer over het schrijven en uitvoeren van Deno-programma’s in de documentatie.
