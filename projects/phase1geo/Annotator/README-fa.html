حاشیه‌نویس

نمای کلی

تصاویر خود را حاشیه‌نویسی کنید و بگذارید تصویر هزاران حرف بگوید.

  • بارگذاری تصویر از سیستم فایل، کلیپ‌بورد یا ایجاد اسکرین‌شات برای حاشیه‌نویسی.
  • افزودن اشکال، برچسب‌ها، تصاویر، متن، نقاشی و سایر نشانه‌گذاری‌ها برای برجسته‌کردن جزئیات تصویر.
  • افزودن ذره‌بین برای افزایش جزئیات تصویر.
  • محو کردن بخش‌هایی از تصویر برای مخفی‌سازی داده‌ها.
  • برش، تغییر اندازه و افزودن حاشیه به تصویر.
  • کنترل رنگ‌ها، ضخامت خطوط و جزئیات فونت.
  • پشتیبانی از بزرگ‌نمایی.
  • پشتیبانی از انتخابگر رنگ در تصویر بارگذاری‌شده.
  • بازگردانی/انجام مجدد نامحدود هر تغییری.
  • کشیدن و رها کردن نسخه PNG تصویر حاشیه‌نویسی‌شده.
  • صادر کردن به فرمت‌های تصویری JPEG، PNG، TIFF، BMP، PDF، SVG و WebP.
  • پشتیبانی از کپی تصویر حاشیه‌نویسی‌شده به کلیپ‌بورد.
  • پشتیبانی از چاپگر.

نصب

دبیان (از سورس)

برای ساخت Annotator به وابستگی‌های زیر نیاز دارید:

  • meson
  • valac
  • debhelper
  • gobject-2.0
  • glib-2.0
  • libgee-0.8-dev
  • libgranite-7-dev
  • libgtk-4-dev
  • libxml2-dev
  • libarchive-dev
  • libwebp-dev

برای نصب Annotator از سورس، دستور ./app install را اجرا کنید.

برای اجرای Annotator، دستور com.github.phase1geo.annotator را اجرا کنید.

اوبونتو (PPA)

می‌توانید از PPA که توسط @PandaJim نگهداری می‌شود استفاده کنید. این PPA از اوبونتو ۲۰.۰۴ به بالا پشتیبانی می‌کند. دستورات زیر را یکی‌یکی وارد کنید

sudo add-apt-repository ppa:ubuntuhandbook1/annotator
sudo apt update
sudo apt install com.github.phase1geo.annotator

آرچ لینوکس

اگر کاربر آرچ لینوکس هستید، یک بسته AUR به نام annotator وجود دارد:

% yay -S annotator

فلت‌پک

علاوه بر این، Annotator می‌تواند از طریق فلت‌پک نصب و اجرا شود.

برای ساخت فلت‌پک المنتری از سورس، دستور ./app elementary را اجرا کنید.

برای ساخت فلت‌پک فلت‌هاب از سورس، دستور ./app flathub را اجرا کنید.

سپس می‌توانید آن را با دستور: ./app run-flatpak اجرا کنید.
