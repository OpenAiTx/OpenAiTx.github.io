Awesome 3D Tiles

Uma lista curada de softwares open source incríveis, bibliotecas, ferramentas e recursos para 3D Tiles.

Contribuições são bem-vindas. Adicione links através de pull requests ou crie uma issue para iniciar uma discussão.

3D Tiles é uma especificação aberta para compartilhar, visualizar, fundir e interagir com grandes volumes de conteúdo geoespacial 3D heterogêneo em aplicativos desktop, web e mobile.

3D Tiles é um padrão comunitário OGC.

Visualizadores

  • CesiumJS - Biblioteca JavaScript para criar globos 3D e mapas 2D em navegadores web.
  • 3DCityDB-Web-Map-Client - Visualizador baseado em Cesium para CityGML e 3D Tiles.
  • loaders.gl - Parsers e encoders para diversos formatos principais 3D, geoespaciais e tabulares.
  • deck.gl - Framework baseado em WebGL para análise visual exploratória de grandes conjuntos de dados.
  • 3d-tiles-renderer - Renderizador para 3D Tiles baseado em Three.js.
  • three-loader-3dtiles - Módulo loader para Three.js para lidar com OGC 3D Tiles.
  • threedtiles - Visualizador de 3DTiles para three.js.
  • mapbox-3dtiles - Camada personalizada do Mapbox GL JS para 3D Tiles.
  • A-Frame component - Componente para A-Frame usando 3D Tiles.
  • iTowns - Framework JS/WebGL baseado em Three.js.
  • giro3d - Framework JS/WebGL baseado em Three.js (sucessor do iTown).
  • UD-Viz - Visualização de Dados Urbanos. Framework para criar aplicações web para visualização e interação com dados urbanos geoespaciais 3D (baseado em iTowns/Three.js).
  • vsgCs - Biblioteca para usar 3D Tiles e outros conteúdos geoespaciais em aplicações Vulkan Scene Graph (VSG).

Integrações com engines de jogos, GIS desktop e ferramentas de modelagem 3D (Open source)

  • QGIS - Sistema de informação geográfica desktop multiplataforma (desde a versão 3.34).
  • 3D Tiles for Godot - Extensão para Godot 4 que integra capacidades do Cesium 3D Tiles ao Godot Engine.
  • Open 3D Engine (O3DE) - Cesium para O3DE.
  • blender-3d-tiler - Ferramenta para Blender para tiling de modelos 3D.

Criação de Tiles (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - Biblioteca C++ para streaming de 3D Tiles e processamento de glTF.
  • 3D Tiles Tools - Ferramentas e utilitários para conversão, otimização, processamento e análise de dados 3D Tiles.
  • glTF Pipeline - Ferramentas Javascript para conversão e otimização glTF / GLB.

Criação de Tiles (Comunidade)

  • py3dtiles - Leitura e escrita de pnts e b3dm via API ou CLI.
  • Python 3DTiles Tilers - Ferramenta e biblioteca Python para construir tilesets 3D Tiles a partir de vários formatos geométricos, como OBJ, GeoJSON, IFC ou CityGML via bancos de dados 3dCityDB.
  • pg2b3dm - Conversão de geometrias 3D do PostGIS para tiles b3dm.
  • mago 3DTiler - Converte vários dados espaciais em OGC 3D Tiles.
  • PLATEAU GIS Converter - Converte modelos 3D de cidades (CityGML) para vários formatos geoespaciais, incluindo 3D Tiles, MVT e GeoPackage.
  • Obj2Tiles - Converte arquivos OBJ para OGC 3D Tiles.
  • 3dtiles - Converte arquivos Osgb (OpenSceneGraph Binary), Esri Shapefile e Fbx em 3D Tiles.
  • gltf-to-3d-tiles - Converte modelos glTF para Glb, b3dm ou formato 3D Tiles.
  • citygml-to-3dtiles - Conversão de CityGML para Cesium 3D Tiles.
  • vts23dtiles - Converte tilesets VTS em tilesets 3D Tiles.
  • 3D City Database - Banco de dados geoespacial para armazenar, representar e gerenciar modelos virtuais 3D de cidades.
  • Cesium Point Cloud Generator - XYZ → Tiles de nuvem de pontos 3D.
  • loaders.gl tile-converter - Converte I3S para 3D Tiles.
  • tyler - Tiling de modelos de cidades 3D codificados em CityJSON.

Ferramentas glTF

  • glTF-Transform - Suporta leitura, edição e escrita de modelos 3D em formato glTF 2.0.
  • Open Asset Import Library (assimp) - Biblioteca e conversor para mais de 40 formatos de arquivos 3D para glTF e outros (formatos suportados).
  • Exportador do Blender - Exportador glTF 2.0 do Blender.

Terreno

Quantized Mesh é um formato para codificar malhas de terreno para renderização eficiente de terreno no lado do cliente. Arquivos desse tipo são suportados no Cesium e deck.gl.

Veja awesome-quantized-mesh-tiles para mais recursos.
