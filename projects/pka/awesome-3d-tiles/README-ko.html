Awesome 3D Tiles

3D 타일을 위한 멋진 오픈 소스 소프트웨어, 라이브러리, 도구 및 리소스를 선별한 목록입니다.

기여를 환영합니다. pull requests를 통해 링크를 추가하거나, 논의를 시작하려면 issue를 생성하세요.

3D Tiles는 대규모 이기종 3D 지리공간 콘텐츠를 데스크탑, 웹, 모바일 애플리케이션 전반에 걸쳐 공유, 시각화, 융합 및 상호작용할 수 있도록 하는 오픈 규격입니다.

3D Tiles는 OGC 커뮤니티 표준입니다.

뷰어

  • CesiumJS - 웹 브라우저에서 3D 지구본과 2D 지도를 생성하는 JavaScript 라이브러리.
  • 3DCityDB-Web-Map-Client - CityGML 및 3D Tiles용 Cesium 기반 뷰어.
  • loaders.gl - 주요 3D, 지리공간, 테이블 형식 파서 및 인코더.
  • deck.gl - 대용량 데이터셋의 시각적 탐색적 데이터 분석을 위한 WebGL 기반 프레임워크.
  • 3d-tiles-renderer - Three.js 기반 3D Tiles 렌더러.
  • three-loader-3dtiles - OGC 3D Tiles 처리를 위한 Three.js 로더 모듈.
  • threedtiles - three.js용 3DTiles 뷰어.
  • mapbox-3dtiles - 3D Tiles용 Mapbox GL JS 커스텀 레이어.
  • A-Frame component - 3D-Tiles를 사용하는 A-Frame 컴포넌트.
  • iTowns - Three.js 기반 JS/WebGL 프레임워크.
  • giro3d - Three.js 기반 JS/WebGL 프레임워크(iTown의 후속작).
  • UD-Viz - 도시 데이터 시각화. 지리공간 3D 도시 데이터를 시각화하고 상호작용하는 웹 애플리케이션을 만들기 위한 프레임워크(iTowns/Tree.js 기반).
  • vsgCs - Vulkan Scene Graph(VSG) 애플리케이션 내에서 3D Tiles 및 기타 지리공간 콘텐츠 사용을 위한 라이브러리.

게임 엔진, 데스크탑 GIS 및 3D 모델링 툴 통합(오픈소스)

  • QGIS - 멀티플랫폼 데스크탑 지리정보 시스템(3.34 버전부터 지원).
  • 3D Tiles for Godot - Godot 4 엔진에 Cesium 3D Tiles 기능을 통합하는 Godot 4 확장.
  • Open 3D Engine (O3DE) - O3DE용 Cesium.
  • blender-3d-tiler - Blender 3D 모델 타일링 도구.

타일 생성(Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles 변환.
  • Cesium Native - 3D Tiles 스트리밍, glTF 처리용 C++ 라이브러리.
  • 3D Tiles Tools - 3D Tiles 데이터 변환, 최적화, 처리 및 분석용 도구 및 유틸리티.
  • glTF Pipeline - glTF / GLB 변환 및 최적화를 위한 자바스크립트 도구.

타일 생성(커뮤니티)

  • py3dtiles - API 또는 CLI를 통한 pnts 및 b3dm 읽기/쓰기.
  • Python 3DTiles Tilers - OBJ, GeoJSON, IFC, CityGML 등 다양한 기하 형식에서 3dCityDB 데이터베이스를 통해 3D Tiles 타일셋을 구축할 수 있는 Python 도구 및 라이브러리.
  • pg2b3dm - PostGIS 3D 지오메트리를 b3dm 타일로 변환.
  • mago 3DTiler - 다양한 공간 데이터를 OGC 3D Tiles로 변환하는 mago 3DTiler.
  • PLATEAU GIS Converter - 3D 도시 모델(CityGML)을 3D Tiles, MVT, GeoPackage 등 다양한 지리공간 형식으로 변환.
  • Obj2Tiles - OBJ 파일을 OGC 3D Tiles로 변환.
  • 3dtiles - Osgb(OpenSceneGraph Binary), Esri Shapefile, Fbx 파일을 3D Tiles로 변환.
  • gltf-to-3d-tiles - glTF 모델을 Glb, b3dm 또는 3D Tiles 형식으로 변환.
  • citygml-to-3dtiles - CityGML을 Cesium 3D Tiles로 변환.
  • vts23dtiles - VTS 타일셋을 3D Tileset으로 변환.
  • 3D City Database - 가상 3D 도시 모델을 저장, 표현, 관리하는 지리 데이터베이스.
  • Cesium Point Cloud Generator - XYZ → 3D 포인트 클라우드 타일.
  • loaders.gl tile-converter - I3S를 3D Tiles로 변환.
  • tyler - CityJSON으로 인코딩된 3D 도시 모델 타일링.

glTF 도구

  • glTF-Transform - glTF-Transform은 glTF 2.0 형식의 3D 모델을 읽기, 편집, 쓰기를 지원.
  • Open Asset Import Library (assimp) - 40개 이상의 3D 파일 포맷을 glTF 등으로 변환하는 라이브러리 및 변환기(지원 파일 포맷).
  • Blender exporter - Blender glTF 2.0 내보내기 도구.

지형

Quantized Mesh는 클라이언트 측에서 효율적인 지형 렌더링을 위해 지형 메시를 인코딩하는 형식입니다. 해당 파일은 Cesium 및 deck.gl에서 지원됩니다.

추가 자료는 awesome-quantized-mesh-tiles를 참고하세요.
