Awesome 3D Tiles

3D Tilesのための素晴らしいオープンソースソフトウェア、ライブラリ、ツール、リソースの厳選リスト。

貢献は歓迎します。プルリクエストでリンクを追加するか、issueを作成してディスカッションを始めてください。

3D Tilesは、デスクトップ、Web、モバイルアプリケーション間で膨大かつ異種の3D地理空間コンテンツを共有、可視化、融合、対話するためのオープンスペックです。

3D TilesはOGCコミュニティ標準です。

ビューア

  • CesiumJS - Webブラウザ上で3D地球儀や2D地図を作成するJavaScriptライブラリ。
  • 3DCityDB-Web-Map-Client - CityGMLおよび3D TilesのためのCesiumベースのビューア。
  • loaders.gl - 主要な3D、地理空間、表形式フォーマットのパーサーとエンコーダー。
  • deck.gl - 大規模データセットのWebGLによる視覚的探索データ分析フレームワーク。
  • 3d-tiles-renderer - Three.jsベースの3D Tilesレンダラー。
  • three-loader-3dtiles - Three.js用OGC 3D Tilesローダーモジュール。
  • threedtiles - three.js用3DTilesビューア。
  • mapbox-3dtiles - Mapbox GL JSカスタムレイヤーによる3D Tiles対応。
  • A-Frame component - A-Frameで3D-Tilesを利用するコンポーネント。
  • iTowns - Three.jsベースのJS/WebGLフレームワーク。
  • giro3d - Three.jsベースのJS/WebGLフレームワーク（iTownの後継）。
  • UD-Viz - アーバンデータの可視化。都市地理空間3Dデータの可視化・対話型Webアプリ作成用フレームワーク（iTowns/Tree.jsベース）。
  • vsgCs - Vulkan Scene Graph (VSG) アプリケーション内で3D Tilesや他の地理空間コンテンツを利用するためのライブラリ

ゲームエンジン・デスクトップGIS・3Dモデリングツール統合（オープンソース）

  • QGIS - マルチプラットフォームデスクトップ地理情報システム（バージョン3.34以降）。
  • 3D Tiles for Godot - GodotエンジンでCesium 3D Tiles機能を統合するGodot 4拡張
  • Open 3D Engine (O3DE) - O3DE向けCesium
  • blender-3d-tiler - Blender用3Dモデルタイル化ツール。

タイル生成（Cesium GS Inc）

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles。
  • Cesium Native - 3D TilesストリーミングやglTF処理のためのC++ライブラリ。
  • 3D Tiles Tools - 3D Tilesデータの変換、最適化、処理、解析用ツール群。
  • glTF Pipeline - glTF / GLBの変換・最適化用JavaScriptツール。

タイル生成（コミュニティ）

  • py3dtiles - APIやCLI経由でpntsやb3dmの読み書き。
  • Python 3DTiles Tilers - OBJ, GeoJSON, IFC, CityGML等の各種ジオメトリフォーマットを3dCityDB経由で3D Tilesタイルセット化するPythonツール・ライブラリ。
  • pg2b3dm - PostGISの3Dジオメトリをb3dmタイルへ変換。
  • mago 3DTiler - 各種空間データをOGC 3D Tilesに変換。
  • PLATEAU GIS Converter - 3D都市モデル（CityGML）を3D TilesやMVT、GeoPackage等の各種地理空間フォーマットへ変換。
  • Obj2Tiles - OBJファイルをOGC 3D Tilesに変換。
  • 3dtiles - Osgb(OpenSceneGraph Binary)、Esri Shapefile、Fbxファイルを3D Tilesに変換。
  • gltf-to-3d-tiles - glTFモデルをGlb、b3dmまたは3d tiles形式に変換。
  • citygml-to-3dtiles - CityGMLからCesium 3D Tilesへの変換。
  • vts23dtiles - VTSタイルセットを3D Tilesetに変換。
  • 3D City Database - 仮想3D都市モデルの格納・表示・管理用ジオデータベース。
  • Cesium Point Cloud Generator - XYZ → 3D点群タイル。
  • loaders.gl tile-converter - I3Sから3D Tilesへの変換。
  • tyler - CityJSONでエンコードされた3D都市モデルのタイル化。

glTFツール

  • glTF-Transform - glTF-TransformはglTF 2.0形式の3Dモデルの読み込み、編集、書き出しをサポート。
  • Open Asset Import Library (assimp) - 40種類以上の3Dファイル形式をglTF等へ変換するライブラリ・コンバータ（file formats）。
  • Blender exporter - BlenderのglTF 2.0エクスポーター。

地形

Quantized Meshは、効率的なクライアントサイド地形レンダリングのための地形メッシュエンコードフォーマットです。これらのファイルはCesiumやdeck.glでサポートされています。

詳細はawesome-quantized-mesh-tilesを参照してください。
