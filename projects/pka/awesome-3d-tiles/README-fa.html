3D Tiles شگفت‌انگیز

یک فهرست منتخب از نرم‌افزارهای متن‌باز، کتابخانه‌ها، ابزارها و منابع عالی برای 3D Tiles.

مشارکت خوش‌آمدید. لینک‌ها را از طریق pull requests اضافه کنید یا یک issue ایجاد کنید تا بحثی را آغاز نمایید.

3D Tiles یک مشخصات باز برای به‌اشتراک‌گذاری، بصری‌سازی، ترکیب و تعامل با محتوای عظیم و ناهمگون سه‌بعدی مکانی در برنامه‌های دسکتاپ، وب و موبایل است.

3D Tiles یک استاندارد جامعه OGC است.

بیننده‌ها

  • CesiumJS - کتابخانه جاوااسکریپت برای ایجاد کره‌های سه‌بعدی و نقشه‌های دوبعدی در مرورگر وب.
  • 3DCityDB-Web-Map-Client - بیننده مبتنی بر Cesium برای CityGML و 3D Tiles.
  • loaders.gl - تجزیه‌کننده‌ها و رمزگذارها برای بسیاری از فرمت‌های عمده سه‌بعدی، مکانی و جدولی.
  • deck.gl - چهارچوب مبتنی بر WebGL برای تحلیل داده‌های بزرگ به صورت بصری.
  • 3d-tiles-renderer - رندرکننده مبتنی بر Three.js برای 3D Tiles.
  • three-loader-3dtiles - ماژول بارگذار Three.js برای مدیریت OGC 3D Tiles.
  • threedtiles - بیننده 3DTiles برای three.js.
  • mapbox-3dtiles - لایه سفارشی Mapbox GL JS برای 3D Tiles.
  • A-Frame component - مولفه A-Frame با استفاده از 3D-Tiles.
  • iTowns - چهارچوب JS/WebGL مبتنی بر Three.js.
  • giro3d - چهارچوب JS/WebGL مبتنی بر Three.js (جانشین iTown).
  • UD-Viz - مصورسازی داده شهری. چهارچوبی برای ایجاد برنامه‌های وب جهت نمایش و تعامل با داده‌های سه‌بعدی شهری (مبتنی بر iTowns/Tree.js).
  • vsgCs - کتابخانه‌ای برای استفاده از 3D Tiles و محتوای مکانی دیگر در برنامه‌های Vulkan Scene Graph (VSG)

ادغام با موتور بازی، GIS دسکتاپ و ابزارهای مدل‌سازی سه‌بعدی (متن‌باز)

  • QGIS - سامانه اطلاعات جغرافیایی دسکتاپ چندسکویی (از نسخه 3.34).
  • 3D Tiles for Godot - افزونه Godot 4 که قابلیت‌های Cesium 3D Tiles را به موتور Godot اضافه می‌کند.
  • Open 3D Engine (O3DE) - Cesium برای O3DE
  • blender-3d-tiler - ابزار Blender برای تکه‌بندی مدل‌های سه‌بعدی.

ایجاد تایل (Cesium GS Inc)

  • Cesium Ion (سرویس SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - کتابخانه ++C برای پخش جریانی 3D Tiles، پردازش glTF.
  • 3D Tiles Tools - ابزارها و امکانات تبدیل، بهینه‌سازی، پردازش و تحلیل داده‌های 3D Tiles.
  • glTF Pipeline - ابزارهای جاوااسکریپت برای تبدیل و بهینه‌سازی glTF / GLB.

ایجاد تایل (جامعه)

  • py3dtiles - خواندن و نوشتن pnts و b3dm از طریق API یا CLI.
  • Python 3DTiles Tilers - ابزار و کتابخانه پایتون برای ساخت مجموعه‌تایل‌های 3D Tiles از فرمت‌های مختلف هندسی مانند OBJ، GeoJSON، IFC یا CityGML با استفاده از پایگاه داده‌های 3dCityDB.
  • pg2b3dm - تبدیل هندسه‌های سه‌بعدی PostGIS به تایل‌های b3dm.
  • mago 3DTiler - mago 3DTiler داده‌های مکانی مختلف را به OGC 3D Tiles تبدیل می‌کند.
  • PLATEAU GIS Converter - تبدیل مدل‌های شهری سه‌بعدی (CityGML) به فرمت‌های مختلف مکانی، شامل 3D Tiles، MVT و GeoPackage.
  • Obj2Tiles - تبدیل فایل‌های OBJ به OGC 3D tiles.
  • 3dtiles - تبدیل فایل‌های Osgb(OpenSceneGraph Binary)، Esri Shapefile و Fbx به 3D Tiles.
  • gltf-to-3d-tiles - تبدیل مدل glTF به فرمت Glb، b3dm یا 3d tiles.
  • citygml-to-3dtiles - تبدیل از CityGML به Cesium 3D Tiles.
  • vts23dtiles - تبدیل مجموعه‌تایل‌های VTS به 3D Tileset.
  • 3D City Database - پایگاه داده مکانی برای ذخیره، نمایش و مدیریت مدل‌های سه‌بعدی مجازی شهری.
  • Cesium Point Cloud Generator - XYZ → تایل‌های ابر نقطه‌ای سه‌بعدی.
  • loaders.gl tile-converter - تبدیل I3S به 3D Tiles.
  • tyler - تکه‌بندی مدل‌های سه‌بعدی شهری کدگذاری شده در CityJSON

ابزارهای glTF

  • glTF-Transform - glTF-Transform از خواندن، ویرایش و نوشتن مدل‌های سه‌بعدی در فرمت glTF 2.0 پشتیبانی می‌کند.
  • Open Asset Import Library (assimp) - کتابخانه و تبدیل‌کننده برای بیش از 40 فرمت فایل سه‌بعدی به glTF و سایر فرمت‌ها (فرمت‌های فایل).
  • Blender exporter - صادرکننده glTF 2.0 برای Blender.

زمین (Terrain)

Quantized Mesh فرمتی برای کدگذاری مش‌های زمین برای رندرینگ کارآمد سمت کاربر است. چنین فایل‌هایی در Cesium و deck.gl پشتیبانی می‌شوند.

برای منابع بیشتر به awesome-quantized-mesh-tiles مراجعه کنید.
