Awesome 3D Tiles

精选的开源软件、库、工具和3D Tiles相关资源列表。

欢迎贡献。如需添加链接，请通过 pull requests 或创建 issue 开始讨论。

3D Tiles 是一个开放规范，用于在桌面、网页和移动应用程序之间共享、可视化、融合和交互大规模异构三维地理空间内容。

3D Tiles 是 OGC 社区标准。

查看器

  • CesiumJS - 用于在网页浏览器中创建3D地球和2D地图的JavaScript库。
  • 3DCityDB-Web-Map-Client - 基于Cesium的CityGML和3D Tiles查看器。
  • loaders.gl - 支持多种主要3D、地理空间和表格格式的解析和编码器。
  • deck.gl - 基于WebGL的大型数据集可视探索分析框架。
  • 3d-tiles-renderer - 基于Three.js的3D Tiles渲染器。
  • three-loader-3dtiles - Three.js 的OGC 3D Tiles加载模块。
  • threedtiles - 基于three.js的3DTiles查看器。
  • mapbox-3dtiles - Mapbox GL JS的3D Tiles自定义图层。
  • A-Frame component - A-Frame 的3D Tiles组件。
  • iTowns - 基于Three.js的JS/WebGL框架。
  • giro3d - 基于Three.js的JS/WebGL框架（iTown的继任者）。
  • UD-Viz - 城市数据可视化。用于创建可视化和交互地理空间3D城市数据的Web应用框架（基于iTowns/Tree.js）。
  • vsgCs - 在Vulkan Scene Graph (VSG) 应用中使用3D Tiles和其他地理空间内容的库

游戏引擎、桌面GIS和3D建模工具集成（开源）

  • QGIS - 跨平台桌面地理信息系统（自3.34版本起支持）。
  • 3D Tiles for Godot - Godot 4扩展，将Cesium 3D Tiles功能集成到Godot引擎中
  • Open 3D Engine (O3DE) - O3DE的Cesium集成
  • blender-3d-tiler - Blender 3D模型切片工具。

瓦片生成（Cesium GS 公司）

  • Cesium Ion（SaaS）
  • CDB to 3D Tiles - OGC CDB → 3D Tiles。
  • Cesium Native - 用于3D Tiles流式传输和glTF处理的C++库。
  • 3D Tiles Tools - 3D Tiles数据的转换、优化、处理和分析工具。
  • glTF Pipeline - glTF / GLB转换与优化的JavaScript工具。

瓦片生成（社区）

  • py3dtiles - 通过API或CLI读取和写入pnts及b3dm。
  • Python 3DTiles Tilers - 用于从多种几何格式（如OBJ、GeoJSON、IFC或CityGML，通过3dCityDB数据库）构建3D Tiles瓦片集的Python工具和库。
  • pg2b3dm - 将PostGIS 3D几何体转换为b3dm瓦片。
  • mago 3DTiler - mago 3DTiler将各种空间数据转换为OGC 3D Tiles。
  • PLATEAU GIS Converter - 将3D城市模型（CityGML）转换为多种地理空间格式，包括3D Tiles、MVT和GeoPackage。
  • Obj2Tiles - 将OBJ文件转换为OGC 3D Tiles。
  • 3dtiles - 将Osgb（OpenSceneGraph Binary）、Esri Shapefile和Fbx文件转换为3D Tiles。
  • gltf-to-3d-tiles - 将glTF模型转换为Glb、b3dm或3d tiles格式。
  • citygml-to-3dtiles - 从CityGML转换为Cesium 3D Tiles。
  • vts23dtiles - 将VTS瓦片集转换为3D Tileset。
  • 3D City Database - 用于存储、表示和管理虚拟3D城市模型的地理数据库。
  • Cesium Point Cloud Generator - XYZ → 3D点云瓦片。
  • loaders.gl tile-converter - I3S转3D Tiles。
  • tyler - 对用CityJSON编码的3D城市模型进行切片

glTF工具

  • glTF-Transform - glTF-Transform支持读取、编辑和写入glTF 2.0格式的3D模型。
  • Open Asset Import Library (assimp) - 支持40多种3D文件格式转换为glTF及更多格式的库和转换器（文件格式）。
  • Blender导出器 - Blender的glTF 2.0导出器。

地形

Quantized Mesh 是一种用于高效客户端地形渲染的地形网格编码格式。Cesium和deck.gl均支持此格式。

更多资源请参见 awesome-quantized-mesh-tiles。
