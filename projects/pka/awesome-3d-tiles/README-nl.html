Awesome 3D Tiles

Een samengestelde lijst van geweldige open source software, bibliotheken, tools en bronnen voor 3D tiles.

Bijdragen zijn welkom. Voeg links toe via pull requests of maak een issue om een discussie te starten.

3D Tiles is een open specificatie voor het delen, visualiseren, combineren en interacteren met enorme heterogene 3D geospatiale inhoud over desktop-, web- en mobiele applicaties.

3D Tiles is een OGC community standard.

Viewers

  • CesiumJS - JavaScript-bibliotheek voor het maken van 3D-globes en 2D-kaarten in een webbrowser.
  • 3DCityDB-Web-Map-Client - Cesium gebaseerde Viewer voor CityGML en 3D Tiles.
  • loaders.gl - Parsers en encoders voor vele belangrijke 3D-, geospatiale en tabelvormige formaten.
  • deck.gl - WebGL-aangedreven framework voor visuele verkennende data-analyse van grote datasets.
  • 3d-tiles-renderer - Three.js gebaseerde renderer voor 3D Tiles.
  • three-loader-3dtiles - Three.js loader-module voor het verwerken van OGC 3D Tiles.
  • threedtiles - 3DTiles viewer voor three.js.
  • mapbox-3dtiles - Mapbox GL JS custom layer voor 3D Tiles.
  • A-Frame component - A-Frame component die 3D-Tiles gebruikt.
  • iTowns - Three.js gebaseerde JS/WebGL framework.
  • giro3d - Three.js gebaseerde JS/WebGL framework (opvolger van iTown).
  • UD-Viz - Urban Data Visualisatie. Framework voor het maken van webapplicaties voor het visualiseren en interacteren met geospatiale 3D stedelijke data (gebaseerd op iTowns/Three.js).
  • vsgCs - een bibliotheek voor het gebruiken van 3D Tiles en andere geospatiale inhoud binnen een Vulkan Scene Graph (VSG) applicatie

Game engine, desktop GIS en 3D-modelleringsintegraties (Open source)

  • QGIS - Multi-platform desktop geografisch informatiesysteem (sinds versie 3.34).
  • 3D Tiles for Godot - Godot 4-extensie die Cesium 3D Tiles mogelijkheden integreert in Godot Engine
  • Open 3D Engine (O3DE) - Cesium voor O3DE
  • blender-3d-tiler - Blender tool voor het tegelen van 3D-modellen.

Tile creatie (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles - OGC CDB → 3D Tiles.
  • Cesium Native - C++-bibliotheek voor 3D Tiles streaming, glTF-verwerking.
  • 3D Tiles Tools - Tools en hulpprogramma's voor het converteren, optimaliseren, verwerken en analyseren van 3D Tiles-data.
  • glTF Pipeline - Javascript-tools voor glTF / GLB-conversie en optimalisatie.

Tile creatie (Community)

  • py3dtiles - Lezen en schrijven van pnts en b3dm via API of CLI.
  • Python 3DTiles Tilers - Python-tool en -bibliotheek waarmee 3D Tiles tilesets uit diverse geometrische formaten kunnen worden gebouwd, zoals OBJ, GeoJSON, IFC of CityGML via 3dCityDB-databases.
  • pg2b3dm - Conversie van PostGIS 3D-geometrieën naar b3dm-tiles.
  • mago 3DTiler - mago 3DTiler converteert diverse ruimtelijke data naar OGC 3D Tiles.
  • PLATEAU GIS Converter - Converteert 3D stadsmodellen (CityGML) naar verschillende geospatiale formaten, inclusief 3D Tiles, MVT en GeoPackage.
  • Obj2Tiles - Converteert OBJ-bestanden naar OGC 3D tiles.
  • 3dtiles - Converteert Osgb (OpenSceneGraph Binary), Esri Shapefile en Fbx-bestanden naar 3D Tiles.
  • gltf-to-3d-tiles - Converteert glTF-model naar Glb, b3dm of 3d tiles-formaat.
  • citygml-to-3dtiles - Converteert van CityGML naar Cesium 3D Tiles.
  • vts23dtiles - Converteert VTS-tileset naar 3D Tileset.
  • 3D City Database - Geodatabank voor het opslaan, weergeven en beheren van virtuele 3D stadsmodellen.
  • Cesium Point Cloud Generator - XYZ → 3D Point Cloud Tiles.
  • loaders.gl tile-converter - Converteer I3S naar 3D Tiles.
  • tyler - Tegelen van 3D stadsmodellen gecodeerd in CityJSON

glTF-tools

  • glTF-Transform - glTF-Transform ondersteunt het lezen, bewerken en schrijven van 3D-modellen in glTF 2.0-formaat.
  • Open Asset Import Library (assimp) - Bibliotheek en converter voor 40+ 3D-bestandsformaten naar glTF en meer (bestandstypen).
  • Blender-exporteur - Blender glTF 2.0 Exporteur.

Terrein

Quantized Mesh is een formaat om terrein-meshes te coderen voor efficiënte client-side terreinweergave. Dergelijke bestanden worden ondersteund in Cesium en deck.gl.

Zie awesome-quantized-mesh-tiles voor meer bronnen.
