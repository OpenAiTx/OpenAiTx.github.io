Awesome 3D Tiles

Кураторский список отличного открытого программного обеспечения, библиотек, инструментов и ресурсов для 3D tiles.

Добро пожаловать к участию. Добавляйте ссылки через pull requests или создайте issue для начала обсуждения.

3D Tiles — это открытая спецификация для обмена, визуализации, объединения и взаимодействия с массивным неоднородным трехмерным геопространственным контентом на настольных, веб- и мобильных приложениях.

3D Tiles — это стандарт сообщества OGC.

Просмотрщики

  • CesiumJS — JavaScript библиотека для создания 3D глобусов и 2D карт в веб-браузере.
  • 3DCityDB-Web-Map-Client — Просмотрщик на основе Cesium для CityGML и 3D Tiles.
  • loaders.gl — Парсеры и кодеры для многих основных 3D, геопространственных и табличных форматов.
  • deck.gl — Фреймворк на WebGL для визуального анализа больших наборов данных.
  • 3d-tiles-renderer — Рендерер 3D Tiles на базе Three.js.
  • three-loader-3dtiles — Модуль загрузчика Three.js для работы с OGC 3D Tiles.
  • threedtiles — Просмотрщик 3DTiles для three.js.
  • mapbox-3dtiles — Кастомный слой Mapbox GL JS для 3D Tiles.
  • A-Frame component — Компонент A-Frame с использованием 3D-Tiles.
  • iTowns — JS/WebGL-фреймворк на основе Three.js.
  • giro3d — JS/WebGL-фреймворк на основе Three.js (преемник iTown).
  • UD-Viz — Визуализация городских данных. Фреймворк для создания веб-приложений для визуализации и взаимодействия с геопространственными 3D городскими данными (на основе iTowns/Three.js).
  • vsgCs — библиотека для использования 3D Tiles и другого геопространственного контента в приложениях Vulkan Scene Graph (VSG).

Интеграция с игровыми движками, настольными ГИС и 3D-моделирующими инструментами (Open source)

  • QGIS — Мультиплатформенная настольная геоинформационная система (с версии 3.34).
  • 3D Tiles for Godot — Расширение для Godot 4, интегрирующее возможности Cesium 3D Tiles в движок Godot.
  • Open 3D Engine (O3DE) — Cesium для O3DE.
  • blender-3d-tiler — Инструмент Blender для нарезки 3D моделей на тайлы.

Создание тайлов (Cesium GS Inc)

  • Cesium Ion (SaaS)
  • CDB to 3D Tiles — OGC CDB → 3D Tiles.
  • Cesium Native — C++ библиотека для стриминга 3D Tiles, обработки glTF.
  • 3D Tiles Tools — Инструменты для конвертации, оптимизации, обработки и анализа данных 3D Tiles.
  • glTF Pipeline — Javascript-инструменты для конвертации и оптимизации glTF / GLB.

Создание тайлов (Сообщество)

  • py3dtiles — Чтение и запись pnts и b3dm через API или CLI.
  • Python 3DTiles Tilers — Python-инструмент и библиотека для построения наборов 3D Tiles из различных геометрических форматов, например, OBJ, GeoJSON, IFC или CityGML через базы 3dCityDB.
  • pg2b3dm — Конвертация 3D-геометрий PostGIS в b3dm-тайлы.
  • mago 3DTiler — mago 3DTiler преобразует различные пространственные данные в OGC 3D Tiles.
  • PLATEAU GIS Converter — Конвертация 3D городских моделей (CityGML) в различные геопространственные форматы, включая 3D Tiles, MVT и GeoPackage.
  • Obj2Tiles — Преобразует OBJ-файлы в OGC 3D tiles.
  • 3dtiles — Конвертирует Osgb (OpenSceneGraph Binary), Esri Shapefile и Fbx-файлы в 3D Tiles.
  • gltf-to-3d-tiles — Конвертация glTF модели в Glb, b3dm или 3d tiles формат.
  • citygml-to-3dtiles — Конвертация из CityGML в Cesium 3D Tiles.
  • vts23dtiles — Конвертация набора тайлов VTS в 3D Tileset.
  • 3D City Database — Геобаза данных для хранения, отображения и управления виртуальными 3D городскими моделями.
  • Cesium Point Cloud Generator — XYZ → 3D Point Cloud Tiles.
  • loaders.gl tile-converter — Конвертация I3S в 3D Tiles.
  • tyler — Нарезка 3D городских моделей, закодированных в CityJSON.

Инструменты для glTF

  • glTF-Transform — Поддержка чтения, редактирования и записи 3D моделей в формате glTF 2.0.
  • Open Asset Import Library (assimp) — Библиотека и конвертер для 40+ 3D-форматов файлов в glTF и другие (форматы файлов).
  • Blender exporter — Экспортер Blender для glTF 2.0.

Рельеф

Quantized Mesh — это формат кодирования рельефных мешей для эффективного клиентского рендеринга рельефа. Такие файлы поддерживаются в Cesium и deck.gl.

Смотрите awesome-quantized-mesh-tiles для дополнительных ресурсов.
