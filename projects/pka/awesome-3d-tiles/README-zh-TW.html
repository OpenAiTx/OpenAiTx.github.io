Awesome 3D Tiles

精選的 3D Tiles 開源軟體、函式庫、工具及資源列表。

歡迎貢獻。請透過 pull requests 新增連結，或建立 issue 開始討論。

3D Tiles 是一項開放規範，用於在桌面、網頁及行動應用間，分享、視覺化、融合並互動處理大規模異質三維地理空間內容。

3D Tiles 是 OGC 社群標準。

檢視器

  • CesiumJS - 用於在網頁瀏覽器中建立 3D 地球儀與 2D 地圖的 JavaScript 函式庫。
  • 3DCityDB-Web-Map-Client - 基於 Cesium 的 CityGML 與 3D Tiles 檢視器。
  • loaders.gl - 支援多種主流 3D、地理空間及表格式格式的解析器與編碼器。
  • deck.gl - 基於 WebGL 的大型資料集探索式視覺分析框架。
  • 3d-tiles-renderer - 基於 Three.js 的 3D Tiles 渲染器。
  • three-loader-3dtiles - Three.js 的 OGC 3D Tiles 載入模組。
  • threedtiles - three.js 的 3DTiles 檢視器。
  • mapbox-3dtiles - Mapbox GL JS 3D Tiles 自定義圖層。
  • A-Frame component - A-Frame 的 3D-Tiles 元件。
  • iTowns - 基於 Three.js 的 JS/WebGL 框架。
  • giro3d - 基於 Three.js 的 JS/WebGL 框架（iTown 的後繼者）。
  • UD-Viz - 城市資料視覺化。基於 iTowns/Tree.js 的 3D 城市地理資料視覺化及互動應用框架。
  • vsgCs - 用於 Vulkan Scene Graph (VSG) 應用中載入 3D Tiles 及其他地理空間內容的函式庫。

遊戲引擎、桌面 GIS 與 3D 建模工具整合（開源）

  • QGIS - 跨平台桌面地理資訊系統（自 3.34 版起）。
  • 3D Tiles for Godot - Godot 4 擴充，整合 Cesium 3D Tiles 功能至 Godot 引擎。
  • Open 3D Engine (O3DE) - O3DE 的 Cesium 整合。
  • blender-3d-tiler - Blender 3D 模型分割工具。

瓦片產製（Cesium GS Inc）

  • Cesium Ion（SaaS）
  • CDB to 3D Tiles - OGC CDB → 3D Tiles。
  • Cesium Native - 3D Tiles 串流與 glTF 處理 C++ 函式庫。
  • 3D Tiles Tools - 3D Tiles 資料轉換、最佳化、處理與分析工具。
  • glTF Pipeline - glTF / GLB 轉換與最佳化 Javascript 工具。

瓦片產製（社群）

  • py3dtiles - 透過 API 或 CLI 讀寫 pnts 與 b3dm。
  • Python 3DTiles Tilers - Python 工具與函式庫，可將 OBJ、GeoJSON、IFC 或 CityGML 等多種格式，經由 3dCityDB 資料庫建構 3D Tiles tileset。
  • pg2b3dm - PostGIS 3D 幾何轉換為 b3dm 瓦片。
  • mago 3DTiler - mago 3DTiler 支援多種空間資料轉換為 OGC 3D Tiles。
  • PLATEAU GIS Converter - 將 3D 城市模型（CityGML）轉換為 3D Tiles、MVT、GeoPackage 等多種地理空間格式。
  • Obj2Tiles - 將 OBJ 檔轉換成 OGC 3D tiles。
  • 3dtiles - 將 Osgb（OpenSceneGraph Binary）、Esri Shapefile 及 Fbx 轉換為 3D Tiles。
  • gltf-to-3d-tiles - glTF 模型轉換為 Glb、b3dm 或 3d tiles 格式。
  • citygml-to-3dtiles - CityGML 轉 Cesium 3D Tiles。
  • vts23dtiles - VTS tileset 轉換為 3D Tileset。
  • 3D City Database - 儲存、表示與管理虛擬 3D 城市模型的地理資料庫。
  • Cesium Point Cloud Generator - XYZ → 3D 點雲瓦片。
  • loaders.gl tile-converter - I3S 轉 3D Tiles。
  • tyler - CityJSON 編碼 3D 城市模型分割為瓦片。

glTF 工具

  • glTF-Transform - glTF-Transform 支援 glTF 2.0 格式 3D 模型的讀取、編輯與寫入。
  • Open Asset Import Library (assimp) - 支援 40+ 種 3D 檔案格式轉換至 glTF 及其他格式的函式庫（檔案格式）。
  • Blender exporter - Blender glTF 2.0 匯出外掛。

地形

Quantized Mesh 是一種高效客戶端地形渲染的地形網格編碼格式。Cesium 與 deck.gl 支援該格式。

更多資源請參見 awesome-quantized-mesh-tiles。
