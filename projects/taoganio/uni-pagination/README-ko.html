Uni-Pagination 페이징 프레임워크

Uni Pagination은 Java 기반의 범용 페이징 프레임워크로, 추상화된 설계를 통해 다양한 데이터 소스에 대해 통일된 페이징 쿼리 인터페이스를 제공합니다. 프레임워크는 모듈화 아키텍처를 채택하여 현재 JDBC, Mybatis, MongoDB, Elasticsearch 등 여러 데이터 소스를 통합하고 유연한 확장 메커니즘을 제공합니다.

핵심 설계

프레임워크의 핵심 설계는 네 가지 주요 컴포넌트를 중심으로 전개됩니다:

  • 페이징 문장(PaginationStatement)
  • 모든 언어의 페이징 쿼리 문장 정의 지원
  • 통일된 페이징 파라미터 인터페이스 제공
  • 커스텀 페이징 로직 허용

  • 페이징 실행기(PaginationStatementExecutor)
  • 페이징 쿼리 문장 실행 담당
  • 페이징 정보 조회 기능 제공
  • 확장 가능한 실행기 메커니즘

  • 페이징 결과 집합(PaginationResultSet)
  • 통일된 결과 집합 인터페이스
  • 컬럼명 및 인덱스로 데이터 접근 지원
  • 타입 변환 기능 제공
  • JDBC ResultSet과 유사한 설계

  • 결과 집합 처리기(PaginationResultSetHandler)
  • 커스텀 결과 집합 변환 지원
  • 객체 매핑 기능 제공

  • 풍부한 결과 집합 처리 방식 제공

  • 모듈화 설계
  • 신규 데이터 소스 빠른 접속 지원
  • 확장 포인트 메커니즘 제공

  • 사용 편의성

  • 풍부한 유틸리티 클래스 지원

적용 시나리오

  • 다양한 데이터 소스 페이징을 지원해야 하는 엔터프라이즈 애플리케이션

핵심 모듈

uni-pagination

uni-pagination-jdbc

JDBC 데이터 소스 지원 모듈로, JDBC 기반 페이징 구현 제공.

uni-pagination-jdbc-mybatis

MyBatis 통합 모듈로, MyBatis 프레임워크와의 통합 지원 제공.

uni-pagination-mongodb

MongoDB 통합 컴포넌트로, MongoDB 페이징 쿼리 지원.

spring-boot-starter-web-uni-pagination

Spring MVC 환경에서의 페이징 지원 모듈로, Spring MVC와의 통합 지원을 제공하며 일반 컨트롤러 작성처럼 간단함.
  • 페이징 파라미터 자동 바인딩
  • 페이징 결과 자동 변환

web-uni-pagination-test

Spring Boot Web 환경에서의 페이징 테스트 모듈로, 페이징 기능 통합 테스트 제공.

간단한 예제

public class PaginationTest {
   
    public void page() {
        PaginationStatementTemplate operations = new PaginationStatementTemplate();
        operations.addStatementExecutor(new Sql2oPaginationNamedParameterStatementExecutor(dataSource));
        
        // 1. 创建分页语句
        PaginationStatement statement = of(b -> b
                .sql("SELECT * FROM user WHERE username LIKE :username")
                .paramMap("username", "%admin%")
                .pageable(0, 10)
                .sort("id", Sort.Direction.DESC));
        
        // 2. 执行查询
        PageInformation information = operations.queryForInformation(statement);
        Page page = operations.queryForResultSet(statement, new JdbcBeanPropertyPaginationRowMapper<>(User.class));
    }
    // User实体类
    class User {
        private Long id;
        private String username;
        // getters and setters
    }
}
