欢迎来到 BGPy！这是全球领先的 Python BGP 安全模拟器之一。你身处良好团队中；该模拟器已被全球众多团队使用，包括 NIST。BGPy 还是同行评审的研究，并已在多篇出版物中出现，且更多研究正在进行中。

使用 BGPy，你可以在笔记本电脑上模拟整个互联网拓扑。你可以编程实现自己的安全策略和攻击，使用拓扑进行自定义分析，使用真实的 ROV 数据等。BGPy 还配备了自定义测试套件，便于测试安全协议。我们支持超过 20 种策略，包括 ROV、ASPA 和 ASRA。

如果你想深入了解功能及使用方法，强烈推荐阅读教程。

如果你只想运行/可视化攻击/防御场景而不编程，推荐使用该工具的前端：BGPSimulator.com
