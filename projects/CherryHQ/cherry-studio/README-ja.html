🍒 Cherry Studio

Cherry Studioは、複数のLLMプロバイダーに対応したデスクトップクライアントで、Windows、Mac、Linuxで利用可能です。

👏 Telegramグループ｜Discord | QQグループ(575014769)に参加しよう

❤️ Cherry Studioが気に入ったら、スター🌟を付けるか、スポンサーで開発を応援してください！

🌠 スクリーンショット

🌟 主な機能

  • 多様なLLMプロバイダー対応:

  • ☁️ 主要なLLMクラウドサービス：OpenAI、Gemini、Anthropicなど
  • 🔗 AI Webサービス統合：Claude、Peplexity、Poeなど
  • 💻 Ollama、LM Studioによるローカルモデルサポート

  • AIアシスタント＆会話:

  • 📚 300以上のプリセットAIアシスタント
  • 🤖 カスタムアシスタント作成
  • 💬 複数モデルの同時会話

  • ドキュメント＆データ処理:

  • 📄 テキスト、画像、Office、PDFなどに対応
  • ☁️ WebDAVファイル管理とバックアップ
  • 📊 Mermaidチャートによる可視化
  • 💻 コード構文ハイライト

  • 実用的なツール統合:

  • 🔍 グローバル検索機能
  • 📝 トピック管理システム
  • 🔤 AI搭載翻訳
  • 🎯 ドラッグ＆ドロップ並べ替え
  • 🔌 ミニプログラム対応
  • ⚙️ MCP（モデルコンテキストプロトコル）サーバー

  • ユーザー体験の向上:

  • 🖥️ Windows、Mac、Linux対応のクロスプラットフォーム
  • 📦 すぐに使える、環境構築不要
  • 🎨 ライト／ダークテーマと透明ウィンドウ
  • 📝 完全なMarkdownレンダリング
  • 🤲 簡単なコンテンツ共有

📝 ロードマップ

現在、以下の機能と改善に積極的に取り組んでいます：

  • 🎯 コア機能

  • 選択アシスタント - スマートなコンテンツ選択強化
  • ディープリサーチ - 高度な調査機能

  • ドキュメント前処理 - 文書処理の改善

  • 🗂 ナレッジマネジメント

  • ダイナミックキャンバス可視化
  • OCR機能
  • TTS（テキスト読み上げ）対応

  • 📱 プラットフォーム対応

🔧 開発者共創プログラム

Cherry Studio開発者共創プログラムを開始し、オープンソースエコシステム内で健全なポジティブフィードバックループを促進します。素晴らしいソフトウェアは協力によって生まれ、マージされたプルリクエストごとにプロジェクトは新たな命を吹き込まれます。

心よりご参加いただき、私たちとCherry Studioの未来を共に創造してください。

コントリビューター報酬プログラム

コア貢献者に還元し好循環を生むため、以下の長期インセンティブプランを設けています。

本プログラムの最初の追跡期間は2025年第3四半期（7月、8月、9月）となります。このサイクルの報酬は10月1日に配布されます。 任意のトラッキング期間内（例：第1サイクルの場合は7月1日〜9月30日）、Cherry StudioのGitHub上のオープンソースプロジェクトに30件以上の意義あるコミットを行った開発者は、以下の特典を受け取る資格があります。

  • Cursorサブスクリプション支援：AIを最も効率的なコーディングパートナーにするため、Cursorサブスクリプションに対して70米ドルのクレジットまたは払い戻しが受けられます。
  • 無制限モデル利用：DeepSeekおよびQwenモデルのAPIコールが無制限に利用できます。
  • 最先端技術アクセス：Claude、Gemini、OpenAIなどのモデルAPIアクセスを含む特典を随時提供し、常に最先端の技術を体験できます。

共に成長する & 将来の計画

活気あるコミュニティは、持続可能なオープンソースプロジェクトの原動力です。Cherry Studioの成長とともに、リワードプログラムも進化していきます。業界最高水準のツールとリソースに特典を継続的に合わせることをお約束します。これにより、コア貢献者には実質的な支援が提供され、開発者・コミュニティ・プロジェクトが共に成長する正の循環が生まれます。

今後は、全オープンソースコミュニティに還元するため、よりオープンな姿勢を強化していきます。

🏢 エンタープライズエディション

コミュニティエディションを基盤に、Cherry Studio Enterprise Editionを新たにご紹介します。これは、現代のチームや企業向けに設計された、プライベートデプロイ可能なAI生産性・管理プラットフォームです。

エンタープライズエディションは、AIリソース・知識・データの管理を一元化し、チームコラボレーションの課題を解決します。組織が効率を高め、イノベーションを促進し、コンプライアンスを確保しつつ、100%データコントロールと安全性を維持できます。

コアの強み

  • 統合モデル管理：OpenAI、Anthropic、Google GeminiなどのクラウドLLMやローカル設置モデルを一元管理。従業員は個別設定なしで即利用可能。
  • エンタープライズ向けナレッジベース：チーム全体で知識ベースを構築・管理・共有。知識の継承と一貫性を保ち、正確な情報でAIと対話できます。
  • きめ細やかなアクセス制御：従業員アカウントを一元管理し、モデル・知識ベース・機能ごとに役割ベースの権限を割り当て可能。
  • 完全なプライベートデプロイ：バックエンド全体をオンプレミスサーバーやプライベートクラウドに導入し、データの100%プライバシーとコントロール、厳格なセキュリティ・コンプライアンス要件に対応。
  • 信頼性の高いバックエンドサービス：安定したAPIサービスやエンタープライズ向けのデータバックアップ・リカバリ機構でビジネス継続性を確保。
