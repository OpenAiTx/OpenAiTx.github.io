🍒 Cherry Studio

Cherry Studio 是一款支持多种 LLM 提供商的桌面客户端，适用于 Windows、Mac 和 Linux。

👏 加入 Telegram 群组｜Discord | QQ群(575014769)

❤️ 喜欢 Cherry Studio？请为它点个星 🌟 或 赞助 支持开发！

🌠 截图

🌟 主要特性

  • 多样化LLM服务商支持：

  • ☁️ 主流LLM云服务：OpenAI、Gemini、Anthropic等
  • 🔗 AI网络服务集成：Claude、Perplexity、Poe等
  • 💻 本地模型支持：Ollama、LM Studio

  • AI助手与对话：

  • 📚 300+预配置AI助手
  • 🤖 支持自定义助手创建
  • 💬 多模型同时对话

  • 文档与数据处理：

  • 📄 支持文本、图片、Office、PDF等多种格式
  • ☁️ WebDAV文件管理与备份
  • 📊 Mermaid图表可视化
  • 💻 代码语法高亮显示

  • 实用工具集成：

  • 🔍 全局搜索功能
  • 📝 主题管理系统
  • 🔤 AI驱动翻译
  • 🎯 拖拽排序
  • 🔌 小程序支持
  • ⚙️ MCP（模型上下文协议）服务器

  • 增强用户体验：

  • 🖥️ 跨平台支持：Windows、Mac、Linux
  • 📦 即装即用，无需环境配置
  • 🎨 支持浅色/深色主题及透明窗口
  • 📝 完整Markdown渲染
  • 🤲 便捷内容分享

🏢 企业版

在社区版基础上，我们隆重推出 Cherry Studio 企业版——面向现代团队与企业的私有化部署 AI 生产力及管理平台。

企业版聚焦团队协作核心难题，集成管理 AI 资源、知识与数据，帮助组织提升效率、激发创新，并确保合规性，实现数据 100% 私有可控与安全。

核心优势

  • 统一模型管理：集中集成和管理各类云端 LLM（如 OpenAI、Anthropic、Google Gemini）及本地私有部署模型，员工无需单独配置即可开箱即用。
  • 企业级知识库：构建、管理并共享团队知识库，保障知识留存与一致性，让团队成员基于统一、准确的信息与 AI 互动。
  • 细粒度权限管控：通过统一后台轻松管理员工账户，并为不同模型、知识库和功能分配基于角色的权限。
  • 全私有化部署：可将全部后端服务部署在本地服务器或私有云，确保数据 100% 私有、可控，满足最严苛的安全与合规要求。
  • 可靠后端服务：提供稳定 API 服务及企业级数据备份与恢复机制，保障业务连续性。
