🍒 Cherry Studio

Cherry Studio 是一款支持多种 LLM 服务商的桌面客户端，适用于 Windows、Mac 和 Linux。

👏 欢迎加入 Telegram 群｜Discord | QQ群(575014769)

❤️ 喜欢 Cherry Studio 吗？给我们点个星 🌟 或 赞助 支持开发！

🌠 截图

🌟 主要特性

  • 多样化大语言模型（LLM）服务商支持：

  • ☁️ 主流LLM云服务：OpenAI、Gemini、Anthropic等
  • 🔗 AI网络服务集成：Claude、Perplexity、Poe等
  • 💻 本地模型支持：Ollama、LM Studio

  • AI助手与对话：

  • 📚 300+ 预设AI助手
  • 🤖 自定义助手创建
  • 💬 多模型同时对话

  • 文档与数据处理：

  • 📄 支持文本、图片、Office、PDF等
  • ☁️ WebDAV文件管理与备份
  • 📊 Mermaid图表可视化
  • 💻 代码语法高亮

  • 实用工具集成：

  • 🔍 全局搜索功能
  • 📝 主题管理系统
  • 🔤 AI驱动翻译
  • 🎯 拖放排序
  • 🔌 小程序支持
  • ⚙️ MCP（模型上下文协议）服务器

  • 优化用户体验：

  • 🖥️ 跨平台支持Windows、Mac、Linux
  • 📦 即装即用 - 无需环境配置
  • 🎨 浅色/深色主题与透明窗口
  • 📝 完整Markdown渲染
  • 🤲 便捷内容分享

🏢 企业版

在社区版的基础上，我们自豪地推出Cherry Studio企业版——专为现代团队和企业打造的本地可部署AI生产力与管理平台。

企业版通过集中管理AI资源、知识与数据，解决团队协作的核心难题。它帮助组织提升效率、激发创新、确保合规，同时在安全环境下实现100%数据自主管控。

核心优势

  • 统一模型管理：集中集成和管理多种云端LLM（如OpenAI、Anthropic、Google Gemini）及本地私有模型。员工可即开即用，无需单独配置。
  • 企业级知识库：构建、管理并共享团队级知识库。保证知识留存与一致性，让团队成员基于统一、准确的信息与AI互动。
  • 细粒度权限控制：通过统一管理后台，轻松管理员工账号，并为不同模型、知识库和功能分配基于角色的权限。
  • 全私有化部署：可将全部后端服务部署于本地服务器或私有云，确保数据100%私有且受控，满足最严格的安全与合规要求。
  • 高可靠后端服务：提供稳定的API服务及企业级数据备份与恢复机制，保障业务连续性。
