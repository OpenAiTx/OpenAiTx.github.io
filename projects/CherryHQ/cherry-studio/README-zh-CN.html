🍒 Cherry Studio

Cherry Studio 是一款支持多种大语言模型（LLM）服务商的桌面客户端，可在 Windows、Mac 和 Linux 上使用。

👏 加入 Telegram 群组｜Discord | QQ群(575014769)

❤️ 喜欢 Cherry Studio？请为我们点个星🌟或赞助支持开发！

🌠 截图

🌟 主要特性

  • 多元 LLM 服务商支持：

  • ☁️ 主流 LLM 云服务：OpenAI、Gemini、Anthropic 等
  • 🔗 AI 网络服务集成：Claude、Peplexity、Poe 等
  • 💻 本地模型支持：Ollama、LM Studio

  • AI 助手与会话功能：

  • 📚 300+ 预设 AI 助手
  • 🤖 自定义助手创建
  • 💬 多模型同时对话

  • 文档与数据处理：

  • 📄 支持文本、图片、Office、PDF 等多种格式
  • ☁️ WebDAV 文件管理与备份
  • 📊 Mermaid 流程图可视化
  • 💻 代码语法高亮

  • 实用工具集成：

  • 🔍 全局搜索功能
  • 📝 主题管理系统
  • 🔤 AI驱动翻译
  • 🎯 拖拽排序
  • 🔌 小程序支持
  • ⚙️ MCP（模型上下文协议）服务器

  • 增强用户体验：

  • 🖥️ 跨平台支持 Windows、Mac 和 Linux
  • 📦 开箱即用，无需环境配置
  • 🎨 明亮/暗色主题与透明窗口
  • 📝 完整 Markdown 渲染
  • 🤲 轻松内容分享

🔧 开发者共创计划

我们正在启动 Cherry Studio 开发者共创计划，旨在促进开源生态内的健康正向反馈循环。我们相信卓越的软件源自协作，每一次合并的 PR 都为项目注入新的活力。

我们诚挚邀请你加入贡献者行列，与我们共同塑造 Cherry Studio 的未来。

贡献者奖励计划

为了回馈核心贡献者并形成良性循环，我们制定了如下长期激励计划。

本计划首个追踪周期为 2025 年第三季度（7月、8月、9月）。本周期奖励将于 10 月 1 日发放。 在任意一个统计周期内（如第一轮为7月1日至9月30日），任何开发者只要为 Cherry Studio 的任意 GitHub 开源项目贡献超过 30 次有意义的提交，即可获得以下福利：

  • Cursor 订阅赞助：获得一份 70 美元 的 Cursor 订阅抵扣或报销，让 AI 成为你最高效的编程伙伴。
  • 模型无限访问：可对 DeepSeek 和 Qwen 模型进行无限量 API 调用。
  • 前沿技术体验：不定期获得如 Claude、Gemini、OpenAI 等模型的 API 体验资格，始终站在技术前沿。

共同成长与未来规划

充满活力的社区是可持续开源项目的核心动力。随着 Cherry Studio 的发展，激励计划也将不断升级。我们承诺持续将福利与业界最优秀的工具和资源保持同步，确保核心贡献者获得有意义的支持，促进开发者、社区与项目的正向循环共同成长。

未来，项目也将持续拥抱开放，回馈整个开源社区。

🏢 企业版

基于社区版，我们自豪地推出 Cherry Studio 企业版——一款可私有化部署的 AI 生产力与管理平台，专为现代团队与企业设计。

企业版聚焦于解决团队协作中的核心难题，集中管理 AI 资源、知识与数据，帮助组织提升效率、激发创新、保障合规，并在安全环境下实现数据 100% 自主可控。

核心优势

  • 统一模型管理：集中集成和管理各类云端大模型（如 OpenAI、Anthropic、Google Gemini）和本地私有模型，员工无需单独配置即可开箱即用。
  • 企业级知识库：团队统一构建、管理与共享知识库，确保知识沉淀和一致性，助力团队成员基于精准信息与 AI 交互。
  • 细粒度权限控制：通过统一管理后台，轻松管理员工账户，并为不同模型、知识库和功能分配基于角色的权限。
  • 全私有化部署：支持将全部后端服务部署于自有服务器或私有云，确保数据 100% 私有并自主可控，满足最严苛的安全与合规需求。
  • 可靠的后端服务：提供稳定的 API 服务、企业级数据备份与恢复机制，确保业务连续性。
