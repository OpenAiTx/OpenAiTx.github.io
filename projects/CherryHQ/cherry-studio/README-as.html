আমিসকল আন্তৰিকভাৱে আপোনাক আমাৰ contributor-ৰ শাৰীত যোগ দিবলৈ আহ্বান জনাইছো আৰু Cherry Studio-ৰ ভৱিষ্যৎ গঠন কৰিবলৈ আমিসকলৰ সৈতে।

Contributor Rewards Program

আমাৰ মূল contributor-সকলক কৃতজ্ঞতা জনাব আৰু সদগুণৰ চক্ৰ সৃষ্টি কৰিবলৈ, আমি তলৰ দীঘলীয়া incentive plan স্থাপন কৰিছো।

এই প্ৰোগ্ৰামৰ প্ৰথম tracking period হ’ব Q3 2025 (জুলাই, আগষ্ট, ছেপ্টেম্বৰ)। এই চক্রৰ বাবে পুরস্কাৰসমূহ ১ অক্টোবৰ তাৰিখে বিতৰণ কৰা হ’ব। যিকোনো অনুসৰণ সময়ছোৱাত (উদাহৰণস্বৰূপ, প্ৰথম চক্রৰ বাবে জুলাই ১ তাৰিখৰ পৰা ছেপ্টেম্বৰ ৩০ তাৰিখলৈ), Cherry Studio-ৰ GitHub-ত থকা যিকোনো মুক্ত উৎস প্ৰকল্পত ৩০ টা অৰ্থপূৰ্ণ কমিটৰ অধিক অৱদান আগবঢ়োৱা যিকোনো ডেভেলপাৰে তলত উল্লেখ কৰা সুবিধাসমূহ পাবলৈ যোগ্য হ’ব:

  • Cursor Subscription Sponsorship: আপোনাৰ Cursor চাবস্ক্ৰিপচনলৈ $৭০ USD ক্ৰেডিট অথবা ৰিয়েম্বাৰ্ছমেন্ট লাভ কৰক, যাতে AI আপোনাৰ আটাইতকৈ দক্ষ কোডিং সহযোগী হৈ উঠে।
  • Unlimited Model Access: DeepSeek আৰু Qwen মডেলৰ বাবে অসীম API কল পাব।
  • Cutting-Edge Tech Access: সময়ে সময়ে বিশেষ সুবিধা লাভ কৰক, যেনে Claude, Gemini, আৰু OpenAI মডেলসমূহৰ API এক্সেছ, যাতে আপুনি প্ৰযুক্তিৰ আগভাগত থাকিব পাৰে।

একেলগে বৃদ্ধি আৰু ভৱিষ্যৎ পৰিকল্পনা

এক উজ্জ্বল সামুদায়িক পৰিৱেশ যিকোনো দীঘলীয়া মুক্ত উৎস প্ৰকল্পৰ মূল চালিকা শক্তি। Cherry Studio-ৰ বৃদ্ধি সৈতে আমাৰ ৰিওৱাৰ্ড প্ৰগ্ৰামো বৃদ্ধি পাব। আমি প্ৰতিশ্ৰুতিবদ্ধ যে, আমাৰ সুবিধাসমূহ সদায় ইণ্ডাষ্ট্ৰীৰ সৰ্বশ্ৰেষ্ঠ টুল আৰু ৰিছৰ্চসমূহৰ সৈতে সংহতি ৰাখি আগবঢ়াম। ইয়াৰ দ্বাৰা আমাৰ মূল অৱদানকাৰীসকলৰ প্ৰতি অৰ্থপূৰ্ণ সহায় নিশ্চিত হয়, যাৰ ফলত ডেভেলপাৰ, সামুদায়িক, আৰু প্ৰকল্পে একেলগে বৃদ্ধি পাব।

ভৱিষ্যতে, প্ৰকল্পটো অধিক মুক্ত নীতি গ্ৰহণ কৰিব যাতে সমগ্ৰ মুক্ত উৎস সামুদায়িকক সহায় আগবঢ়াব পৰা যায়।

Enterprise Edition-এ দলীয় সহযোগিতাৰ মূল সমস্যা সমাধান কৰে, AI ৰিছৰ্চ, জ্ঞান, আৰু ডাটাৰ ব্যৱস্থাপনা কেন্দ্ৰীয়কৰণ কৰি। এইয়ে সংস্থা সমূহক অধিক দক্ষতা, উদ্ভাৱন, আৰু নিয়ম মানি চলাৰ সুবিধা দিয়ে, লগতে ডাটা ১০০% নিয়ন্ত্ৰণ আৰু নিৰাপত্তা নিশ্চিত কৰে।

মূল সুবিধাসমূহ

  • একত্ৰিত মডেল মেনেজমেন্ট: কেন্দ্ৰীয়ভাৱে OpenAI, Anthropic, Google Gemini আদি ক্লাউড-ভিত্তিক LLM আৰু স্থানীয়ভাৱে ডিপ্লয় কৰা ব্যক্তিগত মডেলসমূহ সংহতি আৰু ব্যৱস্থাপনা কৰক। কৰ্মচাৰীয়ে পৃথক কনফিগাৰেশ্যন নকৰাকৈ ব্যৱহাৰ কৰিব পাৰে।
  • এণ্টাৰপ্ৰাইজ-গ্ৰেড জ্ঞানভঁৰাল: দল-ভিত্তিক জ্ঞানভঁৰাল গঢ়ি তুলক, পৰিচালনা কৰক আৰু ভাগ-বতৰা কৰক। ইয়াৰে জ্ঞান সংৰক্ষণ আৰু একৰূপতা নিশ্চিত হয়, যাতে দলৰ সদস্যই AI-ৰ সৈতে একত্ৰিত আৰু সঠিক তথ্যৰ ভিত্তিত যোগাযোগ কৰিব পাৰে।
  • সূক্ষ্ম এক্সেছ নিয়ন্ত্ৰণ: কৰ্মচাৰী একাউন্ট সহজে পৰিচালনা কৰক আৰু বিভিন্ন মডেল, জ্ঞানভঁৰাল, আৰু ফিচাৰসমূহৰ বাবে ৰোল-ভিত্তিক অনুমতি একত্ৰিত এডমিন বেকএণ্ডৰ জৰিয়তে নিযুক্ত কৰক।
  • সম্পূৰ্ণ ব্যক্তিগত ডিপ্লয়মেন্ট: সমগ্ৰ বেকএণ্ড সেৱা আপোনাৰ স্থানীয় ছাৰ্ভাৰ বা ব্যক্তিগত ক্লাউডত ডিপ্লয় কৰক, যাতে আপোনাৰ ডাটা ১০০% ব্যক্তিগত আৰু নিয়ন্ত্ৰণত থাকে, আৰু কঠোৰ সুৰক্ষা আৰু নিয়ম মানি চলাৰ মানদণ্ড পূৰণ হয়।
  • বিশ্বাসযোগ্য বেকএণ্ড সেৱাসমূহ: স্থিতিশীল API সেৱা আৰু এণ্টাৰপ্ৰাইজ-গ্ৰেড ডাটা বেকআপ আৰু পুনৰুদ্ধাৰ ব্যৱস্থা আগবঢ়াই, যাতে ব্যৱসায়িক ধাৰাবাহিকতা নিশ্চিত হয়।
