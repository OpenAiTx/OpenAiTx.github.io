🍒 Cherry Studio

Cherry Studio adalah klien desktop yang mendukung banyak penyedia LLM, tersedia di Windows, Mac, dan Linux.

👏 Bergabunglah dengan Grup Telegram｜Discord | Grup QQ (575014769)

❤️ Suka dengan Cherry Studio? Beri bintang 🌟 atau Sponsor untuk mendukung pengembangan!

🌠 Tangkapan Layar

🌟 Fitur Utama

  • Dukungan Beragam Penyedia LLM:

  • ☁️ Layanan Cloud LLM Utama: OpenAI, Gemini, Anthropic, dan lainnya
  • 🔗 Integrasi Layanan Web AI: Claude, Perplexity, Poe, dan lainnya
  • 💻 Dukungan Model Lokal dengan Ollama, LM Studio

  • Asisten AI & Percakapan:

  • 📚 300+ Asisten AI Pra-konfigurasi
  • 🤖 Pembuatan Asisten Kustom
  • 💬 Percakapan Multi-model Secara Simultan

  • Pemrosesan Dokumen & Data:

  • 📄 Mendukung Teks, Gambar, Office, PDF, dan lainnya
  • ☁️ Manajemen dan Backup File WebDAV
  • 📊 Visualisasi Grafik Mermaid
  • 💻 Penyorotan Sintaks Kode

  • Integrasi Alat Praktis:

  • 🔍 Fungsi Pencarian Global
  • 📝 Sistem Manajemen Topik
  • 🔤 Terjemahan Berbasis AI
  • 🎯 Pengurutan Seret dan Lepas
  • 🔌 Dukungan Mini Program
  • ⚙️ Server MCP (Model Context Protocol)

  • Pengalaman Pengguna yang Ditingkatkan:

  • 🖥️ Dukungan lintas platform untuk Windows, Mac, dan Linux
  • 📦 Siap Pakai - Tidak Perlu Setup Lingkungan
  • 🎨 Tema Terang/Gelap dan Jendela Transparan
  • 📝 Rendering Markdown Lengkap
  • 🤲 Berbagi Konten dengan Mudah

🔧 Program Kolaborasi Pengembang

Kami meluncurkan Program Kolaborasi Pengembang Cherry Studio untuk mendorong ekosistem open-source yang sehat dan penuh umpan balik positif. Kami percaya bahwa perangkat lunak hebat dibangun secara kolaboratif, dan setiap pull request yang diterima memberikan kehidupan baru bagi proyek ini.

Kami dengan tulus mengundang Anda untuk bergabung dengan jajaran kontributor kami dan membentuk masa depan Cherry Studio bersama kami.

Program Penghargaan Kontributor

Untuk memberikan apresiasi kepada kontributor inti dan menciptakan siklus yang positif, kami telah menetapkan rencana insentif jangka panjang berikut.

Periode pelacakan perdana untuk program ini adalah Q3 2025 (Juli, Agustus, September). Hadiah untuk siklus ini akan didistribusikan pada 1 Oktober.

Dalam periode pelacakan mana pun (misalnya, 1 Juli hingga 30 September untuk siklus pertama), setiap pengembang yang memberikan lebih dari 30 commit bermakna ke proyek open-source Cherry Studio di GitHub akan memenuhi syarat untuk manfaat berikut:

  • Sponsorship Langganan Cursor: Terima kredit atau penggantian sebesar $70 USD untuk langganan Cursor Anda, menjadikan AI sebagai mitra coding Anda yang paling efisien.
  • Akses Model Tanpa Batas: Dapatkan panggilan API tanpa batas untuk model DeepSeek dan Qwen.
  • Akses Teknologi Terkini: Nikmati manfaat sesekali, termasuk akses API ke model seperti Claude, Gemini, dan OpenAI, menjaga Anda di garis depan teknologi.

Bertumbuh Bersama & Rencana Masa Depan

Komunitas yang dinamis adalah kekuatan utama di balik proyek open-source yang berkelanjutan. Seiring Cherry Studio berkembang, program hadiah kami pun akan berkembang. Kami berkomitmen untuk terus menyelaraskan manfaat kami dengan alat dan sumber daya terbaik di industri. Hal ini memastikan kontributor inti kami menerima dukungan yang berarti, menciptakan siklus positif di mana pengembang, komunitas, dan proyek tumbuh bersama.

Ke depan, proyek ini juga akan mengadopsi sikap yang semakin terbuka untuk memberikan manfaat kepada seluruh komunitas open-source.

🏢 Enterprise Edition

Melanjutkan dari Community Edition, kami dengan bangga memperkenalkan Cherry Studio Enterprise Edition—platform produktivitas dan manajemen AI yang dapat diterapkan secara privat, dirancang untuk tim dan perusahaan modern.

Enterprise Edition menjawab tantangan inti dalam kolaborasi tim dengan memusatkan pengelolaan sumber daya AI, pengetahuan, dan data. Ini memberdayakan organisasi untuk meningkatkan efisiensi, mendorong inovasi, dan memastikan kepatuhan, sekaligus menjaga 100% kontrol atas data mereka dalam lingkungan yang aman.

Keunggulan Utama

  • Manajemen Model Terpadu: Integrasikan dan kelola berbagai LLM berbasis cloud (misal: OpenAI, Anthropic, Google Gemini) serta model privat yang diterapkan secara lokal. Karyawan dapat langsung menggunakannya tanpa konfigurasi individual.
  • Basis Pengetahuan Tingkat Perusahaan: Bangun, kelola, dan bagikan basis pengetahuan tim secara luas. Menjamin retensi dan konsistensi pengetahuan, memungkinkan anggota tim berinteraksi dengan AI berdasarkan informasi yang terpadu dan akurat.
  • Kontrol Akses Terperinci: Kelola akun karyawan dan tetapkan izin berbasis peran untuk berbagai model, basis pengetahuan, dan fitur melalui backend admin yang terpusat.
  • Deploy Sepenuhnya Privat: Terapkan seluruh layanan backend di server lokal Anda atau cloud privat, memastikan data Anda tetap 100% privat dan di bawah kendali Anda untuk memenuhi standar keamanan dan kepatuhan yang paling ketat.
  • Layanan Backend Andal: Menyediakan layanan API yang stabil serta mekanisme backup dan pemulihan data tingkat perusahaan guna memastikan kelangsungan bisnis.

🔗 Proyek Terkait

  • new-api: Sistem gateway LLM generasi berikutnya dan manajemen aset AI mendukung berbagai bahasa.

  • one-api: Sistem manajemen dan distribusi API LLM mendukung model utama seperti OpenAI, Azure, dan Anthropic. Memiliki antarmuka API terpadu, cocok untuk manajemen kunci dan distribusi sekunder.
