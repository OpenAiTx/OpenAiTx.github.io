🍒 Cherry Studio

Cherry Studio 是一款支持多種 LLM 供應商的桌面客戶端，適用於 Windows、Mac 和 Linux。

👏 歡迎加入 Telegram 群組｜Discord | QQ 群(575014769)

❤️ 喜歡 Cherry Studio 嗎？請給我們一顆星星 🌟 或 贊助 支持開發！

🌠 截圖預覽

🌟 核心特色

  • 多樣化 LLM 供應商支持：

  • ☁️ 主流 LLM 雲端服務：OpenAI、Gemini、Anthropic 等
  • 🔗 AI 網頁服務整合：Claude、Peplexity、Poe 及其他
  • 💻 本地模型支持：Ollama、LM Studio

  • AI 助手與對話功能：

  • 📚 300+ 預設 AI 助手
  • 🤖 自訂助手創建
  • 💬 多模型同時對話

  • 文件與數據處理：

  • 📄 支援文字、圖片、Office、PDF 等格式
  • ☁️ WebDAV 檔案管理與備份
  • 📊 Mermaid 圖表視覺化
  • 💻 程式碼語法高亮

  • 實用工具整合：

  • 🔍 全域搜尋功能
  • 📝 主題管理系統
  • 🔤 AI 驅動翻譯
  • 🎯 拖放排序
  • 🔌 小程序支援
  • ⚙️ MCP（模型上下文協議）伺服器

  • 強化使用者體驗：

  • 🖥️ 支援 Windows、Mac、Linux 跨平台
  • 📦 即裝即用，無需環境配置
  • 🎨 明亮/暗色主題與透明視窗
  • 📝 完整 Markdown 渲染
  • 🤲 內容輕鬆分享

為回饋核心貢獻者並建立良性循環，我們制定了以下長期激勵方案。

本計畫的首個追蹤週期為 2025 年第三季（7 月、8 月、9 月）。本周期獎勵將於 10 月 1 日發放。 在任何追蹤週期內（例如第一輪為 7 月 1 日至 9 月 30 日），任何對 Cherry Studio 在 GitHub 上的開源專案貢獻超過 30 次有意義提交 的開發者，皆有資格獲得以下福利：

  • Cursor 訂閱贊助：獲得 $70 美元 的 Cursor 訂閱抵用金或報銷，讓 AI 成為你最高效的編碼夥伴。
  • 模型無限存取：獲得 DeepSeek 和 Qwen 模型的 API 無限調用權限。
  • 前沿技術特權：享有如 Claude、Gemini、OpenAI 等模型的 API 使用權等不定期福利，讓你始終站在技術前沿。

🏢 企業版

在社群版的基礎上，我們自豪地推出 Cherry Studio 企業版——一個可私有部署的 AI 生產力與管理平台，專為現代團隊及企業設計。

企業版針對團隊協作中的核心挑戰，集中管理 AI 資源、知識與數據。讓企業提升效率、促進創新並確保合規，同時在安全環境下 100% 掌控自有數據。

核心優勢

  • 統一模型管理：集中整合與管理雲端 LLM（如 OpenAI、Anthropic、Google Gemini）及本地私有模型。員工開箱即用，無需個別配置。
  • 企業級知識庫：構建、管理並分享團隊級知識庫。確保知識傳承一致，團隊成員基於統一且正確資訊與 AI 互動。
  • 細緻權限管控：輕鬆管理員工帳號，透過統一後台為不同模型、知識庫和功能分配基於角色的權限。
  • 全私有部署：將整套後端服務部署於本地伺服器或私有雲，數據 100% 私有且可控，符合最嚴格安全與合規標準。
  • 高可用後端服務：提供穩定 API 服務、企業級數據備份與恢復機制，保障業務持續運行。
