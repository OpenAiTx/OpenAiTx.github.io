🍒 Cherry Studio

Cherry Studio 是一款支援多個 LLM 提供者的桌面客戶端，適用於 Windows、Mac 與 Linux。

👏 歡迎加入 Telegram 群組｜Discord | QQ 群(575014769)

❤️ 喜歡 Cherry Studio？點個星星 🌟 或 贊助 支持開發！

🌠 截圖

🌟 主要特點

  • 多元化 LLM 供應商支援：

  • ☁️ 主要 LLM 雲端服務：OpenAI、Gemini、Anthropic 等
  • 🔗 AI 網路服務整合：Claude、Perplexity、Poe 及其他
  • 💻 本地模型支援：Ollama、LM Studio

  • AI 助理與對話功能：

  • 📚 超過 300 款預先配置 AI 助理
  • 🤖 客製化助理創建
  • 💬 多模型同時對話

  • 文件與資料處理：

  • 📄 支援文字、圖片、Office、PDF 等
  • ☁️ WebDAV 檔案管理與備份
  • 📊 Mermaid 圖表視覺化
  • 💻 程式碼語法高亮

  • 實用工具整合：

  • 🔍 全域搜尋功能
  • 📝 主題管理系統
  • 🔤 AI 驅動翻譯
  • 🎯 拖曳排序功能
  • 🔌 小程式支援
  • ⚙️ MCP（模型上下文協定）伺服器

  • 強化使用者體驗：

  • 🖥️ 跨平台支援 Windows、Mac 與 Linux
  • 📦 開箱即用，無需環境配置
  • 🎨 明亮/深色主題及透明視窗
  • 📝 完整 Markdown 呈現
  • 🤲 內容輕鬆分享

🔧 開發者共創計畫

我們正在啟動 Cherry Studio 開發者共創計畫，促進開源生態系統中的健康正向反饋循環。我們相信優秀的軟體是協作產生的，每一次合併的 pull request 都為專案注入新生命。

誠摯邀請您加入我們的貢獻者行列，與我們共創 Cherry Studio 的未來。

貢獻者獎勵計畫

為了回饋核心貢獻者並建立良性循環，我們制定了以下長期激勵方案。

本計畫的首個追蹤週期將為 2025 年第三季（七月、八月、九月）。本週期的獎勵將於 10 月 1 日發放。

在任何一個追蹤週期內（例如，第一輪的 7 月 1 日至 9 月 30 日），任何開發者只要對 Cherry Studio 在 GitHub 上的任一開源專案貢獻超過 30 次有意義的提交，即可獲得以下福利：

  • Cursor 訂閱贊助：可獲得您的 Cursor 訂閱 70 美元的信用額度或報銷，讓 AI 成為您最高效的程式夥伴。
  • 模型無限存取：可獲得 DeepSeek 及 Qwen 模型的 API 無限調用權限。
  • 前沿技術體驗：不定期享有如 Claude、Gemini、OpenAI 等模型的 API 存取權限，助您站在技術最前線。

共同成長與未來規劃

充滿活力的社群是任何永續開源專案的驅動力。隨著 Cherry Studio 的成長，我們的獎勵計畫也將隨之擴大。我們承諾持續將福利與業界頂尖工具和資源接軌，確保核心貢獻者獲得有意義的支持，打造開發者、社群與專案共同成長的正向循環。

未來，專案也將擁抱更開放的態度，回饋整個開源社群。

🏢 企業版

在社群版的基礎上，我們自豪地推出 Cherry Studio 企業版——一個可私有部署的 AI 生產力與管理平台，專為現代團隊與企業打造。

企業版聚焦於團隊協作的核心挑戰，集中管理 AI 資源、知識與數據，協助組織提升效率、激發創新並確保合規，同時在安全環境下 100% 掌控您的資料。

核心優勢

  • 統一模型管理：集中整合並管理多種雲端 LLM（如 OpenAI、Anthropic、Google Gemini）及本地部署的私有模型。員工可直接使用，無需個人配置。
  • 企業級知識庫：建置、管理並分享全團隊知識庫。確保知識留存與一致性，讓團隊成員以統一且準確的資訊與 AI 互動。
  • 細緻權限管控：透過統一管理後台輕鬆管理員工帳號，並針對不同模型、知識庫與功能指派基於角色的權限。
  • 全私有部署：可將整套後端服務部署於您的本地伺服器或私有雲，確保資料 100% 私密且受您掌控，符合最嚴格的安全與合規標準。
  • 可靠後端服務：提供穩定的 API 服務及企業級資料備份與復原機制，確保業務持續運作。
