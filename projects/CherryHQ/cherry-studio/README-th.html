🍒 Cherry Studio

Cherry Studio คือโปรแกรมเดสก์ท็อปที่รองรับผู้ให้บริการ LLM หลากหลาย พร้อมใช้งานบน Windows, Mac และ Linux

👏 เข้าร่วม กลุ่ม Telegram｜Discord | กลุ่ม QQ(575014769)

❤️ ชอบ Cherry Studio ใช่ไหม? กดดาว 🌟 หรือ สนับสนุน เพื่อสนับสนุนการพัฒนา!

🌠 ภาพหน้าจอ

🌟 คุณสมบัติเด่น

  • รองรับผู้ให้บริการ LLM หลากหลาย:

  • ☁️ บริการ LLM คลาวด์หลัก: OpenAI, Gemini, Anthropic และอื่น ๆ
  • 🔗 การเชื่อมต่อบริการ AI บนเว็บ: Claude, Peplexity, Poe และอื่น ๆ
  • 💻 รองรับโมเดลโลคัลด้วย Ollama, LM Studio

  • ผู้ช่วย AI & การสนทนา:

  • 📚 ผู้ช่วย AI ที่ตั้งค่าไว้ล่วงหน้ามากกว่า 300 รายการ
  • 🤖 สร้างผู้ช่วยแบบกำหนดเอง
  • 💬 สนทนาแบบหลายโมเดลพร้อมกัน

  • ประมวลผลเอกสาร & ข้อมูล:

  • 📄 รองรับข้อความ, รูปภาพ, Office, PDF และอื่น ๆ
  • ☁️ จัดการไฟล์และสำรองข้อมูลผ่าน WebDAV
  • 📊 แสดงผลกราฟ Mermaid
  • 💻 ไฮไลต์โค้ดตามไวยากรณ์

  • การผสานรวมเครื่องมือใช้งานจริง:

  • 🔍 ฟังก์ชันค้นหาทั่วโลก
  • 📝 ระบบจัดการหัวข้อ
  • 🔤 การแปลภาษาด้วย AI
  • 🎯 การจัดลำดับด้วยการลากและวาง
  • 🔌 รองรับมินิโปรแกรม
  • ⚙️ เซิร์ฟเวอร์ MCP (Model Context Protocol)

  • ประสบการณ์ผู้ใช้ที่ดียิ่งขึ้น:

  • 🖥️ รองรับหลายแพลตฟอร์มทั้ง Windows, Mac และ Linux
  • 📦 พร้อมใช้งาน ไม่ต้องตั้งค่าสภาพแวดล้อม
  • 🎨 ธีมสว่าง/มืด และหน้าต่างโปร่งใส
  • 📝 การแสดงผล Markdown ครบถ้วน
  • 🤲 แชร์เนื้อหาได้ง่าย

📝 แผนงาน

เรากำลังดำเนินการพัฒนาและปรับปรุงฟีเจอร์ดังต่อไปนี้:

  • 🎯 ฟีเจอร์หลัก

  • Selection Assistant - เพิ่มประสิทธิภาพการเลือกเนื้อหาอย่างชาญฉลาด
  • Deep Research - ความสามารถในการวิจัยขั้นสูง
  • Memory System - การรับรู้บริบททั่วโลก
  • Document Preprocessing - ปรับปรุงการจัดการเอกสาร
  • MCP Marketplace - ระบบนิเวศ Model Context Protocol

  • 🗂 การจัดการความรู้

  • โน้ตและคอลเลกชัน
  • การแสดงผล Canvas แบบไดนามิก
  • ความสามารถ OCR
  • รองรับ TTS (Text-to-Speech)

  • 📱 รองรับแพลตฟอร์ม

  • HarmonyOS Edition (PC)
  • แอป Android (ระยะที่ 1)
  • แอป iOS (ระยะที่ 1)
  • รองรับหลายหน้าต่าง
  • ฟังก์ชันปักหมุดหน้าต่าง

  • 🔌 ฟีเจอร์ขั้นสูง

  • ระบบปลั๊กอิน
  • ASR (การรู้จำเสียงพูดอัตโนมัติ)
  • ปรับโครงสร้างการโต้ตอบผู้ช่วยและหัวข้อ

🤝 การมีส่วนร่วม

เรายินดีต้อนรับทุกการมีส่วนร่วมกับ Cherry Studio! คุณสามารถมีส่วนร่วมได้ดังนี้:

  • ร่วมพัฒนาโค้ด: สร้างฟีเจอร์ใหม่หรือปรับปรุงโค้ดเดิม
  • แก้ไขบั๊ก: ส่งฟิกซ์สำหรับข้อบกพร่องที่คุณพบ
  • ดูแล Issues: ช่วยจัดการปัญหาใน GitHub
  • ออกแบบผลิตภัณฑ์: ร่วมพูดคุยด้านการออกแบบ
  • เขียนคู่มือ: ปรับปรุงคู่มือและเอกสารแนะนำผู้ใช้
  • ร่วมกิจกรรมชุมชน: เข้าร่วมพูดคุยและช่วยเหลือผู้ใช้
  • ส่งเสริมการใช้งาน: เผยแพร่ Cherry Studio ให้เป็นที่รู้จัก

ดูแนวทางการมีส่วนร่วมได้ที่ Branching Strategy

เริ่มต้นใช้งาน

  • Fork Repository: Fork และ clone มายังเครื่องของคุณ
  • สร้าง Branch: สำหรับการแก้ไขของคุณ
  • ส่งการเปลี่ยนแปลง: Commit และ push การแก้ไขของคุณ
  • เปิด Pull Request: อธิบายการเปลี่ยนแปลงและเหตุผล

สำหรับแนวทางเพิ่มเติม โปรดดู Contributing Guide ของเรา

🔧 โปรแกรมร่วมสร้างโดยนักพัฒนา

เรากำลังเปิดตัวโปรแกรม Cherry Studio Developer Co-creation เพื่อส่งเสริมระบบนิเวศโอเพ่นซอร์สที่แข็งแรงและมีฟีดแบ็คเชิงบวก เราเชื่อว่าซอฟต์แวร์ที่ดีเกิดจากความร่วมมือ และทุก pull request ที่ถูกรวมจะช่วยให้โปรเจกต์มีชีวิตชีวามากขึ้น

ขอเชิญคุณเข้าร่วมเป็นผู้ร่วมพัฒนาและกำหนดอนาคตของ Cherry Studio ไปด้วยกัน

โปรแกรมรางวัลผู้ร่วมพัฒนา

เพื่อขอบคุณผู้ร่วมพัฒนาหลักและสร้างวงจรที่ดี เราได้กำหนดแผนจูงใจระยะยาวดังนี้

รอบติดตามครั้งแรกของโปรแกรมนี้คือไตรมาส 3 ปี 2025 (กรกฎาคม, สิงหาคม, กันยายน) รางวัลสำหรับรอบนี้จะจ่ายในวันที่ 1 ตุลาคม ในช่วงระยะเวลาการติดตามใด ๆ (เช่น 1 กรกฎาคม ถึง 30 กันยายน สำหรับรอบแรก) นักพัฒนาคนใดที่มีส่วนร่วม มากกว่า 30 คอมมิตที่มีความหมาย กับโปรเจกต์โอเพนซอร์สของ Cherry Studio บน GitHub จะมีสิทธิได้รับสิทธิประโยชน์ดังต่อไปนี้:

  • สปอนเซอร์ค่าสมัคร Cursor: รับเครดิตหรือการคืนเงิน $70 ดอลลาร์สหรัฐ สำหรับค่าสมัคร Cursor เพื่อให้ AI เป็นคู่หูเขียนโค้ดที่มีประสิทธิภาพสูงสุดของคุณ
  • เข้าถึงโมเดลไม่จำกัด: ได้รับสิทธิ API ไม่จำกัดสำหรับโมเดล DeepSeek และ Qwen
  • เข้าถึงเทคโนโลยีล้ำสมัย: รับสิทธิประโยชน์พิเศษเป็นครั้งคราว รวมถึงการเข้าถึง API ของโมเดลอย่าง Claude, Gemini และ OpenAI เพื่อให้คุณอยู่แถวหน้าของเทคโนโลยี

เติบโตไปด้วยกัน & แผนในอนาคต

ชุมชนที่มีชีวิตชีวาคือพลังขับเคลื่อนที่สำคัญของทุกโปรเจกต์โอเพนซอร์สที่ยั่งยืน เมื่อ Cherry Studio เติบโตขึ้น โปรแกรมรางวัลของเราก็จะเติบโตตามไปด้วย เรามุ่งมั่นที่จะปรับสิทธิประโยชน์ของเราให้สอดคล้องกับเครื่องมือและทรัพยากรที่ดีที่สุดในอุตสาหกรรมอยู่เสมอ เพื่อให้ผู้ร่วมพัฒนาหลักได้รับการสนับสนุนที่มีความหมาย เกิดวัฏจักรบวกที่นักพัฒนา ชุมชน และโปรเจกต์เติบโตไปด้วยกัน

ในอนาคต โปรเจกต์จะเปิดกว้างมากยิ่งขึ้น เพื่อคืนกลับให้กับชุมชนโอเพนซอร์สทั้งหมด

🏢 Enterprise Edition

ต่อยอดจาก Community Edition เราภูมิใจนำเสนอ Cherry Studio Enterprise Edition—แพลตฟอร์ม AI สำหรับการจัดการและเพิ่มประสิทธิภาพงาน ที่สามารถติดตั้งใช้งานภายในองค์กร เหมาะสำหรับทีมและองค์กรยุคใหม่

Enterprise Edition ตอบโจทย์ปัญหาหลักของการทำงานร่วมกันในทีม ด้วยการรวมศูนย์จัดการทรัพยากร AI ความรู้ และข้อมูล ช่วยให้องค์กรเพิ่มประสิทธิภาพ ส่งเสริมนวัตกรรม และรักษากฎระเบียบ โดยยังคงควบคุมข้อมูลได้ 100% ในสภาพแวดล้อมที่ปลอดภัย

ข้อได้เปรียบหลัก

  • การจัดการโมเดลรวมศูนย์: รวมศูนย์และจัดการ LLM บนคลาวด์หลากหลาย (เช่น OpenAI, Anthropic, Google Gemini) และโมเดลส่วนตัวที่ติดตั้งในเครื่อง พนักงานสามารถใช้งานได้ทันทีโดยไม่ต้องตั้งค่าทีละคน
  • ฐานความรู้ระดับองค์กร: สร้าง จัดการ และแชร์ฐานความรู้สำหรับทั้งทีม ให้ความรู้ถูกเก็บรักษาและมีความสม่ำเสมอ สมาชิกทีมสามารถโต้ตอบกับ AI ด้วยข้อมูลที่ถูกต้องและเป็นหนึ่งเดียว
  • ควบคุมสิทธิ์เข้าถึงละเอียด: จัดการบัญชีพนักงานและกำหนดสิทธิ์ตามบทบาทสำหรับแต่ละโมเดล ฐานความรู้ และฟีเจอร์ต่าง ๆ ได้ง่ายผ่านหลังบ้านเดียว
  • ติดตั้งแบบส่วนตัวเต็มรูปแบบ: ติดตั้งบริการหลังบ้านทั้งหมดบนเซิร์ฟเวอร์องค์กรหรือคลาวด์ส่วนตัว เพื่อให้ข้อมูลของคุณเป็นส่วนตัว 100% และควบคุมได้เองตามมาตรฐานความปลอดภัยและข้อบังคับสูงสุด
  • บริการหลังบ้านที่เชื่อถือได้: ให้บริการ API ที่เสถียร รองรับการสำรองและกู้คืนข้อมูลระดับองค์กร เพื่อความต่อเนื่องของธุรกิจ
