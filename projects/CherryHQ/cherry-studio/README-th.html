🍒 Cherry Studio

Cherry Studio เป็นโปรแกรมเดสก์ท็อปที่รองรับผู้ให้บริการ LLM หลายราย ใช้งานได้บน Windows, Mac และ Linux

👏 เข้าร่วม กลุ่ม Telegram｜Discord | กลุ่ม QQ(575014769)

❤️ ชอบ Cherry Studio ใช่ไหม? ให้ดาว 🌟 หรือ สนับสนุน เพื่อสนับสนุนการพัฒนา!

🌠 ภาพหน้าจอ

🌟 คุณสมบัติเด่น

  • รองรับผู้ให้บริการ LLM หลากหลาย:

  • ☁️ บริการ LLM คลาวด์หลัก: OpenAI, Gemini, Anthropic และอื่นๆ
  • 🔗 การเชื่อมต่อบริการ AI บนเว็บ: Claude, Peplexity, Poe และอื่นๆ
  • 💻 รองรับโมเดลโลคัลด้วย Ollama, LM Studio

  • ผู้ช่วย AI & การสนทนา:

  • 📚 มีผู้ช่วย AI ที่ตั้งค่าล่วงหน้ามากกว่า 300 รายการ
  • 🤖 สร้างผู้ช่วยแบบกำหนดเองได้
  • 💬 สนทนาแบบหลายโมเดลพร้อมกัน

  • ประมวลผลเอกสารและข้อมูล:

  • 📄 รองรับข้อความ รูปภาพ เอกสาร Office, PDF และอื่นๆ
  • ☁️ การจัดการไฟล์และสำรองข้อมูลด้วย WebDAV
  • 📊 สร้างภาพแผนภูมิ Mermaid
  • 💻 ไฮไลต์ไวยากรณ์โค้ด

  • การผสานเครื่องมือที่ใช้งานจริง:

  • 🔍 ฟังก์ชันค้นหาทั่วโลก
  • 📝 ระบบจัดการหัวข้อ
  • 🔤 การแปลภาษาโดยใช้ AI
  • 🎯 การจัดลำดับด้วยการลากและวาง
  • 🔌 รองรับมินิโปรแกรม
  • ⚙️ เซิร์ฟเวอร์ MCP (Model Context Protocol)

  • 🖥️ รองรับข้ามแพลตฟอร์มสำหรับ Windows, Mac, และ Linux
  • 📦 พร้อมใช้งานทันที - ไม่ต้องตั้งค่าสภาพแวดล้อม

📝 Roadmap

เรากำลังพัฒนาและปรับปรุงฟีเจอร์ต่อไปนี้อย่างต่อเนื่อง:

  • 🎯 ฟีเจอร์หลัก

  • ตัวช่วยเลือกเนื้อหาด้วยการเพิ่มประสิทธิภาพการเลือกอย่างชาญฉลาด

  • ความสามารถ OCR
  • รองรับ TTS (Text-to-Speech)

  • 📱 รองรับแพลตฟอร์ม

  • HarmonyOS Edition (PC)
  • แอป Android (เฟส 1)

🤝 การมีส่วนร่วม

เรายินดีต้อนรับทุกการมีส่วนร่วมกับ Cherry Studio! นี่คือวิธีที่คุณสามารถร่วมพัฒนา:

  • ร่วมพัฒนาซอร์สโค้ด: สร้างฟีเจอร์ใหม่หรือปรับปรุงโค้ดที่มีอยู่
  • แก้ไขบั๊ก: ส่งการแก้ไขสำหรับข้อผิดพลาดที่คุณพบ
  • ดูแล Issue: ช่วยจัดการกับ Issue บน GitHub
  • ออกแบบผลิตภัณฑ์: เข้าร่วมในการอภิปรายการออกแบบ
  • เขียนเอกสาร: ปรับปรุงคู่มือผู้ใช้และคำแนะนำ
  • มีส่วนร่วมกับชุมชน: เข้าร่วมพูดคุยและช่วยเหลือผู้ใช้
  • ส่งเสริมการใช้งาน: เผยแพร่ Cherry Studio ให้เป็นที่รู้จัก

ดู Branching Strategy สำหรับแนวทางการร่วมพัฒนา

เริ่มต้นใช้งาน

  • Fork Repository: Fork และ clone ไปยังเครื่องของคุณ
  • สร้าง Branch: สำหรับการเปลี่ยนแปลงของคุณ
  • ส่งการเปลี่ยนแปลง: Commit และ push การเปลี่ยนแปลงของคุณ
  • เปิด Pull Request: อธิบายการเปลี่ยนแปลงและเหตุผล

สำหรับแนวทางละเอียดเพิ่มเติม โปรดดู Contributing Guide ของเรา

🔧 โปรแกรมร่วมสร้างสรรค์สำหรับนักพัฒนา

เรากำลังเปิดตัวโปรแกรมร่วมสร้างสรรค์ Cherry Studio เพื่อเสริมสร้างระบบนิเวศโอเพ่นซอร์สที่แข็งแกร่งและมีฟีดแบ็กเชิงบวก เราเชื่อว่าซอฟต์แวร์ที่ยอดเยี่ยมเกิดจากการร่วมมือกัน และทุก Pull Request ที่ถูกรวมคือชีวิตใหม่ของโปรเจกต์

เราขอเชิญคุณเข้าร่วมเป็นหนึ่งในผู้ร่วมพัฒนากับเราและร่วมกำหนดอนาคตของ Cherry Studio

โปรแกรมรางวัลสำหรับผู้ร่วมพัฒนา

เพื่อขอบคุณผู้ร่วมพัฒนาหลักและสร้างวัฏจักรที่ดี เราจึงจัดตั้งแผนจูงใจระยะยาวดังนี้

ช่วงติดตามรอบแรกของโปรแกรมนี้จะเป็นไตรมาส 3 ปี 2025 (กรกฎาคม, สิงหาคม, กันยายน) รางวัลสำหรับรอบนี้จะมอบให้ในวันที่ 1 ตุลาคม ในช่วงระยะเวลาการติดตามใด ๆ (เช่น 1 กรกฎาคม ถึง 30 กันยายน สำหรับรอบแรก) นักพัฒนาคนใดที่มีส่วนร่วม มากกว่า 30 คอมมิทที่มีความหมาย ในโปรเจกต์โอเพนซอร์สของ Cherry Studio บน GitHub จะมีสิทธิ์ได้รับสิทธิประโยชน์ดังต่อไปนี้:

  • สนับสนุนค่าสมัคร Cursor: รับเครดิตหรือคืนเงินค่าสมัคร Cursor มูลค่า $70 USD เพื่อให้ AI กลายเป็นคู่หูเขียนโค้ดที่ทรงประสิทธิภาพที่สุดของคุณ
  • เข้าถึงโมเดลไม่จำกัด: ใช้งาน API ของโมเดล DeepSeek และ Qwen ได้ ไม่จำกัด
  • เข้าถึงเทคโนโลยีล้ำสมัย: รับสิทธิพิเศษเป็นระยะ เช่น การเข้าถึง API ของโมเดล Claude, Gemini และ OpenAI เพื่อให้คุณไม่ตกขบวนเทคโนโลยี

เติบโตไปด้วยกัน & แผนในอนาคต

ชุมชนที่มีชีวิตชีวาคือแรงขับเคลื่อนของโปรเจกต์โอเพนซอร์สที่ยั่งยืน เมื่อ Cherry Studio เติบโตขึ้น โปรแกรมรางวัลของเราก็จะขยายตาม เรายึดมั่นที่จะปรับสิทธิประโยชน์ให้สอดคล้องกับเครื่องมือและทรัพยากรระดับแนวหน้าของอุตสาหกรรมอยู่เสมอ เพื่อให้ผู้มีส่วนร่วมหลักได้รับการสนับสนุนที่มีความหมาย สร้างวัฏจักรบวกที่นักพัฒนา ชุมชน และโปรเจกต์เติบโตไปพร้อมกัน

ต่อจากนี้ โปรเจกต์จะเปิดกว้างมากขึ้นเพื่อคืนประโยชน์ให้กับชุมชนโอเพนซอร์สโดยรวม

🏢 Enterprise Edition

ต่อยอดจาก Community Edition เราขอแนะนำ Cherry Studio Enterprise Edition—แพลตฟอร์ม AI เพื่อการบริหารและเพิ่มประสิทธิภาพที่ปรับใช้ได้แบบส่วนตัว สำหรับทีมและองค์กรยุคใหม่

Enterprise Edition มุ่งแก้ไขปัญหาหลักในการทำงานร่วมกัน โดยรวบรวมการจัดการทรัพยากร AI ความรู้ และข้อมูลไว้ศูนย์กลาง ช่วยให้องค์กรเพิ่มประสิทธิภาพ กระตุ้นนวัตกรรม และมั่นใจในข้อกำหนดด้านความปลอดภัยและกฎระเบียบ โดยข้อมูลยังคงอยู่ในความควบคุม 100% ของคุณ

ข้อได้เปรียบหลัก

  • จัดการโมเดลแบบศูนย์กลาง: รวมและบริหาร LLM บนคลาวด์ (เช่น OpenAI, Anthropic, Google Gemini) และโมเดลส่วนตัวที่ติดตั้งในองค์กร พนักงานใช้งานได้ทันทีโดยไม่ต้องตั้งค่ารายบุคคล
  • คลังความรู้ระดับองค์กร: สร้าง จัดการ และแบ่งปันคลังความรู้ระดับทีม เพื่อเก็บรักษาและถ่ายทอดความรู้ให้ตรงกัน ให้สมาชิกทีมโต้ตอบกับ AI ด้วยข้อมูลที่ถูกต้องและเป็นหนึ่งเดียว
  • ควบคุมสิทธิ์การเข้าถึงละเอียด: จัดการบัญชีพนักงานและกำหนดสิทธิ์การใช้งานตามบทบาทสำหรับโมเดล คลังความรู้ และฟีเจอร์ต่าง ๆ ได้ผ่านแอดมินแบ็คเอนด์ศูนย์กลาง
  • ปรับใช้แบบส่วนตัวเต็มรูปแบบ: ติดตั้งบริการแบ็คเอนด์ทั้งหมดบนเซิร์ฟเวอร์องค์กรหรือคลาวด์ส่วนตัวของคุณเอง เพื่อให้ข้อมูลปลอดภัย 100% ภายใต้การควบคุมของคุณ และเป็นไปตามมาตรฐานความปลอดภัยสูงสุด
  • บริการแบ็คเอนด์ที่เชื่อถือได้: ให้บริการ API ที่เสถียร และระบบสำรอง-กู้คืนข้อมูลระดับองค์กร เพื่อให้ธุรกิจดำเนินต่อเนื่อง
