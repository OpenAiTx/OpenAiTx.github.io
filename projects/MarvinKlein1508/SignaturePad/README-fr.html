SignaturePad

Un composant Blazor simple à utiliser pour dessiner une signature. Il prend en charge les entrées à la souris et au toucher et fonctionne sur Blazor Server et Blazor WebAssembly.

Voir une démo en direct ici même sur GitHub.

Installation

Vous pouvez l’installer depuis Nuget en utilisant la commande suivante :

Install-Package Blazor.SignaturePad

Ou via le gestionnaire de paquets de Visual Studio.

Utilisation basique

Commencez par ajouter la directive using suivante à votre fichier racine _Imports.razor. 
@using SigPad

Ensuite, vous devez définir une propriété dans votre classe. Par exemple : 

public class MyInput
{
    public byte[] Signature { get; set; } = Array.Empty();
}

Vous pouvez ensuite l'utiliser où vous le souhaitez.

Le contrôle vous fournit les données d'image sous forme de byte[] en base64

Pour obtenir l'image, vous devrez convertir le byte[] en une chaîne de caractères. Par exemple :

public class MyInput
{
    public byte[] Signature { get; set; } = Array.Empty();
    public string SignatureAsBase64 => System.Text.Encoding.UTF8.GetString(Signature);
}

Fournir des options

Vous pouvez configurer le SignaturePad en fournissant une instance de SignaturePadOptions au composant. 

@code {
    public MyInput Input { get; set; } = new();
    private SignaturePadOptions _options = new SignaturePadOptions
    {
        LineCap = LineCap.Round,
        LineJoin = LineJoin.Round,
        LineWidth = 20
    };
}

Style personnalisé

Vous pouvez personnaliser l'apparence de SignaturePad en remplaçant les classes CSS ou en spécifiant vos propres classes.

Pour le SignaturePad lui-même, vous pouvez utiliser le paramètre Class et pour le bouton permettant d'effacer le SignaturePad, vous pouvez utiliser le paramètre ClearButtonClass.
