Matrix Morpheus GRUB 主题

红药丸 vs 蓝药丸

一个极简风格的矩阵灵感 GRUB 主题，特色是全屏动态背景，在 Linux 和 Windows 之间切换。

注意： 目前，主题仅包含 Arch Linux 和 Windows 图标。 此外，虽然图标在屏幕上是水平排列的， 但你仍然使用正常 GRUB 菜单的上和下箭头键进行导航。

安装

  • 克隆仓库

git clone https://github.com/Priyank-Adhav/Matrix-Morpheus-GRUB-Theme
  • 进入文件夹 

cd Matrix-Morpheus-GRUB-Theme

  • 使安装程序可执行

chmod +x install.sh

  • 以管理员身份执行安装脚本

sudo ./install.sh

  • 重启以测试您的新主题

--- 可选：简化您的 GRUB 菜单

我设计此主题时采用了两条目布局，尚未考虑如何视觉处理额外条目。

如果您的 GRUB 菜单当前有额外条目，如：

  • “Arch Linux 高级选项”
  • “UEFI 固件设置”
