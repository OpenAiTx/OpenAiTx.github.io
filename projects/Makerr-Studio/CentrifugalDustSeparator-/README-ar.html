فاصل الغبار بالطرد المركزي - MK2 - ملفات ثلاثية الأبعاد للطباعة (إصدار تجريبي)
ادعمني على: https://www.patreon.com/CapturingDust

قد تختلف مورّدات الأنابيب حسب موقعك الجغرافي، وكذلك أبعاد الأنابيب.
تم تعديل الملفات وفقًا لذلك، راجع ملفات README في إصدارات البناء المحددة لمعلومات الموردين:

*ملاحظة: اطبع أولاً حلقة القياس للتحقق من أبعاد الأنبوب لديك. شاهد: https://www.youtube.com/shorts/JZXLTLeB2oE
ملاحظة: بالنسبة للأجزاء غير المدرجة في المجلدات الفرعية (مثل قطع Y)، راجع إصدار MK2 العادي.

جميع الأجزاء مصممة للطباعة على حجم بناء بامبو لاب (WxDxH) 256 × 256 × 256 مم³.
فقط قطع Y غير متوافقة مع لوحة بناء Prusa.

YouTube: @CapturingDust
