CentrifugalDustSeparator

Центробежный пылеотделитель - MK2 - 3D-файлы для печати (бета-версия)
Поддержите меня на: https://www.patreon.com/CapturingDust

Поставщики труб могут различаться в зависимости от вашего географического положения, также как и размеры труб.
Файлы скорректированы соответственно, смотрите файлы README в конкретных версиях сборки для информации о поставщиках:

*Примечание: Сначала распечатайте кольцо для масштабирования, чтобы проверить размеры вашей трубы. Смотрите: https://www.youtube.com/shorts/JZXLTLeB2oE
Примечание: Для деталей, которые не указаны в подпапках (например, Y-образные элементы), смотрите обычную версию MK2.

Все детали спроектированы для печати на объеме сборки bambulab (ШxГxВ) 256 x 256 x 256 мм³.
Только Y-образные элементы не совместимы с платформой Prusa.
