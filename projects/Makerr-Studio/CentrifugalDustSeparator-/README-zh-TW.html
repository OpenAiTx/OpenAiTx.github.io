CentrifugalDustSeparator

離心集塵器 - MK2 - 3D列印檔案（Beta 版）
支持我：https://www.patreon.com/CapturingDust

根據您所在地區不同，管材供應商及管徑尺寸也會有所不同。
檔案已相應調整，請參閱各個版本資料夾中的 README 檔案以獲得供應商資訊：

*註：請先列印縮放環以檢查您的管材尺寸。參見：https://www.youtube.com/shorts/JZXLTLeB2oE
註：未在子資料夾列出的零件（如 Y 型接頭），請參見標準 MK2 版本。

所有零件皆設計於 bambulab 的列印體積（WxDxH）256 x 256 x 256 mm³ 內列印。
僅 Y 型接頭不相容於 Prusa 列印平台。
