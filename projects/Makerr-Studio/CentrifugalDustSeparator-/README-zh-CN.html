CentrifugalDustSeparator

离心除尘器 - MK2 - 3D 打印文件（测试版发布）
支持我：https://www.patreon.com/CapturingDust

根据您的地理位置不同，管道供应商可能会有所不同，管道尺寸也会有差异。
文件已相应调整，请查看特定构建版本的 README 文件以获取供应商信息：

*注意：请先打印标定环以检查您的管道尺寸。参考：https://www.youtube.com/shorts/JZXLTLeB2oE
注意：对于未在子文件夹中列出的零件（如 Y 型件），请参见普通 MK2 版本。

所有部件均设计为适配 bambulab 打印体积 (宽x深x高) 256 x 256 x 256 mm³。
只有 Y 型件不兼容 Prusa 打印平台。
