CentrifugalDustSeparator

เครื่องแยกฝุ่นแบบแรงเหวี่ยง - MK2 - ไฟล์ 3D สำหรับการพิมพ์ (รุ่นเบต้า)
สนับสนุนฉันได้ที่: https://www.patreon.com/CapturingDust

ผู้จัดจำหน่ายท่ออาจแตกต่างกันไปตามพื้นที่ของคุณ เช่นเดียวกับขนาดของท่อ
ไฟล์ต่างๆ ได้รับการปรับให้เหมาะสม โปรดดูไฟล์ README ในแต่ละเวอร์ชันสำหรับข้อมูลผู้จำหน่าย:

*หมายเหตุ: ให้พิมพ์แหวนสำหรับเทียบขนาดก่อนเพื่อเช็คขนาดท่อของคุณ ดูได้ที่: https://www.youtube.com/shorts/JZXLTLeB2oE
หมายเหตุ: สำหรับชิ้นส่วนที่ไม่ได้ระบุไว้ในโฟลเดอร์ย่อย (เช่น Y-pieces) ให้ดูในเวอร์ชัน MK2 ปกติ

ทุกชิ้นส่วนออกแบบมาให้พิมพ์กับขนาดพื้นที่ของ bambulab (กว้าง x ลึก x สูง) 256 x 256 x 256 มม.³
เฉพาะ Y-pieces เท่านั้นที่ไม่สามารถใช้กับแผ่นฐานของ Prusa ได้
