CentrifugalDustSeparator

Separador de Poeira Centrífugo - MK2 - Arquivos 3D para impressão (Versão Beta)
Apoie-me em: https://www.patreon.com/CapturingDust

Os fornecedores de tubos podem variar conforme sua localização geográfica, assim como as dimensões dos tubos.
Os arquivos são ajustados de acordo, veja os arquivos README nas versões específicas de construção para informações sobre fornecedores:

*nota: Imprima primeiro o anel de escala para verificar as dimensões do seu tubo. Veja: https://www.youtube.com/shorts/JZXLTLeB2oE
nota: Para peças que não estão listadas nas subpastas (como as peças Y), veja a versão normal MK2.

Todas as peças são projetadas para impressão no volume de construção da bambulab (LxPxA) 256 x 256 x 256 mm³.
Somente as peças Y não são compatíveis com a mesa de impressão da Prusa.
