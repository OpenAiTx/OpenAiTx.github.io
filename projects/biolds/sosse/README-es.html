Sosse 🦦

Descubre Sosse — el Motor de Búsqueda de Código Abierto basado en Selenium, creado para archivado web potente, rastreo y búsqueda. Explora todas sus características y capacidades en el sitio web oficial.

Ya seas desarrollador, investigador o entusiasta de datos, Sosse está listo para apoyar tus proyectos. Únete a la comunidad en GitHub o GitLab para enviar solicitudes de funciones, reportar errores, contribuir con código o iniciar una discusión.

Características clave

  • 🌍 Búsqueda de Páginas Web: Busca el contenido de páginas web, incluyendo las renderizadas dinámicamente, con consultas avanzadas.
(doc)

  • 🕑 Rastreo Recurrente: Rastrea páginas a intervalos fijos o adapta la frecuencia según los cambios en el contenido.
(doc)

  • 🔖 Archivado de Páginas Web: Archiva contenido HTML, ajusta enlaces para uso local, descarga los recursos necesarios y soporta
contenido dinámico. (doc)

  • 🏷️ Etiquetas: Organiza y filtra páginas rastreadas o archivadas usando etiquetas para mejorar la búsqueda y gestión.
(doc)
  • 📂 Descargas de Archivos: Descarga en lote archivos binarios desde páginas web.
(doc)

  • 📡 Webhooks: Integra con servicios externos usando webhooks altamente flexibles. Conéctate a plataformas de IA propietarias
(doc) o soluciones alojadas localmente (doc) para habilitar extracción avanzada de datos, resumen, auto-etiquetado, notificaciones y más.

  • 🔔 Feeds Atom: Genera feeds de contenido para sitios web que no los tienen, o recibe actualizaciones cuando se publica una página nueva
que contenga una palabra clave. (doc)

  • 🔒 Autenticación: El rastreador puede autenticarse para acceder a páginas privadas y recuperar contenido.
(doc)

  • 👥 Permisos: Los administradores pueden configurar rastreadores y ver estadísticas, mientras que los usuarios autenticados pueden buscar o hacerlo de forma anónima.
(doc)

  • 👤 Funciones de Búsqueda: Incluye historial de búsqueda privado (doc),
y accesos directos a motores de búsqueda externos (doc), etc.

Explora la 📚 documentación y echa un vistazo a algunas 📷 capturas de pantalla.

Sosse está escrito en Python y se distribuye bajo la licencia GNU AGPLv3. Usa rastreo basado en navegador con Mozilla Firefox o Google Chromium junto con Selenium para indexar páginas que dependen de JavaScript. Para un rastreo más rápido, también puede usarse Requests. Sosse utiliza PostgreSQL para almacenamiento de datos.

Pruébalo

Para probar rápidamente la última versión con Docker:

docker run -p 8005:80 biolds/sosse:stable

Luego, abra http://127.0.0.1:8005/ e inicie sesión con el nombre de usuario admin y la contraseña admin.

Para la persistencia de datos de Docker o métodos alternativos de instalación, consulte la guía de instalación.
