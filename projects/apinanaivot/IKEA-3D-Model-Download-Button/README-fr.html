Téléchargeur de Modèles 3D IKEA

Ce script Tampermonkey ajoute un bouton de téléchargement pour les modèles 3D sur les pages produit IKEA, vous permettant de sauvegarder facilement des fichiers .GLB des meubles et décorations IKEA. Il fonctionne sur les différentes versions linguistiques des sites IKEA et nomme automatiquement les fichiers téléchargés en fonction du nom du produit et de la couleur. Les fichiers peuvent être ouverts dans des logiciels 3D comme Blender. Cela vous permet d'essayer les meubles dans votre propre logiciel de planification 3D avant de prendre une décision d'achat.

MISE À JOUR : 28.11.2025 - Fonctionne sur le dernier site IKEA

Fonctionnalités

  • Ajoute un bouton "Télécharger 3D" à côté du bouton "Voir en 3D" sur les pages produit IKEA
  • Fonctionne sur toutes les versions linguistiques des sites IKEA
  • Nomme automatiquement les fichiers téléchargés avec le nom du produit et la couleur

Installation

  • Installez l’extension navigateur Tampermonkey pour votre navigateur.
  • Cliquez sur ce lien pour installer ou mettre à jour ce script, ou créez alternativement un nouveau script dans Tampermonkey et collez-y le contenu de ikea-3d-model-downloader.user.js.
  • Enregistrez le script et assurez-vous qu’il est activé dans Tampermonkey.

Utilisation

  • Rendez-vous sur n’importe quelle page produit IKEA qui possède un bouton "Voir en 3D".
  • Vous verrez un nouveau bouton "Télécharger 3D" à côté du bouton "Voir en 3D".
  • Cliquez sur le bouton "Télécharger 3D" pour télécharger le fichier GLB du modèle 3D.
  • Le fichier sera sauvegardé avec un nom au format : [Nom du Produit] - [Couleur].glb

Dépannage

  • Si le bouton de téléchargement n’apparaît pas, assurez-vous d’être sur une page produit avec un modèle 3D disponible et actualisez la page.
  • Si vous utilisez Chrome, essayez d’activer le mode développeur
  • Si rien ne fonctionne, créez un rapport de bug ici et je tenterai de le corriger au plus vite

Avertissement

Cet outil est conçu pour la planification et la visualisation personnelles à domicile. Respectez toujours les conditions d’utilisation d’IKEA et n’utilisez les modèles sauvegardés que pour des projets personnels de design d’intérieur.
