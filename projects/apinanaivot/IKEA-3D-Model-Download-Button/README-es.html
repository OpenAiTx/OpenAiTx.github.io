Descargador de Modelos 3D de IKEA

Este script de Tampermonkey añade un botón de descarga para modelos 3D en las páginas de productos de IKEA, permitiéndote guardar fácilmente archivos .GLB de muebles y decoraciones de IKEA. Funciona en diferentes versiones de idioma de los sitios web de IKEA y nombra automáticamente los archivos descargados según el nombre del producto y el color. Los archivos pueden abrirse en software 3D como Blender. Esto te permite probar los muebles en tu propio software de planificación de hogar 3D antes de tomar una decisión de compra.

ACTUALIZACIÓN: 28.11.2025 - Funciona en el último sitio web de IKEA

Características

  • Añade un botón "Descargar 3D" junto al botón "Ver en 3D" en las páginas de productos de IKEA
  • Funciona en todas las versiones de idioma de los sitios web de IKEA
  • Nombra automáticamente los archivos descargados usando el nombre del producto y el color

Instalación

  • Instala la extensión del navegador Tampermonkey para tu navegador.
  • Haz clic en este enlace para instalar o actualizar este script, o alternativamente crea un nuevo script en Tampermonkey y pega el contenido de ikea-3d-model-downloader.user.js en él.
  • Guarda el script y asegúrate de que esté habilitado en Tampermonkey.

Uso

  • Navega a cualquier página de producto de IKEA que tenga un botón "Ver en 3D".
  • Verás un nuevo botón "Descargar 3D" junto al botón "Ver en 3D".
  • Haz clic en el botón "Descargar 3D" para descargar el archivo GLB del modelo 3D.
  • El archivo se guardará con un nombre en el formato: [Nombre del Producto] - [Color].glb

Solución de Problemas

  • Si el botón de descarga no aparece, asegúrate de estar en una página de producto con un modelo 3D disponible y actualiza la página.
  • Si usas Chrome, intenta activar el modo desarrollador
  • Si nada funciona, crea un informe de error aquí y trataré de solucionarlo lo antes posible
