자세한 내용은 [온라인 문서]를 참고하세요

개발자용

본 프로젝트는 monorepo 구조를 사용하며, pnpm으로 관리되므로 먼저 pnpm을 설치해야 합니다: npm i -g pnpm

packages 디렉터리 내에는:

  • /core 도구 함수와 데이터 처리의 핵심을 포함

apps 디렉터리 내에는 최종 애플리케이션들이 위치

  • /web 는 웨이보 데이터를 최종적으로 볼 수 있는 웹페이지로, 현재 [vercel]에 호스팅 중
  • /monkey 는 유저스크립트(오일몽키 스크립트)로 패키징하는 용도
  • /cli Nodejs 명령어 버전

> vercel에 직접 배포하기 전에, vercel 프로젝트 설정에서 Root Directory를 apps/web-v2로 설정해야 프로젝트를 올바르게 인식하고 빌드할 수 있습니다.

server 디렉토리에는 go 언어로 작성된 로컬 서버가 있으며, 이미지 다운로드와 웨이보 데이터 탐색에 사용됩니다. 구체적인 사용법은 로컬 서버 사용 설명에서 확인할 수 있습니다.

감사의 말씀

  • 기본 아이디어를 제공해 주신 [speechless]께 감사드립니다.
  • [vite-plugin-monkey]는 유저스크립트로 패키징하는 도구를 제공합니다.
  • Shadcn/ui는 아름다운 UI 컴포넌트 라이브러리를 제공합니다.
  • [vercel]은 온라인 클라우드 플랫폼 배포를 지원합니다😚
  • Github Copilot & v0.dev에 감사드립니다😇
