qcc

一个用于质量控制图表和统计过程控制的R包。


qcc包提供了统计过程控制的质量控制工具：

  • Shewhart质量控制图，适用于连续型、属性型和计数数据。
  • Cusum和EWMA图表。
  • 操作特性曲线。
  • 过程能力分析。
  • 帕累托图和因果图。
  • 多变量控制图。

安装

您可以从CRAN安装发布版本的qcc：

`` r install.packages("qcc") 


或者从 GitHub 获取开发版本：
r

install.packages("devtools")

devtools::install_github("luca-scr/qcc", build = TRUE, build_opts = c("--no-resave-data", "--no-manual"))
