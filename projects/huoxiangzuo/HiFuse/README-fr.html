HiFuse

Ce dépôt est l’implémentation officielle de HiFuse : Réseau de Fusion Hiérarchique Multi-Échelle pour la Classification d’Images Médicales Auteurs : Xiangzuo Huo, Gang Sun, Shengwei Tian, Yan Wang, Long Yu, Jun Long, Wendong Zhang et Aolun Li. Exécution

  • Prérequis :
  • python3
  • pytorch 1.10
  • torchvision 0.11.1
  • Entraînement :
  • Préparez les images requises et rangez-les par catégories, configurez les dossiers d’images d’entraînement et de validation respectivement
  • Lancez python train.py
  • Reprendre l’entraînement :
  • Modifiez parser.add_argument('--RESUME', type=bool, default=True) dans python train.py
  • Lancez python train.py
  • Test :
  • Lancez python test.py

TensorBoard
