仕組み

Supabase はオープンソースツールの組み合わせです。私たちはエンタープライズグレードのオープンソースプロダクトを利用し、Firebase の機能を構築しています。MIT、Apache2、または同等のオープンライセンスで利用できるツールやコミュニティが存在する場合は、それらを活用し、サポートします。存在しない場合は、自ら開発しオープンソース化します。Supabase は Firebase の完全な 1 対 1 のマッピングではありません。私たちの目標は、開発者がオープンソースツールを使って Firebase のような開発体験を得られるようにすることです。

アーキテクチャ

Supabase はホスト型プラットフォームです。インストール不要でサインアップしてすぐに利用を開始できます。 また、セルフホスティングやローカル開発も可能です。

  • Postgres は30年以上にわたる活発な開発により、高い信頼性、豊富な機能、優れたパフォーマンスで評価されているオブジェクトリレーショナルデータベースシステムです。
  • Realtime は、Elixir サーバーで、PostgreSQL へのインサート、アップデート、デリートをウェブソケット経由で監視できます。Realtime は Postgres の組み込みレプリケーション機能で変更を検知し、JSON へ変換、認証済みクライアントにウェブソケットで配信します。
  • PostgREST は PostgreSQL データベースを直接 RESTful API に変換するウェブサーバーです。
  • GoTrue は JWT ベースの認証 API で、アプリケーションのユーザー登録、ログイン、セッション管理を簡単にします。
  • Storage は、S3 上のファイル管理用 RESTful API で、Postgres が権限管理を担当します。
  • pg_graphql は、GraphQL API を公開する PostgreSQL 拡張です。
  • postgres-meta は Postgres の管理用 RESTful API で、テーブル取得、ロール追加、クエリ実行などが可能です。
  • Kong はクラウドネイティブな API ゲートウェイです。
