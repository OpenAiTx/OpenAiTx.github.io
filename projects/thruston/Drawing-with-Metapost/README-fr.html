Dessiner-avec-Metapost

Toby Thurston -- 28 janv. 2025

Ce projet fournit un document qui explique comment dessiner des diagrammes techniques avec le langage Metapost de John Hobby. Il comprend plus de 200 illustrations créées avec Metapost, accompagnées du code source comme source d'inspiration et exemples. Le niveau visé est intermédiaire à avancé plutôt que débutant complet. Pour des introductions, tutoriels et autres articles sur Metapost, voir http://www.tug.org/metapost.html

Commencez par Dessiner-avec-Metapost.pdf dans le répertoire principal.

Le répertoire src contient

  • la source TeX du document principal
  • le fichier de style utilisé pour baliser le code source Metapost
  • le code source Metapost de chaque illustration utilisée dans le document principal
  • les fichiers PDF correspondants créés à partir de chaque fichier source MP

Le texte comprend une discussion des techniques utilisées pour dessiner les illustrations, et inclut des listes complètes ou partielles pour la plupart d’entre elles. Ces listes sont incluses directement depuis le code source utilisé, donc dans tous les cas la sortie doit correspondre exactement à ce que vous voyez sur la page.

Vous pouvez lire d'abord le document principal, mais vous pouvez aussi parcourir les PDF dans le répertoire src, et lorsque vous en trouvez un intéressant, jeter un œil au fichier source MP correspondant. Il y a une correspondance un à un entre les noms PDF et les noms source MP, ainsi geometry-apollonius.pdf est créé à partir de geometry-apollonius.mp. Le répertoire src contient quelques dessins qui ne sont pas inclus dans le document principal.

Si vous trouvez des erreurs, ou avez des suggestions d'amélioration, veuillez ouvrir un problème ou soumettre une demande de tirage sur Github.
