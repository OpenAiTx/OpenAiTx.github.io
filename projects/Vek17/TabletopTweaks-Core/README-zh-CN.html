TabletopTweaks-Core

⭐ 80 stars Simplified Chinese by Vek17

最新版本

﻿该模块为所有 TabletopTweaks 模组以及任何希望使用 TabletopTweaks 系统和工具的第三方模组提供共享核心系统。

官方 Tabletop Tweaks 模组

  • TabletopTweaks-Base。
  • 主要内容模组，基于桌面版开拓者，提供众多修正和新内容。
  • TabletopTweaks-Reworks。
  • 较小的辅助模组，提供我个人觉得猫头鹰工作室新创作中稍显不足的自制改动。
  • TabletopTweaks-Flanking。
  • 较小的辅助模组，提供桌面角色扮演游戏风格的侧击及相关机制。

第三方 Tabletop Tweaks 模组

  • Microscopic Content Expansion
  • 添加极少量新功能。可能会继续扩展范围，后者更为可能。
  • SparsSorcerousSundries
  • 该模块为《正义之怒》提供基于桌面版的修正和新内容。
  • Tome of the Firebrand
  • 包含法术、专长、其他能力及一些修正的合集。

安装方法

  • 下载并安装 Unity Mod Manager，确保版本不低于 0.23.0
  • 运行 Unity Mod Manager 并设置其识别《正义之怒》
  • 下载 Tabletop Tweaks 模组
  • 将第 3 步下载的 zip 文件拖入 Unity Mod Manager 窗口的 Mods 标签页中进行安装
