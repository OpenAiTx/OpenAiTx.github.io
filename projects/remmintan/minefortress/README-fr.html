MineFortress

Le mod qui transforme votre Minecraft en RTS. Prenez le rôle d’un chef de village, rassemblez des ressources, construisez des maisons, engagez des professionnels pour développer et étendre votre village et même conquérir vos voisins ! Oubliez la vue à la première personne et contrôlez tout avec votre curseur de souris comme dans les jeux RTS classiques.

Pour les joueurs

Ce dépôt est destiné aux développeurs. Si vous voulez jouer au mod, veuillez le télécharger depuis CurseForge ou Modrinth.

Le mod nécessite Fabric Loader et Fabric API pour fonctionner. Suivez les instructions d’installation détaillées dans notre wiki.

Pour les développeurs

Prérequis

Je suppose que tous ceux qui souhaitent contribuer au projet sont familiers avec le développement Java/Kotlin en général. Je ne détaillerai pas l’installation de la configuration de base pour développeur, mais je vais lister tous les outils nécessaires pour travailler sur le projet :
  • Java 17 - Je préfère utiliser les builds Adoptium
  • IntelliJ IDEA - Pour une expérience de développement la plus confortable et obtenir des résultats identiques aux miens, je recommande d’utiliser IntelliJ IDEA. Vous pouvez le télécharger ici.
  • Git - Pour cloner le dépôt et valider vos modifications. Vous pouvez le télécharger ici.

Si vous avez besoin d’aide pour configurer votre environnement de développement, veuillez lire les articles disponibles ici : Développement Java avec IntelliJ IDEA

N’hésitez pas à demander de l’aide sur notre serveur Discord.

Premiers pas / Installation

  • Clonez le dépôt avec Git.
  • Assurez-vous que le projet est cloné dans un répertoire nommé minefortress. Cela est nécessaire pour que Gradle fonctionne correctement avec IDEA.
  • Ouvrez le projet dans IntelliJ IDEA.
  • Attendez que Gradle ait fini d’importer le projet.
  • Nous devons encore faire fonctionner les configurations de lancement. Le plugin Fabric le fera automatiquement, mais nous devons fermer le projet.
  • Fermez le projet et ouvrez-le à nouveau.
  • De nouvelles configurations de lancement devraient apparaître. Si ce n’est pas le cas, essayez de redémarrer IDEA.
  • Une configuration de lancement peut avoir une petite croix à côté. Cela signifie que le plugin Fabric a utilisé le mauvais module lors de la configuration.
  • Pour corriger cela, éditez la configuration et sélectionnez dans le menu déroulant du classpath du module minefortress.main.
  • Vous pouvez maintenant exécuter le projet en utilisant la configuration Minecraft Client.

Contribution

Toute aide est grandement appréciée ! Si vous souhaitez contribuer au projet, veuillez suivre ces étapes :
  • Forkez le dépôt.
  • Créez une nouvelle branche à partir de la branche master.
  • Apportez vos modifications.
  • Committez vos modifications et poussez-les sur votre fork.
  • Créez une pull request vers la branche master de ce dépôt.

Il n’y a pas encore de guide de contribution, mais vous pouvez consulter les issues pour trouver quelque chose sur lequel travailler. J’essaierai de rassembler les bugs et demandes de fonctionnalités dans les issues, mais si vous avez des questions, n’hésitez pas à les poser sur notre serveur Discord.

N’hésitez pas également à créer des issues si vous trouvez des bugs ou avez des demandes de fonctionnalités.

Avec le temps, j’essaierai de créer un guide de contribution et une liste d’issues adaptées aux débutants.
