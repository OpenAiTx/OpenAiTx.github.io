MineFortress

这个模组将你的Minecraft变成RTS。扮演村庄领袖的角色，收集资源，建造房屋，雇佣专业人士发展和扩展你的村庄，甚至征服邻居！忘记第一人称视角，像经典RTS游戏一样用鼠标指针控制一切。

给开发者

前提条件

我假设所有想为项目贡献代码的人对Java/Kotlin开发有基本了解。 我不会详细介绍基本开发环境的安装，但会列出所有需要的工具：
  • Java 17 - 我更喜欢使用Adoptium的版本
  • IntelliJ IDEA - 为了获得最便捷的开发体验并确保结果与我一致，推荐使用IntelliJ IDEA。你可以从这里下载。
  • Git - 用于克隆仓库和提交更改。你可以从这里下载。

如果你需要开发环境配置支持，请阅读这里的文章：使用IntelliJ IDEA进行Java开发

欢迎在我们的Discord服务器寻求帮助。

入门 / 安装

  • 使用Git克隆仓库。
  • 确保项目被克隆到名为minefortress的目录中。这是让Gradle在IDEA中正常工作的前提。
  • 在IntelliJ IDEA中打开项目。
  • 等待Gradle完成项目导入。
  • 我们还需要让运行配置生效。Fabric插件会自动完成，但我们需要关闭项目。
  • 关闭项目然后重新打开。
  • 新的运行配置应该会出现。如果没有，尝试重启IDEA。
  • 运行配置旁可能会有一个小叉号，表示Fabric插件在设置配置时使用了错误的模块。
  • 解决方法是编辑配置，选择minefortress.main模块的类路径下拉菜单。
  • 现在你可以使用 Minecraft Client 配置运行项目。
