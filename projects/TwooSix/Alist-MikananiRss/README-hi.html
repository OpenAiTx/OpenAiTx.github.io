Alist-MikananiRss

मिकन योजना या अन्य एनीमे श्रृंखला संबंधित RSS सब्सक्रिप्शन स्रोतों से अपने आप एनीमे अपडेट प्राप्त करें और Alist के माध्यम से ऑफ़लाइन डाउनलोड करके संबंधित क्लाउड ड्राइव में सेव करें।

और ChatGPT का उपयोग करके संसाधन नामों का विश्लेषण करें, ताकि संसाधनों का नाम Emby द्वारा解析 योग्य प्रारूप में बदल सकें।

---

उपयोग दस्तावेज़

फ़ीचर्स

  • स्वचालित रूप से एनीमे अपडेट प्राप्त करें और संबंधित क्लाउड डिस्क में डाउनलोड करें
  • PushPlus, Telegram आदि चैनलों के माध्यम से अपडेट सूचना भेजें
  • स्वचालित रूप से emby द्वारा पहचाने जाने योग्य प्रारूप में पुनः नामकरण करें, साथ ही स्वचालित पार्सिंग के परिणामों का कस्टम रीमैपिंग भी समर्थित है, जिससे नामकरण और अधिक सटीक हो सके

तैयारी कार्य

  • कृपया स्वयं Alist प्रोजेक्ट दस्तावेज़ के अनुसार Alist को डिप्लॉय करें (संस्करण ≥3.42.0 होना चाहिए), और Aria2/qBittorrent ऑफलाइन डाउनलोड सेटअप करें
  • स्वयं मिकान योजना खाता रजिस्टर करें, एनीमे सब्सक्राइब करें, और सब्सक्रिप्शन लिंक प्राप्त करें

संलग्न: अन्य RSS सब्सक्रिप्शन स्रोतों के लिए भी कुछ अनुकूलन किया गया है, सैद्धांतिक रूप से अधिकांश सब्सक्रिप्शन स्रोतों (एनीमे संबंधित) का समर्थन करता है, असमर्थित RSS के लिए, कृपया issue दर्ज करें

उपयोग कैसे करें

Docker, सोर्स कोड रनिंग आदि के लिए अधिक चलाने के तरीके विस्तार से उपयोग दस्तावेज़ में देखें

pip से इंस्टॉल और रन करें

  • कृपया सुनिश्चित करें कि आपका python संस्करण 3.11 से ऊपर है
  • pip से इंस्टॉल करें: pip install alist-mikananirss
  • निर्देशिका में एक नया config.yaml कॉन्फ़िगरेशन फ़ाइल बनाएं, और निम्नानुसार कॉन्फ़िगरेशन फ़ाइल भरें (पूर्ण फ़ीचर उदाहरण विवरण के लिए देखें कॉन्फ़िगरेशन निर्देश)
``yaml common: interval_time: 300 alist: base_url: https://example.com # 修改为你的alist访问地址 token: alist-xxx # 修改为你的alist token；可在"管理员后台->设置->其他"中找到 downloader: qBittorrent # 或者 aria2 download_path: Onedrive/Anime # 修改为你的下载路径(Alist中的路径)

mikan: subscribe_url:

  • https://mikanani.me/RSS/MyBangumi?token=xxx # 修改为你的蜜柑订阅地址
# - https://mikanani.me/RSS/MyBangumi?token=xxx2 # 多条RSS订阅链接情况 filters:
  • 非合集 # 程序暂不支持合集等形式的重命名，若使用重命名功能推荐使用此过滤器
`
  • कोड चलाएँ: python -m alist_mikananirss --config /path/to/config.yaml`
