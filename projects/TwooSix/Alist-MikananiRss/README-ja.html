Alist-MikananiRss

蜜柑計画やその他のアニメ番組関連のRSSフィードから自動で最新話を取得し、Alist経由で対応するクラウドストレージにオフラインダウンロードします

さらにChatGPTによるリソース名解析を組み合わせ、Embyで認識可能な形式にリネームします。

使用ドキュメント

機能

  • アニメの更新を自動取得し、対応するクラウドストレージにダウンロード
  • PushPlus、Telegramなどのチャネルを通じて更新通知を送信
  • embyが認識可能な形式に自動リネーム、さらに自動解析結果のカスタムリマッピングもサポートし、リネーム結果の精度向上

準備作業

  • Alistプロジェクトドキュメントを参照してAlistをデプロイしてください（バージョンは>=3.42.0必要）、Aria2/qBittorrentのオフラインダウンロードも構築
  • 自分でMikan Projectのアカウントを登録し、アニメを購読、購読リンクを取得

付記：他のRSS購読元にも一定の適応を行っており、理論上ほとんどの購読元（アニメ関連）をサポートしています。未対応RSSについてもissue提出歓迎です。

使用方法

Dockerやソースコード実行など、その他の実行方法は使用ドキュメントをご参照ください。

pipでインストールして実行

  • Pythonのバージョンが3.11以上であることを確認してください
  • pipでインストール: pip install alist-mikananirss
  • ディレクトリ内にconfig.yaml設定ファイルを新規作成し、以下のように設定ファイルを記入（完全な機能例と詳細は設定説明参照）
``yaml common: interval_time: 300 alist: base_url: https://example.com # 修改为你的alist访问地址 token: alist-xxx # 修改为你的alist token；可在"管理员后台->设置->其他"中找到 downloader: qBittorrent # 或者 aria2 download_path: Onedrive/Anime # 修改为你的下载路径(Alist中的路径)

mikan: subscribe_url:

  • https://mikanani.me/RSS/MyBangumi?token=xxx # 修改为你的蜜柑订阅地址
# - https://mikanani.me/RSS/MyBangumi?token=xxx2 # 多条RSS订阅链接情况 filters:
  • 非合集 # 程序暂不支持合集等形式的重命名，若使用重命名功能推荐使用此过滤器
`
  • コードを実行：python -m alist_mikananirss --config /path/to/config.yaml`
  • お楽しみください
