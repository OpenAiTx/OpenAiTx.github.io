Alist-MikananiRss

Automatyczne pobieranie aktualizacji anime z Mikan Project lub innych RSS powiązanych z anime oraz ich offline'owe pobieranie na odpowiedni dysk sieciowy przez Alist

W połączeniu z analizą nazw zasobów przez ChatGPT, pliki są automatycznie zmieniane na format rozpoznawalny przez Emby.

---

Dokumentacja użytkownika

Funkcje

  • Automatyczne pobieranie aktualizacji anime i zapisywanie ich na odpowiednim dysku sieciowym
  • Wysyłanie powiadomień o aktualizacjach przez PushPlus, Telegram i inne kanały
  • Automatyczne zmienianie nazw plików na format rozpoznawalny przez emby, z możliwością własnego mapowania wyników, co zwiększa dokładność nazw

Przygotowanie

  • Samodzielnie wdroż Alist według dokumentacji projektu Alist (wymagana wersja >=3.42.0) oraz skonfiguruj Aria2/qBittorrent do pobierania offline
  • Zarejestruj konto na stronie Mikan Project, subskrybuj anime i uzyskaj link do subskrypcji

Uwaga: Skrypt został częściowo dostosowany także do innych źródeł RSS, teoretycznie obsługuje większość źródeł subskrypcji (związanych z anime), jeśli jakieś RSS nie są obsługiwane, zgłoś issue

Jak używać

Więcej metod uruchamiania jak Docker, uruchamianie ze źródeł itp. opisano w dokumentacji użytkownika

Instalacja i uruchomienie przez pip

  • Upewnij się, że używasz Pythona w wersji 3.11 lub wyższej
  • Zainstaluj pakiet przez pip: pip install alist-mikananirss
  • Utwórz w katalogu plik config.yaml i wypełnij go zgodnie z poniższym przykładem (pełny opis funkcji i konfiguracji znajdziesz w instrukcji konfiguracji)
``yaml common: interval_time: 300 alist: base_url: https://example.com # 修改为你的alist访问地址 token: alist-xxx # 修改为你的alist token；可在"管理员后台->设置->其他"中找到 downloader: qBittorrent # 或者 aria2 download_path: Onedrive/Anime # 修改为你的下载路径(Alist中的路径)

mikan: subscribe_url:

  • https://mikanani.me/RSS/MyBangumi?token=xxx # 修改为你的蜜柑订阅地址
# - https://mikanani.me/RSS/MyBangumi?token=xxx2 # 多条RSS订阅链接情况 filters:
  • 非合集 # 程序暂不支持合集等形式的重命名，若使用重命名功能推荐使用此过滤器
`
  • Uruchom kod: python -m alist_mikananirss --config /path/to/config.yaml`
