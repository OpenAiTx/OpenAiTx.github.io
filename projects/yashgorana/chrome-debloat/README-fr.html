Chrome Debloat

Un outil pour générer des politiques pour les navigateurs basés sur Chromium (Chrome, Brave et Edge) qui désactivent les fonctionnalités inutiles, la télémétrie et les logiciels superflus tout en activant certaines améliorations de qualité de vie.

Fonctionnalités

  • Tente de désactiver la télémétrie et le rapport d’utilisation
  • Supprime les fonctionnalités inutiles et les logiciels superflus préinstallés
  • Bloque le contenu promotionnel et les éléments d’interface utilisateur inutiles
  • Maintient la fonctionnalité du navigateur tout en réduisant l’utilisation des ressources
  • Préconfigure les extensions essentielles :
  • uBlock Origin
  • Cookie AutoDelete
  • Don't f* with paste
  • I still don't care about cookies
  • SponsorBlock
  • BlockTube
  • BlankTab
  • Decentraleyes

Démarrage rapide

Windows

  • Téléchargez le fichier .reg pour votre navigateur depuis generated/windows/.
  • Ouvrez le fichier .reg téléchargé pour ajouter les paramètres au Registre Windows.
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy) et cliquez sur « Recharger les politiques ».

macOS

  • Téléchargez le fichier .mobileconfig pour votre navigateur depuis generated/macos/.
  • Ouvrez le fichier .mobileconfig téléchargé pour lancer l’installation du profil.
  • Allez dans Réglages Système > Confidentialité et sécurité > Profils et approuvez le nouveau profil.
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy) et cliquez sur « Recharger les politiques ».

Linux

  • Téléchargez le fichier .json pour votre navigateur depuis generated/linux/.
  • Déplacez le fichier téléchargé dans le répertoire de politique approprié (créez-le si nécessaire) :
  • Chrome : /etc/opt/chrome/policies/managed/chrome.json
  • Edge : /etc/opt/edge/policies/managed/edge.json
  • Brave : /etc/brave/policies/managed/brave.json
Remarque : Vous pourriez avoir besoin des droits sudo pour cela.*
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy) et cliquez sur "Recharger les politiques".

Configuration personnalisée

Si vous souhaitez personnaliser les politiques :

  • Clonez ce dépôt
  • Installez les dépendances :
``bash uv sync `
  • Modifiez policies.yaml selon vos besoins
  • Générez de nouveaux fichiers de configuration :
`bash uv run main.py `
  • Trouvez les fichiers générés dans le répertoire generated/

Désinstallation des politiques

Windows :

  • Naviguez vers le répertoire uninstall/windows/ dans ce dépôt.
  • Exécutez le fichier .reg correspondant à votre navigateur (par exemple, uninstall_chrome.reg). Cela supprimera les clés de registre ajoutées lors de l'installation.
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy) et cliquez sur "Reload policies".

macOS :
  • Allez dans Réglages Système > Confidentialité et sécurité > Profils.
  • Sélectionnez le profil associé à votre navigateur (par exemple, "Chrome Debloat Policies").
  • Cliquez sur le bouton '-' (moins) pour supprimer le profil.
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy) et cliquez sur "Reload policies".

Linux :
  • Supprimez le fichier JSON de politique dans le répertoire spécifique au navigateur (vous pourriez avoir besoin des droits sudo) :
  • Chrome : sudo rm /etc/opt/chrome/policies/managed/chrome.json
  • Edge : sudo rm /etc/opt/edge/policies/managed/edge.json
  • Brave : sudo rm /etc/brave/policies/managed/brave.json
  • Redémarrez votre navigateur ou allez sur chrome://policy (ou edge://policy, brave://policy`) et cliquez sur "Reload policies".
