Chrome Debloat

Chromiumベースのブラウザ（Chrome、Brave、Edge）向けに、不要な機能、テレメトリー、不要なソフトウェアを無効化し、一部の利便性向上機能を有効化するポリシーを生成するツール。

Features

  • テレメトリーと使用状況レポートの無効化を試みる
  • 不要な機能とプリインストールされた不要なソフトウェアを削除
  • プロモーションコンテンツと不要なUI要素をブロック
  • ブラウザの機能を維持しつつリソース使用量を削減
  • 必須拡張機能を事前設定：
  • uBlock Origin
  • Cookie AutoDelete
  • Don't f* with paste
  • I still don't care about cookies
  • SponsorBlock
  • BlockTube
  • BlankTab
  • Decentraleyes

Supported Browsers

Quick Start

Windows

  • ブラウザ用の .reg ファイルを generated/windows/ からダウンロードします。
  • ダウンロードした .reg ファイルを開き、Windowsレジストリに設定を追加します。
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy）にアクセスして「ポリシーを再読み込み」をクリックします。

macOS

  • ブラウザ用の .mobileconfig ファイルを generated/macos/ からダウンロードします。
  • ダウンロードした .mobileconfig ファイルを開き、プロファイルのインストールを開始します。
  • 「システム設定」>「プライバシーとセキュリティ」>「プロファイル」に移動し、新しいプロファイルを承認します。
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy）にアクセスして「ポリシーを再読み込み」をクリックします。

Linux

  • ブラウザ用の .json ファイルを generated/linux/ からダウンロードします。
  • ダウンロードしたファイルを正しいポリシーディレクトリに移動します（必要に応じて作成してください）：
  • Chrome: /etc/opt/chrome/policies/managed/chrome.json
  • Edge: /etc/opt/edge/policies/managed/edge.json
  • Brave: /etc/brave/policies/managed/brave.json
注意: これを行うには sudo 権限が必要な場合があります。*
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy）にアクセスして「ポリシーの再読み込み」をクリックします。

カスタム設定

ポリシーをカスタマイズしたい場合：

  • このリポジトリをクローンします
  • 依存関係をインストールします：

``bash uv sync `
  • 必要に応じて policies.yaml を修正してください
  • 新しい設定ファイルを生成します:
`bash uv run main.py `
  • 生成されたファイルは generated/ ディレクトリにあります

ポリシーのアンインストール

Windows:

  • このリポジトリの uninstall/windows/ ディレクトリに移動します。
  • ブラウザに対応する .reg ファイル（例：uninstall_chrome.reg）を実行します。これによりインストール時に追加されたレジストリキーが削除されます。
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy）にアクセスして「ポリシーの再読み込み」をクリックします。

macOS:
  • 「システム設定」＞「プライバシーとセキュリティ」＞「プロファイル」に移動します。
  • ブラウザに関連付けられたプロファイル（例：「Chrome Debloat Policies」）を選択します。
  • 「-」（マイナス）ボタンをクリックしてプロファイルを削除します。
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy）にアクセスして「ポリシーの再読み込み」をクリックします。

Linux:
  • ブラウザ固有のディレクトリからポリシーの JSON ファイルを削除します（sudo 権限が必要な場合があります）：
  • Chrome: sudo rm /etc/opt/chrome/policies/managed/chrome.json
  • Edge: sudo rm /etc/opt/edge/policies/managed/edge.json
  • Brave: sudo rm /etc/brave/policies/managed/brave.json
  • ブラウザを再起動するか、chrome://policy（または edge://policy、brave://policy`）にアクセスして「ポリシーの再読み込み」をクリックします。
