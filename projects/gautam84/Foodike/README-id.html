🍔 Foodike

Foodike adalah aplikasi pengantaran makanan yang sederhana dan mudah digunakan. Aplikasi ini dibangun dengan Android-Jetpack Compose dan menerapkan prinsip MVVM dengan Komponen Arsitektur Android Modern.

Dibangun Dengan 🛠

  • Kotlin - Bahasa pemrograman resmi dan utama untuk pengembangan Android.
  • Jetpack Compose - Jetpack Compose adalah toolkit modern Android untuk membangun UI native.
  • Coroutines - Coroutine adalah pola desain konkurensi yang dapat digunakan di Android untuk menyederhanakan kode yang berjalan secara asinkron.
  • Flow - Flow adalah versi asinkron dari Sequence, tipe koleksi yang nilainya dihasilkan secara malas.
  • Android Architecture Components -
Kumpulan pustaka yang membantu Anda merancang aplikasi yang kuat, dapat diuji, dan mudah dipelihara.
  • Stateflow - StateFlow adalah flow yang dapat diamati dan menyimpan state, yang mengirimkan pembaruan state saat ini dan baru ke kolektornya.
  • Flow - Flow adalah versi asinkron dari Sequence, tipe koleksi yang nilainya dihasilkan secara malas.
  • ViewModel - Menyimpan data terkait UI yang tidak hilang saat terjadi perubahan UI.
  • Jetpack Compose Navigation - Komponen Navigasi menyediakan dukungan untuk aplikasi Jetpack Compose.
  • DataStore - Jetpack DataStore adalah solusi penyimpanan data yang memungkinkan Anda menyimpan pasangan key-value atau objek bertipe dengan protokol buffer. DataStore menggunakan coroutine dan Flow Kotlin untuk menyimpan data secara asinkron, konsisten, dan transaksional.
  • Dagger-Hilt - Hilt adalah pustaka dependency injection untuk Android yang mengurangi boilerplate dalam melakukan dependency injection manual di proyek Anda. Melakukan dependency injection manual membutuhkan Anda untuk membangun setiap kelas dan dependensinya secara manual, serta menggunakan container untuk mengelola dan menggunakan kembali dependensi.
  • Splash API - Mulai dari Android 12, SplashScreen API memungkinkan aplikasi diluncurkan dengan animasi, termasuk gerakan masuk aplikasi saat peluncuran, layar splash yang menampilkan ikon aplikasi Anda, dan transisi ke aplikasi itu sendiri. SplashScreen adalah Window sehingga menutupi sebuah Activity.
  • Material Components for Android
  • Komponen UI Material Design yang modular dan dapat disesuaikan untuk Android.
  • Figma - Figma adalah editor grafis vektor dan alat prototipe yang berbasis web utama.

Arsitektur 🗼

Aplikasi ini menggunakan arsitektur *MVVM (Model View View-Model)*.
