🍔 Foodike

Foodikeはシンプルで使いやすいフードデリバリーアプリです。Android-Jetpack Composeを使用して構築されており、MVVMとモダンAndroidアーキテクチャコンポーネントの原則に基づいています。

使用技術 🛠

  • Kotlin - Android開発のための公式の第一級プログラミング言語。
  • Jetpack Compose - Jetpack ComposeはAndroidの
最新のネイティブUI構築ツールキットです。
  • Coroutines - コルーチンは
Androidで非同期に実行されるコードを簡素化するための並行性設計パターンです。
  • Flow - Flowは、値を遅延生成する
シーケンスの非同期バージョンです。
  • Android Architecture Components -
頑健でテスト可能かつ保守しやすいアプリ設計を支援するライブラリ群です。
  • Stateflow - StateFlowは
現在および新しい状態更新をコレクターへ発行する状態保持型のオブザーバブルFlowです。
  • Flow - Flowは、値を遅延生成する
シーケンスの非同期バージョンです。
  • ViewModel - UI関連データを
UI変更時にも破棄されずに保持します。
  • Jetpack Compose Navigation - この
NavigationコンポーネントはJetpack Composeアプリ向けナビゲーションをサポートします。
  • DataStore - Jetpack
DataStoreは、プロトコルバッファを用いてキー・バリューや型付きオブジェクトを保存できる データ保存ソリューションです。DataStoreはKotlin CoroutinesおよびFlowを使い、非同期・ 一貫性・トランザクション性のあるデータ保存を実現します。
  • Dagger-Hilt - HiltはAndroid向けの依存性注入ライブラリで、手動依存性注入のボイラープレートを削減します。手動で依存性注入を行う場合、各クラスとその依存関係をすべて手で構築し、依存性を再利用・管理するためのコンテナも使う必要があります。
  • Splash API - Android 12以降、SplashScreen APIによりアニメーション付きでアプリ起動、アプリアイコンを表示するスプラッシュ画面、アプリ本体への遷移などが可能です。SplashScreenはウィンドウの一種で、アクティビティ全体を覆います。
  • Material Components for Android
  • Android向けのモジュール化可能でカスタマイズ可能なMaterial Design UIコンポーネント。
  • Figma - Figmaは主にウェブベースのベクターグラフィックエディタ兼
プロトタイピングツールです。

アーキテクチャ 🗼

このアプリは*MVVM (Model View View-Model)*アーキテクチャを使用しています。

ビルドツール 🧰

このプロジェクトをビルドするには Android Studio Beta 3 以上 が必要です。
