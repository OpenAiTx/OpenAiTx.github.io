🍔 Foodike

Foodike এটা সহজ, ব্যৱহাৰ কৰিবলৈ সৰল খাদ্য ডেলিভাৰী এপ। এইটো Android-Jetpack Compose ব্যৱহাৰ কৰি নিৰ্মিত আৰু MVVM আৰু আধুনিক Android Architecture Components-ৰ নীতি অনুসৰি নিৰ্মিত।

নিৰ্মাণ কৰা হৈছে 🛠

  • কটলিন - Android ৰ বাবে প্ৰথম শ্ৰেণীৰ আৰু চৰকাৰী প্ৰগ্ৰামিং ভাষা।
  • Jetpack Compose - Jetpack Compose হৈছে Android ৰ আধুনিক টুলকিট, যি Native UI নিৰ্মাণৰ বাবে ব্যৱহাৰ হয়।
  • Coroutines - Coroutine হৈছে এটা concurrency design pattern, যি Android ত asynchronous code execute কৰিবলৈ সহজ কৰি তোলে।
  • Flow - Flow হৈছে Sequence ৰ asynchronous সংস্কৰণ, এটা collection, যাৰ মান slow-ভাৱে উত্পন্ন হয়।
  • Android Architecture Components -
robust, testable, আৰু maintainable app ডিজাইন কৰিবলৈ সহায়ক library ৰ collection।
  • Stateflow - StateFlow হৈছে state-holder observable flow, যি current আৰু নতুন state update সমূহ collectors লৈ emit কৰে।
  • Flow - Flow হৈছে Sequence ৰ asynchronous সংস্কৰণ, এটা collection, যাৰ মান slow-ভাৱে উত্পন্ন হয়।
  • ViewModel - UI-সংক্রান্ত data সংৰক্ষণ কৰে, যি UI পৰিবর্তনত নষ্ট নোহোৱা হয়।
  • Jetpack Compose Navigation - Navigation component, Jetpack Compose application সমূহৰ বাবে support প্ৰদান কৰে।
  • DataStore - Jetpack DataStore হৈছে data storage solution, যি key-value pair বা typed object store কৰিব পাৰে protocol buffers সহ। DataStore, Kotlin coroutine আৰু Flow ব্যৱহাৰ কৰি data asynchronously, consistently, আৰু transactionally store কৰে।
  • Dagger-Hilt - Hilt হৈছে Android ৰ বাবে এটা dependency injection library, যি project ত manual dependency injection ৰ boilerplate কমাই। Manual dependency injection ৰ বাবে প্ৰতিটো class আৰু তাৰ dependency হাতেদি নিৰ্মাণ কৰিব লগা হয়, আৰু dependency reuse আৰু management ৰ বাবে container ব্যৱহাৰ কৰিব লাগে।
  • Splash API - Android 12 পৰা আৰম্ভ কৰি, SplashScreen API apps launch কৰোঁতে animation support দিয়ে, য’ত launch সময়ত into-app motion, app icon সহ splash screen, আৰু app ত transition থাকে। SplashScreen এটা Window আৰু Activity ক cover কৰে।
  • Material Components for Android
  • Android ৰ বাবে modular আৰু customizable Material Design UI component।
  • Figma - Figma এটা vector graphics editor আৰু prototyping tool, যি প্ৰধানকৈ web-based।
