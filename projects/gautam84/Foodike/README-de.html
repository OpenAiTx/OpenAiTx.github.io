🍔 Foodike

Foodike ist eine einfache, benutzerfreundliche Essenslieferungs-App. Sie wurde mit Android-Jetpack Compose entwickelt und basiert auf dem Prinzip von MVVM mit modernen Android Architekturkomponenten.

Entwickelt mit 🛠

  • Kotlin – Erstklassige und offizielle Programmiersprache für die Android-
Entwicklung.
  • Jetpack Compose – Jetpack Compose ist das moderne Toolkit von Android zum Erstellen nativer UIs.
  • Coroutines – Eine Coroutine ist ein
Nebenläufigkeits-Designmuster, mit dem Sie auf Android Code vereinfachen können, der asynchron ausgeführt wird.
  • Flow – Ein Flow ist eine asynchrone
Version einer Sequence, eine Sammlung, deren Werte verzögert erzeugt werden.
  • Android Architecture Components –
Sammlung von Bibliotheken, die Ihnen helfen, robuste, testbare und wartbare Apps zu entwickeln.
  • Stateflow – StateFlow ist ein
zustandshaltender, beobachtbarer Flow, der aktuelle und neue Statusaktualisierungen an seine Collector sendet.
  • Flow – Ein Flow ist eine asynchrone
Version einer Sequence, eine Sammlung, deren Werte verzögert erzeugt werden.
  • ViewModel – Speichert
UI-bezogene Daten, die bei UI-Änderungen nicht verloren gehen.
  • Jetpack Compose Navigation – Die
Navigationskomponente bietet Unterstützung für Jetpack Compose-Anwendungen.
  • DataStore – Jetpack
DataStore ist eine Datenspeicherlösung, mit der Sie Schlüssel-Wert-Paare oder typisierte Objekte mit Protokollpuffern speichern können. DataStore verwendet Kotlin Coroutines und Flow, um Daten asynchron, konsistent und transaktional zu speichern.
  • Dagger-Hilt – Hilt ist eine Dependency-Injection-Bibliothek für Android, die den Boilerplate-Code für manuelle Dependency Injection im Projekt reduziert. Bei manueller Dependency Injection müssen Sie jede Klasse und ihre Abhängigkeiten von Hand konstruieren und Container zur Wiederverwendung und Verwaltung von Abhängigkeiten verwenden.
  • Splash API – Ab Android 12 ermöglicht die SplashScreen API das Starten von Apps mit Animation, einschließlich einer Bewegung beim Start, einem Splashscreen mit Ihrem App-Symbol und einer Überleitung zur App selbst. Ein SplashScreen ist ein Fenster und deckt daher eine Aktivität ab.
  • Material Components for Android
– Modulare und anpassbare Material-Design-UI-Komponenten für Android.
  • Figma – Figma ist ein vektorbasiertes Grafik- und Prototyping-Tool, das
hauptsächlich webbasiert ist.

Architektur 🗼
