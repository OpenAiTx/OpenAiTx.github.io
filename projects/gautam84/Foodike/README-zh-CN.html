🍔 Foodike

Foodike 是一个简单、易用的外卖应用。它使用 Android-Jetpack Compose 构建，基于 MVVM 原则和现代 Android 架构组件开发。

构建技术 🛠

  • Kotlin - Android 开发的第一语言和官方编程语言。
  • Jetpack Compose - Jetpack Compose 是 Android 的现代原生 UI 构建工具包。
  • Coroutines - 协程是一种并发设计模式，可用于 Android 上简化异步执行的代码。
  • Flow - Flow 是序列（Sequence）的异步版本，一种值惰性产生的集合类型。
  • Android Architecture Components -
一组帮助设计健壮、可测试和易维护应用的库。
  • Stateflow - StateFlow 是一个状态持有的可观察流，向收集者发出当前及新的状态更新。
  • Flow - Flow 是序列（Sequence）的异步版本，一种值惰性产生的集合类型。
  • ViewModel - 存储与 UI 相关的数据，UI 变化时数据不会被销毁。
  • Jetpack Compose Navigation - Navigation 组件为 Jetpack Compose 应用提供支持。
  • DataStore - Jetpack DataStore 是一种数据存储解决方案，允许使用键值对或协议缓冲区的类型化对象存储数据。DataStore 使用 Kotlin 协程和 Flow 以异步、一致和事务的方式存储数据。
  • Dagger-Hilt - Hilt 是 Android 的依赖注入库，减少了项目中手动依赖注入的样板代码。手动依赖注入需要手工构造每个类及其依赖，并使用容器来重用和管理依赖。

  • Splash API - 从 Android 12 开始，SplashScreen API 允许应用启动时带有动画，包括启动时的应用内动画、显示应用图标的启动屏幕，以及过渡到应用本身。SplashScreen 是一个窗口，因此会覆盖一个 Activity。
  • Material Components for Android
  • Android 的模块化且可定制的 Material Design UI 组件。
  • Figma - Figma 是一个矢量图形编辑器和原型设计工具，主要基于网页。

Architecture 🗼

该应用使用 MVVM（模型-视图-视图模型） 架构。

Build-tool 🧰

你需要安装 Android Studio Beta 3 及以上版本 来构建此项目。
