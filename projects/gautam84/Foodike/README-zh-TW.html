🍔 Foodike

Foodike 是一款簡單易用的外送美食應用程式。它是基於 Android-Jetpack Compose 構建，並採用 MVVM 原則與現代 Android 架構元件實現。

開發技術 🛠

  • Kotlin - Android 官方的一流程式語言。
  • Jetpack Compose - Jetpack Compose 是 Android 的現代原生 UI 建構工具包。
  • Coroutines - 協程是一種並發設計模式，可簡化 Android 非同步執行程式碼。
  • Flow - Flow 是序列的非同步版本，集合的值會延遲產生。
  • Android Architecture Components - 一系列協助設計堅固、可測試及可維護應用程式的函式庫。
  • Stateflow - StateFlow 是一種狀態持有者，可觀察流，向收集器發送當前及新狀態更新。
  • Flow - Flow 是序列的非同步版本，集合的值會延遲產生。
  • ViewModel - 儲存 UI 相關資料，並不會因 UI 改變而被銷毀。
  • Jetpack Compose Navigation - 導航元件提供 Jetpack Compose 應用程式的導航支援。
  • DataStore - Jetpack DataStore 是資料儲存解決方案，可儲存鍵值對或型別物件（協議緩衝區）。DataStore 使用 Kotlin 協程及 Flow 以非同步、穩定及交易方式儲存資料。
  • Dagger-Hilt - Hilt 是 Android 的依賴注入函式庫，減少專案手動依賴注入的樣板程式碼。手動依賴注入需自行構建每個類及其依賴，並使用容器管理與重用依賴。
  • Splash API - 從 Android 12 起，SplashScreen API 讓應用啟動時有動畫，包括進入應用動作、顯示應用圖示的啟動畫面，以及過渡到應用本身。SplashScreen 是一個 Window，因此會覆蓋 Activity。
  • Material Components for Android
  • Android 的模組化與可自訂 Material Design UI 元件。
  • Figma - Figma 是主要基於網頁的向量圖編輯器與原型設計工具。

架構 🗼

此應用程式採用 MVVM（模型-視圖-視圖模型） 架構。

建置工具 🧰

您需要安裝 Android Studio Beta 3 或以上版本 來建置本專案。
