🍔 Foodike

Foodike คือแอปส่งอาหารที่ใช้งานง่ายและเรียบง่าย ถูกสร้างขึ้นโดยใช้ Android-Jetpack Compose และยึดหลักการ MVVM พร้อมด้วย Modern Android Architecture Components

สร้างด้วย 🛠

  • Kotlin - ภาษาโปรแกรมหลักและทางการสำหรับการพัฒนา Android
  • Jetpack Compose - Jetpack Compose เป็นชุดเครื่องมือสมัยใหม่สำหรับสร้าง UI แบบเนทีฟบน Android
  • Coroutines - Coroutine เป็นรูปแบบการออกแบบการทำงานร่วมกันที่ช่วยให้คุณเขียนโค้ดแบบอะซิงโครนัสบน Android ได้ง่ายขึ้น
  • Flow - Flow เป็นเวอร์ชันอะซิงโครนัสของ Sequence ซึ่งเป็นคอลเลกชันที่ผลิตค่าแบบ Lazy
  • Android Architecture Components -
กลุ่มไลบรารีที่ช่วยออกแบบแอปให้มีความแข็งแกร่ง ทดสอบได้ และดูแลรักษาง่าย
  • Stateflow - StateFlow คือ flow แบบถือสถานะที่สามารถสังเกตได้ และปล่อยอัปเดตสถานะปัจจุบันและใหม่ให้กับผู้เก็บรวบรวม
  • Flow - Flow เป็นเวอร์ชันอะซิงโครนัสของ Sequence ซึ่งเป็นคอลเลกชันที่ผลิตค่าแบบ Lazy
  • ViewModel - เก็บข้อมูลที่เกี่ยวข้องกับ UI ซึ่งจะไม่ถูกทำลายเมื่อ UI มีการเปลี่ยนแปลง
  • Jetpack Compose Navigation - คอมโพเนนท์ Navigation รองรับการนำทางสำหรับแอป Jetpack Compose
  • DataStore - Jetpack DataStore เป็นโซลูชันการจัดเก็บข้อมูลที่ให้คุณจัดเก็บ key-value หรืออ็อบเจกต์แบบ typed ด้วย protocol buffers โดยใช้ Kotlin coroutines และ Flow เพื่อจัดเก็บข้อมูลแบบอะซิงโครนัส สม่ำเสมอ และเชิงธุรกรรม
  • Dagger-Hilt - Hilt เป็นไลบรารีสำหรับ Dependency Injection บน Android ที่ช่วยลดโค้ดซ้ำซ้อนจากการฉีด Dependency แบบ manual ในโปรเจกต์ของคุณ การฉีด Dependency แบบ manual ต้องสร้างแต่ละคลาสและ Dependency ของมันด้วยมือ และใช้คอนเทนเนอร์ในการนำกลับมาใช้ใหม่และจัดการ Dependency
  • Splash API - ตั้งแต่ Android 12 เป็นต้นไป SplashScreen API ช่วยให้แอปเปิดด้วยแอนิเมชัน รวมถึงการเคลื่อนไหวเข้าสู่แอปเมื่อเริ่มต้น หน้าจอสแปลชที่แสดงไอคอนแอป และการเปลี่ยนเข้าสู่แอปจริง SplashScreen เป็น Window จึงคลุม Activity ไว้ด้วย
  • Material Components for Android
  • คอมโพเนนท์ UI แบบ Material Design ที่ปรับแต่งและใช้งานแบบโมดูลาร์ได้สำหรับ Android
  • Figma - Figma เป็นโปรแกรมแก้ไขกราฟิกแบบเวกเตอร์และเครื่องมือสร้างต้นแบบที่เน้นการใช้งานบนเว็บ
