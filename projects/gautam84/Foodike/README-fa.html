🍔 فودایک

فودایک یک اپلیکیشن ساده و کاربرپسند برای سفارش آنلاین غذا است. این برنامه با استفاده از Android-Jetpack Compose توسعه یافته و بر پایه اصل MVVM با اجزای معماری مدرن اندروید ساخته شده است.

ساخته شده با 🛠

  • Kotlin - زبان برنامه‌نویسی رسمی و سطح اول برای توسعه اندروید.
  • Jetpack Compose - جت‌پک کامپوز ابزار مدرن اندروید برای ساخت رابط کاربری بومی است.
  • Coroutines - کوروتین یک الگوی طراحی همزمانی است که در اندروید برای ساده‌سازی کدهای اجرای غیرهمزمان استفاده می‌شود.
  • Flow - فلو نسخه غیرهمزمانی از Sequence است؛ نوعی مجموعه که مقادیر آن به صورت تنبل تولید می‌شوند.
  • اجزای معماری اندروید -
مجموعه‌ای از کتابخانه‌ها که به طراحی اپ‌های قوی، قابل تست و قابل نگهداری کمک می‌کنند.
  • Stateflow - StateFlow یک جریان مشاهده‌پذیر نگهدارنده وضعیت است که وضعیت فعلی و جدید را به جمع‌کننده‌های خود ارسال می‌کند.
  • Flow - فلو نسخه غیرهمزمانی از Sequence است؛ نوعی مجموعه که مقادیر آن به صورت تنبل تولید می‌شوند.
  • ViewModel - داده‌های مرتبط با UI را ذخیره می‌کند که در تغییرات UI از بین نمی‌روند.
  • ناوبری جت‌پک کامپوز - کامپوننت ناوبری پشتیبانی از برنامه‌های جت‌پک کامپوز را فراهم می‌کند.
  • DataStore - جت‌پک دیتاستور راه‌حل ذخیره‌سازی داده است که امکان ذخیره جفت‌های کلید-مقدار یا اشیاء تایپ شده با پروتکل بافرها را فراهم می‌کند. دیتاستور از کوروتین‌های Kotlin و Flow برای ذخیره داده‌ها به صورت غیرهمزمان، پیوسته و تراکنشی استفاده می‌کند.
  • Dagger-Hilt - هیلت یک کتابخانه تزریق وابستگی برای اندروید است که حجم کار تزریق دستی وابستگی‌ها در پروژه را کاهش می‌دهد. تزریق وابستگی دستی نیاز دارد هر کلاس و وابستگی‌های آن را به صورت دستی بسازید و برای مدیریت و استفاده مجدد وابستگی‌ها از کانتینرها بهره بگیرید.
  • Splash API - از اندروید ۱۲، SplashScreen API به اپ‌ها اجازه می‌دهد با انیمیشن راه‌اندازی شوند، شامل حرکت ورود به اپ، نمایش اسپلش اسکرین با آیکن اپ و انتقال به خود اپ. اسپلش اسکرین یک Window است و به همین خاطر یک Activity را می‌پوشاند.
  • اجزای متریال برای اندروید
  • کامپوننت‌های UI متریال دیزاین ماژولار و قابل سفارشی‌سازی برای اندروید.
  • Figma - فیگما یک ویرایشگر گرافیک برداری و ابزار نمونه‌سازی است که عمدتاً مبتنی بر وب می‌باشد.
