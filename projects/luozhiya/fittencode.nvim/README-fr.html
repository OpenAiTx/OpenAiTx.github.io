fittencode.nvim

Le fittencode.nvim refactorisé est prêt.

✨ Fonctionnalités

  • 🚀 Complétion rapide grâce à Fitten Code
  • ⚡️ E/S asynchrones pour des performances améliorées
  • ✏️ Complétion en ligne améliorée
  • ✨ Complétion en ligne
  • ✂️ Édition de la complétion
  • 📂 Complétion de projet
  • 🧩 C/C++
  • 🔄 Acceptation/Annulation par plage
  • 📝 Caractère
  • 📖 Mot
  • 📜 Ligne
  • 📋 Tout
  • 🀄️ Complétion chinoise améliorée avec prise en charge de la segmentation de caractères
  • 🔌 Sources pour blink.cmp et serveur LSP natif de Neovim

⚡️ Exigences

  • Neovim 0.11+
  • curl

📦 Installation

Installez le plugin avec votre gestionnaire de paquets préféré :

Par exemple avec lazy.nvim :

{
  'luozhiya/fittencode.nvim',
  opts = {},
}

🚀 Utilisation

| Nom de la commande | Description | |-------------------------|-----------------------| | register | Enregistrer un compte | | login | Connexion au compte | | login3rd | Connexion tierce | | logout | Déconnexion du compte | | ask_question | Poser une question | | user_guide | Guide utilisateur | | enable_completions | Activer les complétions | | disable_completions | Désactiver les complétions | | onlyenable_completions | Activer les complétions pour des fichiers spécifiques | | onlydisable_completions | Désactiver les complétions pour des fichiers spécifiques |

✏️ API

| Fonction | Description | |-------------------------|-------------------------------------------| | has_completions() | Vérifier si des suggestions de complétion sont disponibles | | accept(scope) | Accepter la suggestion de complétion actuelle | | revoke() | Annuler la dernière opération de complétion | | completion_cancel() | Annuler la suggestion de complétion actuelle | | get_status() | Obtenir le statut actuel du système de complétion |
