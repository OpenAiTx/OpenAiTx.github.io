Over

Hoi! Ik ben Spook 👻 en ik ben een custom integratie voor gebruik met Home Assistant. Ik breid je Home Assistant instantie uit met een enorme set angstaanjagend krachtige tools. 🛠️

Leer alles over mij in de uitgebreide documentatie

⚠️ Om het heel duidelijk te maken...

Spook is niet gelieerd aan, goedgekeurd, aanbevolen of ondersteund door het Home Assistant project.

Deze custom integratie wordt geleverd zoals hij is, zonder enige garantie.
