frontier-station-14

⭐ 0 stars Korean by new-frontiers-14

Frontier Station은 Space Station 14를 기반으로 하여 Robust Toolbox 엔진(C#로 작성됨) 위에서 구동되는 포크입니다.

이 저장소는 Frontier Station의 주요 저장소입니다.

Frontier Station을 호스팅하거나 콘텐츠를 제작하고 싶다면 이 저장소를 사용해야 합니다. 이 저장소에는 RobustToolbox와 신규 콘텐츠 팩 개발을 위한 콘텐츠 팩이 모두 포함되어 있습니다.

cd frontier-station-14
python RUN_THIS.py
  • 솔루션을 컴파일합니다:

서버를 빌드하려면 dotnet build를 사용하세요.
