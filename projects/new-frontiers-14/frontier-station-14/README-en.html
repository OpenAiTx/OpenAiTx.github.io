frontier-station-14

⭐ 0 stars English by new-frontiers-14

Frontier Station is a fork of Space Station 14 that runs on the Robust Toolbox engine written in C#.

This is the primary repository for Frontier Station.

If you want to host or create content for Frontier Station, this is the repository you need. It contains both RobustToolbox and the content pack for the development of new content packs.

Contributing

We are happy to accept contributions from anyone. Join our Discord if you want to help. We have a list of ideas that anyone can take on. Don't hesitate to ask for help either!

We are not currently accepting translations of the game in our main repository. If you would like to translate the game into another language, consider creating a fork or contributing to a fork.

If you make any contributions, please make sure to read the markers section in MARKERS.md Any changes made to files belonging to our upstream should be properly marked in accordance with what is specified there.

Building

  • Clone this repository:
git clone https://github.com/new-frontiers-14/frontier-station-14.git
  • Go to the project folder and run RUN_THIS.py to initialize the submodules and load the engine:
cd frontier-station-14
python RUN_THIS.py
  • Compile the solution:

Build the server using dotnet build.

More detailed instructions on building the project.
