사전 요구 사항

다음이 설치되어 있는지 확인하세요:

  • Bun: 빠른 자바스크립트 런타임, 패키지 관리자, 번들러 및 테스트 러너입니다.
  • 설치 가이드
  • Docker Desktop: PostgreSQL 데이터베이스 실행용.

  • BETTER_AUTH_SECRET: 터미널에서 openssl rand -hex 32를 실행하여 안전한 비밀 키를 생성하세요.

  • GOOGLE_CLIENT_ID 및 GOOGLE_CLIENT_SECRET:

  • Better Auth 문서의 1단계를 따라 Google OAuth 자격 증명을 설정하세요.
    • 
  
  • Google Cloud Console APIs에서 Google Calendar API를 활성화합니다.
    • 
  
  • 테스트 사용자로 자신을 추가하세요:

데이터베이스 설정
Analog는 Drizzle ORM과 함께 PostgreSQL을 사용합니다. Docker를 사용하여 데이터베이스를 실행할 수 있습니다:
  •  PostgreSQL 데이터베이스 컨테이너 시작:
    • 
    
`bash
    bun run docker:up
    `

    이 명령어는 docker-compose.yml을 사용하여 PostgreSQL 컨테이너를 실행합니다.
  •  데이터베이스 초기화:
    • 
    
데이터베이스 컨테이너가 실행 중이고 정상 상태가 되면, 데이터베이스를 초기화합니다:
    `bash
    bun run db:push
    `

애플리케이션 실행하기
환경과 데이터베이스 설정을 마친 후, 개발 서버를 시작할 수 있습니다:
bash bun run dev ``

이제 브라우저에서 http://localhost:3000 에서 애플리케이션에 접근할 수 있습니다.
