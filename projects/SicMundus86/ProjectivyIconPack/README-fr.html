🔴 À propos du pack d’icônes Projectivy

Le pack d’icônes Projectivy est conçu spécifiquement pour une utilisation avec l’application Projectivy Launcher. Il comprend désormais plus de 800 icônes.

Créé et basé sur le projet Blueprint — qui a rendu ce pack d’icônes possible — il est facile à appliquer, avec des icônes configurées pour s’assigner automatiquement à toutes les applications supportées, vous faisant gagner du temps et évitant de sélectionner manuellement les icônes pour chaque application.

Toutes les icônes sont faites sur mesure pour offrir un aspect moderne et cohérent, avec des fonds transparents qui améliorent la cohérence visuelle et vous permettent d’assigner vos couleurs de fond préférées dans le lanceur.

Astuce : Pour de meilleurs résultats, il est recommandé d’utiliser un fond sombre pour les icônes, car les icônes sont conçues pour être les plus belles sur des fonds sombres.

Note : Le pack d’icônes Projectivy est conçu et testé avec Android / Google TV. Si vous utilisez un appareil TV fonctionnant sous une variante mobile non conçue pour les téléviseurs, certaines icônes peuvent ne pas s’assigner automatiquement car certaines applications utilisent différentes activités pour les systèmes mobiles. Dans ce cas, vous devrez peut-être les assigner manuellement.

🔴 Comment télécharger et configurer

Téléchargez en utilisant le code de téléchargement 9257057 ou obtenez le fichier APK depuis ici.

Une fois installé, assignez le pack d’icônes directement dans l’application du pack d’icônes en sélectionnant le bouton d’application, ou dans Projectivy Launcher en suivant ces étapes :

  • Depuis le menu du lanceur, sélectionnez Paramètres Projectivy Launcher.
  • Sélectionnez Apparence.
  • Sélectionnez Cartes.
  • Sélectionnez Pack d’icônes.
  • Sélectionnez Pack d’icônes Projectivy.

🔴 Comment mettre à jour le pack d’icônes

Le moyen le plus simple de mettre à jour l'application est via le système de mise à jour intégré. Une fois qu'une mise à jour est disponible, vous verrez une invite à l'ouverture de l'application indiquant qu'une nouvelle version est disponible. Vous pouvez également télécharger directement le fichier APK depuis GitHub ou via Downloader si vous préférez.

Note : Lorsqu'une nouvelle version est installée, vous devrez réappliquer le pack d'icônes pour voir les changements.

Changer la couleur de fond des icônes

Pour améliorer davantage l'apparence du pack d'icônes, vous pouvez changer la couleur de fond des cartes :

  • Dans le menu du Projectivy Launcher, sélectionnez Paramètres du Projectivy Launcher.
  • Sélectionnez Apparence.
  • Sélectionnez Cartes.
  • Sélectionnez Couleur de fond

🔴 Demande de nouvelles icônes

Si vous constatez qu'une icône manque pour une application compatible avec les appareils TV, vous pouvez soumettre une demande pour qu'elle soit ajoutée au pack d'icônes Projectivy ici.

Avant de soumettre une demande, veuillez vérifier que l'icône n'est pas déjà incluse dans le pack ou n'a pas déjà été demandée. Une liste complète des icônes d'applications prises en charge est disponible ici.

Note : Bien que nous nous efforçons de satisfaire toutes les demandes, il n'y a aucune garantie que chaque demande sera réalisée, ni de délai spécifique pour l'ajout des icônes demandées.

🔴 Problèmes de mappage automatique des icônes

Bien que j'aie pris soin de mapper et de tester chaque icône, il est toujours possible que quelque chose ait été manqué. Si vous remarquez que certaines applications n'ont pas automatiquement leur icône attribuée, assurez-vous d'abord que vous utilisez la dernière version de l'application et que vous avez redémarré votre appareil. Si le problème persiste, veuillez signaler un problème de mappage ici.

Note : Le mappage est configuré et testé uniquement sur les appareils Android/Google TV. Les appareils fonctionnant avec des variantes mobiles non officielles ne seront pas officiellement pris en charge. Il peut également être impossible de mapper automatiquement chaque icône en raison des différences régionales et d'appareil dans les noms des paquets d'applications et les activités. Dans ces cas, vous pouvez toujours appliquer les icônes manuellement depuis le launcher.

🔴 Demande d'améliorations d'icônes

Toutes les icônes ne sont pas parfaites, et parfois un meilleur design peut améliorer l'apparence générale de l'application. Si vous avez découvert ou créé un design que vous pensez améliorer une icône existante, veuillez créer un ticket et joindre une image de l'icône proposée, en vous assurant qu'elle soit de haute qualité, claire et nette. Vous pouvez soumettre une demande d'amélioration d'icône ici.

Note : Les demandes d'amélioration d'icônes sont considérées comme une priorité moindre par rapport aux demandes de nouvelles icônes. Toutes les suggestions d'amélioration peuvent ne pas être acceptées, car elles peuvent être basées sur des préférences personnelles et ne pas constituer une réelle amélioration.

🔴 Dépannage

Il y a quelques petits dysfonctionnements qui échappent à mon contrôle car ils dépendent du comportement du lanceur, mais ils sont faciles à contourner et ne causeront pas trop de problèmes.

  • "J'ai appliqué le pack d'icônes pour la première fois et mes anciennes icônes s'affichent toujours".

Cela se produit si vous avez attribué manuellement vos propres icônes à chaque application. Le pack d'icônes ne remplacera pas celles-ci, vous devrez donc réinitialiser manuellement l'icône pour chaque application.
  • "Les nouvelles icônes ajoutées au pack d'icônes ne sont pas attribuées à mes applications".
Lors de la mise à jour du pack d'icônes, les nouvelles icônes peuvent ne pas être visiblement attribuées aux applications. Vous devrez appliquer à nouveau le pack d'icônes en suivant les étapes de la section « comment télécharger et installer » de ce document.

  • "Les icônes mises à jour ne s'affichent pas. Je vois toujours les anciennes versions".

Parfois, après la mise à jour du pack d'icônes, le cache du lanceur projectivy peut encore afficher des icônes plus anciennes de celles qui ont été remplacées par des versions améliorées. Redémarrer votre appareil devrait résoudre ce problème.

  • "Les icônes des applications dans le menu « toutes les applications » affichent des icônes incorrectes à côté".

Je pense que cela peut être un bug du lanceur lui-même. Après avoir appliqué/mis à jour le pack d'icônes, il arrive que le cache du lanceur projectivy affiche les mauvaises icônes à côté des applications dans le menu « toutes les applications ». Redémarrer votre appareil devrait résoudre cela.

  • "Une de mes applications vient de se mettre à jour, et maintenant l'icône a été réinitialisée et l'application ne s'ouvre plus."

Cela arrive parce que le développeur de l'application a changé l'activité de l'application, donc l'icône est toujours associée à une activité qui n'existe plus dans l'application. J'ai signalé cela au développeur de projectivy, donc j'espère que ce sera corrigé dans le lanceur.

Pour le réparer :

  • Retirez l'application de votre catégorie
– Allez dans votre liste de catégories et supprimez l'entrée de l'application.

  • Ajoutez-la à nouveau
– L'application devrait maintenant s'ouvrir correctement. – Vous pouvez réattribuer l'icône manuellement si nécessaire.

  • Signalez un problème de correspondance sur GitHub
– Signalez le problème afin que l'auto-mappage puisse être mis à jour et restauré dans les futures versions.
