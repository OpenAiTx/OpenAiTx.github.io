نصب / راه‌اندازی

  • مخزن را کلون کنید / کلون کردن مخزن:
``bash git clone https://github.com/grtsinry43/grtblog.git cd grtblog `
  • نصب وابستگی‌های فرانت‌اند / نصب وابستگی‌های بخش جلویی:

`bash cd frontend npm install `
  • نصب وابستگی‌های بک‌اند / نصب وابسته‌های پشتی:

`bash cd ../backend mvn install `

اجرای برنامه / Running the Application

  • سرور بک‌اند را راه‌اندازی کنید / Start the back-end server:
`bash cd backend mvn spring-boot:run `
  • راه‌اندازی سرور توسعه فرانت‌اند /启动前端开发服务器:

`bash cd ../frontend npm start ``
