TerminalStart

复古风格，模块化的新标签页仪表盘，专为专注和速度而设计。

在线演示

在 Edge / Chrome 中安装

  • 打开 edge://extensions（或 chrome://extensions）
  • 启用 开发者模式
  • 点击 加载已解压的扩展程序
  • 选择 extension 文件夹

Todoist 集成

使用个人 API 令牌同步您的待办小部件与 Todoist。

  • 进入 设置 > 高级 > 待办小部件
  • 启用 与 Todoist 同步
  • 粘贴您的 API 令牌（设置 > 集成 > 开发者，访问 todoist.com）
  • 提示时授予主机权限

截止日期 — 创建任务时添加自然语言日期：

| 输入 | 任务 | 截止时间 | |---|---|---| | meet john tomorrow at 2pm | meet john | 明天下午2点 | | buy groceries next monday | buy groceries | 下周一 | | call mom every friday | call mom | 每周五 | | submit report jan 15 | submit report | 1月15日 |

Todoist 的自然语言处理负责解析。离线模式仍支持仅时间语法（例如 standup 9am）。

构建

要求

  • Node.js v16+
  • Python 3

步骤

  • 安装依赖项：

npm install

  • 构建：

npm run build

构建脚本会自动将资源同步到 extension/ 文件夹。

  • 打包：

python scripts/package_addon.py
输出：terminal-start-v1.0.0.zip

测试

npx playwright test

使用 Chromium 运行针对扩展的端到端测试。

说明

  • 将鼠标悬停在右上角区域以设置。
  • API 令牌存储在您的浏览器本地，除 Todoist API 外，绝不发送到任何其他地方。
