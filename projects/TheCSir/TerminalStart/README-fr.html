TerminalStart

Tableau de bord de nouvel onglet modulaire, inspiré du rétro, conçu pour la concentration et la rapidité.

Démo en ligne

Installer dans Edge / Chrome

  • Ouvrez edge://extensions (ou chrome://extensions)
  • Activez le Mode développeur
  • Cliquez sur Charger l’extension non empaquetée
  • Sélectionnez le dossier extension

Intégration Todoist

Synchronisez votre widget todo avec Todoist en utilisant un jeton API personnel.

  • Allez dans Paramètres > Avancé > Widget Todo
  • Activez Synchronisation avec Todoist
  • Collez votre jeton API (Paramètres > Intégrations > Développeur sur todoist.com)
  • Accordez l'autorisation d'hôte lorsque cela est demandé

Dates d'échéance — ajoutez des dates en langage naturel lors de la création des tâches :

| Entrée | Tâche | Échéance | |---|---|---| | rencontrer john demain à 14h | rencontrer john | demain à 14h | | acheter des courses lundi prochain | acheter des courses | lundi prochain | | appeler maman tous les vendredis | appeler maman | tous les vendredis | | soumettre rapport 15 janv | soumettre rapport | 15 janv |

Le NLP de Todoist gère l'analyse. Le mode local prend toujours en charge la syntaxe horaire seule (ex. standup 9h).

Compilation

Exigences

  • Node.js v16+
  • Python 3

Étapes

  • Installer les dépendances :

npm install

  • Construction :

npm run build

Le script de construction synchronise automatiquement les ressources dans le dossier extension/.

  • Emballer :

python scripts/package_addon.py

Sortie : terminal-start-v1.0.0.zip

Tests

npx playwright test

Exécute des tests e2e contre l’extension en utilisant Chromium.

Notes

  • Survolez la section en haut à droite pour les paramètres.
  • Les jetons API sont stockés localement dans votre navigateur et ne sont jamais envoyés ailleurs que vers l’API Todoist.
