Aero7-11

ธีม Aero 7 สำหรับ Windows 11 23H2+!

[!หมายเหตุ]
Aero7-11 ถูกสร้างขึ้นโดยใช้ aero.msstyles ดั้งเดิมใน Windows 11 ซึ่งได้มีการสร้างและทดสอบบน Windows 11 23H2 และ 24H2
Aero7-11 รองรับ 22H2 ถึง 23H2 ยังไม่มีการทดสอบบน 25H2 และ DWM mods อาจมีบั๊กเล็กน้อยใน 24H2

คำเตือน

[!NOTE]
แนะนำให้ใช้โหมดแอปสว่างและโหมด Windows มืด ธีมนี้ ไม่ รองรับโหมดแอปมืด โหมด Windows ไม่มีผลมากนัก แต่ส่วนใหญ่ mods ของทาสก์บาร์จะทำงานได้ดีกว่าเมื่อใช้ฐานมืดมากกว่าฐานสว่าง
>
24H2 พิสูจน์แล้วว่ามีปัญหา หากคุณอัปเดต Windows ในไม่กี่เดือนที่ผ่านมา มีโอกาสที่การแก้ไข DWM ใด ๆ อาจทำให้ระบบของคุณเสียหาย โปรดดำเนินการด้วยความระมัดระวังและสำรองข้อมูลไว้เมื่อใช้ mods ใน 24H2 และ 25H2

[!WARNING]
โปรดทราบว่าการใช้ธีมนี้เป็นความเสี่ยงของคุณเอง ฉัน ไม่สามารถ รับประกันความเข้ากันได้กับการอัปเดต และไม่รับผิดชอบต่อความเสียหายใด ๆ ฉันทำงานเฉพาะกับ Visual Style และชี้ไปที่เครื่องมือของบุคคลที่สามเท่านั้น mod เดียวที่ฉันสร้างคือ Borders mod ใน repository นี้ ซึ่งสร้างขึ้นสำหรับ 23H2
>
หากคุณแก้ไขไฟล์ระบบใด ๆ ฉัน แนะนำอย่างยิ่ง ให้สำรอง คำแนะนำและไฟล์ทั้งหมด (ทั้งต้นฉบับและใหม่) ไว้ในที่ที่คุณสามารถเข้าถึงได้หากระบบของคุณเสียหาย เพื่อให้คุณสามารถยกเลิกการเปลี่ยนแปลงได้

ภาพตัวอย่าง:

นี่คือลักษณะของธีมขณะใช้งาน จะเห็นได้ว่าไม่ตรง 1:1 และน่าเสียดายเนื่องจากข้อจำกัด ฉันไม่สามารถทำให้มันตรง 1:1 ได้ด้วยเครื่องมือที่ใช้อยู่ในปัจจุบัน

การติดตั้ง

ส่วนนี้ถือว่าคุณได้ patch ระบบเพื่อรองรับธีมของบุคคลที่สามแล้ว หากยังไม่ได้ทำ กรุณาศึกษาคู่มือวิธีการ patch คุณจะต้องใช้เครื่องมือเช่น UltraUXThemePatcher หรือ SecureUxTheme

  • ดาวน์โหลด repository เป็นไฟล์ ZIP และแตกไฟล์ไปยังโฟลเดอร์ที่คุณต้องการ
  • คัดลอกไฟล์จากโฟลเดอร์ 1 - Theme ไปยัง %SYSTEMROOT%\Resources\Themes
  • ทำตามคำแนะนำในโฟลเดอร์ 2 - Borders เพื่อเซ็ตขอบหน้าต่างโดยใช้ Windhawk และ mod Bring Back the Borders!
  • ทำตามคำแนะนำในโฟลเดอร์ 3 - Glass เพื่อเซ็ตเอฟเฟกต์กระจก
  • (ไม่บังคับ) เปิดโฟลเดอร์ 4 - (Optional) Extras%20Extras/) เพื่อดูธีมและทรัพยากรเพิ่มเติม
  • ใช้ธีมโดยคลิกขวาบนเดสก์ท็อป เลือก Personalize จากนั้นไปที่ Themes และเลือกธีมที่ต้องการ

เครดิต

ธีมนี้เป็น "Open Source" ในแง่ที่ทุกคนสามารถนำไปใช้และแก้ไขได้ น่าเสียดายที่มันอยู่ในรูปแบบไบนารี จึงไม่มีโค้ดให้ปล่อย ฉันไม่จำกัดสิทธิ์หรือต้องชำระเงินใด ๆ

ใครก็ได้สามารถคัดลอกหรือแจกจ่ายต่อได้ เงื่อนไขเดียวคือจะต้องฟรีและต้องให้เครดิตทุกคนด้านล่างนี้สำหรับผลงานปรับแต่ง Windows:

  • teknixstuff: สร้าง mod "Bring Back the Borders!" ดั้งเดิมบน Windhawk โดยอิงจาก uDWM patches ของ Ittr
  • Zee-Ay (Za9-118): มีส่วนร่วมกับ Shellstyle dll และสร้างธีม Vista
  • vaporvance: สร้าง Shellstyle dll ในธีม Aero10 ดั้งเดิม
  • Ittr: สร้างแพตช์ uDWM.dll ที่รวมไว้แล้ว
  • SandTechStuff: อัปเดตคำแนะนำเพื่อให้อ่านง่ายขึ้น
  • TheDolphinSlayer: เพิ่มทรัพยากร Windows 7 alt tab
  • OG Nimbi: ดำเนินการโฟลเดอร์ Optional Extras และอัปเดตคำแนะนำและรูปแบบไฟล์หลายไฟล์เพื่อให้อ่านง่ายและจัดการได้สะดวก
  • นักพัฒนาธีม Windows 7 Aero สำหรับ Windows 10/11 ทุกท่าน: ผมนำธีมจากหลายแหล่งมารวมกัน

ผมไม่สามารถรับเครดิตสำหรับธีมนี้ นอกจากบางจุดเท่านั้น ผมแค่นำดีไซน์ Windows 7 มารวมกับ aero.msstyles ของ Windows 11 แล้วนำทุกอย่างมารวมไว้ที่เดียวเพื่อใช้งาน ผมทำสิ่งนี้ด้วยความหวังว่าจะช่วยให้ผู้ใช้ Windows 11 ที่กำลังมองหาธีม Windows 7 ดีๆ สำหรับระบบของตน
