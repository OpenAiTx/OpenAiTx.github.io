Aero7-11

Windows 11 23H2+ ৰ বাবে Aero 7 থীম!

[!NOTE]
Aero7-11 Windows 11-ৰ ডিফল্ট aero.msstyles ব্যৱহাৰ কৰি তৈয়াৰ কৰা হৈছে। এইটো Windows 11 23H2 আৰু 24H2-ত তৈয়াৰ আৰু পৰীক্ষা কৰা হৈছে।
Aero7-11 22H2 ৰ পৰা 23H2-লৈ সমৰ্থন কৰে। 25H2-ত পৰীক্ষা কৰা হোৱা নাই, আৰু 24H2-ত DWM মডসমূহ কিছুটা বাগি হ’ব পাৰে।

সতৰ্কবাণী

[!NOTE]
মই লাইট এপ মোড আৰু ডাৰ্ক Windows মোড ব্যৱহাৰ কৰিবলৈ পৰামৰ্শ দিছোঁ। এইত ডাৰ্ক এপ মোডৰ সমৰ্থন নাই। Windows মোডটো প্ৰাসংগিক নহয়, কিন্তু অধিকাংশ টাস্কবাৰ মডসমূহ ডাৰ্ক বেছত ভাল কাম কৰে লাইট বেছতকৈ।
>
24H2-ত সমস্যাসমূহ দেখা গৈছে। যদি আপুনি বিগত কেইমাহত Windows আপডেট কৰিছে, তেন্তে যিকোনো DWM সংশোধন আপোনাৰ ব্যৱস্থাক ক্ষতি কৰিব পাৰে। অনুগ্ৰহ কৰি সতর্কতাৰে আগবাঢ়ক আৰু 24H2 আৰু 25H2-ত মডসমূহ প্ৰয়োগ কৰোঁতে ব্যাকআপ ৰাখক।

[!WARNING]
অনুগ্ৰহ কৰি মনত ৰাখক, এই থীম ব্যৱহাৰ আপোনাৰ নিজৰ জুখত। মই কোনো আপডেটৰ সৈতে সংগতিত নিশ্চয়তা দিব নোৱাৰোঁ, আৰু কোনো ক্ষতিৰ বাবে দায়ী নহ’ব। মই Visual Style-ৰে কাম কৰোঁ আৰু কেৱল third party tools-লৈ নিৰ্দেশ দিছোঁ। মই তৈয়াৰ কৰা একমাত্ৰ মড এই ৰিপ’জিটৰিত থকা Borders mod, যি 23H2-ৰ বাবে।
>
যদি আপুনি যিকোনো system file সংশোধন কৰে মই অতি পৰামৰ্শ দিছোঁ সকলো নিৰ্দেশনা আৰু file (মূল আৰু নতুন উভয়) এখন সুবিধাজনক ঠাইত সংৰক্ষণ কৰিবলৈ, যাতে আপোনাৰ ব্যৱস্থা বিকল হ’লে যিকোনো পৰিবর্তন উলটাব পৰা যায়।

স্ক্ৰিনশ্বট:

এইটো থীম ব্যৱহাৰকালত। আপুনি দেখিবলৈ পাব, এইটো 1:1 নহয়, আৰু দুঃখজনকভাৱে সমস্যাৰ বাবে মই ইয়াক 1:1 কৰিব নোৱাৰোঁ মোৰ বৰ্তমান টুলৰ সৈতে।

সংস্থাপন

এই অংশত ধৰা হৈছে যে আপুনি ইতিমধ্যে আপোনাৰ ব্যৱস্থা third party themes-ৰ বাবে প্যাচ কৰিছে। যদি কৰা নাই, অনুগ্ৰহ কৰি সেই বিষয়ে গাইডসমূহ চাওক। UltraUXThemePatcher বা SecureUxTheme-ৰ দৰে টুলৰ প্ৰয়োজন হব।

  • ৰিপ’জিটৰিটো ZIP ফাইল হিচাপে ডাউনলোড কৰি আপোনাৰ ইচ্ছা অনুসৰি এখন ফোল্ডাৰত extract কৰক।
  • 1 - Theme ফোল্ডাৰৰ সমগ্ৰ বিষয়বস্তু %SYSTEMROOT%\Resources\Themes-লৈ কপি কৰক।
  • Windhawk আৰু Bring Back the Borders! mod ব্যৱহাৰ কৰি সীমা স্থাপন কৰিবলৈ 2 - Borders ফোল্ডাৰৰ নিৰ্দেশনা অনুসৰণ কৰক।
  • গ্লাস ইফেক্ট স্থাপন কৰিবলৈ 3 - Glass ফোল্ডাৰৰ নিৰ্দেশনা অনুসৰণ কৰক।
  • (ঐচ্ছিক) 4 - (Optional) Extras%20Extras/) ফোল্ডাৰ খুলিব আৰু অতিৰিক্ত থীম আৰু সম্পদসমূহ অন্বেষণ কৰক।
  • থীমটো প্ৰয়োগ কৰিবলৈ আপোনাৰ ডেস্কটপত right-click কৰক, Personalize বাছনি কৰক, তাৰপিছত Themes-লৈ গৈ ইচ্ছা কৰা থীমটো বাছনি কৰক।

ক্ৰেডিটসমূহ

এই থীম "Open Source" অৰ্থাৎ যিকোনো লোকৰ ব্যৱহাৰ আৰু সম্পাদনা কৰিবলৈ মুক্ত। দুঃখজনকভাৱে এইটো binary format-ত, অৰ্থাৎ মই কোনো code মুক্ত কৰিব নোৱাৰোঁ। মই কোনো সীমাবদ্ধতা বা মূল্য আৰোপ নকৰোঁ।

যিকোনো লোক কপি বা পুনঃবিতৰণ কৰিব পাৰে। যদিও, একমাত্ৰ শর্ত হৈছে এইটো সদায় বিনামূল্যে থাকিব লাগিব আৰু Windows customization-ত অৱদান ৰখা তলত দিয়া সকলো লোকক ক্ৰেডিট দিব লাগিব:

  • teknixstuff: Windhawk-ত মূল "Bring Back the Borders!" mod-টো সৃষ্টি কৰে, Ittr-ৰ uDWM patches-ৰ ভিত্তিত।
  • Zee-Ay (Za9-118): Shellstyle dll-ত অৱদান আগবঢ়ায় আৰু Vista Theme সৃষ্টি কৰে।
  • vaporvance: মূল Aero10 থীমত Shellstyle dll সৃষ্টি কৰে।
  • Ittr: uDWM.dll পেটচটো সৃষ্টি কৰি সংলগ্ন কৰা হৈছে।
  • SandTechStuff: পঢ়িবলৈ সুবিধাজনক কৰি নিৰ্দেশনা আপডেট কৰা হৈছে।
  • TheDolphinSlayer: Windows 7 ৰ alt tab resources যোগ কৰা হৈছে।
  • OG Nimbi: Optional Extras ফোল্ডাৰ কাৰ্যকৰী কৰা আৰু বহুতো ফাইলত নিৰ্দেশনা আৰু ফৰ্মেটিং আপডেট কৰি পঢ়িবলৈ আৰু ব্যৱস্থাপনা সহজ কৰা হৈছে।
  • Windows 10/11 ৰ বাবে সকলো Windows 7 Aero থিমৰ ডেভেলপাৰ: মই এইবোৰ থিমৰ পৰা বিভিন্ন অংশ লৈছোঁ।

এই থিমটো বাবে মই ক্ৰেডিট দাবী কৰিব নোৱাৰোঁ, কেৱল কিছুমান ঠাইত। মই Windows 7 ৰ ডিজাইন লৈ, Windows 11 ৰ aero.msstyles লগত মিলাই, তাৰ পিছত সকলো একেলগ কৰি কাৰ্যকৰী কৰিছোঁ। মই এইটো সেই আশাৰে কৰোঁ যাতে Windows 11 ব্যৱহাৰকাৰীসকলে নিজৰ ছিস্টেমৰ বাবে এটা ভাল Windows 7 থিম পাব পাৰে।
