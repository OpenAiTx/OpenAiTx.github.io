Aero7-11

Tema Aero 7 para Windows 11 23H2+!

[!NOTA]
Aero7-11 é feito usando o aero.msstyles padrão do Windows 11. Este tema foi criado e testado no Windows 11 23H2 e 24H2.
Aero7-11 suporta as versões 22H2 até 23H2. Não foi testado em 25H2, e os mods do DWM podem apresentar alguns bugs no 24H2.

Avisos

[!NOTE]
Recomendo usar o modo claro para aplicativos e o modo escuro para o Windows. Este tema NÃO tem suporte para modo escuro de aplicativos. O modo do Windows não importa, mas a maioria dos mods da barra de tarefas funcionam melhor com base escura do que clara.
>
O 24H2 demonstrou apresentar problemas. Se você atualizou o Windows nos últimos meses, há uma chance de qualquer modificação do DWM quebrar seu sistema. Prossiga com cautela e faça backups ao aplicar mods no 24H2 e 25H2.

[!WARNING]
Observe que o uso deste tema é por sua conta e risco. Eu NÃO posso garantir compatibilidade com atualizações, e não posso ser responsabilizado por quaisquer danos. Trabalho apenas com o Estilo Visual e indico ferramentas de terceiros. O único mod que criei é o Borders neste repositório, feito para 23H2.
>
Se você modificar qualquer arquivo do sistema, eu recomendo fortemente manter um backup de TODAS as instruções e arquivos (originais e novos) em um local acessível caso seu sistema seja corrompido, para que você possa desfazer as alterações.

Instalação

Esta seção pressupõe que você já tenha corrigido seu sistema para permitir temas de terceiros. Se não fez isso, consulte guias sobre como proceder. Você precisará usar uma ferramenta como UltraUXThemePatcher ou SecureUxTheme.

  • Baixe o repositório como um arquivo ZIP e extraia em uma pasta de sua preferência.
  • Copie o conteúdo da pasta 1 - Theme para %SYSTEMROOT%\Resources\Themes.
  • Siga as instruções da pasta 2 - Borders para configurar as bordas usando o Windhawk e o mod Bring Back the Borders!.
  • Siga as instruções da pasta 3 - Glass para configurar os efeitos de vidro.
  • (Opcional) Abra a pasta 4 - (Optional) Extras%20Extras/) para explorar temas e recursos adicionais.
  • Aplique o tema clicando com o botão direito na área de trabalho, selecionando Personalizar, depois indo em Temas e escolhendo o tema desejado.

Créditos

Este tema é "Open Source" no sentido de estar aberto para uso e edição por qualquer pessoa. Infelizmente está em formato binário, o que significa que não há código que eu possa liberar. Não imponho restrições ou cobranças sobre isso.

Qualquer pessoa pode criar uma cópia ou redistribuir. Contudo, a única condição é que permaneça gratuito e que todas as seguintes pessoas sejam creditadas por suas contribuições à personalização do Windows:

  • teknixstuff: Criou o mod original "Bring Back the Borders!" no Windhawk, baseado nos patches uDWM de Ittr.
  • Zee-Ay (Za9-118): Contribuiu com o Shellstyle dll e criou o Tema Vista.
  • vaporvance: Criou o Shellstyle dll no tema Aero10 original.
  • Ittr: Criou o patch uDWM.dll incluído.
  • SandTechStuff: Atualizou as instruções para melhor legibilidade.
  • TheDolphinSlayer: Adicionou os recursos do alt tab do Windows 7.
  • OG Nimbi: Implementou a pasta de Extras Opcionais e atualizou instruções e formatação em vários arquivos para facilitar a leitura e o gerenciamento.
  • Todos os desenvolvedores do tema Aero do Windows 7 para Windows 10/11: Eu aproveitei partes de vários desses temas.

Não posso levar o crédito por este tema, exceto em alguns poucos pontos. Eu apenas peguei os designs do Windows 7, mesclei com o aero.msstyles do Windows 11, depois reuni tudo em um só lugar para implementar. Faço isso na esperança de ajudar usuários do Windows 11 que procuram um bom tema do Windows 7 para seus sistemas.
