Aero7-11

Aero 7 Thema für Windows 11 23H2+!

[!HINWEIS]
Aero7-11 wird unter Verwendung des Standard-aero.msstyles in Windows 11 erstellt. Dies wurde unter Windows 11 23H2 und 24H2 erstellt und getestet.
Aero7-11 unterstützt 22H2 bis 23H2. Es wurde in 25H2 nicht getestet, und DWM-Modifikationen können in 24H2 etwas fehlerhaft sein.

Warnungen

[!HINWEIS]
Ich empfehle, den hellen App-Modus und den dunklen Windows-Modus zu verwenden. Dies hat KEINE Unterstützung für den dunklen App-Modus. Der Windows-Modus ist egal, aber die meisten Taskleisten-Mods funktionieren auf dunkler Basis besser als auf heller Basis.
>
24H2 hat nachweislich Probleme. Wenn Sie Windows in den letzten Monaten aktualisiert haben, besteht die Möglichkeit, dass DWM-Modifikationen Ihr System beschädigen könnten. Bitte seien Sie vorsichtig und erstellen Sie Sicherungen, wenn Sie Mods auf 24H2 und 25H2 anwenden.

[!WARNUNG]
Bitte beachten Sie, dass die Nutzung dieses Themes auf Ihr eigenes Risiko erfolgt. Ich kann keine Kompatibilität mit Updates garantieren und hafte nicht für Schäden. Ich arbeite nur mit dem Visual Style und verweise auf Drittanbieter-Tools. Die einzige Modifikation, die ich gemacht habe, ist die Borders-Mod in diesem Repository, die für 23H2 erstellt wurde.
>
Wenn Sie Systemdateien ändern, empfehle ich dringend, ALLE Anweisungen und Dateien (sowohl original als auch neu) an einem Ort zu sichern, den Sie erreichen können, falls Ihr System unbrauchbar wird, sodass Sie Änderungen rückgängig machen können.

Installation

Dieser Abschnitt setzt voraus, dass Ihr System bereits für die Verwendung von Drittanbieter-Themes gepatcht wurde. Falls nicht, lesen Sie bitte Anleitungen dazu. Sie benötigen ein Tool wie UltraUXThemePatcher oder SecureUxTheme.

  • Laden Sie das Repository als ZIP-Datei herunter und entpacken Sie es in einen Ordner Ihrer Wahl.
  • Kopieren Sie den Inhalt des Ordners 1 - Theme nach %SYSTEMROOT%\Resources\Themes.
  • Befolgen Sie die Anweisungen im Ordner 2 - Borders, um die Ränder mit Windhawk und dem Bring Back the Borders!-Mod einzurichten.
  • Befolgen Sie die Anweisungen im Ordner 3 - Glass, um Glaseffekte einzurichten.
  • (Optional) Öffnen Sie den Ordner 4 - (Optional) Extras%20Extras/), um zusätzliche Themes und Ressourcen zu entdecken.
  • Wenden Sie das Theme an, indem Sie mit der rechten Maustaste auf den Desktop klicken, Personalisieren auswählen, dann zu Designs navigieren und das gewünschte Theme auswählen.

Danksagungen

Dieses Theme ist „Open Source“ in dem Sinne, dass es jeder benutzen und bearbeiten kann. Leider liegt es in binärer Form vor, daher kann ich keinen Code veröffentlichen. Ich lege keinerlei Beschränkungen oder Kosten darauf.

Jeder kann eine Kopie erstellen oder es weitergeben. Die einzige Bedingung ist jedoch, dass es kostenlos bleibt und alle folgenden Personen für ihre Beiträge zur Windows-Anpassung genannt werden:

  • teknixstuff: Erstellte das ursprüngliche „Bring Back the Borders!“-Mod auf Windhawk, basierend auf Ittrs uDWM-Patches.
  • Zee-Ay (Za9-118): Trug die Shellstyle-dll bei und erstellte das Vista-Theme.
  • vaporvance: Erstellte die Shellstyle-dll im ursprünglichen Aero10-Theme.
  • Ittr: Hat den uDWM.dll-Patch erstellt, der enthalten ist.
  • SandTechStuff: Hat die Anweisungen zur besseren Lesbarkeit aktualisiert.
  • TheDolphinSlayer: Hat die Windows 7 Alt-Tab-Ressourcen hinzugefügt.
  • OG Nimbi: Hat den Ordner „Optionale Extras“ implementiert und Anweisungen sowie Formatierungen in mehreren Dateien zur besseren Lesbarkeit und einfacheren Verwaltung aktualisiert.
  • Alle Entwickler des Windows 7 Aero-Designs für Windows 10/11: Ich habe Teile aus verschiedenen solchen Designs übernommen.

Ich kann mir dieses Design nur in wenigen Punkten zuschreiben. Ich habe einfach die Windows 7 Designs genommen, sie mit den Windows 11 aero.msstyles zusammengeführt und alles an einem Ort zusammengefügt, um es zu implementieren. Ich tue dies in der Hoffnung, dass es Windows 11-Nutzern hilft, die ein gutes Windows 7-Design für ihr System suchen.
