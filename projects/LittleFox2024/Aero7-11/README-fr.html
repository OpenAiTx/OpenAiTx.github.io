Aero7-11

Thème Aero 7 pour Windows 11 23H2+ !

[!NOTE]
Aero7-11 est réalisé en utilisant le fichier aero.msstyles par défaut de Windows 11. Il a été conçu et testé sur Windows 11 23H2 et 24H2.
Aero7-11 prend en charge les versions 22H2 à 23H2. Il n'a pas été testé sur 25H2, et les modifications DWM peuvent être instables sur 24H2.

Avertissements

[!NOTE]
Je recommande d'utiliser le mode d'application clair et le mode Windows sombre. Ce thème ne prend PAS en charge le mode d'application sombre. Le mode Windows n'a pas d'importance, mais la plupart des mods de la barre des tâches fonctionnent mieux sur une base sombre que sur une base claire.
>
24H2 s'est avéré poser des problèmes. Si vous avez mis à jour Windows ces derniers mois, il est possible que toute modification DWM puisse casser votre système. Veuillez procéder avec prudence et conserver des sauvegardes lors de l'application de mods sur 24H2 et 25H2.

[!WARNING]
Veuillez noter que l'utilisation de ce thème se fait à vos propres risques. Je ne peux pas garantir la compatibilité avec les mises à jour, et je ne peux être tenu responsable des dommages. Je travaille uniquement sur le Style Visuel et je pointe vers des outils tiers. Le seul mod que j'ai créé est le mod Borders dans ce dépôt, qui est conçu pour 23H2.
>
Si vous modifiez des fichiers système, je recommande fortement de conserver une sauvegarde de TOUTES les instructions et fichiers (originaux et nouveaux) dans un endroit accessible si votre système est briqué, afin de pouvoir annuler les modifications.

Installation

Cette section suppose que vous avez déjà patché votre système pour permettre les thèmes tiers. Si ce n'est pas le cas, veuillez consulter des guides à ce sujet. Vous devrez utiliser un outil tel que UltraUXThemePatcher ou SecureUxTheme.

  • Téléchargez le dépôt au format ZIP et extrayez-le dans le dossier de votre choix.
  • Copiez le contenu du dossier 1 - Theme dans %SYSTEMROOT%\Resources\Themes.
  • Suivez les instructions du dossier 2 - Borders pour configurer les bordures avec Windhawk et le mod Bring Back the Borders!.
  • Suivez les instructions du dossier 3 - Glass pour activer les effets de verre.
  • (Optionnel) Ouvrez le dossier 4 - (Optional) Extras%20Extras/) pour découvrir des thèmes et ressources supplémentaires.
  • Appliquez le thème en faisant un clic droit sur votre bureau, en sélectionnant Personnaliser, puis en allant dans Thèmes et en choisissant le thème désiré.

Remerciements

Ce thème est « Open Source » dans le sens où il est ouvert à tous pour utilisation et modification. Malheureusement, il est bloqué dans un format binaire, ce qui signifie qu'il n'y a pas de code à publier. Je n'impose aucune restriction ou paiement.

Chacun peut en faire une copie ou le redistribuer. Cependant, la seule condition est qu'il doit rester gratuit et que toutes les personnes suivantes soient créditées pour leurs contributions à la personnalisation de Windows :

  • teknixstuff : A créé le mod original « Bring Back the Borders! » sur Windhawk, basé sur les patches uDWM d’Ittr.
  • Zee-Ay (Za9-118) : A contribué au Shellstyle dll, et créé le thème Vista.
  • vaporvance : A créé le Shellstyle dll du thème Aero10 original.
  • Ittr : Création du correctif uDWM.dll inclus.
  • SandTechStuff : Mise à jour des instructions pour une meilleure lisibilité.
  • TheDolphinSlayer : Ajout des ressources Alt Tab de Windows 7.
  • OG Nimbi : Mise en place du dossier Extras optionnels et mise à jour des instructions et du formatage sur plusieurs fichiers pour plus de lisibilité et de facilité de gestion.
  • Tous les développeurs de thèmes Windows 7 Aero pour Windows 10/11 : J’ai utilisé des éléments issus de différents thèmes similaires.

Je ne peux pas m’attribuer le mérite de ce thème, sauf à quelques endroits. J’ai simplement pris les designs de Windows 7, les ai fusionnés avec l’aero.msstyles de Windows 11, puis tout rassemblé pour l’implémenter. Je fais cela dans l’espoir que cela aidera les utilisateurs de Windows 11 à trouver un bon thème Windows 7 pour leur système.
