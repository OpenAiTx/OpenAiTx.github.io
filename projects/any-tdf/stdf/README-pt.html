Introdução

Biblioteca de componentes web mobile baseada em Svelte e Tailwind .

Simples • Pequena (Tiny) • Design • Rápida (Fast)

Funcionalidades

  • 🔥 Suporte completo para Svelte v5, Tailwind CSS v4 e TypeScript.
  • 🚀 Sem runtime, sem DOM virtual — executa mais rápido online.
  • 🧰 API rica — configure facilmente os estilos dos componentes conforme suas necessidades.
  • 🍭 Suporta modo escuro e múltiplas configurações de tema.
  • 📖 Documentação extensa em chinês e inglês, além de exemplos de componentes.
  • 🌍 Suporte à internacionalização, com mais de 60 pacotes de idiomas integrados.
  • 🫰 Interação amigável, parâmetros de animação configuráveis e suporte a temas personalizados.
  • 🤝 Suporte a bibliotecas Tailwind como UnoCSS.
  • 🫡 Sem dependências de terceiros — sem preocupações com versões de bibliotecas ou questões de segurança.
  • 📦 Suporte a importação sob demanda para reduzir o tamanho do bundle.
  • 🪜 Ferramenta de scaffolding para criação rápida de projetos.
  • 🔗 Plugin de mesclagem de SVG para unir arquivos SVG rapidamente, com suporte a Iconify.
  • 🍺 Plugin IDE para visualização rápida das APIs dos componentes.

Iniciando

pnpm create stdf@latest
