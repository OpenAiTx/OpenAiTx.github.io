Microsoft Activation Scripts (MAS)

開源的 Windows 和 Office 啟用工具，提供 HWID、Ohook、TSforge、KMS38 及線上 KMS 等多種啟用方式，並包含進階疑難排解功能。

如何啟用 Windows / Office？

方法一 - PowerShell（Windows 8 及以後版本）❤️

  • 開啟 PowerShell
請按下 Windows 鍵 + X，然後選擇 PowerShell 或終端機。

  • 複製並貼上下方代碼，然後按下 Enter。 
irm https://get.activated.win | iex
另外也可以使用以下指令（未來將會淘汰）： 
irm https://massgrave.dev/get | iex

  • 你將會看到啟用選項。請選擇綠色標示的啟用選項。

  • 完成

---

方法二 - 傳統方式（Windows Vista 及以後版本）

點此展開查看
  • 請使用以下其中一個連結下載檔案：
https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip 或 https://git.activated.win/massgrave/Microsoft-Activation-Scripts/archive/master.zip
  • 下載後，右鍵點擊壓縮檔並選擇解壓縮。
  • 在解壓縮後的資料夾中找到名為 All-In-One-Version 的資料夾。
  • 執行 MAS_AIO.cmd 檔案。
  • 你將會看到啟用選項。請依照畫面指示操作。
  • 完成。

  • 若要啟用其他產品，如 macOS 版 Office、Visual Studio、RDS CALs 和 Windows XP，請參閱這裡。
  • 若要以無人值守模式執行腳本，請參閱這裡。

---

無法啟用 ❓

  • 如果你無法使用 PowerShell 方法啟動 MAS，請參考上方的方法二。
  • 若 MAS 已經啟動，腳本卻顯示錯誤訊息，請檢查螢幕上以藍色標示的疑難排解步驟並依照指示操作。
  • 如果遇到其他問題，歡迎隨時聯繫我們 這裡。

---

[!NOTE]
>
- PowerShell 中的 IRM 指令會從指定網址下載腳本，IEX 指令則是執行該腳本。
- 在執行指令之前，請務必再次確認網址正確，並於手動下載檔案時核對來源。
- 請小心，有些人會假冒 MAS，利用不同的網址在 IRM 指令中散播惡意軟體。
