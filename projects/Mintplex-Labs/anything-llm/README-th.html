👉 AnythingLLM สำหรับเดสก์ท็อป (Mac, Windows, & Linux)! ดาวน์โหลดตอนนี้

แอปพลิเคชันแบบฟูลสแตกที่ช่วยให้คุณเปลี่ยนเอกสาร ทรัพยากร หรือเนื้อหาใดๆ ให้กลายเป็นบริบทที่ LLM ใดๆ สามารถใช้เป็นข้อมูลอ้างอิงระหว่างการแชทได้ แอปพลิเคชันนี้เปิดโอกาสให้คุณเลือก LLM หรือ Vector Database ที่ต้องการใช้ รวมถึงรองรับการจัดการผู้ใช้หลายคนและการกำหนดสิทธิ์

ดูตัวอย่างการใช้งาน!

ภาพรวมของผลิตภัณฑ์

AnythingLLM คือแอปพลิเคชันฟูลสแตกที่คุณสามารถใช้ LLM เชิงพาณิชย์ทั่วไปหรือ LLM โอเพ่นซอร์สยอดนิยมและโซลูชัน vectorDB เพื่อสร้าง ChatGPT ส่วนตัวโดยไม่มีข้อจำกัด ซึ่งสามารถรันได้ทั้งในเครื่องของคุณเองหรือโฮสต์แบบรีโมต และสามารถสนทนาอย่างชาญฉลาดกับเอกสารใดๆ ที่คุณให้ไว้

AnythingLLM จะแบ่งเอกสารของคุณออกเป็นอ็อบเจกต์ที่เรียกว่า workspaces โดย Workspace จะทำงานคล้ายกับเธรด แต่เพิ่มความสามารถในการบรรจุเอกสารของคุณเป็นกลุ่ม Workspaces สามารถแชร์เอกสารกันได้ แต่จะไม่สื่อสารข้ามกัน คุณจึงสามารถรักษาบริบทของแต่ละ workspace ให้สะอาดได้

ภาพรวมทางเทคนิค

monorepo นี้ประกอบด้วยส่วนหลัก 6 ส่วน:

  • frontend: ส่วนหน้าด้วย viteJS + React สำหรับสร้างและจัดการเนื้อหาทั้งหมดที่ LLM สามารถใช้ได้อย่างง่ายดาย
  • server: เซิร์ฟเวอร์ NodeJS express สำหรับจัดการอินเทอร์แอคชันทั้งหมด รวมถึงการจัดการ vectorDB และการติดต่อกับ LLM
  • collector: เซิร์ฟเวอร์ NodeJS express สำหรับประมวลผลและแปลงเอกสารจาก UI
  • docker: คำแนะนำการใช้งาน Docker และกระบวนการ build + ข้อมูลสำหรับการคอมไพล์จากซอร์ส
  • embed: ซับโมดูลสำหรับการสร้าง วิดเจ็ตฝังเว็บไซต์
  • browser-extension: ซับโมดูลสำหรับ ส่วนขยายเบราว์เซอร์ Chrome

วิธีตั้งค่าสำหรับการพัฒนา

  • yarn setup เพื่อกรอกไฟล์ .env ที่จำเป็นในแต่ละส่วนของแอปพลิเคชัน (จาก root ของ repo)
  • กรอกข้อมูลให้ครบก่อนดำเนินการต่อ ตรวจสอบให้แน่ใจว่า server/.env.development กรอกครบแล้ว มิฉะนั้นระบบจะทำงานไม่ถูกต้อง
  • yarn dev:server เพื่อบูตเซิร์ฟเวอร์ในเครื่อง (จาก root ของ repo)
  • yarn dev:frontend เพื่อบูตส่วนหน้าของแอปในเครื่อง (จาก root ของ repo)
  • yarn dev:collector เพื่อรัน document collector (จาก root ของ repo)

การเก็บข้อมูลและความเป็นส่วนตัว

AnythingLLM โดย Mintplex Labs Inc มีฟีเจอร์ telemetry สำหรับเก็บข้อมูลการใช้งานแบบไม่ระบุตัวตน

รายละเอียดเพิ่มเติมเกี่ยวกับ Telemetry & Privacy สำหรับ AnythingLLM

ทำไม?

เราใช้ข้อมูลนี้เพื่อช่วยให้เราเข้าใจการใช้งาน AnythingLLM ช่วยในการจัดลำดับความสำคัญของฟีเจอร์ใหม่และแก้ไขบั๊ก รวมถึงปรับปรุงประสิทธิภาพและความเสถียรของ AnythingLLM

การเลือกไม่เข้าร่วม

ตั้งค่า DISABLE_TELEMETRY ใน server หรือ docker .env ของคุณเป็น "true" เพื่อปิดการใช้งาน telemetry คุณยังสามารถทำสิ่งนี้ในแอปได้โดยไปที่แถบด้านข้าง > Privacy แล้วปิดใช้งาน telemetry

เราติดตามอะไรอย่างชัดเจนบ้าง?

เราจะติดตามเฉพาะรายละเอียดการใช้งานที่ช่วยให้เราตัดสินใจเกี่ยวกับผลิตภัณฑ์และเส้นทางการพัฒนาเท่านั้น โดยเฉพาะ:

  • ประเภทของการติดตั้งของคุณ (Docker หรือ Desktop)

  • เมื่อมีการเพิ่มหรือลบเอกสาร โดยไม่มีข้อมูล _เกี่ยวกับ_ เอกสารนั้น แค่เหตุการณ์นั้นเกิดขึ้นเท่านั้น เพื่อให้เราเข้าใจรูปแบบการใช้งาน

  • ประเภทของฐานข้อมูลเวกเตอร์ที่ใช้งาน เพื่อช่วยให้เราจัดลำดับความสำคัญเมื่อมีการอัปเดตสำหรับผู้ให้บริการนั้น

  • ประเภทของผู้ให้บริการ LLM & แท็กโมเดลที่ใช้ เพื่อช่วยให้เราจัดลำดับความสำคัญเมื่อมีการอัปเดตสำหรับผู้ให้บริการหรือโมเดล หรือการผสมผสานต่างๆ เช่น reasoning กับ regular, multi-modal models ฯลฯ

  • เมื่อมีการส่งแชท นี่คือ "เหตุการณ์" ที่เกิดขึ้นบ่อยที่สุดและช่วยให้เราทราบกิจกรรมประจำวันของโปรเจ็กต์นี้ในทุกการติดตั้ง อีกครั้ง เฉพาะ เหตุการณ์ เท่านั้นที่ถูกส่ง - เราไม่มีข้อมูลเกี่ยวกับลักษณะหรือเนื้อหาของแชทเลย

คุณสามารถตรวจสอบได้ด้วยตัวเองโดยค้นหาทุกจุดที่เรียกใช้ Telemetry.sendTelemetry นอกจากนี้ เหตุการณ์เหล่านี้จะถูกบันทึกลงใน log output ด้วย ดังนั้นคุณจะเห็นข้อมูลเฉพาะที่ถูกส่งออกไป - หากเปิดใช้งาน ไม่มีการเก็บข้อมูล IP หรือข้อมูลระบุตัวตนอื่นๆ ผู้ให้บริการ Telemetry คือ PostHog - เป็นบริการเก็บข้อมูล telemetry แบบโอเพ่นซอร์ส

เราให้ความสำคัญกับความเป็นส่วนตัวอย่างจริงจัง และเราหวังว่าคุณจะเข้าใจว่าเราต้องการเรียนรู้วิธีการที่เครื่องมือนี้ถูกใช้งาน โดยไม่ต้องใช้แบบสอบถามกวนใจ เพื่อที่เราจะได้สร้างสิ่งที่มีคุณค่า ข้อมูลนิรนามเหล่านี้จะ _ไม่มีทาง_ ถูกแบ่งปันกับบุคคลที่สามโดยเด็ดขาด
