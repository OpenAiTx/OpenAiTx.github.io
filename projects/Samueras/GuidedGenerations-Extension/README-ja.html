GuidedGenerations-Extension

⭐ 167 stars Japanese by Samueras

SillyTavern用Guided Generations拡張機能

この拡張機能は、元の「Guided Generations」クイックリプライセットの全機能をSillyTavernにネイティブ拡張としてもたらします。ロールプレイ、物語、キャラクター駆動型チャットに理想的な、AI応答を形作り、洗練し、導くためのモジュール式でコンテキスト認識のツールを提供します。すべての機能はSillyTavernのUIに統合された直感的なボタンとメニューからアクセス可能です。

コードレベルのドキュメントはJSDoc.mdを参照してください。

特徴

🐕 ガイド付き応答

AIの返信前に指示を挿入します。
  • 指示を入力して🐕を押します。
  • あなたの指示が次のAI応答を導きます。
  • 動画例

👈 ガイド付きスワイプ

新しい指示で最後のAIメッセージを再生成します。
  • 新しい指示を入力し、👈を押して新しいスワイプを生成します。
  • 最後のメッセージがAIからの場合にのみ利用可能です。
  • 動画例

なりきり（1人称: 👤、2人称: 👥、3人称: 🗣️）

概要を豊かなキャラクター内語りに展開します。
  • 簡単な概要を入力し、視点を選択（設定で切り替え可能）して対応するボタン（👤/👥/🗣️）を押します。
  • あなたの概要が選択された視点からの完全なメッセージに展開されます。
  • 設定により個別に非表示または表示可能。1人称はデフォルトで表示されます。
  • 動画例：
  • 1人称
  • 2人称
  • 3人称

📖 永続ガイドメニュー

永続的なシナリオコンテキストを管理します。
  • 📖 ボタンをクリックして永続ガイドメニューを開きます。
  • ガイドタイプを選択（下記参照）してコンテキストを生成または管理します。

ガイドタイプ：
  • 🗺️ 状況別：最近のチャットやユーザーの焦点からコンテキストを生成。
  • 🧠 思考：キャラクターの思考を生成（自動トリガーはオプション）。
  • 👕 服装：キャラクターの服装を記述（自動トリガーはオプション）。
  • 🧍 状態：キャラクターの位置や状態を詳細に記述（自動トリガーはオプション）。
  • 📜 ルール：物語内のルールを定義または更新。
  • ➕ カスタム：ユーザー定義のコンテキストを注入。

管理操作：
  • ✏️ ガイド編集：ポップアップで既存のガイド注入を修正。
  • 👁️ ガイド表示：すべての有効なガイドを表示。
  • 🗑️ ガイド削除：選択したガイドまたはすべてのガイドを削除。
  • 思考、服装、状態の自動トリガーは設定で切り替え可能。

🔖 ツールメニュー

追加ユーティリティにアクセス
  • 🔧 修正： 最後のAIメッセージをターゲット指示で編集。
  • ✅ スペルチェッカー： 入力の文法、句読点、流れを整える。
  • ✈️ シンプル送信： モデル応答をトリガーせずにユーザーメッセージとして送信。
  • 🖋️ イントロ編集： イントロメッセージを即座に書き換えや変換。
  • ↩️ 入力復元： 以前にクリアした入力を復元。

使用方法

  • すべての主要機能は送信ボタンの隣または左側の歯車メニューにボタンとして表示されます。
  • ホバーツールチップやコンテキストメニューでガイダンスや高度な機能へのクイックアクセスが提供されます。
  • 機能の切り替えや自動ガイド設定についてはアプリ内設定を参照してください。
  • 詳細な技術情報はJSDoc.mdをご覧ください。

⚙️ 設定

すべての拡張機能設定はSillyTavernの拡張機能設定パネルで管理されます：

  • 自動トリガー：以下の自動実行を切り替えます：
  • Thinking Guide
  • State Guide
  • Clothes Guide

  • ボタンの表示：アクションボタンの表示/非表示を切り替えます：
  • 1人称なりきり（👤）
  • 2人称なりきり（👥）
  • 3人称なりきり（🗣️）
  • ガイド付き応答（🐕）
  • ガイド付きスワイプ（👈）
  • 永続ガイドメニュー（📖）

  • 挿入ロール：命令を挿入する際に使用するロール（system、assistant、user）を選択します。

  • デバッグモード：有効にするとブラウザのコンソールに詳細なデバッグ情報を表示します。トラブルシューティングに便利ですが通常使用時はコンソールが煩雑になる可能性があります。

  • プリセット：各ガイド/ツール（Clothes, State, Thinking, Situational, Rules, Custom, Corrections, Spellchecker, Edit Intros, 1/2/3人称なりきり）に対して任意のSillyTavernプリセットを選択できます。ガイド/ツール実行前に拡張機能はそのプリセット（および設定されたAPI/モデル）に切り替え、動作を実行した後に前のプリセットに戻します。これによりガイドごとに異なるモデルを使用可能です。

  • プロンプトオーバーライド：各ガイド/ツールの生のプロンプトテンプレートをカスタマイズします。{{input}}を入力テキストとして使用し、サポートされる他のプレースホルダーも利用可能です。オーバーライド可能なプロンプト：
  • Clothes Guide プロンプト
  • State Guide プロンプト
  • Thinking Guide プロンプト
  • Situational Guide プロンプト
  • Rules Guide プロンプト
  • Corrections プロンプト
  • Spellchecker プロンプト
  • 1人称/2人称/3人称のプロンプトのなりきり機能
  • ガイド付き応答プロンプト
  • ガイド付きスワイププロンプト

トラブルシューティング

  • ボタンが表示されない場合: SillyTavernが最新（v1.12.9以上）であり、LALibがインストールされ有効になっているか確認してください。
  • コンテキストメニューが表示されない場合: チャットを切り替えるか、クイックリプライメニューで拡張機能を再追加してみてください。
  • その他の問題: SillyTavernを再起動し、アップデートを確認し、SillyTavernのドキュメントをご参照ください。
