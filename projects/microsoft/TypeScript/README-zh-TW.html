TypeScript

TypeScript 是一種用於應用程式規模 JavaScript 的語言。TypeScript 為 JavaScript 增加了可選型別，支援大型 JavaScript 應用程式的工具，適用於任何瀏覽器、任何主機、任何作業系統。TypeScript 會編譯成可讀性高、符合標準的 JavaScript。您可以在 playground 試用，並透過 我們的部落格 和 Twitter 帳號 獲得最新資訊。

在 我們的社群頁面 找到其他正在使用 TypeScript 的人。

安裝

取得最新穩定版本：

npm install -D typescript

取得我們的夜間版本：

npm install -D typescript@next

貢獻

有許多方式可以貢獻給 TypeScript。

  • 提交錯誤回報 並協助我們在修正後進行驗證。
  • 檢視原始碼變更。
  • 在 StackOverflow 上與其他 TypeScript 使用者和開發者互動。
  • 在 TypeScript 社群 Discord 上互相幫助。
  • 加入 Twitter 上的 #typescript 討論。
  • 貢獻錯誤修正。
