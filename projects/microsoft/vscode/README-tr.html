Visual Studio Code - Açık Kaynak ("Code - OSS")

Depo

Bu depo ("Code - OSS"), Visual Studio Code ürününü topluluk ile birlikte geliştirdiğimiz yerdir (Microsoft olarak). Burada sadece kod ve sorunlar üzerinde çalışmakla kalmayıp, aynı zamanda yol haritamızı, aylık iterasyon planlarımızı ve endgame planlarımızı da yayımlıyoruz. Bu kaynak kodu, standart MIT lisansı kapsamında herkesin kullanımına açıktır.

Visual Studio Code

Visual Studio Code, Microsoft'a özgü özelleştirmelerle birlikte gelen, geleneksel bir Microsoft ürün lisansı altında yayımlanan Code - OSS deposunun bir dağıtımıdır.

Visual Studio Code, bir kod editörünün sadeliğini, geliştiricilerin temel düzenleme-derleme-hata ayıklama döngüsü için ihtiyaç duyduğu her şeyle birleştirir. Kapsamlı kod düzenleme, gezinme ve anlama desteğiyle birlikte hafif hata ayıklama, zengin bir genişletilebilirlik modeli ve mevcut araçlarla hafif entegrasyon sunar.

Visual Studio Code, her ay yeni özellikler ve hata düzeltmeleriyle güncellenir. Visual Studio Code'un web sitesinden Windows, macOS ve Linux için indirebilirsiniz. En son sürümleri her gün almak için Insiders sürümünü yükleyin.

Katkıda Bulunma

Bu projeye katılabileceğiniz birçok yol vardır, örneğin:

  • Hata ve özellik istekleri gönderin ve eklenenleri doğrulamamıza yardımcı olun
  • Kaynak kod değişikliklerini inceleyin
  • Dokümantasyonu gözden geçirin ve yazım hatalarından yeni içerik eklemeye kadar her konuda pull request gönderin

Sorunları düzeltmek ve doğrudan kod tabanına katkı sağlamak istiyorsanız, lütfen aşağıdaki konuları içeren Nasıl Katkıda Bulunulur belgesine göz atın:

  • Kaynak koddan nasıl derlenir ve çalıştırılır
  • Geliştirme iş akışı, hata ayıklama ve test çalıştırma dahil
  • Kodlama yönergeleri
  • Pull request gönderme
  • Üzerinde çalışılacak bir sorun bulma
  • Çevirilere katkıda bulunma

Geri Bildirim

  • Stack Overflow'da bir soru sorun
  • Yeni bir özellik isteği gönderin
  • Popüler özellik isteklerine oy verin
  • Bir sorun bildirin
  • GitHub Discussions veya Slack üzerinden eklenti geliştirici topluluğu ile bağlantı kurun
  • @code hesabını takip edin ve görüşlerinizi bizimle paylaşın!

Bu kanalların her birinin açıklaması ve diğer topluluk odaklı kanallar hakkında bilgi için wiki'mize göz atın.

İlgili Projeler

VS Code'un temel bileşenlerinin ve uzantılarının birçoğu GitHub'da kendi depolarında bulunmaktadır. Örneğin, node debug adapter ve mono debug adapter depoları birbirinden ayrıdır. Tüm listenin tamamı için wiki üzerindeki İlgili Projeler sayfasını ziyaret edin.

Dahili Uzantılar

VS Code, birçok dil için dilbilgisi ve snippet içeren extensions klasöründe bulunan bir dizi yerleşik uzantı ile birlikte gelir. Bir dil için zengin dil desteği (kod tamamlama, Tanıma Git) sağlayan uzantılar language-features ekiyle adlandırılır. Örneğin, json uzantısı JSON için renklendirme sağlarken, json-language-features uzantısı JSON için zengin dil desteği sunar.

Geliştirme Konteyneri

Bu depo, Visual Studio Code Dev Containers / GitHub Codespaces geliştirme konteynerini içerir.

  • Dev Containers için, Dev Containers: Clone Repository in Container Volume... komutunu kullanarak macOS ve Windows'ta daha iyi disk G/Ç için bir Docker volume oluşturabilirsiniz.
  • Zaten VS Code ve Docker kuruluysa, başlamak için buraya tıklayabilirsiniz. Bu, gerekirse Dev Containers uzantısının otomatik olarak yüklenmesini, kaynak kodun bir konteyner volume'una klonlanmasını ve kullanıma hazır bir geliştirme konteynerinin başlatılmasını sağlar.

  • Codespaces için, VS Code'da GitHub Codespaces uzantısını kurun ve Codespaces: Create New Codespace komutunu kullanın.

Docker / Codespace'in tam derleme için en az 4 Çekirdek ve 6 GB RAM (8 GB önerilir) olması gerekmektedir. Daha fazla bilgi için geliştirme konteyneri README dosyasına bakınız.
